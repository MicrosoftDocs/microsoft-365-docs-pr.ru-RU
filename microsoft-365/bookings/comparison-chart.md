# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Диаграмма сравнения: сравнение веб-приложения Bookings и приложения Bookings Teams

Приложение Bookings в Teams позволяет расписаниям выполнять свои основные задачи и изменять некоторые параметры. Однако веб-сайт Bookings включает расширенные функции и параметры, которые пока недоступны в приложении Teams.

Функции приложения Teams добавляются постоянно, и мы продолжим обновлять этот список. Дополнительные сведения см. в разделе "Все функции исходного веб-приложения **Bookings в Microsoft Teams"?** [](bookings-faq.md)

| Возможность | Веб-приложение Bookings | Приложение Bookings Teams |
|:---|:---|:---|
| Создание новых календарей резервирования | Да | Да |
| Добавление и удаление сотрудников из календаря резервирования | Да | Да |
| Создание новых типов встреч | Да | Да |
| Расписание онлайн-резервирования | Да | Да |
| Изменение бизнес-сведений | Да | Да |
| Добавление сотрудников с разрешениями гостя для людей за пределами организации | Да | Нет |
| Запланировать резервирование на личном или автономном режиме | Да | Нет |
| Запланировать резервирование групп или нескольких клиентов | Да | Нет |
| Настройка логотипа для бизнеса | Да | Нет |
| Настройка часов работы | Да | Нет |
| Публикация страницы самостоятельного планирования | Да | Нет |
| Управление контактами клиентов | Да | Нет |
| Настройка времени отключки для сотрудников | Да | Нет |
