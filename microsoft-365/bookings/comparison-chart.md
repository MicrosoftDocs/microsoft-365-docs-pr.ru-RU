# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Диаграмма сравнения. Бронирование веб-приложения против Teams приложения

Приложение Bookings в Teams позволяет расписаниям выполнять основные задачи и изменять некоторые параметры. Однако веб-сайт Bookings включает расширенные функции и параметры, которые еще не доступны в Teams приложении.

Teams постоянно добавляются функции приложения, и мы будем продолжать обновлять этот список. Дополнительные сведения см. в разделе Все ли функциональные возможности исходного веб-приложения **Bookings доступны в Microsoft Teams?** в разделе [FAQ.](bookings-faq.yml)

| Возможность | Бронирование веб-приложения | Бронирование Teams приложение |
|:---|:---|:---|
| Создание новых календарей бронирования | Да | Да |
| Добавление и удаление сотрудников из календаря бронирования | Да | Да |
| Создание новых типов встреч | Да | Да |
| Расписание онлайн-бронирований | Да | Да |
| Изменение бизнес-сведений | Да | Да |
| Добавление сотрудников с гостевых разрешений для людей за пределами вашей организации | Да | Нет |
| Запланировать бронирование в личном или автономном режиме | Да | Нет |
| Расписание групповых или много клиента бронирований | Да | Нет |
| Настройка логотипа для бизнеса | Да | Нет |
| Настройка часов работы | Да | Нет |
| Публикация страницы планирования самообслуживки | Да | Нет |
| Управление контактами клиентов | Да | Нет |
| Настройка времени для сотрудников | Да | Нет |
