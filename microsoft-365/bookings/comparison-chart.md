# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Сравнительная диаграмма: "книги" — "веб-приложение" и "книги" — приложение Teams

Приложение для работы с книгами в Teams позволяет планировщикам обрабатывать свои основные задачи и изменять некоторые параметры. Тем не менее, в веб-сайте резервирования есть расширенные функции и параметры, которые пока недоступны в приложении Teams.

Компоненты приложений Teams добавляются все время, и мы будем продолжать обновлять этот список. В разделе вопросы и ответы для получения дополнительных сведений ознакомьтесь с **разделом** в разделе [вопросы и ответы](bookings-faq.md) .

| Компонент | Веб-приложение резервирования | Приложение Teams для работы с книгами |
|:---|:---|:---|
| Создание новых календарей резервирования | Да | Да |
| Добавление и удаление персонала из календаря резервирования | Да | Да |
| Создание новых типов встреч | Да | Да |
| Планирование онлайновых резервирований | Да | Да |
| Изменение бизнес-сведений | Да | Да |
| Добавление сотрудников с разрешениями гостя для людей за пределом Организации | Да | Нет |
| Планирование для пользователей или автономных резервирований | Да | Нет |
| Группа расписаний или несколько пользовательских резервирований | Да | Нет |
| Настройка логотипа для вашего бизнеса | Да | Нет |
| Настройка рабочих часов | Да | Нет |
| Публикация страницы планирования самообслуживания | Да | Нет |
| Управление контактами клиента | Да | Нет |
| Задать нерабочее время для персонала | Да | Нет |
