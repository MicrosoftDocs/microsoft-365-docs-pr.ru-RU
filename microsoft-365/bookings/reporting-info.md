# <a name="reporting-info-for-bookings"></a>Отчетные сведения для Bookings

Теперь вы можете просмотреть 120-дневное представление календаря Booking в TSV-файле. TSV-файлы — это файлы значений, разделенные вкладками. Вы можете просматривать или редактировать подобный файл с помощью любого текстового редактора или программы электронных таблиц, например Excel.

## <a name="see-120-days-booking-activity"></a>См. действия по резервированиям за 120 дней

1. На информационной панели календаря резервирования выберите "Экспорт дополнительных данных в качестве TSV-данных".

:::image type="content" source="../media/bookings-activity.png" alt-text="Screenshot: 30 days Booking activities":::

1. Сохраните файл с новым именем и укажите формат XLS или XLSX.

1. Откройте файл, чтобы просмотреть 120-дневное представление календаря Резервирования.

Файл содержит строку для каждого события в календаре резервирования за предыдущие 60 и ближайшие 60 дней. Каждая строка содержит сведения о дате и времени для события, имя клиента, электронную почту клиента, номер телефона клиента, адрес клиента, назначенного персонала, выбранную службу, расположение и длительность встречи.
