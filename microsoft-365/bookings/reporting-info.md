# <a name="reporting-info-for-bookings"></a>Отчеты о резервировании

Теперь вы можете просмотреть 120 день в календаре резервирования в TSV файле. TSV — файлы с разделителями табуляцией. Вы можете просмотреть или изменить файл, подобный этому, с помощью любого текстового редактора или программы для работы с электронными таблицами, например Excel.

## <a name="see-120-days-booking-activity"></a>Просмотр действий по резервированию 120 дней

1. На панели мониторинга календарь резервирования выберите пункт Экспорт дополнительных данных в виде TSV.

:::image type="content" source="../media/bookings-activity.png" alt-text="Снимок экрана: действия по резервированию за 30 дней":::

1. Сохраните файл под новым именем и укажите формат XLS или XLSX.

1. Откройте файл, чтобы просмотреть представление календаря резервирования 120 дня.

Файл содержит строку для каждого события в календаре резервирования за последние 60 дней, а также ближайшие 60 дней. Каждая строка содержит сведения о дате и времени для события, имени клиента, адреса электронной почты клиента, номера телефона клиента, адреса клиента, назначенного сотрудника, службы, выбранного местоположения и длительность встречи.
