# <a name="cortana-in-microsoft-365"></a>Кортана в Microsoft 365

Кортана — ваш персональный помощник по повышению производительности, предлагает возможности на основе ИИ, чтобы сэкономить время и сосредоточиться на наиболее важных возможностях. Кортана поможет пользователям увеличить свою личность на весь день как на работе, так и на вжимании. При входе с помощью действительных рабочих или учебных учетных записей пользователи могут получить облачные услуги с помощью Кортаны в средах Microsoft 365, соответствующих требованиям корпоративного уровня конфиденциальности, безопасности и соответствия требованиям в Office 365 ("Корпоративные службы Кортаны"). 

Используя личный помощник по вышестоящему продукту, Кортана создана для предоставления функций, безопасно и безопасных обрабатываемых данным Office 365, таких как сообщения электронной почты, файлы, чаты и т. д., для экономии времени, повышения эффективности и повышения производительности труда пользователей.

Двигаясь вперед, мы фокусируемся на производительности предприятия.

- Как и в случае других служб Office 365, корпоративные службы Кортаны соответствуют тем же требованиям конфиденциальности, безопасности и соответствия требованиям, что и в Условиях использования [веб-служб (OST).](https://www.microsoft.com/licensing/product-licensing/products)

- Новые возможности Microsoft 365, такие как "Краткое сообщение" и "Воспроизведение моих сообщений", будут включены с помощью корпоративных служб Кортаны и полностью будут выполняться при помощи этих обещаний. В настоящее время эти функции доступны на английском языке (США) для пользователей с почтовыми ящиками в регионе США, а также для сотрудников многонациональных организаций с почтовыми ящиками в США. Дополнительные сведения о поиске расположения использования см. в разделе "Просмотр [дополнительных свойств для учетных записей".](https://docs.microsoft.com/microsoft-365/enterprise/view-user-accounts-with-microsoft-365-powershell?view=o365-worldwide#view-additional-property-values-for-accounts)

- Пользователи могут подключаться к корпоративным службам Кортаны, описанным здесь, через Кортану в Windows 10 (версии 2004 и более поздних версий), а также клиентские приложения, такие как Outlook для iOS и Android, с помощью отдельных условий лицензирования. 

- Существующие пользовательские интерфейсы, включая Кортану в Windows 10 (версии 1909 и более ранних версий), а также приложение для Кортаны на iOS и Android, регулируется соглашением об условиях обслуживания [Майкрософт](https://www.microsoft.com/licensing/product-licensing/products) [и Заявлением о конфиденциальности](https://privacy.microsoft.com/privacystatement) Майкрософт (см. раздел "Существующие службы для потребителей" ниже). На данный момент термины также регулирует корпоративные службы Кортаны, предоставленные пользователю, выполнив вход в систему с помощью его потребительских учетных данных.

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Какие данные обрабатываются корпоративными службами Кортаны? 

Для обработки запросов пользователя Кортаны, данных Office необходимы для выполнения запросов пользователей и других телеметрических данных, созданных системами Майкрософт. Данные, собираемые корпоративными службами Кортаны, включают текстовое представление запросов пользователя (транскрибирование от распознавания речи). Эти текстовые данные предусмана в соответствии с условиями [использования веб-служб.](https://www.microsoft.com/licensing/product-licensing/products) Она используется только для разработки и улучшения моделей машинного обучения в соответствии с условиями использования веб-служб.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Какая модель управления для данных клиентов в корпоративных службах Кортаны?

Согласно другим службам Office 365, корпоративные службы Кортаны защищены и регулируются [условиями онлайн-служб.](https://www.microsoft.com/licensing/product-licensing/products) Сюда входит набор обещаний для защиты данных клиента от случайной потери, настройки, несанкционированного раскрытия или доступа или незаконного уничтожения. Для данных клиента также распространяются строгие ограничения доступа. Корпорация Майкрософт использует Данные клиента только для предоставления услуг, полюбившихся в качестве оних служб и совместимых с этими службами. Дополнительные сведения см. в следующей таблице.

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Корпоративные службы Кортаны в приложениях Microsoft 365

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Кортана в Windows 10 (версия 2004 и более поздние)

Приложение Кортаны в Windows 10 помогает пользователям быстро получать сведения в Microsoft 365, используя типизированные или произносимые запросы для связи с людьми, проверки календаря, настройки напоминаний и прочего.

Кортана в Windows может облегчить управление расписанием и задачами. Они могут добавлять их в свои списки в Microsoft To-Do, искать локальные сведения, получать определения и отслеживать новейшие новости, сведения о финансовом языке и финансов поисковой системе Bing.

Кортана в Windows 10 версии 2004 и более поздних версиях соответствует нормативным требованиям предприятия к конфиденциальности, безопасности и соответствию требованиям для корпоративных служб Кортаны, как указано в условиях использования [онлайн-служб (OST).](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>Как отказаться от Кортаны в Windows 10

Администраторы могут настроить Кортану в Windows 10 для своей организации с помощью политики MDM Experience\AllowCortana или через групповую политику: Конфигурация компьютера\Административные шаблоны\Компоненты Windows\Поиск\Разрешить Использование Кортаны.

Начиная с Windows 10 версии 2004 Кортана — это приложение универсальной платформы Windows, предустановленное в Windows. Регулярно обновляется через Microsoft Store. Чтобы получать последние обновления Кортаны, необходимо включить [обновления через Microsoft Store.](https://docs.microsoft.com/windows/configuration/stop-employees-from-using-microsoft-store)

[Дополнительные сведения о Кортане в Windows 10](https://docs.microsoft.com/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>Помощь в работе с голосовой связью Кортаны в Teams

Помощь в мобильном приложении Teams для работы с мобильным приложением Teams позволяет пользователям Microsoft 365 корпоративный упрощать задачи связи, взаимодействия и усилий, связанные с собраниями, с помощью эссегольного естественного языка. Чтобы говорить с Кортаной, пользователи могут щелкнуть кнопку микрофона, расположенную в правом верхнем углу мобильного приложения Teams. Пользователи могут связаться с кому-то в ходе миграции, отправив голосовый запрос, например "Звонок меган" или "отправить сообщение следующему собранию". Пользователи также могут присоединяться к собраниям, сказав сообщение "Присоединиться к следующему собранию" и восположить помощь голосовой масикой, чтобы обмениваться файлами, проверять календарь и т. д. Эти возможности голосовой связи поддерживаютиспользование служб корпоративного уровня Кортаны, которые полностью удовлетворяют требованиям конфиденциальности, безопасности и соответствия требованиям Office 365, как указано в Условиях использования [веб-служб (OST).](https://www.microsoft.com/licensing/product-licensing/products)

**Управление для администраторов**

Поддержка голосовой связи Кортаны будет включена по умолчанию для клиентов. Администраторы могут управлять тем, кто в клиенте может использовать помощь через голосовую связь Кортаны в Teams, с помощью политики (TeamsCortanaPolicy). Эту политику можно установить на уровне учетной записи пользователя или клиента. Администраторы также могут использовать поле CortanaVoiceInvocationMode с этим параметром управления политикой, чтобы определить, отключена ли Кортана, включается только вызов кнопки пальцем или включен с помощью принудительного вызова кнопки (применимо к устройствам, поддерживающим эту функцию). Обратите внимание, что во время первоначального выпуска для пользователей Microsoft 365 корпоративный в Английском языках мобильное приложение Teams не поддерживает активацию прозрачных слов, но это приложение сможет поддерживаться в будущем.

**Пользовательский элемент управления**

Отдельные пользователи могут попробовать использовать помощь с кортаной в мобильном приложении Teams, нажав кнопку микрофона. Кроме того, они могут контролировать, включена ли Кортана в Teams на своем устройстве, используя параметр в мобильном приложении Teams. 

1. Откройте мобильное приложение Teams.

2. Перейдите в раздел **"Параметры".**

3. Выберите **Кортану.**

4. Переключатель в положение в положение Вкл. или выключаете, в зависимости от того, нужна ли помощь с голосовой связью Кортаны на этом устройстве.

[Дополнительные сведения о работе с голосовой связью Кортаны в Teams](https://support.microsoft.com/en-us/office/using-cortana-voice-assistance-in-teams-274bb2f0-d962-4182-b45d-307435cea256?ui=en-us&rs=en-us&ad=us)

### <a name="play-my-emails"></a>Воспроизведение моих сообщений электронной почты

Воспроизведение сообщений электронной почты (подключенных к Outlook Mobile) — это функция голосового ввода, при которой пользователи могут прослушивать в своей почты новые сообщения в отсортированных отрывах, а изменения в день посредством динамиков на телефоне, наушниках или подключенных голосовых устройствах. Пользователи могут попросить Кортану прочитать свои последние сообщения электронной почты вслух, а Кортана будет выполнять такие действия, как пометка, архивация, удаление и пропуск сообщений. Эта функция особенно полезна для ознакомления с электронной почтой в ходе работы, работы с несколькими задачами или в пути. Когда пользователь взаимодействует с Кортаной в разделе Воспроизведение моих сообщений электронной почты, голосовой запрос поступит непосредственно в корпоративные службы Кортаны. Чтение текста в речь в электронной почте пользователя обрабатывается в облаке Office 365. Во время этого процесса данные Office 365 не обрабатываются на мобильном устройстве пользователя, а данные электронной почты не сохраняются. Запись в радуссы произносимой команды (т. е. "пометить как прочитанное", "флажок" и т. д.) могут храниться в соответствии с условиями защиты данных, указанными в условиях использования Microsoft [Online Services.](https://www.microsoft.com/licensing/product-licensing/products)

Кортана будет выведена, когда сообщение защищено, и кратко приостановится перед чтением сообщения, чтобы предоставить пользователю достаточно времени на приостановку воспроизведения или переходка к следующему сообщению. Как и в случае частного телефонного звонка, пользователям следует соблазнуться осторожностью при запуске воспроизведения в местах, где конфиденциальные данные могут быть переслышкой. В таких случаях при использовании функции воспроизведения моих сообщений электронной почты в Outlook Mobile сотрудникам вашей организации рекомендуется использовать голосовые наушники.

### <a name="opt-out-of-play-my-emails"></a>Отказ от воспроизведения моих сообщений электронной почты

Отдельные лица могут отказаться от воспроизведения моих сообщений электронной почты, выполнив следующие действия.

1. Откройте Outlook Mobile.

2. Перейдите в раздел **"Параметры".**
  
3. Выберите **"Воспроизведение сообщений электронной почты".**

4. Переместите переключатель в положение выключите учетные записи, которые нужно отключить.

[Дополнительные сведения о воспроизведении сообщений электронной почты](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>Краткое сообщение

Кортана отправляет персонализированное краткое сообщение вместе с задачами и обязательствами, сделанными вами с помощью простого способа пометить их как **завершенные** или запланировать время фокусировки для выполнения. Кроме того, она включает в себя сводка собраний и соответствующих документов за день. Кортана извлекает сведения из сообщений электронной почты пользователя и сохраняет их в почтовом ящике Exchange Online, пока не будет консолидировано в совокупности электронной почты. В любой момент доступны персональные данные за пределами вашего почтового ящика Exchange Online.

### <a name="how-to-opt-out-of-briefing-email"></a>Как отказаться от свободного письма

Администраторы могут настраивать короткие меры в своей организации с [помощью PowerShell](https://docs.microsoft.com/briefing/be-admin) в Exchange Online. Пользователи могут отказаться от краткосрочной работы Кортаны, нажав **кнопку "Отменить подписку"** в нижнем колонтитуле сообщения.

[Дополнительные сведения о краткой почте](https://support.microsoft.com/help/4558259)

Мы будем продолжать вводить больше функций, таких как описано выше, для повышения эффективности вашей организации.

[Дополнительные сведения о предложениях для соответствия требованиям Майкрософт](https://docs.microsoft.com/microsoft-365/compliance/offering-home)

## <a name="how-is-the-delivery-of-cortana-enterprise-services-different-from-the-delivery-of-other-cortana-features-i-may-have-previously-experienced"></a>Как доставка корпоративных служб Кортаны отличается от доставки других функций Кортаны?

Ниже приведено два подхода к работе Кортаны на вашем предприятии.

**Новые возможности для организаций, использующих корпоративные**службы Кортаны: корпоративные службы Кортаны предназначены для удовлетворения потребностей организаций в области безопасности и соответствия требованиям: 

1. Это новая служба, и она рассматривается в этом документе.

2. Обработчиком данных, для которых действует Условия использования веб-служб, корпорация Майкрософт собирает и использует Данные клиента только для предоставления клиентами веб-служб, запрашиваемых нашими клиентами, и для целей, указанных нашими клиентами. В Соответствии с Общим регламентом по защите данных В ЕС клиент выступит в отношении их данных. См. раздел ["Условия использования веб-служб"](https://www.microsoft.com/licensing/product-licensing/products) и "Общие сведения о дополнительной прозрачности [конфиденциальности" для наших коммерческих облачных клиентов.](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)

3. Например, воспроизведение моих сообщений электронной почты — это служба Кортаны, которую пользователи могут подключаться через Outlook для iOS и использовать корпоративные службы Кортаны. 

4. У ИТ-администраторов всегда будут элементы управления для необязательных сетевых функций кортаны, аналогичных необязательным сетевым функциям при использовании приложений Office профессиональный плюс.  

**Существующие службы для потребителей:** необязательные связанные с Кортаной службы создаются в основном для потребительских интерфейсов и в настоящее время доставляются в Windows 10 (версия 1909 и более ранних версий) и приложение Кортаны на iOS и Android.

1. Эти возможности дают возможность использовать такие функции, как прогноз погода, новостей и трафик.

2. Администраторы клиента могут управлять возможностью использования Кортаны в Windows 10 (версия 1909 и более ранних версий), а также с помощью приложения Кортаны на iOS и Android разрешить Кортане подключаться к данным клиента Office 365.

Отключить доступ Кортаны к данным вашей организации, размещенным в корпорации Майкрософт

1. В Центре администрирования Microsoft 365 выберите **"Параметры**  >  **организации" и** выберите **"Параметры кортаны".**

2. Снимите флажок **"Разрешить Использование Кортаны" в Windows 10 (версия 1909 и более ранних версий), а также приложение Кортаны на iOS и Android, чтобы получать доступ к данным, размещенным** в корпорации Майкрософт, от имени пользователей в вашей организации для отключения сетевых функций Кортаны.

3. Нажмите кнопку **Сохранить изменения**.
