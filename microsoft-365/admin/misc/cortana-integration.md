# <a name="cortana-in-microsoft-365"></a>Кортана в Microsoft 365

Кортана, ваш персональный помощник по работе, предоставляет возможности для экономии времени, а также уделять внимание большей важности. Кортана поможет вашим пользователям повысить производительность труда в течение всего дня и на рабочем, и в жизни. При входе с использованием действительных рабочих или учебных учетных записей пользователи могут получать облачные службы помощи Кортаны в Microsoft 365, которые отвечают за конфиденциальность, безопасность и соответствие требованиям Office 365 на уровне предприятия ("Кортана Enterprise Services"). 

В качестве персонального помощника по повышению производительности Кортана предназначена для обеспечения надежной и безопасной обработки и причин последующего использования данных Office 365, таких как электронные письма, файлы, беседы и т. д., для экономии времени, повышения эффективности и повышения производительности пользователей.

При переходе на более высокую производительность мы сосредоточены на Кортане в масштабах предприятия.

- В соответствии с другими службами Office 365, службы Кортаны Enterprise Services отвечают тому же уровню конфиденциальности, безопасность и соответствие требованиям на уровне предприятия, что отражено в [терминах веб-служб (OST)](https://www.microsoft.com/licensing/product-licensing/products).

- Новые возможности Microsoft 365, такие как информационный бюллетень и воспроизводимые сообщения электронной почты, будут включены с использованием служб Кортаны Enterprise Services и полностью соответствуют этим обещаниям. В настоящее время эти функции доступны на английском (США) рынке для пользователей с почтовыми ящиками в компании US GEO, а также для сотрудников с несколькими национальными национальными почтовыми ящиками, основанными на US. Дополнительные сведения о том, как найти место использования, можно найти в разделе [Просмотр дополнительных значений свойств для учетных записей](https://docs.microsoft.com/microsoft-365/enterprise/view-user-accounts-with-microsoft-365-powershell?view=o365-worldwide#view-additional-property-values-for-accounts).

- Пользователи могут подключаться к Enterprise Services Кортаны Enterprise Services с помощью Кортаны в Windows 10 (версия 2004 и более поздние версии), а также клиентские приложения, такие как Outlook для iOS и Android, подчиняются отдельным условиям лицензирования. 

- Существующие возможности для потребителей, в том числе Кортана в Windows 10 (версия 1909 и более ранние версии) и приложение Кортана на iOS и Android, управляются [соглашением об использовании служб Майкрософт](https://www.microsoft.com/licensing/product-licensing/products) и [заявлением о конфиденциальности Майкрософт](https://privacy.microsoft.com/privacystatement) (раздел "существующие службы для потребителей" ниже). Эти термины также будут управлять службами Кортаны Enterprise Services, предоставленными пользователю при входе с использованием учетных данных клиента.

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Какие данные обрабатываются службами Кортаны Enterprise? 

Службы кортаны Enterprise Services обрабатывают запросы от пользователя, данные Office, необходимые для выполнения запроса пользователя, и другие телеметрии, создаваемые системами Microsoft для запуска службы. Данные, собираемые службами Кортаны Enterprise Services, включают текстовое представление голосовых запросов пользователя (например, для распознавания речи). Это текстовые данные, которые управляются в соответствии с  [условиями веб-служб](https://www.microsoft.com/licensing/product-licensing/products). Он используется только для разработки и усовершенствования моделей машинного обучения, совместимых с терминами интерактивной службы.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Что такое модель управления для данных клиента в службах Кортаны Enterprise?

Согласованность с другими службами Office 365, службы Кортаны Enterprise Services защищены и могут подчиняться [условиям интернет-служб](https://www.microsoft.com/licensing/product-licensing/products). Это включает набор обещаний для защиты данных клиентов от случайных потерь, изменения, несанкционированного разглашения, доступа или уничтожения незаконному. Кроме того, для данных клиентов действуют ограничения на ограниченный доступ. Корпорация Майкрософт использует данные клиентов только для предоставления согласованных служб, а также для целей, совместимых с этими службами. В приведенной ниже таблице содержится подробная информация.

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Функции кортаны Enterprise Services в Microsoft 365

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Кортана в Windows 10 (версия 2004 и более поздние)

Приложение Кортаны в Windows 10 помогает пользователям быстро получать информацию в Microsoft 365, используя типизированные или речевые запросы для связи с людьми, проверки календарей, установки напоминаний и многого другого.

Кортана в Windows может помочь пользователям лучше управлять своими расписаниями и задачами. Они могут добавлять в свои списки в Майкрософт, искать локальную информацию, получать определения и отслеживать последние новости, погода и финансовую информацию с помощью поиска Bing.

Кортана в Windows 10 версии 2004 и более поздней соответствуют тому же уровню конфиденциальности, безопасности и соответствия требованиям корпоративного уровня, что и в [терминах веб-служб (OST)](https://www.microsoft.com/licensing/product-licensing/products).

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>Как отказаться от Кортаны в Windows 10

Администраторы могут настроить Кортану в Windows 10 для своей организации с помощью политики Експериенце\алловкортана MDM или с помощью групповой политики: Конфигурация компьютера \ административные шаблоны \ административные шаблоны \ Компонентс\сеарч\аллов Кортана.

Начиная с Windows 10, версия 2004, Кортана — это приложение для универсальной платформы Windows (UWP), предустановленное с помощью Windows, которое регулярно обновляется с помощью Microsoft Store. Для получения последних обновлений для Кортаны необходимо [Включить обновления через Microsoft Store](https://docs.microsoft.com/windows/configuration/stop-employees-from-using-microsoft-store).

[Дополнительные сведения о Кортане в Windows 10](https://docs.microsoft.com/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>Помощник по голосовой связи кортаны в Teams

Помощник по голосовой связи кортаны в приложении Teams и на устройствах отображения Microsoft Teams позволяет корпоративным пользователям Microsoft 365 осуществлять оптимизацию общения, совместной работы и задач, связанных с собраниями, с помощью голосового языка. Пользователи могут поговорить с Кортаной, нажав кнопку микрофона, расположенную в правом верхнем углу приложения Teams, или произнося &#8220;Кортаны&#8221; на экране Microsoft Teams. Чтобы быстро подключаться к участникам команды и во время перехода, пользователи могут иметь такие запросы, как &#8220;вызовов Меган&#8221; или &#8220;отправить сообщение на следующий&#8221; собраний. Пользователи также могут присоединяться к собраниям, произнося &#8220;присоединиться к следующему собранию&#8221; и использовать голосовую помощь для совместного использования файлов, проверки календаря и других возможностей. Эти возможности для голосовой связи доставляются с помощью служб Кортаны Enterprise, которые полностью соответствуют конфиденциальности, безопасности и соответствия требованиям Office 365, что отражается в [терминах веб-служб (OST)](https://www.microsoft.com/licensing/product-licensing/products).

**Элемент управления администратора**

Помощник по голосовой связи Кортана будет включен по умолчанию для клиентов. Администраторы могут управлять тем, кто в своем клиенте может использовать голосовую поддержку Кортаны в Teams с помощью политики (Теамскортанаполици). Эту политику можно задать на уровне учетной записи пользователя или на уровне клиента. Администраторы также могут использовать поле Кортанавоицеинвокатионмоде внутри этого элемента управления политики, чтобы определить, отключена ли Кортана, включена ли функция "Отправить нажатием кнопки" или "включить" с вызовом функции "Пробуждение Word" (это относится к устройствам, поддерживающим эту возможность, например, для отображения Microsoft Teams). Обратите внимание, что во время первоначального выпуска для корпоративных пользователей Microsoft 365 в США на английском языке приложение Teams не будет поддерживать активацию Word для пробуждения, но будет поддерживаться в будущем.

**Пользовательский элемент управления**

Отдельные пользователи могут испытать голосовую поддержку Кортаны в мобильном приложении Teams, нажав кнопку MIC. Они могут попытаться использовать голосовые команды Кортаны в Microsoft Teams для отображения устройств, просто произнося &#8220;Кортану. &#8221; они также могут управлять включением Кортаны в Teams для устройств с помощью параметров в приложении Teams или в Microsoft teams: 

1. Откройте мобильное приложение Teams или перейдите на экран внешний (домашний) экран в Microsoft Teams.

2. В мобильном приложении Teams перейдите в раздел **Параметры** . На панели отображения Microsoft Teams выберите аватар пользователя и нажмите кнопку Параметры. Если Кортана включена, скажите, &#8220;Кортану, перейдите к разделу Параметры. &#8221;

3. Выберите **Кортана** .

4. Установите переключатель в положение **включено** или **выключено** , в зависимости от того, хотите ли вы использовать голосовую поддержку кортаны на устройстве.

[Дополнительные сведения о голосовой поддержке Кортаны в Teams](https://docs.microsoft.com/microsoftteams/cortana-in-teams)

### <a name="play-my-emails"></a>Проигрывать мои сообщения

Воспроизведение электронных писем (как подключенных через Outlook Mobile) — это управляемая голосовым интерфейсом, с помощью которой пользователи могут прослушивать новые сообщения в папке "Входящие" и вносить в них изменения на динамиках, на телефонах, наушниках и подключенных звуковых устройствах. Пользователи могут попросить Кортане читать последние сообщения электронной почты вслух и попросить Кортане выполнить такие действия, как помечать, архивировать, удалять и пропускать сообщения. Эта функция особенно полезна для отслеживания сообщений электронной почты во время работы, многозадачности или в дороге. Когда пользователь приобщения к Кортане в воспроизведении электронной почты, запрос голосового голосового сигнала передается непосредственно в службы Кортаны Enterprise Services. Текстовая блок-речь для электронной почты пользователя обрабатывается в облаке Office 365. Во время этого процесса данные Office 365 не обрабатываются на мобильном устройстве пользователя, и данные электронной почты не сохраняются. Транскрипция голосовых команд (например, "пометить как прочтенный", "Далее", "" ") может быть сохранена в соответствии с условиями защиты данных в [терминах Microsoft Online Services](https://www.microsoft.com/licensing/product-licensing/products).

Кортана будет вызываться, когда электронная почта защищена и ненадолго задерживается, прежде чем считывать сообщение, чтобы предоставить пользователям достаточно времени для приостановки воспроизведения или перехода к следующему сообщению. Как и для частного звонка, пользователи должны соблюдать осторожность при инициации воспроизведения в местах, где потенциально могут передаваться конфиденциальные данные. В этих случаях рекомендуется, чтобы сотрудники вашей организации наизноси наушники в соответствующих средах при воспроизведении почты в Outlook Mobile.

### <a name="opt-out-of-play-my-emails"></a>Отказ от проигрывания моих сообщений электронной почты

Пользователи могут отказаться от проигрывания моих сообщений, выполнив указанные ниже действия.

1. Откройте Outlook Mobile.

2. Перейдите в раздел **Параметры** .
  
3. Выберите пункт **проигрывать мои сообщения** .

4. Установите переключатель в положение отключено для учетных записей, которые требуется отключить.

[Дополнительные сведения о воспроизведении электронной почты](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>Информационные сообщения

Кортана отправляет персонализированное информационное сообщение с помощью задач и обязательств, которые вы сделали с помощью удобного способа пометить их как **Завершенные** или запланировать время фокусировки для их выполнения. Кроме того, она включает в себя сводную информацию о собраниях и соответствующих документах в течение вашего дня. Кортана извлекает сведения из сообщений электронной почты пользователя и сохраняет их в почтовом ящике Exchange Online до тех пор, пока они не будут объединены в информационные сообщения. Доступ к персональным данным вне почтового ящика Exchange Online возможен без времени.

### <a name="how-to-opt-out-of-briefing-email"></a>Как отказаться от использования информационного сообщения

Администраторы могут настроить информационный отдел для своей организации с помощью [PowerShell](https://docs.microsoft.com/briefing/be-admin) в Exchange Online. Пользователи могут отказаться от использования информационного сообщения Кортаны, выбрав пункт **отказ от подписки** в нижнем колонтитуле сообщения.

[Дополнительные сведения об информационном письме](https://support.microsoft.com/help/4558259)

Мы будем продолжать добавлять больше возможностей, как показано выше, чтобы повысить производительность организации.

[Дополнительные сведения о решениях Майкрософт для обеспечения соответствия требованиям](https://docs.microsoft.com/microsoft-365/compliance/offering-home)

1. Это новая служба, которая обсуждается в этом документе.

2. Для служб, которые подчиняются терминам Интернет-служб, корпорация Майкрософт является обработчиком данных: Корпорация Майкрософт собирает и использует клиентские данные от клиентов только для предоставления услуг, запрашиваемых нашими клиентами, а также в целях, с которыми наши клиенты. В рамках общего регулирования по защите данных ЕС (GDPR) клиент — это контроллер данных своих данных. Ознакомьтесь с [терминами веб-служб](https://www.microsoft.com/licensing/product-licensing/products) и [Узнайте больше о прозрачности для наших коммерческих облачных пользователей](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/).
