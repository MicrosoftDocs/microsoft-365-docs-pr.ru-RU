# <a name="cortana-in-microsoft-365"></a>Кортана в Microsoft 365

Кортана, ваш личный помощник по повышению производительности, предлагает возможности на основе ИИ, чтобы сэкономить время и сосредоточиться на наиболее важных вопросах. Кортана поможет пользователям повысить свою личные производительность в течение всего дня работы и жизни. При выполнении этого действия с помощью действительных учетных записей на работе или в учебном зачете пользователи могут получать облачные службы помощи с Кортаной в решениях Microsoft 365, которые соответствуют корпоративным обещаниям конфиденциальности, безопасности и соответствия требованиям Office 365 ("Корпоративные службы Кортаны"). 

Кортана как личный помощник по повышению производительности предназначена для предоставления функций, которые безопасно и безопасно обрабатывает данные Office 365, такие как сообщения электронной почты, файлы, чаты и т. д., для экономии времени, повышения эффективности и повышения производительности пользователей.

В будущем мы уделяем кортане особое внимание повышению производительности предприятия.

- В соответствии с другими службами Office 365 корпоративные службы Кортаны соответствуют тем же обещаниям конфиденциальности, безопасности и соответствия требованиям на уровне предприятия, что и в условиях обслуживания Online [Services (OST).](https://www.microsoft.com/licensing/product-licensing/products)

- Новые возможности Microsoft 365, такие как электронная почта для сводки и воспроизведения электронной почты, будут включены с помощью корпоративных служб Кортаны и полностью соответствуют этим обещаниям. Эти функции в настоящее время доступны по всему миру (стандартные мульти клиента). Дополнительные сведения о расположении использования можно найти в дополнительных значениях свойств [учетных записей.](https://docs.microsoft.com/microsoft-365/enterprise/view-user-accounts-with-microsoft-365-powershell?view=o365-worldwide#view-additional-property-values-for-accounts)

- Пользователи могут подключаться к корпоративным службам Кортаны, описанным здесь, с помощью Кортаны в Windows 10 (версии 2004 и более поздних версий), а также клиентских приложений, таких как Outlook для iOS и Android, на которые лагаются отдельные условия лицензирования. 

- Существующие решения для потребителей, включая Кортану в Windows 10 (версии 1909 и более ранних версий) и приложение Кортаны для iOS и Android, регулируются соглашением об обслуживании [Майкрософт](https://www.microsoft.com/licensing/product-licensing/products) и заявлением о конфиденциальности [(см.](https://privacy.microsoft.com/privacystatement) раздел "Существующие службы для потребителей" ниже). Эти условия также регулируют корпоративные службы Кортаны, предоставляемые пользователю при входах с учетными данными потребителя.

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Какие данные обрабатываются корпоративными службами Кортаны? 

Корпоративные службы Кортаны обрабатывает запросы от пользователя, данные Office, необходимые для выполнения запроса пользователя, и другую телеметрию, созданную системами Майкрософт для запуска службы. Данные, собранные корпоративными службами Кортаны, включают текстовое представление запросов пользователя (т. е. транскрибации из распознавания речи). Эти текстовые данные — данные клиента и управляются в соответствии с условиями [веб-служб.](https://www.microsoft.com/licensing/product-licensing/products) Он используется только для разработки и улучшения моделей машинного обучения в соответствии с условиями использования веб-службы.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Какова модель управления данными клиента в корпоративных службах Кортаны?

В соответствии с другими службами Office 365 корпоративные службы Кортаны защищены и подчиняются [условиям веб-служб.](https://www.microsoft.com/licensing/product-licensing/products) К ним относится набор обещаний по защите данных клиента от случайной потери, изменения, несанкционированного раскрытия или доступа или незаконного уничтожения. К данным клиента также накладываются строгие ограничения на доступ. Корпорация Майкрософт использует данные клиента только для предоставления согласованных служб и для целей, совместимых с этими службами. Подробные сведения см. в таблице ниже.

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Корпоративные службы Кортаны в microsoft 365

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Кортана в Windows 10 (версия 2004 и более поздние)

Приложение Кортаны в Windows 10 помогает пользователям быстро получать информацию в Microsoft 365: с помощью введите или произнесенные запросы для связи с людьми, проверки календарей, напоминаний и т. д.

Кортана в Windows помогает пользователям лучше управлять своим расписанием и задачами. Они могут добавлять в свои списки в Microsoft To Do, искать локальные сведения, получать определения и отслеживать последние новости, погоду и финансовую информацию с помощью поиска Bing.

Кортана в Windows 10 версии 2004 и более поздних версий соответствует тем же обещаниям конфиденциальности, безопасности и соответствия требованиям на уровне предприятия для корпоративных служб Кортаны, как отражено в условиях обслуживания Online [Services (OST).](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>Как отказаться от Кортаны в Windows 10

Администраторы могут настроить Кортану в Windows 10 для своей организации с помощью политики MDM Experience\AllowCortana или с помощью групповой политики: Конфигурация компьютера\Административные шаблоны\Компоненты Windows\Поиск\Разрешить Кортану.

Начиная с Windows 10 версии 2004, Кортана — это приложение универсальной платформы Windows (UWP), предустановленное в Windows и регулярно обновляемая через Microsoft Store. Чтобы получать последние обновления Для Кортаны, необходимо включить обновления [через Microsoft Store.](https://docs.microsoft.com/windows/configuration/stop-employees-from-using-microsoft-store)

[Подробнее о Кортане в Windows 10](https://docs.microsoft.com/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>Голосовая поддержка Кортаны в Teams

Голосовая поддержка Кортаны в мобильном приложении Teams и на устройствах отображения Microsoft Teams позволяет пользователям Microsoft 365 корпоративный оптимизировать взаимодействие, совместную работу и задачи, связанные с собраниями, с использованием разговорного естественного языка. Пользователи могут говорить с Кортаной, нажав кнопку микрофона, расположенную в верхней правой части мобильного приложения Teams, или &#8220;Кортану&#8221; на экране Microsoft Teams. Чтобы быстро связаться со своей командой без рук и во время ее проведения, пользователи могут отправлять такие запросы, как &#8220;позвонить&#8221;&#8221; или &#8220;отправить сообщение на следующее собрание&#8221;. Пользователи также могут присоединяться к собраниям, &#8220;присоединиться к следующему собранию&#8221; использовать голосовую поддержку для обмена файлами, проверки календаря и других данных. Эти услуги голосовой помощи ставляются с помощью служб корпоративного уровня Кортаны, которые полностью соответствуют обязательствам Office 365 в области конфиденциальности, безопасности и соответствия требованиям, как отражено в условиях использования веб-служб [(OST).](https://www.microsoft.com/licensing/product-licensing/products)

**Управление администратором**

Голосовая поддержка Кортаны будет включена по умолчанию для клиентов. Администраторы могут управлять тем, кто в клиенте может использовать голосовую поддержку Кортаны в Teams с помощью политики (TeamsCortanaPolicy). Эту политику можно установить на уровне учетной записи пользователя или клиента. Администраторы также могут использовать поле CortanaVoiceInvocationMode в этом средстве управления политикой, чтобы определить, отключена ли Кортана, включена ли только вызов push-кнопки или включена вызовом слова для запуска (применимо к устройствам, которые ее поддерживают, например, дисплею Microsoft Teams). Обратите внимание, что во время первоначального выпуска для пользователей Microsoft 365 корпоративный в США на английском языке мобильное приложение Teams не будет поддерживать активацию слова после активации, но оно будет поддерживаться в будущем.

**Пользовательский контроль**

Отдельные пользователи могут попробовать голосовую поддержку Кортаны в мобильном приложении Teams, нажав кнопку микрофона. Они могут попробовать голосовую поддержку Кортаны на устройствах отображения Microsoft Teams, просто говоря &#8220;Cortana.&#8221; Они также могут управлять тем, включена ли кортана в Teams для своего устройства с помощью параметра в мобильном приложении Teams или на дисплее Microsoft Teams: 

1. Откройте мобильное приложение Teams или перейдите на внешний (домашний) экран отображения Microsoft Teams.

2. В мобильном приложении Teams перейдите в **"Параметры".** На экране Microsoft Teams выберите аватар пользователя, а затем выберите "Параметры". Если Кортана включена, скажем, &#8220;, перейдите на &#8221;

3. Выберите **Кортану.**

4. Переместите этот переключение в "Вкл." или "Выкл." **в** зависимости от того, нужна ли вам голосовая помощь Кортаны на устройстве. 

[Узнайте больше о голосовой поддержке Кортаны в Teams](https://docs.microsoft.com/microsoftteams/cortana-in-teams)

### <a name="play-my-emails"></a>Play My Emails

Play My Emails (as connected to through Outlook mobile) is a voice-driven, hands-free experience for users to listen to new messages in their Focused Inbox and changes to their day via the speakers on their phone, headphones, or connected audio device. Пользователи могут попросить Кортану прочитать свои последние сообщения электронной почты вслух и попросить Кортану принять такие действия, как пометка, архивировать, удалять и пропускать сообщения. Эта функция особенно полезна для того, чтобы перехватить электронную почту во время поездок, многозадачности или в пути. Когда пользователь говорит с Кортаной в "Play My Emails", речевой звуковой запрос передается непосредственно в корпоративные службы Кортаны. Текст для чтения речи электронной почты пользователя обрабатывается в облаке Office 365. В ходе этого процесса данные Office 365 не обрабатываются на мобильном устройстве пользователя, а данные электронной почты не сохраняются. Запись разговорных команд (т. е. "пометить как прочтенные", "далее", "флаг" и т. д.) может храниться в соответствии с условиями защиты данных в условиях Microsoft [Online Services.](https://www.microsoft.com/licensing/product-licensing/products)

Кортана будет вызывать сообщение, защищенное и кратковременно приостанавливовыв его перед чтением, чтобы дать пользователям достаточно времени для приостановки воспроизведения или перехода к следующему сообщению. Как и при закрытом телефонном звонке, пользователям следует соблюдать осторожность при инициации воспроизведения в тех местах, где конфиденциальные сведения могут быть переслушана. В таких случаях при использовании воспроизведения электронной почты в Outlook mobile сотрудникам вашей организации рекомендуется использовать наушники в соответствующих средах.

### <a name="opt-out-of-play-my-emails"></a>Отказ от воспроизведения сообщений электронной почты

Отдельные люди могут отказаться от воспроизведения электронной почты с помощью следующих действий.

1. Откройте Outlook Mobile.

2. Перейдите **в "Параметры".**
  
3. Выберите **"Играть в мои сообщения электронной почты".**

4. Отключите этот переключите для учетных записей, которые нужно отключить.

[Дополнительные информация о воспроизведения сообщений электронной почты](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>Электронная почта с кратким сообщением

Кортана отправляет персонализированное сообщение электронной почты с задачами и обязательствами, которые вы сделали удобным способом пометить их как выполненные или запланировать время для их выполнения.  Здесь также содержится сводка по собраниям и соответствующим документам для вашего дня. Кортана извлекает сведения из сообщений электронной почты пользователя и сохраняет их в почтовом ящике Exchange Online, пока не будет консолидирована в сообщении электронной почты с кратким сообщением. Персональные данные не доступны за пределами почтового ящика Exchange Online.

### <a name="how-to-opt-out-of-briefing-email"></a>Как отказаться от сообщения электронной почты с кратким сообщением

Администраторы могут настраивать краткие сводки для своей организации с помощью [PowerShell](https://docs.microsoft.com/briefing/be-admin) в Exchange Online. Отдельные лица могут отказаться от электронной почты Кортаны для краткого сообщения, выбрав "Отписаться" в нижбике сообщения. 

[Дополнительные информация о сообщении электронной почты с кратким сообщением](https://support.microsoft.com/help/4558259)

Мы по-прежнему будем внедрять дополнительные опытом, такие как выше, чтобы повысить производительность вашей организации.

**Новые возможности для организаций с** корпоративными службами Кортаны: корпоративные службы Кортаны предназначены для удовлетворения потребностей организации в обеспечении безопасности и соответствия требованиям: 

1. Это новая служба, которая обсуждается здесь в этом документе.

2. Для служб, на которые налагаются условия использования веб-служб, корпорация Майкрософт является обработчиком данных: Корпорация Майкрософт собирает и использует данные клиента от клиентов только для предоставления веб-служб, запрашиваемой нашими клиентами, и для целей, о чем они сообщили. В соответствии с Общим регламентом ЕС по защите данных (GDPR) клиент является контролером данных своих данных. См. [условия работы с веб-службами](https://www.microsoft.com/licensing/product-licensing/products) и знакомство с дополнительной прозрачностью конфиденциальности для наших [коммерческих облачных клиентов.](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)

3. Например, "Play My Emails" — это служба Кортаны, к которую пользователи могут подключаться через Outlook для iOS и использовать корпоративные службы Кортаны. 

4. ИТ-администраторы всегда будут иметь элементы управления для необязательных подключенных функций для Кортаны, как и необязательные подключенные функции при использовании приложений Office профессиональныйplus.  

Существующие службы для потребителей: необязательные подключенные службы Кортаны предназначены в основном для потребителей и в настоящее время поставляются в Windows 10 (версии 1909 и более ранних версий) и приложении Кортаны на iOS и Android.

1. Эти возможности позволяют использовать такие функции, как погода, новости и трафик.

2. Администраторы клиентов могут контролировать, могут ли Кортана в Windows 10 (версии 1909 и более ранних версиях) и приложение Кортаны для iOS и Android разрешить Кортане подключаться к данным клиента Office 365.

Отключение доступа Кортаны к данным, которые корпорация Майкрософт в вашей организации

1. В Центре администрирования Microsoft 365 выберите "Параметры организации" и выберите   >   **Кортану.**

2. Чтобы отключить подключенные к Кортане возможности доступа к данным, которые корпорация Майкрософт использует от имени пользователей вашей организации, отключите подключение Кортаны, отключите его, отключив доступ к данным, подключенным к Кортане, в Windows **10 (версии 1909** и более ранних версиях) и отключите доступ к данным, которые были доступны корпорации Майкрософт.

3. Нажмите кнопку **Сохранить изменения**.
