# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Обновление записей MX для перехода на глобальную службу Exchange Online

1. Войдите на [портал администрирования Microsoft 365](https://admin.microsoft.com)и перейдите к **доменам**  >  **параметров**

2. Состояние будет показано справа для каждого домена. Если домены вашей организации указывают на Microsoft Cloud в Германии Exchange Online, вам потребуется обновить запись MX.

3. Щелкните домен, затем щелкните **обнаруженные ошибки DNS, щелкните здесь, чтобы просмотреть.**

4. На этой странице будут указаны инструкции по исправлению записи MX. Если регистратор домена использует [Domain Connect,](../setup/add-domain.md#registrars-with-domain-connect)вы можете нажать кнопку "Исправить мои записи" вверху. В противном случае вы можете следовать по ссылке в мастере, чтобы получить пошаговую инструкцию **для** своего регистратора.
