# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Ошибка отключения учетной записи в Outlook в Интернете

Если при попытке  открыть Outlook в Интернете (прежнее название — Outlook Web App) учетная запись была отключена, администратор мог отключить доступ к `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Outlook в Интернете.

Администратор может исправить эту ошибку, выработав действия, Outlook Web App для [почтового ящика.](https://docs.microsoft.com/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app)
