# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>Регистрация на Office 365 с личным адресом электронной почты

## <a name="when-you-skip-adding-a-custom-domain"></a>При пропуске добавления настраиваемой доменной области


При регистрации на Office 365 с личным адресом электронной почты и пропустить добавление пользовательского домена на данный момент, вы ограничиваете доступ к определенным функциям. [](../get-help-with-domains/what-is-a-domain.md)

Если вы решите добавить настраиваемый домен сейчас, вы получите доступ ко всем премиум-функциям ниже:

|**Функция**|**Office 365 личной электронной почты**|**Office 365 с настраиваемой областью**|
|:-----|:-----|:-----|
|**OneDrive <sup>1, 2</sup>**| [Личные OneDrive](https://onedrive.live.com/about/plans/)| [OneDrive для бизнеса](https://onedrive.live.com/about/en-us/business/) |
|**Office: Word, Excel, PowerPoint, OneNote, Outlook, Access (только для ПК),**| Да | Да
|**Бизнес-приложения <sup>3:</sup> Microsoft Bookings и MileIQ**| Нет | Да
|**Доступ к Центр администрирования Microsoft 365**| Ограниченный доступ (установка биллинга, поддержки и домена) | Да
|**Добавление пользователей**| Нет | Да
|**Office 365: Exchange Online, Skype для бизнеса, Microsoft Teams, SharePoint Online**| Нет | Да
|**Средства безопасности и соответствия требованиям**| Нет | Да

> <sup>1</sup> Вам потребуется перенести файлы персональных OneDrive [на OneDrive для бизнеса.](move-email-and-data-to-office-365-business-premium.md)<br/>
