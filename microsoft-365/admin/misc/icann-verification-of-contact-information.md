# <a name="icann-verification-of-contact-information"></a>Проверка контактных данных в ЕАО

При покупке домена или внесении изменений в контактные данные для домена регистратору необходимо проверить ваши контактные данные в соответствии с нормативными актами ЕАО.

Для проверки на адрес регистратора отправляется сообщение электронной почты.

 **Чтобы проверить контактные данные, выберите ссылку в сообщении для проверки.** Если владельцы домена не проверяют адрес электронной почты, домен будет приостановлен. Затем перейдите на страницу ["Домены"](https://admin.microsoft.com/adminportal/home?ref=Domains) в Центре администрирования и проверьте состояние домена.
