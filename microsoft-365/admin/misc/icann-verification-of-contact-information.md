# <a name="icann-verification-of-contact-information"></a>Проверка контактных данных ICANN

При покупке домена или внесении изменений в контактные данные для домена регистратор должен проверить ваши контактные данные в соответствии с правилами ICANN.

Для проверки по адресу регистратора отправляется сообщение электронной почты.

 **Чтобы проверить контактные данные, выберите ссылку в электронной почте проверки.** Если владельцы домена не проверяют адрес электронной почты, домен будет приостановлен. Затем перейдите на страницу [Домены](https://admin.microsoft.com/adminportal/home?ref=Domains) в центре администрирования и проверьте состояние домена.
