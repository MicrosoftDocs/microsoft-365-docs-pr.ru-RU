# <a name="quick-help-deleted-users-checklist"></a>Быстрая справка: контрольный список для удаления пользователей

Итак, вы решили удалить пользователя из своей организации. Например, это можно сделать после увольнения сотрудника. До и после удаления пользователя необходимо выполнить несколько действий, которые указаны в этом кратком контрольном списке.
  
## <a name="remove-the-microsoft-365-license-from-the-user"></a>Удаление лицензии Microsoft 365 у пользователя

Вы можете удалить лицензию у пользователя, ища пользователя на домашней странице или на странице **Активные** пользователи, а затем выбрав их. На **вкладке Лицензии и Приложения** снимите чековую шкатулку для лицензии, которая необходимо удалить у пользователя. [Прочитать статью](../manage/remove-licenses-from-users.md).
  
## <a name="block-sign-in"></a>Блокирование входа

Вы можете заблокировать пользователя, чтобы он не мог войти в организацию. Выполните поиск пользователя на домашней странице или выберите его в списке **Активные пользователи**. В верхней части области, под именем пользователя,  выберите блок этого значка пользователя, а затем выберите блок этого пользователя при входе **в** поле. [Прочитать статью](../add-users/assign-admin-roles.md).
  
## <a name="get-their-data"></a>Получение данных

Вы можете отправить электронную почту пользователя другому сотруднику или пользователю в вашей организации, выполнив эти [действия](../add-users/remove-former-employee.md). Кроме того, можно заархивировать и сохранить все данные бывшего сотрудника, выполнив эти [действия](../add-users/get-access-to-and-back-up-a-former-user-s-data.md).
  
## <a name="delete-user"></a>Удаление пользователя

Выполните поиск пользователя на домашней странице или а странице **Удаленные пользователи**. Выберите пользователя и выберите **Удалить пользователя**. [Прочитать статью](../add-users/delete-a-user.md).
  
## <a name="what-to-do-with-the-unassigned-license"></a>Что делать с неназначенными лицензиями

Если у вас есть новый сотрудник или сотрудник, который присоединяется к организации, вы можете назначить им эту лицензию, чтобы они могли использовать Microsoft 365. Выполните поиск пользователя на домашней странице или выберите его в списке **Активные пользователи**. На **вкладке Лицензии и Приложения** выберите поле для лицензии, которая будет добавлена новому пользователю. [Прочитать статью](../manage/assign-licenses-to-users.md).
  
## <a name="remove-license-so-you-dont-have-to-pay-for-it"></a>Как удалить лицензию, чтобы не платить за нее

Вы также можете удалить лицензию из подписки, чтобы не платить за нее. На домашней странице центра администрирования введите **подписку** и выберите **просмотр подписки**. Выберите **Добавить или удалить лицензии.** В поле **Total licenses** введите общее количество лицензий, необходимых для этой подписки, а затем выберите **Отправка изменений.** Например, если у вас 5 лицензий, из которых вам нужно удалить 1, следует ввести **4**. [Прочитать статью.](../../commerce/licenses/buy-licenses.md)
