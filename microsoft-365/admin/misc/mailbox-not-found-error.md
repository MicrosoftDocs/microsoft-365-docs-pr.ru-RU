# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a><span data-ttu-id="c007a-103">Получение почтового ящика, не найденного в Outlook в Интернете?</span><span class="sxs-lookup"><span data-stu-id="c007a-103">Getting a mailbox not found error in Outlook on the web?</span></span>

<span data-ttu-id="c007a-104">Если вы используете Outlook в Интернете и  не нашли почтовый ящик из-за ошибки, у учетной записи, которая использовалась для подключения к Outlook в Интернете, нет лицензии Exchange Online, поэтому с этой учетной записью не связан ни один почтовый ящик.</span><span class="sxs-lookup"><span data-stu-id="c007a-104">If you're using Outlook on the web and you get a  **Mailbox couldn't be found for**  error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account.</span></span> <span data-ttu-id="c007a-105">Администратор может назначить лицензию вашей учетной записи, вы предприняв указанные ниже действия.</span><span class="sxs-lookup"><span data-stu-id="c007a-105">Your admin can assign a license to your account by following these steps:</span></span>

1. <span data-ttu-id="c007a-106">Откройте Центр  [администрирования Microsoft 365](https://portal.office.com/adminportal/home#/homepage) и  перейдите к активным пользователям в разделе "Пользователи" и выберите пользователя, который видит ошибку.</span><span class="sxs-lookup"><span data-stu-id="c007a-106">Open the  [Microsoft 365 admin center](https://portal.office.com/adminportal/home#/homepage)  and go to  **Active users**  under the  **Users**  section, and select the user who is seeing the error.</span></span>
2. <span data-ttu-id="c007a-107">На открываемой странице пользователя  перейдите в раздел "Лицензии и приложения", выберите соответствующее значение расположения и назначьте лицензию с Exchange Online (разместите лицензию, чтобы увидеть ее сведения). </span><span class="sxs-lookup"><span data-stu-id="c007a-107">In the user page that opens, go to the  **Licenses and Apps**  section, select the appropriate  **Location**  value, and assign a license that contains Exchange Online (expand the license to see its details).</span></span> <span data-ttu-id="c007a-108">После завершения нажмите кнопку **"Сохранить изменения".**</span><span class="sxs-lookup"><span data-stu-id="c007a-108">When you're finished, click  **Save changes**.</span></span>
