# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a>Получение почтового ящика, не найденного в Outlook в Интернете?

Если вы используете Outlook в Интернете и  не нашли почтовый ящик из-за ошибки, у учетной записи, которая использовалась для подключения к Outlook в Интернете, нет лицензии Exchange Online, поэтому с этой учетной записью не связан ни один почтовый ящик. Администратор может назначить лицензию вашей учетной записи, вы предприняв указанные ниже действия.

1. Откройте Центр  [администрирования Microsoft 365](https://portal.office.com/adminportal/home#/homepage) и  перейдите к активным пользователям в разделе "Пользователи" и выберите пользователя, который видит ошибку.
2. На открываемой странице пользователя  перейдите в раздел "Лицензии и приложения", выберите соответствующее значение расположения и назначьте лицензию с Exchange Online (разместите лицензию, чтобы увидеть ее сведения).  После завершения нажмите кнопку **"Сохранить изменения".**
