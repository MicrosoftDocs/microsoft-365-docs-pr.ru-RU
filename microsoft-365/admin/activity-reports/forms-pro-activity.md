# <a name="microsoft-365-reports-in-the-admin-center---dynamics-365-customer-voice-activity"></a>Отчеты Microsoft 365 в Центре администрирования — действия с голосовой почтой клиентов Dynamics 365

На информационной панели  "Отчеты Microsoft 365" представлен обзор действий в продуктах организации. Вы можете просмотреть отчеты по отдельным продуктам, чтобы получить более подробные сведения о действиях с каждым приложением. Ознакомьтесь со статьей [Обзор отчетов](activity-reports.md).
  
Например, вы можете понять действия каждого пользователя, у кого есть лицензия на использование голосовой связи клиента Microsoft Dynamics 365, изумив их взаимодействие с голосовой службой клиента Dynamics 365. Кроме того, это помогает понять уровень совместной работы, изумив количество созданных опросов Pro и опросов Pro, на которые ответили пользователи. 
  
> [!NOTE]
> Чтобы увидеть отчеты, вы должны быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, службе Teams, коммуникациях Teams или администраторе Skype для бизнеса. 

## <a name="how-to-get-to-the-forms-pro-activity-report"></a>Как добраться до отчета об активности Forms Pro

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы.

    
2. В **выпадаемом окплексе** "Выбор отчета" выберите действие "Голосовая почта клиента **Dynamics 365".** \> 

## <a name="interpret-the-dynamics-365-customer-voice-activity-report"></a>Интерпретация отчета об активности голосовой почты клиентов Dynamics 365

Вы можете просмотреть действия пользователя в Dynamics 365 Customer Voice, просмотрев диаграммы "Действия **и** **пользователи".** 

![Отчет о действиях с формами](../../media/formsproactivity.png)

|Элемент|Описание|
|:-----|:-----|
|1.  <br/> |В отчете о действиях с голосовой связи клиентов **Dynamics 365** можно просмотреть тенденции за последние 7, 30, 90 и 180 дней. Однако если выбрать определенный день в отчете, в таблице (7) будут отображиться данные за 28 дней от текущей даты (а не даты, когда был создан отчет).   <br/> |
|2.  <br/> |Данные в каждом отчете обычно являются последними за последние 48 часов.  <br/> |
|3.  <br/> |Представление **"Пользователи"** помогает понять тенденцию числа активных пользователей голосовой связи клиента Dynamics 365. Пользователь считается активным, если он выполнял действие вокруг опроса Pro (создание, редактирование, просмотр и т. д.) в течение определенного периода времени.  <br/> |
|4.  <br/> |Представление **"Действие"** помогает понять тенденцию числа активных пользователей. Пользователь считается активным, если он выполнял действия с файлом (сохранение, синхронизацию, изменение или предоставление общего доступа) или посещал страницу в течение заданного периода времени.<br/> ПРИМЕЧАНИЕ. Действие может происходить несколько раз для одного опроса, но будет учитываться только как один активный опрос. Например, можно создать опрос Pro и продолжить редактирование одного опроса несколько раз за указанный период времени. Он будет учитываться только как один опрос. <br>|
|5.<br/>|На **диаграмме "Пользователи"** ось Y — это количество уникальных пользователей. Ось X — это дата, в которую активны уникальные пользователи. Эти легенды:<br/><br/>**Дизайнеры** означает, что пользователь создал или редактировал опрос голосовой службы клиента Dynamics 365.<br><br>На **диаграмме действий** ось Y — это количество ответов dynamics 365 customer Voice на опрос. Ось X — это дата проведения опроса или ответа. Эти легенды:<br/><br/>**Созданные опросы** — это количество уникальных опросов Dynamics 365 Customer Voice, созданных пользователями<br>**Ответы —** это количество анонимных или неанимных ответов, отправленных пользователями, которые получили опрос. |
|6.<br/>|Вы можете отфильтровать ряды, которые вы видите на диаграмме, выбрав элемент в легенде. Например, на диаграмме "Пользователи" выберите конструкторов, ответчиков или пользователей, чтобы увидеть только сведения, связанные с каждым из них. Изменение этого выделения не изменяет сведения в таблице таблицы под ней.|
|7.<br/>|В таблице показана разбивка действий на уровне пользователя. Эти легенды:<br/><br/>**Имя пользователя —** это адрес электронной почты пользователя, выполнившего действие в Microsoft Forms.<br/>**Дата последнего действия (UTC)** — это последняя дата действия формы, выполненного пользователем для выбранного диапазона дат. Для просмотра действий, которые произошли в определенный день, выберите эту дату непосредственно на диаграмме.<br/>Это позволит отфильтровать таблицу для отображения данных о действиях с файлами только для пользователей, выполнивших действие в этот день.<br/><br/>**Количество созданных опросов** — это количество опросов, созданных пользователем.<br/> **Количество ответов на опрос** — это количество ответов от ответчиков, которым был распространен опрос.|
|8.<br/>|Выберите **значок "Управление столбцами",** чтобы добавить или удалить столбцы из отчета.|
|9.<br/>|Вы также можете экспортировать данные отчета в CSV-файл Excel, выбрав ссылку **"Экспорт".** Это позволяет экспортировать данные для всех пользователей и позволяет легко агрегировать, сортировать и фильтровать данные для дальнейшего анализа. Если у вас менее 100 пользователей, можно сортировать и фильтровать таблицу в самом отчете. Если у вас более 100 пользователей, для фильтрации и сортировки необходимо экспортировать данные.|
