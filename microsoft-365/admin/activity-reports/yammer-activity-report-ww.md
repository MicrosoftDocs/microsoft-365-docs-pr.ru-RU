# <a name="microsoft-365-reports-in-the-admin-center---yammer-activity-report"></a>Отчеты Microsoft 365 в центре администрирования — отчет о активности Yammer

В качестве администратора Microsoft 365 панель **отчетов** отображает данные об использовании продуктов в организации. Ознакомьтесь [с отчетами об активности в центре администрирования.](activity-reports.md) В отчете **Yammer Activity** вы можете понять уровень взаимодействия вашей организации с Yammer, если посмотреть количество уникальных пользователей, использующих Yammer для публикации, а также прочитать сообщение и объем активности, генерируемой в организации. 
  
> [!NOTE]
> Чтобы увидеть отчеты, необходимо быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, Teams Service, Teams Communications или Skype для бизнеса.  
 
## <a name="how-do-i-get-to-the-yammer-activity-report"></a>Как добраться до отчета об активности Yammer?

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы. 
2. На домашней странице панели мониторинга нажмите кнопку **Просмотр** дополнительных на карточке Yammer.

  
## <a name="interpret-the-yammer-activity-report"></a>Анализ отчета об активности в Yammer

Вы можете просмотреть действия в отчете Yammer, выбрав вкладку **Activity.**<br/>![Отчеты Microsoft 365 — отчет об активности Microsoft Yammer.](../../media/9b251183-c2b3-430c-ab2d-58bf11e7e3ae.png)

Выберите **выберите столбцы для** добавления или удаления столбцов из отчета.  <br/> ![Отчет о активности Yammer — выберите столбцы](../../media/7ef6351d-f7e9-4504-913d-2c2df9062bf6.png)

Вы также можете экспортировать данные отчета в файл Excel .csv, выбрав ссылку **Экспорт.** При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные.
