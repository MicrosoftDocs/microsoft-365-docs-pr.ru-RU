# <a name="microsoft-365-reports-in-the-admin-center---yammer-activity-report"></a>Microsoft 365 Отчеты в центре администрирования — Yammer отчет об активности

В Microsoft 365 администратора панель **отчетов** отображает данные об использовании продуктов в организации. Ознакомьтесь [с отчетами об активности в центре администрирования.](activity-reports.md) В **отчете Yammer** действий можно понять уровень взаимодействия организации с Yammer, если посмотреть на количество уникальных пользователей, использующих Yammer для публикации, чтения или чтения сообщения, а также объема активности, генерируемой в организации. 
  
> [!NOTE]
> Чтобы увидеть отчеты, вы должны быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, Teams Service, Teams Communications или Skype для бизнеса администратором.  
 
## <a name="how-do-i-get-to-the-yammer-activity-report"></a>Как добраться до отчета Yammer действий?

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы. 
2. На домашней странице панели мониторинга нажмите кнопку **Просмотр** дополнительных на Yammer карте.

  
## <a name="interpret-the-yammer-activity-report"></a>Анализ отчета об активности в Yammer

Вы можете просмотреть действия в отчете Yammer, выбрав вкладку **Activity.**<br/>![Microsoft 365 отчеты — отчет о Yammer майкрософт.](../../media/9b251183-c2b3-430c-ab2d-58bf11e7e3ae.png)

Выберите **выберите столбцы для** добавления или удаления столбцов из отчета.  <br/> ![Yammer действий — выберите столбцы](../../media/7ef6351d-f7e9-4504-913d-2c2df9062bf6.png)

Вы также можете экспортировать данные отчета в Excel .csv, выбрав ссылку **Экспорт.** При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
  
|Item|Описание|

В этой сетке показаны пользователи, которые вошли Yammer с помощью Microsoft 365 учетной записи или вошел в сеть с помощью единого входа. <br/> |
