# <a name="microsoft-365-reports-in-the-admin-center---forms-activity"></a>Отчеты Microsoft 365 в Центре администрирования — действия с формами

На информационной панели  "Отчеты Microsoft 365" представлен обзор действий в продуктах организации. Вы можете просмотреть отчеты по отдельным продуктам, чтобы получить более подробные сведения о действиях с каждым приложением. Ознакомьтесь со статьей [Обзор отчетов](activity-reports.md).
  
Например, вы можете понять действия каждого пользователя, у кого есть лицензия на использование Microsoft Forms, изумив их взаимодействие с формами. Это также помогает понять уровень совместной работы, изумив количество созданных форм и форм, на которые пользователь ответил.
  
> [!NOTE]
> Чтобы увидеть отчеты, вы должны быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, службе Teams, коммуникациях Teams или администраторе Skype для бизнеса. 

## <a name="how-to-get-to-the-forms-activity-report"></a>Как добраться до отчета о действиях Forms

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы.

    
2. В **выпадаемом окплексе "Выбор** отчета" выберите **действие** \> **"Формы".**

## <a name="interpret-the-forms-activity-report"></a>Интерпретация отчета о действиях с формами

Вы можете просмотреть действия пользователя с формами, просмотрев **диаграммы "Действия"** и **"Пользователи".** 

![Отчет о действиях с формами](../../media/adminformsactivity.png)

|Элемент|Описание|
|:-----|:-----|
|1.  <br/> |В **отчете о** действиях с формами можно просмотреть тенденции за последние 7, 30, 90 и 180 дней. Однако если выбрать определенный день в отчете, в таблице (7) будут отображиться данные за 28 дней от текущей даты (а не даты, когда был создан отчет).  <br/> |
|2.  <br/> |Данные в каждом отчете обычно охватывают до последних 24-48 часов.  <br/> |
|3.  <br/> |Представление **"Пользователи"** помогает понять тенденцию числа активных пользователей форм. Пользователь считается активным, если он или она выполнили действие вокруг формы (создание, редактирование, просмотр и т. д.) или ответили на форму в течение определенного периода времени.  <br/> |
|4.  <br/> |Представление **"Действие"** помогает понять тенденцию числа активных пользователей. Пользователь считается активным, если он выполнял действия с файлом (сохранение, синхронизацию, изменение или предоставление общего доступа) или посещал страницу в течение заданного периода времени.<br/> ПРИМЕЧАНИЕ. Действие может происходить несколько раз для одной формы, но будет учитываться только как одна активная форма. Например, можно создать форму и продолжить редактировать ее несколько раз за указанный период времени, но она будет учитываться только как одна форма. Однако если пользователь отправил несколько ответов для одной формы, каждый ответ будет по-прежнему отдельным ответом и, следовательно, будет учитываться несколько раз. <br/> |
|5.<br/>|На **диаграмме "Пользователи"** ось Y — это количество уникальных пользователей. Ось X — это дата, в которую активны уникальные пользователи. Эти легенды:<br/><br/>**Конструкторы** означает, что пользователь создал или редактировал форму.<br/>**Ответчики** означает, что пользователь отправил ответы в форму.<br/> **Общее количество пользователей** означает всех пользователей в компании, которые были дизайнером или ответчиком.<br/><br/> На **диаграмме действий** ось Y — это количество уникальных форм или ответов. Ось X — это дата действия формы или ответа. Эти легенды:<br/><br/>**Созданные** формы — это количество уникальных форм, созданных пользователями.<br/> **Количество подписанных** ответов, полученных пользователями в организации.<br/> **Анонимные ответы** — это количество анонимных ответов, полученных пользователями в организации.<br/><br/>|
|6.<br/>|Вы можете отфильтровать ряды, которые вы видите на диаграмме, выбрав элемент в легенде. Например, на диаграмме "Пользователи" выберите конструкторов, ответчиков или пользователей, чтобы увидеть только сведения, связанные с каждым из них. Изменение этого выделения не изменяет сведения в таблице таблицы под ней.|
|7.<br/>|В таблице показана разбивка действий на уровне пользователя. Эти легенды:<br/><br/>**Имя пользователя —** это адрес электронной почты пользователя, выполнившего действие в Microsoft Forms.<br/>**Дата последнего действия (UTC)** — это последняя дата действия формы, выполненного пользователем для выбранного диапазона дат. Для просмотра действий, которые произошли в определенный день, выберите эту дату непосредственно на диаграмме.<br/><br/>Это позволит отфильтровать таблицу для отображения данных о действиях с файлами только для пользователей, выполнивших действие в этот день.<br/><br/>**Количество созданных форм** — это количество форм, созданных пользователем.<br/> **Количество форм ответа —** это количество форм, на которые пользователь отправил ответы.|
|8.<br/>|Выберите **значок "Управление столбцами",** чтобы добавить или удалить столбцы из отчета.|
|9.<br/>|Вы также можете экспортировать данные отчета в CSV-файл Excel, выбрав ссылку **"Экспорт".** Это позволяет экспортировать данные для всех пользователей и позволяет легко агрегировать, сортировать и фильтровать данные для дальнейшего анализа. Если количество пользователей меньше 100, можно сортировать и фильтровать их в самой таблице отчета. Если у вас более 100 пользователей, для фильтрации и сортировки необходимо экспортировать данные.|
