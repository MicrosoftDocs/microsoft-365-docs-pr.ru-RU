# <a name="microsoft-365-reports-in-the-admin-center---sharepoint-activity"></a>Отчеты Microsoft 365 в центре администрирования — действия SharePoint

В качестве администратора Microsoft 365 панель отчетов показывает обзор действий различных продуктов в организации.  Вы можете детализировать их, чтобы получить более подробную информацию о действиях для каждого продукта. Ознакомьтесь с отчетами об активности в центре администрирования [Microsoft 365.](activity-reports.md)
  
Например, вы можете понять, чем занимается каждый пользователь, имеющий лицензию на SharePoint, изучив его работу с файлами. Кроме того, можно оценить активность совместной работы, просмотрев количество файлов, которыми поделились сотрудники.
  
> [!NOTE]
> Чтобы увидеть отчеты, необходимо быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, Teams Service, Teams Communications или Skype для бизнеса. 
 
## <a name="how-do-i-get-to-the-to-the-sharepoint-activity-report"></a>Как перейти к отчету о действиях в SharePoint?

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы. 
2. На домашней странице панели мониторинга нажмите кнопку **Просмотр дополнительных** на карте SharePoint.
  
## <a name="interpret-the-sharepoint-activity-report"></a>Интерпретация отчета о действиях SharePoint

Действия в отчете SharePoint можно просмотреть, выбрав вкладку **Activity.**<br/>![Отчеты Microsoft 365 — отчет об активности Microsoft SharePoint.](../../media/5a0a96f-0e4f-4fb9-8baa-3262275b3d1f.png)

Выберите **выберите столбцы для** добавления или удаления столбцов из отчета.  <br/> ![Отчет о действиях SharePoint — выберите столбцы](../../media/3c396cd1-9701-4712-8eaa-eb7bba702aa8.png)

Вы также можете экспортировать данные отчета в файл Excel .csv, выбрав ссылку **Экспорт.** При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. 
  
|Item|Описание|
|:-----|:-----|
|**Метрика**|**Определение**|
|Имя пользователя  <br/> |Адрес электронной почты пользователя, который выполнял действие на сайте SharePoint.  <br/> |
|Последняя дата действия (UTC)  <br/> |Последняя дата выполнения действия файла или посещения страницы для выбранного диапазона дат. Для просмотра действий, которые произошли в определенный день, выберите эту дату непосредственно на диаграмме.  <br/> |
|Просмотр или редактирование файлов  <br/> |Количество файлов, которые пользователь загрузил, скачал, изменил или просмотрел.   <br/> |
|Синхронизированные файлы  <br/> |Количество файлов, синхронизированных с локального устройства пользователя на сайт SharePoint. <br/> |
|Файлы, общие для внутренней  <br/> | Количество файлов, которые были общими для пользователей в организации или пользователей в группах (в том числе внешних пользователей).  <br/> |
|Файлы, общие для внешнего  <br/> |Количество файлов, которые были совместно с пользователями за пределами организации. <br/>|
|Страницы, посещаемые  <br/> |Посещение пользователем уникальных страниц. <br/>|
|Deleted  <br/> | Это означает, что лицензия пользователя была удалена.  <br/>  **ПРИМЕЧАНИЕ:** Действие для удаленного пользователя будет отображаться в отчете до тех пор, пока он или она была лицензирована в течение определенного периода времени. Столбец "Удален" говорит о том, что пользователь больше не является активным, но его данные использовались в отчете.  <br/> |
|Дата удаления  <br/> |Дата, в которую была удалена лицензия пользователя. <br/>|
|Назначен продукт  <br/> |Продукты Microsoft 365, лицензированные пользователю.|
