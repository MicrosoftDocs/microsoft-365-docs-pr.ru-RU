# <a name="microsoft-365-reports-in-the-admin-center---microsoft-teams-user-activity"></a>Отчеты Microsoft 365 в Центре администрирования — действия пользователей Microsoft Teams

На информационной панели  "Отчеты Microsoft 365" представлен обзор действий в продуктах организации. Вы можете просмотреть отчеты по отдельным продуктам, чтобы получить более подробные сведения о действиях с каждым приложением. Ознакомьтесь с [общими сведениями о панели отчетов](activity-reports.md). С помощью отчета о действиях пользователей Microsoft Teams вы можете получить представление об активности этих пользователей в вашей организации.
  
> [!NOTE]
> Чтобы увидеть отчеты, вы должны быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, службе Teams, коммуникациях Teams или администраторе Skype для бизнеса.  
 
## <a name="how-to-get-to-the-microsoft-teams-user-activity-report"></a>Как найти отчет о действиях пользователей Microsoft Teams?

1. В центре администрирования перейдите в раздел **отчеты о** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использование</a> страницы.
2. На домашней странице панели мониторинга  нажмите кнопку "Подробнее" на карточке действия Microsoft Teams.

## <a name="interpret-the-microsoft-teams-user-activity-report"></a>Интерпретация отчета о действиях пользователей Microsoft Teams

Вы можете просмотреть действия пользователей в отчете Teams, выбрав вкладку **"Действия пользователя".** <br/>![Отчеты Microsoft 365 : активность пользователей Microsoft Teams.](../../media/1011877f-3cf0-4417-9447-91d0b2312aab.png)

Выберите **"Выбрать столбцы",** чтобы добавить или удалить столбцы из отчета.  <br/> ![Teams user activity report - choose columns](../../media/a1513028-cf09-4186-93a6-8a203cd22475.png)

Вы также можете экспортировать данные отчета в CSV-файл Excel, выбрав ссылку **"Экспорт".** При этом данные всех пользователей будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 пользователей, вы можете сортировать и фильтровать значения в самой таблице отчета. Если пользователей больше 2000, для фильтрации и сортировки потребуется экспортировать данные. Экспортируемая формат для **времени звука,** времени **видео** и времени совместной работы с экраном следует формату длительности ISO8601. 

Чтобы обеспечить качество данных, мы ежедневно проверяем данные за последние три дня и заполняем все обнаруженные пробелы. Во время процесса можно заметить различия в исторических данных.

|Элемент|Описание|
|:-----|:-----|
|**Метрика**|**Определение**|

|Сообщения канала   <br/> |Количество уникальных сообщений, которые пользователь опубликовал в чате группы за указанный период времени.  <br/> |

|Общее количество собраний   <br/> |Количество собраний по сети, в которые пользователь принимал участие в течение указанного периода времени.  <br/> |

|Запланированные собрания  <br/> |Количество запланированных собраний, организованных пользователем в указанный период времени.  <br/> |
|Лицензируется ли |Выбирается, если у пользователя есть лицензия на использование Teams.|
|Другие действия|Пользователь активен, но выполнял другие действия, кроме предложенных в отчете типов действий (отправка или ответ на сообщения каналов и сообщений чата, планирование или участие в звонках и собраниях по 1:1). Примеры действий: когда пользователь изменяет состояние Teams или сообщение о состоянии Teams или открывает сообщение канала, но не отвечает. |
