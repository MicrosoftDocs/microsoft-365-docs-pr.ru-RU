# <a name="microsoft-365-reports-in-the-admin-center---onedrive-for-business-usage"></a>Отчеты Microsoft 365 в Центре администрирования : использование OneDrive для бизнеса

На информационной панели  "Отчеты Microsoft 365" представлен обзор действий в продуктах организации. Вы можете просмотреть отчеты по отдельным продуктам, чтобы получить более подробные сведения о действиях с каждым приложением. Ознакомьтесь со статьей [Обзор отчетов](activity-reports.md).
  
Например, с помощью карточки OneDrive на панели мониторинга вы можете получить общее представление о том, насколько активно используется OneDrive для бизнеса с точки зрения количества хранимых файлов и объема хранилища, используемого в вашей организации. Затем вы можете детализировать данные, чтобы выявить тенденции использования активных учетных записей OneDrive, узнать, сколько файлов используют сотрудники и какой объем в хранилище занимают их данные. Вы также сможете узнать сведения о OneDrive каждого пользователя.
  
> [!NOTE]
> Чтобы увидеть отчеты, вы должны быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, службе Teams, коммуникациях Teams или администраторе Skype для бизнеса.  
 
## <a name="how-do-i-get-to-the-onedrive-usage-report"></a>Как получить отчет об использовании OneDrive?

1. В Центре администрирования перейдите в **"Отчеты об** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">использовании".</a>

    
2. В **выпадаемом окплексе "Выбор** отчета" выберите **"Использование OneDrive".** \>  
  
## <a name="interpret-the-onedrive-usage-report"></a>Анализ отчета об использовании OneDrive

Вы можете просмотреть представления **Учетные записи**, **Файлы** и **Хранилище**, чтобы проанализировать использование OneDrive для бизнеса. 
  
![OneDrive Usage Report](../../media/49c5b93b-d081-436e-8992-236343a6d46b.png)
  
|Элемент|Описание|
|:-----|:-----|
|1.  <br/> |Отчет **Использование OneDrive** показывает тенденции за последние 7, 30, 90 или 180 дней. Однако если выбрать определенный день в отчете, в таблице (7) будут отображиться данные за 28 дней от текущей даты (а не даты, когда был создан отчет).  <br/> |
|2.  <br/> |Данные в каждом отчете обычно охватывают до последних 24-48 часов. <br/>|
|3.  <br/> |В представлении **Учетные записи** отображаются тенденции количества активных и общего количества OneDrive учетных записей. К активным учетным записям относятся все, в которых пользователи просматривают, изменяют, загружают, скачивают или синхронизируют файлы.  <br/> |
|4.  <br/> |В **представлении** "Файлы" показано общее число активных файлов. Файл считается активным, если он был сохранен, синхронизирован, изменен или общим в течение определенного периода времени.  <br/> ПРИМЕЧАНИЕ. Действия с файлом могут происходить несколько раз для одного файла, но будут учитываться только как один активный файл. Например, вы можете сохранить и синхронизировать файл несколько раз за определенный период времени, но он будет учитываться как один активный файл и один синхронизированный файл.           |
|5.  <br/> |В представлении **Хранилище** показана тенденция изменения объема хранилища, которое используется в OneDrive. Если вы хотите проверить ограничения хранилища, проверьте, есть ли у пользователя ограничение по умолчанию или [определенное ограничение.](https://docs.microsoft.com/onedrive/set-default-storage-space#check-if-a-user-has-the-default-storage-limit-or-a-specific-limit)  <br/> ПРИМЕЧАНИЕ. Размер включает все версии и метаданные, связанные с файлами.           |
|6.  <br/> | На диаграмме **Учетные записи** ось Y отображает число учетных записей OneDrive.  <br/>  На диаграмме **Файлы** ось Y представляет число файлов, которые хранятся в учетных записях OneDrive.  <br/>  На оси Y диаграммы **Хранилище** показан объем хранилища, которое используется для OneDrive.  <br/>  На оси X всех диаграмм представлен диапазон дат, выбранный для отчета.  <br/> |
|7.  <br/> |Вы можете отфильтровать ряды, которые вы видите на диаграмме, выбрав элемент в легенде. Например, на диаграмме **"Файлы"** выберите **"Всего файлов" или** **"Активные файлы".** На **диаграмме "Учетные записи"** выберите **"Всего** учетных записей" или **"Активные учетные записи".** Или на **диаграмме хранилища** выберите **используемую память.** При выборе другого значения данные в таблице не меняются.  <br/> |
|8.  <br/> | В таблице показана разбивка данных для каждого пользователя OneDrive. Для отображения в таблице необходимо присвоить пользователю лицензию, включающую OneDrive, и включить SharePoint Online. Пользователю также необходимо войти в клиент синхронизации OneDrive или перейти к OneDrive с помощью веб-браузера.  <br/>  Если в OneDrive выполнялись действия с файлами, будет указана последняя дата такого действия. Строки в таблице сортируются по значению **Дата последнего действия**, поэтому OneDrive с недавними действиями будут отображаться в верхней части списка.  <br/>  Вы можете добавить столбцы в таблицу или удалить их из нее.  <br/> ![Параметры столбцов](../../media/onedriveusage-columns.png)  <br/> **URL-адрес** — это веб-адрес oneDrive пользователя.  <br/> **Удалено**  это состояние удаленияOneDrive. Учетные записи помечаются как удаленные не ранее чем через 7 дней.  <br/> **Владелец сайта**  имя пользователя, который является основным администратором OneDrive.  <br/> **Имя владельца —** это адрес электронной почты владельца OneDrive.  <br/> **Дата последнего действия (UTC)**  дата последнего действия с файлами в OneDrive. Если в OneDrive не выполнялись действия с файлами, значение будет пустым.  <br/> **Файлы**  число файлов в OneDrive.  <br/> **Активные файлы**  число активных файлов за определенный период времени.<br/> ПРИМЕЧАНИЕ. Если файлы были удалены в течение указанного периода времени для отчета, число активных файлов, показанных в отчете, может быть больше текущего количества файлов в OneDrive. Удаленные пользователи будут по-прежнему отображаться в отчетах в течение 180 дней.<br/>**Используемое хранилище (МБ)**  объем хранилища (в МБ), которое используется OneDrive. <br/>  Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с **разделом "Как скрыть** сведения об уровне пользователя?", в разделе "Отчеты об активности" в Центре администрирования [Microsoft 365.](activity-reports.md)  <br/> |
|9.  <br/> |Выберите **значок "Управление столбцами"** "Управление столбцами", чтобы добавить или удалить ![ ](../../media/13d2e536-de88-4db3-80c7-7a3a57298eb4.png) столбцы из отчета.  <br/> |
|10.  <br/> |Вы также можете экспортировать данные отчета в CSV-файл Excel, выбрав ссылку **"Экспорт".** При этом данные всех OneDrive будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 учетных записей OneDrive, вы можете сортировать и фильтровать значения в самой таблице отчета. Если учетных записей OneDrive больше 2000, для фильтрации и сортировки потребуется экспортировать данные.  <br/> ПРИМЕЧАНИЕ. При экспорте данных в файл Excel дата, когда был создан отчет о контенте, отражается в файле в **столбце "Данные"**  <br/> |
