# <a name="microsoft-365-reports-in-the-admin-center---onedrive-for-business-usage"></a>Отчеты Microsoft 365 в центре администрирования — Использование OneDrive для бизнеса

Панель мониторинга отчетов  Microsoft 365 показывает обзор действий в продуктах организации. Вы можете просмотреть отчеты по отдельным продуктам, чтобы получить более подробные сведения о действиях с каждым приложением. Ознакомьтесь со статьей [Обзор отчетов](activity-reports.md).
  
Например, с помощью карточки OneDrive на панели мониторинга вы можете получить общее представление о том, насколько активно используется OneDrive для бизнеса с точки зрения количества хранимых файлов и объема хранилища, используемого в вашей организации. Затем вы можете детализировать данные, чтобы выявить тенденции использования активных учетных записей OneDrive, узнать, сколько файлов используют сотрудники и какой объем в хранилище занимают их данные. Вы также сможете узнать сведения о OneDrive каждого пользователя.
  
> [!NOTE]
> Чтобы увидеть отчеты, необходимо быть глобальным администратором, глобальным читателем или читателем отчетов в Microsoft 365 или Exchange, SharePoint, Teams Service, Teams Communications или Skype для бизнеса.  
 
## <a name="how-do-i-get-to-the-onedrive-usage-report"></a>Как получить отчет об использовании OneDrive?

1. В центре администрирования перейдите к **использованию** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">отчетов</a>.

    
2. Из **ссылки Выберите отсев** отчетов, выберите **Использование OneDrive.** \>  
  
## <a name="interpret-the-onedrive-usage-report"></a>Анализ отчета об использовании OneDrive

Вы можете просмотреть представления **Учетные записи**, **Файлы** и **Хранилище**, чтобы проанализировать использование OneDrive для бизнеса. 
  
![OneDrive Usage Report](../../media/49c5b93b-d081-436e-8992-236343a6d46b.png)
  
|Item|Описание|
|:-----|:-----|
|1.  <br/> |Отчет **Использование OneDrive** показывает тенденции за последние 7, 30, 90 или 180 дней. Однако если выбрать определенный день в отчете, в таблице (7) будут показываться данные на срок до 28 дней с текущей даты (а не даты, когда был создан отчет).  <br/> |
|2.  <br/> |Данные в каждом отчете обычно охватывают до последних 24-48 часов. <br/>|
|3.  <br/> |В представлении **Учетные записи** отображаются тенденции количества активных и общего количества OneDrive учетных записей. К активным учетным записям относятся все, в которых пользователи просматривают, изменяют, загружают, скачивают или синхронизируют файлы.  <br/> |
|4.  <br/> |В **представлении Files** показано количество общего и активного файлов. Файл считается активным, если он был сохранен, синхронизирован, изменен или общий в течение определенного периода времени.  <br/> ПРИМЕЧАНИЕ. Действие файла может происходить несколько раз для одного файла, но будет считаться только одним активным файлом. Например, вы можете сохранить и синхронизировать файл несколько раз за определенный период времени, но он будет учитываться как один активный файл и один синхронизированный файл.           |
|5.  <br/> |В представлении **Хранилище** показана тенденция изменения объема хранилища, которое используется в OneDrive. Если вы хотите проверить ограничения хранения, см. в этой записи проверьте, есть ли у пользователя ограничение хранилища по умолчанию или [определенное ограничение.](/onedrive/set-default-storage-space#check-if-a-user-has-the-default-storage-limit-or-a-specific-limit)  <br/> ПРИМЕЧАНИЕ. Размер включает все версии и метаданные, связанные с файлами.           |
|6.  <br/> | На диаграмме **Учетные записи** ось Y отображает число учетных записей OneDrive.  <br/>  На диаграмме **Файлы** ось Y представляет число файлов, которые хранятся в учетных записях OneDrive.  <br/>  На оси Y диаграммы **Хранилище** показан объем хранилища, которое используется для OneDrive.  <br/>  На оси X всех диаграмм представлен диапазон дат, выбранный для отчета.  <br/> |
|7.  <br/> |Вы можете фильтровать серии, которые вы видите на диаграмме, выбрав элемент в легенде. Например, на диаграмме **Files** выберите **общие** файлы или **активные файлы.** На **диаграмме Учетные записи** выберите **Общие учетные записи** или **Активные учетные записи.** Или на **диаграмме хранения** выберите **используемую хранилище.** При выборе другого значения данные в таблице не меняются.  <br/> |
|8.  <br/> | В таблице показана разбивка данных для каждого пользователя OneDrive. Для отображения в таблице необходимо присвоить пользователю лицензию, включающую OneDrive, и включить SharePoint Online. Пользователю также необходимо войти в клиент синхронизации OneDrive или перейти к OneDrive с помощью веб-браузера.  <br/>  Если в OneDrive выполнялись действия с файлами, будет указана последняя дата такого действия. Строки в таблице сортируются по значению **Дата последнего действия**, поэтому OneDrive с недавними действиями будут отображаться в верхней части списка.  <br/>  Вы можете добавить столбцы в таблицу или удалить их из нее.  <br/> ![Параметры столбцов](../../media/onedriveusage-columns.png)  <br/> **URL-адрес** — это веб-адрес OneDrive пользователя.  <br/> **Удалено**  это состояние удаленияOneDrive. Учетные записи помечаются как удаленные не ранее чем через 7 дней.  <br/> **Владелец сайта**  имя пользователя, который является основным администратором OneDrive.  <br/> **Основное имя владельца** — адрес электронной почты владельца OneDrive.  <br/> **Дата последнего действия (UTC)**  дата последнего действия с файлами в OneDrive. Если в OneDrive не выполнялись действия с файлами, значение будет пустым.  <br/> **Файлы**  число файлов в OneDrive.  <br/> **Активные файлы**  число активных файлов за определенный период времени.<br/> ПРИМЕЧАНИЕ. Если файлы были удалены в течение указанного периода времени для отчета, количество активных файлов, показанных в отчете, может быть больше текущего числа файлов в OneDrive. Удаленные пользователи будут продолжать появляться в отчетах в течение 180 дней.<br/>**Используемое хранилище (МБ)**  объем хранилища (в МБ), которое используется OneDrive. <br/>  Если политики организации не позволяют просматривать отчеты, в которых есть личные сведения пользователей, можно изменить параметр конфиденциальности для всех отчетов. Ознакомьтесь с **разделом Как** скрыть сведения об уровне пользователя? в разделе Отчеты об активности в центре администрирования [Microsoft 365.](activity-reports.md)  <br/> |
|9.  <br/> |Выберите **значок Управление столбцами** Управление столбцами для добавления или удаления ![ ](../../media/13d2e536-de88-4db3-80c7-7a3a57298eb4.png) столбцов из отчета.  <br/> |
|10.  <br/> |Вы также можете экспортировать данные отчета в файл Excel .csv, выбрав ссылку **Экспорт.** При этом данные всех OneDrive будут экспортированы в формат, позволяющий сортировать и фильтровать их для дальнейшего анализа. Если у вас менее 2000 учетных записей OneDrive, вы можете сортировать и фильтровать значения в самой таблице отчета. Если учетных записей OneDrive больше 2000, для фильтрации и сортировки потребуется экспортировать данные.  <br/> ПРИМЕЧАНИЕ. Когда данные экспортируются в файл Excel, дата сгенерированного отчета о контенте отражается в файле **в столбце Data.**  <br/> |
