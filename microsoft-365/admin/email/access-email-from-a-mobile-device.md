# <a name="options-for-accessing-email-from-your-mobile-device"></a>Варианты доступа к электронной почте с мобильного устройства

В организации настраиваются параметры, которые контролируют, какие мобильные приложения можно использовать для доступа к своей учетной записи для электронной почты, календаря и контактов.

Мобильные приложения Outlook для Android и Outlook для iOS предназначены для вашей работы или учебной электронной почты. Эти приложения предоставляют такие функции, как "Отслужив входящие", которые обеспечивают поддержку важных сообщений переднего и центрального помощник по планированию, что помогает находить время собрания, которое работает с коллегами, и просто находит доступные конференц-залы. [](https://support.microsoft.com/office/f445ad7f-02f4-4294-a82e-71d8964e3978) [](https://support.microsoft.com/office/scheduling-made-easy-in-outlook-mobile-11c5bee5-d78a-4a2b-80c2-2b386ddb4470)
  
## <a name="end-user-experience"></a>Интерфейс конечных пользователей

### <a name="outlook-only"></a>Только Outlook

Вашей организации необходимо использовать мобильные приложения Outlook для Android или Outlook для iOS для доступа к электронной почте, календарю и контактам компании. Ваши данные начнут синхронизироваться после скачивания и установки Outlook для Android или Outlook для iOS.

![Пример сообщения электронной почты для синхронизации электронной почты с помощью Outlook](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)

Дополнительные функции Outlook можно узнать в приложении "Оптимизация мобильного приложения Outlook" для [телефона с iOS](https://support.microsoft.com/office/de075b19-b73c-4d8a-841b-459982c7e890) или Android. Если у вас есть проблемы, перейдите в Центр справки Outlook для iOS и [Android.](https://support.microsoft.com/office/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6)

### <a name="any-email-app"></a>Любое почтовое приложение

Ваша организация рекомендует использовать мобильное приложение Outlook для Android или iOS для доступа к своей учетной записи для работы или учебного заведения для электронной почты, календаря и контактов. Если вы не будете ничего делать в течение следующих нескольких часов, ваша электронная почта автоматически начнет синхронизироваться. Если вы решите использовать мобильное приложение Outlook для Android или iOS, ваши данные будут синхронизироваться только с мобильным приложением Outlook для Android или Outlook для iOS. Если вы решите продолжить использовать сторонное приложение, ваши данные немедленно начнут синхронизироваться.
