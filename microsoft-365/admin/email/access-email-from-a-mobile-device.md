# <a name="options-for-accessing-email-from-your-mobile-device"></a>Параметры доступа к электронной почте с мобильного устройства

Организация настраивает параметры, которые контролируют, какие мобильные приложения можно использовать для доступа к вашей учетной записи для электронной почты, календаря и контактов.

Приложение Outlook для Android и Outlook для мобильных приложений iOS предназначено для вашей работы или школьной электронной почты. Эти приложения предоставляют такие функции, как [Сфокусированный](https://support.microsoft.com/office/f445ad7f-02f4-4294-a82e-71d8964e3978)почтовый ящик, который сохраняет важные электронные [почты](https://support.microsoft.com/office/scheduling-made-easy-in-outlook-mobile-11c5bee5-d78a-4a2b-80c2-2b386ddb4470)передней и центральной, и помощник по планированию , что помогает вам найти время собраний, которые работают с коллегами и просто находит доступные конференц-залы.
  
## <a name="end-user-experience"></a>Интерфейс конечных пользователей

### <a name="outlook-only"></a>Outlook Только

Для доступа к электронной почте, календарю и контактам в организации Outlook для Android или Outlook для мобильных приложений iOS. Ваши данные начнут синхронизироваться после загрузки и установки Outlook для Android или Outlook для iOS.

![Пример электронной почты для Outlook для синхронизации электронной почты](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)

Дополнительные [функции Outlook](https://support.microsoft.com/office/de075b19-b73c-4d8a-841b-459982c7e890) мобильного приложения для iOS или Outlook Android-телефона. Если у вас Outlook проблемы, перейдите в центр помощи для iOS и [Android.](https://support.microsoft.com/office/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6)

### <a name="any-email-app"></a>Любое приложение электронной почты

Организация рекомендует использовать приложение Outlook android или iOS для доступа к вашей учетной записи для работы или учебного заведения для электронной почты, календаря и контактов. Если вы не будете принимать никаких действий в течение следующих нескольких часов, электронная почта автоматически начнет синхронизироваться. Если вы решите использовать Outlook для android или iOS мобильного приложения, ваши данные синхронизируются только с Outlook для Android или Outlook для мобильного приложения iOS. Если вы решите продолжить использование сторонних приложений, данные начнут синхронизироваться мгновенно.
