# <a name="configure-shared-mailbox-settings"></a>Настройка параметров общего почтового ящика

После создания [общего](create-a-shared-mailbox.md)почтового ящика необходимо настроить некоторые параметры для пользователей почтовых ящиков, такие как пересылание электронной почты и автоматические ответы. Позже может потребоваться изменить другие параметры, такие как имя почтового ящика, члены или разрешения участников. 

## <a name="change-the-name-or-email-alias-of-a-shared-mailbox-or-change-the-primary-email-address"></a>Изменение имени или псевдонима общего почтового ящика или изменение основного адреса электронной почты

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **Изменить** рядом с **псевдонимами Name, Email, Email.**

3. Введите новое имя или добавьте другой псевдоним. Если вы хотите изменить основной адрес электронной почты, ваш почтовый ящик должен иметь несколько псевдонимов электронной почты.

4. Нажмите **Сохранить**.

## <a name="forward-emails-that-are-sent-to-a-shared-mailbox"></a>Пересылка сообщений, отправленных на общий почтовый ящик

Вам не нужно назначать лицензию общему почтовому ящику для пересылаемой им электронной почты. Вы можете переададовать сообщения на любой допустимый адрес электронной почты или список рассылки.

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **изменить переададку** \> **электронной почты.**
    
3. Установите перенастройку **в On** и введите один адрес электронной почты для отправки сообщений. Это может быть любой допустимый адрес электронной почты. Чтобы переадпределить несколько [](/office365/admin/setup/create-distribution-lists) адресов, необходимо создать группу рассылки для адресов, а затем ввести имя группы в этом поле.
    
4. Нажмите **Сохранить**.

## <a name="send-automatic-replies-from-a-shared-mailbox"></a>Отправка автоматических ответов с общего почтового ящика

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **Автоматические ответы** \> **Изменить**.
    
3. Установите toggle **on** и выберите, отправлять ли ответ людям внутри организации или за ее пределами.

4. Введите текст ответа, который должен отправляться пользователям в организации. Добавить можно только текст без изображений.

5. Если вы хотите *также* отправить ответ людям за пределами организации, выберите контрольный ящик, который вы хотите получить ответ, и введите текст. Сделать так, чтобы ответ отправлялся только пользователям за пределами организации и не отправлялся пользователям внутри нее, невозможно.

6. Нажмите **Сохранить**.

## <a name="allow-everyone-to-see-the-sent-email-the-replies"></a>Предоставление всем пользователям разрешения на просмотр отправленных сообщений (ответов)

По умолчанию сообщения, отправленные с общего почтового ящика, не сохраняются в папке "Отправленные" этого почтового ящика. Вместо этого они сохраняются в папке "Отправленные" отправителя.

Если вы хотите разрешить всем видеть отправленную электронную почту в центре администрирования, изменить параметры общих почтовых ящиков и выбрать **пункты Sent** \> **Edit**.


## <a name="choose-the-apps-that-a-shared-mailbox-can-use-to-access-microsoft-email"></a>Выберите приложения, которые общий почтовый ящик может использовать для доступа к электронной почте Майкрософт

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **изменить приложения** \> **электронной почты.**

3. Установите для всех  приложений, которые пользователи могут использовать для доступа к общему почтовому ящику. Установите отключку **для** любых приложений, которые вы не хотите использовать. 

4. Нажмите **Сохранить**.


## <a name="put-a-shared-mailbox-on-litigation-hold"></a>Поместить общий почтовый ящик на удержание судебного разбирательства

Дополнительные дополнительные информацию о удержании судебного разбирательства см. в дополнительных подробной информации [о создании судебного удержания.](../../compliance/create-a-litigation-hold.md)

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **Редактирование удержания** \> **для судебного разбирательства.**

3. Установите toggle **on**. 

4. Необязательно введите продолжительность, примечание о удержании и URL-адрес с дополнительными сведениями.  

5. Нажмите **Сохранить**.


## <a name="add-or-remove-members"></a>Добавление или удаление участников

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **Members** \> **Edit**.

3. Выполните одно из следующих действий.
   - Чтобы добавить участников, выберите **Добавить участников,** найти или выбрать члена, чтобы добавить, а затем выберите **Сохранить**.
   - Чтобы удалить участников, при необходимости используйте поле Поиск для поиска участника, выберите **X** рядом с именем участника, а затем выберите **Сохранить**. 

4. Выберите **Сохранить** снова.

## <a name="add-or-remove-permissions-of-members"></a>Добавление или удаление разрешений участников

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **разрешения На** \> **настройку членов.**

3. Выберите **Изменить** рядом с разрешением, который необходимо изменить для участника. 

4. Выполните одно из следующих действий.
   - Чтобы предоставить это разрешение дополнительному члену, выберите **Добавить** разрешения, найти или выбрать члена, чтобы добавить, а затем **выберите Сохранить**.
   - Чтобы удалить разрешение у участника, в случае необходимости используйте поле Поиск для поиска участника, выберите **X** рядом с именем участника, а затем выберите **Сохранить**. 

4. Выберите **Сохранить** снова.

## <a name="show-or-hide-a-shared-mailbox-in-the-global-address-list"></a>Показать или скрыть общий почтовый ящик в глобальном списке адресов

Если вы решите не показывать общий почтовый ящик в глобальном списке адресов, почтовый ящик не будет отображаться в списке адресов организации, но он по-прежнему будет получать отправленную им электронную почту. 

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

2. Выберите общий почтовый ящик, который необходимо изменить, а затем выберите **Показать в глобальном списке** \> **адресов Изменить**.

3. Установите переключеть **на или** **отключить**. 

4. Нажмите **Сохранить**.

> [!NOTE]
> Сокрытие общего почтового ящика из списка адресов сделает невозможным добавление скрытого почтового ящика в профиль Outlook до тех пор, пока общий почтовый ящик не будет снова показан в списке адресов.
