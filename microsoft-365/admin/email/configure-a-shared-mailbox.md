# <a name="configure-shared-mailbox-settings"></a>Настройка параметров общего почтового ящика

После создания [общего почтового](create-a-shared-mailbox.md)ящика необходимо настроить некоторые параметры для пользователей почтового ящика, такие как переадстройка электронной почты и автоматические ответы. Позднее может потребоваться изменить другие параметры, такие как имя почтового ящика, члены или разрешения участников. 

## <a name="change-the-name-or-email-alias-of-a-shared-mailbox-or-change-the-primary-email-address"></a>Изменение имени или псевдонима общего почтового ящика или изменение основного адреса электронной почты

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

::: moniker-end 

::: moniker range="o365-germany"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

::: moniker range="o365-21vianet"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

2. Выберите общий почтовый ящик, который нужно  изменить, а затем выберите "Изменить рядом с **именем", "Электронная почта", "Псевдонимы электронной почты".**

3. Введите новое имя или добавьте другой псевдоним. Если вы хотите изменить основной адрес электронной почты, ваш почтовый ящик должен иметь несколько псевдонимов электронной почты.

4. Нажмите **Сохранить**.

## <a name="forward-emails-that-are-sent-to-a-shared-mailbox"></a>Пересылка сообщений, отправленных на общий почтовый ящик

Вам не нужно назначать лицензию общему почтовому ящику для пересылаемой в него электронной почты. Вы можете переададовать сообщения на любой допустимый адрес электронной почты или в список рассылки.

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

::: moniker-end 

::: moniker range="o365-germany"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

::: moniker range="o365-21vianet"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

2. Выберите общий почтовый ящик, который нужно изменить, а затем выберите "Изменить **пересылку** \> **электронной почты".**
    
3. Установите для этого toggle **вкл.** и введите один адрес электронной почты, на который будут переададратовать сообщения. Это может быть любой допустимый адрес электронной почты. Чтобы переададратовать на [](https://docs.microsoft.com/office365/admin/setup/create-distribution-lists) несколько адресов, необходимо создать группу рассылки для этих адресов, а затем ввести имя группы в этом поле.
    
4. Нажмите **Сохранить**.

## <a name="send-automatic-replies-from-a-shared-mailbox"></a>Отправка автоматических ответов с общего почтового ящика

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

::: moniker-end 

::: moniker range="o365-germany"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

::: moniker range="o365-21vianet"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

2. Выберите общий почтовый ящик, который нужно изменить, а затем выберите **"Изменить автоматические** \> **ответы".**
    
3. Установите для этого toggle **"Вкл."** и выберите, следует ли отправлять ответ людям внутри организации или за ее пределами.

4. Введите текст ответа, который должен отправляться пользователям в организации. Добавить можно только текст без изображений.

5. Если вы хотите *также* отправить ответ людям за пределами вашей организации, уберите его и введите текст. Сделать так, чтобы ответ отправлялся только пользователям за пределами организации и не отправлялся пользователям внутри нее, невозможно.

6. Нажмите **Сохранить**.

## <a name="allow-everyone-to-see-the-sent-email-the-replies"></a>Предоставление всем пользователям разрешения на просмотр отправленных сообщений (ответов)

По умолчанию сообщения, отправленные с общего почтового ящика, не сохраняются в папке "Отправленные" этого почтового ящика. Вместо этого они сохраняются в папке "Отправленные" отправителя.

Если вы хотите, чтобы все могли видеть отправленную электронную почту, в Центре  администрирования отредактируете параметры общего почтового ящика и выберите пункт "Изменить \> **отправленные".**

## <a name="add-or-remove-members"></a>Добавление или удаление участников

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

::: moniker-end 

::: moniker range="o365-germany"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

::: moniker range="o365-21vianet"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

2. Выберите общий почтовый ящик, который нужно изменить, а затем выберите **"Члены"** \> **"Изменить".**

3. Выполните одно из указанных ниже действий.
   - Чтобы добавить участников, выберите **"Добавить участников",**"Найти" или "Добавить" и выберите **"Сохранить".**
   - Чтобы удалить участников, при необходимости используйте поле поиска, чтобы найти его, выберите **X** рядом с именем участника, а затем выберите **"Сохранить".** 

4. Снова **выберите "Сохранить".**

## <a name="show-or-hide-a-shared-mailbox-in-the-global-address-list"></a>Показать или скрыть общий почтовый ящик в глобальном списке адресов

Если общий почтовый ящик не будет отображаться в глобальном списке адресов, он не будет отображаться в списке адресов организации, но по-прежнему будет получать отправленную на него электронную почту. 

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Группы** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2066847" target="_blank">Общие почтовые ящики</a>.

::: moniker-end 

::: moniker range="o365-germany"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

::: moniker range="o365-21vianet"

1. В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">Центре администрирования</a> откройте страницу **Группы** > **Общие почтовые ящики**. 

::: moniker-end

2. Выберите общий почтовый ящик, который нужно изменить, а затем выберите "Показать в глобальном списке **адресов"** \> **"Изменить".**

3. Установите для этого переключия **"Вкл."** или **"Выкл."** 

4. Нажмите **Сохранить**.

> [!NOTE]
> Скрытие общего почтового ящика из списка адресов сделает невозможным добавление скрытого почтового ящика в профиль Outlook новыми участниками общего почтового ящика до тех пор, пока общий почтовый ящик не будет снова показан в списке адресов.
