# <a name="remove-a-license-from-a-shared-mailbox"></a>Удаление лицензии из общего почтового ящика

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

::: moniker-end

Для общих почтовых ящиков обычно не требуется лицензия. Следуйте этим инструкциям, чтобы удалить лицензию из общего почтового ящика, чтобы назначить ее пользователю или вернуть лицензию, чтобы не платить за лицензию, которая вам не нужна.

> [!NOTE]
> Лицензия требуется в следующих сценариях:
> 1. Общий почтовый ящик использует более 50 ГБ памяти.
> 2. Общий почтовый ящик использует архив на месте.
> 3. Общий почтовый ящик помещается на судебное удержание.
> 4. Общему почтовому ящику назначена лицензия Microsoft Defender.

  
## <a name="remove-the-license"></a>Удаление лицензии

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

   > [!NOTE]
   > Необходимо удалить лицензию со страницы "Активные пользователи". Лицензию нельзя удалить со страницы общего почтового ящика, так как лицензии являются пользовательскими настройками. 
  
2. Выберите общий почтовый ящик.

3. На одной **вкладке "Лицензии и** приложения" разоберите "Лицензии" и снимите этот список. 

4. Нажмите кнопку **Сохранить изменения**.

5. Когда вы вернетесь на страницу **"Активные** пользователи", состояние общего почтового ящика будет **нелицензным.**

6. Вы по-прежнему платите за лицензию. Чтобы прекратить оплату, [удалите лицензию из подписки.](../../commerce/licenses/remove-licenses-from-subscription.md)

::: moniker-end

::: moniker range="o365-germany"

 1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">Активные пользователи</a>.

    > [!NOTE]
    > Необходимо удалить лицензию со страницы "Активные пользователи". Лицензию нельзя удалить со страницы общего почтового ящика, так как лицензии являются пользовательскими настройками.

2. Выберите общий почтовый ящик и выберите "Изменить **рядом** с **лицензиями на продукты".**

3. На одной **странице "Лицензии на** продукты" установите для отключаемой лицензии выключение. 

4. Нажмите кнопку **Сохранить**.

5. Когда вы вернетесь на страницу **"Активные** пользователи", состояние общего почтового ящика будет **нелицензным.**

6. Вы по-прежнему платите за лицензию. Чтобы прекратить оплату, [удалите лицензию из подписки.](../../commerce/licenses/remove-licenses-from-subscription.md)

::: moniker-end
