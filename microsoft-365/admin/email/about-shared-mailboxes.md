<span data-ttu-id="4b1e1-105">Общие почтовые ящики используются, если нескольким людям необходим доступ к одному и тому же почтовому ящику, например к электронному адресу службы поддержки, почтовому ящику со сведениями о компании, столу регистрации или к другой функции, которую совместно используют несколько человек.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-105">Shared mailboxes are used when multiple people need access to the same mailbox, such as a company information or support email address, reception desk, or other function that might be shared by multiple people.</span></span>

<span data-ttu-id="4b1e1-106">Пользователи с разрешениями для почтового ящика группы "Отправить как" или "Отправить от имени" могут отправлять сообщения с электронного адреса почтового ящика, если это разрешено администратором.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-106">Users with permissions to the group mailbox can send as or send on behalf of the mailbox email address if the administrator has given that user permissions to do that.</span></span> <span data-ttu-id="4b1e1-107">Эта возможность особенно полезна для почтовых ящиков службы справки и поддержки, так как пользователи могут отправлять письма из таких групп, как "Служба поддержки Contoso" или "Создание стола регистрации".</span><span class="sxs-lookup"><span data-stu-id="4b1e1-107">This is particularly useful for help and support mailboxes because users can send emails from "Contoso Support" or "Building A Reception Desk."</span></span>

<span data-ttu-id="4b1e1-108">Прежде чем [создавать общий почтовый ящик](create-a-shared-mailbox.md), необходимо знать следующее:</span><span class="sxs-lookup"><span data-stu-id="4b1e1-108">Before you [create a shared mailbox](create-a-shared-mailbox.md), here are some things you should know:</span></span>

- <span data-ttu-id="4b1e1-115">**Разрешения пользователя:** Необходимо предоставить пользователям разрешения (членство) для использования общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-115">**User permissions:** You need to give users permissions (membership) to use the shared mailbox.</span></span> <span data-ttu-id="4b1e1-116">Использовать общий почтовый ящик могут только пользователи в пределах организации.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-116">Only people inside your organization can use a shared mailbox.</span></span>

- <span data-ttu-id="4b1e1-117">**Внешние пользователи:** Вы не можете предоставлять пользователям за пределами организации (например, с помощью учетной записи Gmail) доступ к общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-117">**External users:** You can't give people outside your business (such as people with a Gmail account) access to your shared mailbox.</span></span> <span data-ttu-id="4b1e1-118">Если вы хотите сделать это, подумайте о том, чтобы вместо этого создать группу для Outlook.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-118">If you want to do this, consider creating a group for Outlook instead.</span></span> <span data-ttu-id="4b1e1-119">Чтобы узнать больше, ознакомьтесь со статьей [Создание группы Microsoft 365 в центре администрирования](../create-groups/create-groups.md).</span><span class="sxs-lookup"><span data-stu-id="4b1e1-119">To learn more, see [Create a Microsoft 365 group in the admin center](../create-groups/create-groups.md).</span></span>

-  <span data-ttu-id="4b1e1-120">**Использование с Outlook:** Кроме использования Outlook в Интернете из браузера для доступа к общим почтовым ящикам, вы также можете использовать приложение Outlook для iOS или приложение Outlook для Android.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-120">**Use with Outlook:** In addition to using Outlook on the web from your browser to access shared mailboxes, you can also use the Outlook for iOS app or the Outlook for Android app.</span></span> <span data-ttu-id="4b1e1-121">Чтобы узнать больше, ознакомьтесь <a href="https://support.microsoft.com/office/f866242c-81b2-472e-8776-6c49c5473c9f" target="_blank">со статьей Добавление общего почтового ящика в Outlook Mobile</a>.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-121">To learn more, see <a href="https://support.microsoft.com/office/f866242c-81b2-472e-8776-6c49c5473c9f" target="_blank">Add a shared mailbox to Outlook mobile</a>.</span></span> <span data-ttu-id="4b1e1-122">Кроме того, можно создать группу для общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-122">Another option is to create a group for your shared mailbox.</span></span> <span data-ttu-id="4b1e1-123">Чтобы узнать больше, ознакомьтесь со статьей [Сравнение групп](../create-groups/compare-groups.md).</span><span class="sxs-lookup"><span data-stu-id="4b1e1-123">To learn more, see [Compare Groups](../create-groups/compare-groups.md).</span></span>

- <span data-ttu-id="4b1e1-133">**Вход:** Общий почтовый ящик не предназначен для прямого входа с помощью связанной с ним учетной записи пользователя.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-133">**Signing in:** A shared mailbox is not intended for direct sign-in by its associated user account.</span></span> <span data-ttu-id="4b1e1-134">Вы всегда можете заблокировать вход для учетной записи общего почтового ящика и сохранить его блокировку.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-134">You should always block sign-in for the shared mailbox account and keep it blocked.</span></span>

- <span data-ttu-id="4b1e1-135">**Слишком много пользователей:** При одновременном доступе к общему почтовому ящику слишком большого числа назначенных пользователей могут возникать сбои при подключении к этому почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-135">**Too many users:** When there are too many designated users concurrently accessing a shared mailbox, they may intermittently fail to connect to this mailbox.</span></span> <span data-ttu-id="4b1e1-136">В этом случае можно сократить количество пользователей или использовать другую рабочую нагрузку, например группу Microsoft 365 или общую папку.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-136">In this case, you can consider reducing the number of the users or using a different workload, such a Microsoft 365 group or Public folder.</span></span>

- <span data-ttu-id="4b1e1-137">**Удаление сообщения:** К сожалению, вы не можете запретить пользователям удалять сообщения в общем почтовом ящике.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-137">**Message deletion:** Unfortunately, you can't prevent people from deleting messages in a shared mailbox.</span></span> <span data-ttu-id="4b1e1-138">Единственный способ сделать это — создать группу Microsoft 365 вместо общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-138">The only way around this is to create a Microsoft 365 group instead of a shared mailbox.</span></span> <span data-ttu-id="4b1e1-139">Группа в Outlook аналогична общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="4b1e1-139">A group in Outlook is like a shared mailbox.</span></span> <span data-ttu-id="4b1e1-140">Сравнение этих двух способов приведено в разделе [Сравнение групп](../create-groups/compare-groups.md).</span><span class="sxs-lookup"><span data-stu-id="4b1e1-140">For a comparison of the two, see [Compare groups](../create-groups/compare-groups.md).</span></span> <span data-ttu-id="4b1e1-141">Дополнительные сведения о группах приведены в [статье дополнительные сведения о группах](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2).</span><span class="sxs-lookup"><span data-stu-id="4b1e1-141">To learn more about groups, see [Learn more about groups](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2).</span></span>
