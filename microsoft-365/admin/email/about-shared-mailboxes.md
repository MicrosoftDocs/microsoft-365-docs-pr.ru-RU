<span data-ttu-id="13bfb-106">Пользователи с разрешениями для почтового ящика группы "Отправить как" или "Отправить от имени" могут отправлять сообщения с электронного адреса почтового ящика, если это разрешено администратором.</span><span class="sxs-lookup"><span data-stu-id="13bfb-106">Users with permissions to the group mailbox can send as or send on behalf of the mailbox email address if the administrator has given that user permissions to do that.</span></span> <span data-ttu-id="13bfb-107">Эта возможность особенно полезна для почтовых ящиков службы справки и поддержки, так как пользователи могут отправлять письма из таких групп, как "Служба поддержки Contoso" или "Создание стола регистрации".</span><span class="sxs-lookup"><span data-stu-id="13bfb-107">This is particularly useful for help and support mailboxes because users can send emails from "Contoso Support" or "Building A Reception Desk."</span></span>

- <span data-ttu-id="13bfb-115">**Разрешения пользователя:** Необходимо предоставить пользователям разрешения (членство) для использования общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="13bfb-115">**User permissions:** You need to give users permissions (membership) to use the shared mailbox.</span></span> <span data-ttu-id="13bfb-116">Использовать общий почтовый ящик могут только пользователи в пределах организации.</span><span class="sxs-lookup"><span data-stu-id="13bfb-116">Only people inside your organization can use a shared mailbox.</span></span>

- <span data-ttu-id="13bfb-117">**Внешние пользователи:** Вы не можете предоставлять пользователям за пределами организации (например, с помощью учетной записи Gmail) доступ к общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="13bfb-117">**External users:** You can't give people outside your business (such as people with a Gmail account) access to your shared mailbox.</span></span> <span data-ttu-id="13bfb-118">Если вы хотите сделать это, подумайте о том, чтобы вместо этого создать группу для Outlook.</span><span class="sxs-lookup"><span data-stu-id="13bfb-118">If you want to do this, consider creating a group for Outlook instead.</span></span> <span data-ttu-id="13bfb-119">Чтобы узнать больше, ознакомьтесь со статьей [Создание группы Microsoft 365 в центре администрирования](../create-groups/create-groups.md).</span><span class="sxs-lookup"><span data-stu-id="13bfb-119">To learn more, see [Create a Microsoft 365 group in the admin center](../create-groups/create-groups.md).</span></span>

- <span data-ttu-id="13bfb-124">**Шифрование:** Вы не можете шифровать сообщения, отправленные с общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="13bfb-124">**Encryption:** You can't encrypt email sent from a shared mailbox.</span></span> <span data-ttu-id="13bfb-125">Это связано с тем, что общий почтовый ящик не имеет собственного контекста безопасности (имени пользователя и пароля), поэтому ему не может быть назначен ключ.</span><span class="sxs-lookup"><span data-stu-id="13bfb-125">This is because a shared mailbox does not have its own security context (username/password) so it cannot be assigned a key.</span></span> <span data-ttu-id="13bfb-126">Если участнику назначено несколько пользователей, они отправляют и получают сообщения, зашифрованные с помощью собственных ключей, другие участники могут читать электронную почту, а другие — в зависимости от того, какой открытый ключ использовался для шифрования электронной почты.</span><span class="sxs-lookup"><span data-stu-id="13bfb-126">If more than one person is a member, and they send/receive emails they encrypted with their own keys, other members might be able to read the email and others might not, depending which public key the email was encrypted with.</span></span>

- <span data-ttu-id="13bfb-133">**Вход:** Общий почтовый ящик не предназначен для прямого входа с помощью связанной с ним учетной записи пользователя.</span><span class="sxs-lookup"><span data-stu-id="13bfb-133">**Signing in:** A shared mailbox is not intended for direct sign-in by its associated user account.</span></span> <span data-ttu-id="13bfb-134">Вы всегда можете заблокировать вход для учетной записи общего почтового ящика и сохранить его блокировку.</span><span class="sxs-lookup"><span data-stu-id="13bfb-134">You should always block sign-in for the shared mailbox account and keep it blocked.</span></span>

- <span data-ttu-id="13bfb-135">**Слишком много пользователей:** При одновременном доступе к общему почтовому ящику слишком большого числа назначенных пользователей могут возникать сбои при подключении к этому почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="13bfb-135">**Too many users:** When there are too many designated users concurrently accessing a shared mailbox, they may intermittently fail to connect to this mailbox.</span></span> <span data-ttu-id="13bfb-136">В этом случае можно сократить количество пользователей или использовать другую рабочую нагрузку, например группу Microsoft 365 или общую папку.</span><span class="sxs-lookup"><span data-stu-id="13bfb-136">In this case, you can consider reducing the number of the users or using a different workload, such a Microsoft 365 group or Public folder.</span></span>

- <span data-ttu-id="13bfb-137">**Удаление сообщения:** К сожалению, вы не можете запретить пользователям удалять сообщения в общем почтовом ящике.</span><span class="sxs-lookup"><span data-stu-id="13bfb-137">**Message deletion:** Unfortunately, you can't prevent people from deleting messages in a shared mailbox.</span></span> <span data-ttu-id="13bfb-138">Единственный способ сделать это — создать группу Microsoft 365 вместо общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="13bfb-138">The only way around this is to create a Microsoft 365 group instead of a shared mailbox.</span></span> <span data-ttu-id="13bfb-139">Группа в Outlook аналогична общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="13bfb-139">A group in Outlook is like a shared mailbox.</span></span> <span data-ttu-id="13bfb-140">Сравнение этих двух способов приведено в разделе [Сравнение групп](../create-groups/compare-groups.md).</span><span class="sxs-lookup"><span data-stu-id="13bfb-140">For a comparison of the two, see [Compare groups](../create-groups/compare-groups.md).</span></span> <span data-ttu-id="13bfb-141">Дополнительные сведения о группах приведены в [статье дополнительные сведения о группах](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2).</span><span class="sxs-lookup"><span data-stu-id="13bfb-141">To learn more about groups, see [Learn more about groups](https://support.microsoft.com/office/b565caa1-5c40-40ef-9915-60fdb2d97fa2).</span></span>
