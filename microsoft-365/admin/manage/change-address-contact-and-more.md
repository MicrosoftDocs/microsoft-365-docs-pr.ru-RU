# <a name="change-your-organizations-address-technical-contact-and-more"></a>Изменение адреса организации и других данных профиля

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet).

::: moniker-end
  
Вы можете внести изменения в профиль организации, такие как название организации, адрес, телефон и контактная информация. **Для обновления этих сведений вы должны быть глобальным администратором.**
  
Чтобы изменить адрес, связанный с вашим счетом или подпиской, см. "Изменение адресов вы выставления счета [для Microsoft 365 для бизнеса".](../../commerce/billing-and-payments/change-your-billing-addresses.md)

## <a name="edit-organization-information"></a>Изменение сведений об организации

> [!IMPORTANT]
> Вы не можете изменить страну или регион для своей подписки. Это необходимо потому, что страна или регион, в котором расположена организация, определяет, какие службы доступны вам, налоги и валюту вы выставления счета, а также расположение центра обработки данных. Чтобы изменить страну или регион вашей организации, зарегистриройте новую учетную запись, выберите нужную страну или регион и приобретите новую подписку.

Чтобы изменить другие сведения на странице профиля вашей компании:
  
1. В Центре администрирования перейдите **на** страницу параметров организации \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**параметров.**</a>

2. На **вкладке "Профиль организации"** выберите **"Сведения об организации".**

3. Обновите сведения организации, а затем выберите **"Сохранить изменения".** Обязательно заполните все необходимые поля с пометкой *, чтобы сохранить изменения.

Ниже приведены описания всех полей.

## <a name="change-your-alternate-email-address"></a>Изменение альтернативного адреса электронной почты

Как администратор вы можете изменить альтернативный адрес электронной почты для сброса паролей.

1. Перейдите в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Центр администрирования Microsoft 365.</a>

2. В заголке выберите значок профиля, а затем выберите **"Просмотреть учетную запись".**

3. Слева выберите сведения **о безопасности.**

4. В разделе "Электронная почта" выберите **"Изменить"**

5. Edit the address in the **Alternate email** box, then select **Next**.

6. Введите код, отправленный на альтернативный адрес электронной почты, и выберите **"ОК",** чтобы проверить изменение.
Чтобы узнать об изменении других сведений профиля, см. сведения об изменении личных [настроек](change-contact-preferences.md) контактов [или изменении языка отображения.](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b.aspx)
