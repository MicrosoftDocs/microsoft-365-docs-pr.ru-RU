# <a name="search-in-the-microsoft-365-admin-center"></a>Поиск в Центре администрирования Microsoft 365

Администраторы организации Microsoft 365 могут искать пользователей, выполнять действия, переходить к другим настройкам и читать документацию. Благодаря новым функциям поиска скорость поиска была улучшена, и теперь вы можете искать со всех страниц в Центре администрирования. Поле поиска перемещено в область баннеров в верхней части центра администрирования. Вы можете использовать **ярлык ALT+S** для поиска с любой страницы.

:::image type="content" source="../../media/search-bar.png" alt-text="Screenshot: Search bar in the Microsoft 365 admin center":::

Результаты поиска организованы по разным категориям. Большинство категорий — это элементы в Центре администрирования. Например, пользователи, группы, общие почтовые ящики или домены. В других категориях покажутся места, в которые можно перейти, действия, которые можно принять, или параметры уровня приложения, которые можно изменить. Кроме того, существует категория, связанная с документацией.

В следующих разделах описываются различные области и категории в Центре администрирования, которые можно найти.

## <a name="before-you-begin"></a>Прежде чем начать

Для поиска в Центре администрирования необходимо быть администратором. Областью результатов поиска являются разрешения администратора, которые есть у во время входа пользователя. Например, если администратор SharePoint не видит область или категорию в Центре администрирования, он не увидит ее в поиске.

## <a name="users"></a>Пользователи

Пользователей можно найти по отображаемой имени, фамилии, имени, имени пользователя, основному адресу электронной почты или псевдонимам электронной почты. Выберите изменение имени пользователя, чтобы изменить сведения о пользователе.

:::image type="content" source="../../media/User-search-stock-picture-domain-blanked.png" alt-text="Screenshot: Search for a userNderson":::

Если рядом с их именем выбрано меню "Дополнительные действия", можно сбросить пароль.

:::image type="content" source="../../media/User-search-stock-picture-reset-password-domain-blanked.png" alt-text="Screenshot: User search with the option to reset password from search result":::

**Советы по улучшению результатов поиска пользователей**

- Убедитесь, что имена пользователей написаны правильно, так как поисковые запросы пользователей точно соответствуют указанным выше свойствам. Например, в примере выше Юс или Малц будут работать, но опечатка, например, Йосина вместо Зумы не найдет этого пользователя.
- Чтобы получить точное совпадение, наберем основной адрес электронной почты или имя пользователя.

## <a name="groups"></a>Группы

Вы можете искать группы по имени или адресу электронной почты группы. Вы можете выбрать группу и изменить ее на любой странице.

:::image type="content" source="../../media/search-for-groups-mac.png" alt-text="Screenshot: Search results for groups in the admin center":::

**Советы по улучшению результатов поиска в группах**

Убедитесь, что имя группы написано правильно.

## <a name="actions"></a>Actions

В Центре администрирования M365 можно найти категории "Действия", которые содержат часто используемые действия. Подумайте о действиях как о глаголе в системе. Например, можно также искать "сбросить пароль" на любой странице, а затем сбросить один или несколько паролей для пользователей. Вы можете найти "удалить пользователя" и удалить его со страницы "Удалить пользователя".

:::image type="content" source="../../media/search-for-actions-mac.png" alt-text="Screenshot: Search results showing actions you can perform in the admin center":::

## <a name="navigation"></a>Навигация

Результаты предоставляют возможность быстро перейти на определенную страницу в Центре администрирования. Например, при поиске RBAC вы будете перенаться на страницу "Роли" для ролей Azure AD.

:::image type="content" source="../../media/search-for-navigation-mac.png" alt-text="Screenshot: Search results showing navigation path in the admin center":::

## <a name="settings"></a>Параметры

Поиск поддерживаемых параметров уровня приложений, связанных с вашей организацией, служб, на которые вы подписаны, а также параметров безопасности и конфиденциальности.

:::image type="content" source="../../media/search-for-settings-mac.png" alt-text="Screenshot: Search results showing settings in the admin center":::

## <a name="domain"></a>Домен

Вы можете найти быстрые ссылки на свои домены, а затем по этой ссылке перейдите на страницу обзора этого домена.

:::image type="content" source="../../media/search-for-domains-mac.png" alt-text="Результаты поиска с доменами, владельцем которых является Центр администрирования":::

## <a name="documentation"></a>Документация

Поиск в документации предоставляет соответствующую справку на основе фразы поиска. Щелкните раздел, чтобы узнать больше.

:::image type="content" source="../../media/search-for-docs-mac.png" alt-text="Screenshot: Search showing documentation results in the admin center":::

## <a name="send-us-feedback"></a>Отправка отзывов и предложений

Используйте этот раздел для отправки отзывов об использовании поиска. Мы не можем отвечать на все отзывы, но читаем все эти отзывы и используем их для улучшения поиска. Не забудьте предоставить как можно больше подробностей в своих отзывах.
