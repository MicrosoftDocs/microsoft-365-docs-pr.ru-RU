# <a name="message-center"></a>Центр сообщений


Чтобы отслеживать будущие изменения, в том числе новые и измененные функции, запланированное обслуживание или другие важные объявления, перейдите в <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Центр сообщений</a>. 
  
Чтобы открыть центр сообщений:

::: moniker range="o365-worldwide"

- В центре администрирования перейдите в **Health** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Центр сообщений об</a>исправности.

::: moniker-end

::: moniker range="o365-germany"

- В <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">центре администрирования</a>перейдите в центр сообщений **об исправности** > **Message center**.

::: moniker-end

::: moniker range="o365-21vianet"
 
- В <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">центре администрирования</a>перейдите в центр сообщений **об исправности** > **Message center**.

::: moniker-end

Вы также можете использовать [приложение Microsoft 365 Admin](https://go.microsoft.com/fwlink/p/?linkid=627216) на мобильном устройстве для просмотра центра сообщений, что является прекрасным способом поддержания актуальности при push-уведомлениях.

### <a name="messages"></a>Сообщения

В центре сообщений представлено представление всех активных сообщений в формате таблицы. По умолчанию в начале списка отображается Последнее сообщение. Вы можете выбрать вкладку для просмотра **всех активных сообщений**, сообщений **с высоким уровнем важности** , **непрочитанных сообщений**и **закрытых сообщений**. Кроме того, с помощью раскрывающегося меню **Фильтр** можно просмотреть все сообщения в определенной категории сообщений.

Ниже приведен краткий обзор сведений, которые будут отображаться в каждом столбце. Выберите один из заголовков столбцов, чтобы изменить порядок сортировки.
  
|**Столбец**|**Описание**|
|:-----|:-----|

### <a name="major-updates"></a>Основные обновления

Основные обновления можно просмотреть, выбрав вкладку **Высокая важность** в верхней части центра сообщений.

Основные обновления сообщаются по крайней мере за 30 дней, когда требуется действие и могут включать:
  
- Изменение ежедневных продуктивностей, таких как папки "Входящие", "собрания", "Делегирование" и "доступ"

- Изменения, вносимые в темы, веб-части и другие компоненты, которые могут повлиять на индивидуальные функции

- Увеличивается или уменьшается до видимой емкости, такой как хранилище, количество правил, элементов или продолжительность

- Изменения фирменного стиля продукта, которые могут:

  - Приведут к путанице конечного пользователя

  - Приводятся изменения в процессах и справочных материалах службы поддержки, а также

  - Изменение URL-адреса

- Новая служба или приложение

- Изменения, требующие действия администратора (исключительные проблемы, связанные с предотвращением или исправлением)

- Изменения места хранения данных
  
### <a name="preferences"></a>Настройках

Если администрирование распространяется в пределах организации, вам может не потребоваться просматривать записи о всех службах Microsoft 365. Каждый администратор может выполнять следующие действия:

- Задайте параметры, определяющие, какие сообщения будут отображаться в центре сообщений.
- Фильтрация сообщений
- Настройка параметров электронной почты для получения еженедельного дайджеста всех сообщений, сообщений электронной почты только для основных обновлений и сообщений электронной почты для сообщений о конфиденциальности данных.  

::: moniker range="o365-worldwide"

1. В верхней части центра сообщений выберите **изменить параметры** .

2. Убедитесь, что установлен флажок для каждой службы, которую необходимо отслеживать. Снимите флажки для служб, которые требуется отфильтровать из представления "центр сообщений".

3. Сообщения дайджеста включены по умолчанию и отправляются на основной адрес электронной почты. Чтобы прекратить получение еженедельного дайджеста, снимите флажок **Отправить дайджест-электронную почту из "Мои сообщения** ". <br/><br/>Уведомление по электронной почте для основных обновлений является отдельным элементом управления. Если вы хотите получать уведомления по электронной почте о важных обновлениях, убедитесь, что выбран параметр **отправлять мне электронные письма для основных обновлений** . Снимите этот флажок, чтобы прекратить почтовые сообщения об основных обновлениях. <br><br/>Установите флажок рядом с пунктом **отправлять мне сообщения** электронной почты о конфиденциальности данных для получения отдельных уведомлений о конфиденциальности данных (сообщения о конфиденциальности данных не включаются в еженедельный дайджест). <br><br/>Вы можете выбрать или очистить основной адрес электронной почты, но не можете изменить его. Чтобы указать другие адреса электронной почты, на которые отправляется сообщение о конфиденциальности данных, установите флажок **другие адреса электронной почты** и введите другие адреса электронной почты, на которые необходимо отправить сообщение. Введите адрес электронной почты для группы Microsoft 365 или списка рассылки, если требуется, чтобы более двух пользователей получили сообщение о конфиденциальности данных.

4. Нажмите кнопку **сохранить** , чтобы сохранить изменения.
  
::: moniker-end

3. Сообщения дайджеста включены по умолчанию и отправляются на основной адрес электронной почты. Если вы хотите получать уведомления по электронной почте о важных обновлениях, убедитесь **в**том, что для **основных обновлений включена отправка электронных писем** . Чтобы указать другие адреса электронной почты **, на которые**отправляется Еженедельная сводка по электронной почте, убедитесь, что включен **Еженедельный дайджест моих сообщений** . Введите адрес электронной почты для группы Microsoft 365 или списка рассылки, если необходимо получить более двух пользователей из сводной почты.

4. Нажмите кнопку **сохранить** , чтобы сохранить изменения.<br/>
