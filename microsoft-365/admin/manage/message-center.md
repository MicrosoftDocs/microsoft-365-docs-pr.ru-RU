# <a name="message-center"></a>Центр сообщений


Чтобы отслеживать предстоящие изменения, включая новые и измененные функции, запланированное обслуживание и другие важные объявления, перейдите в <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Центр сообщений.</a> 
  
Чтобы открыть Центр сообщений, скроем:

::: moniker range="o365-worldwide"

- В Центре администрирования перейдите в **Центр** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">сообщений о состоянии.</a>

::: moniker-end

::: moniker range="o365-germany"

- В Центре <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">администрирования перейдите</a>в **Центр** > **сообщений о состоянии.**

::: moniker-end

::: moniker range="o365-21vianet"
 
- В Центре <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">администрирования перейдите</a>в **Центр** > **сообщений о состоянии.**

::: moniker-end

Вы также можете использовать приложение ["Администратор Microsoft 365"](https://go.microsoft.com/fwlink/p/?linkid=627216) на мобильном устройстве для просмотра Центра сообщений, что позволяет получать push-уведомления.

### <a name="messages"></a>Сообщения

Центр сообщений представляет представление всех активных сообщений в табличные форматы. По умолчанию в верхней части списка показано самое последнее сообщение. Вы можете выбрать вкладку, чтобы увидеть все **активные** **сообщения,** сообщения высокой важности, непрочитанные и **отклоненные сообщения.** Кроме того, вы можете **использовать** меню "Фильтр", чтобы увидеть все сообщения в определенной категории сообщений.

Ниже представлен краткий обзор сведений, которые вы увидите в каждом столбце. Выберите любой из заголовков столбцов, чтобы изменить порядок сортировки.
  
|**Столбец**|**Описание**|
|:-----|:-----|
|Флажок  <br/> |При выборе контрольного знака в строке заголовка столбца будут выбраны все сообщения, которые отображаются в данный момент. Если выбрать этот контрольный знак рядом с одним или более сообщениями, вы сможете принять меры с этими сообщениями.  <br/> |
|Значок приложения <br/> |Значки указывают приложение, к которому применяется сообщение.<br/> |
|Название сообщения  <br/> |Заголовки сообщений — это краткое описание предстоящих изменений. Если полный заголовок не отображается, наведите на него курсор и весь заголовок появится во всплывающее окно.  <br/> |
|Дополнительные параметры <br/> |Дополнительные параметры позволяют отклонять сообщение, пометить его как прочитанные или непрочитанные или поделиться им с другим администратором. Чтобы восстановить отклоненное сообщение,  выберите вкладку "Отклоненные сообщения", выберите метку рядом с сообщением и выберите **"Восстановить".** <br/> |

Предотвращение или устранение проблем отличается от сообщений о состоянии службы, так как они предупреждают вас о том, чтобы избежать проблем. <br/> <br/> **Планирование изменений:** информирует вас об изменениях в Microsoft 365, которые могут потребовать от вас действий во избежание сбоев в работе службы. ### <a name="major-updates"></a>Основные обновления

Основные обновления можно просмотреть, выбрав вкладку "Высокая **важность"** в верхней части центра сообщений.

Основные обновления обновляются по крайней мере за 30 дней до того, когда требуется действие, и могут включать:
  
- Изменения ежедневной производительности, такие как "Входящие", собрания, делегирования, общий доступ и доступ

- Изменения тем, веб-частей и других компонентов, которые могут повлиять на пользовательские функции

- Увеличивает или уменьшается до видимой емкости, такой как хранение, количество правил, элементов или длительности

- Изменения фирменой марки продукта, которые могут:

  - Привести к путанице для конечных пользователей.

  - Внесение изменений в процессы службы поддержки и справочные материалы или

  - Изменение URL-адреса

- Новая служба или приложение

- Изменения, требующие действий администратора (исключительно для предотвращения или устранения проблем)

- Изменения места хранения данных
  
### <a name="preferences"></a>Параметры

Если администрирование распространяется по всей организации, возможно, вам не потребуется видеть публикации обо всех службах Microsoft 365. Каждый администратор может:

- Задав параметры, которые контролируют, какие сообщения отображаются в Центре сообщений.
- Фильтрация сообщений
- Установите параметры электронной почты, чтобы получать еженедельный дайджест всех сообщений, сообщений электронной почты только для основных обновлений и сообщений электронной почты для сообщений о конфиденциальности данных.  

::: moniker range="o365-worldwide"

1. Выберите **"Изменить параметры"** в верхней части центра сообщений.

2. Убедитесь, что для каждой службы, которую вы хотите отслеживать, будет выбран этот контроль. Сойите флажки для служб, которые нужно отфильтровать, из представления Центра сообщений.

3. Дайджест-сообщения по умолчанию отключены и отправляются на ваш основной адрес электронной почты. Чтобы перестать получать еженедельный  дайджест, спишите еженедельный дайджест моих сообщений. <br/><br/>Уведомление по электронной почте для основных обновлений — это отдельный контроль. Если вы хотите получать уведомления по электронной почте о крупных обновлениях, убедитесь, что выбрана отправка сообщений электронной почты для **основных** обновлений. Чтобы прекратить получение электронной почты о важных обновлениях, спишите этот ящик. <br><br/>Для получения отдельных  уведомлений о конфиденциальности данных по электронной почте (сообщения о конфиденциальности данных не включаются в еженедельный дайджест) включайте в него отдельные уведомления о конфиденциальности данных. <br><br/>Основной адрес электронной почты можно выбрать или очистить, но изменить его нельзя. Чтобы указать другие адреса электронной почты, на  которые отправляется сообщение о конфиденциальности данных, введите другие адреса электронной почты, на которые будет отправлено сообщение электронной почты. Введите адрес электронной почты для группы Microsoft 365 или списка рассылки, если более двух человек должны получать электронную почту о конфиденциальности данных.

4. Выберите **"Сохранить",** чтобы сохранить изменения.
  
::: moniker-end

4. Выберите **"Сохранить",** чтобы сохранить изменения.<br/>
