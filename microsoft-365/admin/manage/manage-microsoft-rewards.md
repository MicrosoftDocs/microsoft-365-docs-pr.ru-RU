# <a name="manage-microsoft-rewards"></a>Управление Microsoft Rewards

Пользователи, у которых есть личные учетные записи Майкрософт, могут получать очки [Microsoft Rewards](https://www.microsoft.com/rewards) в связи с их поиском в работе, если включена функция Link *Microsoft Azure Active Directory (Azure AD) с* функцией Rewards.

Если пользователь не откажется от этой функции, его личная учетная запись Майкрософт будет связана с учетной записью Azure AD на рабочем месте. Microsoft Rewards получает и регистрит сведения о том, что поиск произошел, но не получает сведений о содержимом поиска. Содержимое запросов из организации не будет делиться с Microsoft Rewards в рамках этой функции.

Для администраторов образовательных организаций с несовершеннолетними детьми для участия в программе Microsoft Rewards для ребенка требуется родительская учетная запись Microsoft. Родительская учетная запись не будет связана с организационной учетной записью учащегося. Дополнительные сведения о учетных записях Майкрософт для детей см. в записях родительского согласия и [учетных записях детей Майкрософт.](https://support.microsoft.com/account-billing/c6951746-8ee5-8461-0809-fbd755cd902e)

Эта функция недоступна для государственных пользователей. Администраторы должны убедиться, что политики соответствия требованиям их организации позволяют использовать личные учетные записи Microsoft Rewards с помощью поиска в работе.
