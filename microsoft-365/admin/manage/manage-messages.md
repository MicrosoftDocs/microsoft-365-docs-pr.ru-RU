# <a name="manage-messages-in-message-center"></a>Управление уведомлениями в Центре сообщений

В Центре сообщений публикуются важные объявления о новых и измененных функциях. В этой статье вы найдете подробные сведения о том, как использовать эти сообщения в рамках управления изменениями.
  
## <a name="read-a-message"></a>Чтение сообщения

Чтобы перейти в Центр сообщений, выберите [Центр администрирования](https://go.microsoft.com/fwlink/p/?linkid=2024339), **Работоспособность** > **Центр сообщений**. 

Щелкните заголовок сообщения, чтобы открыть его в области чтения. Каждое сообщение содержит описание изменения, сведения о том, как оно может повлиять на вас, инструкции по подготовке организации к нему, а также (в большинстве случаев) ссылку на дополнительные сведения. Дополнительные сведения будут доступны даже после того, как истечет срок действия сообщения.

## <a name="give-feedback-on-a-post"></a>Оставить отзыв о публикации

В Центре сообщений можно выбрать сообщение, чтобы просмотреть подробные сведения.

Чтобы оставить отзыв о сообщении, выберите значок **Нравится** или **Не нравится** и оставьте свой отзыв в появившемся текстовом поле. Не предоставляйте личные сведения. При необходимости можно выбрать **Со мной можно связаться по поводу отзыва**, а затем **Отправить**.

## <a name="share-a-message"></a>Пересылка сообщения

Действия, требуемые в сообщении, должен выполнить кто-то другой? Вы можете поделиться содержимым сообщения, отправив его по электронной почте:
  
1. Выберите сообщение, чтобы открыть его, а затем нажмите **Поделиться**.
  
2. Чтобы поделиться сообщением, введите до двух адресов электронной почты, разделяя их точкой с запятой. Вы можете отправлять сообщения на индивидуальные электронные адреса или в группы адресов электронной почты. При желании можно получить копию сообщения в электронном письме (сообщение будет отправлено на основной адрес электронной почты) или добавить личное сообщение, предоставив получателям больше контекста.
  
3. Нажмите **Поделиться**, чтобы отправить сообщение.
    
## <a name="get-a-link"></a>Получение ссылки

Вам нужно убедиться, что другой администратор знает об изменении и принимает соответствующие меры? Вы можете создать ссылку на уведомление и отправить ее по электронной почте или через мгновенные сообщения. Пользователь, с которым вы делитесь ссылкой, должен быть администратором Microsoft 365 в вашей организации, чтобы иметь доступ к сообщению.

::: moniker range="o365-worldwide"

1. Выберите сообщение, чтобы открыть его.

2. Выберите значок (ссылка) **Копировать ссылку в буфер обмена**.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка будет скопирована в буфер обмена.

4. Чтобы вставить ее, нажмите клавиши CTRL+V или щелкните правой кнопкой мыши и выберите **Вставить**.

::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. На панели действий выберите **Еще** \> **Получить ссылку**.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка будет скопирована в буфер обмена.

4. Чтобы вставить ее, нажмите клавиши CTRL+V или щелкните правой кнопкой мыши и выберите **Вставить**.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. На панели действий выберите **Еще** \> **Получить ссылку**.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка будет скопирована в буфер обмена.

4. Чтобы вставить ее, нажмите клавиши CTRL+V или щелкните правой кнопкой мыши и выберите **Вставить**.

## <a name="read-and-unread-states"></a>Состояния "Прочитанное" и "Непрочитанное"

Любое непрочитанное сообщение в Центре выделяется полужирным шрифтом. При открытии сообщения оно помечается как прочитанное. Сообщение можно пометить как непрочитанное.

::: moniker range="o365-worldwide"

1. На главной странице Центра сообщений выберите сообщение.

2. В верхней части сообщения щелкните значок (конверт) **Пометить как непрочитанное**.
  
::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

## <a name="archive-and-restore"></a>Архивация и восстановление

Если вы видите сообщение, которое не относится к вам или которое уже не актуально, вы можете заархивировать сообщение, чтобы удалить его из Центра сообщений. Представление, которое вы видите в Центре сообщений, связано только с вашей учетной записью, поэтому архивация сообщения не повлияет на других администраторов.

::: moniker range="o365-worldwide"

- На главной странице Центра сообщений выберите сообщение, а затем щелкните **Архивировать** над списком сообщений.

- Откройте сообщение и выберите **Архивировать** в нижней части сообщения.

Необходимо вернуть архивное сообщение? Это не проблема.
  
1. В верхней части Центра сообщений выберите вкладку **Архивные сообщения**. Отобразится список архивных сообщений. 

2. Выберите сообщение, нажмите **Восстановить**, и оно будет восстановлено в активных сообщениях.

::: moniker-end

::: moniker range="o365-germany"

Архивировать сообщения можно двумя способами.
  
- Выберите значок X справа от идентификатора сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение и выберите **Архивировать** в верхней части сообщения.

Нужно вернуть удаленное сообщение? Никаких проблем!
  
1. Выберите фильтр **Архивные сообщения** в раскрывающемся меню **Представления**. Отобразится список архивных сообщений.

2. Щелкните **Восстановить** справа от идентификатора сообщения, и оно появится в списке активных сообщений.

::: moniker-end

::: moniker range="o365-21vianet"

Архивировать сообщения можно двумя способами.
  
- Выберите значок X справа от идентификатора сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение и выберите **Архивировать** в верхней части сообщения.

Необходимо вернуть архивное сообщение? Это не проблема.
  
1. Выберите фильтр **Архивные сообщения** в раскрывающемся меню **Представления**. Отобразится список архивных сообщений.

2. Щелкните **Восстановить** справа от идентификатора сообщения, и оно появится в списке активных сообщений.

::: moniker-end

## <a name="track-your-message-center-tasks-in-planner"></a>Отслеживание задач центра сообщений в Планировщике

Многие полезные сведения об изменениях служб Microsoft 365 поступают в центр сообщений Microsoft 365. Может быть сложно отслеживать, какие изменения требуют выполнения задач, сроки выполнения и исполнителей, а также завершение каждой задачи. Кроме того, вам может потребоваться создать заметку о чем-нибудь и пометку для проверки позже. Вы можете выполнять все эти и другие действия, синхронизировав сообщения из Центра администрирования Microsoft 365 с Планировщиком (Майкрософт). Дополнительные сведения см. в статье [Отслеживание своих задач Центра сообщений в Планировщике](https://docs.microsoft.com/office365/planner/track-message-center-tasks-planner).


Общие сведения о Центре сообщений см. в статье [Центр сообщений в Microsoft 365](message-center.md). Сведения о настройке языковых параметров для включения машинного перевода записей Центра сообщений, см. в статье [Перевода записей Центра сообщений](language-translation-for-message-center-posts.md). Если вы хотите получать актуальные сведения о работоспособности служб и уведомления Центра сообщений программным путем, см. [Обзор API взаимодействия служб Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=848507).
