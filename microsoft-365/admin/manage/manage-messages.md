# <a name="manage-messages-in-message-center"></a>Управление уведомлениями в Центре сообщений

В Центре сообщений публикуются важные объявления о новых и измененных функциях. В этой статье вы найдете подробные сведения о том, как использовать эти сообщения в рамках управления изменениями.
  
## <a name="read-a-message"></a>Чтение сообщения

Чтобы перейти в центр сообщений, перейдите в [центр администрирования](https://go.microsoft.com/fwlink/p/?linkid=2024339)и выберите Центр сообщений об **исправности**  >  **Message center**. 

Выберите заголовок сообщения, чтобы открыть его в области чтения. Каждое сообщение содержит сводное описание изменения, сведения о том, как это изменение может повлиять на вас, сведения о том, как подготовить Организацию к изменению и (в большинстве случаев) дополнительную информационную ссылку, где можно узнать больше. Дополнительная информация будет доступна даже после истечения срока действия сообщения.

## <a name="give-feedback-on-a-post"></a>Обратная связь для записи

В центре сообщений можно выбрать сообщение, чтобы просмотреть подробные сведения.

Если вы хотите оставить отзыв о сообщении, выберите значок " **нравится** " или "не **нравится** " и введите отзыв в появившемся текстовом поле. Не предоставляйте никаких личных сведений. При необходимости вы можете установить **связь со мной** и нажать кнопку **Отправить**.

## <a name="share-a-message"></a>Пересылка сообщения

Действия, требуемые в сообщении, должен выполнить кто-то другой? Вы можете поделиться содержимым сообщения, отправив его по электронной почте:
  
1. Выберите сообщение, чтобы открыть его, а затем выберите пункт **общий доступ**.
  
2. Чтобы поделиться сообщением, введите до двух адресов электронной почты, разделенных двоеточием. Вы можете отправлять электронные адреса пользователей и групп. Кроме того, можно выбрать получение копии сообщения по электронной почте (сообщение будет отправлено на основной адрес электронной почты) или добавить личное сообщение, чтобы предоставить получателям больше контекста.
  
3. Выберите **общий доступ** , чтобы отправить сообщение электронной почты.
    
## <a name="get-a-link"></a>Создание ссылки

Вам нужно убедиться, что другой администратор знает об изменении и принимает соответствующие меры? Вы можете создать ссылку на уведомление и отправить ее по электронной почте или через мгновенные сообщения. Пользователь, которому предоставляется доступ к ссылке, должен быть администратором Microsoft 365 для вашей организации или не иметь доступа к связанному сообщению.

::: moniker range="o365-worldwide"

1. Выберите сообщение, чтобы открыть его.

2. Выберите значок **Копировать ссылку на буфер обмена** (ссылка).

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Нажмите клавиши CTRL + V или щелкните правой кнопкой мыши и выберите команду **Вставить** , чтобы вставить ссылку.

::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. Нажмите кнопку **More** \> **Get (получить ссылку** ) на панели действий.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Нажмите клавиши CTRL + V или щелкните правой кнопкой мыши и выберите команду **Вставить** , чтобы вставить ссылку.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. Нажмите кнопку **More** \> **Get (получить ссылку** ) на панели действий.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Нажмите клавиши CTRL + V или щелкните правой кнопкой мыши и выберите команду **Вставить** , чтобы вставить ссылку.

::: moniker-end

## <a name="read-and-unread-states"></a>Состояния "Прочитанное" и "Непрочитанное"

Любое непрочитанное сообщение в Центре выделяется полужирным шрифтом. При открытии сообщения оно помечается как прочитанное. Вы можете пометить сообщение как непрочтенное.

::: moniker range="o365-worldwide"

1. На главной странице центра сообщений выберите сообщение.

2. Выберите значок **Пометить как непрочтенный** (конверт) в верхней части сообщения.
  
::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

## <a name="archive-and-restore"></a>Архивация и восстановление

Если вы видите сообщение, которое не относится к вам или вы уже выполнили действия, можно заархивировать сообщение, чтобы удалить его из представления "центр сообщений". Представление, которое отображается в центре сообщений, относится только к учетной записи пользователя, поэтому их архивация из представления не влияет на других администраторов.

::: moniker range="o365-worldwide"

- На главной странице центра сообщений выберите сообщение, а затем выберите пункт **Архив** над списком сообщений.

- Откройте сообщение, а затем в нижней части сообщения выберите **Архивировать** .

Нужно вернуть Архивное сообщение? Никаких проблем!
  
1. Выберите вкладку **заархивированные сообщения** в верхней части центра сообщений. Отобразится список архивных сообщений. 

2. Выберите сообщение, нажмите кнопку **восстановить**, и сообщение будет восстановлено до активных сообщений.

::: moniker-end

::: moniker range="o365-germany"

Существует два способа архивации сообщений.
  
- Выберите значок X, расположенный справа от идентификатора сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение, а затем выберите **Архивировать** в верхней части сообщения.

Нужно вернуть удаленное сообщение? Никаких проблем!
  
1. В раскрывающемся меню **представления** выберите фильтр **архивных сообщений** . Отобразится список архивных сообщений.

2. Выберите вариант **восстановить** справа от идентификатора сообщения, после чего сообщение будет восстановлено до активных сообщений.

::: moniker-end

::: moniker range="o365-21vianet"

Существует два способа архивации сообщений.
  
- Выберите значок X, расположенный справа от идентификатора сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение, а затем выберите **Архивировать** в верхней части сообщения.

Нужно вернуть Архивное сообщение? Никаких проблем!
  
1. В раскрывающемся меню **представления** выберите фильтр **архивных сообщений** . Отобразится список архивных сообщений.

2. Выберите вариант **восстановить** справа от идентификатора сообщения, после чего сообщение будет восстановлено до активных сообщений.

::: moniker-end
