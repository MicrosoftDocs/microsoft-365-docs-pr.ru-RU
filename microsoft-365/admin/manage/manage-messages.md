# <a name="manage-messages-in-message-center"></a>Управление уведомлениями в Центре сообщений

В Центре сообщений публикуются важные объявления о новых и измененных функциях. В этой статье вы найдете подробные сведения о том, как использовать эти сообщения в рамках управления изменениями.
  
## <a name="read-a-message"></a>Чтение сообщения

Чтобы перейти в Центр сообщений, перейдите в Центр [администрирования](https://go.microsoft.com/fwlink/p/?linkid=2024339)и выберите **"Центр**  >  **сообщений о состоянии".** 

Выберите заголовок сообщения, чтобы открыть его в области чтения. Каждое сообщение содержит сводку об изменении, описание того, как это изменение может повлиять на вас, сведения о подготовке организации к изменению и (в большинстве случаев) дополнительную ссылку для получения дополнительных сведений. Дополнительные сведения по-прежнему доступны даже после истечения срока действия сообщения.

## <a name="give-feedback-on-a-post"></a>Обратная связь с публикацией

В Центре сообщений можно выбрать сообщение, чтобы увидеть подробные сведения.

Если вы хотите отправить отзыв о сообщении,  выберите значок "Нравится" или "Не нравится" и в текстовом поле.  Не предоставлять никаких персональных данных. При желании вы можете выбрать "ОК", чтобы связаться со мной **по этому** отзыву, а затем выбрать "Отправить". 

## <a name="share-a-message"></a>Пересылка сообщения

Действия, требуемые в сообщении, должен выполнить кто-то другой? Вы можете поделиться содержимым сообщения, отправив его по электронной почте:
  
1. Выберите сообщение, чтобы открыть его, а затем выберите **"Поделиться".**
  
2. Чтобы поделиться сообщением, введите до двух адресов электронной почты, разделенных двоеточием. Вы можете отправлять сообщения на отдельные адреса электронной почты и групп. При желании вы можете получить копию сообщения по электронной почте (сообщение будет отправлено на ваш основной адрес электронной почты) или добавить личное сообщение, чтобы предоставить получателям дополнительный контекст.
  
3. Выберите **"Поделиться",** чтобы отправить сообщение электронной почты.
    
## <a name="get-a-link"></a>Создание ссылки

Вам нужно убедиться, что другой администратор знает об изменении и принимает соответствующие меры? Вы можете создать ссылку на уведомление и отправить ее по электронной почте или через мгновенные сообщения. Человек, с которого вы поделились ссылкой, должен быть администратором Microsoft 365 для вашей организации или у него не будет доступа к связанному сообщению.

::: moniker range="o365-worldwide"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **ссылку "Копировать" на значок буфера** обмена (ссылки).

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Используйте CTRL+V или щелкните правой кнопкой мыши и выберите **"Вставить",** чтобы вставить ссылку.

::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. Choose **More** \> **Get a link** from the action bar.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Используйте CTRL+V или щелкните правой кнопкой мыши и выберите **"Вставить",** чтобы вставить ссылку.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. Choose **More** \> **Get a link** from the action bar.

3. Разрешите веб-странице доступ к буферу обмена. Прямая ссылка копируется в буфер обмена.

4. Используйте CTRL+V или щелкните правой кнопкой мыши и выберите **"Вставить",** чтобы вставить ссылку.

::: moniker-end

## <a name="read-and-unread-states"></a>Состояния "Прочитанное" и "Непрочитанное"

Любое непрочитанное сообщение в Центре выделяется полужирным шрифтом. При открытии сообщения оно помечается как прочитанное. Вы можете пометить сообщение как непрочитанные.

::: moniker range="o365-worldwide"

1. На главной странице центра сообщений выберите сообщение.

2. В **верхней** части сообщения выберите значок "Пометить как непрочитанный (конверт)".
  
::: moniker-end

::: moniker range="o365-germany"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

::: moniker range="o365-21vianet"

1. Выберите сообщение, чтобы открыть его.

2. Выберите **Еще** \> **Пометить как непрочитанное**.

3. Закройте сообщение.

::: moniker-end

## <a name="archive-and-restore"></a>Архив и восстановление

Если вы видите сообщение, которое не относится к вам, или, возможно, вы уже сделали это, вы можете заархивировать сообщение, чтобы удалить его из представления Центра сообщений. Представление, которое вы видите в Центре сообщений, является специфическим для вашей учетной записи пользователя, поэтому ее архивы не затрагивают других администраторов.

::: moniker range="o365-worldwide"

- На главной странице Центра сообщений выберите сообщение,  а затем выберите "Архивировать" над списком сообщений.

- Откройте сообщение и выберите **"Архив"** в нижней части сообщения.

Требуется вернуть архивное сообщение? Никаких проблем!
  
1. Выберите **вкладку "Архивные сообщения"** в верхней части центра сообщений. Появится список архивных сообщений. 

2. Выберите сообщение, выберите **"Восстановить",** и сообщение будет восстановлено в активные сообщения.

::: moniker-end

::: moniker range="o365-germany"

Архивировать сообщение можно двумя способами.
  
- Выберите X, который отображается справа от ИД сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение и выберите **"Архивировать"** в верхней части сообщения.

Нужно вернуть удаленное сообщение? Никаких проблем!
  
1. Выберите фильтр **архивных сообщений** в **меню** "Представления". Появится список архивных сообщений.

2. Выберите **"Восстановить"** справа от ИД сообщения, и сообщение будет восстановлено в активные сообщения.

::: moniker-end

::: moniker range="o365-21vianet"

Архивировать сообщение можно двумя способами.
  
- Выберите X, который отображается справа от ИД сообщения. Сообщение будет удалено из списка. Или:

- Откройте сообщение и выберите **"Архивировать"** в верхней части сообщения.

Требуется вернуть архивное сообщение? Никаких проблем!
  
1. Выберите фильтр **архивных сообщений** в **меню** "Представления". Появится список архивных сообщений.

2. Выберите **"Восстановить"** справа от ИД сообщения, и сообщение будет восстановлено в активные сообщения.

::: moniker-end

## <a name="track-your-message-center-tasks-in-planner"></a>Отслеживание задач центра сообщений в Планировщике

Многие полезные сведения об изменениях служб Microsoft 365 поступают в центр сообщений Microsoft 365. Может быть трудно отслеживать, какие изменения требуют выполнения задач, когда и кем, а также отслеживать выполнение каждой задачи. Кроме того, вам может потребоваться создать заметку о чем-нибудь и пометку для проверки позже. Вы можете сделать все это и больше при синхронизации сообщений из Центра администрирования Microsoft 365 с Планировщиком (Майкрософт). Дополнительные сведения см. в отслеживании [задач центра сообщений в Планировщике.](https://docs.microsoft.comoffice365/planner/track-message-center-tasks-planner)


Обзор Центра сообщений см. в [центре сообщений в Microsoft 365.](message-center.md) Или, чтобы узнать, как настроить языковые параметры, чтобы включить машинный перевод для записей Центра сообщений, см. перевод языков для записей [Центра сообщений.](language-translation-for-message-center-posts.md) Если вы хотите запрограммировать альтернативный способ получения сведений о состоянии службы в режиме реального времени и сообщений центра сообщений, обратитесь к обзору API связи со службами [Microsoft 365.](https://go.microsoft.com/fwlink/p/?linkid=848507)
