# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Получите последнюю версию аналитики использования Microsoft 365

Приложение шаблона может обновляться с помощью новых данных или новых визуализаций несколько раз в год. Существующий экземпляр продолжит работать, но если вы хотите получить последнюю версию, необходимо создать новый экземпляр и применить все настройки к новому экземпляру. См. ["Включить аналитику использования Microsoft 365".](enable-usage-analytics.md)
