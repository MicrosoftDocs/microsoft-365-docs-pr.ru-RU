# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Получите последнюю версию Microsoft 365 аналитики использования

Приложение шаблона может обновляться с помощью новых данных или новых визуализаций несколько раз в год. Существующий экземпляр будет продолжать работать, но если вы хотите получить последнюю версию, необходимо создать новый экземпляр и применить любые настройки к новому экземпляру. См. [Microsoft 365 аналитики использования.](enable-usage-analytics.md)
