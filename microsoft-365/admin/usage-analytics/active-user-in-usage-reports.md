# <a name="active-user-in-microsoft-365-usage-reports"></a>Активный пользователь в отчетах об использовании Microsoft 365

## <a name="active-user-in-usage-reports"></a>Активный пользователь в отчетах об использовании

Активный пользователь продуктов Microsoft 365 для анализа использования Microsoft [](../activity-reports/activity-reports.md) [365](usage-analytics.md) и отчетов об активности в Центре администрирования определяется следующим образом. 
  
|**Продукт**|**Определение активного пользователя**|**Примечания**|
|:-----|:-----|:-----|
|Exchange Online  <br/> |Любой пользователь, который прочитал или отправил по крайней мере одно сообщение электронной почты.  <br/> |Данные календаря не учитываются (эта возможность будет добавлена в будущем).  <br/> |
|SharePoint Online  <br/> |Любой пользователь, который создал, изменил, просмотрел, удалил файл, предоставил к нему внутренний или внешний доступ, синхронизировался с клиентами на любом сайте или просмотрел страницу на любом сайте.  <br/> |Метрика активного пользователя для SharePoint Online в приложении шаблона "Аналитика использования Microsoft 365" отражает только пользователей, которые сделали действия с файлами на сайте группы SharePoint или на сайте группы. Приложение шаблона будет обновлено для синхронизации определения с тем же, что и в отчетах об использовании в Центре администрирования.  <br/> |
|OneDrive для бизнеса  <br/> |Любой пользователь, который создал, изменил, просмотрел, удалил файл, предоставил к нему внутренний или внешний доступ либо синхронизировался с клиентами на любом сайте.  <br/> ||
|Yammer  <br/> |Любой пользователь, который прочитал, опубликовал сообщение или поставил отметку "Нравится" в Yammer.  <br/> ||
|Skype для бизнеса  <br/> |Любой пользователь, который участвовал в одноранговом сеансе (включая обмен мгновенными сообщениями, голосовые вызовы и видеозвонки, общий доступ к приложениям и передачу файлов), организовал конференцию или принимал участие в ней.  <br/> ||
|Office  <br/> |Любой пользователь, активировавший подписку на Microsoft 365 профессиональный плюс, Visio Pro или Project Pro хотя бы на одном устройстве.  <br/> ||
|Группы Microsoft 365  <br/> |Любой участник группы, который выполнял действия в почтовом ящике (если в группу было отправлено хотя бы одно сообщение)  <br/> |Это определение будет расширено с помощью действий с файлами на сайте группы и группы Yammer (действия с файлами на сайте группы и сообщение, опубликованное в группе Yammer, связанной с группой).) В настоящее время эти данные недоступны в приложении "Аналитика использования Microsoft 365"  <br/> |
|Microsoft Teams  <br/> |Любой пользователь, который участвовал в сообщениях чата, приватных чатах, звонках, собраниях или других действиях. Другие действия определяются как количество других действий в команде, некоторые из которых включают и не ограничиваются: сообщения, приложения, работа с файлами, поиск, следующие команды и канал и их предпочтение.  <br/> ||
   
## <a name="adoption-metrics"></a>Показатели внедрения

[Аналитика использования Microsoft 365](usage-analytics.md) содержит дополнительные показатели внедрения, связанные с активными пользователями, чтобы со временем показать внедрение продуктов. Эти показатели действительны для выбранного месяца, года и продукта и определяются следующим образом. 
  
|**Метрика**|**Описание**|
|:-----|:-----|
|EnabledUsers  <br/> |Количество пользователей, которые могут использовать продукт за месяц.  <br/> |
|ActiveUsers  <br/> |Количество активных пользователей за месяц.  <br/> |
|MoMReturningUsers  <br/> |Количество активных пользователей за месяц, которые также были активны в предыдущем месяце.  <br/> |
|FirstTimeUsers  <br/> |Количество активных пользователей за месяц, которые ранее никогда не использовали службу.  <br/> |
|CumulativeActiveUsers  <br/> |Количество активных пользователей в месяце, а также любой предыдущий месяц.  <br/> |
|ActiveUsers(%)  <br/> |Процент пользователей, округлив до ближайшей десятой активной части месяца по сравнению с числом пользователей, включенных в этом месяце.  <br/> |
|MoMReturningUsers(%)  <br/> |Процент пользователей, округливших до ближайшей десятой части активного месяца, которые также были активны в предыдущем месяце по сравнению с количеством активных пользователей.  <br/> |
   
С 1 января 2018 г. были сброшены moMReturningUsers, FirstTimeUsers, CumulativeActiveUsers с включением &amp; Microsoft Teams.
