# <a name="how-to-get-help-in-the-microsoft-365-admin-center"></a>Как получить помощь в Центре администрирования Microsoft 365

Если вы администратор, [admin.microsoft.com](https://admin.microsoft.com) вы можете управлять подпиской на Microsoft 365 и эффективно ее эффективно. Иногда может не оказаться нужной задачей, требуется дополнительный контекст перед началом работы в потоке задач или просто невозможно быть уверенным в области и влиянии действий администратора. Чтобы охватить эти ситуации, мы предоставляем современную и интеллектуальную помощь в Центре администрирования Microsoft 365:

* **Встроенная справка** — справка в Центре администрирования

* **Интеллектуальная помощь** — самопо себе на движухом ИИ

* **Руководства и статьи — полный** контент в Интернете

## <a name="integrated-help"></a>Интегрированная справка

Справка интегрирована во всем Центре администрирования, поэтому она там же, где вам нужна. В верхней части многих страниц можно найти текст справки, который содержит информационный обзор задачи, а также ссылки на статьи, которые помогут быстро найти официальной документации для более подробного изучения.

![Страница "Группы" со справкой и ссылками на статьи](../../media/integrated-help.png)

## <a name="modern-self-help-powered-by-ai"></a>Современная самоповеряемая справка на движухом ИИ

Чтобы открыть современный интерфейс для самостоятельного решения задач,  созданный с помощью искусственного интеллекта, выберите кнопку "Нужна помощь" в Центре администрирования Microsoft 365. Например, если вы ищете "проверить мой домен", вы получите шаги и несколько статей, которые, как мы считаем, помогут вам. Мы используем машинное обучение для решения, которое поможет другим администраторам, которые ввели похожие запросы.

Кроме того, для тех ситуаций, когда вы не знаете, как что-то сделать в конкретной ситуации, используйте помощник по поддержке. В настоящее время этот опыт доступен только на английском языке. Чтобы включить помощник по поддержке, просто используйте выключателем  в верхней части области "Нужна помощь". Помощник по поддержке предоставляет интерфейс беседы, который поможет вам. После ввода запроса чат-робот задает уточняющие вопросы, чтобы получить правильный ответ для конкретной ситуации. Думайте о нем как о виртуальном помощнике для обнаружения решений и выполнения задач.

![Современная самопо себе](../../media/help-options.png)

Конечно, иногда на вопросы лучше всего отвечать людям. Если у нашей современной самостоятельной справки нет ответа, у вас всегда есть возможность обратиться к нашим агентам поддержки.
