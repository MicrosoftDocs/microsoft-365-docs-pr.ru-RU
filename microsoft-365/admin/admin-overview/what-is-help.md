# <a name="how-to-get-help-in-the-microsoft-365-admin-center"></a>Как получить помощь в центре Microsoft 365 администрирования

Если вы [администратор,](https://admin.microsoft.com) admin.microsoft.com место для управления подпиской на Microsoft 365. Иногда вы можете не найти правильной задачи, необходимо больше контекста, прежде чем приступить к потоку задач, или просто не может быть уверены в области и последствиях ваших действий в качестве администратора. Чтобы охватить эти ситуации, мы предоставляем современную помощь и интеллектуальную помощь во всем центре администрирования Microsoft 365 такими способами:

* **Интегрированная справка** — помощь в центре администрирования

* **Интеллектуальная помощь** — самопомощи с помощью ИИ

* **Руководства и статьи** — полный контент в Интернете

## <a name="integrated-help"></a>Интегрированная справка

Справка интегрирована во всем центре администрирования, поэтому она прямо там, когда она вам нужна. В верхней части многих страниц вы найдете текст inline help, который содержит информационный обзор поставленной задачи, а также ссылки на статьи, которые помогут быстро найти официальную документацию для более углубленного обучения.

![Страница Группы, показывающая влияемую справку и ссылки на статьи](../../media/integrated-help.png)

## <a name="modern-self-help-powered-by-ai"></a>Современная самопомощи с питанием от ИИ

Чтобы открыть современный интерфейс самопомощи, который используется с помощью искусственного интеллекта, выберите кнопку Need **Help** в центре Microsoft 365 администрирования. Например, если вы ищете для "проверить мой домен", вы получите шаги, а также несколько статей, которые, как мы думаем, помогут вам. Мы используем машинное обучение, чтобы создать ближайшее решение, которое помогло другим администраторам, которые ввели аналогичные запросы.

Или в те моменты, когда вы не знаете, как что-то сделать в конкретной ситуации, используйте помощник поддержки. В настоящее время этот опыт доступен только на английском языке. Чтобы включить помощник поддержки, просто используйте переключеть в верхней части области **справки** о необходимости. Помощник по поддержке предоставляет диалоговую интерфейс, чтобы помочь вам. После ввода запроса чат-бот задает уточняющие вопросы, чтобы получить правильный ответ для конкретной ситуации. Подумайте об этом как о виртуальном помощнике для обнаружения решений и выполнения задач.

![Современная самопомеха](../../media/help-options.png)

Конечно, иногда на вопросы лучше всего отвечать людям. Если наша современная самопомощная помощь не имеет ответа, у вас всегда есть возможность связаться с нашими агентами поддержки.
