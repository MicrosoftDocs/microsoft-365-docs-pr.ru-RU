# <a name="how-to-sign-up---admin-help"></a>Регистрация для получения справки для администраторов

::: moniker range="o365-worldwide"

Подпишитесь на Microsoft 365 для бизнеса, чтобы ваша команда могла начать использовать последние версии Word, Excel, PowerPoint и других программ Office в вашей организации.
  
::: moniker-end

::: moniker range="o365-germany"

Подпишитесь на Microsoft 365 для бизнеса, чтобы ваша команда могла начать использовать последние версии Word, Excel, PowerPoint и других программ Office в вашей организации.
  
::: moniker-end

::: moniker range="o365-21vianet"

Служба Office 365 под управлением 21Vianet разработана в соответствии с потребностями безопасной, надежной и масштабируемой облачной службы в Китае. В ее основе лежат технологии, лицензии на которые корпорация Майкрософт предоставила 21Vianet. Обслуживанием и предоставлением этой службы занимается не корпорация Майкрософт, а 21Vianet. 21Vianet работает, предоставляет и управляет доставкой службы. В сферу его деятельности входит размещение на собственных серверах, управление сетями и обеспечение инфраструктуры облачных вычислений. По лицензированию технологий Microsoft 21Vianet работает с локальными центрами обработки данных Майкрософт, чтобы обеспечить возможность использования служб Майкрософт и хранения данных в Китае. 21Vianet также предоставляет службы подписки и выставления счетов, а также поддержку.
  
> [!NOTE]
> Предоставление этих услуг регулируется китайским законодательством.
  
Когда вы подписываетесь на Office 365 под управлением 21Vianet, команда может начать использовать последние версии Word, Excel, PowerPoint и других программ Office в вашей организации.
  
Готовы к регистрации? [Выберите план](https://products.office.com/zh-cn/business/compare-office-365-for-business-plans).
  
::: moniker-end

## <a name="youll-be-asked-for-the-following-information-when-you-sign-up"></a>При регистрации вам будет предложено ввести следующие сведения:

- **Адрес и контактные данные для подписки**

  - **Страна**, в которой будут использоваться службы. Вы **не сможете** изменить сведения о стране ни позже, ни даже во время процесса регистрации. Чтобы указать другую страну, вам придется начать регистрацию заново.

  - **Электронная почта** и **номер телефона** для связи с вами в случае необходимости для решения вопросов, связанных с вашей подпиской. Например, если вы забыли пароль, мы используем эти сведения для отправки вам временного пароля. На указанный вами адрес электронной почты мы также будем отправлять вам сведения об оплате.

    > [!NOTE]
    > Адрес электронной почты, который вы вводите здесь, отличается от вашего адреса электронной почты Microsoft 365 (имя для входа ниже). Так как на этот адрес отправляются данные для выставления счетов, рекомендуем использовать адрес, подходящий для деловой переписки.
  
- **Имя для входа (идентификатор пользователя):** Этот адрес становится первым адресом электронной почты Microsoft 365, а также позволяет быстро начать работу.

    Это адрес электронной почты, который используется для входа. Например, если ваша компания называется Fourth Coffee, в качестве идентификатора пользователя можно выбрать адрес pavel@fourthcoffee.onmicrosoft.com.

    Большинство пользователей добавляют свой собственный домен в ближайшее время, чтобы они могли приступить к работе с электронной почтой. Например, если у вас есть домен с именем fourthcoffee.com, вы можете настроить свой адрес электронной почты следующим образом: pavel@fourthcoffee.com.

- **Сведения для оплаты**

    Вы можете заплатить за подписку с помощью кредитной карты. Если стоимость достигнет определенного уровня, вы также сможете оплачивать ее по счету.

    > [!IMPORTANT]
    >  При регистрации убедитесь, что выбран вариант лучший платеж для вашей организации. Чтобы изменить способ платежа, обратитесь в службу поддержки по вопросам, связанным с выставлением счетов.
