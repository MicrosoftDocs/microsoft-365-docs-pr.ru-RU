# <a name="about-the-microsoft-365-admin-mobile-app"></a>О мобильном приложении Microsoft 365 администратора

Вы администратор, который обычно в го? Даже если это не так, могут возникнуть моменты, когда вам потребуется управлять Microsoft 365 с телефона или планшета. Ознакомьтесь с бесплатным [приложением администрирования Microsoft 365,](https://go.microsoft.com/fwlink/?LinkID=627216)идеальным компаньоном для веб-центра администрирования Microsoft 365 администратора. Вы можете скачать приложение из [Магазина приложений Apple](https://apps.apple.com/app/apple-store/id761397963?pt=80423&ct=docsaboutadminapp&mt=8)и магазина Google [Play.](https://play.google.com/store/apps/details?id=com.ms.office365admin&referrer=utm_source%3Ddocsaboutadminapp%26utm_campaign%25docsaboutadminapp)

:::image type="content" source="../../media/DocsMicrosoftFinalImage.PNG" alt-text="Снимок экрана: главная страница мобильного приложения администратора, отображающий поиск, центр сообщений, здоровье и быстрые ссылки":::

Приложение администрирования имеет множество возможностей, которые позволяют управлять Microsoft 365 с мобильного или планшетного устройства, когда вы не можете добраться до компьютера. Вот список некоторых задач, которые можно выполнять из приложения:

- **Управление пользователями и устройствами** Добавьте или отредактировать пользователя, сбросите пароль пользователя, назначьте роль, заблокируете пользователя, удалите пользователя, управляйте псевдонимом, назначьте лицензии, стереть данные устройства и т. д.
- **Управление группами** Добавьте группу, добавьте или удалите пользователей из групп.
- **Управление лицензией и выставление счета** Просмотр списка приобретенных и назначенных лицензий, назначение лицензий пользователям, покупка или удаление лицензий, просмотр и загрузка счетов-фактур.
- **Поддержка** Создайте новый запрос на службу и отслеживайте все обновления, связанные с запросами на обслуживание, пока вы находитесь в пути.
- **Центр сообщений** Оставайтесь на вершине всех предстоящих изменений, планового обслуживания или других важных сообщений, связанных с Microsoft 365
- **Здоровье службы** Отслеживайте состояние всех служб, просматривая текущее состояние службы и сведения о сбоях и отключениях службы.
- **Уведомления** Оставайтесь на вершине всех важных сведений и обновлений, связанных с сообщениями центра сообщений, состоянием службы и выставлением счета с помощью push-уведомлений. Вы даже можете настроить то, о чем нужно уведомить.

Если вы администратор и несете ответственность за несколько Microsoft 365 организации, вы можете войти в несколько организаций и быстро переключаться между ними. Приложение поддерживает темную тему и доступно на 39 языках.
  
> [!IMPORTANT]
> Если у вас возникли проблемы с использованием мобильного приложения Admin [](mailto:feedback365@microsoft.com) на iOS или Android, feedback365@microsoft.com пишите нам.

## <a name="frequently-asked-questions"></a>Вопросы и ответы

Ниже приведены ответы на часто задаваемые вопросы.
  
### <a name="what-do-i-need-to-do-to-be-able-to-use-the-app"></a>Что нужно сделать, чтобы начать использовать приложение?

Чтобы использовать приложение, необходимо иметь разрешения администратора и действительная подписка на одну из следующих Microsoft 365 подписки:

- Приложения Microsoft 365 для предприятий
- Приложения Microsoft 365 для бизнеса
- Microsoft 365 бизнес премиум
- Microsoft 365 бизнес стандарт
- Microsoft 365 бизнес базовый
- Microsoft 365 E3/E5
- Office 365: E1, E3/E5
- Exchange Online
- OneDrive бизнес-плана
  
### <a name="can-i-use-the-app-with-my-microsoft-365-family-subscription"></a>Могу ли я использовать приложение с Microsoft 365 для семьи подпиской?

Нет, приложение не работает с Microsoft 365 для семьи или Microsoft 365 персональный подписками.

### <a name="will-the-app-work-if-my-organization-has-directory-synchronization-enabled"></a>Будет ли работать приложение, если в моей организации включена синхронизация службы каталогов?

Да, но с уменьшенной функциональностью. Вы сможете войти и просмотреть сведения о службе, но большинство функций управления пользователями будут только для чтения. Вы не сможете добавлять, изменять или удалять пользователей. Однако вы сможете назначать лицензии пользователям в организации и получать уведомления.
  
### <a name="what-languages-are-supported-by-the-app"></a>Какие языки поддерживаются в приложении?

Приложение поддерживает все 39 языков, Microsoft 365 центр администрирования. Чтобы изменить язык, из левого меню навигации в центре администрирования **перейдите** в Параметры language или выберите значок Language в разделе Быстрые ссылки домашней страницы приложения  >   администратора.  
  
### <a name="how-can-i-share-the-service-incidents-and-messages-with-the-rest-of-my-organization"></a>Как рассказать об инцидентах обслуживания и поделиться другими сообщениями с сотрудниками моей организации?

Если выбран определенный инцидент службы или сообщение, параметр share будет в правом верхнем углу.
  
### <a name="can-i-use-this-app-with-multiple-accounts-or-tenants"></a>Могу ли я использовать это приложение с несколькими учетными записями или клиентами?

Да, можно настроить несколько учетных записей или организаций.

### <a name="im-unable-to-login-or-my-app-is-acting-funny-what-can-i-do-to-troubleshoot-or-fix-the-issue"></a>Я не могу войти в систему или мое приложение действует смешно. Что можно сделать для устранения неполадок или устранения проблемы?

Вы можете попробовать выполнить некоторые стандартные действия.

1. Закройте и снова откройте приложение.
1. Удалите и переустановите приложение. Убедитесь, что вы находитесь в последней версии приложения.
1. Если на вашем устройстве Microsoft Authenticator или приложение портала компании, попробуйте переустановить его или обновить до последней версии.
1. Если это не работает, вы можете отправить нам по электронной почте feedback365@microsoft.com, чтобы мы знали.

### <a name="how-do-i-manage-notifications-in-the-app"></a>Как управлять уведомлениями в приложении?

Из левого меню навигации перейдите **Параметры**  >  **уведомления**. Здесь можно управлять службами, центрами сообщений и уведомлениями о выставлении счета.

### <a name="what-do-i-do-if-my-question-isnt-answered"></a>Что делать, если ответ на мой вопрос отсутствует?

Электронная [feedback365@microsoft.com](mailto:feedback365@microsoft.com) сообщить о проблеме с приложением. Или вы можете дать обратную связь в нижней части этой статьи.
