# <a name="about-the-microsoft-365-admin-center"></a>Сведения о Центре администрирования Microsoft 365

**Если вы приобрели Microsoft 365 бизнес-плана** и у вас есть разрешения администратора, у вас есть доступ к центру администрирования. Эта статья для вас!

**Если вы пользователь или имеете Microsoft 365 для семьи, у вас нет центра администрирования.** Чтобы настроить Microsoft 365, переустановите или установите Microsoft 365 или [Office 2019](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658)на ПК или Mac .

С помощью Центра администрирования можно настроить организацию в облаке, управлять пользователями, управлять подписками и т. д. Из этой статьи вы узнаете, как войти в Центр администрирования, а также какие возможности и настройки доступны.

Посмотрите короткий видеоролик о Центре администрирования. <br><br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfvDL]

Если этот видео помогло вам, ознакомьтесь с [полным учебным курсом для малых предприятий и новых пользователей Microsoft 365](../../business-video/index.yml).

## <a name="how-to-get-to-the-admin-center"></a>Переход в Центр администрирования

1. Войдите на сайт <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">admin.microsoft.com</a> с учетной записью администратора.

2. В левом верхнем углу нажмите значок средства запуска приложений и выберите элемент **Администратор**.

    Плитка **администратора** отображается только для людей, у которых Microsoft 365 [разрешения администратора.](../add-users/about-admin-roles.md) Если вы не видите эту плитку, значит, у вас нет разрешений на доступ к центру администрирования в организации.

Ниже описаны функции и параметры, расположенные в левой панели навигации Центра администрирования. Узнать больше о задачах администрирования можно в разделе [справки для администраторов](../../business-video/admin-center-overview.md).

Добавляйте и обновляйте доменные имена, такие как contoso.com. Настройте профиль организации и параметры выпуска. |**Отчеты** <br/> |Посмотрите сразу, как ваша организация использует Microsoft 365 подробные отчеты об использовании электронной почты, Office активаций и других. Научитесь пользоваться новыми [отчетами об активности](../activity-reports/activity-reports.md).  <br/> |
|**Работоспособность** <br/> |Просмотр состояния здоровья с первого взгляда. Вы также можете узнать больше сведений и историю здоровья. Дополнительные [сведения см. в](../../enterprise/view-service-health.md) дополнительных сведениях о проверке состояния [службы](/windows/deployment/update/check-release-health) и Windows состояния выпуска.  <br/><br/>С помощью Центра сообщений можно отслеживать предстоящие изменения функций и служб. В Центре администрирования Skype для бизнеса можно настраивать уведомления о мгновенных сообщениях, конференции с телефонным подключением и сведения о присутствии в сети.  <br/> Узнайте больше о Центрах администрирования [Exchange](/exchange/exchange-admin-center) и [SharePoint](/sharepoint/sharepoint-online).<br/> **Примечание.** Набор доступных центров администрирования зависит от вашего плана и региона.           |
