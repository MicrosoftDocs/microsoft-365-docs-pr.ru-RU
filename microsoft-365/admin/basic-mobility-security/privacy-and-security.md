# <a name="privacy-and-security-in-basic-mobility-and-security"></a>Конфиденциальность и безопасность в базовой мобильности и безопасности

Basic Mobility and Security — это облачная служба с Microsoft Intune, которая помогает управлять и обеспечивать безопасность мобильных устройств в организации. После активации Basic Mobility и Security можно создать политики управления мобильными устройствами. Эти политики можно развернуть на мобильных устройствах, зарегистрированных лицензированными пользователями Microsoft 365 организации.

Microsoft Intune отправляет сведения Microsoft 365 о состоянии соответствия требованиям для каждого управляемого устройства, а затем можно создавать отчеты, в которых покажут, соответствуют ли управляемые устройства в организации установленным политикам. Дополнительные данные о приверженности Корпорации Майкрософт конфиденциальности и безопасности см. в центре [доверия Майкрософт.](https://www.microsoft.com/trust-center)
