# <a name="privacy-and-security-in-basic-mobility-and-security"></a>Конфиденциальность и безопасность в базовой мобильности и безопасности

Basic Mobility and Security — это облачная служба, основанная на Microsoft Intune, которая помогает управлять мобильными устройствами в организации и обеспечивать их безопасность. После активации Basic Mobility and Security можно создать политики управления мобильными устройствами. Затем эти политики можно развернуть на мобильных устройствах, зарегистрированных лицензированными пользователями Microsoft 365 в вашей организации.

Microsoft Intune отправляет в Microsoft 365 сведения о состоянии соответствия требованиям для каждого управляемого устройства, а затем вы можете создавать отчеты, которые показывают, соответствуют ли управляемые устройства в организации требованиям на основе установленных политик. Чтобы узнать больше о обязательствах корпорации Майкрософт в отношении конфиденциальности и безопасности, см. [центр управления безопасностью Майкрософт.](https://www.microsoft.com/trust-center)
