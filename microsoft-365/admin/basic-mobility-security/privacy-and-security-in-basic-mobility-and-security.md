# <a name="privacy-and-security-in-basic-mobility-and-security"></a>Конфиденциальность и безопасность в базовом мобильном и системном обеспечении безопасности

Basic Mobility and Security — это облачная служба, основанная на Microsoft Intune, которая помогает управлять мобильными устройствами, используемыми в Организации, и защищать их. После включения базовых служб мобильной связи и безопасности можно создавать политики управления мобильными устройствами. Эти политики затем можно развернуть на мобильных устройствах, зарегистрированных в Организации с лицензированными пользователями Microsoft 365.

Microsoft Intune отправляет сведения в Microsoft 365 о состоянии соответствия требованиям каждого управляемого устройства, а затем вы можете создавать отчеты, показывающие, соответствуют ли управляемые устройства в Организации требованиям, на основе установленных политик. Чтобы узнать больше о приверженности корпорации Майкрософт в отношении конфиденциальности и безопасности, ознакомьтесь со статьей [центр управления безопасностью Майкрософт](https://www.microsoft.com/trust-center).
