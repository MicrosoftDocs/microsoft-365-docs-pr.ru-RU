# <a name="what-is-a-domain"></a>Что такое домен?

 Если вы не нашли то, что вы ищете, см. раздел **[Вопросы и ответы по доменам](../setup/domains-faq.yml)**. 
  
Домен  это уникальное имя, которое стоит после символа **@** в адресах электронной почты и после **www.** в веб-адресах. Обычно он принимает название организации и стандартный суффикс Интернета, например *бизнес-бизнес. <span> com* или *stateuniversity. <span> edu*. 
  
Использование пользовательского домена, *например rob@contoso. <span> Com* с Microsoft 365 может помочь создать доверии и распознавание для вашей торговой марки. 
  
## <a name="choose-the-experience-thats-best-for-you"></a>Выберите оптимальный вариант.

 **Купить новый домен:**
  
Использование Microsoft 365 для покупки нового домена — мы настроим все. [Подробнее.](buy-a-domain-name.md)
  
 **Используйте домен, который у вас уже есть:**
  
Если у вас уже есть домен,  *например you@yourcompany. <span> вы*  можете добавить его и сразу начать использовать OneDrive и приложения Майкрософт. Настройка почты и Skype для бизнеса займет у вас буквально несколько минут. 
  
> [!TIP]
> Не знаете точно, где зарегистрирован ваш домен? [Получите помощь в определении своего регистратора доменных имен](find-your-domain-registrar.md).
  
 **Используйте домен по умолчанию, пока вам это удобно:**
  
Вы можете использовать домен по умолчанию, например  *yourcompany.onmicrosoft. <span> com*  для входа в систему и добавления пользовательского домена позже. Изменить или переименовать доменное имя по умолчанию нельзя, поэтому перед переходом подумайте, подойдет ли оно вам.
