# <a name="explaining-microsoft-365-groups-to-your-users"></a>Разъяснить Microsoft 365 группы пользователям

Microsoft 365 Группы позволяют настроить коллекцию ресурсов для общего доступа, включая общий почтовый ящик и календарь, веб SharePoint сайт с OneNote ноутбуком и microsoft Planner среди других. Microsoft Teams также можно включить при создании группы или добавить ее позже. Ресурсы групп разрешений управляются через группу.

Группы можно создать, создав любой из общих ресурсов. Создание группы в Outlook дает тот же результат, что и создание сайта SharePoint группы или плана в Planner. Если ваши пользователи не Microsoft 365 групп, они могут этого не понимать. Это может привести к путанице для пользователей и возможности создания дублирующихся ресурсов. (Например, кто-то может создать SharePoint для совместной работы с документами, а затем создать отдельный экземпляр Planner, не понимая, что планировщик уже доступен в составе группы.)

Так как группы могут создаваться несколькими способами, рекомендуется обуять пользователей использовать метод, который лучше всего подходит вашей организации:

- Если ваша организация делает большую часть общения с помощью электронной почты, поручите пользователям создавать группы в Outlook.
- Если ваша организация активно использует SharePoint или мигрирует из локального SharePoint, поручите пользователям создавать сайты SharePoint для совместной работы.
- Если ваша организация развернула Teams, научите пользователей создавать команду, когда им требуется пространство для совместной работы.

[![image desc ](../../media/03.png)](../../media/03.png#lightbox)

Если вы научите пользователей всегда использовать метод создания групп, который наиболее соответствует их способу работы, когда им требуется пространство для совместной работы с другими пользователями, вы можете избежать путаницы и дублирования ресурсов. По мере того как пользователи становятся более опытными, они будут лучше понимать коллекцию служб, которые приходят с группой, и что различные методы создания приводят к одному результату.

В качестве отправной точки для обучающих презентаций для пользователей можно использовать шаблон Microsoft 365 groups [for Business User - PowerPoint.](https://www.microsoft.com/download/details.aspx?id=102396)
