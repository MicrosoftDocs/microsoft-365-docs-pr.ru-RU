# <a name="office-365-is-now-microsoft-365"></a>Теперь Office 365 называется Microsoft 365

Мы изменили некоторые названия подписок на Office 365 и Microsoft 365. Вам ничего не нужно делать. Службы, приложения и функции остаются такими же, как и цена подписки. Ваше имя подписки будет автоматически обновляться в Центр администрирования Microsoft 365, а выписки о выставлении счета будут обновляться 21 апреля 2020 г. или после нее.

- **Office 365 бизнес базовый** теперь **Microsoft 365 бизнес базовый**
- **Office 365 бизнес премиум** теперь **Microsoft 365 бизнес стандарт**
- **Microsoft 365 бизнес** теперь **Microsoft 365 бизнес премиум**
- **Office 365 бизнес** теперь **Приложения Microsoft 365 для бизнеса**
- **Office 365 профессиональный плюс** теперь **Приложения Microsoft 365 для предприятий**
