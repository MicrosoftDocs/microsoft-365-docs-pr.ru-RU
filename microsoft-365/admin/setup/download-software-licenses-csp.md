# <a name="download-perpetual-software-and-product-license-keys"></a>Скачайте бессрочные ключи лицензий на программное обеспечение и продукт

В этой статье рассказывается о загрузке ключей лицензий на программное обеспечение и продукты для бессрочных программ, купленных поставщик облачных решений (CSP).

## <a name="before-you-begin"></a>Прежде чем начать

Вы должны быть глобальным администратором для действий в этой статье. Дополнительные сведения см. в статье [О ролях администраторов](../add-users/about-admin-roles.md).

## <a name="download-software-and-product-license-keys"></a>Загрузка ключей лицензии программного обеспечения и продуктов

1. В Центре администрирования Microsoft 365 перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке Продукты** в разделе **Программное** обеспечение выберите программное обеспечение, которое необходимо скачать.
3. На странице подробные сведения о подписке **в** разделе & загрузок выберите версию **продукта,** язык и тип файла **&** процессора, а затем выберите **скачать**.
4. Чтобы скачать ключ, выберите **клавишу Скопируйте в буфер обмена.**
5. В правой области выберите **скопируйте,** а затем закроем его.
6. Введите ключ в файл в безопасном месте и введите его по инструкции во время установки программного обеспечения. Ключ необходим для активации скачаного программного обеспечения.
