# <a name="download-perpetual-software-and-product-license-keys"></a>Скачивание бессрочных ключей лицензий на программное обеспечение и продукты

В этой статье объясняется, как скачать ключи лицензий программного обеспечения и продуктов для бессрочных программ, приобретаемых в рамках программы поставщиков облачных решений (CSP).

## <a name="before-you-begin"></a>Прежде чем начать

Для этого необходимо быть глобальным администратором. Дополнительные сведения см. в статье [О ролях администраторов](../add-users/about-admin-roles.md).

## <a name="download-software-and-product-license-keys"></a>Скачивание ключей лицензий на программное обеспечение и продукты

1. В Центре администрирования Microsoft 365 перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" **в** разделе "Программное обеспечение" выберите программное обеспечение, которое необходимо скачать.
3. На странице сведений о  подписке в разделе "& загрузки" выберите тип файла "Версия продукта", "Язык" и **&** ЦП, а затем выберите "Скачать". 
4. Чтобы скачать ключ, выберите **"Скопировать ключ" в буфер обмена.**
5. В правой области выберите **"Копировать"** и закроем ее.
6. Введите ключ в файл в надежном месте и введите его в соответствии с инструкциями во время установки программного обеспечения. Ключ необходим для активации загруженного программного обеспечения.
