# <a name="set-up-file-storage-and-sharing-in-microsoft-365"></a>Настройка хранения файлов и совместного использования в Microsoft 365

Один из лучших способов настройки хранилища файлов и совместного использования для вашего бизнеса заключается в использовании OneDrive и командный сайт вместе. Этот вариант идеально подходит для небольших компаний с небольшим количеством сотрудников.

Посмотрите короткое видео о хранении файлов в Microsoft 365 облаке.<br><br>

## <a name="microsoft-365-document-storage-and-management"></a>Microsoft 365 хранения и управления документами

- OneDrive предназначен для индивидуального использования, с случайным обменом файлами.

- Сайт группы предназначен для регулярного обмена файлами и совместной работы с ней. Сайт команды идеально подходит для хранения файлов, которые имеют общую собственность, где несколько человек владеют файлами и могут сотрудничать с ними. Создайте команду Майкрософт для добавления командный сайт. [Узнайте больше в работе по созданию команды в Teams.](https://support.microsoft.com/office/174adf5f-846b-4780-b765-de1a0a737e2b)

Как OneDrive, так и сайты команды предоставляют доступ в любую точку для вас и ваших сотрудников.

## <a name="start-using-onedrive-and-your-team-site"></a>Начните использовать OneDrive и сайт вашей команды

### <a name="team-members-can-store-their-own-files-in-onedrive"></a>Члены команды могут хранить свои собственные файлы в OneDrive

Каждый человек в вашем бизнесе, у которого Microsoft 365 лицензия на покупку (и SharePoint онлайн выбран), получает OneDrive облачное хранилище. Они могут хранить файлы, связанные с бизнесом, здесь для доступа с любого устройства, и они доступны только для этого пользователя. Например, пользователи могут переносить в хранилище проекты коммерческих предложений, свои заметки к собраниям или сценарии собственных презентаций.
  
Сотрудники также могут обмениваться OneDrive файлами и папками. Если сотрудник находится в стороне или покидает компанию, другие могут получить доступ к общим файлам, хранящимся в OneDrive (OneDrive совместное использование общих папок).
  
Вот как каждый человек в вашей команде может настроить OneDrive обмениваться файлами.

1. Перейдите на <a href="https://portal.office.com/ " target="_blank">Microsoft 365 Portal и</a>войдите в свой имя пользователя и пароль.

2. Из пусковой установки Приложения **выберите OneDrive.**

3. В OneDrive члены группы могут хранить свои собственные файлы, связанные с бизнесом. Вы можете поделиться либо отдельными файлами, либо целой папкой. Выберите файл или папку, нажмите правой кнопкой мыши, а затем **выберите Share**.

    ![Совместное использование папки](../../media/e8df9df3-aea5-404d-a320-92d7826c260c.png)
  
4. На странице **Отправить ссылку,** оставьте выбор по умолчанию **Любой человек со ссылкой может просматривать и редактировать**.

    Ввемийте имена или адреса электронной почты членов команды, которые хотите иметь доступ к папке, и добавьте дополнительное сообщение.

    Если вы хотите получить свою собственную копию письма, которое будет отправлено, добавьте свой адрес электронной почты в список.

    ![Поделитесь ссылкой диалоговое окно, показывающее ввод и выбор имени.](../../media/877e6587-db9d-4903-a87b-11e570eee926.png)
  
5. Когда вы закончили ввода, кто вы хотите поделиться с, выберите **Отправить**. Электронное письмо будет сразу же выслано всем приглашенным пользователям.

    ![Поделитесь ссылкой, показывая список имен](../../media/e85625ea-7655-43f3-8623-72db68d0ea39.png)
  
6. Вот как выглядит электронная почта. 

    ![Электронная почта со ссылкой на OneDrive папку](../../media/750c92e1-f14f-404c-a6a3-2095e26c680c.png)
  
### <a name="upload-files-to-a-team-site-for-online-collaboration"></a>Upload файлы на сайт команды для онлайн-сотрудничества

Сайты команд приходят с местом для хранения файлов, называемых библиотекой документов.  
  
Вот шаги для добавления файлов:
  
1. На главной странице сайта вашей команды выберите **Документы** из левого навигационного меню. В результате вы перейдете к своей **библиотеке документов**.
  
2. В то время как вы все еще в Microsoft 365, откройте Windows **File Explorer из** панели задач или другого местоположения. Перейдите к файлам, которые нужно отправить на сайт группы.

3. Выберите файлы, которые вы хотите загрузить на сайт вашей команды, а затем перетащите их в **библиотеку документов.**
  
4. После отправки файлы будут храниться как на сайте группы, так и на вашем компьютере.
  
5. Вы можете удалить файлы с вашего компьютера. На следующем этапе, [Синхронизация онлайн-файлов с пк или Mac](#sync-online-files-with-your-pc-or-mac), вы собираетесь создать новое место для этих файлов на вашем компьютере.

    Если у вас есть многочисленные файлы или большие файлы для загрузки на ваш сайт команды, прочитайте эти [советы по загрузке больших или многих файлов в библиотеку.](https://support.microsoft.com/office/da549fb1-1fcb-4167-87d0-4693e93cb7a0#__toc384119242)

    Сведения о том, как приобрести дополнительное дисковое пространство в хранилище, см. в статье [Изменение объема хранилища для своей подписки](../../commerce/add-storage-space.md).

### <a name="sync-online-files-with-your-pc-or-mac"></a>Синхронизация онлайн-файлов с ПК или Mac

Теперь, когда ваши файлы хранятся на сайте группы, вы можете настроить их синхронизацию с ПК или Mac. Таким образом, вы можете работать над файлами с вашего компьютера или Mac вместо того, чтобы работать Microsoft Edge, Chrome или другом браузере. Наличие синхронизированных версий файлов на локальном компьютере также крайне полезно в тех случаях, когда вам нужно просмотреть файл, а подключение к Интернету отсутствует.
  
После настройки файлов для синхронизации с компьютером при подключении к Интернету они будут синхронизированы автоматически.
  
Вот как настроить синхронизацию файлов на сайте группы с компьютером.
  
1. На главной странице сайта вашей команды выберите **Документы** из левого навигационного меню. Это доставит вас в библиотеку Документов.

    > [!TIP]
    > При синхронизации файлов на сайте вашей команды вы синхронизируете каждую библиотеку файлов на сайте, а не на всем сайте.
  
2. Выберите **Sync** для синхронизации всех файлов. Или просмотрите конкретную папку, которую вы хотите синхронизировать.

3. Если предложено переключить приложения, выберите **Да**. OneDrive процесс делает синхронизацию.
  
4. Если вы получаете **набор данных, OneDrive,** воймите с вашей работой или школьной учетной записью.

    ![OneDrive настройка экрана](../../media/82cbb1ac-2ac5-42bd-82de-ba710bf46145.png)
  
5. Если вы еще не синхронизировали OneDrive, вы можете **увидеть, что это ваш OneDrive экрана папки.** Проверьте путь под **папкой OneDrive вашей семьи здесь**. Выберите **место изменения,** если вы хотите использовать другой путь, а затем выберите **Следующий**.

    ![Изменение локальной папки на этом экране](../../media/6395485a-e729-4a9a-8e7d-b35e662435da.png)
  
6. Файлы на сайтах вашей команды будут отображаться в левом стекле File Explorer под названием вашей организации. Файлы в OneDrive появятся под "OneDrive - \<Name of Organization\> "

    ![Посмотрите, что было sync'd в локальной папке](../../media/93e2ca9f-4b5b-4930-a94d-ebc5b95aca84.png)
  
7. Проверьте синхронизацию, открыв файл в папке команды на компьютере. Внесите нужные изменения и выберите команду **Сохранить**.

## <a name="best-practices-for-file-storage-and-sharing"></a>Рекомендации по хранению файлов и предоставлению к ним общего доступа

Вот несколько советов для получения наибольшей от OneDrive вашей SharePoint команды.
  
### <a name="file-storage-and-collaboration-recommendations-for-other-types-of-small-businesses"></a>Рекомендации по использованию хранилища файлов и организации совместной работы для других типов малых компаний

- **Индивидуальные владельцы**: OneDrive для хранения собственных файлов и обмена ими с клиентами на индивидуальной основе.

- **Совладельцы : Оба владельца** используют их OneDrive обмениваться файлами туда и обратно.

- **Предприятия с внешними клиентами или партнерами, которым нужен доступ к файлам:** Создайте новый сайт команды для хранения и обмена документами, предназначенными для конкретного клиента. Настройка сайта, чтобы обеспечить доступ только к этому клиенту. Тогда вам не нужно беспокоиться, что один клиент случайно получит доступ к информации, предназначенной для другого клиента.

### <a name="keep-private-files-private"></a>Защита конфиденциальных файлов

При храним файле OneDrive он доступен только вам, если вы не поделитесь им с другими. При общих файлах можно создать ссылку, которую можно переадресать, или поделиться ею только с конкретными людьми. Кроме того, вы можете создавать в службе OneDrive папки для разных целей, например для общедоступных и личных материалов или отдельных проектов. Общий доступ к каждой из них можно предоставить разным пользователям и группам либо не предоставлять никому.
  
Для получения дополнительной информации об обмене, [а также увидеть Файлы обмена и папки с Microsoft 365](https://support.microsoft.com/office/72f26d6c-bf9e-432c-8b96-e3c2437f5b65).
  
### <a name="track-how-much-space-you-have-left"></a>Отслеживание свободного места

Чтобы узнать, сколько места для хранения у вас осталось OneDrive, [см OneDrive для бизнеса.](https://support.microsoft.com/office/31519161-059C-4764-B6F8-F5CD29F7FE68)
  
### <a name="what-files-can-be-stored-in-onedrive-and-a-team-site"></a>Какие файлы могут храниться OneDrive и на сайте команды?

В то время как вы можете загрузить почти все типы файлов, некоторые имена файлов и символы в именах файлов не допускаются. Для получения дополнительной информации [с OneDrive для бизнеса м.](https://support.microsoft.com/office/64883A5D-228E-48F5-B3D2-EB39E07630FA)
  
### <a name="enable-or-disable-third-party-storage-services"></a>Включить или отключить сторонние службы хранения данных

Вы можете включить сторонняя хранение для ваших пользователей в Microsoft 365, чтобы они могли хранить и обмениваться документами, используя такие службы, как Dropbox в дополнение к OneDrive и командных сайтах. Это отличная возможность обеспечить пользователям доступ к тем службам, которые они уже используют или предпочитают использовать для бизнес-проектов. Если вы не хотите, чтобы люди в вашей организации Office для открытия файлов в сторонняя служба, следуйте этим шагам, чтобы отключить его.
  
> [!IMPORTANT]
> Сторонняя хранилище включено по умолчанию, поэтому вам нужно выполнить эти шаги сразу, если вы не хотите, чтобы он был доступен для ваших пользователей.
  
1. Воехать в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">центр администратора.</a>

2. Перейдите на **страницу Параметры** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**Org.**</a>

3. На **вкладке Услуги** выберите **Office в Интернете.**

4. Выберите или отберите чековую коробку, чтобы включить или выключить хранилище сторонних участников, а затем выберите **изменения Save.**
