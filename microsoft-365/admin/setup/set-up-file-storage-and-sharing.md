# <a name="set-up-file-storage-and-sharing-in-microsoft-365"></a>Настройка хранилища файлов и совместного использования в Microsoft 365

Один из лучших способов настроить хранилище файлов и общий доступ для бизнеса — использовать OneDrive и сайт группы вместе. Этот вариант идеально подходит для небольших компаний с небольшим количеством сотрудников.

Посмотрите короткое видео о хранении файлов в облаке Microsoft 365.<br><br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX] 

Если это видео помогло вам, ознакомьтесь с [полным учебным курсом для малых предприятий и новых пользователей Microsoft 365](https://support.microsoft.com/office/6ab4bbcd-79cf-4000-a0bd-d42ce4d12816).

## <a name="microsoft-365-document-storage-and-management"></a>"Хранилище документов Microsoft 365 и управление ими"

- OneDrive предназначен для индивидуального использования с периодическим совместным доступом к файлам.

- Сайт группы предназначен для регулярного обмена файлами и совместной работы с файлами. Сайт группы идеально подходит для хранения файлов с общими владениями, в которых несколько человек владеют файлами и могут совместно работать с ними. Создайте microsoft Team, чтобы добавить сайт группы. [Дополнительные возможности можно узнать в этой теме: создание команды в Teams.](https://support.microsoft.com/office/174adf5f-846b-4780-b765-de1a0a737e2b)

Как OneDrive, так и сайты группы предоставляют доступ в любом месте для вас и ваших сотрудников.

## <a name="start-using-onedrive-and-your-team-site"></a>Начало работы с OneDrive и сайтом группы

### <a name="team-members-can-store-their-own-files-in-onedrive"></a>Участники группы могут хранить собственные файлы в OneDrive

Каждый человек в вашей компании, которому назначена лицензия Microsoft 365 (и выбран SharePoint Online), получает облачное хранилище OneDrive. Они могут хранить здесь бизнес-файлы для доступа с любого устройства и доступны только этому пользователю. Например, пользователи могут переносить в хранилище проекты коммерческих предложений, свои заметки к собраниям или сценарии собственных презентаций.
  
Сотрудники также могут совместно использовать файлы и папки OneDrive. Если сотрудник находится в компании или покидает ее, другие могут получить доступ к общим файлам, хранимым в OneDrive (общий доступ к папок группы OneDrive).
  
Вот как каждый человек в вашей команде может настроить OneDrive и поделиться файлами.

1. Перейдите на <a href="https://portal.office.com/ " target="_blank">портал Microsoft 365 и</a>войдите с помощью имени пользователя и пароля.

2. В оке запуска приложений выберите **OneDrive.**

3. В OneDrive участники группы могут хранить собственные бизнес-файлы. Вы можете совместно использовать отдельные файлы или всю папку. Выберите файл или папку, щелкните правой кнопкой мыши и выберите **"Поделиться".**

    ![Общий доступ к папке](../../media/e8df9df3-aea5-404d-a320-92d7826c260c.png)
  
4. На странице **"Отправить ссылку"** оставьте выбор по умолчанию, который любой, у кого есть ссылка, **может просматривать и редактировать.**

    Введите имена или адреса электронной почты участников группы, которым вы хотите получить доступ к папке, и добавьте необязательное сообщение.

    Если вы хотите отправить собственную копию сообщения электронной почты, добавьте свой адрес электронной почты в список.

    ![Диалоговое окно "Поделиться ссылкой" с отображением ввода текста и выбора имени.](../../media/877e6587-db9d-4903-a87b-11e570eee926.png)
  
5. Когда вы введите, с кем вы хотите поделиться, выберите **"Отправить".** Электронное письмо будет сразу же выслано всем приглашенным пользователям.

    ![Поделиться ссылкой со списком имен](../../media/e85625ea-7655-43f3-8623-72db68d0ea39.png)
  
6. Вот как выглядит сообщение электронной почты. 

    ![Электронная почта со ссылкой для обмена папкой OneDrive](../../media/750c92e1-f14f-404c-a6a3-2095e26c680c.png)
  
### <a name="upload-files-to-a-team-site-for-online-collaboration"></a>Отправка файлов на сайт группы для совместной работы в Интернете

На сайтах группы есть место для хранения файлов, называемых библиотекой документов.  
  
Вот как добавить файлы:
  
1. На домашней странице сайта группы выберите **"Документы"** в левом меню навигации. В результате вы перейдете к своей **библиотеке документов**.
  
2. Хотя вы еще вписались в Microsoft 365, откройте проводник **Windows** на панели задач или в другом месте. Перейдите к файлам, которые нужно отправить на сайт группы.

3. Выберите файлы, которые нужно отправить на сайт группы, а затем перетащите их в **библиотеку документов.**
  
4. После отправки файлы будут храниться как на сайте группы, так и на вашем компьютере.
  
5. Файлы можно удалить с компьютера. На следующем этапе синхронизация сетевых файлов с компьютером [или Mac](#sync-online-files-with-your-pc-or-mac)вы создайте новое расположение для этих файлов на компьютере.

    Если у вас есть множество файлов или больших файлов для отправки на сайт группы, ознакомьтесь с этими советами по отправке больших или многих файлов [в библиотеку.](https://support.microsoft.com/office/da549fb1-1fcb-4167-87d0-4693e93cb7a0#__toc384119242)

    Сведения о том, как приобрести дополнительное дисковое пространство в хранилище, см. в статье [Изменение объема хранилища для своей подписки](../../commerce/add-storage-space.md).

### <a name="sync-online-files-with-your-pc-or-mac"></a>Синхронизация файлов из Интернета с компьютером или Mac

Теперь, когда ваши файлы хранятся на сайте группы, вы можете настроить их синхронизацию с ПК или Mac. Таким образом, вы сможете работать с файлами на компьютере с Mac, а не в Microsoft Edge, Chrome или другом браузере. Наличие синхронизированных версий файлов на локальном компьютере также крайне полезно в тех случаях, когда вам нужно просмотреть файл, а подключение к Интернету отсутствует.
  
После того как вы настроите синхронизацию файлов с компьютером, при под подключением к Интернету они будут синхронизированы автоматически.
  
Вот как настроить синхронизацию файлов на сайте группы с компьютером.
  
1. На домашней странице сайта группы выберите **"Документы"** в левом меню навигации. После этого вы будете перенабираться в библиотеку документов.

    > [!TIP]
    > При синхронизации файлов на сайте группы синхронизируется каждая библиотека файлов на сайте, а не весь сайт.
  
2. Выберите **"Синхронизация",** чтобы синхронизировать все файлы. Или перейдите к определенной папке, которая вы хотите синхронизировать.

3. Если вам будет предложено переключить приложения, выберите **"Да".** Синхронизацию можно синхронизировать в OneDrive.
  
4. Если после этого вы получите запрос на настройка **OneDrive,** во sign in with your work or school account.

    ![Экран настройки OneDrive](../../media/82cbb1ac-2ac5-42bd-82de-ba710bf46145.png)
  
5. Если вы еще не синхронизировали OneDrive, вы можете увидеть экран **папки "Это ваш OneDrive".** Проверьте путь к **папке OneDrive здесь.** Выберите **"Изменить расположение",** если нужно использовать другой путь, а затем выберите **"Далее".**

    ![Изменение локальной папки на этом экране](../../media/6395485a-e729-4a9a-8e7d-b35e662435da.png)
  
6. Файлы на сайтах группы появятся в левой области проводника под именем вашей организации. Файлы в OneDrive появятся в папке "OneDrive - \<Name of Organization\> "

    ![Посмотрите, что синхронизировали в локальной папке](../../media/93e2ca9f-4b5b-4930-a94d-ebc5b95aca84.png)
  
7. Проверьте синхронизацию, открыв файл в папке команды на компьютере. Внесите нужные изменения и выберите команду **Сохранить**.

## <a name="best-practices-for-file-storage-and-sharing"></a>Рекомендации по хранению файлов и предоставлению к ним общего доступа

Вот несколько советов по большей части из OneDrive или сайта группы SharePoint.
  
### <a name="file-storage-and-collaboration-recommendations-for-other-types-of-small-businesses"></a>Рекомендации по использованию хранилища файлов и организации совместной работы для других типов малых компаний

- **Индивидуальные владельцы:** используйте OneDrive для хранения собственных файлов и обмена ими с клиентами в индивидуальном порядке.

- **Совместное владение:** оба владельца используют OneDrive и совместно используют файлы.

- **Компании с внешними клиентами или партнерами,** которым нужен доступ к файлам: создайте новый сайт группы для хранения документов, предназначенных для определенного клиента, и обмена ими. Настройка сайта, чтобы разрешить доступ только этому клиенту. После этого вам не нужно беспокоиться, что один клиент случайно получит доступ к информации, предназначенной для другого клиента.

### <a name="keep-private-files-private"></a>Защита конфиденциальных файлов

Когда вы сохраняете файл в OneDrive, он доступен только вам, если вы не поделитесь им с другими. Когда вы делитесь файлами, вы можете создать ссылку, которую можно переадлить, или поделиться только определенными людьми. Кроме того, вы можете создавать в службе OneDrive папки для разных целей, например для общедоступных и личных материалов или отдельных проектов. Общий доступ к каждой из них можно предоставить разным пользователям и группам либо не предоставлять никому.
  
Дополнительные сведения о совместном использовании см. также в разделе "Общий доступ к файлам и [папок в Microsoft 365".](https://support.microsoft.com/office/72f26d6c-bf9e-432c-8b96-e3c2437f5b65)
  
### <a name="track-how-much-space-you-have-left"></a>Отслеживание свободного места

Чтобы узнать, сколько места на диске осталось в OneDrive, см. в подзагодре "Управление [хранилищем OneDrive для бизнеса".](https://support.microsoft.com/office/31519161-059C-4764-B6F8-F5CD29F7FE68)
  
### <a name="what-files-can-be-stored-in-onedrive-and-a-team-site"></a>Какие файлы можно хранить в OneDrive и на сайте группы?

Хотя вы можете отправить почти все типы файлов, некоторые имена файлов и символы в именах файлов не разрешены. Дополнительные сведения см. в сведениях о недопустимых символах и типах [файлов в OneDrive для бизнеса.](https://support.microsoft.com/office/64883A5D-228E-48F5-B3D2-EB39E07630FA)
  
### <a name="enable-or-disable-third-party-storage-services"></a>Включить или отключить сторонние службы хранения данных

Вы можете включить сторонное хранилище для пользователей в Microsoft 365, чтобы они могли хранить документы и делиться ими с помощью таких служб, как Dropbox, в дополнение к OneDrive и сайтам группы. Это отличная возможность обеспечить пользователям доступ к тем службам, которые они уже используют или предпочитают использовать для бизнес-проектов. Если вы не хотите, чтобы люди в вашей организации, использующие Office, открывали файлы в стороной службе, выполните следующие действия, чтобы отключить его.
  
> [!IMPORTANT]
> Сторонное хранилище включено по умолчанию, поэтому вам необходимо выполнить эти действия сразу, если вы не хотите, чтобы оно было доступно вашим пользователям.
  
1. Во входе в Центр <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">администрирования.</a>

2. Перейдите на **страницу** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**параметров организации параметров.**</a>

3. На **вкладке "Службы"** выберите **Office в Интернете.**

4. Чтобы включить или отключить стороннее хранилище, выберите или отключите его, а затем выберите "Сохранить **изменения".**
