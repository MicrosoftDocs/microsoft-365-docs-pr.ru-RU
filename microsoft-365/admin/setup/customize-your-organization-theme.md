# <a name="customize-the-microsoft-365-theme-for-your-organization"></a>Настройка темы Microsoft 365 для организации

В качестве администратора организации можно создать несколько тем для сотрудников организации и выбрать, какие темы применяются к различным членам организации. Тема организации — это то, что отображается в верхней панели навигации для людей в организации.

Вы можете добавить или обновить тему по умолчанию, которая применяется ко всем в вашей организации. Вы также можете создать до четырех дополнительных групповых тем, которые могут быть назначены нескольким Microsoft 365 группам.
  
## <a name="add-or-update-your-organizations-theme"></a>Добавление или обновление темы организации

1. В центре администрирования перейдите на страницу **Параметры** org Параметры, а затем выберите вкладку \>  **профилей** Организации.

2. На **вкладке профилей Организации** выберите **настраиваемые темы.**

Все темы можно настроить с помощью следующих вкладок.

|**Tab**|**What can you do?**|
|:-----|:-----|
|[Общие](#general-modify-a-theme) <br/> |Измените имя темы и назначьте до пяти групп (если применимо).  <br/> |
|[Логотипы](#logos-specify-your-theme-logos) <br/> |Добавьте логотип темы, включая Office темную тему и мобильные параметры.  <br/> |
|[Colors](#colors-choose-theme-colors) <br/> |Настройка цветовой схемы путем указания панели навигации, акцента, текста и значка. <br/> |

## <a name="general-modify-a-theme"></a>Общие. Изменение темы

Ваш опыт работы на вкладке General зависит от того, добавляете или измените тему по умолчанию или групповую тему.

### <a name="update-the-default-theme"></a>Обновление темы по умолчанию

Тема по умолчанию — это первая отображаемая тема.  

1. Если вы ранее настраивали тему для  организации, выберите тему по умолчанию и используйте одну из сохраненных настроек или выберите **тему Добавить**.
2. На **главной** странице можно запретить пользователям переопределять тему и отображать имя пользователя.
3. Нажмите кнопку **Сохранить**, чтобы сохранить изменения.  

> [!IMPORTANT]
> Тема по умолчанию уникальна, ее нельзя переименовать и использовать для всех в организации. Чтобы удалить тему по умолчанию, сначала необходимо удалить все остальные темы.

:::image type="content" source="../../media/default-theme-general.png" alt-text="Снимок экрана: общая вкладка, показывающая тему по умолчанию для организации":::

### <a name="create-a-group-theme"></a>Создание групповой темы

Можно создать до четырех дополнительных групповых тем.

1. На странице **Общие** введите имя для темы.

2. В **группах** можно выбрать до 5 Microsoft 365 групп, которые могут видеть групповую тему, а не использовать тему по умолчанию. Вы также можете запретить пользователям переопределять тему и показывать имя отображения пользователя.

3. Нажмите **Сохранить**.

:::image type="content" source="../../media/default-theme-general-users.png" alt-text="Снимок экрана: общая вкладка, показывающая тему по умолчанию для группы пользователей в организации":::

## <a name="logos-specify-your-theme-logos"></a>Logos: Specify your theme logos

На странице **Logos** можно добавить свои логотипы и указать URL-адрес, в который будут ориентироваться пользователи при выборе логотипа.

- **Логотип по умолчанию.** Добавьте расположение URL-адреса, которое указывает на ваш логотип. Убедитесь, что URL-адрес использует HTTPS. Если вы загружаете логотип, убедитесь, что он меньше 10kb. Ваш логотип по умолчанию может быть в формате JPG, PNG, GIF или SVG. Для изображений SVG они будут повторно размером до 24 пикселей по вертикали. Изображения JPG, PNG, GIF будут масштабироваться до 200 x 48 пикселей.
- **Альтернативный логотип.** Добавьте расположение URL-адреса, которое указывает на ваш логотип. Ваш альтернативный логотип должен быть оптимизирован для использования в Office темах. Те же требования, что и к логотипу по умолчанию.
- **Ссылка на кнопку мыши:** Добавьте расположение URL-адреса, которое указывает на ваш логотип. Логотип можно использовать в качестве ссылки на любой ресурс компании, например веб-сайт вашей компании.

Нажмите кнопку **Сохранить**, чтобы сохранить изменения.

Вы можете удалить свои логотипы в любое время. Просто вернись на **страницу Logos** и выберите **Удалить**.
  
## <a name="colors-choose-theme-colors"></a>Цвета: выберите цвета темы

На странице **Цвета** можно установить цвета по умолчанию и выбрать, какой логотип следует использовать.

- **Цвет панели навигации.** Выберите цвет для фона панели навигации. На каждой странице в верхней части отображается планка навигации.
- **Цвет текста и значка.** Выберите цвет для текста и значков на верхней панели навигации.
- **Цвет акцента.** Выберите тот, который хорошо отображается на белом или светлом фоне. Цвет акцента используется для цвета некоторых ссылок и кнопок, которые показываются на белом или светлом фоне. Например, цвет акцента используется для цветовых элементов в почтовом ящике пользователя и на странице портала Office.com.
- **Сброс цвета.** Выберите эту ссылку для сброса цветов в цвета по умолчанию.
- **Какой логотип следует использовать?**: Выберите логотипы по умолчанию или любые другие созданные вами логотипы.

:::image type="content" source="../../media/default-theme-colors.png" alt-text="Снимок экрана: вкладка Colors, показывающая цвета темы по умолчанию для организации":::

## <a name="frequently-asked-questions"></a>Вопросы и ответы

### <a name="my-organization-already-has-a-theme-for-all-employees-how-will-this-change"></a>В моей организации уже есть тема для всех сотрудников. Как это изменится?

Тема по умолчанию будет по-прежнему показываться всем сотрудникам.Добавление новой групповой темы будет доступно только для Microsoft 365 групп, связанных с этой темой.

### <a name="whydont-isee-group-themes-in-the-admin-center"></a>Почему я не вижу групповые темы в Центре администрирования?

Настраивать темы компании могут только глобальные администраторы.Глобальные читатели имеют доступ только для чтения.

### <a name="how-many-different-themes-can-i-set-up-for-my-organization"></a>Сколько различных тем можно настроить для своей организации?  

Можно создать до пяти тем. Тема по умолчанию и четыре групповых темы.  

### <a name="can-i-use-security-groups-or-distribution-groups-instead-of-microsoft-365-groups"></a>Могу ли я использовать группы безопасности или группы рассылки вместо Microsoft 365 групп?

Нет, новые групповые темы должны быть соопределяться с одной или Microsoft 365 группами, а не группами безопасности или группами рассылки.

> [!NOTE]
> Группы [рассылки можно](../manage/upgrade-distribution-lists.md) преобразовать в Microsoft 365 группы в Outlook.

### <a name="can-imanually-assign-a-theme-independent-ofmicrosoft-365-groups"></a>Могу ли я вручную назначить тему независимо от Microsoft 365 групп?  

Нет, новые групповые темы должны быть соедемы с одной или Microsoft 365 группами. Пользователи, которые являются членами Microsoft 365 группы, получат тему, примененную к их группе. Вы можете создать и добавить новых членов в группу **** [Microsoft 365,](../create-groups/create-groups.md) переехав в Параметры   >  **групп** в центре администрирования.

### <a name="what-happens-if-a-user-is-assigned-to-multiple-group-themes"></a>Что произойдет, если пользователю назначено несколько групповых тем?  

Пользователям, которым назначено несколько групповых тем, будет показана тема по умолчанию.  

### <a name="why-cant-i-delete-the-default-theme"></a>Почему я не могу удалить тему по умолчанию?  

Тема по умолчанию может быть удалена только после удаления всех групповых тем. Прежде чем пытаться удалить групповую тему, убедитесь, что вы удалите все групповые темы.

### <a name="why-am-i-receiving-an-error-message-every-time-i-uploadalogo-url"></a>Почему я получаю сообщение об ошибке каждый раз, когда я загружаю URL-адрес логотипа.  

Убедитесь, что логотип, который вы используете, указан как общедоступный URL-адрес. Выполните следующие действия для загрузки логотипов в [Azure Blob служба хранилища](/azure/storage/blobs/storage-upload-process-images?tabs=dotnet) или Office 365 сеть доставки содержимого с SharePoint [Online](../../enterprise/use-microsoft-365-cdn-with-spo.md).

### <a name="why-am-i-receiving-themessagedoesnt-meet-minimum-color-contrast-ratio-of-451"></a>Почему я получаю сообщение "Не соответствует минимальному соотношению контрастности цвета 4,5:1"?

Рекомендуемое соотношение контраста между цветом текста, значка или кнопки и цветом фона 4.5:1. Вы можете переопредить эту рекомендацию и сохранить тему, так как это не является требованием.

### <a name="if-i-define-a-theme-which-places-in-microsoft-365-will-this-be-used"></a>Если я определяю тему, какие Microsoft 365 будут использоваться?

Любая тема отображается в верхней панели навигации для всех в организации в Microsoft 365 набора.
