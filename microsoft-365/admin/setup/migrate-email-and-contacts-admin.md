# <a name="migrate-email-and-contacts-to-microsoft-365"></a>Перенос электронной почты и контактов в Microsoft 365

Импорт или перенос электронной почты из Gmail или другого поставщика электронной почты в Microsoft 365.
  
 **Нужна помощь в этом?**  [Обратитесь в службу поддержки Microsoft 365 для бизнеса.](../contact-support-for-business-products.md) 
  
Для этой задачи необходимо использовать версию Outlook, установленную на рабочем столе. Outlook включен в большинство планов Microsoft [](https://go.microsoft.com/fwlink/p/?LinkId=723731)365.
  
### <a name="migrate-gmail-to-microsoft-365"></a>Перенос Gmail в Microsoft 365

Выполните следующие действия, чтобы импортировать или перенести электронную почту, контакты и календарь из Gmail в Outlook с помощью Microsoft 365:
  
- [Импорт Gmail в Outlook](https://support.microsoft.com/office/20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7)
    
- [Импорт контактов в Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
- [Импорт календаря Google](https://support.microsoft.com/office/098ed60c-936b-41fb-83d6-7e3786437330)
    
> [!VIDEO https://www.microsoft.com/videoplayer/embed/c1e45708-9a92-475b-910b-f5aa40614d92?autoplay=false]
  
### <a name="import-outlook-pst-files-to-microsoft-365-desktop"></a>Импорт PST-файлов Outlook в Microsoft 365 (настольные ПК)

Выполните следующие действия, чтобы экспортировать электронную почту, контакты и календари из Outlook в PST-файл, а затем импортировать этот файл в Outlook с помощью Microsoft 365:
  
1. [Экспорт электронной почты, контактов и календаря](https://support.microsoft.com/office/14252b52-3075-4e9b-be4e-ff9ef1068f91)
    
2. [Импорт почты, контактов и календаря](https://support.microsoft.com/office/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
    
Если вам нужны только контакты, выполните следующие действия.
  
1. [Экспорт контактов из Outlook](https://support.microsoft.com/office/10f09abd-643c-4495-bb80-543714eca73f)
    
2. [Импорт контактов в Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
Чтобы запустить этот процесс, откройте Outlook и выберите **"Открыть** экспорт и импорт и экспорт \> **&amp;** \> **файлов".**
  
![Меню "Файл" в Outlook 2016](../../media/2f1c39a5-177e-4052-9dd8-90c0d140be2c.png)![Команда &amp; "Открыть экспорт" в Outlook 2016](../../media/eecab6df-c372-45b1-8a8a-2f6d7af0dd68.png)![Кнопка импорта и экспорта в Outlook 2016](../../media/ed90ae47-20db-4be1-b0c0-826008432c6e.png)
  
## <a name="see-other-email-accounts-in-outlook"></a>Другие учетные записи электронной почты в Outlook

Просто хотите увидеть свою электронную почту от другого поставщика (например, Gmail, Yahoo или Live.com) в Outlook? Нет необходимости импортировать или перенести. Вы можете настроить Outlook или Outlook Web App таким образом, чтобы получать доступ к другим учетным записям из того же места, где и почтовый ящик Microsoft 365, а также отправлять, получать и читать сообщения электронной почты, отправляемые на эти учетные записи.
  
### <a name="outlook-desktop"></a>Outlook (рабочий стол)

Добавьте учетную запись, например частную учетную запись Gmail, в Outlook.
  
- Откройте Outlook, а затем перейдите к **учетной** \> **записи "Добавить файл".**
    
Требуются более подробные инструкции? См. ["Добавление учетной записи".](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b)

## <a name="multiple-mailboxes-admins-can-bulk-import-email-contacts-and-calendars-to-microsoft-365"></a>Несколько почтовых ящиков: администраторы могут массово импортировать электронную почту, контакты и календари в Microsoft 365

В зависимости от вашей системы электронной почты можно выбрать один из нескольких способов массовой миграции. Ознакомьтесь с способами переноса нескольких учетных записей электронной почты в [Microsoft 365,](https://docs.microsoft.com/Exchange/mailbox-migration/mailbox-migration) чтобы определить, какой из них вам подходит.
