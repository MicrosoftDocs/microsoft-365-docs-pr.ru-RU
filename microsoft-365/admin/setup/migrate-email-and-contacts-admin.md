# <a name="migrate-email-and-contacts-to-microsoft-365"></a>Перенос электронной почты и контактов в Microsoft 365

Импорт или перенос электронной почты из Gmail или другого поставщика электронной почты в Microsoft 365.
  
 **Хотите помочь с этим?**  [Свяжитесь с Microsoft 365 для поддержки бизнеса.](../contact-support-for-business-products.md) 
  
Для этой задачи необходимо использовать версию Outlook, установленную на рабочем столе. Outlook включен в большинство планов Microsoft [](https://go.microsoft.com/fwlink/p/?LinkId=723731)365.
  
### <a name="migrate-gmail-to-microsoft-365"></a>Перенос Gmail в Microsoft 365

Выполните следующие действия по импорту или переносу электронной почты, контактов и календаря из Gmail в Outlook с помощью Microsoft 365:
  
- [Импорт Gmail в Outlook](https://support.microsoft.com/office/20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7)
    
- [Импорт контактов в Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
- [Импорт календаря Google](https://support.microsoft.com/office/098ed60c-936b-41fb-83d6-7e3786437330)
    
> [!VIDEO https://www.microsoft.com/videoplayer/embed/c1e45708-9a92-475b-910b-f5aa40614d92?autoplay=false]
  
### <a name="import-outlook-pst-files-to-microsoft-365-desktop"></a>Импорт PST-файлов Outlook в Microsoft 365 (настольный компьютер)

Выполните следующие действия для экспорта электронной почты, контактов и календарей из Outlook в файл pst, а затем импортировать этот файл в Outlook с помощью Microsoft 365:
  
1. [Экспорт электронной почты, контактов и календаря](https://support.microsoft.com/office/14252b52-3075-4e9b-be4e-ff9ef1068f91)
    
2. [Импорт почты, контактов и календаря](https://support.microsoft.com/office/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
    
Если вам просто нужны контакты, выполните следующие действия:
  
1. [Экспорт контактов из Outlook](https://support.microsoft.com/office/10f09abd-643c-4495-bb80-543714eca73f)
    
2. [Импорт контактов в Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
Чтобы запустить процесс, откройте Outlook и выберите **файл Open** \> **Export &amp;** \> **Import/Export**.

## <a name="see-other-email-accounts-in-outlook"></a>Другие учетные записи электронной почты в Outlook

Просто хотите, чтобы ваша электронная почта от другого поставщика (например, Gmail, Yahoo или Live.com) в Outlook? Нет необходимости импортировать или мигрировать. Вы можете настроить Outlook или Outlook Web App, чтобы вы могли получать доступ к другим учетным записям из того же места, что и почтовый ящик Microsoft 365, и отправлять, получать и читать сообщения электронной почты, отправленные на эти учетные записи.
  
### <a name="outlook-desktop"></a>Outlook (настольный компьютер)

Добавьте учетную запись, например свою личную учетную запись Gmail, в Outlook.
  
- Откройте Outlook, а затем перейдите к **учетной записи Добавить** \> **файл**.
    
Нужны более подробные действия? См. [статью Добавление учетной записи](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b).

## <a name="multiple-mailboxes-admins-can-bulk-import-email-contacts-and-calendars-to-microsoft-365"></a>Несколько почтовых ящиков: администраторы могут массово импортировать электронную почту, контакты и календари в Microsoft 365

В зависимости от источника системы электронной почты можно выбрать один из нескольких методов массовой миграции. Ознакомьтесь с способами переноса нескольких учетных записей электронной почты в [Microsoft 365,](/Exchange/mailbox-migration/mailbox-migration) чтобы определить, какой метод работает для вас.
