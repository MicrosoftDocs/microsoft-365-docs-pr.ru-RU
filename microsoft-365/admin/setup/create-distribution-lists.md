# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>Создание групп распределения в Microsoft 365 центре
  
Используется, когда вы хотите отправить электронную почту группе людей без введать имя каждого отдельного получателя, группы распределения (списки распределения) организованы определенной темой обсуждения (например, "Маркетинг") или пользователями, которые разделяют общую работу, которая требует от них часто общаться. Они также предоставляют вам возможность автоматически переамыть электронную почту на несколько адресов электронной почты.

Группы распределения иногда называются списками распределения.
  
## <a name="create-a-distribution-group-list"></a>Создание группы распределения (список)

::: moniker range="o365-worldwide"

Перейдите в Центр администрирования <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a>.

::: moniker-end

::: moniker range="o365-germany"

Перейдите в Центр администрирования <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a>.

::: moniker-end

::: moniker range="o365-21vianet"

Перейдите в Центр администрирования <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a>.

::: moniker-end

1. Щелкните значок средства запуска приложений  и выберите **Администратор**.
    
    Не можете найти приложение, которое ищете? Из запуска приложения выберите все **приложения,** чтобы увидеть алфавитный список приложений, доступных для вас. Оттуда вы можете искать определенное приложение. 
    
2. Выберите  \> **группы** групп в левом навигационном стекле, а затем **выберите Добавить группу.** 
      
3. На поле **типа группы выберите** Распределение, **а** затем выберите **Next**.
  
4. Введите псевдоним имени, описания и электронной почты для вашей новой группы и выберите, хотите ли вы, чтобы люди за пределами вашей организации посылали группу по электронной почте. 
    
5. Нажмите **Добавить,** чтобы создать группу, а затем просмотреть вашу группу и **выбрать Закрыть**. 
    
6. Чтобы добавить пользователей в список дистрибутива, [см.](../email/add-user-or-contact-to-distribution-list.md)
    
Проверьте, как использовать группу в Outlook 2016 и Outlook в Интернете [в использовании контактных групп (ранее списки распределения) в Outlook](https://support.microsoft.com/office/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39). 
  
Ознакомьтесь с [проблемами группы распределения](/office365/troubleshoot/groups/distribution-list-issues) для справки о проблемах с списком дистрибутивов.
