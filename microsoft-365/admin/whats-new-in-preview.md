### <a name="multi-tenant-management"></a>Управление несколькими арендаторами

Мы разработали набор функций для нескольких администраторов, таких как вы, чтобы сделать свою работу быстрее и эффективнее. Дополнительные сведения см. в [руб. Управление несколькими арендаторами.](multi-tenant/manage.md)

- **Ваши клиенты.** Быстро переключаться между арендаторами, которые вы управляете.
- **Все** клиенты : новая страница, на которой можно быстро увидеть состояние всех служб клиента, запросы на открытые службы, продукты и выставление счета, рекомендуемые задачи установки и количество пользователей в этом клиенте.
- **Настройка.** Страница установки с несколькими арендаторами предоставляет представление списка страницы Установки, но организована для многих клиентов. Вы можете увидеть, какие функции не включены, какие задачи являются завершены для всех клиентов, задачи, которые по-прежнему необходимо выполнить. Это представление поможет отслеживать принятие функций и убедиться, что рекомендуемые задачи установки безопасности всегда будут выполняться.
- **Здоровье службы.** В представлении о состоянии службы показано, влияют ли какие-либо инциденты или советы на клиентов. В нем даже будет посвеяно, сколько из управляемых клиентов затронуто. Просто выберите инцидент, чтобы получить дополнительные сведения на вкладке обзор, а затем переключение на вкладку Клиенты затронуты, чтобы сверлить и поддерживать этого клиента.
- **Миграция** почтовых ящиков с перекрестным клиентом — это новая служба, которая теперь является общедоступным предварительным просмотром, которая позволяет перемещать почтовые ящики между пользователями без необходимости отключения от борта, а затем и на борту почтовых ящиков. 
- **Совместное использование домена** для нескольких клиентов. В ближайшее время вы можете присоединиться к закрытому предварительному просмотру возможностей, позволяющих совместно использовать домен для нескольких клиентов. Например, если Contoso приобретает wingtip Toys, Contoso может делиться доменом с Wingtip Toys, чтобы люди в обоих клиентах могли использовать "contoso.com" в качестве адресов электронной почты.

### <a name="monitor-your-most-important-accounts"></a>Мониторинг наиболее важных учетных записей

Вы можете отслеживать и отслеживать неудавшие или отложенные сообщения электронной почты, отправленные пользователям, которые имеют большое влияние на бизнес, например вашему генеральному директору. Отслеживайте учетные записи приоритетов, добавляя пользователей в список приоритетных учетных записей в центре Microsoft 365 администрирования. Добавьте руководителей, руководителей, руководителей или других пользователей, которые имеют доступ к конфиденциальной или приоритетной информации.

Учетные записи приоритетов доступны только организациям, которые соответствуют следующим требованиям:

- Office 365 E3 или Microsoft 365 E3, Office 365 E5 или Microsoft 365 E5.
- Не менее 10 000 лицензий и не менее 50 активных Exchange Online пользователей.

![Страница установки для этой функции: мониторинг наиболее важных учетных записей](../media/MAC-WN-PriorityAccounts.png)

Существует два способа начать работу:

- Перейдите **к пользователям,** а затем в меню три точки (больше действий) выберите **Управление** учетной записью приоритета, чтобы добавить пользователей в список.
- Перейдите **к установке,** найдите задачу установки **Monitor ваши** наиболее важные учетные записи, а затем выберите **Начало** работы .

Дополнительные сведения о приоритетных учетных записях можно узнать в [мониторинге приоритетных учетных записей.](./setup/priority-accounts.md)

### <a name="search-faster-and-get-better-results-from-any-page"></a>Поиск быстрее и улучшение результатов с любой страницы

Мы начали развертывание нового интерфейса поиска для центра администрирования, и мы не можем дождаться, когда вы опробуете его. ![Поле поиска переместилось в область баннеров. Alt+S для поиска с любой страницы.](../media/MAC-WN-GlobalSearch.png)

- Поле Поиска переместилось в область загона, где написано "Microsoft 365 центра администрирования", поэтому теперь вы будете искать с любой страницы, а не только на домашней странице. У нас даже есть ярлык: **Alt+S**.
- Поиск является более интеллектуальным и даст вам лучшие результаты, даже быстрее. Попробуйте ввести "2fa", чтобы начать работу.
- Результаты поиска организованы по типу элемента или действию, которые можно принять.
  - **Пользователи.** Выберите имя пользователя, и вы можете изменить его прямо там. Если вы выберите меню три точки (больше действий) рядом с их именем, вы можете сбросить пароль. Вы можете искать по имени, фамилии, имени, имени пользователя или основному адресу электронной почты и псевдонимам электронной почты. Но чтобы получить точное совпадение, поиск по основному адресу электронной почты или имени пользователя.
  - **Группы.** Изменить группу с любой страницы, добавить участников, назначить владельцев.
  - **Действия.** Аналогично тому, как можно искать пользователя, а затем сбросить пароль, можно также сбросить пароль с любой страницы, а затем сбросить один или несколько паролей для пользователей.
  - **Навигация.** Результаты в навигации могут быстро помочь быстро добраться до страницы в центре администрирования. Например, поиск "ролей" будет проходить на страницу Roles для ролей Azure AD.
  - **Параметры:** Поиск любых параметров, связанных с организацией, службами, на которые вы подписаны, и параметров безопасности и конфиденциальности. 
  - **Домены**. Вы можете найти быстрые ссылки на домены, а затем ссылка будет принимать вас к обзору домена и здоровья страницы.
  - **Документация.** Если мы не можем найти результат для вас, мы попытаемся найти документацию, чтобы помочь. Чтобы найти совпадение, для кураторского списка статей требуется немного больше времени, поэтому подождите секунду, чтобы поиск нашел результаты. 
  - **Обратная** связь. Не нашли то, что вы искали? Отправьте нам отзывы из Search. Мы добавим функции поиска для дополнительных страниц и дополнительных функций в центре администрирования.
