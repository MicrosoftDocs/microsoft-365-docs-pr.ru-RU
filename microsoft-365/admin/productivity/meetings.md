# <a name="meetings--people-experiences"></a>Собрания — впечатления от работы с людьми

Оценка производительности позволяет получить представление о переходе вашей организации на цифровую трансформацию благодаря использованию Microsoft 365 и поддерживаемой технологии. Оценка организации отражает показатели опыта людей и технологий и может сравниваться с контрольным показателем от организаций, похожих на ваш. Категория "Собрания" является частью мер по опытом работы пользователей. Чтобы узнать больше, ознакомьтесь с обзором [оценки производительности](productivity-score.md) и ознакомьтесь с [заявлением о конфиденциальности корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Необходимые компоненты

Для начала работы с данными о собраниях у людей в вашей организации должна быть лицензия на:

- Microsoft Teams

Дополнительные сведения см. [в теме "Назначение лицензий пользователям".](../manage/assign-licenses-to-users.md)

После того как люди были активны в Teams по крайней мере один раз в течение последних 28 дней, вы начнете видеть эти сведения.

## <a name="why-your-organizations-meetings-score-matters"></a>Почему важна оценка собраний в организации

Собрания, на которых люди изучают идеи, планируют, решают проблемы и принимают решения, являются фундаментальной основой производительности организации. Исследования показывают, что при эффективном использовании средств собраний по сети они обычно экономят до 104 минут в неделю. См. [свидетельство.](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf)

## <a name="how-we-calculate-the-meetings-score"></a>Как мы вычисляем оценку собраний

Мы предоставляем основную информацию о том, как это сделать, которая содержит ключевые показатели для этой категории. Затем для расчета показателей в организации используется система показателей, описанная ниже.

### <a name="primary-insight"></a>Основные сведения

Microsoft Teams интегрируется с календарем Outlook и предоставляет множество возможностей для более привлекательных и эффективных собраний. Основная информация рассматривает все онлайн-собрания Microsoft Teams, которые были проведены в вашей организации. Основное представление также отслеживает, сколько собраний следует хотя бы одному из лучших методов для собраний Teams.

:::image type="content" source="../../media/primaryinsights-meetings.png" alt-text="PRimary insights for meetings with best practices.":::

1. **Заготовка:** Показывает процент онлайн-собраний в Microsoft Teams, прошедших за последние 28 дней с видео- или экранным доступом во время собрания.
2. **Body:** Предоставляет дополнительные сведения о том, как следующие лучшие методики взаимодействия во время собрания, такие как использование видео или общий доступ к экрану, могут сделать собрания более эффективными.
3. **Визуализация (текущее состояние):**

      - На этой горизонтальной гистограмме синяя (цветная) часть представляет процент, показанный в заголовке
      - Дробь (числовая или деномилиторная) используется для вычисления процента, показанного в загоне
         - Числитель: количество онлайн-собраний Microsoft Teams, включая пользователей из вашей организации, которые использовали видео или общий доступ к экранам.
         - Denominator: количество онлайн-собраний Microsoft Teams, в том числе пользователей из вашей организации, которые проводились за последние 28 дней.
      - Значение одноранговой производительности ключевой метрики также отображается в процентах.
1. **Ссылка на ресурсы:** Выберите эту ссылку, чтобы просмотреть содержимое справки.

### <a name="scoring-framework"></a>Структура озвучки

Оценка собраний для вашей организации измеряет степень, до которой собрания Microsoft Teams по сети в вашей организации следовали лучшим методикам за последние 28 дней. Она взвешна в зависимости от количества пользователей в организации, участвующих в собраниях, и продолжительности собраний.

## <a name="explore-more-about-meetings-in-your-organization"></a>Узнайте больше о собраниях в организации

Мы также предоставляем вам вспомогательную информацию, чтобы вы понимали, как люди в вашей организации, в совокупности, проводят собрания. Эти дополнительные показатели не напрямую влияют на оценку производительности, но могут помочь вам создать план действий в рамках цифровой трансформации.
