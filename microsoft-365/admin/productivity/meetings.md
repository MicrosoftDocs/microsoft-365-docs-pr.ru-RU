# <a name="meetings--people-experiences"></a>Собрания — взаимодействие с пользователями

Оценка производительности позволяет получить подробные сведения о процессе цифрового преобразования в Организации с помощью Microsoft 365 и технологических интерфейсов, поддерживающих эту возможность. Оценка вашей организации отражает измерения использования технологий для людей и технологий и может сравниваться со тестированиями в организациях, сходных с вашими. Категория "собрания" является частью показателей людей с опытом. Чтобы узнать больше, ознакомьтесь с [обзором оценки производительности](productivity-score.md) и ознакомьтесь с заявлением [Майкрософт о конфиденциальности](https://privacy.microsoft.com/privacystatement).

## <a name="why-your-organizations-meetings-score-matters"></a>Почему показатели собраний Организации важны

Собрания, в которых люди изучены идеи, планируют проблемы и принимают решения, являются фундаментальным элементом для продуктивности Организации. Исследование указывает на то, что при эффективном использовании средств для проведения собраний по сети они, как правило, сохраняют до 104 минут в неделю. Посмотрите [свидетельство](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf).

## <a name="how-we-calculate-the-meetings-score"></a>Вычисление оценки собраний

Мы предоставляем основное представление о возможностях, которые содержат ключевые показатели для этой категории. Затем структура оценки, описанная ниже, используется для этих метрик для вычисления оценки Организации.

### <a name="primary-insight"></a>Основное представление

Microsoft Teams интегрируется с календарем Outlook и предоставляет главный набор возможностей, позволяющих повысить эффективность и эффективность собраний. Основной анализ просматривает все собрания Microsoft Teams Online, которые существовали в вашей организации. Кроме того, основное представление отслеживает, сколько собраний выполняет по крайней мере одно из рекомендаций для собраний Teams.

2. **Основной текст:** В этой статье приводятся дополнительные сведения о следующих рекомендациях по задействованию на собрании, таких как использование видео или демонстрации экрана, позволяет повысить эффективность собраний.

      - Значение однорангового эталонного показателя также отображается в процентах.

### <a name="scoring-framework"></a>Структура оценки

Оценка собраний для вашей организации измеряет степень, в которую Интерактивные собрания Microsoft Teams в вашей организации следуют рекомендациям за последние 28 дней. Он задается в зависимости от количества людей в Организации, которые применялись к собраниям и продолжительность собраний.
