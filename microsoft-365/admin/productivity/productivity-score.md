# <a name="microsoft-productivity-score"></a>Оценка производительности (Майкрософт) 

Оценка производительности поддерживает переход на цифровые технологии с аналитикой использования в вашей организации Microsoft 365 и технологий, которые ее поддерживают. Оценка организации показывает измерения, отражающие результаты работы людей и технологий, и сравнивается с показателями эффективности организаций, похожих на вашу.

Предоставляются:

- **Показатели**, которые помогут понять, на каком этапе перехода на цифровые технологии вы находитесь.
- **Аналитические сведения** позволяют выявлять возможности для повышения производительности и удовлетворенности в вашей организации.
- **Рекомендации** для эффективного использования продуктов Microsoft 365 в вашей организации.

Показатели, аналитические сведения и рекомендации представлены в двух разделах: 

- **Эффективность пользователей:** Содержит количественную оценку работы организации с помощью категорий Microsoft 365, таких как совместная работа с контентом, мобильность, общение, собрания и командная работа.  

    Для каждой из упомянутых категорий рассматривается общественное исследование, чтобы найти лучшие рекомендации и связанные с ними преимущества в форме организационной эффективности. Например, исследования компании Forrester показали, что при совместной работе и обмене содержимого в облаке (а не в сообщениях электронной почты) пользователи могут сэкономить до 100 минут в неделю. Кроме того, мы оценили использование этих рекомендаций в вашей организации, чтобы помочь вам узнать, на каком этапе перехода на цифровые технологии вы находитесь. 

- **Эффективность технологий.** Ваша организация зависит не только от эффективного использования Microsoft 365, но и от надежных и хорошо работающих технологий. [Аналитика конечных точек](https://aka.ms/endpointanalytics) помогает понять, как производительность аппаратного и программного обеспечения влияет на вашу организацию. Работоспособность приложений Microsoft 365 позволяет определить, используют ли устройства в вашей организации приложения Microsoft 365 на рекомендуемых каналах.

## <a name="before-you-begin"></a>Подготовка

Дополнительные сведения см. в статье [Что такое аналитика конечных точек?](/mem/analytics/overview). Дополнительные сведения об аналитике сетевого подключения Microsoft 365 см. в статье [Обзор сетевого подключения](../../enterprise/microsoft-365-networking-overview.md).

Для доступа к данным об эффективности сотрудников требуется подписка Microsoft 365 для бизнеса или Office 365 для предприятий. Для доступа к данным аналитики конечных точек клиента необходимо добавить Microsoft Intune в подписку. Intune помогает защитить данные организации, управляя устройствами и приложениями. После добавления Intune можно включить аналитику конечных точек в его интерфейсе. Дополнительные сведения о Microsoft Intune см. в [документации по Microsoft Intune](/mem/intune/). 

> [!NOTE]
> Для получения доступа к возможностям оценки производительности лицензия на Рабочую аналитику не требуется.

Оценка производительности доступна только в Центре администрирования Microsoft 365 и только для ИТ-специалистов с одной из следующих ролей.  

- Глобальный администратор
- Администратор Exchange
- Администратор SharePoint
- Администратор Skype для бизнеса
- Администратор Teams
- Глобальный читатель
- Читатель отчетов
- Средство чтения сводки об использовании

> [!NOTE]
> Только ИТ-специалист с ролью глобального администратора может зарегистрироваться или согласиться на использование клиента для оценки производительности.

Модель управления доступом на основе ролей для Оценки производительности помогает организациям в дальнейшей цифровой трансформации с Microsoft 365, обеспечивая гибкость при назначении ролей ИТ-специалистам в организации.

Корпорация Майкрософт стремится защитить индивидуальную конфиденциальность. Этот [документ о конфиденциальности](privacy.md)  рассказывает об элементах управления, предоставляемых нами в роли ИТ-администратора, чтобы убедиться в том, что сведения были полезны, не ставя под угрозу ваше доверие к корпорации Майкрософт.

Доступ к оценке производительности можно получить на главной странице Microsoft 365 Admin, перейдя в раздел **Отчеты** > **Оценка производительности**.
  
## <a name="how-the-score-is-calculated"></a>Метод вычисления оценки

Оценка производительности зависит от совокупных показателей эффективности сотрудников и технологий. Все категории оцениваются одинаково с максимальным количеством 100 баллов. Максимальный показатель оценки производительности — 800 баллов.

### <a name="score-categories"></a>Категории оценки 

- Общение (100 баллов)
- Собрания (100 баллов)
- Совместная работа над контентом (100 баллов)
- Командная работа (100 баллов)
- Мобильность (100 баллов)
- Аналитика конечных точек (100 баллов)
- Сетевое подключение (100 баллов)
- Работоспособность приложений Microsoft 365 (100 баллов)
- **Максимальное количество баллов = 800**
 
Каждая категория оценки содержит количественную оценку ключевых показателей использования Microsoft 365 в вашей организации при переходе на цифровые технологии. Доступны 28-дневный и 180-дневный обзоры основных видов деятельности. Также доступны вспомогательные показатели, не использующиеся при расчете оценки, но имеющие важное значение для выявления базовой статистики использования и конфигураций, которые можно исправить.

### <a name="products-included-in-productivity-score"></a>Данные, используемые в оценке производительности 

Оценка производительности включает данные из Exchange, SharePoint, OneDrive, Teams, Word, Excel, PowerPoint, OneNote, Outlook, Yammer и Skype.

Оценка организации обновляется ежедневно и отражает действия пользователей, выполненные за последние 28 дней (включая текущий день).

## <a name="interpreting-your-organizations-productivity-score"></a>Анализ Оценки продуктивности организации 

На домашней странице оценки производительности отображается общая оценка организации и ее история, а также основные аналитические сведения для каждой категории.

:::image type="content" source="../../media/prodscore-landing.png" alt-text="Страница оценки производительности в разделе ''Отчеты&quot;":::

**Оценка организации** отображается в процентах и баллах. В числителе отображаются баллы вашей организации, а в знаменателе — максимально возможное число баллов.

**Показатели конкурентов** позволяют сравнивать оценку вашей организации с оценками похожих организаций. Показатель конкурентов для категории эффективности сотрудников рассчитывается как среднее арифметическое показателей множества подобных организаций. Множество организаций состоит из организаций в вашем регионе с похожим количеством лицензированных пользователей, похожими типами лицензий, похожим видом деятельности и похожим сроком использования Microsoft 365.

> [!NOTE]
> Корпорация Майкрософт использует внутренние данные для определения отрасли, с которой сопоставлена организация. Клиенты в родительской организации сопоставляются с той же отраслью, что и родительская организация. Организации не могут просматривать или изменять сопоставление отраслей.

Показатель партнеров аналитики конечных точек включает рекомендации по улучшению производительности устройств и настройке ПО на основе совокупных средних значений всех клиентов.

Для сетевого подключения рекомендованный показатель составляет 80 баллов.

В разделе **Разбивка оценки** приводится разбивка оценки производительности по эффективности сотрудников и технологий.

В разделе ''История оценки" показано изменение оценки для каждой категории в течение последних шести месяцев.

В области **Эффективность сотрудников** и **Эффективность технологий** отображаются основные аналитические сведения для категорий в этих областях. Для просмотра подробных аналитических сведений можно выбрать необходимую категорию.

## <a name="business-continuity-special-report"></a>Специальный отчет о непрерывности бизнеса

Отчет о непрерывности бизнеса — это ограниченный по времени отчет по анализу рабочего места, доступный всем клиентам Microsoft 365, чтобы помочь им управлять организациями в это непростое время.  

Этот отчет помогает организациям понять: 

- Как переход на удаленную работу влияет на совместную работу и общение. 

- Как влияет работа из дома на баланс между работой и личной жизнью. 

- Способствуют ли удаленные собрания эффективному принятию решений.
