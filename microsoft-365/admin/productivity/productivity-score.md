# <a name="microsoft-productivity-score"></a>Оценка производительности корпорации Майкрософт 

Оценка производительности поддерживает путешествие к цифровому преобразованию с подробными сведениями о том, как ваша организация использует Microsoft 365 и технологический технологический интерфейс, поддерживающий эту службу. Оценка вашей организации отражает измерения использования технологий для людей и технологий и может сравниваться с тестами в организациях, аналогичных их размеру.

Он предоставляет:

- **Метрики** , которые помогут вам понять, где находится ваше путешествие с цифровым преобразованием.
- Сведения **о данных** , которые помогут вам определить возможности повышения продуктивности и удовлетворения потребностей Организации.
- **Рекомендуемые действия** , которые могут помочь Организации эффективно использовать продукты Microsoft 365.

Мы предоставляем показатели, аналитические сведения и рекомендации в двух областях: 

- **Возможности для людей:** Сведения о том, как Организация работает с помощью категорий Microsoft 365, таких как совместная работа, мобильность, связь, собрания и Командная работа.  

    Для каждой упомянутой категории мы рассмотрим общедоступные исследования, чтобы определить некоторые рекомендации и связанные с ними преимущества в форме организационной эффективности. Например, [Forrester] ( https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf исследование показывает, что при совместной работе и совместном использовании контента в облаке (вместо вложений электронной почты) они могут сэкономить до 100 минут в неделю. Кроме того, мы вычислим, как использовать эти рекомендации в Организации, чтобы понять, где находится ваше путешествие по цифровому преобразованию. 

- **Технологические интерфейсы:** Ваша организация зависит от надежных и хорошо производительных технологий, а также от эффективного использования Microsoft 365. С помощью [аналитики конечных точек](https://aka.ms/endpointanalytics) вы можете оценить влияние вашей организации на производительность и работоспособность, связанные с оборудованием и программным обеспечением. Проверка работоспособности приложений Microsoft 365 позволяет определить, работают ли на устройствах вашей организации приложения Microsoft 365 на рекомендуемых каналах.

Узнайте, [что такое аналитика конечной точки](https://docs.microsoft.com/mem/analytics/overview) для обзора и предварительных сведений. Чтобы узнать больше о службе сетевого подключения Microsoft 365, прочтите [Обзор сетевого подключения](https://docs.microsoft.com/microsoft-365/enterprise/microsoft-365-networking-overview).
  

## <a name="how-the-score-is-calculated"></a>Способ вычисления оценки

Показатели производительности основываются на совокупных показателях категорий взаимодействия с пользователями и технологическими технологиями. Каждая категория имеет одинаковый вес с суммой в 100 баллов. Наибольший возможный показатель производительности составляет 800.

### <a name="score-categories"></a>Категории оценок 

- Связь (100 точек)
- Собрания (100 точек)
- Совместная работа с контентом (100 точек)
- Совместная работа (100 точек)
- Мобильность (100 точек)
- Аналитика конечной точки (100 точек)
- Сетевое подключение (100 точек)
- Работоспособность приложений Microsoft 365 (100 точек)
- **Общее количество возможных = 800 баллов**
 
 В каждой категории оценки изменяйте ключевые индикаторы того, как организация использует Microsoft 365 во всех случаях, направленных на цифровое преобразование. Мы предоставим представление основных действий в течение 28 дней и 180 дней. Кроме того, мы предоставляем вспомогательные показатели, которые не являются частью вычисления оценки, но важны для определения базовых статистических показателей использования и конфигураций, которые можно использовать.

### <a name="products-included-in-productivity-score"></a>Продукты, включенные в оценку продуктивности 

Оценка производительности включает данные из Exchange, SharePoint, OneDrive, Teams, Word, Excel, PowerPoint, OneNote, Outlook, Yammer и Skype.

Баллы Организации обновляются ежедневно и отражают действия пользователя, выполненные за последние 28 (включая текущий день).


## <a name="pre-requisites"></a>Необходимые условия 

Для получения данных о пользователях необходим Microsoft 365 для бизнеса или Office 365 для корпоративных подписок. Чтобы получить данные аналитики конечной точки для клиента, необходимо добавить Microsoft Intune в подписку. Intune помогает защитить данные Организации за счет управления устройствами и приложениями. После использования Intune вы можете включить аналитику конечных точек в интерфейсе Intune. Узнайте больше о [Microsoft Intune](https://docs.microsoft.com/mem/intune/). 
> [!NOTE]
> Для получения функций оценки производительности не требуется использовать аналитику лицензирования на рабочее место.

Оценка производительности доступна только в центре администрирования Microsoft 365, и доступ к ним возможен только ИТ-специалистам, имеющим одну из следующих ролей:  

- Глобальный администратор 
- администраторы Exchange;
- Администратор SharePoint 
- Администратор Skype для бизнеса 
- Администратор Teams 
- Глобальный читатель 
- Читатель отчетов 

Обратите внимание, что эти сведения предназначены только для дальнейшего применения цифровой преобразования с помощью Microsoft 365, поэтому их необходимо предоставить с осторожностью. 

Корпорация Майкрософт стремится защищать персональную конфиденциальность. В этом [документе](privacy.md)  содержатся сведения об элементах управления, которые мы предоставляем вам, как ИТ-администратор вашей организации, чтобы убедиться в том, что эти сведения поддаются действию, не нарушая отношения доверия в Майкрософт.

Вы можете получить доступ к интерфейсу от имени "Домашняя страница" администратора Microsoft 365 в разделе **отчеты об**  >  **оценке производительности**.

## <a name="interpreting-your-organizations-productivity-score"></a>Интерпретация оценки производительности Организации 

Главная страница оценки производительности показывает общую оценку и историю оценок вашей организации, а также основную информацию о каждой категории.

:::image type="content" source="../../media/prodscore-landing.png" alt-text="Страница оценки производительности в отчетах.":::

**Показатель вашей организации** отображается в виде значения в процентах и в баллах. Вы можете видеть точки в числителе и максимальные возможные точки в знаменателе.

**Оценки одноранговых узлов** позволяют сравнить рейтинг Организации с организациями, такими как ваш. Эталонный тест для категорий взаимодействия с пользователями рассчитывается как среднее значение показателей в пределах набора похожих организаций. Набор организаций состоит из организаций в вашем регионе с аналогичным количеством лицензированных пользователей, типов лицензий, отраслей и тенуре с помощью Microsoft 365. 

Эталонный тест для узла аналитики конечной точки включает целевые показатели производительности при запуске устройств и рекомендуемую конфигурацию программного обеспечения на основе агрегированных значений медианы для всех клиентов.

Для сетевого подключения рекомендуемый контрольный тест составляет 80 баллов.

Раздел " **разбивка оценки** " обеспечивает разбивку оценки производительности с помощью тестовых показателей для людей и технологических областей.

Журнал оценок показывает, как изменился рейтинг в каждой категории за прошедшие 6 месяцев.

Области взаимодействия с **людьми** и **технологическими технологиями** содержат основные сведения о категориях в этих областях. Вы можете щелкнуть каждую категорию, чтобы просмотреть более подробные сведения.

## <a name="business-continuity-special-report"></a>Специальный отчет о непрерывности бизнеса

Отчет о непрерывности бизнеса — это отчет об ограничении рабочих задач в ограниченном времени, доступный всем пользователям Microsoft 365, чтобы помочь им в пошаговой разработке.  

Этот отчет помогает организациям освоить: 

- Влияние сдвига на удаленную работу на совместную работу и взаимодействие. 

- Влияние на баланс на время работы, когда пользователи настраиваются на работу из дома. 

- Поддерживает ли удаленное собрание эффективное принятие решений.
