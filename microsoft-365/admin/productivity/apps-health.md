# <a name="microsoft-365-apps-health--technology-experiences"></a>Работоспособность приложений Microsoft 365 — эффективность технологий

Оценка производительности позволяет получить представление о цифровом преобразовании в вашей организации благодаря использованию Microsoft 365 и поддерживаемой технологии. Оценка организации отражает показатели опыта людей и технологий и может сравниваться с контрольным показателем от организаций, похожих на ваш. Категория состояния приложений является частью показателей, которые подпадают под технологический опыт. Чтобы узнать больше, ознакомьтесь с обзором [оценки](productivity-score.md) производительности и ознакомьтесь с [заявлением о конфиденциальности корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="why-your-organization39s-microsoft-365-apps-health-score-matters"></a>Почему важно&#39;оценку состояния приложений Microsoft 365 в вашей организации

Производительность организации зависит от работоспособной среды приложений. Устройства с самыми актуальными версиями приложений Microsoft 365 в рекомендуемом канале более безопасны и помогают людям в вашей организации использовать все возможности Microsoft 365.

## <a name="how-we-calculate-the-microsoft-365-apps-health-score"></a>Расчет оценки состояния приложений Microsoft 365

Мы вычисляем оценку состояния приложений Microsoft 365, измеряя количество устройств в каждом канале обновления. Мы также определяем, работают ли на устройствах поддерживаемая версия, и самый новый выпуск приложений Microsoft 365.

Мы предоставляем основную информацию о том, как это сделать, которая содержит ключевые показатели для этой категории. Затем для вычисления показателей используется структура показателей, описанная в следующих разделах.

### <a name="primary-insight"></a>Основные сведения

Основная информация рассчитывается на устройствах, на которые запущены приложения Microsoft 365 в рекомендуемом обновленном канале.

1. **Визуализация (текущее состояние):**

### <a name="scoring-framework"></a>Структура озвучки

Оценка состояния работы приложений Microsoft 365 измеряет, работают ли устройства с приложениями Microsoft 365 на рекомендуемом канале и в последних версиях.

## <a name="explore-your-organization-microsoft-365-app-channels-and-versions"></a>Изучение каналов и версий приложений Microsoft 365 для вашей организации

Мы также предоставляем вспомогательные сведения, которые помогают получить дополнительную информацию о том, какие каналы и версии устройств в вашей организации в настоящее время запущены. Эти дополнительные показатели не помогают вам в оценке производительности, но помогут вам создать план действий для повышения оценки работоспособности приложений Microsoft 365, убедившись, что устройства запускают приложения Microsoft 365 по рекомендованным каналам.
