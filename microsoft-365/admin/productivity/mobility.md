# <a name="mobility--people-experiences"></a>Мобильность — возможности для людей

Оценка производительности помогает организациям преобразовывать принципы работы с метриками о том, как люди используют приложения Microsoft 365 и поддерживающие технологические возможности. Показатель показывает, что ваша организация&#39;а эффективность категорий людей и технологий, а также сравнивает свою оценку с организациями, такими как ваш. Категория "Совместная работа" является частью измерений, которые подпадают под опыт работы. Чтобы узнать больше, ознакомьтесь со статьей [Обзор оценки производительности](productivity-score.md) .

## <a name="why-your-organization39s-mobility-score-matters"></a>Причины, по которым Организация&#39;для мобильных устройств имеет значение

Фундаментальный принцип продуктивности Организации состоит в том, насколько хорошо люди могут работать гибко в любом месте. С помощью Microsoft 365 пользователи могут оставаться подключенными к Outlook, Microsoft Teams и Yammer. Кроме того, пользователи могут эффективно совместно работать с контентом с помощью Word, Excel, PowerPoint и OneNote из любого расположения и платформ.

## <a name="how-we-calculate-the-score"></a>Как вычисляется Оценка

Мы предоставляем основное представление о возможностях, которые содержат ключевые показатели для этой категории. Затем для этих метрик используется оценочная структура, описанная ниже.

### <a name="primary-insight"></a>Основное представление

Microsoft 365 позволяет людям гибко работать в различных приложениях, в том числе Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer и Skype. Люди также могут работать в любом месте, используя для настольных систем, веб-приложений и мобильных платформ. В основном сведения о продуктах, включенных для сотрудников Организации, и о том, сколько этих сотрудников активны по крайней мере на двух платформах, доступны для понимания.

### <a name="scoring-framework"></a>Структура оценки

Показатель мобильности определяет, используются ли пользователи Microsoft 365 Apps: Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer и Skype для различных платформ, настольных систем, веб-приложений и мобильных устройств.
