# <a name="mobility--people-experiences"></a>Мобильность — опыт работы с людьми

Оценка производительности позволяет получить представление о цифровом преобразовании в вашей организации благодаря использованию Microsoft 365 и поддерживаемой технологии. Оценка организации отражает показатели опыта людей и технологий и может сравниваться с контрольным показателем от организаций, похожих на ваш. Категория мобильности является частью мер по опытом людей. Чтобы узнать больше, ознакомьтесь с обзором [оценки](productivity-score.md) производительности и ознакомьтесь с [заявлением о конфиденциальности корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Предварительные условия

Чтобы начать работу с mobility insights, у людей в вашей организации должна быть лицензия на:

- Microsoft Teams
- Exchange Online
- Word
- Excel
- PowerPoint
- OneNote

Дополнительные сведения см. [в теме "Назначение лицензий пользователям".](../manage/assign-licenses-to-users.md)

После того как люди были активны в вышеуказанных продуктах хотя бы один раз за последние 28 дней, вы начнете видеть эти сведения.

## <a name="why-your-organization39s-mobility-score-matters"></a>Почему важно&#39;мобильности в вашей организации

Фундаментальным принципом производительности организации является то, насколько хорошо люди могут гибко работать, где бы они ни были. С помощью Microsoft 365 люди могут оставаться на связи с Outlook, Microsoft Teams и Yammer. Кроме того, с помощью Word, Excel, PowerPoint и OneNote можно легко работать над содержимым из любого расположения и на любых платформах.

## <a name="how-we-calculate-the-score"></a>Как вычисляется оценка

Мы предоставляем основную информацию о том, как это сделать, которая содержит ключевые показатели для этой категории. Затем для расчета показателей в организации используется приведенная ниже структура показателей.

### <a name="primary-insight"></a>Основные сведения

Microsoft 365 позволяет гибко работать с приложениями, включая Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer и Skype для бизнеса. Пользователи также могут работать из любого места с помощью простого рабочего стола, веб-платформ и мобильных платформ. Основная информация рассматривает продукты, которые включены для пользователей в вашей организации, и количество активных людей по крайней мере на двух платформах.

:::image type="content" source="../../media/primary-mobility.png" alt-text="Диаграмма, на которой показаны основные сведения о мобильности — оценка производительности для людей.":::

1. **Заглавная:** Показывает процент людей в вашей организации, которые имеют доступ к любому из приложений Microsoft 365 и которые активны по крайней мере в одном из этих приложений на более чем одной платформе.
2. **Body:** Предоставляет дополнительные сведения о том, как использование этих приложений на нескольких платформах может повысить эффективность и удовлетворенность.
3. **Визуализация (текущее состояние):** Показывает, сколько людей используют несколько платформ для настольных компьютеров, мобильных устройств и веб-приложений по крайней мере для одного Microsoft 365 (список ниже):
    - **Горизонтальная полоса,** где синяя (цветная) часть представляет процент, выраженный в заголовке.
    - **Дробь** на панели выделяет (числитель/замеитель), используемую для вычисления процента в заголовке.
        - Числитель: количество пользователей в организации, использующих любое приложение в Microsoft Outlook, Word, Excel, PowerPoint, OneNote и Microsoft Teams, Yammer и Skype на нескольких платформах с настольных компьютеров, интернета и мобильных устройств за последние 28 дней.
        - Denominator: количество пользователей, лицензированных для приложений Microsoft 365, Exchange, Yammer, Microsoft Teams или Skype за последние 28 дней.
    - Значение **одноранговой производительности** для ключевой метрики также отображается в процентах.
4. **Просмотр ресурсов mobility:** Выберите эту ссылку, чтобы просмотреть содержимое справки.

### <a name="scoring-framework"></a>Структура озвучки

Показатель мобильности для вашей организации измеряется на уровне организации (агрегированное), используют ли пользователи приложения Microsoft 365 — Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer и Skype — на различных платформах — настольных компьютерах, в Интернете и мобильных устройствах.

Оценки не предоставляются на уровне отдельных пользователей.
