# <a name="mobility--people-experiences"></a>Mobility — People experiences

Оценка производительности позволяет получить представление о пути цифровой трансформации организации с помощью Microsoft 365 и технологий, которые ее поддерживают. Оценка организации отражает измерения опыта людей и технологий и может сравниться с эталонами организаций, похожих на вашу. Категория мобильности является частью мер, которые испытывают люди. Чтобы узнать больше, ознакомьтесь с обзором [показателей](productivity-score.md) производительности и ознакомьтесь [с заявлением о конфиденциальности Корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Предварительные условия

Чтобы начать работу с данными о мобильности, необходимо получить лицензию на:

- Microsoft Teams
- Exchange Online
- Word
- Excel
- PowerPoint
- OneNote

Дополнительные сведения см. [в дополнительных сведениях о назначении лицензий пользователям.](../manage/assign-licenses-to-users.md)

После того как в течение последних 28 дней по крайней мере один раз в этих продуктах были активны люди, вы начнете видеть сведения.

## <a name="why-your-organization39s-mobility-score-matters"></a>Почему ваша&#39;имеет значение s mobility score

Основополагающим элементом производительности организации является то, насколько хорошо люди могут гибко работать, где бы они ни были. С Microsoft 365 люди могут оставаться на связи с Outlook, Microsoft Teams и Yammer. Кроме того, с помощью Word, Excel, PowerPoint и OneNote из любого расположения и платформ можно легко взаимодействовать с контентом.

## <a name="how-we-calculate-the-score"></a>Как мы вычисляем счет

Мы предоставляем основное представление о том, что содержит ключевые метрики для этой категории. Затем для этих показателей используется структура подсчета баллов, описанная ниже.

### <a name="primary-insight"></a>Основная информация

Microsoft 365 позволяет пользователям гибко работать в приложениях, включая Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer и Skype для бизнеса. Кроме того, пользователи могут работать из любой точки мира с помощью бесшовной работы на настольных, веб-и мобильных платформах. Основное представление рассматривает продукты, которые включены для людей в вашей организации, и сколько из этих людей активны по крайней мере на двух платформах.

:::image type="content" source="../../media/primary-mobility.png" alt-text="Диаграмма, показывающая основные сведения о мобильности — люди испытывают показатели производительности.":::

1. **Заглавная:** Показывает процент людей в вашей организации, которые имеют доступ к любому из Приложения Microsoft 365, и которые активно работают по крайней мере на одном из этих приложений на более чем одной платформе.
2. **Body:** Дополнительные сведения о том, как использование этих приложений на нескольких платформах может повысить эффективность и удовлетворенность.
3. **Визуализация (текущее состояние):** Показывает, сколько пользователей используют несколько платформ на настольных, мобильных и веб-платформах по крайней мере Microsoft 365 (список ниже) следующим образом:
    - **Горизонтальная** полоса, в которой синяя (цветная) часть представляет процент, выраженный в заголовке.
    - **Фракция** на панели выделяет (числитель/знаменатель), используемый для вычисления процента в заголовке.
        - Числитель. Количество пользователей в вашей организации, использующих любое приложение в Microsoft Outlook, Word, Excel, PowerPoint, OneNote и Microsoft Teams, Yammer и Skype на нескольких платформах с настольных компьютеров, веб-сайтов и мобильных устройств за последние 28 дней.
        - Знаменатель: число людей, лицензированных для Приложения Microsoft 365, Exchange, Yammer, Microsoft Teams или Skype, по крайней мере 1 из последних 28 дней.
    - **Одноранговая эталонная** величина для ключевой метрики также показана в процентах.
4. **Просмотр ресурсов Mobility:** Выберите эту ссылку для просмотра контента справки.

### <a name="scoring-framework"></a>Рамки скоринга

Оценка мобильности для организации измеряется на уровне организации (агрегированной) независимо от того, используют ли пользователи Приложения Microsoft 365 — Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer и Skype — на разных платформах — настольных, веб-и мобильных.

Оценки не предоставляются на уровне отдельного пользователя.
