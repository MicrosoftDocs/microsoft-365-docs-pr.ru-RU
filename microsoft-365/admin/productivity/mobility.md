# <a name="mobility--people-experiences"></a>Мобильность — возможности для людей

Оценка производительности позволяет получить подробные сведения о процессе цифрового преобразования в Организации с помощью Microsoft 365 и технологических интерфейсов, поддерживающих эту возможность. Оценка вашей организации отражает измерения использования технологий для людей и технологий и может сравниваться со тестированиями в организациях, сходных с вашими. Категория мобильности является частью показателей людей с опытом. Чтобы узнать больше, ознакомьтесь с [обзором оценки производительности](productivity-score.md) и ознакомьтесь с заявлением [Майкрософт о конфиденциальности](https://privacy.microsoft.com/privacystatement).

## <a name="why-your-organization39s-mobility-score-matters"></a>Причины, по которым Организация&#39;для мобильных устройств имеет значение

Фундаментальный принцип продуктивности Организации состоит в том, насколько хорошо люди могут работать гибко в любом месте. С помощью Microsoft 365 пользователи могут оставаться подключенными к Outlook, Microsoft Teams и Yammer. Кроме того, пользователи могут эффективно совместно работать с контентом с помощью Word, Excel, PowerPoint и OneNote из любого расположения и платформ.

## <a name="how-we-calculate-the-score"></a>Как вычисляется Оценка

Мы предоставляем основное представление о возможностях, которые содержат ключевые показатели для этой категории. Затем для этих метрик используется структура оценки, описанная ниже, чтобы рассчитать оценку Организации.

### <a name="primary-insight"></a>Основное представление

Microsoft 365 позволяет людям гибко работать в различных приложениях, в том числе Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer и Skype для бизнеса. Люди также могут работать в любом месте, используя для настольных систем, веб-приложений и мобильных платформ. В основном сведения о продуктах, включенных для сотрудников Организации, и о том, сколько этих сотрудников активны по крайней мере на двух платформах, доступны для понимания.

### <a name="scoring-framework"></a>Структура оценки

Оценка мобильности Организации измеряется на уровне Организации (совокупный), независимо от того, используете ли люди приложения Microsoft 365: Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer и Skype — для различных платформ, настольных систем, Интернета и мобильных устройств.

Показатели не предоставляются на уровне отдельного пользователя.
