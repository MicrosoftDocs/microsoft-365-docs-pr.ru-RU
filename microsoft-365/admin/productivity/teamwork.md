# <a name="teamwork--people-experiences"></a>Совместная работа — люди

Оценка производительности помогает организациям преобразовывать принципы работы с метриками о том, как люди используют приложения Microsoft 365 и поддерживающие технологические возможности. Показатель показывает, что ваша организация&#39;а эффективность категорий людей и технологий, а также сравнивает свою оценку с организациями, такими как ваш. Категория "Совместная работа" является частью измерений, которые подпадают под опыт работы. Чтобы узнать больше, ознакомьтесь со статьей [Обзор оценки производительности](productivity-score.md) .

## <a name="why-your-org39s-teamwork-score-matters"></a>Причины оценки вашей организации&#39;

Фундаментальный принцип продуктивной работы Организации состоит в том, что группа людей с общими целями совместно работает друг с другом и с общими ресурсами для общего успеха. Исследование показывает, что когда пользователи совместно используют данные и совместно работают в общих рабочих областях, они сохраняют до 4 часов в неделю. Они могут находить связанные документы, искать контекст для предыдущих обсуждений и предоставлять доступ к общим целям. Просмотр[свидетельства](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf)

## <a name="how-we-calculate-the-score"></a>Как вычисляется Оценка

Мы предоставляем основное представление о возможностях, которые содержат ключевые показатели для этой категории. Затем структура оценки, описанная ниже, используется для этих метрик для вычисления оценки.

## <a name="people-in-your-organization"></a>Сотрудники организации

В этом разделе описывается работа с метриками, на которые вы хотите сосредоточиться, предоставляя пользователям подробные сведения о действиях, связанных со всеми аналитическими данными на странице.

Следующие столбцы представлены в таблице на уровне пользователя:

- **Username:**  адрес электронной почты пользователя.
- **Отображаемое имя**  — это полное имя пользователя.
- **Дата последнего действия:**  последний раз, когда пользователь выполнил действия, относящиеся к какой-либо аналитической информации в ходе работы в течение последних 28 дней.
- **Отдел:** Отдел, определенный для пользователя в Azure Active Directory (Azure AD).
- **Организация:** Компания, определенная для пользователя в Azure AD.
- **Страна:** Страна, определенная для пользователя в Azure AD.
- **Состояние:** Состояние, определенное для пользователя в Azure AD.
- **City:** Город, определенный для пользователя в Azure AD.
- **Активные рабочие области:** Количество рабочих областей (в группах Microsoft 365), в которых был активен пользователь.
- **Отправленные дни электронной почты:** Количество дней, в течение которых пользователь отправил электронную почту за последние 28 дней.
- **Дней отправки сообщений чата или канала:** Количество дней, в течение которых пользователь отправил сообщение или канал чата в Microsoft Teams за последние 28 дней.
- **Дни чтения контента:** Количество дней, в течение которых пользователь получил доступ к контенту (файлы типа Microsoft Word/Excel/PowerPoint/OneNote или PDF) в OneDrive или SharePoint за последние 28 дней..
- **Дни создания контента:** Количество дней, в течение которых пользователь создал, изменил или передал контент (файлы типа Microsoft Word/Excel/PowerPoint/OneNote или PDF) в OneDrive или SharePoint за последние 28 дней.
- **Дней отправки электронного почтового ящика группы:** Количество дней, в течение которых пользователь отправил сообщения в группу почтового ящика за последние 28 дней
- **Дней в сообщениях канала:** Количество дней, в течение которых пользователь отправлял сообщения канала в Microsoft Teams за последние 28 дней
- **Дни чтение контента на сайтах групп SharePoint:** Количество дней, в течение которых пользователь получил доступ к контенту (файлы типа Microsoft Word/Excel/PowerPoint/OneNote или PDF) на сайтах группы SharePoint за последние 28 дней
- **Дни создания контента на сайтах групп SharePoint:** Количество дней, в течение которых пользователь создал, изменил или передал контент (файлы типа Microsoft Word/Excel/PowerPoint/OneNote или PDF) на сайтах группы SharePoint за последние 28 дней

Чтобы просмотреть сведения о уровне рабочей области, в раскрывающемся меню выберите пункт **Показать данные рабочей области** .

В таблице на уровне рабочей области представлены следующие столбцы:

- **Имя рабочей области:** Имя общей рабочей области (Microsoft 365 группы)
 - **Дата последнего действия** относится к моменту последнего действия, относящегося к какой бы то ни было аналитике в ходе работы в течение последних 28 дней.
 - **Общее число участников:** Количество участников рабочей области.
 - **Дата создания:** Дата создания рабочей области.
 -  **Содержит команды teams:** Логическое значение, указывающее, связана ли Рабочая область с группой Microsoft Teams.
 - **Дней с любой деятельностью:** Количество дней, в течение которых Рабочая область имела действия, включающие в себя групповую почту, сообщения канала или действия с файлами на связанном сайте SharePoint за последние 28 дней.
 - **Дни с действиями электронной почты:** Количество дней, в течение которых Рабочая область имела действия с электронной почтой в течение последних 28 дней.
 - **Дни с действиями канала:** Количество дней, в течение которых в рабочей области потребовались действия с сообщениями канала в течение последних 28 дней.
 - **Дней с действием "чтение контента":** Количество дней, в течение которых Рабочая область имела файлы Office, такие как Word, Excel, PowerPoint, OneNote или PDF, доступ к которым осуществляется со связанного сайта SharePoint за последние 28 дней.
 - **Дни с действием "создать контент":** Количество дней, в течение которых Рабочая область имела файлы Office, которые были созданы, изменены или отправлены со связанного сайта SharePoint в течение последних 28 дней.
 - **Люди с любыми действиями:** Количество пользователей, которые выполнили какие-либо действия, включающие в себя групповую почту, сообщения канала или действия с файлами на связанном сайте SharePoint за последние 28 дней.
 - **Люди с действиями электронной почты:** Количество пользователей, выполняющих какие – либо групповые действия с электронной почтой в рабочей области за последние 28 дней.
 - **Люди с действиями канала:** Число пользователей, отправляющих сообщения канала в рабочей области за последние 28 дней.
 - **Пользователи с действием "чтение контента":** Количество пользователей, которые обращаются к файлам Office со связанного сайта SharePoint за последние 28 дней.
 - **Пользователи с действием "создать контент":** Количество пользователей, которые создали, изменили или загрузили файлы Office со связанного сайта SharePoint.

## <a name="notes"></a>Заметки

- Все ссылки на общие рабочие области включают группы Microsoft 365 и связанные команды группы, сайт группы SharePoint и почтовый ящик группы.
- Только общие рабочие области с от 2 до 500 участников рассматриваются для работы в показателях продуктивности.
- Совместное использование измеряет задействование сотрудников в Организации. Она не&#39;захватывать действия внешних членов.
- Сообщения канала в команде "Совместная работа" не включают сообщения, отправленные через клиент Microsoft Teams Mobile.
- Различия между метриками электронной почты и сообщениями канала между категориями взаимодействия и работы, которые мы работаем над разрешениями, могут быть несущественными.
