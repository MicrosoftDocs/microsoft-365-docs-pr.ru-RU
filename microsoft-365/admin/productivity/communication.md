# <a name="communication--people-experiences"></a>Communication — People experiences

Оценка производительности поддерживает переход к цифровой трансформации с помощью информации о том, как ваша организация использует Microsoft 365 и технологические опытом, которые ее поддерживают. Оценка организации показывает измерения, отражающие результаты работы людей и технологий, и сравнивается с показателями эффективности организаций, похожих на вашу. Категория связи является частью мер, которые испытывают люди. Чтобы узнать больше, ознакомьтесь с обзором [показателей](productivity-score.md) производительности и ознакомьтесь [с заявлением о конфиденциальности Корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Предварительные требования

Чтобы начать работу с сведениями о связи, необходимо, чтобы у людей в организации была лицензия на:

- Microsoft Teams
- Yammer
- Exchange Online

Дополнительные сведения см. [в дополнительных сведениях о назначении лицензий пользователям.](../manage/assign-licenses-to-users.md)

После того как в течение последних 28 дней по крайней мере один раз в этих продуктах были активны люди, вы начнете видеть сведения.

## <a name="why-your-organizations-communication-score-matters"></a>Почему важна оценка связи в организации 

Корпорация Майкрософт понимает, что у людей разные потребности в общении. Чтобы получить быстрый ответ на вопрос, можно отправить мгновенное сообщение. Если вы хотите отправить руководству обновления состояния, вы можете выбрать сообщение электронной почты. Чтобы охватить более широкую аудиторию, вы можете опубликовать сообщение сообщества. Microsoft 365 эту гибкость в режимах связи, чтобы соответствовать потребностям каждого. Исследования показывают, что использование средств связи в режиме реального времени создает более единую организацию и создает моральный дух независимо от расположения.

## <a name="how-we-calculate-the-communication-score"></a>Как мы вычисляем оценку связи

Для связи мы предоставляем основные сведения, которые содержат ключевые метрики, связанные с коммуникацией в организации, в сочетании с каркасом скоринга для использования этих показателей для вычисления показателей организации.

### <a name="primary-insight"></a>Основная информация

Microsoft 365 предоставляет людям гибкость, чтобы соответствовать каждому стилю общения, предлагая несколько режимов. Основное представление предоставляет показатель людей в вашей организации, которые используют несколько режимов для общения.

:::image type="content" source="../../media/primaryinsight.png" alt-text="Основное представление о режимах связи.":::

1. **Заглавная:** Предоставляет ключевую метрику людей, использующих несколько режимов связи, а именно:

   - Отправка сообщений электронной почты Exchange
   - Отправка сообщений через Teams
   - Публикация в сообществах в Yammer

   Этот ключевой показатель показан в процентах от всех пользователей, использующих два или несколько Exchange, Teams или Yammer. 

2. **Body:** Дополнительные сведения о том, как гибкость в режимах связи является ценной для людей в вашей организации.

3. **Визуализация текущего состояния:**

   - Горизонтальная полоса, где синяя часть представляет процент, выраженный в заголовке
   - Моменты (числитель/знаменатель), используемый для вычисления процента, выраженного в загонах:
     - **Числитель**: # людей, использующих несколько режимов связи за последние 28 дней
     - **Знаменатель:**# людей, помеченных как включенные для более чем одного средства связи за последние 28 дней
   - **Значение peer Benchmark** для ключевой метрики также отображается в процентах

4. **Просмотр коммуникационных ресурсов:** Щелкнув здесь, вы завербовывайте все содержимое поддержки в виде видео/статей, связанных с коммуникацией. Из них можно перейти к пользовательскому списку воспроизведения для связи, а затем по всем категориям показателей производительности.

#### <a name="trend-visualization-of-primary-insight"></a>Визуализация тренда первичного анализа

 На этой диаграмме приведена тенденция цифратора ключевой метрики в основном анализе, т. е. число людей в вашей организации, использующих несколько режимов связи за последние 180 дней. Здесь это ежедневное значение — это совокупность количества людей, которые используют несколько форм коммуникации за последние 28 дней.

:::image type="content" source="../../media/trend-communication.png" alt-text="Диаграмма, на которую указывается число людей, которые используют несколько режимов для общения по сравнению со временем.":::

### <a name="scoring-model"></a>Модель скоринга

Оценка связи для организации измеряется на уровне сводной (организации) независимо от того, регулярно ли люди общаются с помощью нескольких режимов между сообщениями электронной почты, чата и сообщений сообщества в течение 28-дневного окна.  

Оценки не предоставляются на уровне отдельного пользователя.
