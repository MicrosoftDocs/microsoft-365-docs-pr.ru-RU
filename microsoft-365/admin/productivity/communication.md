# <a name="communication--people-experiences"></a>Связь — взаимодействие с пользователями

Оценка производительности поддерживает путешествие к цифровому преобразованию с подробными сведениями о том, как ваша организация использует Microsoft 365 и технологический технологический интерфейс, поддерживающий эту службу. Оценка вашей организации отражает измерения использования технологий для людей и технологий и может сравниваться с тестами в организациях, аналогичных их размеру. Категория "связь" является частью показателей взаимодействия с пользователями. Чтобы узнать больше, ознакомьтесь с [обзором оценки производительности](productivity-score.md) и ознакомьтесь с заявлением [Майкрософт о конфиденциальности](https://privacy.microsoft.com/privacystatement).

## <a name="why-your-organizations-communication-score-matters"></a>Почему показатель связи в Организации имеет значение 

Корпорация Майкрософт понимает, что люди имеют разные требования к общению. Чтобы быстро получить ответ на вопрос, вы можете отправить мгновенное сообщение. Если вы хотите отправлять обновления состояния руководителю, вы можете выбрать электронное сообщение. Чтобы получить доступ к более широкой аудитории, вы можете отправить сообщение сообщества. Microsoft 365 обеспечивает эту гибкость в режимах взаимодействия для удовлетворения потребностей всех пользователей. Исследование показывает, что использование средств связи в режиме реального времени создает более единую организацию и построения, независимо от расположения. [Посмотрите свидетельство](https://go.microsoft.com/fwlink/?linkid=2127669).

## <a name="how-we-calculate-the-communication-score"></a>Вычисление оценки связи

Для общения мы предоставляем основное представление, которое содержит основные метрики, связанные с обменом данными в Организации, в сочетании с инфраструктурой оценки, используемой для вычисления оценки Организации.

### <a name="primary-insight"></a>Основное представление

Microsoft 365 предоставляет пользователям гибкие возможности для соответствия стилю общения для всех пользователей, предлагая несколько режимов. Основной обзор предоставляет меру людей в Организации, которые используют несколько режимов для общения.

:::image type="content" source="../../media/primaryinsight.png" alt-text="Основное представление режимов связи.":::

1. **Верхний колонтитул:** Предоставляет ключевую метрику людей, использующих более одного режима связи, а именно:

   - Отправка электронных сообщений через Exchange
   - Отправка сообщений через Teams
   - Публикация в Yammer

   Эта ключевая метрика отображается в процентах от всех пользователей, использующих две или более Exchange, Teams или Yammer. 

2. **Основной текст:** В этой статье приводятся дополнительные сведения о том, как гибкость в режимах взаимодействия полезна для людей в вашей организации.

3. **Визуализация текущего состояния:**

   - Горизонтальная полоса, где синяя часть представляет процент, выраженный в заголовке
   - Выделяется (числитель/знаменатель), используемый для вычисления процентного соотношения, выраженного в заголовке:
     - **Числитель**: количество пользователей, использующих более одного режима связи за последние 28 дней
     - **Знаменатель**: количество пользователей, помеченных как включенные для нескольких продуктов связи в течение последних 28 дней
   - **Оценочное значение однорангового** показателя для ключевой метрики также отображается в процентах

4. **Просмотр ресурсов взаимодействия:** Щелкнув здесь, можно прикрепить все содержимое поддержки в виде видеороликов и статей, относящихся к общению. С помощью этих средств вы можете перейти к настраиваемому списку воспроизведения для общения, а затем для всех категорий оценки производительности.

#### <a name="trend-visualization-of-primary-insight"></a>Визуализация тенденций основной информации

 На этой диаграмме приводится тенденция числителя ключевого показателя в первичной информации, то есть числа людей в Организации, использующих более одного режима связи за последние 180 дней. Здесь значение ежедневно — это совокупность количества людей, использующих несколько форм связи за последние 28 дней.

:::image type="content" source="../../media/trend-communication.png" alt-text="Диаграмма, на которой показано количество людей, использующих более одного режима для общения и времени.":::

### <a name="scoring-model"></a>Модель оценки

Оценка связи для вашей организации измеряется на уровне совокупности (организации), независимо от того, будут ли пользователи в течение 28-дневного окна обмениваться данными с использованием нескольких режимов.  

Оценки не предоставляются на уровне отдельных пользователей.
