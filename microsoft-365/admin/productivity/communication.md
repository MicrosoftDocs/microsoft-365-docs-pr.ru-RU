# <a name="communication--people-experiences"></a>Связь — взаимодействие с людьми

Показатель производительности поддерживает переход к цифровому преобразованию с помощью статистики о том, как ваша организация использует Microsoft 365 и какие технологии ее поддерживают. Оценка организации показывает измерения, отражающие результаты работы людей и технологий, и сравнивается с показателями эффективности организаций, похожих на вашу. Категория связи является частью мер взаимодействия с людьми. Чтобы узнать больше, ознакомьтесь с обзором [оценки производительности](productivity-score.md) и ознакомьтесь с [заявлением о конфиденциальности корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Необходимые компоненты

Чтобы начать работу с информационной статистикой, у людей в вашей организации должна быть лицензия на:

- Microsoft Teams
- Yammer
- Exchange Online

Дополнительные сведения см. [в теме "Назначение лицензий пользователям".](../manage/assign-licenses-to-users.md)

После того как люди были активны в вышеуказанных продуктах хотя бы один раз за последние 28 дней, вы начнете видеть эти сведения.

## <a name="why-your-organizations-communication-score-matters"></a>Почему важен показатель коммуникаций в организации 

Корпорация Майкрософт понимает, что у людей разные потребности в связи. Чтобы получить быстрый ответ на вопрос, можно отправить мгновенное сообщение. Если вы хотите отправить обновления состояния руководству, вы можете выбрать сообщение электронной почты. Чтобы охватить более широкую аудиторию, вы можете опубликовать сообщение сообщества. Microsoft 365 обеспечивает эту гибкость в режимах связи, чтобы соответствовать всем требованиям. Исследования показывают, что использование средств связи в режиме реального времени создает более унифицированную организацию и создает унифицированную организацию независимо от расположения. [См. свидетельство.](https://go.microsoft.com/fwlink/?linkid=2127669)

## <a name="how-we-calculate-the-communication-score"></a>Как мы вычисляем показатель связи

Для связи мы предоставляем основную информацию, которая содержит ключевые метрики, связанные с взаимодействием в организации, в сочетании с структурой показателей для использования этих показателей для вычисления показателей организации.

### <a name="primary-insight"></a>Основные сведения

Microsoft 365 предоставляет людям гибкие возможности, чтобы соответствовать каждому стилю общения, предлагая несколько режимов. Основная информация предоставляет показатель людей в вашей организации, которые используют несколько режимов для общения.

:::image type="content" source="../../media/primaryinsight.png" alt-text="Основная информация о режимах связи.":::

1. **Заготовка:** Предоставляет ключевую метрику людей, использующих несколько режимов связи, а именно:

   - Отправка сообщений электронной почты через Exchange
   - Отправка сообщений через Teams
   - Публикация в сообществах в Yammer

   Этот ключевой показатель показывается в процентах от всех пользователей, использующих Exchange, Teams или Yammer или более. 

2. **Body:** Предоставляет дополнительные сведения о том, насколько гибкость в режимах общения является полезной для людей в вашей организации.

3. **Визуализация текущего состояния:**

   - Горизонтальная полоса, где синяя часть представляет процентное соотношение, выраженное в заголовке
   - Выделяет (числитель или деномилитор), используемый для вычисления процентного соотношения, выраженного в заголке:
     - **Числитель:**# людей, использующих несколько режимов связи за последние 28 дней
     - **Denominator**: # людей, помеченных как включенные для нескольких продуктов связи за последние 28 дней
   - **Одноранговая** контрольная точка ключевой метрики также показана в процентах

4. **Просмотр коммуникационных ресурсов:** Если щелкнуть здесь, будет зачислиться весь контент службы поддержки в виде видеороликов и статей, связанных с общением. Из них можно перейти к настраиваемом списку воспроизведения для связи, а затем для всех категорий показателей производительности.

#### <a name="trend-visualization-of-primary-insight"></a>Визуализация тенденций первичного анализа

 На этой диаграмме приведена тенденция числовых показателей ключевой метрики в основном анализе, т. е. количество людей в организации, использующих несколько режимов связи за последние 180 дней. Здесь это ежедневное значение — это совокупность людей, которые используют несколько форм общения за последние 28 дней.

:::image type="content" source="../../media/trend-communication.png" alt-text="Диаграмма, на которую указывается количество людей, которые используют несколько режимов для общения и времени.":::

### <a name="scoring-model"></a>Модель озвучки

Показатель коммуникации для вашей организации измеряется на общем уровне (организации), будут ли люди постоянно общаться с помощью нескольких режимов между электронной почтой, чатом и публикациями сообщества в течение 28-дневного окна.  

Оценки не предоставляются на уровне отдельных пользователей.

   - **Люди, отправляя сообщения электронной почты:** Цветная часть и доля представляют процент пользователей, которым включена exchange, отправляя сообщения электронной почты. Эта дробная часть построена из:
   
     - **Числитель:** люди, отправляя сообщения электронной почты за последние 28 дней.
     - **Denominator**: people enabled for Exchange in the last 28 days.
     
   - **Люди, отправляя сообщения в Microsoft Teams:** Цветная часть и дробная часть представляют процент пользователей, для которых включена отправка сообщений в Microsoft Teams. Эта дробная часть построена из:
   
     - **Numerator:** люди, отправляя сообщения в Microsoft Teams за последние 28 дней.
     - **Denominator**: люди, включенные в Microsoft Teams за последние 28 дней.
     
   - **Люди, публикуемые в сообществах:** Цветная часть и доля представляют процент пользователей, включенных в Yammer, которые размещаются в сообществах. Эта дробная часть построена из:
   
     - **Числитель:** Люди, публикующие сообщения в сообществах Yammer за последние 28 дней.
     - **Denominator:** Люди, которые включили Yammer за последние 28 дней.

### <a name="mentions-in-emails"></a>@mentions в сообщениях электронной почты

:::image type="content" source="../../media/emailmentions.png" alt-text="! [Сообщения электронной почты с упоминаниями.]":::

1. **Заготовка:** Выделяет увеличение скорости отклика для новых потоков электронной почты, запущенных за последние 28 дней, когда они содержат @mentions.

3. **Визуализация:** Разбивает скорость отклика для новых потоков электронной почты в зависимости от того, были ли они @mentions или нет:

   - Ответы на сообщения электронной почты с **\@ упоминаниями:** цветная часть и доля представляют скорость отклика для новых потоков электронной почты, содержащих @mentions. Эта дробная часть построена из:
   
     - **Числитель:** Новые потоки электронной почты @mentions, которые были запущены и получили ответ за последние 28 дней.
     - **Denominator:** Новые потоки электронной почты, @mentions, которые были запущены в течение последних 28 дней.
     
   - Ответы на сообщения электронной почты без **\@ упоминаний:** цветная часть и доля представляют скорость отклика для новых потоков электронной почты, содержащих @mentions. Эта дробная часть построена из:
   
     - **Числитель:** Новые потоки электронной почты, не содержащие @mentions, которые были запущены, а также получили ответ за последние 28 дней.
     - **Denominator:** Новые потоки электронной почты, @mentions которые были запущены в течение последних 28 дней.
