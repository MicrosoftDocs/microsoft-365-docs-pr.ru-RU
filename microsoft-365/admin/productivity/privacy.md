# <a name="privacy-controls-for-productivity-score"></a>Управление конфиденциальностью для оценки производительности

Оценка производительности позволяет получить подробные сведения о процессе цифрового преобразования в Организации с помощью Microsoft 365 и технологических интерфейсов, поддерживающих эту возможность.  Оценка вашей организации отражает измерения использования технологий для людей и технологий и может сравниваться со тестированиями в организациях, сходных с вашими. Для получения дополнительных сведений просмотрите [Обзор оценки производительности](productivity-score.md).

Конфиденциальность важна для Майкрософт. Сведения о том, как мы защищаем конфиденциальность, можно узнать в заявлении [корпорации Майкрософт о конфиденциальности](https://privacy.microsoft.com/privacystatement). Оценка производительности позволяет ИТ-администратору организации получить доступ к параметрам конфиденциальности, чтобы убедиться в том, что сведения о показателях производительности, которые вы хотите просмотреть, являются действиями, а не доверяются на то, что ваши Организации не доверяют вашей организации в корпорации Майкрософт.

В области взаимодействия с пользователями метрики доступны только на уровне Организации. В этой области рассматривается, как пользователи используют Microsoft 365, изучив категории совместной работы с контентом, мобильности, собраний, команд и общения. Мы разрешите несколько уровней элементов управления, чтобы обеспечить соответствие требованиям к внутренней политике конфиденциальности.
Элементы управления позволяют:

- Гибкие роли администраторов, позволяющие контролировать, кто может видеть информацию в показателях продуктивности.
- Возможность отказаться от использования области взаимодействия с пользователями.

## <a name="flexible-admin-roles-to-control-who-can-see-the-information-in-productivity-score"></a>Гибкие роли администраторов, позволяющие контролировать, кто может видеть информацию в показателях продуктивности.

Чтобы просмотреть весь показатель продуктивности, вам потребуется одна из следующих ролей администратора:

- Глобальный администратор
- администраторы Exchange;
- Администратор SharePoint
- Администратор Skype для бизнеса
- Администратор Teams
- Глобальный читатель
- Читатель отчетов
- Средство чтения сводки отчетов об использовании

Назначьте роль читателя Reports или сводный отчет об использовании всем, кто отвечает за управление изменениями и внедрение, но не обязательно является ИТ ИТ – администратором. Эта роль дает им доступ к полному показателю продуктивной работы в центре администрирования Microsoft 365.

Роль читателя сводки отчетов об использовании должна быть назначена командлетами PowerShell до тех пор, пока она не будет назначена в центре администрирования Майкрософт 365 позже в 2020.

Чтобы назначить роль читателя сводки отчетов об использовании с помощью PowerShell:

- Выполните следующую команду PowerShell:

```powershell
Connect-AzureAD
Enable-AzureADDirectoryRole -RoleTemplateId '75934031-6c7e-415a-99d7-48dbd49e875e'
$role=Get-AzureADDirectoryRole -Filter "roleTemplateId eq '75934031-6c7e-415a-99d7-48dbd49e875e'"
Get-AzureADDirectoryRoleMember -ObjectId $role.ObjectId
$u=Get-AzureADUser -ObjectId <user upn>
Add-AzureADDirectoryRoleMember -ObjectId $role.ObjectId -RefObjectId $u.ObjectId
```

## <a name="capability-to-opt-out-of-people-experiences"></a>Возможность отказаться от взаимодействия с пользователями

Вы также можете отказаться от использования оценки продуктивности людей. Если вы отказываетесь от вашей организации, никто из вашей организации не сможет просматривать эти метрики, а ваша организация будет удалена из любых вычислений, включающих связь, собрания, Командная работа, совместную работу с контентом и мобильность.

Чтобы выбрать команду "Добавить":

1. В центре администрирования откройте **раздел Параметры**   >   **Организации**, а затем на вкладке **службы** выберите **отчеты**.
2. Снимите флажок  **Разрешить использование данных об использовании Microsoft 365 для получения сведений о пользователях**. Чтобы узнать, как изменить параметры общего доступа к данным для аналитики конечной точки в диспетчере конфигураций Intune, выберите дополнительные **сведения**.
3. Нажмите кнопку  **сохранить**.
