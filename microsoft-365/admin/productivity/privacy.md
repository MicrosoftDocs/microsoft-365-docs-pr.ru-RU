# <a name="privacy-controls-for-productivity-score"></a>Средства контроля конфиденциальности для оценки производительности

Оценка производительности позволяет получить представление о цифровом преобразовании в вашей организации благодаря использованию Microsoft 365 и поддерживаемой технологии.  Оценка организации отражает показатели опыта людей и технологий и может сравниваться с контрольным показателем от организаций, похожих на ваш. Дополнительные сведения можно просмотреть в [обзоре оценки производительности.](productivity-score.md)

Конфиденциальность очень важна для Корпорации Майкрософт. Чтобы узнать, как мы защищаем вашу конфиденциальность, см. заявление [о конфиденциальности корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement) Оценка производительности предоставляет ИТ-администратору вашей организации доступ к настройкам конфиденциальности, чтобы убедиться, что просматриваемая вами информация о показателе производительности является действительной, не нарушая при этом доверие вашей организации в Корпорации Майкрософт.

В области работы с людьми метрики доступны только на уровне организации. В этой области посяг на использование Microsoft 365: категории совместной работы с контентом, мобильности, собраний, командной работы и общения. Мы можем использовать несколько уровней элементов управления, чтобы помочь вам удовлетворить ваши внутренние потребности в политике конфиденциальности.
Элементы управления дают вам:

- Гибкие роли администраторов для управления тем, кто может видеть информацию в оценке производительности.
- Возможность отказаться от области работы с людьми.

## <a name="flexible-admin-roles-to-control-who-can-see-the-information-in-productivity-score"></a>Гибкие роли администраторов для управления тем, кто может видеть информацию в оценке производительности

Чтобы просмотреть всю оценку производительности, вам потребуется одна из следующих ролей администратора:

- Глобальный администратор
- Администратор Exchange
- Администратор SharePoint
- Администратор Skype для бизнеса
- Администратор Teams
- Глобальный читатель
- Читатель отчетов
- Средство чтения сводки об использовании

Назначьте роль "Читатель отчетов" или "Читатель сводных отчетов об использовании" всем, кто несет ответственность за управление изменениями и внедрение, но не обязательно ИТ-администратору. Эта роль предоставляет им доступ к полному интерфейсу оценки производительности в Центре администрирования Microsoft 365.

Роль "Читатель сводных отчетов об использовании" должна быть назначена с помощью cmdlets PowerShell, пока она не станет доступной в Центре администрирования Microsoft 365 позже в 2020 г.

Чтобы назначить роль чтения сводных отчетов об использовании с помощью PowerShell:

- Запустите следующую powerShell:

```powershell
Connect-AzureAD
Enable-AzureADDirectoryRole -RoleTemplateId '75934031-6c7e-415a-99d7-48dbd49e875e'
$role=Get-AzureADDirectoryRole -Filter "roleTemplateId eq '75934031-6c7e-415a-99d7-48dbd49e875e'"
Get-AzureADDirectoryRoleMember -ObjectId $role.ObjectId
$u=Get-AzureADUser -ObjectId <user upn>
Add-AzureADDirectoryRoleMember -ObjectId $role.ObjectId -RefObjectId $u.ObjectId
```

## <a name="capability-to-opt-out-of-people-experiences"></a>Возможность отказаться от работы с людьми

Вы также можете отказаться от области работы с людьми в оценке производительности. В случае отказа никто из вашей организации не сможет просматривать эти показатели, и ваша организация будет удалена из всех вычислений, которые включают связь, собрания, командную работу, совместную работу с контентом и мобильность. Вы должны быть глобальным администратором, чтобы отказаться от отчетов о интерфейсах пользователей в вашей организации.

Чтобы выбрать put:

1. В Центре администрирования перейдите на вкладку "Параметры организации" и на вкладке "Службы"    >   выберите  **"Отчеты".**
2. Un-check the box that says  **Allow Microsoft 365 usage data to be used for people experiences insights**. Чтобы понять, как изменить параметры общего доступа к данным для Endpoint Analytics в диспетчере конфигураций Intune, выберите **дополнительные сведения.**
3. Выберите  **"Сохранить"**.
