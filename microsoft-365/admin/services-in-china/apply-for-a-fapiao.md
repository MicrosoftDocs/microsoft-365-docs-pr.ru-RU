# <a name="apply-for-a-fapiao-for-office-365-operated-by-21vianet"></a>Подача заявки на получение факса для Office 365 под управлением 21Vianet

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

Вы можете отправить запрос Фапьяо в систему управления 21Vianet Fapvia через три дня после оплаты. После отправки запроса Фапьяо он будет обработан в течение двух дней.
  
![Flowchart showing the Fapiao application process.](../../media/bf14884a-53f9-4c53-971c-b9b8ad6ec8d3.png)
  
## <a name="step-1-apply-for-a-fapiao"></a>Шаг 1. Подача заявки на фапьяо

Существует два способа применения к Фапьяо:
  
1. После оплаты на веб-сайте Office 365, который управляется компанией 21Vianet, вы получите сообщение с подтверждением заказа, которое содержит ваш номер заказа и ссылку на систему управления Фапьяо. Ссылку в сообщении электронной почты можно использовать для создания учетной записи в системе управления сведениями <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">компании 21Vianet Fapiao.</a>

    или

2. Вы можете подать заявку на получение факса из Центра <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">администрирования.</a>

    ![Страница "Счета" в Центре администрирования Microsoft 365.](../../media/a6e3b953-abd4-46aa-a910-08c517915a21.png)
  
## <a name="step-2-register-with-the-21vianet-fapiao-management-system"></a>Шаг 2. Регистрация в системе управления 21Vianet Fapvia

> [!NOTE]
> Вы можете использовать то же имя и пароль электронной почты, что и учетную запись Office 365, или использовать другое имя.
  
1. Перейдите в систему <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">управления сведениями компании 21Vianet Fapvia.</a>

2. В форме регистрации введите номер заказа, адрес электронной почты и пароль, а затем выберите **"Регистрация".**

    ![Страница регистрации системы управления 21Vianet Fapioa.](../../media/60d39184-95b2-4ea4-a8a2-3e11763bec87.png)
  
3. После завершения регистрации система отправит сообщение электронной почты для активации на ваш адрес электронной почты. Откройте сообщение электронной почты и выберите ссылку для активации учетной записи.

## <a name="step-3-submit-your-bill-for-a-fapiao"></a>Шаг 3. Отправка счета для фапьяо

1. Войдите в систему <a href="https://go.microsoft.com/fwlink/p/?linkid=837465" target="_blank">управления Фапьяо.</a>

2. Выберите запись вы выставления счета, а затем выберите **"Применить счет".**

    > [!NOTE]
    > Платежная система находится на стороне платформы и занимает три дня, чтобы синхронизировать заказ и платежную запись.
  
    ![Страница системы управления Фапьяо, на которую отображаются последние заказы.](../../media/b319767d-1d10-4cb4-b270-c5fbcee1368e.png)
  
3. Выберите тип Фапьяо, введите необходимые сведения, а затем выберите **"Далее".**

    ![Форма запроса Фапьяо, в которой вы выбираете тип Фапьяо.](../../media/56fe3db1-c20f-4082-a39d-02d7ac41fec8.png)
  
    > [!NOTE]
    > - Для обычного НДС Фапьяо необходимо ввести только имя покупателя.
    > - При необходимости вы можете подать заявку на фапьяо с другим названием. Однако вы можете применить только один заголовок Фапьяо для одного счета в системе. Если вы хотите разделить Фапьяо на разные суммы или названия, отправьте свой запрос <a href="https://portal.partner.microsoftonline.cn/Support/SupportOverview.aspx" target="_blank">в Центре администрирования.</a>
    > - При следующем подаче заявки на получение fap многофайла система автоматически представляет предыдущие сведения о Фапьяо.
    > - Если вам нужен сертификат покупки или возврат денежных средств, имя платетель и название фапьяо должны совпадать.

4. Выберите способ доставки и введите сведения о почте. Вы можете выбрать Y и SF (сбор сборов веских данных). Вы также можете перейти в филиал 21Vianet в Шанхае, чтобы получить фапьяо. select **Next**.

    ![Форма запроса Фапьяо, в которой отображаются сведения о рассылке.](../../media/bba500b4-a51d-477b-81a7-9113b08d39f1.png)
  
    **Адрес компании филиала "21Vianet в Шанхае":**

    В Кейуан-Ноуан-Кейуан-Кейуан в Нью-Йорке. 88, Немецкий центр, здание 3, 657

    **Контактные данные:**

    Zhu qin qin 021-28986102

5. Убедитесь, что сведения верны, а затем выберите **"Подтвердить".**

    ![Страница подтверждения формы запроса Фапьяо.](../../media/18706d9d-defc-4285-8fd3-990448b44a18.png)
  
## <a name="step-4-check-application-progress"></a>Шаг 4. Проверка хода выполнения приложения

Приложение автоматически передается в Центр обслуживания 21Vianet и будет завершено в течение двух рабочих дней.
  
После отправки приложения вы можете проверить ход выполнения в любое время. Мы обновим состояние вашего приложения с помощью таких примечай, как **"Выдано"** или **"Отправлено".**
  
![Страница системы управления 21Vianet Fapvia, на которую отображается состояние запроса.](../../media/6cd696ec-d630-4fce-9f27-935a0d5f0ebe.png)
  
Когда Фапьяо не используется, система отображает уведомление и указывает ожидаемое время для выдачи Фапьяо.
  
![Сообщение об ошибке отображается, когда система управления 21Vianet Fapvia не используется.](../../media/effe0796-83aa-4a91-a488-15d6f58c01dc.png)
  
## <a name="faqs"></a>Вопросы и ответы

### <a name="what-services-can-i-get-from-online-support"></a>Какие службы можно получить из поддержки в Интернете?

Вы можете проверить ход выполнения запроса Фапьяо и узнать, почему вы не получили фапьяо.
  
Если вы хотите изменить заголовок Фапьяо, отправьте нам фапьяо, и мы переименуем Фапьяо. Вы можете отправить запрос через <a href="https://portal.partner.microsoftonline.cn/Support/SupportOverview.aspx" target="_blank">Центр администрирования.</a>
  
![Форма поддержки Центра администрирования, используемая для отправки фапьяо запроса на изменение названия.](../../media/2a413e9e-f30b-4f26-adbf-6287cc217a0f.png)
  
### <a name="how-do-i-change-my-login-email-address-and-password"></a>Как изменить адрес электронной почты и пароль для входа?

1. Войдите в систему управления <a href="https://go.microsoft.com/fwlink/p/?linkid=837465" target="_blank">Фапьяо</a>и выберите **"Изменить адрес электронной почты".**

    ![В выпадаемом меню после входа пользователя в систему управления 21Vianet Fapvia.](../../media/ee6de24b-6be2-41e6-8aec-e0c3cb0ea35e.png)
  
2. Если вы забыли пароль для входа, для сброса пароля можно использовать адрес электронной почты для входа. Система управления Фапьяо отправит новый пароль на ваш адрес электронной почты. Для входа можно использовать новый пароль.

    ![Последовательное изображение сброса пароля в системе управления 21Vianet Fapvia.](../../media/2edb0a47-1286-4792-804d-7e84534c8370.png)
  
3. Если вы забыли свой адрес электронной почты для входа, обратитесь в службу поддержки клиентов 21Vianet по адресу (86) 400-089-0365.

### <a name="how-do-i-find-my-order-id"></a>Как найти свой ИД заказа?

1. В Центре [администрирования перейдите](https://go.microsoft.com/fwlink/p/?linkid=850627)на страницу **"Выставление счетов** \> **& платежей".**

2. Найдите нужный счет, выберите для просмотра или скачайте PDF-файл.

### <a name="what-if-i-enter-the-wrong-email-address-when-i-register"></a>Что делать, если при регистрации я введите неправильный адрес электронной почты?

Если вы введите неправильный адрес электронной почты при регистрации, вы не получите сообщение об активации. Срок действия ссылки на регистрацию в сообщении автоматически истекает через 24 часа. Вы можете вернуться на страницу <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">регистрации и</a> снова зарегистрироваться с правильным адресом электронной почты.
  
### <a name="what-if-i-dont-receive-an-activation-email"></a>Что делать, если я не получаю сообщение об активации?

Если вы не получили сообщение об активации учетной записи в течение 24 часов после регистрации, перейдите в систему управления сведениями <a href="https://go.microsoft.com/fwlink/p/?linkid=837466" target="_blank">компании 21Vianet Fapvia,</a>введите свой адрес электронной почты, а затем выберите "Отправить повторно".  Система будет повторно отправлять электронное письмо для активации учетной записи на зарегистрированный адрес электронной почты.
  
![Страница регистрации системы управления 21Vianet Fapioa.](../../media/60d39184-95b2-4ea4-a8a2-3e11763bec87.png)
  
Если вы по-прежнему не получили сообщение об активации, обратитесь в службу поддержки клиентов 21Vianet по электронной почте (86) 400-089-0365.
