# <a name="office-app-for-ios-for-office-365-operated-by-21vianet"></a>Приложение Office для iOS для Office 365 под управлением 21Vianet

Приложение Microsoft Office для iOS объединяет мобильные приложения Word, Excel и PowerPoint в одно приложение, доступное для скачивания на телефоны iOS. С помощью приложения Office для iOS вы можете подключаться к Office 365 так же, как и с мобильными приложениями Word, Excel и PowerPoint. Скачивание приложения Office для iOS не повлияет на существующие установки Word, Excel и PowerPoint.

Некоторые функции приложения Office для iOS недоступны для клиентов Office 365 под управлением 21Vianet:

- Изображение в текст, а изображение в таблицу 
- Преобразование фотографий в документы Word из Lens 
- Действие передачи файлов 
- Заметки остаются локальными и не синхронизируются с сервером
- Предварительный просмотр ссылок в QR проверки
- Карточки живых людей в разделе "Я"
- Классификация, маркировка и защита (CLP)


## <a name="download-the-office-app-for-ios"></a>Скачивание приложения Office для iOS

- Скачайте приложение Office для iPhone из [Магазина приложений.](https://products.office.com/mobile/office?rtc=2) 

> [!NOTE]
> Приложение Office для iOS в настоящее время доступно только для iPhone. Поддержка iPad будет добавлена позже. 

## <a name="security-considerations"></a>Рекомендации по безопасности

Если ваша организация переназначает приложения на мобильные устройства сотрудников, мы рекомендуем заменить приложения Word, Excel и PowerPoint приложением Office для iOS.
