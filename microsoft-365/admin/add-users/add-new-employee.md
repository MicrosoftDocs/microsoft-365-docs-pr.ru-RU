# <a name="add-a-new-employee-to-microsoft-365"></a>Добавление нового сотрудника в Microsoft 365

В этой статье вы можете помочь новому сотруднику Microsoft 365 для бизнеса. Мы предполагаем, что вы администратор, [](../setup/setup.md)и вы уже Microsoft 365, и теперь у вас есть кто-то новый присоединение к вашей компании.
  
Вы в правильном месте, если вашему новому сотруднику требуется Microsoft 365, и вы используете план [Microsoft 365,](https://products.office.com/business/compare-office-365-for-business-plans) который позволяет установить Office приложения, такие как Word и Excel на компьютере. 
  
 **Вы не являетесь администратором?** [Узнайте, как Microsoft 365](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) помогает бизнесу и домашним пользователям с настройками. 
  
 **Нет Office в вашем плане?** Следуйте ниже шагам, но пропустите разделы для установки приложений. Вместо этого используйте сетевые [версии Office.](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) 
  
Ниже представлен краткий обзор.

|[Шаг 2. Дайте сотруднику свой пользовательский код и пароль](#step-2-give-the-employee-their-user-id-and-password) <br/> |При создании учетной записи вы получите ID и пароль, которые вы можете передать сотруднику, чтобы они могли войти.  <br/> |
|[Шаг 3. Разъяснить, где войти](#step-3-explain-where-to-sign-in) <br/> |Знак в расположении [https://www.office.com](https://www.office.com) <br/> |

## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Шаг 1. Создание учетной записи Microsoft 365 для сотрудника


Дополнительные инструкции см. [в добавлении пользователей и](add-users.md)одновременном назначении лицензий. При настройках нового сотрудника можно отправить сведения о входе в личную учетную запись сотрудника. Таким образом, они получат электронное письмо от Microsoft Online Service Team, в которых рассказывается, как войти в систему Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Шаг 2. Дайте сотруднику свой пользовательский код и пароль


Если вы не отправили его на личный адрес электронной почты, распечатайте подпись сотрудника с именем и паролем и передайте его им. Или сообщите им информацию по телефону.
  
Так как у них еще нет доступа к электронной почте, не отправляйте информацию на этот адрес электронной почты.
  
## <a name="step-3-explain-where-to-sign-in"></a>Шаг 3. Разъяснить, где войти 


Так же, как Facebook, Amazon или Gmail, ваш сотрудник впишется в Microsoft 365. Дайте им следующий знак в сведениях:
  
- Вход в [https://www.office.com](https://www.office.com) .
    
- Выберите **вход,** а затем введите код пользователя и пароль.
    
## <a name="step-4-help-your-employee-get-started"></a>Шаг 4. Помогите сотруднику приступить к работе
