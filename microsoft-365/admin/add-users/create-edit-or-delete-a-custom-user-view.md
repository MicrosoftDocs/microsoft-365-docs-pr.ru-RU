# <a name="create-edit-or-delete-a-custom-user-view"></a>Создание, изменение и удаление настраиваемых представлений пользователей

Если вы глобальный администратор подписки на Microsoft 365 для бизнеса или администратор управления пользователями, вы можете создавать настраиваемые представления пользователей для просмотра определенного подмножества пользователей. Эти представления являются дополнением к стандартному набору представлений. Настраиваемые представления пользователей можно создавать, изменять и удалять, а доступны они всем администраторам.
  
## <a name="custom-user-views-in-the-admin-center"></a>Настраиваемые представления пользователей в Центре администрирования

::: moniker range="o365-worldwide"

При создании, редактировании или удалении пользовательского представления изменения  будут показаны в списке фильтров, который все администраторы компании видят при просмотре на странице "Пользователи".  Можно создать до 50 настраиваемых представлений. 

::: moniker-end

::: moniker range="o365-germany"

При создании, редактировании или удалении пользовательского представления изменения  будут показаны в списке представлений, который все администраторы компании видят при просмотре на странице **"Пользователи".** Можно создать до 50 настраиваемых представлений. 

::: moniker-end

::: moniker range="o365-21vianet"

При создании, редактировании или удалении пользовательского представления изменения  будут показаны в списке представлений, который все администраторы компании видят при просмотре на странице **"Пользователи".** Можно создать до 50 настраиваемых представлений. 

::: moniker-end

> [!TIP]
>  Стандартные представления пользователей по умолчанию  отображаются в выпадаемом списке фильтров. Стандартные фильтры **включают** всех пользователей, лицензированных пользователей, гостевых пользователей, разрешенный вход, блокировку входов, пользователей без лицензий, пользователей с ошибками, администраторов вычетов, глобальных администраторов, администраторов службы, администраторов служб и администраторов управления пользователями.           Изменить или удалить стандартные представления нельзя. 

Ниже приведены другие замечания о стандартных представлениях. 

- В некоторых стандартных представлениях сортировка списков, содержащих более 2000 пользователей, не выполняется. Чтобы найти конкретного пользователя в таком списке, используйте поле поиска. 
- Если вы не приобретли Microsoft 365 у корпорации **Майкрософт,** администраторы вы выставления счета не будут отображаться в стандартном списке представлений. Дополнительные сведения см. в [теме "Назначение ролей администратора".](assign-admin-roles.md) 
  
## <a name="choose-the-filters-for-your-custom-user-view"></a>Выбор фильтров для настраиваемого представления пользователей

Настраиваемые представления можно создавать и редактировать на **настраиваемой области фильтров.** Если выбрать несколько вариантов фильтрации, в результаты будут включены пользователи, соответствующие всем условиям. В следующем примере показано, как создать настраиваемое представление "Пользователи из Канады", в котором показаны все пользователи определенного домена, находящиеся в Канаде. 

  
 **A — домен** Если в вашей организации несколько доменов, вы можете выбрать один из доступных доменов в выпадаемом списке. 
  
 **Б. Состояние для входов** Выберите пользователей, которые разрешены или заблокированы. 
  
 **C — расположение** Выберите расположение в выпадаемом списке стран. 
  
 **Г— назначенная лицензия на продукт** Выберите в выпадаемом списке лицензии, доступные в вашей организации. При выборе этого фильтра отображаются пользователи, которым назначена выбранная лицензия. Отображаемые пользователи могут также иметь и другие лицензии. 
  
Вы также можете отфильтровать пользователей по дополнительным сведениям профиля, используемым в вашей организации, таким как отдел, город, область или край, должность, страна или регион.
  
 **Другие условия**:
  
- **Только синхронизированные пользователи**: при выборе этого параметра отображаются все пользователи, синхронизированные с локальной службой Active Directory независимо от того, были ли они активированы. 
    
- **Пользователи с ошибками**: при выборе этого параметра отображаются пользователи, при подготовке которых возникли ошибки. 
    
- **Пользователи без лицензий**: при выборе этого параметра отображаются все пользователи, которым не назначены лицензии. В результаты этого представления также могут включаться пользователи, которым предоставлен почтовый ящик Exchange, но не назначена лицензия. Чтобы отслеживать таких пользователей отдельно, используйте фильтр **Пользователи без лицензий с почтовыми ящиками или архивами Exchange**. В результаты этого представления также могут включаться пользователи, у которых есть архив Exchange, но нет лицензии.
    
- **Пользователи без лицензий с почтовыми ящиками или архивами Exchange** Выберите это поле, чтобы показать учетные записи пользователей, которые были созданы в Exchange Online и имеют почтовый ящик Exchange, но не были назначены лицензии На Microsoft 365. Результаты применения этого фильтра включают пользователей, имеющих архив Exchange. 

> [!NOTE]
> Пользователи **без лицензий с фильтром почтовых** ящиков Exchange работают, когда:
1. Почтовый ящик был недавно преобразован  из общего в пользовательский **и** у него нет лицензии.
2. Почтовый ящик был недавно перенесен в Microsoft 365, но лицензия не назначена.
3. Почтовый ящик создан с помощью PowerShell и лицензия не назначена.
4. Для пользователя создается новый почтовый ящик, созданный локально New-RemoteMailbox с помощью New-RemoteMailbox.
    
> [!TIP]
> В настраиваемых представлениях, возвращающих более 2 000 пользователей, сортировка списка результатов не выполняется. В этом случае используйте поле поиска, чтобы найти пользователей, или отредактировать настраиваемые представления для уточнения поиска. 
  
## <a name="create-a-custom-user-view"></a>Создание пользовательского представления пользователя

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">"Активные пользователи".</a>
    
2. На странице **"Активные пользователи"** выберите **"Фильтры"** и **"Новый фильтр".**
  
3. На странице **"Настраиваемый фильтр"** введите имя фильтра, выберите условия для пользовательского фильтра и выберите **"Добавить".** Настраиваемые представления теперь включены в выпадаемый список фильтров.
    
::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">"Активные пользователи".</a>  

2. На странице **"Активные пользователи"** выберите **"Представления"** и выберите **"Добавить настраиваемые представления".**
  
3. На странице **"Настраиваемые представления"** введите имя фильтра, выберите условия для настраиваемного фильтра и выберите **"Добавить".** Настраиваемые представления теперь включены в выпадаемый список фильтров.

::: moniker-end


::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">"Активные пользователи".</a> 

2. На странице **"Активные пользователи"** выберите **"Представления"** и выберите **"Добавить настраиваемые представления".**
  
3. На странице **"Настраиваемые представления"** введите имя фильтра, выберите условия для настраиваемного фильтра и выберите **"Добавить".** Настраиваемые представления теперь включены в выпадаемый список фильтров.

::: moniker-end
    

## <a name="edit-or-delete-a-custom-user-view"></a>Изменение или удаление пользовательского представления

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">"Активные пользователи".</a>
    
2. На странице **"Активные пользователи"** выберите **"Фильтр",** выберите фильтр, который нужно изменить, а затем выберите **"Изменить фильтр".** 
    
    > [!TIP]
    > Вы можете изменять только настраиваемые представления. 
  
3. На странице **"Настраиваемый фильтр"** отредактируете нужные сведения и выберите **"Сохранить".** Или, чтобы удалить фильтр, в нижней части страницы выберите **"Удалить".** 
    
::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">"Активные пользователи".</a>  

2. На странице **"Активные пользователи"** выберите **"Представления",** выберите фильтр, который нужно изменить, а затем выберите "Изменить **это представление".** 
    
    > [!TIP]
    > Вы можете изменять только настраиваемые представления. 
  
3. На странице **"Настраиваемые представления"** отредактируете нужные сведения и выберите **"Сохранить".** Чтобы удалить фильтр, в нижней части страницы выберите **"Удалить настраиваемые представления".** 

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите к **пользователям** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">"Активные пользователи".</a> 

2. На странице **"Активные пользователи"** выберите **"Представления",** выберите фильтр, который нужно изменить, а затем выберите "Изменить **это представление".** 
    
    > [!TIP]
    > Вы можете изменять только настраиваемые представления. 
  
3. На странице **"Настраиваемые представления"** отредактируете нужные сведения и выберите **"Сохранить".** Или, чтобы удалить фильтр, в нижней части страницы выберите **"Удалить настраиваемые представления".** 

::: moniker-end
