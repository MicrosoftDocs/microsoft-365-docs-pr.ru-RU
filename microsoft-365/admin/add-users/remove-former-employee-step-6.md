# <a name="step-6---remove-the-microsoft-365-license-from-a-former-employee"></a>Шаг 6 . Удаление Microsoft 365 лицензии у бывшего сотрудника

Если вы не хотите платить за лицензию после того, как кто-то покинет организацию, необходимо удалить Microsoft 365 лицензию и удалить ее из подписки. Вы можете назначить лицензию другому пользователю, если ее не удалить.
  
После освобождения лицензии все данные этого пользователя будут храниться в течение 30 дней. Вы сможете [получить доступ](get-access-to-and-back-up-a-former-user-s-data.md) к данным или [восстановить](restore-user.md) учетную запись, если пользователь вернется. Через 30 дней все данные пользователя (за исключением документов, хранимых в SharePoint Online) удаляются из Microsoft 365 и не могут быть восстановлены.

1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.
2. Выберите имя сотрудника, которого необходимо заблокировать, а затем выберите вкладку **Лицензии и Приложения.**
3. Снимите флажки для необходимой лицензии и выберите **сохранить изменения.**

**Чтобы уменьшить количество лицензий,** за которые вы платите, пока не наймете другого человека, сделайте следующие действия:

1. В центре администрирования перейдите на страницу **Выставление** платы за продукты \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> и выберите **вкладку Продукты.**
2. Выберите подписку, из которой необходимо удалить лицензии.
3. На странице подробные сведения выберите **Удаление лицензий.**
4. В области **Удаление лицензий** в поле "Новое количество" в поле **"Общие** лицензии" введите общее количество лицензий, которые необходимо получить для этой подписки. Например, если у вас есть 25 лицензий и вы хотите удалить одну из них, введите 24.
5. Нажмите **Сохранить**.

Когда вы [добавите](add-users.md) другого человека в свой бизнес, вам будет предложено купить лицензию в то же время, с одним шагом!

Дополнительные сведения об управлении пользовательскими лицензиями для Microsoft 365 для бизнеса см. в рублях Назначение лицензий пользователям в [Microsoft 365](../manage/assign-licenses-to-users.md)для бизнеса и [unassign лицензий](../manage/remove-licenses-from-users.md)от пользователей в Microsoft 365 для бизнеса .
  
## <a name="how-the-deleted-employee-account-affects-skype-for-business"></a>Как удаление учетной записи сотрудника влияет на Skype для бизнеса

Когда вы удаляете лицензию пользователя в Office 365, освобождается связанный с ним вызывающий номер ТСОП. Вы можете назначить его другому пользователю.
  
Если пользователь относится к группе очереди, он станет недоступен в качестве цели для агентов, обрабатывающих очереди звонков. Поэтому мы рекомендуем также удалять пользователя из групп, связанных с очередью звонков.

## <a name="set-up-call-forwarding-to-people-in-your-organization"></a>Настройка переададки вызовов для людей в организации

Если вам необходимо настроить переададку вызовов для номера телефона уволенного сотрудника, параметр переадстройки вызовов в соответствии с политиками вызова может настроить переадстроение, в котором входящие вызовы могут быть переадрены другим пользователям или одновременно звонить другому лицу. Дополнительные сведения см. в [Microsoft Teams.](/microsoftteams/teams-calling-policy)
