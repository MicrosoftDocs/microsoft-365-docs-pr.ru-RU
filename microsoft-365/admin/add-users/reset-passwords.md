# <a name="reset-passwords"></a>Сброс паролей

В этой статье рассказывается, как сбросить пароли для себя и для пользователей, если у вас есть Microsoft 365 для бизнеса подписки.

## <a name="before-you-begin"></a>Прежде чем начать

Эта статья адресована тем, кто устанавливает политику срока действия паролей в компании, учебном заведении или некоммерческой организации. Чтобы выполнить эти действия, вам нужно войти с помощью своей учетной записи администратора Microsoft 365. [Что такое учетная запись администратора?](../../business-video/admin-center-overview.md)

Для выполнения этих [действий](about-admin-roles.md) необходимо быть глобальным администратором или администратором паролей.

## <a name="steps-reset-a-business-password-for-a-user"></a>Действия: сброс бизнес-пароля для пользователя

1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

2. На странице **Активные пользователи** выберите пользователя и выберите **пароль Reset**.

3. Следуйте инструкциям на странице **Сброс** пароля, чтобы автоматически создать новый пароль для пользователя или создать его для них, а затем выберите **Сброс**.  

4. Введите адрес электронной почты, который пользователь может получить для получения нового пароля, и выполните его, чтобы убедиться, что он получил его.

## <a name="let-users-reset-their-own-passwords"></a>Предоставление пользователям прав на самостоятельный сброс пароля

Мы настоятельно рекомендуем вам настроить самостоятельный сброс пароля. В таком случае вам не придется сбрасывать пароли вручную для всех пользователей в организации. Сведения о том, как это сделать, см. в статье [Предоставление пользователям прав на самостоятельный сброс пароля в Office 365](let-users-reset-passwords.md).

## <a name="reset-my-admin-password"></a>Сброс пароля администратора

Используйте эти действия, если вы забыли пароль, но можете войти в Microsoft 365, так как, например, пароль сохранен в браузере:

1. Выберите имя (значок) в правом верхнем углу > **личной** информации моей  >  **учетной записи**.

2. В **соответствии с сведениями**  о контакте убедитесь, что ваша альтернативная электронная почта является точной и что вы предоставили номер мобильного телефона. Если нет, измените их сейчас.

3. Выпишите: выберите свое имя в правом верхнем \> **углу.**

4. Теперь введите еще раз: введите имя пользователя \> **Далее,** \> а затем выберите пароль **Забыли**.

5. Выполните действия мастера по сбросу пароля. Он использует альтернативные контактные данные для проверки правильного сброса пароля.

Если вы забыли пароль и не можете войти:

- Попросите другого глобального администратора в вашем бизнесе сбросить пароль для вас.

- Убедитесь, что вы предоставили альтернативные контактные данные, включая номер мобильного телефона.

- Или [позвоните в службу поддержки Майкрософт.](../../business-video/get-help-support.md)

## <a name="reset-all-business-passwords-for-everyone-in-your-organization-at-the-same-time"></a>Сброс всех бизнес-паролей для всех в организации одновременно
<a name="bkmk_forgot"> </a>

Эти действия работают для бизнеса с десятками пользователей. Если у вас сотни или тысячи пользователей, см. в следующем разделе об сбросе паролей оптом (не более 40 пользователей одновременно).
  
1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

2. Выберите параметр рядом с **отображением имени,** чтобы выбрать всех в вашем бизнесе. Затем снимите флажок напротив себя. Собственный пароль невозможно сбросить одновременно со всеми остальными пользователями.

3. Выберите **пароль Reset**. 

4. Следуйте инструкциям на странице **Сброс пароля** и выберите **Сброс**.  Если вы выбрали автоматическое создание паролей, будут отображаться новые временные пароли.

5. Введите адрес электронной почты, где можно получить временные пароли. Необходимо уведомить пользователей о временных паролях.
