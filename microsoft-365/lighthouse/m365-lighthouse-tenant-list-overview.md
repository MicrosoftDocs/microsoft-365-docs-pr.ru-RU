# <a name="microsoft-365-lighthouse-tenant-list-overview"></a>Microsoft 365 Lighthouse список клиентов

> [!NOTE]
> Функции, описанные в этой статье, подлежат изменениям и доступны только партнерам, которые соответствуют [требованиям.](m365-lighthouse-requirements.md) Если в организации нет Microsoft 365 Lighthouse, [см.](m365-lighthouse-sign-up.md)в Microsoft 365 Lighthouse.

Список Microsoft 365 Lighthouse предоставляет сведения о различных клиентах, с которых заключен контракт, в том числе о состоянии onboarding клиента относительно Microsoft 365 Lighthouse. Кроме того, в списке клиентов можно пометить клиентов для предоставления различных фильтров во Microsoft 365 Lighthouse и сверлить, чтобы узнать больше о том или ином клиенте и состоянии плана развертывания.

После того как ваши клиенты [Microsoft 365 Lighthouse](m365-lighthouse-requirements.md)требованиям к вовсю, их состояние будет показываться как **Active** в списке клиента.

Чтобы получить доступ к списку Microsoft 365 Lighthouse, выберите Клиенты в левой области навигации, чтобы открыть страницу Tenants. 

## <a name="tenant-status"></a>Состояние клиента

В следующей таблице показаны различные сообщения о состоянии и их значение.<br><br>

| Сообщение о состоянии | Описание |
|--|--|
| Активное | Запущена веха и поток данных. |
| В процессе | Клиент обнаружен, но не полностью на борту. |
| Недопустимый, DAP | Требуется делегированная настройка привилегий администратора (DAP). |
| Неудобная, количество пользователей | У клиента больше пользователей, чем разрешено. |
| Неугомонный, лицензионный | У клиента нет необходимой лицензии. |
| Неактивный | Клиент больше не активен. |

После бездействия клиента вы не можете принять меры к арендатору, Microsoft 365 Lighthouse завершить процесс неактивности. Для завершения неактивности может потребоваться до 48 часов.

При повторной активности клиента может потребоваться до 48 часов для повторного получения данных.

## <a name="tenant-tags"></a>Теги клиента

Вы можете пометить клиентов с помощью настраиваемой метки в Microsoft 365 Lighthouse. Эти теги можно использовать для организации клиентов, а также помогут легко отфильтровать существующие представления и сведения, доступные для соответствующих наборов клиентов. Вы также можете управлять тегами и их клиентами со страницы "Клиенты".
