# <a name="create-a-communications-site"></a>Создание информационного сайта

Отличный способ сообщить о приоритетах, поделиться стратегическими документами и выделить предстоящие события — это использовать сайт связи в SharePoint. Сайты коммуникаций для общего доступа к данным в целом по всему бизнесу или кампании; это ваш внутренний сайт стратегии.

## <a name="best-practices"></a>Рекомендации

Включите следующие элементы на сайте Communications:

1. Добавьте логотип и цвета в качестве изображения и темы для загона.
2. Руководство с помощью стратегии, сообщения, важных документов, каталога и задаваемой темы в веб-части **Hero.**
3. Включай заявление генерального директора или кандидата в команду в **веб-части Text.**
4. Добавьте события в **веб-часть События,** чтобы все могли видеть, что происходит.
5. Добавление фотографий, которые пользователи могут использовать или делиться с веб-частью **галереи изображений.**

## <a name="set-it-up"></a>Настройка

1. Вход в https://Office.com.
2. В левом верхнем углу страницы выберите значок запуска приложения,  а затем выберите SharePoint плитку. Если вы не видите  плитку SharePoint, щелкните плитку  **Sites** или All, если SharePoint не отображается.
3. В верхней части домашней SharePoint нажмите **кнопку + Создайте** сайт и выберите вариант **сайта** Communication.

Узнайте все [о сайтах коммуникаций](https://support.office.com/article/What-is-a-SharePoint-communication-site-94A33429-E580-45C3-A090-5512A8070732) и о создании сайта связи [в SharePoint Online](https://support.microsoft.com/en-us/office/create-a-communication-site-in-sharepoint-online-7fb44b20-a72f-4d2c-9173-fc8f59ba50eb).

## <a name="admin-settings"></a>Параметры администратора

Если вы не видите ссылку **+Create** site, создание сайта самообслуживки может быть недоступным в Microsoft 365. Чтобы создать сайт группы, свяжитесь с лицом, Microsoft 365 в вашей организации. Если вы администратор Microsoft 365, см. в SharePoint [Online](/sharepoint/manage-site-creation) создание сайта самообслуживки для [](/sharepoint/manage-sites-in-new-admin-center) организации или управление сайтами в новом центре администрирования SharePoint для создания сайта из центра администрирования SharePoint Online.
