# <a name="customize-your-sign-in-page-with-a-privacy-and-consent-notice"></a>Настройка страницы для входов с уведомлением о конфиденциальности и согласии

Ваша бизнес-кампания может упростить для правоохранительных органов подачу судебных исков в отношении интернет-злоумышленников, добавив уведомление о конфиденциальности и согласии на вход.

Вы можете настроить страницу для входов с помощью фирменой настройки. Вы также можете добавить текст, который поможет пользователям в входе или указать юридические требования или ограничения для получения доступа к ресурсам Microsoft 365.

## <a name="design-customization-the-text-on-your-sign-in-page"></a>Настройка текста на странице для регистрации

Чтобы обновить настраиваемые элементы на странице для входов, необходимо быть глобальным администратором. Конкретные инструкции см. в статье [о добавлении фирменой марки](https://docs.microsoft.com/azure/active-directory/fundamentals/customize-branding) компании.

Элементы, которые можно обновить:

- **Текст страницы для входов** Простое место для добавления заявления о конфиденциальности и согласия.
- Фоновое изображение страницы для входов
- Логотип баннера
- Подсказка имени пользователя

Примеры уведомлений о конфиденциальности и согласии см. в приложении А в статье ["Поиск](https://www.justice.gov/sites/default/files/criminal-ccips/legacy/2015/01/14/ssmanual2009.pdf)и засеяние компьютеров" и "Получение электронных свидетельств в расследованиях по делу о судимости".
