### <a name="stop-log-shipping"></a><span data-ttu-id="adf2a-383">Остановка доставки журналов</span><span class="sxs-lookup"><span data-stu-id="adf2a-383">Stop log shipping</span></span>

<span data-ttu-id="adf2a-p149">Перед восстановлением фермы необходимо остановить доставку журналов. Сначала сделайте это на вспомогательном сервере в Azure, а затем — на основном локальном сервере.
