# <a name="microsoft-365-endpoints"></a>Конечные точки Microsoft 365

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

Конечные точки — это набор IP-адресов назначения, доменных имен DNS и URL-адресов для Microsoft 365 трафика в Интернете. 

Чтобы оптимизировать производительность Microsoft 365 облачных служб, эти конечные точки нуждаются в специальной обработке в клиентских браузерах и устройствах в вашей краевой сети. Эти устройства включают брандмауэры, SSL Break и Inspect и устройства проверки пакетов, а также системы предотвращения потери данных.

Сведения [см. в Microsoft 365 управлении конечными точками.](managing-office-365-endpoints.md)

В настоящее время существует пять различных Microsoft 365 облаков. В этой таблице вы можете найти список конечных точек для каждой из них.

| Облако | Описание |
|:-------|:-----|
| [Глобальные конечные точки](urls-and-ip-address-ranges.md) | Конечные точки для Microsoft 365 по всему миру, в том числе облако сообщества для государственных организаций (GCC). |
| [Конечные точки U.S. Government DoD](microsoft-365-u-s-government-dod-endpoints.md) | Конечные точки для подписок Министерства обороны США (DoD). |
| [Конечные точки U.S. Government GCC High](microsoft-365-u-s-government-gcc-high-endpoints.md) | Конечные точки для подписок GCC High государственных организаций США. |
| [Microsoft 365 с конечными точками 21Vianet](urls-and-ip-address-ranges-21vianet.md) | Конечные точки для Microsoft 365 21Vianet, которая предназначена для удовлетворения потребностей Microsoft 365 в Китае. |
| [Microsoft 365 Конечные точки Германии](microsoft-365-germany-endpoints.md) | Конечные точки отдельного облака в Европе для наиболее регулируемых клиентов в Германии, Европейском Союзе (ЕС) и Европейской ассоциации свободной торговли (ЕАСТ). |
|||
