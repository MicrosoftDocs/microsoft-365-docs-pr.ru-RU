# <a name="migrate-your-organization-data-to-microsoft-365-enterprise"></a>Перенос данных организации в Microsoft 365 корпоративный

## <a name="ready-to-migrate-your-data-to-microsoft-365"></a>Готовы перенести данные в Microsoft 365?

Наши [основные инструкции по](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) настройкам помогают вам получить всех, Microsoft 365 служб и приложений в кратчайшие сроки. Это пропускает гибридные сценарии и более продвинутые методы миграции в Microsoft 365. 
  
Если вы хотите помочь Microsoft 365 настройки, [FastTrack](https://fasttrack.microsoft.com/office) — это самый простой способ развертывания Microsoft 365, вы также можете войти и использовать руководства установки для Microsoft 365 [служб.](setup-guides-for-microsoft-365.md)
