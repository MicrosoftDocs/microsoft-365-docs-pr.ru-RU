# <a name="deploy-microsoft-365-enterprise-for-your-organization"></a>Развертывание Microsoft 365 корпоративный для организации

Готовы ли вы развернуть Microsoft 365 корпоративный, интегрировать его с локальной инфраструктурой и позволить сотрудникам использовать преимущества облачной производительности и совместной работы?

В этой статье описываются два варианта развертывания, независимо от того, нужна ли вам помощь или вы делаете это самостоятельно. В каждом случае корпорация Майкрософт предоставляет вам управляемый путь для успешного развертывания.

## <a name="guided-microsoft-365-enterprise-setup-process-with-fasttrack"></a>Руководство по настройке Microsoft 365 корпоративный с помощью FastTrack

**[FastTrack для Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365)** — лучший способ развертывания Microsoft 365. Служба FastTrack поможет разобраться в одной из самых распространенных конфигураций развертывания и ответит на возникающие в ходе этого процесса вопросы. 

Чтобы получить доступ к ряду руководств по настройке, войдите в свою подписку на Microsoft 365 и перейдите в руководство [по установке.](https://aka.ms/o365fasttrack)

>[!Note]
>Вы также можете получить помощь от партнера [Майкрософт.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-guided-deployment-of-microsoft-365-enterprise"></a>Самостоятельное развертывание Microsoft 365 корпоративный

Для самостоятельного развертывания Microsoft 365 корпоративный необходимо выполнить подробное исследование, чтобы принять проектные решения, упрощающие настройку служб Microsoft 365 и внедрение пользователей. Начните планирование [здесь.](get-your-organization-ready-for-office-365.md)

Чтобы развернуть Microsoft 365 корпоративный самостоятельно после завершения планирования, рекомендуется сделать это.

1. [Настройка сети](set-up-network-for-microsoft-365.md)

   Включает добавление доменов Интернета и оптимизацию производительности сети для пользователей локальной сети.
 
2. [Настройка удостоверений](protect-your-global-administrator-accounts.md)

   Включает определение модели удостоверений (только облачной или гибридной) и гибридной идентификации, настройку синхронизации службы каталогов между локальной доменной службой Active Directory (AD DS) и подпиской на Microsoft 365.

3. [Реализация безопасности](https://docs.microsoft.com/office365/securitycompliance/security-roadmap)

   Включает настройку и развертывание основных и улучшенных мер безопасности, защиты от угроз и информации для клиента и удостоверений в течение первых 30, 90 и более дней.
 
4. [Развертывание клиентского программного обеспечения](https://docs.microsoft.com/DeployOffice/deployment-guide-microsoft-365-apps)

   Включает развертывание приложений Microsoft 365 для предприятий (ранее называлось Office 365 профессиональныйplus), обновленной в облаке и постоянно обновляемой версии набора Office (Word, Excel, PowerPoint и других) на ваших устройствах. Каждая клиентская лицензия Microsoft 365 включает лицензию на приложения Microsoft 365 для предприятий.
 
5. [Настройка управления мобильными устройствами](https://support.office.com/article/set-up-mobile-device-management-mdm-in-office-365-dd892318-bc44-4eb1-af00-9db5430be3cd)

   Microsoft 365 корпоративный включает возможности управления мобильными устройствами, которые помогают защитить мобильные устройства пользователей и управлять ими.
 
6. [Настройка служб и приложений](configure-services-and-applications.md)

   Содержит сведения о переносе данных и ссылки на статьи, в которые вы можете начать работу с ключевыми приложениями для повышения производительности Microsoft 365, такими как Exchange Online, SharePoint Online и Teams.
 
7. [Обучение пользователей](https://docs.microsoft.com/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

   Содержит короткие видеоролики, которые помогут пользователям быстро получить максимальное количество результатов от Microsoft 365.
 

>[!Note]
>Эти действия также необходимы для предприятий и [некоммерческих](https://go.microsoft.com/fwlink/?LinkId=627221) организаций, которые хотят начать с настраиваемого развертывания Microsoft 365 корпоративный. 
>
