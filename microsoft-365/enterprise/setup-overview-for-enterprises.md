# <a name="deploy-microsoft-365-enterprise-for-your-organization"></a>Развертывание Microsoft 365 Enterprise для организации

Готовы ли вы развернуть Microsoft 365 Enterprise, интегрировать ее с локальной инфраструктурой и предоставить сотрудникам возможность использовать преимущества облачной производительности и совместной работы?

В этой статье описываются два варианта развертывания, независимо от того, хотите ли вы помочь или делаете это самостоятельно. В каждом случае Корпорация Майкрософт предоставляет вам управляемый путь для успешного развертывания.

## <a name="guided-microsoft-365-enterprise-setup-process-with-fasttrack"></a>Руководство microsoft 365 Корпоративный процесс установки с FastTrack

**[FastTrack для Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365)** — это лучший способ развертывания Microsoft 365. Служба FastTrack поможет разобраться в одной из самых распространенных конфигураций развертывания и ответит на возникающие в ходе этого процесса вопросы. 

Чтобы получить доступ к ряду руководств по настройке, войдите в подписку на Microsoft 365 и перейдите в [руководство по настройке.](https://aka.ms/o365fasttrack)

>[!Note]
>Вы также можете получить помощь от партнера [Майкрософт.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-guided-deployment-of-microsoft-365-enterprise"></a>Самостоятельное развертывание Microsoft 365 Enterprise

Для самостоятельного развертывания Microsoft 365 Enterprise необходимо выполнить подробные исследования для принятия решений по проектированию, упрощающих конфигурацию служб Microsoft 365 и принятие пользователей. Начните планирование [здесь.](get-your-organization-ready-for-office-365.md)

Для самостоятельного развертывания Microsoft 365 Enterprise после завершения планирования рекомендуется сделать следующие действия.

1. [Настройка сети](set-up-network-for-microsoft-365.md)

   Включает добавление доменов в Интернете и оптимизацию производительности сети для локального пользователя.
 
2. [Настройка удостоверений](protect-your-global-administrator-accounts.md)

   Включает определение модели удостоверений (только облачной или гибридной), а также гибридную идентификацию, настройку синхронизации каталогов между локальной службой доменных служб Active Directory (AD DS) и подпиской на Microsoft 365.

3. [Реализация безопасности](/office365/securitycompliance/security-roadmap)

   Включает в себя настройку и развертывание основных и расширенных защитных элементов безопасности, угроз и информации для клиента и удостоверений в течение первых 30 дней, 90 дней и далее.
 
4. [Развертывание клиентского программного обеспечения](/DeployOffice/deployment-guide-microsoft-365-apps)

   Включает развертывание microsoft 365 Apps для предприятия (ранее именуемого Office 365 ProPlus), облачную и постоянно обновляемую версию пакета Office (Word, Excel, PowerPoint и другие) на ваших устройствах. Каждая клиентская лицензия Microsoft 365 включает лицензию для microsoft 365 Apps для предприятия.
 
5. [Настройка управления мобильными устройствами](https://support.office.com/article/set-up-mobile-device-management-mdm-in-office-365-dd892318-bc44-4eb1-af00-9db5430be3cd)

   Microsoft 365 Enterprise включает возможности управления мобильными устройствами, которые помогают обеспечить безопасность и управление мобильными устройствами пользователей.
 
6. [Настройка служб и приложений](configure-services-and-applications.md)

   Включает сведения о миграции данных и ссылки на статьи, которые вы начинаете с ключевых приложений производительности Microsoft 365, таких как Exchange Online, SharePoint Online и Teams.
 
7. [Обучение пользователей](/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

   Включает короткие видео, которые помогают пользователям быстро получить максимальное количество данных из Microsoft 365.
 

>[!Note]
>Эти действия также для предприятий и [некоммерческих организаций,](https://go.microsoft.com/fwlink/?LinkId=627221) которые хотят начать с настраиваемого развертывания Microsoft 365 Enterprise.
