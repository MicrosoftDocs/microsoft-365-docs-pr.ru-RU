# <a name="integrated-apps-and-azure-ad-for-microsoft-365-administrators"></a>Интегрированные приложения и Azure AD для администраторов Microsoft 365

Управление интегрированными приложениями имеет гораздо большее, чем просто управление согласием [пользователей на приложения.](https://docs.microsoft.com/microsoft-365/admin/misc/integrated-apps) С появлением REST API Microsoft 365 пользователи могут предоставлять приложениям доступ к своим данным Microsoft 365, таким как почта, календари, контакты, пользователи, группы, файлы и папки. По умолчанию пользователям необходимо индивидуально предоставлять разрешения каждому приложению. 

Но это не очень хорошо масштабироваться, если вы хотите авторизировать приложение один раз на уровне глобального администратора и развясти его для всей организации с помощью запуска приложений. Для этого необходимо зарегистрировать приложение в Azure Active Directory (Azure AD). Чтобы зарегистрировать приложение в Azure AD, необходимо предпринять несколько действий, а также некоторые справочные сведения, которые помогут вам управлять приложениями в организации Microsoft 365.
