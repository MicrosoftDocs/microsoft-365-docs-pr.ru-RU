<span data-ttu-id="0b8bb-122">Показанные столбцы с данными:</span><span class="sxs-lookup"><span data-stu-id="0b8bb-122">Data columns shown are:</span></span>

- <span data-ttu-id="0b8bb-p107">**Категория** . Показывает, присвоена ли набору конечной точки категория "Оптимизация", "Разрешение" или "По умолчанию". This column also lists which endpoint sets are required to have network connectivity. For endpoint sets which are not required to have network connectivity, we provide notes in this field to indicate what functionality would be missing if the endpoint set is blocked. If you are excluding an entire service area, the endpoint sets listed as required do not require connectivity.</span></span>

- <span data-ttu-id="0b8bb-p108">**ER** . Значение **Да** , если набор конечной точки поддерживается в Azure ExpressRoute с префиксами маршрутизации Office 365. Если значение ER соответствует **Нет** , это означает, что ExpressRoute не поддерживается для этого набора конечной точки. However, it should not be assumed that no routes are advertised for an endpoint set where ER is **No** .</span></span>

- <span data-ttu-id="0b8bb-p109">**Адреса** . Перечисляет полные доменные имена или подстановочные доменные имена, а также диапазоны IP-адресов для набора конечной точки. - <span data-ttu-id="0b8bb-p110">**Порты** . Перечисляет порты TCP или UDP, которые объединяются с адресами для создания конечной точки сети.
