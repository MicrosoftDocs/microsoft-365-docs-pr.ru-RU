# <a name="overview-of-contoso-corporation"></a>Обзор корпорации Contoso

Корпорация Contoso — это многонациональный бизнес со штаб-квартирой в Париже. Компания — это организация по производству, продажам и поддержке с более чем 100 000 продуктов.

## <a name="contoso-around-the-world"></a>Представительства Contoso по всему миру

На рисунке 1 показан центральный офис в Париже и региональные центральные и спутниковые офисы на различных континентах.

![Офисы Contoso по всему миру](../media/contoso-overview/contoso-overview-fig1.png)

**Рис. 1. Офисы Contoso по всему миру**
 
Contoso имеет три уровня офисов:

- Headquarters

  Штаб-квартира Contoso — это корпоративный кампус на окраине Парижа с десятками зданий для административных, инженерных и производственных объектов. Все центра обработки данных Contoso и его присутствие в Интернете находятся в штаб-квартире в Париже.

  В главном офисе работает 25 000 сотрудников.

- Региональные центральные офисы

  Офисы hub обслуживают определенный регион мира с 60-процентным персоналом по продажам и поддержке. Каждый региональный центр подключен к штаб-квартире в Париже с помощью высокоскоростной WAN-ссылки.

  В региональных центрах в среднем работают 2000 сотрудников.

- Подчиненные офисы

  Спутниковые офисы содержат 80-процентный персонал по продажам и поддержке. Они обеспечивают присутствие клиентов Contoso на месте в ключевых городах или субрегионах. Каждый спутниковый офис подключен к региональному центру с помощью высокоскоростной WAN-ссылки.

  В спутниковых офисах в среднем работают 250 сотрудников.

Около 25 процентов сотрудников Contoso только для мобильных устройств. В региональных центрах и спутниковых офисах доля таких сотрудников выше. Улучшение поддержки мобильных сотрудников — важная бизнес-цель корпорации Contoso.

## <a name="design-considerations-for-microsoft-365-for-enterprise"></a>Разработка соображений для Microsoft 365 для предприятия

ИТ-архитекторы Contoso определили следующие факторы, необходимые для развертывания Microsoft 365 для предприятия:

- Разные географические расположения с местным законодательством и требованиями по соответствию нормам
- Центральный центр обработки данных интрасети в штаб-квартире и региональных серверах приложений, на которые вмещая внутренние бизнес-приложения
- Существующая инфраструктура Microsoft Endpoint Configuration Manager
- Сочетание клиентских вычислительных устройств с Windows, Mac и Linux
- Сочетание личных и корпоративных мобильных устройств, включая смартфоны и планшеты на iOS (iPhone и iPad) и Android
- Много удаленных и мобильных сотрудников
- Большое количество бизнес-партнеров
- Большое количество персональных данных клиента и других конфиденциальных сведений для управления и обеспечения безопасности
- Большой объем ценной интеллектуальной собственности в виде проектных спецификаций для продуктов и производственных тайн
