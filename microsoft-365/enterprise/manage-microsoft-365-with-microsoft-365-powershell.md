# <a name="manage-microsoft-365-with-powershell"></a>Управление Microsoft 365 с помощью PowerShell

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

PowerShell для Microsoft 365 — это мощное средство управления, дополняющие Центр администрирования Microsoft 365. Например, вы можете использовать автоматизацию PowerShell для простого управления несколькими учетных записей пользователей и лицензий, а также для создания отчетов.

Выберите один из следующих разделов, чтобы узнать, как использовать PowerShell для управления Microsoft 365:
  
- [**Начало работы**](getting-started-with-microsoft-365-powershell.md)

    Ознакомьтесь с этой ссылкой, если вы не знакомы с PowerShell для Microsoft 365 и хотите установить модули Microsoft 365 и подключиться к подписке.

- [**Учетные записи пользователей, лицензии и группы**](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Начните с этой статьи, если хотите узнать об использовании команд автоматизации для управления учетной записью пользователя, лицензиями и группами.

- [**SharePoint**](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Начните с этой статьи, если вы хотите использовать команды автоматизации для управления SharePoint.

- [**Exchange Online**](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)

    Начните с этой темы, если вы хотите управлять Exchange Online.

- [**миграции электронной почты.**](use-powershell-for-email-migration-to-microsoft-365.md)

    Начните с этой темы, если вы хотите перенести электронную почту из существующих систем.

- [**Центр безопасности и соответствия требованиям**](https://docs.microsoft.com/powershell/exchange/scc-powershell)

    Начните с этого, если вы хотите управлять функциями Центра & соответствия требованиям.

- [**Партнеры по делегированную доступ (DAP)**](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Начните с этой темы, если вы хотите использовать партнеров по синдикации и поставщиков облачных решений (CSP) для управления клиентами Microsoft 365.

- [**Skype для бизнеса Online**](manage-skype-for-business-online-with-microsoft-365-powershell.md)

    Начните с этого, чтобы управлять Skype для бизнеса Online.
