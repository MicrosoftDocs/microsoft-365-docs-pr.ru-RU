# <a name="manage-microsoft-365-with-powershell"></a>Управление Microsoft 365 с помощью PowerShell

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

PowerShell для Microsoft 365 является мощным средством управления, который дополняет Microsoft 365 центра администрирования. Например, можно использовать автоматизацию PowerShell для легкого управления несколькими учетных записей и лицензий пользователей и создания отчетов.

Выберите из следующих разделов, чтобы узнать, как использовать PowerShell для управления Microsoft 365:
  
- [**Начало работы**](getting-started-with-microsoft-365-powershell.md)

    Начните здесь, если вы не знакомы с PowerShell для Microsoft 365, и вы хотите установить Microsoft 365 и подключиться к подписке.

- [**Учетные записи пользователей, лицензии и группы**](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Начните здесь, если хотите узнать об использовании команд автоматизации для управления учетной записью, лицензиями и группами пользователей.

- [**SharePoint**](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Начните здесь, если вы хотите использовать команды автоматизации для управления SharePoint.

- [**Exchange Online**](/powershell/exchange/exchange-online-powershell)

    Начните здесь, если вы хотите управлять Exchange Online.

- [**миграции электронной почты.**](use-powershell-for-email-migration-to-microsoft-365.md)

    Начните здесь, если вы хотите перенести электронную почту из уже существующих систем.

- [**Центр безопасности и соответствия требованиям**](/powershell/exchange/scc-powershell)

    Начните здесь, если вы хотите управлять функциями Центра & безопасности.

- [**Партнеры по делегированию разрешений доступа (DAP)**](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Начните здесь, если вы хотите использовать партнеров syndication и поставщик облачных решений (CSP) для управления клиентами Microsoft 365 клиентов.
