# <a name="microsoft-365-client-app-support-conditional-access"></a>Microsoft 365 Поддержка клиентских приложений: условный доступ

На современном рабочем месте пользователи могут получать доступ к ресурсам организации с помощью различных устройств и приложений из любой точки мира. В результате этого недостаточно просто сосредоточиться на том, кто может получить доступ к ресурсу. Организация также должна поддерживать доступ к ресурсу в инфраструктуре управления доступом.

Благодаря Azure Active Directory устройству, расположению и многофакторной проверке подлинности условным доступом можно выполнить это новое требование. Условный доступ — это возможность Azure Active Directory, которая позволяет применять элементы управления доступом к приложениям в вашей среде, все зависит от определенных условий и управляется из центра.

Дополнительные дополнительные [Azure Active Directory условного доступа](/azure/active-directory/conditional-access/).
