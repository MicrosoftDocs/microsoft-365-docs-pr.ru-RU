# <a name="cmdlet-references-for-microsoft-365-services"></a>Сведения о командлетах для служб Microsoft 365 

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

В этой статье приведены ссылки на командлеты для различных служб Майкрософт 365 и инструкций по подключению для каждой службы Microsoft 365, поддерживаемой PowerShell.
  
> [!NOTE]
> Чтобы подключиться ко всем службам сразу, ознакомьтесь [со статьей подключение ко всем службам Microsoft 365 в отдельном окне Windows PowerShell](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md).
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Командлеты PowerShell Azure Active Directory

Справочные разделы PowerShell Azure Active Directory PowerShell для Graph представлены в разделе Справочные материалы по [документации PowerShell Active Directory PowerShell для Graph](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0).

Справочные разделы по модулю Azure Active Directory для Windows PowerShell приведены в разделе Справочные материалы по [документации Azure Active Directory (MSOnline)](https://docs.microsoft.com/powershell/azure/active-directory/overview?view=azureadps-1.0).

Инструкции по подключению Microsoft 365 PowerShell приведены в статье [Подключение к microsoft 365 с помощью PowerShell](connect-to-microsoft-365-powershell.md).
  
## <a name="exchange-online-powershell-cmdlets"></a>Командлеты Exchange Online PowerShell

Справочные разделы по командлетам Exchange Online приведены в разделе Справочные материалы [по документации PowerShell для Exchange Online](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell).
  
Инструкции по подключению к Exchange Online PowerShell приведены [в статье подключение к Exchange Online PowerShell](https://go.microsoft.com/fwlink/p/?LinkId=396554).
  
> [!NOTE]
> Командлеты отчетов для других служб, в том числе SharePoint Online, Skype для бизнеса Online и Microsoft 365 User Activity, доступны в Exchange Online PowerShell. Дополнительные сведения см. [в статье командлеты отчетов в Exchange Online](https://go.microsoft.com/fwlink/p/?LinkId=691595). 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>Командлеты PowerShell в SharePoint Online

Командлеты SharePoint Online приведены в статье [индексирование командлетов Windows PowerShell для SharePoint Online](https://go.microsoft.com/fwlink/p/?LinkId=691476).
  
Инструкции по подключению для SharePoint Online PowerShell можно найти [в статье Настройка среды Windows PowerShell для командной консоли SharePoint Online](https://go.microsoft.com/fwlink/p/?LinkId=691603).
  
## <a name="skype-for-business-online-powershell-cmdlets"></a>Командлеты PowerShell в Skype для бизнеса Online

В справочных материалах по командлетам Skype для бизнеса Online можно ознакомиться с [командлетами Skype для бизнеса Online](https://technet.microsoft.com/library/mt228132.aspx).
  
Инструкции по подключению для Skype для бизнеса Online PowerShell можно найти [в статье Управление Skype для бизнеса Online с помощью PowerShell](manage-skype-for-business-online-with-microsoft-365-powershell.md).

## <a name="security--compliance-center-powershell-cmdlets"></a>Командлеты PowerShell центра соответствия требованиям & безопасности

Справочные материалы по командлетам центра безопасности & требованиям приведены в разделе Справочные материалы по [документации по системе безопасности для обеспечения безопасности &](https://docs.microsoft.com/powershell/exchange/scc-powershell).
  
Инструкции по подключению & безопасности для центра соответствия требованиям PowerShell можно найти [в разделе Connect to the security & Security Center The PowerShell](https://docs.microsoft.com/powershell/exchange/connect-to-scc-powershell).
