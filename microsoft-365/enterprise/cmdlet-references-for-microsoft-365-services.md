# <a name="cmdlet-references-for-microsoft-365-services"></a>Сведения о командлетах для служб Microsoft 365 

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

В этой статье содержатся ссылки на различные службы Microsoft 365 и инструкции по подключению для каждой службы Microsoft 365, поддерживаемой PowerShell.
  
> [!NOTE]
> Чтобы подключиться к всем службам одновременно, см. "Подключение к всем службам [Microsoft 365" в](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)одном Windows PowerShell окна.
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Cmdlets Azure Active Directory PowerShell

Справочные разделы по powerShell для Azure Active Directory для Graph см. в справочном разделе документации [по Azure Active Directory PowerShell для Graph.](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)

Справочные статьи о модуле Azure Active Directory для Windows PowerShell командлетов см. в справочном разделе документации [по Azure Active Directory (MSOnline).](https://docs.microsoft.com/powershell/azure/active-directory/overview?view=azureadps-1.0)

Инструкции по подключению к Microsoft 365 PowerShell см. в подключении к [Microsoft 365 с помощью PowerShell.](connect-to-microsoft-365-powershell.md)
  
## <a name="exchange-online-powershell-cmdlets"></a>Командлеты Exchange Online PowerShell

Справочные разделы о cmdlet exchange Online см. в справочном разделе документации [по Exchange Online PowerShell.](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)
