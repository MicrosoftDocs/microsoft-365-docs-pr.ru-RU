# <a name="cmdlet-references-for-microsoft-365-services"></a>Сведения о командлетах для служб Microsoft 365 

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

В этой статье содержатся ссылки на различные службы Microsoft 365 и инструкции по подключению для каждой службы Microsoft 365, поддерживаемой PowerShell.
  
> [!NOTE]
> Чтобы подключиться сразу к всем службам, см. в этой записи [connect to all Microsoft 365 services in a single Windows PowerShell window.](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Cmdlets Azure Active Directory PowerShell

Справочные разделы Azure Active Directory PowerShell для графа находятся в справочном разделе документации [Azure Active Directory PowerShell для Graph.](/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)

Модуль Azure Active Directory для справочных тем Windows PowerShell командлетов находится в справочном разделе документации [Azure Active Directory (MSOnline).](/powershell/azure/active-directory/overview?view=azureadps-1.0)

Инструкции по подключению к Microsoft 365 PowerShell см. в ссылке Подключение к [Microsoft 365 с помощью PowerShell.](connect-to-microsoft-365-powershell.md)
  
## <a name="exchange-online-powershell-cmdlets"></a>Командлеты Exchange Online PowerShell

Справочные темы для веб-пользователей Exchange Online находятся в справочном разделе документации [Exchange Online PowerShell.](/powershell/exchange/exchange-online-powershell)
  
Инструкции по подключению для Exchange Online PowerShell см. в обзоре [Connect to Exchange Online PowerShell.](/powershell/exchange/connect-to-exchange-online-powershell)
  
> [!NOTE]
> В Exchange Online PowerShell доступны разделы отчетности для других служб, таких как SharePoint Online, Skype для бизнеса Online и Microsoft 365. Дополнительные сведения см. в [веб-сайте Reporting cmdlets in Exchange Online.](/powershell/exchange/exchange-online-powershell) 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>Командлеты PowerShell в SharePoint Online

Для разделов SharePoint Online см. в Windows PowerShell [index of Windows PowerShell sharePoint Online.](/powershell/module/sharepoint-online/)
  
Инструкции по подключению для SharePoint Online PowerShell см. в раздел Настройка среды управления [sharePoint Online Windows PowerShell.](/powershell/sharepoint/sharepoint-online/connect-sharepoint-online)

Инструкции по подключению для Skype для бизнеса Online PowerShell см. в этой ссылке Управление Skype для бизнеса [Online с помощью PowerShell.](manage-skype-for-business-online-with-microsoft-365-powershell.md)
