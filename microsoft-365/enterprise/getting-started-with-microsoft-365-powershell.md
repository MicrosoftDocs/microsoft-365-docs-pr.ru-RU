# <a name="get-started-with-powershell-for-microsoft-365"></a>Приступая к работе с PowerShell для Microsoft 365

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

Вы можете использовать команды и сценарии в PowerShell для Microsoft 365 для управления Microsoft 365 и оптимизации ежедневной работы. Узнайте, почему PowerShell для Microsoft 365 очень важно для управления Microsoft 365, для подключения к вашей подписке Microsoft 365 и создания отчетов, а также для получения дополнительных сведений и помощи в сообществе Майкрософт 365.
  
Выберите из следующих разделов:
  
- [**Зачем использовать PowerShell для Microsoft 365**](why-you-need-to-use-microsoft-365-powershell.md)
    
    Начните отсюда, если вы не знакомы с PowerShell для Microsoft 365. Сведения о том, почему необходимо использовать PowerShell для Microsoft 365.
    
- [**Подключение к Microsoft 365 с помощью PowerShell**](connect-to-microsoft-365-powershell.md)
    
    Начните отсюда, чтобы подключиться к вашей подписке Microsoft 365 с помощью PowerShell для Microsoft 365 и выполнять задачи администрирования из командной строки.
    
- [**Подключение ко всем службам Microsoft 365 с помощью единого окна PowerShell**](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    Вы можете управлять Microsoft 365 в отдельном окне для Skype для бизнеса Online, SharePoint Online, Microsoft Exchange Online и учетных записей и лицензий Microsoft 365. Вы также можете управлять ими из одного окна. В этой статье описывается, как это сделать.
    
- [**Создание отчетов в Microsoft 365 с помощью PowerShell**](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    Начните отсюда, если вы установили модули PowerShell для Microsoft 365 и хотите узнать, как использовать команды автоматизации для быстрого создания отчетов.
    
- [**Сведения о командлетах для служб Microsoft 365**](cmdlet-references-for-microsoft-365-services.md)
    
    Узнайте о командлетах PowerShell для модулей Microsoft 365.
    
- [**Ресурсы сообщества Microsoft 365 для PowerShell**](microsoft-365-powershell-community-resources.md)
    
    Начните отсюда, чтобы подключиться к сообществу PowerShell и получить помощь или дополнительные сведения об использовании PowerShell для Microsoft 365.
