# <a name="use-powershell-for-email-migration-to-microsoft-365"></a>Миграция электронной почты в Microsoft 365 с помощью PowerShell

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

Когда администраторы сначала настраивают Microsoft 365, многие из них переносят электронную почту из существующих систем. Это также можно сделать с помощью центра администрирования Microsoft 365. Кроме того, для переноса электронной почты вы можете использовать Windows PowerShell.
  
Используйте Windows PowerShell для переноса электронной почты в Microsoft 365.
