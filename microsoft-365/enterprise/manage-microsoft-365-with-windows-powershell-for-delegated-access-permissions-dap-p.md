# <a name="how-to-manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-partners"></a>Как управлять Microsoft 365 с помощью Windows PowerShell для партнеров с правами делегированного доступа

*Эта статья относится к Microsoft 365 корпоративный и Office 365 корпоративный.*

Партнеры по делегированным правам доступа (DAP)  партнеры по синдикации и поставщики облачных решений (CSP). Многие поставщики сетевых услуг или телекоммуникационных услуг. Они объединяют подписки Microsoft 365 на их обслуживание. При продаже подписки на Microsoft 365 им автоматически предоставляется разрешение "Администрирование от имени" (АОБО) для клиенты клиента, чтобы они могли администрировать и сообщать об этих клиенты. Эти задачи трудно выполнить в центре администрирования Microsoft 365. Использование PowerShell для Microsoft 365 значительно упрощает выполнение административных задач, таких как:
- Перечисление всех клиентов **тенантидс** и их доменов 
- Определение всех пользователей в клиентской лицензии и назначенных им лицензиях
> [!NOTE]
> Некоторые административные задачи можно выполнить только в PowerShell.

В следующих статьях показано, как участники синдикации и поставщики служб шифрования используют PowerShell для администрирования своих клиентов клиенты:
