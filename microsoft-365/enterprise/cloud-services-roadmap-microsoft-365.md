# <a name="cloud-services-roadmap-for-microsoft-365"></a>План облачных служб для Microsoft 365

Чтобы получить творческий потенциал, командную работу и преимущества Microsoft 365 для предприятий, разверните облачные службы, которые наилучшим образом соответствуют потребностям вашей организации.

Если у вас есть серверные развертывания для Exchange, SharePoint или Skype для бизнеса и вы хотите перенести всю организацию в Microsoft 365 для предприятий, см. план развития клиентского и серверного программного [обеспечения.](client-server-software-roadmap-microsoft-365.md) Этот план включает Microsoft Office клиентские продукты, продукты локального сервера Office Server и устройства на основе Microsoft &ndash; Windows.

## <a name="how-microsoft-does-cloud-services-for-microsoft-365"></a>Как корпорация Майкрософт делает облачные службы для Microsoft 365

Сведения о развертывании облачных служб Microsoft 365 или управлении облачными службами Microsoft 365:

1. Перейдите в [ИТ-демонстрацию Майкрософт.](https://www.microsoft.com/itshowcase)
2. Выберите **контент поиска.**
3. В **области результатов уточнения выберите** **"ИТ-профессиональный"** в области аудитории, а затем в области **"Продукт"** выберите облачную службу.
