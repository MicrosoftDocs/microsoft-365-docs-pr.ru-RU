# <a name="enter-your-product-key-for-microsoft-365-business-standard"></a>Введите ключ продукта для Microsoft 365 бизнес стандарт

Если вы приобрели Microsoft 365 бизнес стандартный в розничном магазине, эта статья поможет вам активировать подписку.
  
 **Нужно выполнить другую задачу?**
 **Справку по семейство Microsoft 365 или Microsoft 365 персональный** см. в справке по использованию [ключей продукта в Office.](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)  
 **Участвуете в программе корпоративного лицензирования?** Если вы приобрели Microsoft 365 для бизнеса в рамках программы [microsoft Open License Program,](https://go.microsoft.com/fwlink/p/?LinkID=613298)см. статью, введите ключ продукта, приобретенный [в Microsoft Open.](purchases-from-microsoft-open.md)
  
## <a name="activate-a-new-subscription"></a>Активация новой подписки

1. Перейдите по ссылке <a href="https://go.microsoft.com/fwlink/p/?LinkId=839911" target="_blank">https://microsoft365.com/setupbusiness</a>.

2. Введите ключ продукта и выберите **"Далее".**

3. Введите имя, рабочий адрес электронной почты, рабочий номер телефона, название компании и страну или регион, а затем выберите **"Далее".**

4. Введите номер телефона, чтобы получить текст или вызов для подтверждения личности.

5. Вы можете получить новый домен, использовать доменное имя, которое вы уже владеете, или добавить домен позже. Действия могут отличаться, поэтому следуйте запросам в зависимости от того, какой вариант вы выбрали.

    > [!NOTE]
    > Если вы хотите, чтобы подписка автоматически продлевалась в конце срока ее действия, можно добавить кредитную карту или банковский счет, а затем включить регулярное выставление счетов для этой подписки. [](subscriptions/renew-your-subscription.md#turn-recurring-billing-off-or-on)

6. После завершения перейдите в Центр администрирования, <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">чтобы</a> управлять подпиской.

## <a name="extend-the-expiration-date-or-add-a-license-to-an-existing-subscription"></a>Продление срока действия или добавление лицензии в существующую подписку

Чтобы продлить срок действия или добавить лицензию в существующую подписку Microsoft 365 бизнес стандартный, используйте новый и неиспользимый ключ для того же плана Microsoft 365. Ввод исходного ключа продукта Microsoft 365 не будет работать, так как уже использованный ключ нельзя использовать снова. См. ["Добавление лицензий для или расширение подписки, оплаченной с помощью ключа продукта".](licenses/add-licenses-using-product-key.md)
