# <a name="enter-your-product-key-purchased-from-microsoft-open"></a>Введите ключ продукта, приобретенный в Microsoft Open

Если вы приобрели Microsoft 365 для бизнеса из [программы Microsoft Open License Program,](https://go.microsoft.com/fwlink/p/?LinkID=613298)эта статья для вас. Клиенты, которые приобрели ключ продукта по программе корпоративного лицензирования, могут воспользоваться инструкциями в этой статье, чтобы активировать, продлить лицензии или добавить их в подписку.
  
 **Требуется срочная помощь?** [Вызов службы поддержки Майкрософт](../admin/contact-support-for-business-products.md) . 
  
 **Нужно выполнить другую задачу?**
 
 **Справки по Microsoft 365 home или Personal** см. в справке Использование ключей продукта с [Office.](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)
  
 **Справки о Microsoft 365 бизнес стандарт,** приобретенных в розничном магазине, см. в справке Введите ключ продукта, приобретенный [в розничном магазине.](enter-your-product-key.md) 
  
## <a name="activate-renew-or-add-licenses-to-a-subscription-purchased-from-microsoft-open"></a>Активация, продление подписки, приобретенной по программе Microsoft Open, и добавление в нее лицензий

Если вы Microsoft 365 лицензии Microsoft Open License Program, система автоматически вводит ключ для вас при активации, обновлении или добавлении лицензий в подписку.
  
1. После покупки ключа от партнера Microsoft 365 проверьте почтовый ящик для электронной почты от Microsoft, содержащей ссылку на активацию.
    
    > [!TIP]
    >  Сообщение отправляется на адрес электронной почты лица, указанного в качестве администратора веб-служб в соглашении о корпоративном лицензировании. Если вы не уверены, кто назначен администратором в Интернете, спросите своего Microsoft 365 партнера. 
  
2. Перейдите по полученной ссылке, войдите с помощью своей рабочей или учебной учетной записи, примите условия соглашения Open License и продлите подписку. Для пошаговой работы скачайте руководство по PDF-файлу Активируйте online [Services: Microsoft 365 Subscription,](https://go.microsoft.com/fwlink/p/?LinkId=618100)в котором описывается, как активировать новую подписку, обновить существующую подписку или добавить лицензии в существующую подписку.
    
> [!NOTE]
> Клиенты по лицензированию тома также могут активировать подписки в Центре обслуживания [корпоративного лицензирования.](https://go.microsoft.com/fwlink/p/?LinkID=282016) Чтобы узнать, как это делается, скачайте PDF-руководство [Активация служб Online Services на веб-сайте Volume Licensing Service Center](https://go.microsoft.com/fwlink/p/?LinkId=618096).
