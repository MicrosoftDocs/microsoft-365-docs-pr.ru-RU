# <a name="understand-the-proposal-workflow"></a>Общие сведения о рабочем процессе предложений

Предложение — это формальное предложение от Майкрософт для приобретения продуктов и служб Майкрософт. Вы напрямую работаете с представителем корпорации Майкрософт, чтобы определить конкретные продукты, Услуги и условия для вашего предложения.

Представитель корпорации Майкрософт предложение, содержащее элементы, обсуждаемые вами и вашим представителем. Представитель отправляет вам сообщение электронной почты со ссылкой на сайт предложений. Сайт содержит предложение, специально подготовленное для вас и вашей организации.

После получения уведомления по электронной почте перейдите по ссылке на сайт предложений. После входа на сайт вы можете начать процесс проверки предложения.

## <a name="prerequisites-for-buying-items-with-a-proposal"></a>Необходимые условия для приобретения элементов с предложением

Прежде чем приобретать товары для предложения, необходимо иметь счет выставления счетов и соглашение с корпорацией Майкрософт.

### <a name="billing-account"></a>Счет для выставления счетов

Учетная запись выставления счетов используется для управления параметрами учетных записей, счетов, профилей выставления счетов, а также продуктов и услуг. Если у вас еще нет учетной записи выставления счетов, ваш представитель корпорации Майкрософт создает его.
В противном случае они используют для Организации существующий счет для выставления счетов, при условии, что у вас есть разрешение на использование этой учетной записи выставления счетов.

Управление разрешениями учетной записи выставления счетов осуществляется владельцем учетной записи выставления счетов.
Глобальные администраторы могут назначать себя для роли владельца учетной записи выставления счетов, а затем делать других владельцев учетных записей пользователей для выставления счетов.

Дополнительные сведения об учетных записях для выставления счетов можно узнать в статье [Управление выставлением](manage-billing-accounts.md)счетов.

### <a name="microsoft-customer-agreement"></a>Соглашение клиента Майкрософт

Соглашение об использовании клиентов (MCA) Майкрософт позволяет Организации покупать продукты и службы Майкрософт. Для получения дополнительных сведений обратитесь к разделу [Соглашение клиента Майкрософт](https://www.microsoft.com/en-us/Licensing/how-to-buy/microsoft-customer-agreement).

## <a name="permissions-needed-to-sign-an-agreement-or-pay-for-items"></a>Разрешения, необходимые для подписания соглашения или оплаты за номенклатуры

Если у вас нет назначенной роли в учетной записи выставления счетов, при просмотре предложения вы назначаете базовую роль читателя. Эта роль позволяет просматривать, но не предпринимать никаких действий над предложением. Прежде чем можно будет подписать соглашение или приобрести продукты и службы, необходимо назначить роль участника учетной записи для выставления счетов или партнера по учетным записям для выставления счетов. Владелец учетной записи выставления счетов может назначить вам эту роль.

Дополнительные сведения о ролях счетов выставления счетов приведены [в статье Общие сведения о доступе к учетным записям выставления](manage-billing-accounts.md#understand-access-to-billing-accounts)счетов.

Если это новая учетная запись выставления счетов и никто не принял соглашение, он автоматически становится владельцем учетной записи выставления счетов при условии, что вы:

- — Это пользователь, названный в предложении \
    OR
- Уже является [глобальным администратором Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles) для вашей организации

## <a name="what-is-the-overall-workflow"></a>Общие сведения о рабочем процессе

Общий рабочий процесс предложений выглядит следующим образом:

- Представитель корпорации Майкрософт создает предложение и отправляет ссылку на вас в сообщении электронной почты.

- Используйте ссылку, чтобы перейти на страницу входа в предложение.

- Вы просматриваете сведения об Организации.

- Вы просматриваете предложение, принимаете MCA, если это необходимо, и завершите процесс извлечения.

    > [!IMPORTANT]
    > Необходимо иметь полномочия на подписание MCA от имени вашей организации. Если у вас нет таких полномочий, то кто-то должен выполнить этот шаг.

- После завершения оформления вы задаете дополнительные ссылки для настройки продуктов и услуг.

В следующей таблице содержатся термины и определения, которые отображаются в вашем предложении и на сайте предложений.

| **Термин** | **Определение** |
|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Счет для выставления счетов | Учетная запись, используемая для управления параметрами учетной записи, накладными, методами оплаты и продуктами. |
| Профиль выставления счетов | Сведения о вашей организации, которые позволяют настроить элементы, включаемые в счет, и способ оплаты счетов. Профиль выставления счетов включает имя учетной записи выставления счетов, способы оплаты, используемые для конкретного профиля выставления счетов, контактные данные, параметры накладных и разрешения, позволяющие изменить профиль выставления счетов, счета оплаты и приобрести продукты и службы. |
| Существующие соглашения | Любое соглашение, которое у вашей организации уже есть в корпорации Майкрософт. Это может быть, но не ограниченное, корпоративным соглашением, соглашением о службах & Майкрософт или соглашением о клиентах корпорации Майкрософт. |
| Соглашение о клиентах Майкрософт (MCA) | Соглашение, которое облагают условия и условия учетной записи, используемой Организацией в корпорации Майкрософт. |
| Представитель Майкрософт | Уполномоченный представитель Майкрософт, подготавливающий предложение для вас и вашей организации. |
| Организация | Юридическое лицо, использующее продукты, технологии или службы корпорации Майкрософт. |
| Подготовлено | Адрес электронной почты представителя корпорации Майкрософт, который подготовил предложение. |
| Дополнительные условия | Перемещается на MCA, который содержит термины, характерные для вашей организации. Чтобы принять дополнительные условия, необходимо использовать DocuSign для записи электронной подписи. |

## <a name="step-1-review-organization-information"></a>Шаг 1: Просмотр сведений об Организации

После входа в систему первое, что вы сделаете, просматриваете сведения об Организации.

### <a name="your-organization"></a>Организация

В разделе **ваша организация** отображается связанная с ним учетная запись выставления счетов. Сведения об учетной записи выставления счетов либо извлекаются из существующей учетной записи выставления счетов, либо создаются представителем корпорации Майкрософт. Если ваша организация является дочерней по отношению к другой организации, вы также увидите раздел **Организация по интересу** с именем и адресом этой Организации.

Если **ваша организация устанавливает** коммерческую связь с корпорацией Майкрософт в первый раз, а вы **еще не** подписаны MCA, обратитесь к представителю, чтобы внести изменения. После принятия MCA вы можете просмотреть и изменить адрес и контактные данные своей организации на странице [учетные записи выставления счетов](https://go.microsoft.com/fwlink/p/?linkid=2084771) в центре администрирования Microsoft 365. Если название организации изменяется, откройте запрос службы, чтобы он обновлялся. [Сведения о том, как открыть запрос на обслуживание](../admin/contact-support-for-business-products.md)

### <a name="your-information"></a>Ваши сведения

Если вы являетесь новым клиентом, введите свое имя, адрес электронной почты и номер **телефона, а**затем нажмите кнопку **сохранить**. Если вы используете существующего клиента, убедитесь, что ваши сведения верны. Чтобы внести исправления, выберите **Правка**, внесите необходимые изменения и нажмите кнопку **сохранить**.

Когда вы будете готовы, нажмите кнопку **продолжить** , чтобы перейти к следующему шагу.

## <a name="step-2-review-proposal"></a>Шаг 2: Проверка предложения

В этом предложении содержатся сведения об элементах, которые вы обсуждали у представителя корпорации Майкрософт. Вы можете переадресовать электронное письмо с помощью ссылки "предложение", чтобы поделиться им с другими заинтересованными лицами в вашей организации. Любой пользователь, использующий ссылку, имеет доступное только для чтения представление предложения.

Если вы хотите внести изменения в предложение после проверки, обратитесь к представителю Майкрософт.

### <a name="proposal-contents"></a>Содержимое предложения

Предложение содержит следующие сведения:

| Раздел | Описание |
|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Название организации | Имя Организации, для которой было подготовлено предложение. |
| Действителен до даты | Дата окончания срока действия предложения. Если вы пропустите этот срок, свяжитесь с представителем корпорации Майкрософт, чтобы сообщить вам, что вы по-прежнему заинтересованы в предложении. |
| Валюта | Валюта, используемая для расчета стоимости элементов в предложении. |
| Подготовлено для | Имя учетной записи выставления счетов, адрес, контактный адрес электронной почты и номер телефона пользователя, запросившего предложение. |
| Подготовлено | Адрес электронной почты представителя корпорации Майкрософт, который подготовил предложение. |
| Аннотация | Отображает промежуточные итоги, связанные с предложением. При необходимости он также будет показывать валюту внешнего сервера Exchange (FX), который используется для вычисления затрат. |
| Элементы строки предложения | В этом разделе содержатся количество, Цена за единицу и промежуточные итоги всех элементов, включенных в предложение. |
| Следующий шаг | В этом разделе указаны действия, которые необходимо выполнить. |

Чтобы подписать MCA, нажмите кнопку в разделе **следующий шаг**. Если вы хотите подписать дополнительные условия, ссылка переходит на сайт DocuSign, на котором выполняются действия для подписания документа.

После того как вы подписаны все необходимые соглашения или дополнительные условия, нажмите кнопку **Перейти, чтобы выполнить извлечение**.

## <a name="step-3-checkout"></a>Шаг 3: извлечение

Страница "извлечение" содержит следующие разделы:

### <a name="sold-to"></a>Продано

В этом разделе показан счет выставления счетов, используемый для предложения. Если вам нужно изменить какие бы то ни было сведения, перейдите по ссылке **Edit (изменить** ). Вы также можете использовать ссылку **изменить** , чтобы добавить Налоговый идентификатор организации. Налоговый код должен быть связан с страной, указанной в разделе **продано** . При наличии налогового освобождения необходимо открыть билет в службу поддержки, чтобы запросить статус налогового освобождения.

Чтобы узнать больше о налоговых кодах, а также о том, как применять их для статуса налогового освобождения, ознакомьтесь со статьей [Налоговая информация](billing-and-payments/tax-information.md).

### <a name="billed-to"></a>Выставлен счет на

В этом разделе показан профиль выставления счетов, используемый для определения элементов, включенных в счет, и оплаты счетов. Каждого цикла выставления счетов вы получаете отдельный счет для каждого профиля выставления счетов. Вы оплачиваете счета для выставления счетов, используя проверку или передачу по кабелю или предоплату Azure. Если у вас еще нет профиля выставления счетов, ваш представитель корпорации Майкрософт создает его. Во время извлечения можно выбрать другой профиль выставления счетов, если он есть, изменить имя профиля выставления счетов или добавить П.О. значение. Вы также можете создать новый профиль выставления счетов.

Сведения о профилях выставления счетов можно найти в статье [Управление профилями выставления счетов](billing-and-payments/manage-billing-profiles.md).

### <a name="proposal-items-in-this-order"></a>Элементы предложений в этом порядке

В этом разделе представлен список всех элементов, включенных в предложение. Список может включать одну или несколько из следующих категорий:

- **Дополнительные условия** Список всех поправок MCA, содержащих термины в Организации. Например, в этом списке могут содержаться термины "HIPAA" или "GDPR".

- **Купить сейчас** Список элементов, которые вы оплачиваете во время извлечения в конце рабочего процесса принятия предложения.

- **Скидки (применяется к будущим накладным расходов)** Список скидок, которые вы получаете в рамках предложения.

- **Включен** Список элементов, включенных в состав пакета предложений без дополнительных затрат. Некоторые из этих элементов могут иметь связанные с ними затраты в будущем.

### <a name="summary"></a>Аннотация

В этом разделе показано количество оплаченных товаров, промежуточный итог, предполагаемый налог и общая сумма заказа.

Чтобы разместить заказ, выберите пункт **разместить заказ** или **одобрить Соглашение о том, & заместить заказ**.

После размещения заказа вы получите подтверждение, чтобы выполнить следующие действия. Если вы приобрели план Azure, следующий шаг — настройка учетной записи выставления счетов на портале Azure.

## <a name="step-4-set-up-your-new-billing-account-azure-customers-only"></a>Шаг 4: Настройка новой учетной записи для выставления счетов (только для клиентов Azure)

Если вы являетесь новым клиентом и приобрели продукты Azure в рамках предложения, ваш следующий шаг — настройка новой учетной записи для выставления счетов. Сведения о том, как [настроить учетную запись выставления счетов для соглашения с клиентами корпорации Майкрософт](https://docs.microsoft.com/azure/cost-management-billing/manage/mca-setup-account).

Если вы уже используете клиент Azure с корпоративным соглашением, и вы подписываете MCA в первый раз, выполните следующие действия, чтобы узнать об изменениях между ними и выполнить задачи с помощью новой учетной записи выставления счетов. Чтобы узнать больше, ознакомьтесь [со статьей выполнение корпоративных соглашений в учетной записи выставления счетов для соглашения о клиентах корпорации Майкрософт](https://docs.microsoft.com/azure/cost-management-billing/manage/mca-enterprise-operations).

## <a name="understand-invoicing"></a>Общие сведения о выставлении счетов

После извлечения и выполнения заказа в течение 24-48 часов отправляется начальная накладная. После этого вы получаете счета в 5 – 5 раз в месяц. В месячном счете содержатся расходы за предыдущий месяц. Если у вас есть кредиты для вашей учетной записи, они вычитаются из денежных кредитов профиля выставления счетов и применяются к балансу накладной. Оставшийся баланс после применения кредитов — это задолженность. У вас 30 дней с даты выставления счетов, чтобы оплатить счет.

Инструкции по оплате для того, где следует отправлять проверку или проводки по кабелю, включены в PDF-копию счета. Чтобы просмотреть или скачать счет, ознакомьтесь со статьей [Просмотр счета или счета](billing-and-payments/view-your-bill-or-invoice.md).
