# <a name="add-storage-space-for-your-subscription"></a>Добавление дискового пространства для подписки

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet).

::: moniker-end

Если у вас заканчивается нехватка хранилища для ваших семейств сайтов SharePoint Online, вы можете добавить хранилище к своей подписке, если ваш план соответствует критериям. Если вы не видите **дополнительное хранилище файлов Office 365** в списке доступных надстроек, это означает, что ваш план не подходит. Чтобы получить дополнительные сведения, ознакомьтесь с [различной схемой.](#is-my-plan-eligible-for-office-365-extra-file-storage)

> [!NOTE]
> Если вы приобрели подписку по программе корпоративного лицензирования или поставщику служб шифрования, вы не сможете приобрести **дополнительное хранилище файлов Office 365** для вашей организации непосредственно у корпорации Майкрософт. Обратитесь за помощью к своему представителю или партнеру.

## <a name="view-available-storage"></a>Просмотр доступного хранилища

::: moniker range="o365-worldwide"

1. Перейдите на страницу ["Активные сайты" в новом Центре администрирования SharePoint](https://admin.microsoft.com/sharepoint?page=siteManagement&modern=true) и войдите, используя учетную запись с [правами администратора](https://docs.microsoft.com/sharepoint/sharepoint-admin-role) в вашей организации.

2. В правом верхнем углу страницы отображается объем хранилища, используемого для всех сайтов, и общий объем хранилища для вашей подписки. (Если в организации настроена поддержка нескольких регионов в Office 365, на панели также показывается объем хранилища, используемого для всех географических расположений.) 

::: moniker-end

::: moniker range="o365-germany"

1. Войдите в https://portal.office.de систему как глобальный администратор или администратор SharePoint, а затем выберите плитку администратор, чтобы открыть центр администрирования. (Если отображается сообщение о том, что у вас нет разрешения на доступ к странице, у вас нет прав администратора Microsoft 365 в вашей организации.)

2. В панели слева в разделе **Центры администрирования** выберите **SharePoint**. Если откроется классическая версия Центра администрирования SharePoint, нажмите кнопку **Открыть** в верхней части страницы, чтобы открыть новый Центр администрирования SharePoint.

3. На левой панели нового Центра администрирования SharePoint выберите пункт **Активные сайты**.

4. В правом верхнем углу страницы отображается объем хранилища, используемого для всех сайтов, и общий объем хранилища для вашей подписки.

::: moniker-end

::: moniker range="o365-21vianet"

1. Войдите в https://login.partner.microsoftonline.cn/ систему как глобальный администратор или администратор SharePoint, а затем выберите плитку администратор, чтобы открыть центр администрирования. (Если отображается сообщение о том, что у вас нет разрешения на доступ к странице, у вас нет прав администратора Microsoft 365 в вашей организации.)

2. В панели слева в разделе **Центры администрирования** выберите **SharePoint**. Если откроется классическая версия Центра администрирования SharePoint, нажмите кнопку **Открыть** в верхней части страницы, чтобы открыть новый Центр администрирования SharePoint.

3. На левой панели нового Центра администрирования SharePoint выберите пункт **Активные сайты**.

4. В правом верхнем углу страницы отображается объем хранилища, используемого для всех сайтов, и общий объем хранилища для вашей подписки.  

::: moniker-end

![Панель хранения на странице активных сайтов](https://docs.microsoft.com/sharepoint/sharepointonline/media/active-sites-storage-bar.png)

> [!NOTE]
> Используемое хранилище не включает изменения, внесенные в течение последних 24-48 часов.

Определив объем используемого хранилища, вы можете добавить или удалить дисковое пространство для своей подписки. Чтобы узнать, сколько затрат будет стоить при добавлении дискового пространства, выполните действия, описанные в этой статье, и ознакомьтесь со сведениями о ценах перед покупкой.
  
Сведения о том, как задать пределы хранения для семейства веб-сайтов, можно найти в разделе [Manage site Limit Storage Limits](https://docs.microsoft.com/sharepoint/manage-site-collection-storage-limits).
  
## <a name="add-storage-to-your-subscription"></a>Добавление хранилища к подписке

Если вы еще не приобрели дополнительное хранилище для своей подписки, это можно сделать.

::: moniker range="o365-worldwide"

1. В центре администрирования перейдите на страницу Услуги по **выставлению счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">покупки</a> .

2. В нижней части страницы **услуги покупки** выберите **надстройки**.

3. Выберите **Office 365 Extra Storage File**.

4. На странице **Office 365 Extra Storage File** , если она отображается, выберите базовую подписку, а затем введите количество гигабайтов хранилища, которое вы хотите добавить.

5. Нажмите **кнопку извлечь.**

6. На странице **как это выглядит?** проверьте выбранное количество гигабайтов хранилища, просмотрите сведения о ценах, а затем нажмите кнопку **Далее**.

7. На странице **полный заказ** проверьте итоговое значение. Если требуется внести изменения, выберите пункт **изменить заказ**. Если для заказа требуется проверка кредита, установите этот флажок. Когда все будет готово, нажмите кнопку **поместить заказ** \> **перейдите на домашнюю страницу администратора**.

::: moniker-end

::: moniker range="o365-germany"

2. На странице " **подписки** " выберите подписку, в которую вы хотите добавить дисковое пространство, а затем выберите **надстройки**.

3. Выберите **купить надстройки**.

5. Введите необходимое количество пользовательских лицензий и, если это отображается, выберите базовую подписку. Нажмите **кнопку извлечь.**
  
6. На странице **как это выглядит?** проверьте выбранное количество гигабайтов хранилища, просмотрите сведения о ценах, а затем нажмите кнопку **Далее**.

7. На странице **полный заказ** выберите пункт **разместить заказ**.

## <a name="increase-or-decrease-storage"></a>Увеличение или уменьшение хранилища

Если вы уже приобрели дополнительное хранилище файлов с помощью надстройки **Office 365 Extra Storage** , вы можете использовать эти действия, чтобы увеличить или уменьшить дополнительное место для подписки. Вы можете уменьшить объем дискового пространства, как минимум 1 ГБ. Чтобы удалить все лишнее дисковое пространство, необходимо обратиться в [службу поддержки](../admin/contact-support-for-business-products.md).

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.

2. Выберите подписку, которая содержит надстройку **Office 365 Extra Storage** .

3. Выберите **надстройки**, а затем нажмите **изменить количество**.

4. В области **Добавить/удалить гигабайты** введите общее количество гигабайтов, которое требуется для подписки, а затем нажмите кнопку **послать изменение**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \><a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

2. На странице **подписки** выберите **надстройки**.

    ![Add-ons button used to purchase add-ons.](../media/b4d2beb4-4f6d-435a-b127-01ceebd6eebf.png)
  
    > [!NOTE]
    > Если вы не видите **надстройки**, а ваша подписка была приобретена в партнере, выберите **центр обслуживания корпоративного лицензирования (VLSC)**.
  
3. В разделе **Office 365 дополнительное хранилище файлов**выберите **изменить количество**.

    ![Ссылка "Изменить количество".](../media/96473f2b-6ff6-45ec-b1a3-d7b204ac1f6e.png)
  
4. В правой области введите общее количество гигабайтов, которое вам нужно, а затем нажмите кнопку **послать**.

    Например, если у вас есть 200 ГБ дополнительного места, а вам нужно только 100 ГБ, в поле следует ввести **100**.

5. Нажмите **Закрыть**.

::: moniker-end

## <a name="is-my-plan-eligible-for-office-365-extra-file-storage"></a>Доступна ли в моем плане надстройка "Office 365 с дополнительным хранилищем"?

Надстройка "Office 365 с дополнительным хранилищем" доступна для таких подписок:
  
- Office 365 для предприятий E1

- Office 365 для предприятий E2

- Office 365 для предприятий E3

- Office 365 корпоративный E4

- Office 365 корпоративный E5

- Office для Интернета с SharePoint (план 1)

- Office для Интернета с SharePoint (план 2)

- SharePoint Online (план 1)

- SharePoint Online (план 2)

- Microsoft 365 бизнес базовый

- Microsoft 365 бизнес стандарт

- Microsoft 365 бизнес премиум

- Microsoft 365 E3

- Microsoft 365 E5

- Microsoft 365 F1

> [!NOTE]
> Office 365 дополнительное хранилище файлов также доступно для планов GCC, GCC High и DOD.
