# <a name="add-storage-space-for-your-subscription"></a>Добавление места для хранения подписки

Если у вас заканчивается нехватка хранилища для ваших семейств сайтов SharePoint Online, вы можете добавить хранилище к своей подписке, если ваш план соответствует критериям. Если вы не видите **Office 365 с дополнительным хранилищем доступных** дополнениях, это означает, что ваш план не имеет права. Для получения дополнительной информации [см.](#is-my-plan-eligible-for-office-365-extra-file-storage)

> [!NOTE]
> Если вы купили подписку через Корпоративное лицензирование или CSP, вы не можете **купить Office 365 с дополнительным хранилищем** организации непосредственно у корпорации Майкрософт. Обратитесь за помощью к своему представителю или партнеру.

## <a name="before-you-begin"></a>Подготовка к работе

Вы должны быть глобальным или SharePoint администратором для выполнения задач в этой статье. Дополнительные сведения см. в статье [О ролях администраторов](../admin/add-users/about-admin-roles.md).

## <a name="view-available-storage"></a>Просмотр доступного хранилища

1. В центре SharePoint, перейдите на страницу <a href="https://admin.microsoft.com/sharepoint?page=siteManagement&modern=true" target="_blank">Активных сайтов</a> и войдите в него с учетной записью, [которая имеет разрешения администратора](/sharepoint/sharepoint-admin-role) для вашей организации.

2. В правом верхнем углу страницы отображается объем места в хранилище, используемый всеми сайтами, и общий объем хранилища, доступный для вашей подписки. Если ваша организация настроила Multi-Geo Office 365, на баре также показано количество хранилища, используемого во всех геолокациях.

   ![Панель "Хранилище" на странице "Активные сайты"](/sharepoint/sharepointonline/media/active-sites-storage-bar.png)

   > [!NOTE]
   > Значение используемого объема хранилища не учитывает изменения, сделанные за последние 24–48 часов.

После определения того, сколько хранилища вы используете, вы можете добавить или удалить место для хранения подписки. Чтобы узнать, сколько это будет стоить, чтобы добавить пространство для хранения, следуйте шагам в этой статье, и просмотрите информацию о ценах, прежде чем купить больше.
  
Для получения информации об установлении пределов хранения коллекции сайта [см.](/sharepoint/manage-site-collection-storage-limits)
  
## <a name="add-storage-to-your-subscription"></a>Добавление хранилища в подписку

Если вы еще не купили дополнительное хранилище для подписки, вы можете это сделать.

1. В центре администратора перейдите  на страницу \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">биллинговых услуг.</a>
2. В нижней части страницы **услуг покупки,** в **разделе Дополнения,** найти **Office 365 с дополнительным хранилищем,** и выберите Подробная **информация**.
3. На странице детали продукта, выберите **Следующий**.
4. При необходимости выберите базовую подписку, а затем введите количество гигабайт памяти, которые вы хотите добавить.
5. Выберите **Проверьте сейчас**.
6. На странице **Как это выглядит?** прояснять количество выбранных вами гигабайт памяти, просмотреть информацию о ценах, а затем выбрать **Next**.
7. На странице **Полного заказа** проявим общее количество. Если вам нужно внести какие-либо изменения, выберите **заказ Edit.** Если заказ требует проверки кредитоспособно, выберите чековую коробку. По его словам, выберите заказ **Место** \> **Перейти к администратору Главная**.

## <a name="increase-or-decrease-storage"></a>Увеличение или уменьшение хранилища

Если вы уже купили дополнительное хранилище файлов **через надстройку Office 365 с дополнительным хранилищем,** вы можете использовать эти шаги для увеличения или уменьшения дополнительного места для хранения подписки. Вы можете уменьшить объем хранилища до 1 гигабайта. Чтобы удалить все дополнительное пространство для хранения, обратитесь [в поддержку.](../business-video/get-help-support.md)

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. Во **вкладке** Продукты выберите подписку, **Office 365 с дополнительным хранилищем** надстройку.
3. На странице деталей продукта в **разделе Дополнения** выберите **дополнения Manage.**
4. В **панели надстройок** Manage из **списка Add-on** выберите **Office 365 с дополнительным хранилищем.**
5. В **текстовом поле** Quantity введите количество ГБ складских помещений, которые вы хотите для подписки.
6. Нажмите **Сохранить**.

## <a name="is-my-plan-eligible-for-office-365-extra-file-storage"></a>Доступна ли в моем плане надстройка "Office 365 с дополнительным хранилищем"?

Надстройка "Office 365 с дополнительным хранилищем" доступна для таких подписок:
  
- Office 365 для предприятий E1
- Office 365 для предприятий E2
- Office 365 для предприятий E3
- Office 365 корпоративный E4
- Office 365 корпоративный E5
- Office для Интернета с SharePoint плана 1
- Office для Интернета с SharePoint план 2
- SharePoint Online (план 1)
- SharePoint Online (план 2)
- Microsoft 365 бизнес базовый
- Microsoft 365 бизнес стандарт
- Microsoft 365 бизнес премиум
- Microsoft 365 E3
- Microsoft 365 E5
- Microsoft 365 F1

> [!NOTE]
> Office 365 с дополнительным хранилищем также доступна для GCC, GCC High и DOD.
