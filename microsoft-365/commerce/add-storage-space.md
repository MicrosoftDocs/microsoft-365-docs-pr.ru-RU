# <a name="add-storage-space-for-your-subscription"></a>Добавление места в хранилище для подписки

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

::: moniker-end

Если у вас заканчивается нехватка хранилища для ваших семейств сайтов SharePoint Online, вы можете добавить хранилище к своей подписке, если ваш план соответствует критериям. Если вы не видите дополнительное хранилище файлов **Office 365** в списке доступных надстройок, это означает, что ваш план не имеет права. Дополнительные сведения см. в [подходящем плане?](#is-my-plan-eligible-for-office-365-extra-file-storage)

> [!NOTE]
> Если вы приобрели подписку через корпоративное лицензирование или CSP, вы не сможете приобрести дополнительное хранилище файлов **Office 365** для своей организации непосредственно у корпорации Майкрософт. Обратитесь за помощью к представителю или партнеру.

## <a name="before-you-begin"></a>Прежде чем начать

Для выполнения задач в этой статье необходимо быть глобальным администратором или администратором SharePoint. Дополнительные сведения см. в статье [О ролях администраторов](../admin/add-users/about-admin-roles.md).

## <a name="add-storage-to-your-subscription"></a>Добавление хранилища в подписку

Если вы еще не приобрели дополнительное хранилище для подписки, это можно сделать.

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **"Приобретение** услуг \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">вы выставления счета".</a>
2. В нижней части страницы "Приобретение **служб"** выберите **"Надстройки".**
3. Выберите дополнительное хранилище файлов **Office 365.**
4. На странице "Дополнительное хранилище файлов **Office 365",** если она отображается, выберите базовую подписку, а затем введите количество гигабайт хранилища, который вы хотите добавить.
5. Выберите **"Проверить" сейчас.**
6. На странице "Как это **выглядит?"** проверьте количество выбранных гигабайт хранилища, просмотрите сведения о ценах и выберите **"Далее".**
7. На странице **"Полный заказ"** проверьте итог. Если необходимо внести какие-либо изменения, выберите **"Изменить порядок".** Если заказ требует проверки кредитоспособности, выберите этот ящик. Когда все будет готово, выберите **"Place order** \> **Go to Admin Home".**

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **"Подписки** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">на выставление счета".</a>  

2. На странице **"Подписки"** выберите подписку, в которую нужно добавить место, а затем выберите **"Надстройки".**

    ![Add-ons button used to purchase add-ons.](../media/b4d2beb4-4f6d-435a-b127-01ceebd6eebf.png)
  
    > [!NOTE]
    > Если вы не видите надстроек **и** ваша подписка приобретена у партнера, выберите Volume **Licensing Service Center (VLSC).**
  
3. Выберите **"Купить надстройки".**

    ![Ссылка "Купить надстройки" на странице "Подписки" в Центре администрирования.](../media/f5cbc3fa-90f7-4299-976d-2482f2c69755.png)
  
4. On the **Purchase services** page, mouse over or tap **Office 365 Extra File Storage**, then select Buy **now**.
  
5. Введите необходимое количество пользовательских лицензий и, если они показано, выберите базовую подписку. Выберите **"Проверить" сейчас.**
  
6. На странице "Как это **выглядит?"** проверьте количество выбранных гигабайт хранилища, просмотрите сведения о ценах и выберите **"Далее".**

7. На странице **"Полный заказ"** выберите **"Place order" (Разместить заказ).**

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.

## <a name="increase-or-decrease-storage"></a>Увеличение или уменьшение хранилища

Если вы уже приобрели дополнительное хранилище файлов с помощью надстройки "Дополнительное хранилище файлов **Office 365",** с помощью этих действий можно увеличить или уменьшить дополнительное место в хранилище для подписки. Вы можете уменьшить объем хранилища до 1 гигабайта. Чтобы удалить все дополнительное место в хранилище, [обратитесь в службу поддержки.](../admin/contact-support-for-business-products.md)

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" выберите подписку, которая содержит надстройку "Дополнительное хранилище файлов **Office 365".**
3. На странице сведений о продукте в **разделе** "Надстройки" выберите "Управление **надстройки".**
4. В области **"Управление надстройки"** в списке надстройки выберите "Дополнительное хранилище файлов **Office 365".** 
5. В **текстовом поле** "Количество" введите количество ГБ пространства для хранения, которое необходимо для подписки.
6. Нажмите **Сохранить**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

2. На странице **"Подписки"** выберите **"Надстройки".**

    ![Add-ons button used to purchase add-ons.](../media/b4d2beb4-4f6d-435a-b127-01ceebd6eebf.png)
  
    > [!NOTE]
    > Если вы не видите надстроек **и** ваша подписка приобретена у партнера, выберите Volume **Licensing Service Center (VLSC).**
  
3. В области "Дополнительное хранилище файлов **Office 365"** выберите **"Изменить количество".**

    ![Ссылка "Изменить количество".](../media/96473f2b-6ff6-45ec-b1a3-d7b204ac1f6e.png)
  
4. В правой области введите необходимое количество гигабайт, а затем выберите **"Отправить".**

    Например, если у вас есть 200 ГБ дополнительного места, а вам нужно только 100 ГБ, в поле следует ввести **100**.

5. Нажмите **Закрыть**.

::: moniker-end

## <a name="is-my-plan-eligible-for-office-365-extra-file-storage"></a>Доступна ли в моем плане надстройка "Office 365 с дополнительным хранилищем"?

Надстройка "Office 365 с дополнительным хранилищем" доступна для таких подписок:
  
- Office 365 для предприятий E1

- Office 365 для предприятий E2

- Office 365 для предприятий E3

- Office 365 корпоративный E4

- Office 365 корпоративный E5

- Office для Интернета с SharePoint (план 1)

- Office для Интернета с SharePoint (план 2)

- SharePoint Online (план 1)

- SharePoint Online (план 2)

- Microsoft 365 бизнес базовый

- Microsoft 365 бизнес стандарт

- Microsoft 365 бизнес премиум

- Microsoft 365 E3

- Microsoft 365 E5

- Microsoft 365 F1

> [!NOTE]
> Дополнительное хранилище файлов Office 365 также доступно для планов GCC, GCC High и DOD.
