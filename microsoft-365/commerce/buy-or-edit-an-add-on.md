# <a name="buy-or-manage-add-ons"></a>Покупка и управление надстройки

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

::: moniker-end

Во многих планах Microsoft 365 для бизнеса есть надстройки, которые можно приобрести для подписок. Надстройки предоставляют дополнительные возможности для улучшения работы с подпиской.

Существует два типа надстройки:

1. **Традиционные надстройки** связаны с определенной подпиской. При отмене подписки связанная надстройка также отменяется.
2. **Отдельные подписки на** надстройки отображаются  на странице "Ваши продукты" как отдельная подписка с собственной датой окончания срока действия. В отличие от традиционных надстройки, отдельные подписки на надстройки не связаны с определенной подпиской. Управление подпиской на автономные надстройки происходит так же, как и с любой другой подпиской.

## <a name="before-you-begin"></a>Прежде чем начать

Для выполнения описанных в этой статье действий необходимы права глобального администратора или администратора по выставлению счетов. Дополнительные сведения см. в статье [О ролях администраторов](../admin/add-users/about-admin-roles.md).

## <a name="buy-an-add-on"></a>Приобретение надстройки

Чтобы найти и приобрести надстройки, с помощью следующих действий. Этот список включает только те надстройки, которые вам доступны. Этот список основан на текущих активных подписках и любых других существующих надстройки, которые у вас уже есть. Если надстройки нет в списке, ее нельзя приобрести.

Сведения о надстройках для Skype для бизнеса см. в статье [Лицензирование надстроек Skype для бизнеса и Microsoft Teams](https://docs.microsoft.com/SkypeForBusiness/skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing).

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **"Приобретение** услуг \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">вы выставления счета".</a>
2. В нижней части страницы "Приобретение **служб"** выберите **"Надстройки".**
3. На странице **"Приобретение служб"** выберите надстройку, которую вы хотите приобрести.
4. На странице сведений о продукте выберите **"Купить".**
5. При необходимости заполнять любые дополнительные сведения о заказе. Эти сведения могут включать выбор подписки для надстройки. По завершению выберите **"Проверить".**
6. При необходимости просмотрите сведения о заказах и ценах, а затем выберите **"Далее".**
7. Выберите способ оплаты или добавьте новый, а затем выберите "Place **order** > **Go to Admin Home".**

::: moniker-end

## <a name="manage-an-add-on"></a>Управление надстройка

Вы управляете подпиской на автономные надстройки так же, как управляете любой другой подпиской на странице **"Ваши продукты".** Например, вы можете купить или удалить лицензии, управлять [](billing-and-payments/manage-payment-methods.md) повторяющимся выставлением [счета,](subscriptions/renew-your-subscription.md)а также изменить или заменить способы оплаты для автономных подписок на надстройки. [](licenses/buy-licenses.md)

Вы управляете традиционной надстройой, переходить к подписке, содержаной ее, и вносить в нее изменения. Чтобы управлять традиционной надстройой, с помощью следующих действий.
  
::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" выберите подписку, содержаную надстройку, которую нужно обновить.
3. На странице сведений о подписке **в** разделе "Надстройки" перечислены купленные вами надстройки. Выберите **"Управление надстройки".**
4. В области **"Управление надстройки"** в выпадаемом списке надстройки выберите надстройку, которую нужно изменить. 
5. Введите изменение количества для надстройки и выберите **"Сохранить".**

::: moniker-end

1. В Центре администрирования перейдите на страницу **Выставление счетов** \><a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

## <a name="remove-an-add-on"></a>Удаление надстройки

Чтобы удалить подписку на автономные надстройки, отмените [ее,](subscriptions/cancel-your-subscription.md)как и любую другую подписку. Вы также можете [отключить повторяющееся выставление счета.](subscriptions/renew-your-subscription.md)

Вы не можете удалить традиционную надстройки. Если вы хотите удалить традиционную надстройка, обратитесь за помощью в [службу поддержки.](../admin/contact-support-for-business-products.md)
