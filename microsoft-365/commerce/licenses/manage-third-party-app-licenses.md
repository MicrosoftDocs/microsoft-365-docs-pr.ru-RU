# <a name="manage-isv-app-licenses-in-the-microsoft-365-admin-center"></a>Управление лицензиями приложений isV в центре Microsoft 365 администрирования

Приложение независимого поставщика программного обеспечения (ISV) — это приложение, которое вы покупаете у поставщика программного обеспечения, кроме Microsoft.

## <a name="before-you-begin"></a>Прежде чем начать

Чтобы назначать лицензии необходимо быть глобальным администратором либо администратором лицензией или пользователей. Дополнительные сведения см. в статье [О ролях администраторов](../../admin/add-users/about-admin-roles.md).

## <a name="assign-isv-app-licenses-to-users-or-groups"></a>Назначение лицензий приложений isV пользователям или группам

1. В центре Microsoft 365 администрирования перейдите на страницу  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">"Лицензии на выставление счета".</a>
2. Выберите приложение, для которое необходимо назначить лицензии.
3. На странице Сведения о лицензии выберите **Назначение лицензий**.
4. В области **Назначение лицензий** начните вводить имя пользователя или группы, а затем выберите его из результатов, чтобы добавить его в список.
5. После завершения нажмите **Назначить**, затем — **Закрыть**.

## <a name="unassign-isv-app-licenses-from-users-or-groups"></a>Unassign isV app licenses from users or groups

1. В центре администрирования страница **Биллинг**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">Лицензии.</a>
2. Выберите приложение, для которое необходимо отогнать лицензии.
3. На странице сведения о лицензии выберите пользователей и группы для удаления лицензий, а затем выберите лицензии Unassign.
4. В диалоговом окне подтвердите, что необходимо удалить лицензии, затем выберите Unassign.

## <a name="add-or-remove-isv-app-licenses-for-your-account"></a>Добавление или удаление лицензий приложения isV для учетной записи

Лицензии приложений ISV управляются поставщиком приложений. Обратитесь к поставщику, чтобы добавить или удалить лицензии для вашей учетной записи.

## <a name="next-steps"></a>Дальнейшие действия

В зависимости от купленного приложения ISV следующим шагом может стать установка приложения в среду организации. Установка приложения делает его доступным для пользователей. Для установки приложения ISV в среду используйте следующие действия.

1. В центре администрирования страница **Биллинг**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">Лицензии.</a>
2. Выберите приложение, которое необходимо установить в учетную запись.
3. На странице сведения о лицензии выберите **Установите этот продукт.** Вы перенаправляетсяе на другой сайт платформы, где вы устанавливаете приложение в свою среду.
