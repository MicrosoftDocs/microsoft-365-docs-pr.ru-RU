# <a name="add-licenses-to-a-subscription-purchased-through-the-volume-licensing-service-center"></a>Добавление лицензий для подписки, приобретенной в Центре обслуживания для корпоративного лицензирования

Если вы приобрели microsoft 365 для бизнеса или корпоративный план Office 365 через сторонного партнера, необходимо приобрести дополнительные лицензии через этого партнера. Если Майкрософт известно о партнере, у которого была приобретена подписка, вы можете узнать, как связаться с ним, выполнив следующие действия.
  
::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Ваши продукты</a>.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Ваши продукты</a>.
::: moniker-end

2. Выберите подписку, на которую необходимо добавить лицензии, а затем выберите ссылку Центр службы лицензирования тома **(VLSC).**

3. Выполните действия, описанные в VLSC, для завершения покупки.
