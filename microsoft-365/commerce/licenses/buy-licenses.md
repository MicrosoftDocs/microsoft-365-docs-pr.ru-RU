# <a name="buy-or-remove-licenses"></a>Покупка и удаление лицензий

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

::: moniker-end

Вы можете приобрести дополнительные лицензии или уменьшить количество лицензий для своих подписок, вы предприняв следующие действия.

## <a name="before-you-begin"></a>Перед началом работы

- Для выполнения задач, описанных в этой статье, необходимо быть глобальным администратором или администратором вы выставления счета. Дополнительные сведения см. в статье [О ролях администраторов](../../admin/add-users/about-admin-roles.md).
- Вы можете [добавлять пользователей и назначать лицензии одновременно.](../../admin/add-users/add-users.md)
- Если вы приобрели план Microsoft 365 для бизнеса или Office 365 корпоративный через стороного партнера, вам необходимо приобрести дополнительные лицензии у этого партнера.

## <a name="watch-buy-new-licenses"></a>Watch: Buy new licenses

Если пользователи, для которых вы покупаете лицензии, еще не являются активными пользователями в вашей организации, следующим шагом будет добавление пользователей и одновременное назначение [лицензий.](../../admin/add-users/add-users.md)

## <a name="watch-remove-existing-licenses"></a>Watch: Remove existing licenses

Если вы удалили лицензии из подписки, необходимо удалить пользователей [из вашей организации.](../../admin/add-users/delete-a-user.md)

## <a name="buy-or-remove-licenses-for-your-business-subscription"></a>Покупка и удаление лицензий для подписки на бизнес

::: moniker range="o365-worldwide"

1. В Центре администрирования Microsoft 365 перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" найдите подписку, для которой вы хотите приобрести или удалить лицензии. Выберите **дополнительные действия** (три точки), а затем выберите **"Купить лицензии".** [Что делать, если я не вижу кнопки "Купить лицензии" или "Удалить лицензии"?](#what-if-i-dont-see-the-buy-licenses-or-remove-licenses-buttons)
3. Если вы хотите уменьшить количество лицензий, в  верхней части области "Купить лицензии" выберите "Удалить **лицензии".**
4. Чтобы приобрести или удалить  лицензии,  в поле "Новое количество" введите общее количество лицензий для этой подписки. Например, если у вас 100 лицензий и вы хотите добавить еще пять, введите 105. Если вы хотите удалить пять из них, введите 95.
5. Нажмите **Сохранить**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.
2. На странице **"Подписки"** выберите подписку, для которой вы хотите приобрести или удалить лицензии, а затем выберите **"Добавить/удалить лицензии".** [Что делать, если я не вижу кнопки "Купить лицензии" или "Удалить лицензии"?](#what-if-i-dont-see-the-buy-licenses-or-remove-licenses-buttons)
3. В поле **"Всего лицензий"** введите общее количество лицензий, которое необходимо для этой подписки, а затем выберите **"Отправить** \> **закрыть".** Например, если у вас 100 лицензий и вы хотите добавить еще пять, введите 105. Если вы хотите удалить пять лицензий, введите 95.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.
2. На странице **"Подписки"** выберите подписку, для которой вы хотите приобрести или удалить лицензии, а затем выберите **"Добавить/удалить лицензии".** [Что делать, если я не вижу кнопки "Купить лицензии" или "Удалить лицензии"?](#what-if-i-dont-see-the-buy-licenses-or-remove-licenses-buttons)
3. В поле **"Всего лицензий"** введите общее количество лицензий, которое необходимо для этой подписки, а затем выберите **"Отправить** \> **закрыть".** Например, если у вас 100 лицензий и вы хотите добавить еще 5, введите 105. Если вы хотите удалить 5 из них, введите 95.

::: moniker-end

> [!NOTE]
> Вы не можете уменьшить количество лицензий для вашей подписки, если все лицензии в настоящее время назначены пользователям. Чтобы уменьшить количество лицензий, сначала отоберите одну или несколько лицензий у [пользователей,](../../admin/manage/remove-licenses-from-users.md)а затем удалите лицензии из подписки.

## <a name="when-will-the-new-licenses-be-available-to-assign"></a>Когда будут доступны новые лицензии для назначения?

Способ оплаты, связанный с подпиской или профилем вы выставления счета, взимается, как только вы приобретите дополнительные лицензии для подписки. Лицензии сразу же доступны для назначения пользователям.

Если вы предоплатите подписку с помощью ключа продукта, вы можете добавить дополнительные лицензии с помощью другого ключа продукта, кредитной или дебетовой карты или банковского счета для оплаты дополнительных расходов на новые лицензии. Если ваша подписка предоплачена, вы не сможете удалить лицензии.

## <a name="how-does-buying-or-removing-licenses-affect-my-billing-statements"></a>Как приобретение или удаление лицензий влияет на выписки по счету?

- Лицензии, добавленные в середине периода выставления счетов, отображаются в следующем счете. Если вы платите ежегодно, счета за эти изменения выставляются в течение месяца.
- В следующей выписке по счету вычитается предыдущий платеж за исходное количество лицензий. Мы добавляем ратный сбор за период времени с исходным количеством лицензий и взимаем плату за новое количество лицензий. Кроме того, существует плата за текущее количество лицензий за оставшийся период вы выставления счета.

## <a name="next-steps"></a>Дальнейшие действия

Если вы приобрели дополнительные лицензии для своей подписки, вам следует назначить эти лицензии пользователям [в организации.](../../admin/manage/assign-licenses-to-users.md)

Если вы снизили количество лицензий для вашей подписки из-за того, что кто-то покинул вашу организацию, может потребоваться удалить учетную запись этого пользователя. Дополнительные узнать см. в [подстраховок "Удаление бывшего сотрудника".](../../admin/add-users/remove-former-employee.md)
