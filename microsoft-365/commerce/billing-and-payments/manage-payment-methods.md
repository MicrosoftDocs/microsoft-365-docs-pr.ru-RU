# <a name="manage-payment-methods"></a>Управление методами оплаты

::: moniker range="o365-21vianet"
> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).
::: moniker-end

При покупке бизнес-продуктов или служб у корпорации Майкрософт можно использовать существующий способ оплаты или добавить новый. Для оплаты покупки можно использовать кредитную или дебетовую карту или банковский счет.

Если у вашей бизнес-учетной записи есть профиль выставления счетов, а вы владелец профиля выставления счетов или участник профиля выставления счетов, вы можете использовать профиль выставления счетов, который используется с помощью кредитной карты или счета-фактуры, для покупки или оплаты счетов. Если вы менеджер по выставлению счетов, для оплаты счетов можно использовать только профиль выставления счетов. Дополнительные данные о профилях и ролях вы выставления счета см. в под управлением [профилей вы выставления счета.](manage-billing-profiles.md)

Если у вашей бизнес-учетной записи нет профиля вычета, любой глобальный администратор или администратор вычета может управлять любым банковским счетом, добавленным на бизнес-счет, и использовать его. Однако вы можете управлять только добавляемой кредитной картой или использовать ее.

> [!NOTE]
> Возможность оплаты с помощью банковского счета недоступна в некоторых странах или регионах.
>
> Необходимо использовать способ оплаты, выданный из той же страны, что и клиент.

## <a name="before-you-begin"></a>Прежде чем начать

Для выполнения задач в этой статье необходимо быть глобальным администратором или администратором вы выставления счета. Дополнительные сведения см. в статье [О ролях администраторов](../../admin/add-users/about-admin-roles.md).

## <a name="add-a-payment-method"></a>Добавление способа оплаты

Добавление метода оплаты не связывает с ним подписки. Чтобы назначить одну подписку методу оплаты, см. "Изменение [метода оплаты для одной подписки".](#change-a-payment-method-for-a-single-subscription) Чтобы заменить все подписки, которые используют другой метод оплаты, на новый, см. ссылку ["Замена метода оплаты".](#replace-a-payment-method)

1. В центре администрирования перейдите в раздел **Выставление счетов** > **Счета и платежи** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Методы оплаты</a>.
2. Выберите элемент **Добавление метода оплаты**.
3. На странице **Методы оплаты** выберите метод оплаты в раскрывающемся меню.
4. Введите сведения для новой карты или банковского счета, а затем выберите **"Добавить".**

## <a name="update-payment-method-details"></a>Обновление сведений о способе оплаты

Вы можете изменить имя на кредитной или дебетовой карте, адрес для вычета или дату окончания срока действия существующего способа оплаты. Однако изменить карту или номер учетной записи нельзя. Если номер учетной записи изменился, [замените](#replace-a-payment-method)его другим методом оплаты, а затем [удалите старый.](#delete-a-payment-method)

1. В центре администрирования перейдите в раздел **Выставление счетов** > **Счета и платежи** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Методы оплаты</a>.
2. Выберите строку метода оплаты, который нужно обновить. В области справа выберите **Изменить**.
3. Обновите данные метода оплаты, включая имя на кредитной или дебетовой карте, адрес выставления счетов или дату срока действия, затем нажмите кнопку **Сохранить**.

## <a name="replace-a-payment-method"></a>Замена способа оплаты

При замене метода оплаты он заменяется для всех подписок и профилей вы выставления счета, которые используют один и тот же метод оплаты. Замена метода оплаты не удаляет существующий метод оплаты. Он по-прежнему доступен для выбора и использования для других подписок и профилей вы выставления счета.

Чтобы изменить способ оплаты для одной подписки, см. "Изменение [метода оплаты для одной подписки".](#change-a-payment-method-for-a-single-subscription)

1. В центре администрирования перейдите в раздел **Выставление счетов** > **Счета и платежи** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Методы оплаты</a>.
2. Выберите строку метода оплаты, который нужно заменить. В области справа выводится список всех профилей выставления счетов и отдельных подписок, использующих выбранный метод оплаты.
3. В области справа выберите **Замена способа оплаты для всех элементов**
4. Чтобы использовать существующий способ оплаты, выберите его из раскрывающегося списка, а затем нажмите **Заменить**
    > [!NOTE]
    > Для оплаты подписок, связанных с профилем выставления счетов, можно использовать только кредитную или дебетовую карту. Банковские учетные записи, перечисленные на странице **Способы оплаты**, недоступны в раскрывающемся списке.
5. Чтобы добавить новый способ оплаты, выберите **Добавить метод оплаты**.
6. В области **Добавление метода оплаты** введите сведения об учетной записи и нажмите **Сохранить**.. Необходимо использовать способ оплаты из той же страны, что и у вашего клиента.
7. Новый способ оплаты уже выбран в раскрывающемся списке. Нажмите кнопку **Заменить**.

## <a name="change-a-payment-method-for-a-single-subscription"></a>Изменение способа оплаты для одной подписки

Вы можете изменить способ оплаты, используемый для оплаты одной подписки.

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" найдите подписку, которую вы хотите оплатить с помощью альтернативного способа оплаты.
3. Выберите **дополнительные действия** (три точки), а затем выберите **метод оплаты Replace.**
4. В области **"Заменить** метод оплаты" в выпадаемом списке выберите альтернативный способ оплаты или добавьте метод оплаты.
5. При добавлении метода оплаты введите данные карты или учетной записи, а затем выберите **"Сохранить".**
6. Убедитесь, что выбран правильный способ оплаты, а затем выберите **"Заменить".**

## <a name="delete-a-payment-method"></a>Удаление способа оплаты

Можно удалить только метод оплаты, который не присоединен к подписке или профилю вы выставления счета. Это относится ко всем подпискам независимо от их состояния.

### <a name="delete-a-payment-method-with-no-subscriptions-or-billing-profiles-attached"></a>Удаление метода оплаты без подключенных подписок или профилей вы выставления счета

Если метод оплаты не связан ни с одной подпиской или профилями вы выставления счета, вы можете немедленно удалить его.

1. В центре администрирования перейдите в раздел **Выставление счетов** > **Счета и платежи** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Методы оплаты</a>.
2. Найдите метод оплаты, который нужно удалить, выберите три точки, а затем выберите **"Удалить".**
3. В нижней части правой области выберите **"Удалить".**

### <a name="delete-a-payment-method-with-subscriptions-or-billing-profiles-attached"></a>Удаление метода оплаты с присоединенными подписками или профилями вы выставления счета

Если метод оплаты присоединен к подпискам или профилям вы выставления счета, сначала замените его существующим методом оплаты или добавьте новый, а затем удалите старый метод оплаты.

1. В центре администрирования перейдите в раздел **Выставление счетов** > **Счета и платежи** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">Методы оплаты</a>.
2. Выберите строку для метода оплаты, который нужно удалить. На правой области перечислены существующие подписки, которые используют этот метод оплаты.
3. В правой области выберите **"Удалить".**
4. Чтобы использовать существующий способ оплаты, выберите один из них в выпадаемом списке, выберите "Далее" **и** выберите **"Удалить".**
    > [!NOTE]
    > Если у вас есть подписки, связанные с профилем вычета, для оплаты можно использовать только кредитную карту. Если на странице "Способы  оплаты" указаны банковские счета, их нельзя выбрать в выпадаемом списке.
5. Чтобы добавить новый способ оплаты, выберите **Добавить метод оплаты**.
6. Выберите тип метода оплаты, который вы хотите добавить, введите сведения об учетной записи и выберите **"Сохранить".**
7. Новый способ оплаты уже выбран в раскрывающемся списке. Нажмите кнопку **Далее**.
8. Выберите **Удалить**.
