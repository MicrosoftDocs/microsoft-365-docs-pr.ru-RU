# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Сведения о выставлении Microsoft 365 для бизнеса в Мексике

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. Где можно получить счет для Microsoft 365 для покупки бизнеса?

Ваш счет отправляется на зарегистрированный электронный адрес уведомления о выставлении счетов через 24 часа после подтверждения покупки. Чтобы скачать счет в центре администрирования Microsoft 365, перейдите на страницу счетов &   >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">платежей.</a>

> [!IMPORTANT]
> Если вы хотите, чтобы счет включал RFC, необходимо добавить его перед подтверждением покупки.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. Как добавить RFC (Federal de Contribuyentes) в счет, который я получаю для покупки Microsoft 365 для бизнеса?

### <a name="during-the-purchase-process-checkout"></a>Во время процесса покупки (проверка)

Во время процесса покупки при проверке, когда вы получите шаг **4. Оплата и выставление счета**, выберите поле для ввода RFC. Этот шаг необходим, чтобы вы могли увидеть его отражение в счете покупки.

### <a name="for-your-future-purchases"></a>Для будущих покупок

Вы можете добавить или изменить RFC, чтобы информация отражалась в будущих Microsoft 365 для коммерческих покупок. Чтобы добавить или изменить RFC, перейдите на страницу **счетов**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">биллинга.</a>

> [!NOTE]
> Если вы имеете ежемесячную или годовую подписку на повторные платежи, добавление или изменение RFC отражается на счете следующего заряда.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. Могу ли я добавить или изменить RFC в уже созданный счет?

Важно добавить или изменить RFC перед покупкой. После сгенерировать Microsoft 365 для бизнес-счета вы не сможете добавлять или изменять данные RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. Почему я не получаю CFDI (Comprobante Fiscal Digital Por Internet)?

Корпорация Майкрософт, как налоговый резидент в Соединенных Штатах Америки, может выдавать счета только в соответствии с законами юрисдикции и не обязана предоставлять электронный счет CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. Действителен ли счет,который я получаю для операций бухгалтерского учета в Мексике?

Счет, созданный иностранными организациями (в данном случае корпорацией Майкрософт), который включает соответствующий RFC, действителен для налоговых целей в Мексике. (Мы рекомендуем консультации с бухгалтером для получения дополнительных сведений).
