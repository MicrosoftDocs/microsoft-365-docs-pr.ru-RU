# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Сведения о выставлении счетов для Microsoft 365 для бизнеса в Мексики

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. где можно получить счет для приобретения Microsoft 365 для бизнеса?

Ваш счет отправляется на зарегистрированный электронный адрес электронной почты с уведомлением о выставлении счетов через 24 часа после подтверждения покупки. Чтобы скачать свой счет в центре администрирования Microsoft 365, перейдите на страницу **выставления счетов**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">& платежей</a> .

> [!IMPORTANT]
> Если вы хотите, чтобы счет включал ваш документ RFC, его необходимо добавить перед подтверждением покупки.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. как можно добавить документ RFC (Registro Федеральный де Контрибуйентес) в накладную, полученную для приобретения Microsoft 365 для бизнеса?

### <a name="during-the-purchase-process-checkout"></a>Во время процесса покупки (Checkout)

При получении заказа на покупку при переходе к этапу **4. Оплата и выставление счетов**, установите флажок, чтобы ввести RFC. Это действие необходимо для того, чтобы оно отображалось в счете покупки.

### <a name="for-your-future-purchases"></a>Для будущих покупок

Вы можете добавить или изменить RFC, чтобы информация была отражена в будущих приобретениях Microsoft 365 для бизнеса. Чтобы добавить или изменить документ RFC, перейдите на **Billing**  >  страницу <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">счета выставления</a> счетов для выставления счетов.

> [!NOTE]
> Если вы находитесь в месячной или ежегодной подписке для выставления счетов, добавление или изменение документа RFC отражается на счете вашей следующей зарядки.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. можно ли добавить или изменить документ RFC в уже созданную вами накладную?

Важно добавить или изменить RFC, прежде чем приступать к приобретению. После создания счета Microsoft 365 для бизнеса невозможно добавить или изменить сведения RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. Почему я не получаю КФДИ (Компробанте Digital Digital Por Internet)?

Корпорация Майкрософт в качестве налога, резидентного в США, может выдать накладные в соответствии с законами юрисдикции и не обязан предоставить КФДИ электронной накладную.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. будет ли полученная накладная действительна для моей бухгалтерской операции в Мексика?

Счет, созданный внешними объектами (в данном случае корпорация Майкрософт), включающий соответствующий RFC, действителен для целей налогообложения в Мексики. (Мы рекомендуем консультации для своего бухгалтера за дополнительными сведениями).
