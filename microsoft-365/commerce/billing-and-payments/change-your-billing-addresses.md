# <a name="change-your-billing-addresses"></a>Изменение адресов выставления счетов

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](../../admin/microsoft-365-admin-center-preview.md?preserve-view=true&view=o365-21vianet).

::: moniker-end

Счет или счет содержит три адреса:
  
- **Адрес покупателя**  название и адрес вашей организации, как они указаны в ее профиле.

- **Адрес выставления счета**  адрес вашего подразделения по работе со счетами; обычно совпадает с адресом **покупателя**.

- **Адрес использования службы**  адрес, по которому используется служба. Обычно он совпадает с адресом **покупателя**. Если в организации есть удаленные пользователи или несколько офисов, укажите адрес большинства пользователей.

Как правило, эти адреса совпадают. При необходимости вы можете изменить один или несколько из них. Вы также можете предоставить альтернативный адрес электронной почты для получения уведомлений о выставлении счета и изменить альтернативный адрес электронной почты для других администраторов.

## <a name="change-your-sold-to-address"></a>Изменение адреса покупателя

1. В центре администрирования перейдите на страницу **счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">биллинга.</a>

2. выберите **изменение сведений об учетной записи для биллинга.**

3. Обновите сведения об организации, а затем выберите **Сохранить**.
  
## <a name="change-your-bill-to-address"></a>Изменение адреса выставления счетов

::: moniker range="o365-worldwide"

1. В центре администрирования перейдите на страницу **Методы оплаты** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">биллинга.</a>

2. Выберите кредитную карту или банковский счет, которые необходимо изменить.

3. На странице **Сведения о методе оплаты** выберите **Изменить**.

4. Обновите адрес вы выставления счета, а затем выберите **Сохранить** \> **сделано**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

2. Выберите подписку, которая необходимо изменить, а затем выберите **сведения об изменении оплаты.**

    ![The Payment method section of a Subscription card for a subscription that pays by credit card.](../../media/6c9d9cae-6086-4687-a979-bb971f35f1b4.png)
  
3. *Клиенты кредитных карт и банковских счетов:*  На странице **Сведения** об изменениях оплаты выберите кредитную карту или банковскую учетную запись, которую необходимо обновить, выберите сведения об изменении, обновите свой платежный адрес, а затем выберите **Отправить**.
  
    > [!NOTE]
    > В некоторых странах или регионах возможность оплаты банковским счетом недоступна.

     *Клиенты-фактуры:*  На странице **Сведения об изменениях оплаты** обновите свой платежный адрес, а затем выберите **Отправить**.
  
    > [!NOTE]
    > Возможность оплатить счет чеком недоступна в некоторых странах или регионах.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.

2. Выберите подписку, которая необходимо изменить, а затем выберите **сведения об изменении оплаты.**

    ![The Payment method section of a Subscription card for a subscription that pays by invoice.](../../media/51ab38aa-6e15-4e51-9f27-261c38c98fed.png)
  
3. На странице **Сведения об изменениях оплаты** обновите свой платежный адрес и выберите **Отправить**.

    Метод оплаты Office 365, выполняемый 21Vianet, можно изменить с счета-фактуры на оплату в Интернете с помощью Alipay или China UnionPay или наоборот.

::: moniker-end

## <a name="change-your-service-usage-address"></a>Изменение адреса использования службы

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На вкладке **Продукты** выберите подписку, которую хотите изменить.
3. На странице сведения о подписке в разделе Адрес использования **службы** выберите **адрес использования службы Редактирование.**
4. В области **адресов использования** службы редактирования обновите адрес, а затем выберите **Сохранить**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

2. Выберите подписку, которая необходимо изменить, выберите **дополнительные действия,** а затем выберите адрес использования **службы Редактирование.**

    ![Закройте меню "Дополнительные действия".](../../media/befa74b7-62c1-42a3-a38e-db76a1c97dba.png)
  
3. В области **изменить адрес использования службы** обновите адрес, а затем выберите **Отправить**.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.

2. Выберите подписку, которая необходимо изменить, выберите **дополнительные действия,** а затем выберите адрес использования **службы Редактирование.**

    ![Закройте меню "Дополнительные действия".](../../media/befa74b7-62c1-42a3-a38e-db76a1c97dba.png)
  
3. В области **изменить адрес использования службы** обновите адрес, а затем выберите **Отправить**.

::: moniker-end

## <a name="change-the-alternate-email-address-your-billing-notifications-are-sent-to"></a>Изменение запасного адреса электронной почты, на который отправляются уведомления о выставлении счетов

По умолчанию уведомления о выставлении счета вашей организации отправляются на адрес электронной почты Майкрософт и альтернативный адрес электронной почты каждого глобального администратора и администратора биллинга в вашей организации. Каждый администратор может изменить свой запасной адрес электронной почты в данных своего профиля.
  
1. Вопишитесь в Microsoft 365 со своей учетной записью работы или учебного заведения в качестве глобального [администратора. Узнайте, как войти.](https://support.microsoft.com/office/e9eb7d51-5430-4929-91ab-6157c5a050b4)

2. Выберите **значок Пользователя,** а затем выберите **мою учетную запись.**
  
3. Выберите **личную информацию**.
  
4. В поле **Сведения о контакте** выберите **Изменить**.

5. В **альтернативном поле электронной** почты введите альтернативный адрес электронной почты, который необходимо использовать, а затем выберите **Сохранить**.
  
## <a name="change-the-alternate-email-address-for-another-admin"></a>Изменение запасного адреса электронной почты для другого администратора

Вы также можете изменить запасные адреса электронной почты других глобальных администраторов и администраторов выставления счетов в вашей организации.

::: moniker range="o365-worldwide"
  
1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=853212" target="_blank">Уведомления о выставлении счетов</a>.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=853213" target="_blank">Уведомления о выставлении счетов</a>.

::: moniker-end

::: moniker range="o365-21vianet"

1. В центре администрирования перейдите на страницу **Уведомления** о выставлении \> <a href="https://go.microsoft.com/fwlink/p/?linkid=853215" target="_blank">биллинга.</a>

::: moniker-end

2. На странице **Уведомления о выставлении счета** выберите администратора, который необходимо изменить.

3. В области **Изменить роли пользователей** введите альтернативный адрес электронной почты, который необходимо использовать, а затем выберите **Сохранить**.
