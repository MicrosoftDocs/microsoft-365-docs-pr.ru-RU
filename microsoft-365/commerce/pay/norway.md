КОД SWIFT: CITIGB2L\
Код IBAN: GB94CITI18500812207621\
Номер учетной записи: 12207621\
Имя учетной записи: MICROSOFT

Банк: Citibank Europe plc\

Код IBAN: IE43CITI99005170000129\
Номер учетной записи: 70000129\
Имя учетной записи: MICROSOFT
