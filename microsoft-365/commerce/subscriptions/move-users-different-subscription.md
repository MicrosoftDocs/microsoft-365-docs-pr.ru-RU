## <a name="move-users-to-a-different-subscription"></a><span data-ttu-id="cdeba-108">Перевод пользователей на другую подписку</span><span class="sxs-lookup"><span data-stu-id="cdeba-108">Move users to a different subscription</span></span>

::: moniker range="o365-worldwide"

1. <span data-ttu-id="cdeba-109">В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.</span><span class="sxs-lookup"><span data-stu-id="cdeba-109">In the admin center, go to the **Users** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Active users</a> page.</span></span>

::: moniker-end

::: moniker range="o365-germany"

 1. <span data-ttu-id="cdeba-110">В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">Активные пользователи</a>.</span><span class="sxs-lookup"><span data-stu-id="cdeba-110">In the admin center, go to the **Users** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">Active users</a> page.</span></span>

::: moniker-end

::: moniker range="o365-21vianet"

 1. <span data-ttu-id="cdeba-111">В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">Активные пользователи</a>.</span><span class="sxs-lookup"><span data-stu-id="cdeba-111">In the admin center, go to the **Users** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">Active users</a> page.</span></span>

::: moniker-end

2. <span data-ttu-id="cdeba-112">Выберите круги рядом с именами пользователей, для которых необходимо заменить существующие лицензии.</span><span class="sxs-lookup"><span data-stu-id="cdeba-112">Select the circles next to the names of the users that you want to replace existing licenses for.</span></span>

3. <span data-ttu-id="cdeba-113">В верхней части выберите **Управление лицензиями на продукты.**</span><span class="sxs-lookup"><span data-stu-id="cdeba-113">At the top, select **Manage product licenses**.</span></span>

4. <span data-ttu-id="cdeba-114">В области **Управление лицензиями на** продукты выберите **Замените**  и выберите лицензии, которые необходимо назначить пользователям.</span><span class="sxs-lookup"><span data-stu-id="cdeba-114">In the **Manage product licenses** pane, select **Replace**  and select the licenses that you would like to assign to the users.</span></span>

5. <span data-ttu-id="cdeba-115">В нижней части выберите **Сохранить изменения** \> **Закрыть**.</span><span class="sxs-lookup"><span data-stu-id="cdeba-115">At the bottom, select **Save Changes** \> **Close**.</span></span>
