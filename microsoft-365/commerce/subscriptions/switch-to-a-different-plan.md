# <a name="switch-to-a-different-microsoft-365-for-business-plan"></a>Переход на другой план Microsoft 365 для бизнеса

Если в вашей компании происходят изменения или требуются дополнительные функции, вы можете переключить планы.  

Проще всего это сделать в Центре администрирования с помощью кнопки **Переключение планов**. Однако использование кнопки **"Переключить планы"** поддерживается не во всех ситуациях. В некоторых случаях можно переключить планы вручную.


**Нужно выполнить другую задачу?**
- [Отмена подписки](cancel-your-subscription.md)
- [Обновление семейства Microsoft 365 до подписки для бизнеса](https://support.microsoft.com/office/9322ffb8-a35d-4407-8ebe-ed6ea0859b9f.aspx)
- [Позвоните в службу поддержки Майкрософт](../../admin/contact-support-for-business-products.md)

## <a name="use-the-switch-plans-button"></a>Использование кнопки "Переключить планы"

Когда вы  используете кнопку "Переключить планы", вы проходите процесс покупки нового плана, на который вы можете переключить текущий план, всем пользователям автоматически назначены лицензии в новом плане, а старый план отменяется.

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Подписки</a>.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.

::: moniker-end

2. Выберите **"Переключить** планы", чтобы просмотреть список доступных новых планов.

    [Кнопка "Переключить планы" не существует](#the-switch-plans-button-isnt-there)  |  [Я не вижу нужный план](#i-dont-see-the-plan-i-want)

3. Выберите план, чтобы увидеть новые общие затраты в месяц. Обязательно ознакомьтесь с **информацией о ключе,** а затем выберите **"Далее",** чтобы проверить.

    [Помогите мне выбрать новый план.](https://go.microsoft.com/fwlink/p/?linkid=842056)

4. Выберите **"Чат"** сейчас на **странице "Проверка",** если вам нужна помощь.

    Когда вы закончите проверку, Microsoft 365 займет несколько минут, чтобы завершить переход. Вы можете сразу приступить к использованию новой подписки.

## <a name="the-switch-plans-button-isnt-there"></a>Кнопка "Переключить планы" не существует

Если **кнопка "Переключить** планы" недоступна, можно попробовать переключить планы [вручную](switch-plans-manually.md) или [вызвать службу поддержки.](../../admin/contact-support-for-business-products.md) Дополнительные сведения см. [в этой теме.](why-can-t-i-switch-plans.md)
  
Вот почему это может произойти:
  
- Вы используете несколько планов Microsoft 365. Кнопку "Переключить **планы"** можно использовать только в том случае, если все пользователи подписаны на один и тот же план.

- Вы уже используете Office 365 корпоративный E5, поэтому у вас уже есть все функции, доступные в Office 365.

## <a name="i-dont-see-the-plan-i-want"></a>Я не вижу нужный план

При использовании кнопки **"Переключить** планы" планы, на которые можно переключиться, отображаются в зависимости от служб в текущем плане. Кнопку "Переключить **планы"** можно использовать только для переключения на план с одинаковыми службами, связанными с данными, или на более высокую версию. В этом случае при переключении пользователи гарантированно не потеряют данные, связанные с этими службами.
  
Сравнение планов перед переключение см. на странице сравнения продуктов [Microsoft 365 для](https://go.microsoft.com/fwlink/p/?linkid=842056) бизнеса. Технические спецификации см. в описании [службы Microsoft 365.](https://go.microsoft.com/fwlink/p/?linkid=842275)
  
Если вы хотите переключиться на план с меньшем количеством служб, вы можете переключить планы вручную или вызвать службу поддержки Майкрософт [за](../../admin/contact-support-for-business-products.md) помощью. [](switch-plans-manually.md)
  
## <a name="why-some-switches-take-longer"></a>Почему для некоторых коммутаторов нужно больше времени

 **Проверка кредитоспособности** при переключении планов: если вы платите по счету или достигаете определенного уровня стоимости, может потребоваться проверка кредитоспособности. Проверка кредитоспособности может занять до двух дней. У пользователей будет полный доступ к текущему плану, пока вы не переключили их на новый.
  
 **Проверка состояния некоммерческой** организации: обычно для завершения процесса проверки вашего некоммерческого статуса требуется 30 дней. Если это было более 30 дней, обратитесь в службу поддержки.
