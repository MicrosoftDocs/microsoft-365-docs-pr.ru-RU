# <a name="switch-to-a-different-microsoft-365-for-business-plan"></a>Переход на другой план Microsoft 365 для бизнеса

При смене бизнеса или необходимости в дополнительных функциях можно изменить планы.  

Проще всего это сделать в Центре администрирования с помощью кнопки **Переключение планов**. Однако использование кнопки **Switch plans** не поддерживается во всех ситуациях. В некоторых случаях можно переключать планы вручную.


**Нужно выполнить другую задачу?**
- [Отмена подписки](cancel-your-subscription.md)
- [Обновление microsoft 365 Family до подписки на бизнес](https://support.microsoft.com/office/9322ffb8-a35d-4407-8ebe-ed6ea0859b9f.aspx)
- [Позвоните в службу поддержки Майкрософт](../../admin/contact-support-for-business-products.md)

## <a name="use-the-switch-plans-button"></a>Используйте кнопку "Планы коммутатора"

Когда вы используете кнопку **Планы** коммутатора, вы проходите процесс покупки нового плана, на который можно переключить текущий план, все пользователи автоматически получают лицензии в новом плане, и старый план отменяется для вас.

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Подписки</a>.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Подписки</a>.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Подписки</a>.

::: moniker-end

2. Выберите **планы Switch,** чтобы просмотреть список доступных новых планов.

    [Кнопка "Планы коммутатора" не существует](#the-switch-plans-button-isnt-there)  |  [Я не вижу нужный план](#i-dont-see-the-plan-i-want)

3. Выберите план, чтобы увидеть новую общую стоимость в месяц. Обязательно ознакомьтесь с **ключевыми сведениями,** а затем выберите **Далее,** чтобы проверить.

    [Помогите мне выбрать новый план](https://go.microsoft.com/fwlink/p/?linkid=842056).

4. Выберите **Чат сейчас** на странице **Checkout,** если вам нужна помощь.

    После завершения проверки microsoft 365 занимает несколько минут, чтобы завершить переход. Вы можете начать использовать новую подписку сразу.

## <a name="the-switch-plans-button-isnt-there"></a>Кнопка "Планы коммутатора" не существует

Если **кнопка Switch plans** недоступна, можно попытаться переключить планы [вручную](switch-plans-manually.md) или [вызвать службу поддержки.](../../admin/contact-support-for-business-products.md) Дополнительные сведения [см. в статью Почему нельзя переключать планы?](why-can-t-i-switch-plans.md)
  
Вот почему это может произойти:
  
- Вы используете несколько планов Microsoft 365. Вы можете использовать кнопку **Switch plans только** в том случае, если все пользователи подписываются на один и тот же план.

- Вы уже используете Office 365 Enterprise E5, поэтому у вас уже есть все функциональные возможности, доступные в Office 365.

## <a name="i-dont-see-the-plan-i-want"></a>Я не вижу нужный план

Когда вы используете **кнопку Switch plans,** планы, на которые можно перейти, отображаются на основе служб текущего плана. Вы можете использовать только кнопку **Switch plans** для перехода на план, который имеет те же службы, связанные с данными, или на более высокую версию. В этом случае при переключении пользователи гарантированно не потеряют данные, связанные с этими службами.
  
Чтобы сравнить планы перед переключение, см. на странице [Microsoft 365 для сравнения бизнес-продуктов.](https://go.microsoft.com/fwlink/p/?linkid=842056) Технические характеристики см. [в описании службы Microsoft 365.](/office365/servicedescriptions/office-365-service-descriptions-technet-library)
  
Если вы хотите перейти на план с большим количеством служб, вы можете переключать планы вручную [или](switch-plans-manually.md)вызвать службу [поддержки Майкрософт](../../admin/contact-support-for-business-products.md) за помощью.
  
## <a name="why-some-switches-take-longer"></a>Почему некоторые переключатели дольше

 **Проверка кредитоспособности** при переключении планов: если вы оплачиваете счет или достигаете определенного уровня затрат, может потребоваться проверка кредита. Проверка кредита может занять до двух бизнес-дней. Пользователи будут иметь полный доступ к текущему плану, пока вы не переключить их на новый.
  
 **Проверка состояния некоммерческой** организации. Обычно для завершения процесса проверки для некоммерческого статуса обычно требуется 30 дней. Если это было более 30 дней, обратитесь в службу поддержки.
