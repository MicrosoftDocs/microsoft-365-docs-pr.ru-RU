# <a name="reactivate-your-subscription"></a>Повторная активация подписки

Вы можете возобновить подписку в центре администратора, если срок действия подписки истек, был отключен корпорацией Майкрософт или если вы отменили ее в середине срока подписки.
  
## <a name="before-you-begin"></a>Подготовка к работе

Вы должны быть администратором Global или Billing для повторной активации подписки. Дополнительные сведения см. в статье [О ролях администраторов](../../admin/add-users/about-admin-roles.md).

**Вы не являетесь администратором?** Свяжитесь с администрацией, чтобы активировать подписку.

## <a name="reactivate-a-subscription"></a>Повторная активация подписки

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Ваши продукты</a>.

::: moniker-end

::: moniker range="o365-21vianet"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Ваши продукты</a>.

::: moniker-end

2. На вкладке **Продукты** найдите подписку, которую нужно повторно активировать. Выберите три точки (больше действий), затем **выберите Reactivate эту подписку.**
    Если вы не видите **Reactivate эту подписку, обратитесь** [в поддержку.](../../business-video/get-help-support.md)
3. В области **Повторно активировать эту подписку** выберите способ оплаты из раскрывающегося списка или нажмите **Добавить способ оплаты**.
4. Нажмите **Сохранить**.
