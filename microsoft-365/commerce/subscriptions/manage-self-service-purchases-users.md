# <a name="manage-self-service-purchases-users"></a>Управление самостоятельно приобретенными лицензиями (пользователи)

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet&preserve-view=true).

::: moniker-end

Как пользователь вы можете приобрести подписки на определенные продукты и назначить лицензии на эти подписки пользователям в вашей команде. Вы несете ответственность за самостоятельное приобретение. Вы можете управлять подписками в Центре администрирования Microsoft 365 по <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">admin.microsoft.com.</a>

Администратор может просматривать все приобретаемые подписки только для чтения. Они могут видеть продукт, имя покупателей, приобретенные подписки, дату окончания срока действия, цену покупки и назначенного пользователя для каждой приобретаемой подписки.

## <a name="view-your-subscriptions"></a>Просмотр подписок

Вы можете просмотреть список всех приобретенных вами самообслуживаемых подписок.

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" выберите значок фильтра, а затем выберите **"Самообслуживка".**

## <a name="how-to-buy-more-or-reduce-licenses"></a>Как приобрести дополнительные или уменьшить количество лицензий

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На **вкладке** "Продукты" выберите подписку, для которую вы хотите приобрести дополнительные лицензии или уменьшить количество лицензий.
3. Выберите **"Купить лицензии"** или **"Удалить лицензии".**
4. В правой области в  поле "Всего лицензий" введите общее количество лицензий, которое необходимо для этой подписки, а затем выберите **"Сохранить".** Например, если у вас 100 лицензий и вы хотите добавить еще 5, введите 105.

## <a name="assign-or-unassign-licenses"></a>Назначение и назначение лицензий

### <a name="to-assign-licenses"></a>Назначение лицензий

1. В Центре администрирования перейдите на страницу **"Лицензии**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">на выставление счета".</a>
2. Выберите подписку, для которую нужно назначить лицензии.
3. Choose **Assign licenses**.
4. В области **Назначение лицензий пользователям** начните вводить имя и выберите его в результатах, чтобы добавить в список. Вы можете добавить до 20 пользователей одновременно.
    > [!NOTE]
    > Лицензии можно назначать только людям в организации.
5. Choose **Assign**, then choose **Close**.

### <a name="to-unassign-licenses"></a>Чтобы отоименовать лицензии

1. В Центре администрирования перейдите на страницу **"Лицензии**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">на выставление счета".</a>
2. Выберите продукт, для который нужно отоименовать лицензии.
3. Выберите пользователей, у которые вы хотите отоименовать лицензии.
4. Выберите **"Отоименуть лицензии".**
5. В поле **"Unassign licenses"** (Отоименуть лицензии) выберите **"Unassign" (Отоименуть лицензии).**

## <a name="cancel-a-subscription"></a>Отмена подписки

1. В Центре администрирования перейдите на страницу **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ваши продукты</a>.
2. На вкладке **Продукты** найдите подписку, которую хотите отменить. Нажмите **Дополнительные действия** (три точки) и выберите **Отменить подписку**.
3. В области **Отмена подписки** выберите причину, по которой вы отменяете подписку. При желании вы можете предоставить свой отзыв.
4. Нажмите кнопку **Сохранить**.

## <a name="manage-your-payment-details"></a>Управление реквизитами платежа

1. В Центре администрирования перейдите на страницу **"Выставление** счетов  >  **& платежей".**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2103629" target="_blank"></a>
2. Выберите профиль вы выставления счета в списке.
3. На странице "Сведения о профиле вы выставления счета" в **меню "Способ оплаты"** выберите один из следующих параметров:
    - Если вы хотите обновить существующий способ оплаты, выберите **"Изменить".**
    - Если вы хотите добавить новый способ оплаты, выберите **"Заменить".**
4. В правой области введите сведения о карточке и выберите **"Сохранить".**

### <a name="update-an-existing-payment-method"></a>Обновление существующего способа оплаты

1. В Центре администрирования перейдите на страницу **"Выставление счетов**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">& платежей".</a>
2. На **вкладке "Методы оплаты"** выберите в списке существующий способ оплаты.
3. В правой области выберите **"Изменить".** Вы можете изменить код безопасности, дату окончания срока действия, имя на карточке и адрес.
4. Внести необходимые изменения и выбрать **"Сохранить".**

### <a name="add-a-new-payment-method"></a>Добавление нового способа оплаты

1. В Центре администрирования перейдите на страницу **"Выставление счетов**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">& платежей".</a>
2. На **вкладке "Методы оплаты"** выберите **"Добавить метод оплаты".**
3. В области **"Добавление метода оплаты"** введите сведения о новом методе оплаты и выберите **"Добавить".**

## <a name="view-your-invoices"></a>Просмотр счетов

1. В Центре администрирования перейдите на страницу **"Счета**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">& платежи".</a>
2. На **вкладке "Счета"** выберите счет, который нужно просмотреть. Если накладные не видны, измените последние **3 месяца** на последние **6** месяцев или укажите **диапазон дат.**
