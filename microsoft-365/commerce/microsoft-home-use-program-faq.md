# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Вопросы и ответы по программе домашнего использования (Майкрософт)

Клиенты корпоративного лицензирования используют программу microsoft Home Use Program (HUP) на [веб-сайте Microsoft Store для бизнеса.](https://go.microsoft.com/fwlink/?linkid=2139192) 

Все остальные клиенты имеют доступ к HUP с помощью этого шага:

- В Центре администрирования Майкрософт перейдите на страницу "Выставление счета вашим продуктам", выберите вкладку "Преимущества" и перейдите на вкладку  >  [](https://go.microsoft.com/fwlink/p/?linkid=842054) **HUP.** 

## <a name="what-is-the-microsoft-home-use-program"></a>Что такое программа microsoft Home Use?

Программа microsoft Home Use Program (HUP) позволяет сотрудникам соответствующих компаний приобретать потребительские продукты Майкрософт со скидкой. Вы можете проверить текущие скидки для каждого предложения продукта, посетив страницу продукта программы [домашнего использования.](https://www.microsoft.com/home-use-program)

## <a name="in-which-markets-is-the-home-use-program-offered"></a>На каких рынках предлагается программа домашнего использования?

Скидки на подписки на семейство Microsoft 365 и личные подписки предоставляются подходящим компаниям по всему миру за такими исключениями:

- Макао, Россия и Венесуэла: это предложение не доступно на этих рынках.
- Китай и Гонконг: на этих рынках доступна одногодовая подписка. Для Китая посетите [Microsoft 365 в Китае.](https://www.microsoftstore.com.cn/home-use-program/invite) В Гонконге посетите сайт ["Купить Microsoft 365" по сниженной цене.](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK)
- Япония: семейство Microsoft 365 не доступно.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>Как организация может получить скидки HUP?

Чтобы сотрудники могли воспользоваться скидкой на программу домашнего использования Майкрософт, им необходим действительный корпоративный адрес электронной почты и активная учетная запись Майкрософт, и ваша организация должна соответствовать одному из следующих критериев:

- Клиенты с активной программой Software Assurance для проверки приложений Office 

- Клиенты, у которых есть Microsoft 365/Microsoft 365 E3/E5 from Software Assurance

- Коммерческие и государственные клиенты, которые приобрели ≥ 2000 лицензий на одну или несколько продуктов Microsoft 365 E3/E5

- Клиенты из образовательных и некоммерческих организаций

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>Почему мне нужно создать учетную запись Azure Active Directory (AAD) для использования программы домашнего использования или Microsoft Store для бизнеса?

Чтобы обновить и модернизировать программу домашнего использования, чтобы получить более широкий спектр преимуществ лицензий и продуктов, нам нужно было изменить способ выполнения HUP. Microsoft Store для бизнеса используется для проверки прав и назначения преимуществ HUP, а также для этого использует AAD для подключения информации из VSLC и проверки доменов электронной почты, в которые находятся сотрудники вашей организации. Дополнительные сведения см. в подпишитесь на учетные записи [Azure AD в Office 365 для бизнеса.](https://docs.microsoft.com/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome)

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>Как стать ИТ-администратором управляемого клиента?

Необходимо создать специальную запись в DNS, чтобы подтвердить владение доменом компании ex:contoso.com.

Пошаговую процедуру по управлению клиентом см. в подзагорном управлении неуправленным каталогом от администратора [в Azure Active.](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover)

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>Какие ресурсы доступны для продвижения преимуществ программы домашнего использования для сотрудников?

Используйте ресурсы, расположенные [](https://www.microsoft.com/home-use-program/resources) на домашней странице программы использования для дома, чтобы помочь вам продвигать программу для сотрудников. Вы найдете цифровые и печатные листовки, подробные инструкции по использованию в электронных письмах и информационных бюллетенях, а также рекомендации по лучшему продвижению программы домашнего использования.

Вы можете получить доступ к материалам по входу на нескольких языках на [сайте HUP Material for Customers.](https://microsofteur.sharepoint.com/teams/HUPMaterial) Выберите язык для начала загрузки файлов. Файл содержит ресурсы, которые можно использовать для информирования сотрудников о программе и продвижения ее внутри компании.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>Каковы действия, необходимые для приобретения программы домашнего использования Майкрософт для соответствующих сотрудников?

1. **Sign Up** Перейдите на [страницу программы microsoft Home Use Program](https://www.microsoft.com/home-use-program)и выберите кнопку "Начало работы". Сотрудники введите свой корпоративный адрес электронной почты.

2. **Подтверждение**  На их корпоративный адрес электронной почты отправляется сообщение электронной почты со ссылкой, которая позволяет проверить их право на участие в программе.

3. **Вход** После проверки подлинности они начинают процесс покупки, предоставляя учетную запись Майкрософт, связанную с личным адресом электронной почты, что позволяет им просматривать скидки на доступные им продукты в зависимости от преимущества программы домашнего использования организации. Дополнительные сведения см. в [сведениях о входе в учетную запись Майкрософт.](https://support.microsoft.com/help/4028195/microsoft-account-sign-in)

4. **Магазин** Покупка обрабатывается и завершится в Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Если сотрудник подписывается на Microsoft 365 с помощью программы домашнего использования Майкрософт, как это повлияет на подписку на Microsoft 365 бизнес и ограничение устройств в организации?

Установка с помощью программы домашнего использования не влияет на подписку на Microsoft 365 бизнес вашей организации. Когда сотрудники активируют Microsoft 365 на своих личных устройствах, они используют свою личную учетную запись Майкрософт, а не свою учетную запись. ИТ-администраторы вашей организации не управляют покупками, сделанными в рамках программы домашнего использования, и не поддерживают их.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>I'm having trouble managing my organization's Home Use Program benefit. Где можно получить поддержку?

О поддержке проблем, которые могут быть перенесены из Volume Licensing Service Center или для доступа к Microsoft Store для бизнеса, см. в технической [поддержке VLSC.](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0)

Проблемы с просмотром преимущества после того, как вы уже получили доступ к Microsoft Store для бизнеса, см. в службе поддержки [администраторов Магазина для бизнеса](https://docs.microsoft.com/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>Почему у меня нет возможности приобрести Visio профессиональный 2019 или Project профессиональный 2019?

Если один из ваших сотрудников воспользовался предыдущей программой и ищет скидки на Visio профессиональный 2019 и(или) Project профессиональный 2019, пройдите процедуру проверки на соответствие требованиям и, войдите с помощью личной учетной записи Майкрософт, перейдите в другие продукты **Office,** чтобы приобрести эти продукты. Если ожидаемые скидки не применяются, обратитесь к администратору программы домашнего использования своего компании, чтобы узнать об этих дополнительных преимуществах.

## <a name="what-if-my-employees-have-questions-about-this-program-or-need-support-with-purchasing-and-installation"></a>Что делать, если у сотрудников возникли вопросы по этой программе или нужна поддержка по покупке и установке?

Если у сотрудников есть вопросы, обратитесь к ним по следующим ресурсам:

- [Post purchase and installation FAQ](https://products.office.com/microsoft-office-for-home-and-school-faq)

- [Home Use Program Employee FAQ](https://www.microsoft.com/home-use-program/frequently-asked-questions)

 Справку по техническим вопросам см. в [справке службы поддержки Майкрософт.](https://support.microsoft.com/) Вы найдете справку по:

- Состояние заказа

- Выставление счетов

- Установка Office

- Вход в Office
