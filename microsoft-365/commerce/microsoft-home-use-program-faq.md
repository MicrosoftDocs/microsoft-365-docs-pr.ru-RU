# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>программа использования ПО Майкрософт на домашних компьютерах часто задаваемые вопросы (часто задаваемые вопросы)

Клиенты volume Licensing используют программа использования ПО Майкрософт на домашних компьютерах (HUP) на [веб-Microsoft Store для бизнеса.](https://go.microsoft.com/fwlink/?linkid=2139192) 

Все остальные клиенты имеют доступ к HUP с помощью этого шага:

- В центре администрирования Майкрософт перейдите на страницу **Выставление** платы за продукты, выберите вкладку Преимущества, а затем  >  [](https://go.microsoft.com/fwlink/p/?linkid=842054)выберите **вкладку HUP.** 

## <a name="what-is-the-microsoft-home-use-program"></a>Что такое программа использования ПО Майкрософт на домашних компьютерах?

Служба программа использования ПО Майкрософт на домашних компьютерах (HUP) позволяет сотрудникам в соответствующих компаниях покупать продукты Microsoft со скидкой. Вы можете проверить текущие скидки для каждого предложения продукта, посетив страницу продукта [программы домашнего использования](https://www.microsoft.com/home-use-program).

## <a name="in-which-markets-is-the-home-use-program-offered"></a>На каких рынках предлагается программа домашнего использования?

Скидки на Microsoft 365 для семьи и личные подписки предоставляются компаниям по всему миру, за исключением этих исключений:

- Макао, Россия и Венесуэла. Это предложение не доступно на этих рынках.
- Китай и Гонконг: на этих рынках доступна однолетная подписка. Для Китая посетите Microsoft 365 [в Китае](https://www.microsoftstore.com.cn/home-use-program/invite). В Гонконге посетите [сайт Buy Microsoft 365 по сниженной цене.](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK)
- Япония: Microsoft 365 для семьи недоступна.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>Как организация может претендовать на скидки huP?

Чтобы сотрудники могли воспользоваться скидкой программа использования ПО Майкрософт на домашних компьютерах, им необходим действительный корпоративный адрес электронной почты и активная учетная запись Майкрософт, и ваша организация должна соответствовать одному из следующих критериев:

- Клиенты, которые имеют активную гарантию программного обеспечения для Office приложений 

- Клиенты, Microsoft 365/Microsoft 365 E3/E5 из обеспечения программного обеспечения

- Коммерческие и государственные клиенты, ≥ 2000 лицензий одной или Microsoft 365 E3/E5

- Образовательные и некоммерческие клиенты

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>Зачем мне создавать учетную запись Azure Active Directory (AAD) для использования программы домашнего использования/ Microsoft Store для бизнеса?

Чтобы обновить и модернизировать программу домашнего использования до более широкого диапазона преимуществ лицензии и продуктов, нам необходимо изменить способ выполнения HUP. Microsoft Store для бизнеса используется для проверки прав и назначения преимуществ HUP, а для этого она полагается на AAD для подключения сведений из VSLC и проверки доменов электронной почты, на которые работают сотрудники организации. Дополнительные сведения см. в документе Для регистрации учетных [записей Azure AD](/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome)Office 365 для бизнеса.

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>Как стать назначенным ИТ-администратором управляемого клиента?

Необходимо создать специальную запись в DNS, чтобы доказать, что вы владеете доменом компании, ex:contoso.com.

О пошаговом процессе становления управляемого клиента см. в статью [Take over an unmanaged directory as administrator in Azure Active.](/azure/active-directory/users-groups-roles/domains-admin-takeover)

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>Какие ресурсы доступны для продвижения преимущества программы домашнего использования для сотрудников?

Используйте ресурсы, расположенные на домашней странице [программы](https://www.microsoft.com/home-use-program/resources) домашнего использования, чтобы помочь вам продвигать программу своим сотрудникам. Вы найдете цифровые и печатные листовки, подробные инструкции для использования в электронных письмах и информационных бюллетенях, а также рекомендации по наилучшему продвижению программы домашнего использования.

Вы можете получить доступ к бортовых материалов на нескольких языках в [HUP Material для клиентов](https://microsofteur.sharepoint.com/teams/HUPMaterial). Выберите язык для начала загрузки файлов. Файл содержит ресурсы, которые можно использовать для информирования сотрудников о программе и ее внутренней рекламы в вашей компании.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>Какие действия необходимо предпринять для того, чтобы соответствующий сотрудник программа использования ПО Майкрософт на домашних компьютерах покупку?

1. **Зарегистрив** Перейдите на [программа использования ПО Майкрософт на домашних компьютерах страницу](https://www.microsoft.com/home-use-program)и выберите кнопку Начало работы. Сотрудники введите корпоративный адрес электронной почты.

2. **Подтверждение**  В почтовый ящик корпоративного почтового ящика отправляется сообщение электронной почты со ссылкой на проверку подлинности их прав на участие в программе.

3. **Вход** После проверки подлинности они начинают процесс покупки, предоставляя учетную запись Майкрософт, связанную с личным адресом электронной почты, которая позволяет им просматривать скидки на продукты, доступные им на основе преимущества программы домашнего использования организации. Дополнительные сведения см. [в документе How to sign in to your Microsoft account.](https://support.microsoft.com/help/4028195/microsoft-account-sign-in)

4. **Магазин** Покупка обрабатывается и завершится в Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Если сотрудник подписывается на Microsoft 365 через программа использования ПО Майкрософт на домашних компьютерах, как это влияет на Microsoft 365 нашей организации подписку и ограничение устройств?

Установка с помощью программы домашнего использования не влияет на Microsoft 365 организации. Когда сотрудники активируют Microsoft 365 на личных устройствах, они используют свою личную учетную запись Майкрософт, а не свою учетную запись. ИТ-администраторы организации не управляют и не поддерживают покупки, сделанные с помощью программы домашнего использования.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>У меня возникли проблемы с управлением преимуществами программа использования ПО Майкрософт на домашних компьютерах организации. Где можно получить поддержку?

Для поддержки проблем, переносимых из Центра службы лицензирования томов или для доступа к Microsoft Store для бизнеса, см. в выпуске [VLSC Support.](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0)

О проблемах с просмотром преимущества после того, как вы уже получили доступ к Microsoft Store для бизнеса, см. в поддержку [Store for Business Admin](/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>Почему я не вижу возможности приобрести Visio профессиональный 2019 Project профессиональный 2019?

Если кто-то из сотрудников воспользовался предыдущей программой и ищет скидки на Visio профессиональный 2019 и/или Project профессиональный 2019 г., перейдите по процедуре проверки на соответствие требованиям и, войдите в личную учетную запись Майкрософт, перейдите на другие продукты **Office,** чтобы приобрести эти продукты. Если вы не видите ожидаемых скидок, обратитесь к администратору программы домашнего использования вашего работодателя, чтобы узнать об этих дополнительных преимуществах.
