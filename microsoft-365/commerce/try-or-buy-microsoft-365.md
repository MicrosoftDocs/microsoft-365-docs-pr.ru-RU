# <a name="try-or-buy-a-microsoft-365-for-business-subscription"></a>Попробуйте или купите подписку на Microsoft 365 для бизнеса

::: moniker range="o365-21vianet"

> [!NOTE]
> Изменяется Центр администрирования. Если ваш интерфейс не соответствует приведенным здесь сведениям, см. раздел [О новом Центре администрирования Microsoft 365](../admin/microsoft-365-admin-center-preview.md?view=o365-21vianet).

::: moniker-end

Microsoft 365 для бизнеса — это служба подписки, позволяющая работать над вашей организацией в облаке, пока корпорация Майкрософт следит за ее. Майкрософт управляет устройствами, обеспечивает защиту от реальных угроз и предоставляет в Организации последние сведения о корпоративном программном обеспечении. Вы можете подписаться на бесплатную пробную подписку на Microsoft 365 бизнес Standard, Microsoft 365 бизнес премиум или Microsoft 365 для бизнеса, а затем испытать ее в течение 30 дней.

> [!NOTE]
> Вы должны использовать кредитную карту при регистрации бесплатной пробной версии. В конце бесплатного пробного периода пробная подписка автоматически преобразуется в платную подписку. Выставление счетов по кредитной карте осуществляется только после завершения пробного периода.

## <a name="before-you-begin"></a>Прежде чем начать

Чтобы получить бесплатную пробную версию, вам не нужна учетная запись Майкрософт, имеющая существующую учетную запись Майкрософт. Для всех остальных процедур, описанных в этой статье, вы должны быть глобальным администратором или администратором по выставлению счетов для вашей организации. Более подробную информацию можно узнать в статье [сведения о ролях администратора](../admin/add-users/about-admin-roles.md).

## <a name="try-a-free-trial-subscription"></a>Попробуйте бесплатную пробную подписку.

Вы являетесь новым клиентом и у вас еще нет учетной записи Майкрософт? Выполните указанные ниже действия, чтобы создать учетную запись и зарегистрироваться для получения бесплатной пробной подписки на Microsoft 365 бизнес Standard, Microsoft 365 бизнес премиум или Microsoft 365 Apps для бизнеса.

1. Перейдите на <a href="https://www.aka.ms/office365signup" target="_blank">сайт продукты Microsoft 365</a>.
2. Выберите план, для которого вы хотите подписаться, например **Microsoft 365 Business Standard**, прокрутите страницу вниз и выберите вариант **попробовать бесплатно в течение 1 месяца**.
3. На следующей странице выполните действия, чтобы настроить свою учетную запись.
4. Для завершения процесса регистрации может потребоваться несколько минут. После \' завершения этого процесса вы можете запустить мастер установки для своей подписки. Дополнительные сведения о настройке подписки приведены в разделе Дополнительные [действия](#next-steps).

## <a name="buy-a-subscription-from-your-free-trial"></a>Приобретение подписки из бесплатной пробной версии

По окончании бесплатного пробного периода пробная подписка автоматически преобразуется в платную подписку. По умолчанию платной подписке назначен текущий план. Вы можете приобрести другой план, выполнив действия, описанные в статье [Покупка другой подписки](#buy-a-different-subscription).

Если вы хотите приобрести подписку перед пробной версией, выполните указанные ниже действия.

1. В центре администрирования Microsoft 365 перейдите на страницу **выставление счетов**по   \>  <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">продуктам</a>   .
2. На странице " **продукты**"   Найдите подписку, которую вы хотите купить.
3. В разделе **лицензии**   выберите пункт **приобрести подписку**.
4. Выберите ежемесячное или годовое обязательство для подписки, а затем выберите пункт **Checkout**.
5. На следующей странице проверьте подписку и выберите пункт **Checkout**.
6. На следующей странице проверьте адрес **проданных продаж**   , **счет, к которому выставлен счет**   , и **элементы в этом заказе**. Если требуется внести изменения, нажмите кнопку **изменить**   рядом с соответствующим разделом.
7. По \' завершении нажмите **одобрить Соглашение & заместить заказ**.

## <a name="cancel-your-free-trial-subscription"></a>Отмена бесплатной пробной подписки

Если вы решили отменить пробную подписку до завершения бесплатного пробного периода, перейдите в центр администрирования Microsoft 365 и отключите периодическую [выставление счетов](subscriptions/renew-your-subscription.md#turn-recurring-billing-off-or-on). Срок действия пробной версии автоматически истечет, а ваша кредитная карта будет \' оплачиваться.

## <a name="try-a-different-subscription"></a>Попробуйте другую подписку.

Если у вас уже есть подписка на Microsoft 365 для бизнеса, вы можете использовать центр администрирования Microsoft 365, чтобы попробовать другую подписку.

Когда вы добавляете подписку с помощью центра администрирования Microsoft 365, Новая подписка связана с той же организацией (пространством имен домена), что и существующая подписка. Эта связь упрощает перемещение пользователей в Организации между подписками или назначение им лицензии на дополнительные продукты, которые им нужны.

1. В центре администрирования перейдите на страницу Услуги по **выставлению счетов**   \>  <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">покупки</a>   .
2. На странице **приобрести службы**   вы увидите планы, доступные в Организации. Выберите план Microsoft 365, который вы хотите испытать.
3. На следующей странице выберите **получение бесплатной пробной версии**. В пробной версии предоставляется 25 пользовательских лицензий на один месяц.
4. Выберите, чтобы получить текст или вызов, введите свой номер телефона, затем выберите пункт " **текстовые**   или **звонить мне**".
5. Введите код проверки, а затем выберите **начать бесплатную пробную версию**.
6. На странице **извлечь**   нажмите кнопку **попробовать сейчас**.
7. На странице **прием заказов**   нажмите кнопку **продолжить**.

## <a name="buy-a-different-subscription"></a>Приобретение другой подписки

Если у вас уже есть подписка на Microsoft 365 для бизнеса, вы можете приобрести другую подписку для вашей организации с помощью центра администрирования Microsoft 365.

При покупке другой подписки через центр администрирования новая подписка связывается с той же организацией (пространством имен домена), что и существующая подписка. Это упрощает перемещение пользователей в Организации между подписками или предоставление им лицензии на дополнительные подписки, которые им необходимы.

1. В центре администрирования перейдите на страницу Услуги по **выставлению счетов**   \>  <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">покупки</a>   .
2. На странице " **Покупка служб**"   выберите план, который вы хотите купить, выберите **сведения**, а затем нажмите кнопку **купить**.
3. Введите необходимое количество лицензий и укажите, следует ли платить ежемесячно или за весь год. Укажите, хотите ли вы автоматически назначать лицензии всем пользователям, у которых в настоящее время нет лицензии. Затем нажмите **кнопку извлечь.**
4. Ознакомьтесь со сведениями о ценах и нажмите кнопку **Далее**.
5. Введите сведения об оплате, а затем выберите пункт **разместить заказ**   \>  **перейдите в домашнюю страницу администратора**.

> [!NOTE]
> Вы должны переместить пользователей из бесплатной пробной подписки на новую подписку, прежде чем срок действия льготного периода составляет 90 дней после истечения срока действия пробной подписки. Это позволяет хранить данные, учетные записи и конфигурацию. В противном случае эта информация удаляется.
