# <a name="manage-recipients-in-standalone-eop"></a>Управление получателями в автономной службе EOP

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**Область применения**
-  [Автономный exchange Online Protection](exchange-online-protection-overview.md)

Автономные организации Exchange Online Protection (EOP) без почтовых ящиков Exchange Online поддерживают следующие типы получателей:

- **Пользователи почты.** Пользователи почты являются основным типом учетных записей пользователей в вашей автономных организациях EOP. Пользователи почты имеют учетные данные с логотипами в организации EOP, но у них есть внешние адреса электронной почты (их почтовые ящики расположены за пределами организации EOP).

  Дополнительные сведения об управлении почтовыми пользователями в EOP см. в разделе [Управление почтовыми пользователями в EOP](manage-mail-users-in-eop.md).

- **Группы.** Можно создать следующие типы групп:

  - группы рассылки;
  - группы безопасности с включенной поддержкой почты.

  Дополнительные сведения об управлении группами в EOP см. в разделе [Управление группами в EOP](manage-groups-in-eop.md).
