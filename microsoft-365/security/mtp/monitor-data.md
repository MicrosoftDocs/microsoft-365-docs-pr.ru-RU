# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Мониторинг данных и отчетность в Центре безопасности Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Отчеты категории **Данные** помогают отслеживать активность пользователей, которая может привести к несанкционированному разглашению данных. Они являются работой существующих отчетов о политиках DLP, а также отчета о совпадениях с политикой DLP сторонних сторон.

Вы можете отслеживать:

- пользователей, предоставляющих больше всего файлов из облачных приложений;
- Количество совпадений с политикой DLP
- количество переопределений политик DLP или ложных срабатываний;
- Количество совпадений с политикой DLP в сторонних облачных службах через Microsoft Cloud App Security
