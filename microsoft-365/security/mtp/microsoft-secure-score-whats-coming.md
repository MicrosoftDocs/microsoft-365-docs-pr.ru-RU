# <a name="whats-coming-to-microsoft-secure-score"></a>Новые возможности службы "Оценка безопасности (Майкрософт)"

Чтобы [сделать оценку безопасности (Майкрософт)](microsoft-secure-score.md) лучшим превосходной представительством вашей безопасности и повысить удобство использования, мы внесли некоторые изменения в скором будущем. Оценка и максимальная оценка могут меняться.

## <a name="proposed-changes"></a>Предлагаемые изменения

Эти изменения в скором будут поступать в службу "Оценка безопасности (Майкрософт)", в зависимости от вашей лицензии. Сведения о недавних изменениях см. в [разделе "Новые возможности оценки безопасности (Майкрософт)"?](microsoft-secure-score-whats-new.md)

- Изменение максимального показателя с 7 на 8 для действия улучшения Azure Active Directory, которое называется "Разрешить политику блокировать устаревшую проверку подлинности".
