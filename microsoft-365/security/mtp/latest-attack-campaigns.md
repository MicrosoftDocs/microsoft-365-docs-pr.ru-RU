# <a name="understand-the-latest-attack-campaigns-and-techniques-with-threat-analytics"></a>Обзор последних кампаний и методов атак с помощью аналитики угроз 

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Киберугрозы становятся все более частыми и изощренными. Очень важно, чтобы организации быстро оценивали состояние безопасности и устойчивость к определенным возникающим угрозам.

**Аналитика угроз** — это набор интерактивных отчетов, публикующихся сразу после обнаружения новых угроз — новых методов, кампаний по атакам и эпидемий вредоносных программ. Отчеты помогают оценить влияние угроз на среду, а также предоставляют рекомендуемые действия по их предотвращению, а также по предотвращению определенных угроз, а также по повышению устойчивости организации.

[Узнайте больше об аналитике угроз в Microsoft Defender для конечной точки.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/threat-analytics)
