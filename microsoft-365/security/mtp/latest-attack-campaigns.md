# <a name="understand-the-latest-attack-campaigns-and-techniques-with-threat-analytics"></a>Понимание последних кампаний и методов атак с помощью аналитики угроз 

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Киберугрозы становятся все более частыми и изощренными. Важно, чтобы организации быстро оценили свою позицию безопасности и устойчивость к определенным возникающим угрозам.

**Аналитика угроз** — это набор интерактивных отчетов, опубликованных после обнаружения новых угроз, новых методов, кампаний атак и вспышек вредоносных программ. Отчеты помогают оценить влияние угроз на окружающую среду и предоставляют рекомендуемые действия по сдерживанию, повышению устойчивости организации и предотвращению определенных угроз.

[Дополнительные данные об аналитике угроз в Microsoft Defender для конечной точки.](/windows/security/threat-protection/microsoft-defender-atp/threat-analytics)
