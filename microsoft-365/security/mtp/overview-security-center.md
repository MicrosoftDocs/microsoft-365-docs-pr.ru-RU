# <a name="overview-of-the-microsoft-365-security-center"></a>Общие сведения о Центре безопасности Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

> Хотите испытать Microsoft 365 Defender? Вы можете [оценить его в лабораторной среде](https://aka.ms/mtp-trial-lab) или запустить [пилотный проект в производственной среде.](https://aka.ms/m365d-pilotplaybook)
>
Управление безопасностью вашего бизнеса для защиты от постоянно меняющихся угроз вызывает множество проблем. Возможно, у вас слишком много решений для обеспечения безопасности с различными областями для настройки большого количество элементов управления. Вам может быть сложно узнать, какие элементы управления являются наиболее эффективными и которые будут представлять новые проблемы для ваших сотрудников. Группам безопасности может быть трудно найти правильный баланс между безопасностью и производительностью.

Введите Центр безопасности Microsoft 365 — новый сайт для мониторинга и управления безопасностью в удостоверениях, данных, устройствах, приложениях и инфраструктуре Майкрософт. Здесь можно легко просмотреть состояние системы безопасности вашей организации, настроить устройства, пользователей и приложения, а также получать оповещения о подозрительных действиях. Центр безопасности Microsoft 365 помогает администраторам безопасности и группам операций безопасности управлять своей организацией и защищать ее.

Новый Центр безопасности Microsoft 365 и Центр соответствия требованиям [Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/microsoft-365-compliance-center) — это специализированные области, предназначенные для удовлетворения потребностей групп безопасности и соответствия требованиям. Эти решения интегрированы в службы Microsoft 365 и предоставляют информацию с действиями для снижения рисков и защиты цифрового актива.

## <a name="lets-take-a-closer-look"></a>Рассмотрим подробнее

На **домашней** странице показаны многие распространенные карточки, необходимые группам безопасности. Композиция карточек и данных зависит от роли пользователя. Так как Центр безопасности Microsoft 365 использует управление доступом на основе ролей, различные роли будут видеть карточки, которые более значимы для их ежедневного задания.  

Эти обзорные сведения помогают вам следить за последними действиями в организации. Центр безопасности Microsoft 365 объединяет сигналы из разных источников, чтобы представить целостное представление среды Microsoft 365.

Разделы Центра безопасности Microsoft 365 перечислены ниже.

* **Главная** — быстро просматривайте общую безопасность вашей организации.
* **Инциденты** — просмотр более широкой истории атаки путем подключения точек, которые видны в отдельных оповещениях об сущностям. Вы точно узнаете, где началась атака, какие устройства были затронуты, кто был затронут и куда прошла угроза.
* **Оповещения —** более наглядное оповещение во всей среде Microsoft 365. Включает оповещения из Microsoft Cloud App Security, Microsoft Defender для Office 365, Azure Active Directory, Microsoft Defender для удостоверений и Microsoft Defender для конечной точки. Доступно для пользователей E3 и E5.  
* **Центр действий** — уменьшите количество оповещений, которые группа безопасности должна адресовать вручную, позволяя группе безопасности сосредоточиться на более сложных угрозах и других ценных инициатив.
* **Отчеты** — получите подробные сведения и сведения, необходимые для лучшей защиты пользователей, устройств, приложений и других данных.
* **Оценка безопасности**. Улучшение состояния общей безопасности благодаря службе "Оценка безопасности (Майкрософт)". На этой странице приводится сводка различных включенных функций и возможностей безопасности, а также рекомендации по улучшению областей.
* **Расширенный поиск** — упреждающий поиск вредоносных программ, подозрительных файлов и действий в организации Microsoft 365.
* **Классификация**. Защита от потери данных за счет добавления меток для классификации документов, сообщений электронной почты, документов, сайтов и т. д. При применении метки (автоматически или пользователем) контент или сайт защищен в зависимости от параметров, которые вы выбрали. Например, можно создать метки для шифрования файлов, добавления маркировки содержимого и управления доступом пользователей к определенным сайтам.
* **Политики**. Настройка политик для управления устройствами, защиты от угроз и получения оповещений о различных действиях в вашей организации.
* **Разрешения —** управление доступом сотрудников организации к просмотру контента и выполнению задач в Центре безопасности Microsoft 365. Разрешения для Microsoft 365 можно также назначить на портале Azure AD.
