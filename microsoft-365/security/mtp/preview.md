# <a name="microsoft-365-defender-preview-features"></a>Функции предварительной версии защитника Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

>[!IMPORTANT]
>Ознакомительные версии предоставляются без соглашения об уровне обслуживания и не рекомендуются для рабочих рабочих нагрузок. Некоторые компоненты могут быть не поддерживаются или могут иметь ограниченные возможности.

**Область применения:**
- Защитник Microsoft 365

Служба защитника Microsoft 365 постоянно обновляется, добавляя новые функции и возможности.

Узнайте о новых возможностях в ознакомительной версии защитника Microsoft 365, а также в первую очередь для последующей работы, включив ознакомительную версию.

Для получения дополнительных сведений о новых возможностях, которые обычно доступны, ознакомьтесь со статьей [новые возможности защитника Microsoft 365](whats-new.md).

## <a name="turn-on-preview-features"></a>Включение предварительных функций
Вы получите доступ к предстоящим функциям, которые помогут вам повысить эффективность работы, прежде чем они будут доступны.

Включите параметр предварительной версии, чтобы быть в числе тех, кто первым попробует новые функции.

1. В области навигации выберите **Параметры**.

2. Выберите **защитник Microsoft 365**.


3. Выберите **Предварительные функции** > **Включить предварительные функции**. 

3. Нажмите **Сохранить**.

Вы узнаете, что предварительные функции включены, когда увидите установленный флажок **Включить предварительные функции**. 

## <a name="preview-features"></a>Предварительные функции
В настоящий момент в предварительной версии доступны следующие функции и улучшения.

- **[API-интерфейсы защитника microsoft 365](api-overview.md)** — API-интерфейсы защитника Microsoft 365 LOP, позволяющие автоматизировать рабочие процессы, основанные на общем инциденте и расширенных таблицах поискового запроса. 
- **[Выполнение действий в расширенном поиске](advanced-hunting-take-action.md)** — быстрое включение угроз или адресов скомпрометированных ресурсов, которые вы найдете в [расширенном поиске](advanced-hunting-overview.md).
- **[Справочник по схемам на портале](advanced-hunting-schema-tables.md#get-schema-information-in-the-security-center)** — получение сведений о расширенных таблицах схемы подпоисков непосредственно в центре обеспечения безопасности. Помимо описаний таблиц и столбцов, эта ссылка включает поддерживаемые типы событий ( `ActionType` значения) и примеры запросов.
