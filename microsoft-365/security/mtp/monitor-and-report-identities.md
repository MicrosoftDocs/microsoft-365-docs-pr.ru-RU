# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Мониторинг удостоверений и отчетность в Центре безопасности Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Отслеживайте удостоверения в организации и отслеживайте подозрительные или рискованные действия. В категории **"Удостоверения"** **можно** отслеживать:

* пользователей с наибольшим количеством обнаруженных аномалий;
* количество пользователей, подверженных риску согласно политикам условного доступа;
* количество глобальных администраторов в организации.

![Страница "Удостоверения" отчетов](../../media/identities.png)

Для пользователей с определенными обнаружениями вы можете изучить конкретное оповещение и изучить в Центре безопасности Microsoft Defender. Обнаружение включает аномалии, такие как пользователи, выехавшие из незнакомых мест.

Полный набор событий риска см. в этой [теме.](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events)
