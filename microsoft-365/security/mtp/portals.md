# <a name="microsoft-security-portals-and-admin-centers"></a>Порталы безопасности и центры администрирования Майкрософт

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Хотя Центр безопасности [Microsoft 365](overview-security-center.md) — это новый сайт для мониторинга и управления безопасностью в удостоверениях, данных, устройствах и приложениях, вам потребуется доступ к различным порталам для выполнения определенных специализированных задач.

> [!TIP] 
> Чтобы получить доступ к различным релевантным порталам из Центра безопасности Microsoft 365, выберите дополнительные ресурсы **в** области навигации.

## <a name="security-portals"></a>Порталы безопасности

Операторы безопасности и администраторы могут перейти на следующие порталы, чтобы управлять настройками безопасности, исследовать возможные угрозы, реагировать на активные угрозы и сотрудничать с ИТ-администраторами для устранения проблем.
<p></p>

| Имя портала | Description | ссылок. |
|---|---|---| 
| Центр безопасности Microsoft 365 | Мониторинг угроз и реагирование на них, а также усиление безопасности удостоверений, электронной почты, данных, конечных точек и приложений с помощью [Microsoft 365 Defender](microsoft-threat-protection.md) | [security.microsoft.com](https://security.microsoft.com/) |
| Центр безопасности Microsoft Defender | Мониторинг и реагирование на угрозы на конечных точках с помощью возможностей, предоставляемых [в Microsoft Defender для конечной точки](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-advanced-threat-protection) | [securitycenter.windows.com](https://securitycenter.microsoft.com/) |
| Центр безопасности и соответствия требованиям | Управление [Exchange Online Protection](https://docs.microsoft.com/microsoft-365/security/office-365-security/exchange-online-protection-overview?view=o365-worldwide) и Microsoft Defender для Office [365](https://docs.microsoft.com/microsoft-365/security/office-365-security/office-365-atp?view=o365-worldwide) для защиты электронной почты и служб совместной работы, а также обеспечения соответствия различным нормативным требованиям по обработке данных | [protection.office.com](https://protection.office.com) |
| Портал Защитника Azure | Использование [Защитника Azure](https://docs.microsoft.com/azure/security-center/security-center-intro) для усиления безопасности центров обработки данных и гибридных рабочих нагрузок в облаке | [portal.azure.com/#blade/Microsoft_Azure_Security](https://portal.azure.com/#blade/Microsoft_Azure_Security/SecurityMenuBlade/0) |
| Портал Microsoft Defender для удостоверений | Выявлять, обнаруживать и исследовать сложные угрозы, скомпрометированную личность и вредоносные действия внутри программы предварительной проверки с помощью сигналов Active Directory в [Microsoft Defender для удостоверений](https://docs.microsoft.com/azure-advanced-threat-protection/what-is-atp) | [portal.atp.azure.com](https://portal.atp.azure.com/) |
| Портал Cloud App Security | Использование [Microsoft Cloud App Security](https://docs.microsoft.com/cloud-app-security/what-is-cloud-app-security) для обеспечения высокой наглядности, контроля над перемещениями данных и сложной аналитики для идентификации и борьбы с киберугрозами в облачных службах | [portal.cloudappsecurity.com](https://portal.cloudappsecurity.com/) |
| Портал аналитики безопасности в Microsoft Defender | Получите обновления аналитики безопасности для Microsoft Defender для конечной точки, отправьте примеры и изучите encyclopedia угроз | [microsoft.com/wdsi](https://microsoft.com/wdsi) |

## <a name="portals-for-other-workloads"></a>Порталы для других рабочих нагрузок

Хотя эти порталы не предназначены специально для управления безопасностью, они поддерживают различные рабочие нагрузки и задачи, которые могут повлиять на безопасность. На этих порталах можно управлять удостоверениями, разрешениями, настройками устройств и политиками обработки данных.
<p></p>

| Имя портала | Description | ссылок. | 
|---|---|---| 
| Портал Azure | Просмотр и управление [Azure Active Directory и](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-whatis) всеми [ресурсами Azure](https://docs.microsoft.com/azure/azure-resource-manager/management/overview)  | [portal.azure.com](https://portal.azure.com/) |
| Центр соответствия требованиям Microsoft 365 | Управление политиками обработки данных и обеспечение [соответствия нормативным требованиям](https://docs.microsoft.com/microsoft-365/compliance/offering-home?view=o365-worldwide) | [compliance.microsoft.com](https://compliance.microsoft.com/) |
| Центр администрирования Microsoft 365 | Настройка служб Microsoft 365; управление ролями, лицензиями и отслеживание обновлений служб Microsoft 365 | [admin.microsoft.com](https://admin.microsoft.com/) |
| Центр администрирования Microsoft Endpoint Manager | Использование [Microsoft Endpoint Manager для](https://docs.microsoft.com/mem/configmgr/) управления и защиты устройств с помощью объединенных возможностей Intune и Configuration Manager | [devicemanagement.microsoft.com](https://devicemanagement.microsoft.com/) |
| Портал Microsoft Intune | Использование [Microsoft Intune для развертывания](https://docs.microsoft.com/intune/fundamentals/what-is-intune) политик устройств и отслеживания устройств на предмет соответствия требованиям | [endpoint.microsoft.com](https://endpoint.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/DevicesMenu/overview)
