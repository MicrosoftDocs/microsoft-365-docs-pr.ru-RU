# <a name="set-the-time-zone-for-microsoft-365-defender"></a>Настройка часового пояса для защитника Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Область применения:**
- Защитник Microsoft 365



Защитник Microsoft 365 может отображать сведения о дате и времени с помощью местного часового пояса или времени в формате UTC. Выбранный часовой пояс будет применен ко всем сведениям о дате и времени, которые указаны в следующих функциях Центра безопасности Microsoft 365:
- Инциденты
- Автоматизированное исследование и исправления, включая Центр уведомлений.
- Расширенный поиск

Чтобы задать часовой пояс для этих компонентов, перейдите к разделу **Настройка**  >  часового пояса **защитника Microsoft 365**  >  **Time zone**.
