# <a name="create-a-microsoft-365-defender-trial-lab-or-pilot-environment"></a>Создание пробной лаборатории или пилотной среды защитника Microsoft 365 

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Область применения:**
- Microsoft 365 Defender

Цель создания пробной лаборатории или пилотной среды — продемонстрировать комплексные и интегрированные возможности защитника Microsoft 365. Сведения о том, как данное интеллектуальное решение для обеспечения безопасности обнаруживает, предотвращает и автоматически исследует и реагирует на дополнительные угрозы вашей организации. 

В этом руководстве описано, как запустить оценку защитника Microsoft 365 на основе рекомендуемых путей развертывания. Цель состоит в том, чтобы помочь вам настроить решение для обеспечения безопасности в лабораторной среде с пробной учетной записью или в пилотной среде в рабочей среде с полной лицензией. Подготовка пробной лаборатории или пилотной среды может помочь в обеспечении использования операций по обеспечению безопасности для лиц, принимающих решения в Организации. Когда вы закончите работу с имитацией атаки и удовлетворены результатами, вы сможете полностью развернуть и оператионализе его в вашей организации с помощью технических специалистов Майкрософт по продажам или экспертов в вашей организации. 

Это руководство поможет вам:
- Настройка сервера лаборатории и компьютеров
- Настройка Active Directory для пользователей и групп
- Настройка и настройка защитника Майкрософт для идентификации, защитник Майкрософт для Office 365, защитник Майкрософт для конечной точки и Microsoft Cloud App Security
- Настройка локальных политик для сервера и компьютеров
- Имитация атаки на угрозу для создания тестового инцидента или оповещения в защитнике Microsoft 365

>[!IMPORTANT]
>Для достижения оптимальных результатов выполните инструкции по настройке лаборатории, насколько это возможно.

## <a name="in-scope"></a>В области

В этом руководстве представлены следующие задачи:
-   Настройка Azure Active Directory
-   Настройка защитника Microsoft 365
    -   Подпишитесь на пробную версию Microsoft 365 и используйте полную лицензию, если вы используете пилотный проект
    -   Настройка домена
    -   Назначение лицензий на Microsoft 365
    -   Завершение работы мастера установки на портале
-   Настройка всех базовых принципов защитника Microsoft 365 на основе рекомендаций
    -   Microsoft Defender для Office 365
    -   Microsoft Defender для удостоверений
    -   Microsoft Cloud App Security
    -   Microsoft Defender для конечной точки

-   Тестирование уязвимости в рабочей среде
