## <a name="create-roles-and-assign-the-role-to-an-azure-active-directory-group"></a>Создание ролей и назначение роли группе Azure Active Directory

Следующие действия по созданию ролей в Центр безопасности в Microsoft Defender. Предполагается, что вы уже создали Azure Active Directory групп пользователей.

1. Войдите [Центр безопасности в Microsoft Defender](https://securitycenter.windows.com/) учетную запись с помощью администратора безопасности или роли глобального администратора.

2. В области навигации выберите **Параметры > роли**.

3. Выберите **элемент Добавить**.

4. Введите имя роли, описание и разрешения, которые необходимо назначить роли.

5. Выберите **Далее,** чтобы назначить роль группе безопасности Azure AD.

6. Используйте фильтр, чтобы выбрать группу Azure AD, к которую необходимо добавить эту роль.

7. **Сохранить и закрыть**.

8. Применить параметры конфигурации.

> [!IMPORTANT]
> После создания ролей необходимо создать группу устройств и предоставить доступ к группе устройств, назначив ее только что созданной роли.

### <a name="permission-options"></a>Параметры разрешений

- **Просмотр данных**
    - **Операции безопасности** — просмотр всех данных операций безопасности на портале
    - **Угроза и управление уязвимостями** — просмотр контроль угроз и уязвимостей данных на портале

- **Активные действия по исправлению**
    - **Операции безопасности** . Принятие ответных действий, утверждение или отклонение в ожидании действий по исправлению, управление разрешенными и заблокированными списками для автоматизации и индикаторов
    - **Обработка управление уязвимостями** и управление уязвимостями - Создание новых исключений и управление активными исключениями
    - **Обработка угроз и управление уязвимостями -** Отправка новых запросов на исправление, создание билетов и управление существующими действиями по исправлению

- **Проверка** оповещений . Управление оповещениями, инициирование автоматических расследований, выполнение сканов, сбор пакетов расследований, управление тегами устройств и скачивание только переносных исполняемых (PE) файлов 

- **Управление настройками** системы портала — Настройка параметров хранения, параметров API siem и угроз intel (применяется глобально), расширенных параметров, автоматических загрузок файлов, ролей и групп устройств

    > [!NOTE]
    > Этот параметр доступен только в роли администратора конечной точки Microsoft Defender для конечной точки (по умолчанию).

- **Управление настройками** безопасности в Центре безопасности — настройка параметров подавления оповещений, управление исключениями папок для автоматизации, устройств на борту и на борту, а также управление уведомлениями электронной почты, управление лабораторией оценки

- **Возможности живого ответа**
    - **Основные** команды:
        - Запуск сеанса живого ответа
        - Выполните чтение только команд живого ответа на удаленном устройстве (за исключением копирования и выполнения файлов
    - **Расширенные** команды:
        - Скачивание файла с удаленного устройства с помощью живого ответа
        - Скачивание файлов PE и non-PE со страницы файла
        - Upload файл на удаленное устройство
        - Просмотр сценария из библиотеки файлов
        - Выполнение скрипта на удаленном устройстве из библиотеки файлов

Дополнительные сведения о доступных командах см. в сайте [Investigate devices using Live response.](live-response.md)
  
## <a name="edit-roles"></a>Изменение ролей

1. Войдите [Центр безопасности в Microsoft Defender](https://securitycenter.windows.com/) учетную запись с помощью администратора безопасности или роли глобального администратора.

2. В области навигации выберите **Параметры > роли**.

3. Выберите роль, которую вы хотите изменить.

4. Щелкните **Изменить**.

5. Измените сведения или группы, которые назначены роли. 

6. Нажмите **кнопку Сохранить и закрыть**.

## <a name="delete-roles"></a>Удаление ролей

1. Войдите [Центр безопасности в Microsoft Defender](https://securitycenter.windows.com/) учетную запись с помощью администратора безопасности или роли глобального администратора.

2. В области навигации выберите **Параметры > роли**.

3. Выберите роль, которую необходимо удалить.

4. Нажмите кнопку drop-down и выберите **роль Удалить**.
