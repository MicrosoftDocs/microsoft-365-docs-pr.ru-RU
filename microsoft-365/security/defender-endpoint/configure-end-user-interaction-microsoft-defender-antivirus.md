# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Настройка взаимодействия конечных пользователей с антивирусная программа в Microsoft Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Область применения:**

- [Microsoft Defender для конечной точки](/microsoft-365/security/defender-endpoint/)

Можно настроить взаимодействие пользователей конечных точек в сети с антивирусная программа в Microsoft Defender. Можно настроить, видят ли пользователи интерфейс антивирусная программа в Microsoft Defender, какие уведомления они видят, и могут ли они локально переопределять глобально развернутые параметры групповой политики.

Используйте следующие статьи для настройки взаимодействия конечных пользователей с антивирусная программа в Microsoft Defender

- [Настройка уведомлений, которые отображаются в конечных точках](configure-notifications-microsoft-defender-antivirus.md) Настройка и настройка уведомлений, настраиваемых текстов для уведомлений и уведомлений о перезагрузке для устранения

- [Запретить пользователям видеть](prevent-end-user-interaction-microsoft-defender-antivirus.md) или взаимодействовать с антивирусная программа в Microsoft Defender интерфейсом пользователя Сокрытие пользовательского интерфейса от пользователей

- [Предотвращение локального изменения параметров политики пользователями](configure-local-policy-overrides-microsoft-defender-antivirus.md) Запретить (или разрешить) пользователям переопределять параметры политики на отдельных конечных точках
