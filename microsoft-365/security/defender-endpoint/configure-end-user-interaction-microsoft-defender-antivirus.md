# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Настройка взаимодействия конечных пользователей с антивирусная программа в Microsoft Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Область применения:**

- [Microsoft Defender для конечной точки](/microsoft-365/security/defender-endpoint/)

Вы можете настроить, как пользователи конечных точек в вашей сети могут взаимодействовать с антивирусная программа в Microsoft Defender. Можно настроить, видят ли пользователи интерфейс антивирусная программа в Microsoft Defender, какие уведомления они видят, и могут ли они локально переопределить глобально развернутые настройки групповой политики.

Используйте следующие статьи для настройки взаимодействия конечных пользователей с антивирусная программа в Microsoft Defender

- **[Настройка уведомлений, которые отображаются в конечных точках](configure-notifications-microsoft-defender-antivirus.md)** Настройка и настройка уведомлений, включая текст для уведомлений, и уведомлений о перезагрузках, необходимых для исправления.

- **[Предотвращение просмотра или взаимодействия пользователей антивирусная программа в Microsoft Defender интерфейсом](prevent-end-user-interaction-microsoft-defender-antivirus.md)** Скрыть вирус **& защиты пользовательского интерфейса** от конечных пользователей.

- **[Предотвращение локального изменения параметров политики пользователями](configure-local-policy-overrides-microsoft-defender-antivirus.md)** Предотвращение (или разрешить) пользователям переопределять параметры политики в своих отдельных конечных точках.
