# <a name="manage-automation-folder-exclusions"></a>Управление исключениями из папки автоматизации 

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Область применения:**
- [Microsoft Defender для конечной точки](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>Хотите испытать Defender для конечной точки? [Зарегистрився для бесплатной пробной.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-automationexclusionfolder-abovefoldlink)

Исключения из папки автоматизации позволяют указать папки, которые будет пропускать автоматическое расследование. 

Вы можете управлять следующими атрибутами в папке, которую необходимо пропустить:
- Folders 
- Расширения файлов
- Имена файлов


**Folders**<br>
Вы можете указать папку и ее подмостки, которые необходимо пропустить. 


>[!NOTE]
>В настоящее время использование wild cards в качестве способа исключения файлов в каталоге еще не поддерживается. 


**Расширения**<br>
Можно указать расширения, которые необходимо исключить в определенном каталоге. Расширения — это способ предотвратить использование злоумышленником исключенной папки для сокрытия эксплойта. Расширения явно определяют, какие файлы игнорировать. 

**Имена файлов**<br>
Вы можете указать имена файлов, которые необходимо исключить в определенном каталоге. Имена — это способ предотвратить использование злоумышленником исключенной папки для сокрытия эксплойта. Имена явно определяют, какие файлы игнорировать. 



## <a name="add-an-automation-folder-exclusion"></a>Добавление исключения папки автоматизации
1. В области навигации выберите **исключения из папки "Автоматизация**  >  **параметров".**  

2. Щелкните **исключение из новой папки**.  

3. Введите сведения о папке:

    - Folder
    - Расширения
    - Имена файлов
    - Описание
    

4. Нажмите кнопку **Сохранить**.

>[!NOTE]
> Команды Live Response для сбора или изучения исключенных файлов сбой с ошибкой: "Файл исключен". Кроме того, автоматические исследования будут игнорировать исключенные элементы.

## <a name="edit-an-automation-folder-exclusion"></a>Изменение исключения папки автоматизации 
1. В области навигации выберите **исключения из папки "Автоматизация**  >  **параметров".** 

2. Нажмите **кнопку Изменить** исключение папки.  

3. Обновите сведения о правиле и нажмите кнопку **Сохранить**.

## <a name="remove-an-automation-folder-exclusion"></a>Удаление исключения папки автоматизации 
1. В области навигации выберите **исключения из папки "Автоматизация**  >  **параметров".**  
2. Нажмите **кнопку Удалить исключение**.
