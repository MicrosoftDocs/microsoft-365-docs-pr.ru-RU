# <a name="microsoft-defender-for-endpoint-device-control-removable-storage-protection"></a>Microsoft Defender для управления конечными устройствами, которые можно служба хранилища защиты

[!INCLUDE [Prerelease](../includes/prerelease.md)]

Защита Microsoft Defender для удаления служба хранилища устройств с конечными точками не позволяет пользователю или машине или обоим использовать несанкционированные съемные носитли хранения.

## <a name="protection-policies"></a>Политики защиты

### <a name="device-installation"></a>Установка устройства

**Возможности** . Предотвратить установку с исключением или без исключения на основе различных свойств устройства.

**Windows 10 сведения о поддержке**
- Применяется на уровне машины: та же политика применяется и для любого пользователя, зарегистрированного в журнале.
- Поддерживает MEM и GPO.
- Поддерживаемые свойства[устройств,](#device-properties)как указано в списке.
- Дополнительные сведения о Windows см. в веб-сайте [How to control USB devices and other removable media using Microsoft Defender for Endpoint.](control-usb-devices-using-intune.md)

**Поддерживаемая платформа** — Windows 10

**сведения о поддержке macOS**
- Применяется на уровне машины: та же политика применяется и для любого пользователя, зарегистрированного в журнале.
- Сведения о macOS см. в [специальном см. для управления устройствами macOS.](mac-device-control-overview.md)
 
**Поддерживаемая платформа** — macOS Catalina 10.15.4+ (с включенными расширениями системы)

### <a name="removable-storage-access-control"></a>Управление доступом к съемным хранилищам

**Capabilities**
- *Аудит* Чтение или написание или выполнение доступа к съемным хранилищам на основе различных свойств устройства с исключением или без него.
- *Предотвращение* Чтение или написание или выполнение доступа с исключением или без него . Разрешить определенное устройство на основе различных свойств устройства.

**Windows 10 сведения о поддержке**
- Применяется либо на компьютере, либо на пользователе, либо на обоих устройствах — только разрешить определенным пользователям, которые выполняют чтение/записи или выполняют доступ к определенному съемного хранилища на определенном компьютере.
- Поддержка MEM OMA-URI и GPO.
- Поддерживаемые свойства[устройств,](#device-properties)как указано в списке.
- Для функции в Windows см. в этой Windows [Элемент управления доступом к хранилищам.](device-control-removable-storage-access-control.md)

**Поддерживаемая платформа** — Windows 10

**сведения о поддержке macOS**
- Применяется на уровне машины: та же политика применяется и для любого пользователя, зарегистрированного в журнале.
- Сведения о macOS см. в [специальном см. для управления устройствами macOS.](mac-device-control-overview.md)
 
**Поддерживаемая платформа** — macOS Catalina 10.15.4+ (с включенными расширениями системы)

### <a name="windows-portable-device-access-control"></a>Windows Управление доступом к портативному устройству

**Возможности** — запретить чтение или [](/windows-hardware/drivers/portable/)записывать доступ к Windows портативному устройству, например: Планшет, iPhone.

**Описание**
- Применяется либо на компьютере, либо на пользователе, либо на обоих устройствах.
- Поддержка MEM OMA-URI и GPO.

**Поддерживаемая платформа** — Windows 10

### <a name="endpoint-dlp-removable-storage"></a>Съемное хранилище конечной точки DLP

**Возможности** — аудит или предупреждение или предотвращение копирования элемента или сведений на съемное мультимедиа или USB-устройство.

**Описание** . Дополнительные сведения о Windows см. в Microsoft 365 [предотвращении потери данных конечной точки.](../../compliance/endpoint-dlp-learn-about.md)

**Поддерживаемая платформа** — Windows 10

### <a name="bitlocker"></a>BitLocker 

**Capabilities**
- Блокируют данные, которые будут записаны на съемные диски, которые BitLocker защищены.
- Блокируют доступ к съемным дискам, если они не зашифрованы на компьютере, который принадлежит вашей организации.
 
**Описание** . Дополнительные сведения о Windows см. в [BitLocker - Съемный диск Параметры](/mem/intune/protect/endpoint-security-disk-encryption-profile-settings).

**Поддерживаемая платформа** — Windows 10
