# <a name="access-the-microsoft-defender-for-endpoint-community-center"></a>Получение доступа к Центру сообщества Microsoft Defender для конечной точки

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Область применения:**
- [Microsoft Defender для конечной точки](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>Хотите испытать Defender для конечной точки? [Зарегистрився для бесплатной пробной.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-assignaccess-abovefoldlink)

Центр сообщества Defender для конечных точек — это место, где члены сообщества могут учиться, сотрудничать и обмениваться опытом о продукте. 

Существует несколько пробелов, которые можно изучить, чтобы узнать о конкретных сведениях:
- Объявления 
- Новые возможности
- Аналитика угроз


Существует несколько способов доступа к Центру сообщества:
- В области навигации Центра безопасности Microsoft Defender выберите **центр сообщества.**  Откроется новая вкладка браузера, которая переназначит вас на страницу "Защитник для конечной точки". 
- Доступ к сообществу через страницу [Microsoft Defender for Endpoint Tech Community](https://techcommunity.microsoft.com/t5/Windows-Defender-Advanced-Threat/ct-p/WindowsDefenderAdvanced)


Вы можете мгновенно просматривать и читать беседы, которые были размещены в сообществе. 

Чтобы получить полный опыт работы в сообществе, например возможность комментировать публикации, необходимо присоединиться к сообществу. Дополнительные сведения о том, как начать работу в сообществе Microsoft Tech, см. в [веб-сайте Microsoft Tech Community: Getting Started](https://techcommunity.microsoft.com/t5/Getting-Started/Microsoft-Tech-Community-Getting-Started-Guide/m-p/77888#M15).
