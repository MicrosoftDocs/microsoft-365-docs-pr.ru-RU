# <a name="investigate-an-ip-address-associated-with-a-microsoft-defender-for-endpoint-alert"></a>Исследование IP-адреса, связанного с оповещением Microsoft Defender for Endpoint

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Область применения:**
- [Microsoft Defender для конечной точки](https://go.microsoft.com/fwlink/p/?linkid=2146631)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


>Хотите испытать Defender для конечной точки? [Зарегистрився для бесплатной пробной.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-investigateip-abovefoldlink)

Изучите возможную связь между устройствами и внешними IP-адресами протокола Интернета.

Определение всех устройств в организации, которые связывались с подозрительным или известным вредоносным IP-адресом, например серверами Command and Control (C2), помогает определить потенциальные области нарушения, связанные файлы и зараженные устройства.

Сведения из следующих разделов можно найти в представлении IP-адресов:

- IP во всем мире
- Обратные имена DNS
- Оповещений, связанных с этим IP
- IP в организации
- Распространенность

## <a name="ip-worldwide-and-reverse-dns-names"></a>Ip Worldwide и обратные имена DNS

В разделе сведения об IP-адресе показаны атрибуты IP-адреса, такие как ASN и имена обратных DNS.

## <a name="alerts-related-to-this-ip"></a>Оповещений, связанных с этим IP

**Оповещений, связанных с этим** ip-разделом, содержится список оповещений, связанных с IP-адресом.

## <a name="ip-in-organization"></a>IP в организации

В **разделе IP в организации** приводится подробная информация о распространенности IP-адреса в организации.

## <a name="prevalence"></a>Распространенность

В **разделе "Распространенность"** отображается количество устройств, подключенных к этому IP-адресу, а также время первого и последнего отображения IP-адреса. Результаты этого раздела можно отфильтровать по времени; по умолчанию — 30 дней.

## <a name="most-recent-observed-devices-with-ip"></a>Самые последние наблюдаемые устройства с IP

Последние **наблюдаемые устройства** с разделом IP предоставляют хронологическое представление о событиях и связанных оповещениях, которые наблюдались на IP-адресе.

**Исследование внешнего IP-адреса:**

1. Выберите **IP-адрес** **из** выпадаемого меню панели поиска.
2. Введите IP-адрес в **поле Поиска.**
3. Щелкните значок поиска или нажмите **кнопку Ввод**.

Отображаются сведения об IP-адресе, в том числе: сведения о регистрации (при наличии), обратные IP-адреса (например, домены), распространенность устройств в организации, связывающихся с этим IP-адресом (в течение определенного периода времени), а также устройства в организации, наблюдаемые при общении с этим IP-адресом.

> [!NOTE]
> Результаты поиска будут возвращены только для IP-адресов, наблюдаемых при общении с устройствами в организации.

Для определения критериев поиска используйте фильтры поиска. Вы также можете использовать поле поиска по временной шкале, чтобы отфильтровать отображаемые результаты всех устройств в организации, наблюдающих связь с IP-адресом, файлом, связанным с сообщением, и последней датой.

Щелкнув любое из имен устройств, вы сможете просмотреть представление этого устройства, где можно продолжить изучение сообщений о оповещениях, поведении и событиях.
