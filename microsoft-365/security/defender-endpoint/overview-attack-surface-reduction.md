# <a name="overview-of-attack-surface-reduction-capabilities"></a>Обзор возможностей уменьшения поверхности атаки

Поверхности атак — это все места, в которых ваша организация уязвима для киберугроз и атак. Defender for Endpoint включает несколько возможностей, которые помогут уменьшить количество поверхностей атак. Просмотрите следующее видео, чтобы узнать больше о сокращении поверхности атаки.<p>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4woug]

## <a name="resources-to-learn-more-about-attack-surface-reduction"></a>Ресурсы, чтобы узнать больше о снижении поверхности атаки

Как упоминалось в видео, Defender for Endpoint включает несколько возможностей уменьшения поверхности атаки. Дополнительные возможности использования следующих ресурсов:

| Статья | Описание |
|:---|:---|
| [Аппаратная изоляция](/windows/security/threat-protection/microsoft-defender-application-guard/md-app-guard-overview) | Защита и сохранение целостности системы по мере ее запуска и работы. Проверка целостности системы с помощью локальной и удаленной проверки. И используйте изоляцию контейнеров для Microsoft Edge защиты от вредоносных веб-сайтов. |
| [Элемент управления приложениями](/windows/security/threat-protection/windows-defender-application-control/windows-defender-application-control) | Используйте управление приложениями, чтобы ваши приложения заработали доверие для запуска. |
| [Контролируемый доступ к папкам](controlled-folders.md) | Помогите предотвратить внесение изменений в файлы в папках ключевой системы (требуется антивирусная программа в Microsoft Defender) |
| [Защита сети](network-protection.md) | Расширение защиты сетевого трафика и подключения на устройствах организации. (Требуется антивирусная программа в Microsoft Defender) |
| [Защита от эксплойтов](exploit-protection.md) | Защита операционных систем и приложений, которые использует организация, от использования. Защита от эксплойтов также работает с сторонними антивирусными решениями. |
| [Правила сокращения направлений атак](attack-surface-reduction.md) | Уменьшение уязвимостей (направлений атак) в ваших приложениях с помощью интеллектуальных правил, которые помогают остановить вредоносные программы. (Требуется антивирусная программа в Microsoft Defender). |
| [Управление устройством](device-control-report.md) | Защищает от потери данных, отслеживая и контролируя средства массовой информации, используемые на устройствах, таких как съемные накопители и USB-накопители, в вашей организации. |
