# <a name="evaluate-microsoft-365-defender-for-endpoint-overview"></a>Оценка Microsoft 365 Defender для обзора конечных точек

**Область применения:**

- Microsoft 365 Defender

![Microsoft 365 Defender и процесс пилотирования](../../media/defender/m365-defender-eval-process.png)

Комплексные оценки продуктов безопасности могут быть сложным процессом, требующим громоздких конфигураций среды и устройств, прежде чем может быть фактически сделано моделирование атак. Сложность заключается в отслеживании того, где во время оценки отражаются действия моделирования, оповещения и результаты.

Лаборатория оценки Microsoft Defender для конечной точки предназначена для устранения сложностей конфигурации устройства и среды. Это позволяет:

- Фокус на оценке возможностей платформы
- Запуск имитаций
- См. функции предотвращения, обнаружения и восстановления в действии
<br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4woug]

С помощью упрощенной системы можно сосредоточиться на выполнении собственных сценариев тестирования и предварительного моделирования, чтобы узнать, как работает Defender для endpoint.

У вас будет полный доступ к мощным возможностям платформы, таким как автоматизированные расследования, расширенные поиски и аналитика угроз, что позволит протестировать комплексный стек защиты, который предлагает Defender для Endpoint.

Можно добавить Windows 10 или Windows Server 2019, которые предварительно настроены для установки последних версий ОС и нужных компонентов безопасности, а также Office стандарта 2019 г.

Вы также можете установить симуляторы угроз. Defender for Endpoint имеет партнерские отношения с ведущими отраслевыми платформами моделирования угроз, чтобы помочь вам проверить возможности Defender для конечной точки, не покидая портал.

 Установите предпочтительный симулятор, запустите сценарии в лаборатории оценки, а затем немедленно посмотрите, как работает платформа. Эта возможность удобно доступна без дополнительных затрат для вас. Вы также будете иметь удобный доступ к широкому спектру имитаций, к которым можно получить доступ и запустить из каталога моделирования.
