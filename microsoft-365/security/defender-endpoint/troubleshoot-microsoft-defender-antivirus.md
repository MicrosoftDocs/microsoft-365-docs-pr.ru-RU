## <a name="microsoft-defender-antivirus-event-ids"></a>ID-ID событий антивирусного события Microsoft Defender

Антивирус Microsoft Defender записи событий в журнале событий Windows.

Вы можете напрямую просмотреть журнал событий, или если у вас есть сторонний инструмент управления данными безопасности и событиями (SIEM), вы также можете использовать [ID-данные](troubleshoot-microsoft-defender-antivirus.md#windows-defender-av-ids) клиентских событий антивирусной программы Microsoft Defender для проверки определенных событий и ошибок с конечных точек.

В таблице в этом разделе перечислены основные ID событий антивирусного события Защитника Майкрософт и, по возможности, предлагаются предлагаемые решения для исправления или устранения ошибки. 

## <a name="to-view-a-microsoft-defender-antivirus-event"></a>Просмотр антивирусного события Microsoft Defender

1.  Open **Event Viewer**.
2.  В дереве консоли разойдите **журналы приложений** и **служб,** **затем Microsoft,** **затем Windows,** а затем Защитник Windows .
3.  Дважды нажмите кнопку **Оперативный**.
4.  В области сведений просмотреть список отдельных событий, чтобы найти событие.
5.  Щелкните событие, чтобы увидеть конкретные сведения о событии в нижней области в вкладке **Общие** и **Подробные** сведения.
