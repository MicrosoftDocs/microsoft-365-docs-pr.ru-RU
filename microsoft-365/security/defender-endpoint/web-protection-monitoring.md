# <a name="monitor-web-browsing-security"></a>Мониторинг безопасности просмотра веб-страниц

**Область применения:**
- [Microsoft Defender для конечной точки](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>Хотите испытать Microsoft Defender для конечной точки? [Зарегистрився для бесплатной пробной.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-main-abovefoldlink&rtc=1)

Веб-защита позволяет отслеживать безопасность просмотра веб-страниц в  организации с помощью отчетов в > веб-защиты в Центре безопасности Защитника Майкрософт. Отчет содержит карточки, которые предоставляют статистику обнаружения веб-угроз.

- Обнаружение **веб-угроз** со временем — эта карта тренда отображает количество веб-угроз, обнаруженных по типу в течение выбранного периода времени (Последние 30 дней, Последние 3 месяца, Последние 6 месяцев)
 
    ![Изображение карты, показывающая обнаружение защиты от веб-угроз со временем](images/wtp-blocks-over-time.png)

- **Сводка по** защите от веб-угроз — эта карта отображает общее количество обнаружений веб-угроз за последние 30 дней, демонстрируя распространение различных типов веб-угроз. Выбор фрагмента открывает список доменов, найденных на вредоносных или нежелательных веб-сайтах.

    ![Изображение карты с сводкой защиты от веб-угроз](images/wtp-summary.png)

>[!Note]
>Это может занять до 12 часов, прежде чем блок будет отражен в картах или списке домена.

## <a name="types-of-web-threats"></a>Типы веб-угроз

Веб-защита классифицируют вредоносные и нежелательные веб-сайты как:

- **Фишинг —** веб-сайты, содержащие поддельные веб-формы и другие механизмы фишинга, предназначенные для обмана пользователей в разглашение учетных данных и других конфиденциальных сведений
- **Вредоносные** — веб-сайты с вредоносными программами и кодом эксплойтов
- **Настраиваемый индикатор** — веб-сайты, URL-адреса или домены которых добавлены в [настраиваемый](manage-indicators.md) список индикаторов для блокировки

## <a name="view-the-domain-list"></a>Просмотр списка доменов

Выберите определенную категорию веб-угроз в сводной карте **защиты** от веб-угроз, чтобы открыть страницу **Домены.** На этой странице отображается список доменов этой категории угроз. На странице приводится следующая информация для каждого домена:

- **Количество запросов** на URL-адреса в домене
- **Блоки** — количество запросов было заблокировано
- **Тенденция доступа** — изменение количества попыток доступа
- **Категория угрозы** — тип веб-угрозы
- **Устройства** — число устройств с попытками доступа

Выберите домен, чтобы просмотреть список устройств, которые пытались получить доступ к URL-адресам в этом домене, и список URL-адресов.
