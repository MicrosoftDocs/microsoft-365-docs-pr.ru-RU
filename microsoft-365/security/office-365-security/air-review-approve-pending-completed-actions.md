# <a name="review-and-manage-remediation-actions-in-office-365"></a>Просмотр и управление действиями по исправлению в Office 365

**Область применения**
- [Microsoft Defender для Office 365 (план 2)](defender-for-office-365.md)

По мере автоматического расследования & контента совместной работы создаются определенные действия по исправлению, например вредоносные или подозрительные.  В Microsoft Defender для Office 365 действий по исправлению можно включить следующие действия:

- Мягкое удаление сообщений электронной почты или кластеров
- Отключение внешней пересылаемой почты

Эти действия по исправлению не принимаются до тех пор, пока их не утвердит группа операций безопасности. Мы рекомендуем как можно скорее просмотреть и утвердить все ожидающие действия, чтобы автоматические расследования были завершены своевременно. В некоторых случаях можно пересмотреть представленные действия.  Вы должны быть частью роли & очистки перед принятием каких-либо действий.


## <a name="approve-or-reject-pending-actions"></a>Утверждение (или отклонение) ожидающих действий
Существует четыре различных способа поиска и проведения действий по автоматическому расследованию:

- [Очередь инцидентов](https://security.microsoft.com/incidents)
- [Центр уведомлений](https://security.microsoft.com/action-center/pending)
- Само расследование (доступ к нему через incident или из оповещений)
- [Очередь расследований и исправлений](https://security.microsoft.com/airinvestigation)

## <a name="incident-queue"></a>Очередь инцидентов
1. Перейдите в [центр Microsoft 365 безопасности и](https://security.microsoft.com) войдите.
2. В области навигации выберите **& оповещения > происшествия**.
3. Выберите имя инцидента, чтобы открыть его сводную страницу.
4. Выберите **вкладку Evidence and Response.**
5. Выберите элемент в списке. Откроется его боковое окно.
6. В боковой области примите утверждение или отклонение действий.

## <a name="investigation-queue"></a>Очередь расследования 
1. Перейдите в [центр Microsoft 365 безопасности и](https://security.microsoft.com) войдите.
2. Перейдите со страницы оповещений и инцидентов. 
3. На странице Исследование перейдите на вкладку **ожидающих действий.** 
4. Выберите элемент в списке. Откроется его боковое окно.  
5. В боковой области примите утверждение или отклонение действий.

## <a name="action-center"></a>Центр уведомлений
1. Перейдите в [центр Microsoft 365 безопасности и](https://security.microsoft.com) войдите.
2. В области навигации выберите **Центр действий.**
3. На **вкладке Ожидание** просмотрите список действий, ожидающих утверждения.
   - Выберите **страницу Open investigation,** чтобы просмотреть дополнительные сведения о расследовании.
   - Выберите **Утверждение,** чтобы инициировать ожидающих действий.
   - Выберите **Отклонение,** чтобы предотвратить ожидающих действий.

## <a name="investigation-and-remediation-investigations-queue"></a>Очередь расследований и исправлений
1. Перейдите в [центр Microsoft 365 безопасности и](https://security.microsoft.com) войдите.
2. Откройте ожидающих расследований. 
3. На странице Исследование перейдите на вкладку **ожидающих действий.**
4. Выберите элемент в списке. Откроется его боковое окно.  
5. В боковой области примите утверждение или отклонение действий.

## <a name="change-or-undo-one-remediation-action"></a>Изменение или отмена одного действия по исправлению

Существует два различных способа пересмотра представленных действий:
   - Через [единый центр действий](https://security.microsoft.com/action-center).
   - Хотя центр [Office действий](https://security.microsoft.com/threatincidents).
   
## <a name="change-or-undo-through-the-unified-action-center"></a>Изменение или отмена с помощью единого центра действий
1. Перейдите в [единый центр действий и](https://security.microsoft.com/action-center) войдите.
2. На **вкладке История** выберите действие, которое необходимо изменить или отменить.
3. В области на правой стороне экрана выберите соответствующеедействие (переходить в почтовый **ящик,** перейти к нежелательной **,** перейти к удаленным элементов , **soft delete", или жесткое **удаление**).

 ## <a name="change-or-undo-through-the-office-action-center"></a>Изменение или отмена через центр Office действий 
1. Перейдите в [центр Office действий и](https://security.microsoft.com/threatincidents) войдите.
2. Выберите соответствующее исправление.
3. В боковой области щелкните запись отправки почты и дождись загрузки списка. 
4. Подождите кнопку Действие в верхней части, чтобы включить и выбрать кнопку Действие, чтобы изменить тип действия. 
5. Это создаст соответствующие действия.
