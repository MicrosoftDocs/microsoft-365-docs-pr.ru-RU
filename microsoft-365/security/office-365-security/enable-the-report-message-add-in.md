# <a name="enable-the-report-message-or-the-report-phishing-add-ins"></a>Включить сообщение отчета или надстройки для фишинга отчетов

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**Область применения**
- [Exchange Online Protection](exchange-online-protection-overview.md)
- [Microsoft Defender для Office 365 (план 1 и план 2)](defender-for-office-365.md)
- [Microsoft 365 Defender](../defender/microsoft-365-defender.md)

> [!NOTE]
> Если вы администратор в организации Microsoft 365 с почтовыми ящиками Exchange Online, рекомендуем использовать портал Отправки на портале Microsoft 365 Defender. Дополнительные сведения см. в материале Использование отправки администратора для отправки в Корпорацию Майкрософт подозрительных [спама, фишинга, URL-адресов и файлов.](admin-submission.md)

Надстройки Для Outlook и Outlook в Интернете (ранее известные как Outlook Web App) позволяют пользователям легко сообщать о ложных срабатываниях (хорошая электронная почта, помеченная как плохие) или ложных отрицательных (разрешенная плохая электронная почта) в Корпорацию Майкрософт и ее филиалы для анализа.

Корпорация Майкрософт использует эти материалы для повышения эффективности технологий защиты электронной почты. Например, предположим, что люди сообщают много сообщений с помощью надстройки Report Phishing. Эти сведения будут использоваться в панели мониторинга безопасности и других отчетах. Группа безопасности организации может использовать эти сведения в качестве указания на необходимость обновления политик защиты от фишинга.

Можно установить надстройку Report Message или Report Phishing. Если вы хотите, чтобы пользователи сообщали о спаме и фишинговых сообщениях, разместите надстройку Report Message в организации. Дополнительные сведения см. в добавлении Enable the Report Message.

Надстройка Report Message предоставляет возможность сообщать о спаме и фишинговых сообщениях. Администраторы могут включить надстройка Report Message для организации, а отдельные пользователи могут установить ее для себя.

Надстройка Report Phishing предоставляет возможность сообщать только о фишинговых сообщениях. Администраторы могут включить надстройку Report Phishing для организации, а отдельные пользователи могут установить ее для себя.

Если вы индивидуальный пользователь, вы можете включить оба надстройки для себя.

Если вы глобальный администратор или администратор Exchange Online, а Exchange настроена на использование проверки подлинности OAuth, вы можете включить надстройку Report Message и надстройку Report Phishing для вашей организации. Обе надстройки теперь доступны через [централизованное развертывание.](../../admin/manage/centralized-deployment-of-add-ins.md)

## <a name="what-do-you-need-to-know-before-you-begin"></a>Что нужно знать перед началом работы

- Надстройка Сообщения отчетов и надстройка Report Phishing работают с большинством подписок Microsoft 365 и следующими продуктами:
  - Outlook в Интернете
  - Outlook 2013 SP1 или более поздний
  - Outlook 2016 для Mac;
  - Outlook, включенный в приложения Microsoft 365 для предприятия
  - Приложение Outlook для iOS и Android

- Обе надстройки недоступны для общих почтовых ящиков или почтовых ящиков в локальной организации Exchange.

- Существующий веб-браузер должен работать как с надстройки Report Message, так и с надстройки Report Phishing. Но если вы заметили, что надстройка недоступна или работает не так, как ожидалось, попробуйте другой браузер.

- Для организационных установок необходимо настроить организацию для использования проверки подлинности OAuth. Дополнительные сведения см. в [рубриках Определение,](../../admin/manage/centralized-deployment-of-add-ins.md)работает ли централизованное развертывание надстройок для вашей организации.

- Администраторы должны быть членами группы ролей глобальных администраторов. Дополнительные сведения см. [в веб-сайте Permissions in the Microsoft 365 Defender portal](permissions-microsoft-365-security-center.md).

- Дополнительные сведения о том, как сообщать о сообщении с помощью функции Сообщение отчетов, см. в обзоре [Report false positives and false negatives in Outlook.](report-false-positives-and-false-negatives.md)

> [!IMPORTANT]
> Мы не рекомендуем встроенный интерфейс отчетности в Outlook, так как он не может использовать политику отправки [пользователей.](./user-submission.md) Рекомендуется использовать надстройку "Сообщение отчетов" или надстройку Report Phishing.

## <a name="get-the-report-message-add-in"></a>Получить надстройки сообщения отчета

### <a name="get-the-report-message-add-in-for-yourself"></a>Получите надстройка "Сообщение отчетов" для себя

1. Перейдите в Microsoft AppSource и <https://appsource.microsoft.com/marketplace/apps> найдите надстройки "Сообщение отчетов". Чтобы перейти непосредственно к надстройки Сообщение отчета, перейдите к <https://appsource.microsoft.com/product/office/wa104381180> .

2. Нажмите **КНОПКУ GET IT NOW**.

   ![Сообщение отчета — получите его сейчас](../../media/ReportMessageGETITNOW.png)

3. В диалоговом окантове, который появляется, просмотрите условия использования и политику конфиденциальности, а затем нажмите **кнопку Продолжить**.

4. Впишитесь с помощью учетной записи работы или учебного заведения (для бизнеса) или учетной записи Майкрософт (для личного использования).

После установки и включения надстройки вы увидите следующие значки:

- В Outlook значок выглядит так:

  > [!div class="mx-imgBorder"]
  > ![Значок надстройки Report Message для Outlook](../../media/OutlookReportMessageIcon.png)

- В Outlook в Интернете значок выглядит так:

  > [!div class="mx-imgBorder"]
  > ![Outlook на значке надстройки "Сообщение о веб-отчете"](../../media/owa-report-message-icon.png)

### <a name="get-the-report-message-add-in-for-your-organization"></a>Получите надстройка "Сообщение отчета" для организации

> [!NOTE]
> Для появления надстройки в организации может занять до 12 часов.

1. В центре администрирования Microsoft 365  перейдите на страницу \> **Надстройки** параметров по <https://admin.microsoft.com/AdminPortal/Home#/Settings/AddIns> крайней мере . Если вы не видите  страницу надстройки, перейдите к ссылке Надстройки "Параметры интегрированных приложений" в верхней части страницы  \>  \>  **Интегрированные** приложения.

2. Выберите **развертывание надстройки** в верхней части страницы, а затем выберите **Далее**.

   ![Страница Службы и надстройки в центре администрирования Microsoft 365](../../media/ServicesAddInsPageNewM365AdminCenter.png)

3. В **развертывании новой** надстройки, которая появится, просмотрите сведения, а затем нажмите **кнопку Далее**.

4. На следующей странице нажмите **кнопку Выберите из магазина**.

   ![Развертывание новой страницы надстройки](../../media/NewAddInScreen2.png)

5. На странице **Выберите надстройку,** которая  появится, щелкните в поле Поиск, **введите** сообщение отчета, а затем нажмите **значок Поиск** поиска ![ ](../../media/search-icon.png) . В списке результатов найдите **сообщение отчета** и нажмите кнопку **Добавить**.

   ![Выбор результатов поиска надстройки](../../media/NewAddInScreen3.png)

6. В диалоговом окте, который появится, просмотрите сведения о лицензировании и конфиденциальности, а затем нажмите **кнопку Продолжить**.

7. На **странице Настройка надстройки,** которая появится, настройте следующие параметры:

   - **Назначены пользователи:** Выберите одно из следующих значений:
     - **Все** (по умолчанию)
     - **Конкретные пользователи / группы**
     - **Только я**

   - **Метод развертывания:** Выберите одно из следующих значений:
     - **Исправлено (по умолчанию).** Надстройка автоматически развертывается для указанных пользователей, и они не могут удалить ее.
     - **Доступно.** Пользователи могут установить надстройки в **Home** \> **Get надстройки,** \> **управляемые администратором.**
     - **Необязательный** вариант: надстройка автоматически развертывается для указанных пользователей, но они могут удалить ее.

   ![Настройка страницы надстройки](../../media/configure-add-in.png)

   По завершению щелкните **Развертывание**.

8. На странице **Развертывание сообщения отчетов,** которая отображается, вы увидите отчет о ходе выполнения, за которым следует подтверждение развертывания надстройки. После прочтя сведений нажмите кнопку **Далее**.

   ![Развертывание страницы сообщения отчета](../../media/deploy-report-message-page.png)

9. На странице **Надстройка** "Объявить", которая появится, просмотрите сведения и нажмите **кнопку Закрыть**.

   ![Объявить страницу надстройки](../../media/announce-add-in-page.png)
