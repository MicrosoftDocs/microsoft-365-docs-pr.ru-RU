# <a name="auto-forwarded-messages-insight-in-the-security--compliance-center"></a>Аналитика автоматически пересланных сообщений в Центре безопасности & требованиям

На **информационной панели** потока обработки [Mail flow dashboard](mail-flow-insights-v2.md) почты в Центре соответствия требованиям безопасности & отображаются сведения о сообщениях, автоматически пересланных из вашей организации получателям во внешние домены.

![Мини-приложение "Автоматически пересылаемые сообщения" в Центре безопасности & соответствия требованиям](../../media/mfi-auto-forwarded-messages.png)

## <a name="auto-forwarded-messages-details"></a>Сведения об автоматической пересылке сообщений

Если щелкнуть количество сообщений в мини-приложении, появится всплывающий элемент, в котором отображаются дополнительные сведения об автоматически пересылаемых сообщениях:

- **Автоматически пересылаемые сообщения путем пересылки методов:**

  - **по правилам потока обработки почты;**
  - **Правилами для папки "Входящие"**
  - **пересылка SMTP;**
  - Ссылка на отчет о [пересылке для получения](view-mail-flow-reports.md#forwarding-report) дополнительных сведений.

- **Автоматически пересылаемые сообщения доменами и пользователями:**

  - **5: перенаправленные 5 доменов**
  - **Новые домены (последняя неделя)**
  - **5 пользователей с пересылкой: 5 пользователей с пересылк**
  - **Новые пользователи (прошлая неделя)**
  - Ссылка на отчет [изменений пересылки для получения дополнительных](mfi-new-users-forwarding-email.md#forwarding-modifications-report) сведений.

![Всплывающую информацию о отчете автоматически пересылаемых сообщений в Центре безопасности & требованиям](../../media/mfi-auto-forwarded-messages-details.png)

## <a name="insights"></a>Аналитика

На основе данных отчета создаются два данных:

- [Новые пользователи пересылают электронную почту](mfi-new-users-forwarding-email.md)
- [Сообщения электронной почты, пересылаемые для новых доменов](mfi-new-domains-being-forwarded-email.md)
