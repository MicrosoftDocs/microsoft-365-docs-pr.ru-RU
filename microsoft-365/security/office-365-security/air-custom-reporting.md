# <a name="custom-or-third-party-reporting-solutions-for-microsoft-defender-for-office-365"></a>Пользовательские или сторонние решения отчетности для Microsoft Defender для Office 365

С [помощью Microsoft Defender для Office 365](defender-for-office-365.md)вы получите подробные сведения об [автоматизированных расследованиях.](air-view-investigation-results.md) Однако в некоторых организациях также используется настраиваемая или сторонная отчетность. Если ваша организация хочет [](office-365-air.md) интегрировать сведения об автоматизированных расследованиях с таким решением, вы можете использовать API Office 365 управления.

**Область применения**
- [Microsoft Defender для Office 365 (план 2)](defender-for-office-365.md)
- [Microsoft 365 Defender](../defender/microsoft-365-defender.md)

С [помощью Microsoft Defender для Office 365](defender-for-office-365.md)вы получите подробные сведения об [автоматизированных расследованиях.](air-view-investigation-results.md) Однако в некоторых организациях также используется настраиваемая или сторонная отчетность. Если ваша организация хочет интегрировать сведения об автоматизированных расследованиях с таким решением, можно использовать API Office 365 управления.
