# <a name="report-messages-and-files-to-microsoft"></a>Передача информации о сообщениях и файлах в корпорацию Майкрософт

У всех организаций Microsoft 365 с почтовыми ящиками в Exchange Online или в автономных организациях Exchange Online Protection (EOP) без почтовых ящиков Exchange Online у обоих пользователей и администраторов есть несколько разных методов, связанных с отчетами сообщений электронной почты и файлов в корпорацию Майкрософт.

|[Включение надстройки Report Message](enable-the-report-message-add-in.md)|Работает с Outlook, Outlook для Mac и Outlook в Интернете (предыдущее название — Outlook Web App) и рекомендована к рекомендуемой надстройке. <br/><br/> Сообщения, отправленные пользователям с надстройкой, доступны на портале [отправки](admin-submission.md)администрирования, результатах автоматического исследования [и реагирования на них,](air-view-investigation-results.md)отчетов о [сообщениях, о](view-email-security-reports.md#user-reported-messages-report)которых пользователь оформлял, и [обозревателе угроз.](threat-explorer-views.md#email--submissions) <br/><br/> Вы можете настроить отправленные сообщения для копирования или перенаправления в заданный почтовый ящик. |[Отправка сообщений корпорации Майкрософт для анализа вручную](submit-spam-non-spam-and-phishing-scam-messages-to-microsoft-for-analysis.md)|Вручную отправлять вложенные сообщения на определенные адреса электронной почты Майкрософт для спама, а не спама и фишинга.|
