# <a name="non-delivery-report-in-the-security--compliance-center"></a>Отчет о недоставке в Центре безопасности & соответствия требованиям

Отчет **о недоставке на панели мониторинга** [потока](mail-flow-insights-v2.md) обработки почты в Центре безопасности & показывает коды ошибок в отчетах о недоставке (также известных как сообщения о недоставке) для пользователей в вашей организации. Этот отчет содержит подробные сведения об отчетах о недоставке электронной почты.

![Мини-приложение с отчетом о недоставке на панели мониторинга потока обработки почты в Центре безопасности & Compliance Center](../../media/mfi-non-delivery-report-widget.png)

## <a name="report-view-for-the-non-delivery-report"></a>Представление отчета о недоставке

Если щелкнуть **мини-приложение отчетов** о недоставке, откроется отчет **о недоставке.**

По умолчанию отображается активность для всех кодов ошибок. Если щелкнуть **"Показать данные",** можно выбрать в раскрывающемся списке определенный код ошибки.

Если наведите указатель мыши на определенный цвет (код ошибки) на конкретном дне диаграммы, вы увидите общее количество сообщений для ошибки.

![Просмотр отчетов в отчете о необслуживаемом домене](../../media/mfi-non-delivery-report-overview-view.png)

## <a name="details-table-view-for-the-non-delivery-report"></a>Представление таблицы подробностей для отчета о недоставке

При нажатии **кнопки "Просмотр таблицы** сведений" в представлении отчета отобразятся следующие сведения:

- **Date**
- **Код отчета о недоставке**
- **Count**
- **Примеры сообщений:** идентификаторы примера соответствующих сообщений.

При выборе **фильтров в представлении** таблицы сведений можно указать диапазон дат с **начальной** и **конечной датами.**

Чтобы отправлять отчету отчету в заданный диапазон дат одному или нескольким получателям, нажмите **кнопку "Запросить загрузку".**

Когда вы выбираете строку в таблице, появится всплывающее окно со следующими сведениями:

- **Date**
- **Код отчета о недоставке. С**помощью ссылки можно получить дополнительные сведения о причинах и решениях конкретных ошибок.
- **Count**
- **Примеры сообщений:** можно нажать **кнопку "Просмотреть примеры** сообщений", [чтобы](message-trace-scc.md) просмотреть результаты трассировки для примера затронутых сообщений.

![Всплывающие элементы подробности после выбора строки в представлении таблицы "Сведения" в отчете о недоставке](../../media/mfi-non-delivery-report-details-flyout.png)
