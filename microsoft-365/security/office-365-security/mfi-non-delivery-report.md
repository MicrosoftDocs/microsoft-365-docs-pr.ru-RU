# <a name="non-delivery-report-in-the-security--compliance-center"></a>Отчет о невывозе в Центре & безопасности

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**Область применения**
- [Exchange Online Protection](exchange-online-protection-overview.md)
- [Microsoft Defender для Office 365 (план 1 и план 2)](defender-for-office-365.md)
- [Microsoft 365 Defender](../defender/microsoft-365-defender.md)

Отчет  о невывозе в панели [](https://protection.office.com) мониторинга потока почты в Центре соответствия требованиям & безопасности показывает наиболее часто встречаемые коды ошибок в отчетах о невывозе (также известных как NDRs или сообщения отказов) для пользователей в вашей организации. [](mail-flow-insights-v2.md) В этом отчете показаны сведения о NDRs, чтобы устранить проблемы с доставкой электронной почты.

![Виджет отчета о невывозе в панели мониторинга потока почты в Центре & безопасности](../../media/mfi-non-delivery-report-widget.png)

## <a name="report-view-for-the-non-delivery-report"></a>Представление отчета для отчета о невывозе

Нажав на виджет **отчета** о невывозе, вы сможете получить отчет **о невывозе.**

По умолчанию отображается действие для всех кодов ошибок. Если вы **нажмете показать данные** для, вы можете выбрать определенный код ошибки из отсев.

Если наведите курсор на определенный цвет (код ошибки) в определенный день на диаграмме, вы увидите общее количество сообщений для ошибки.

![Представление отчета в отчете о не принятых доменах](../../media/mfi-non-delivery-report-overview-view.png)

## <a name="details-table-view-for-the-non-delivery-report"></a>Представление таблицы сведений для отчета о невывозе

Если **щелкнуть таблицу Просмотр** сведений в представлении отчета, показано следующее:

- **Date**
- **Код отчета о невывозе**
- **Count**
- **Пример сообщений:** ID сообщений образца затронутых сообщений.

Если щелкнуть **Фильтры** в представлении таблицы сведений, можно указать диапазон дат с датой начала и  **датой окончания.**

Чтобы отправить отчет для определенного диапазона дат одному или несколько получателей, нажмите кнопку **Запрос скачать**.

При выборе строки в таблице вылет появляется со следующими сведениями:

- **Date**
- **Код отчета о** невывозе. Вы можете нажать на ссылку, чтобы найти дополнительные сведения о причинах и решениях для определенного кода ошибок.
- **Count**
- **Примеры сообщений.** Вы можете нажать **просмотреть примеры** сообщений, чтобы просмотреть результаты трассировки сообщений для примера затронутых сообщений. [](message-trace-scc.md)

![Сведения о вылете после выбора строки в представлении таблицы Details в отчете о невывозе](../../media/mfi-non-delivery-report-details-flyout.png)
