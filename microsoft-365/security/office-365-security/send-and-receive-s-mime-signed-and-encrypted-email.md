# <a name="send-and-receive-smime-signed-and-encrypted-email-in-exchange-online"></a>Отправка и получение электронной почты, подписанной и зашифрованной с помощью S/MIME в Exchange Online

Отправка и ответ на сообщение с зашифрованной с помощью S/MIME в Microsoft Outlook очень похожа на работу с незашифрованным сообщением.
