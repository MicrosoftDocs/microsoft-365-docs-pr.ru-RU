# <a name="mail-flow-map-in-the-security--compliance-center"></a>Карта потока обработки почты в Центре безопасности & безопасности

Карта **потока обработки почты** на [панели](mail-flow-insights-v2.md) мониторинга потока обработки почты в Центре безопасности & дает представление о потоке обработки почты в организации. Эти сведения можно использовать для изучения шаблонов, идентификации аномалий и устранения проблем по мере их возникновения.

![Мини-приложение «Карта потока обработки почты) на информационной панели потока обработки почты в Центре соответствия требованиям & требованиям](../../media/mfi-mail-flow-map-widget.png)

По умолчанию виджет показывает шаблон потока обработки почты за предыдущий день на диаграмме, называемой *диаграммой Sankey.* Для отображения информации от разных дней можно использовать стрелки ![ влево и ](../../media/scc-left-arrow.png) стрелки ![ ](../../media/scc-right-arrow.png) вправо. Каждый цвет представляет поток обработки почты через разные входящие или исходящие соединители (или без использования соединителей). Если навести курсор на определенный цвет, для соединителя этого типа отображается количество сообщений.

## <a name="report-view-for-the-mail-flow-map"></a>Представление отчетов для карты потока обработки почты

Если щелкнуть **мини-приложение "Карта** потока обработки почты", откроется **отчет сопоставления потока обработки** почты.

В представлении отчета доступны следующие диаграммы:

- **Отображение данных для: Обзор**— по существу, это более масштабное представление мини-приложения. Если навести курсор на определенный цвет, для соединителя этого типа отображается количество сообщений.

  ![Обзорное представление в отчете о карте потока обработки почты](../../media/mfi-mail-flow-map-report-overview.png)

- **Показать данные для: Подробности.** В этом представлении отображаются сведения о соединителях и доменах назначения. В списке указаны верхние домены отправителя и получателя, остальные — в **других.** Если наведите указатель мыши на определенный цвет и раздел, отображается число сообщений.

  ![Подробное представление в отчете о карте потока обработки почты](../../media/mfi-mail-flow-map-report-detail.png)

При выборе **фильтров в представлении** отчета можно указать диапазон дат с **начальной** и **конечной датами.**

Чтобы отправлять отчету отчету в заданный диапазон дат одному или нескольким получателям, нажмите **кнопку "Запросить загрузку".**

Содержащаяся аналитика отображается под схемой потока обработки почты, если они доступны (например, [исправить возможный источник данных о почтовом цикле).](mfi-mail-loop-insight.md)

## <a name="details-table-view-for-the-mail-flow-map"></a>Представление таблицы "Сведения" для карты потока обработки почты

При нажатии **кнопки "Просмотр таблицы** сведений" в представлении отчета отобразятся следующие сведения:

- **Date**
- **Категория**
- **Соединитель или сторонний поставщик служб**
- **Домен отправителя или получателя**
- **Число сообщений**

При выборе **фильтров в представлении** таблицы сведений можно указать диапазон дат с **начальной** и **конечной датами.**

При выборе строки похожие сведения будут показаны во всплывающем окне:

![Всплывающем элементе "Сведения" таблицы "Сведения" на карте потока обработки почты](../../media/mfi-mail-flow-map-view-details-table-details.png)

Чтобы отправлять отчету отчету в заданный диапазон дат одному или нескольким получателям, нажмите **кнопку "Запросить загрузку".**

Чтобы вернуться в представление отчетов, нажмите **кнопку "Просмотреть отчет".**
