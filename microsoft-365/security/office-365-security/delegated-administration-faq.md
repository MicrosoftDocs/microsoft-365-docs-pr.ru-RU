# <a name="delegated-administration-faq"></a>Вопросы и ответы по делегированному администрированию

В этом разделе приведены часто задаваемые вопросы и ответы для партнеров и торговых посредников Майкрософт, которые хотят выполнять делегированные задачи администрирования, в ключая возможность управления службой Exchange Online Protection (EOP) для других клиентов (компаний).

## <a name="im-a-reseller-and-i-need-to-manage-my-customers-tenants-how-does-this-work"></a>Я торговый представитель, и мне необходимо управлять клиентами заказчика; как это работает?

Если вы являетесь партнером или торговым посредником Корпорации Майкрософт и подписаны на использование консультанта корпорации Майкрософт, вы можете запросить разрешение на администрирование своего клиента в Центре администрирования. Такое администрирование называется делегированным администрированием, и позволяет управлять их клиентом Microsoft 365 (в том числе параметрами EOP), как если бы вы были администратором в своей организации. Для делегирования администрирования необходимо выполнить следующие действия:

1. Зарегистрируйтесь, чтобы стать [консультантом по Microsoft Office 365](https://aka.ms/cloudbenefits).

2. Зарегистрироваться для делегированного администрирования. Для возможности администрирования учетной записи клиента он должен авторизовать вас в качестве делегированного администратора. Для получения утверждения сначала отправьте заказчику [предложение о делегированном администрировании](https://support.microsoft.com/office/26530dc0-ebba-415b-86b1-b55bc06b073e). (Вы также можете предложить делегированное администрирование заказчику позднее.)

3. Создайте учетную запись делегированного администратора, выполнив действия [в разделе "Добавление, изменение и удаление партнера-консультанта по подпискам".](https://docs.microsoft.com/microsoft-365/admin/misc/add-partner)

Посетите [партнеров. Создание бизнес-подписки и](https://support.microsoft.com/office/30dd1681-47e0-4cbc-abfe-a222cd111319) административного администрирования для получения дополнительных сведений о настройке делегированного администрирования.

## <a name="im-a-customer-not-a-reseller-how-can-set-up-delegated-administrator-for-my-sub-tenants"></a>Как можно настроить полномочного администратора для моих подчиненных клиентов?

В текущий момент делегированное администрирование доступно только для торговых посредников и партнеров. Однако мы предоставляем пример скрипта Windows PowerShell, который поможет вам применить политики к подчиненным клиентам (компаниям). Дополнительные сведения см. в [примере сценария применения параметров EOP к нескольким клиентам.](sample-script-for-applying-eop-settings-to-multiple-tenants.md)

## <a name="can-i-prevent-my-sub-tenant-admin-from-modifying-my-policy"></a>Можно ли запретить администратору подчиненного клиента изменить мою политику?

Microsoft 365 в настоящее время не поддерживает такую возможность.

## <a name="can-i-get-consolidated-reporting-across-all-of-my-sub-tenants"></a>Можно ли получить консолидированный отчет для всех подчиненных клиентов?

В данный момент консолидированные отчеты для всех управляемых вами компаний недоступен в центре администрирования Microsoft 365. Однако это можно сделать с помощью [Microsoft Graph.](https://docs.microsoft.com/graph/overview)
