- <span data-ttu-id="00a24-116">**Изолировать** устройство . - <span data-ttu-id="00a24-118">**Ограничение выполнения** приложения . Чтобы ограничить запуск приложения, применяется политика целостности кода, которая позволяет запускать файлы только в том случае, если они подписаны сертификатом, выданным Корпорацией Майкрософт.</span><span class="sxs-lookup"><span data-stu-id="00a24-118">**Restrict app execution** - To restrict an application from running, a code integrity policy is applied that only allows files to run if they are signed by a Microsoft-issued certificate.</span></span> <span data-ttu-id="00a24-119">Этот метод ограничения может помочь злоумышленнику контролировать скомпрометированную устройства и выполнять дальнейшие вредоносные действия.</span><span class="sxs-lookup"><span data-stu-id="00a24-119">This method of restriction can help prevent an attacker from controlling compromised devices and performing further malicious activities.</span></span>
- <span data-ttu-id="00a24-120">**Запуск антивирусного** сканирования — антивирусная программа в Microsoft Defender может работать вместе с другими антивирусными решениями, независимо от того, является ли антивирус Defender активным антивирусным решением или нет.</span><span class="sxs-lookup"><span data-stu-id="00a24-120">**Run Antivirus scan** - A Microsoft Defender Antivirus scan can run alongside other antivirus solutions, whether Defender Antivirus is the active antivirus solution or not.</span></span> <span data-ttu-id="00a24-121">Если основным решением защиты конечной точки является другой антивирусный продукт, можно запустить антивирус Defender в пассивном режиме.</span><span class="sxs-lookup"><span data-stu-id="00a24-121">If another antivirus vendor product is the primary endpoint protection solution, you can run Defender Antivirus in Passive mode.</span></span>
- <span data-ttu-id="00a24-122">**Инициировать автоматическое** расследование . Вы можете начать новое общее автоматическое расследование на устройстве.</span><span class="sxs-lookup"><span data-stu-id="00a24-122">**Initiate automated investigation** - You can start a new general purpose automated investigation on the device.</span></span> <span data-ttu-id="00a24-123">В ходе расследования любое другое оповещение, сгенерированное с устройства, будет добавлено в текущее автоматическое расследование до завершения этого расследования.</span><span class="sxs-lookup"><span data-stu-id="00a24-123">While an investigation is running, any other alert generated from the device will be added to an ongoing automated investigation until that investigation is completed.</span></span> <span data-ttu-id="00a24-124">Кроме того, если такая же угроза видна на других устройствах, эти устройства добавляются в исследование.</span><span class="sxs-lookup"><span data-stu-id="00a24-124">In addition, if the same threat is seen on other devices, those devices are added to the investigation.</span></span>
- <span data-ttu-id="00a24-125">**Инициировать** живой ответ .
