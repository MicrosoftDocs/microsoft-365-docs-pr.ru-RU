# <a name="set-the-time-zone-for-microsoft-365-defender"></a>Установите часовой пояс для Защитника Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Область применения:**
- Microsoft 365 Defender



Защитник Microsoft 365 может отображать сведения о дате и времени с помощью локального часовой пояса или UTC. Выбранный часовой пояс будет применен ко всем сведениям о дате и времени, которые указаны в следующих функциях Центра безопасности Microsoft 365:
- Инциденты
- Автоматизированное исследование и исправления, включая Центр уведомлений.
- Расширенный поиск

Чтобы установить часовой пояс для этих функций, перейдите в **Параметры** Часовой пояс  >  **Microsoft 365**  >  Defender.
