# <a name="microsoft-365-defender-data-security-and-privacy"></a>Безопасность и конфиденциальность данных в Microsoft 365 Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Область применения:**
- Microsoft 365 Defender

Microsoft 365 Defender работает в центрах обработки данных Microsoft Azure в Европейском союзе, Великобритании и США. Данные о клиентах, собираемые службой, хранятся в режиме покоя в (а) географическом расположении клиента, определяемом во время предварительной обработки, или b) если Microsoft 365 Defender использует другую службу Microsoft Online для обработки таких данных, геолокацию, определяемую правилами хранения данных этой другой онлайн-службы.

Данные клиентов в псевдонимной форме также могут храниться в центральных системах хранения и обработки данных в США.
