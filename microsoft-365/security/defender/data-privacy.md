# <a name="microsoft-365-defender-data-security-and-privacy"></a>Microsoft 365 Безопасность и конфиденциальность данных Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Область применения:**
- Microsoft 365 Defender

Microsoft 365 Defender работает в Microsoft Azure центрах обработки данных в Европейском союзе, Великобритании и США. Данные о клиентах, собираемые службой, хранятся в режиме покоя в (а) географическом расположении клиента, определяемом во время предварительной обработки, или ( b) если Microsoft 365 Defender использует другую онлайн-службу Майкрософт для обработки таких данных, геолокацию, определяемую правилами хранения данных другой службы.

Данные клиентов в псевдонимной форме также могут храниться в центральных системах хранения и обработки данных в США.
