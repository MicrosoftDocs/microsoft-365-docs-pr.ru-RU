# <a name="create-the-microsoft-365-defender-evaluation-environment"></a>Создание среды Microsoft 365 Defender оценки

Существует два распространенных способа сделать следующий шаг в оценке. В этом документе предполагается, что у вас уже есть клиент производства M365, и вы активируете пробные лицензии E5 для оценки M365 Defender в *текущей среде.* Оценка на месте позволит вам сохранить любые методы безопасности при покупке лицензий после периода оценки.

Во-вторых, настройка [среды Microsoft 365 Defender](setup-m365deval.md) лаборатории для целей оценки. Он может не иметь много реальных сигналов от бизнеса, поэтому следует помнить о том, что оговорка.

## <a name="to-activate-e5-trial-licenses-to-evaluate-microsoft-365-defender"></a>Активация пробных лицензий E5 для оценки Microsoft 365 Defender 
1. Войдите на существующий портал администрирования клиента M365.
2. Выберите *Службы покупки* из меню навигации.
3. Прокрутите *вниз до раздела Office 365* и выберите кнопку "Детали" в Office 365 E5 лицензии.

:::image type="content" source="../../media/mdo-eval/2_mdo-eval-license-details.png" alt-text="В Office 365 разделе есть кнопка &quot;Сведения&quot;.":::

4. Выберите *бесплатную пробную ссылку Start.*

:::image type="content" source="../../media/mdo-eval/3-m365-purchase-button.png" alt-text="Нажмите кнопку &quot;Начните бесплатную пробную пробную версия&quot; (за нее взимается плата в размере 35$).":::

5. Подтвердите запрос и *нажмите кнопку "Попробуйте* сейчас".

:::image type="content" source="../../media/mdo-eval/4_mdo-trial-order.png" alt-text="На панели &quot;Проверьте, подтвердите свой заказ&quot; (для пробной Office 365 E5 месяца для 25 пользователей) есть кнопка &quot;Попробуйте сейчас&quot;.":::
