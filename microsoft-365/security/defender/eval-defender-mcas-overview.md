# <a name="evaluate-microsoft-cloud-app-security"></a>Оценка Microsoft Cloud App Security

**Область применения:**
- Microsoft 365 Defender


В этой статье описывается процесс, позволяющий включить и Microsoft Cloud App Security вместе с Microsoft 365 Defender. Перед началом этого процесса убедитесь, что вы [](eval-overview.md) рассмотрели общий процесс оценки Microsoft 365 Defender и создали среду Microsoft 365 Defender [оценки.](eval-create-eval-environment.md)
