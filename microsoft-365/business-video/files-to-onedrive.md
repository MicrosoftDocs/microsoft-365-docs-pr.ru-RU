# <a name="move-files-to-onedrive"></a>Перемещение файлов в OneDrive

## <a name="watch-move-your-files-to-onedrive-for-business"></a>Watch: Перемещение файлов в OneDrive для бизнеса

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

После регистрации на Microsoft 365 бизнес премиум вам&#39;скопировать существующие файлы работы в OneDrive для бизнеса. OneDrive это хорошее место для хранения личных файлов работы или конфиденциальных бизнес-файлов, к которые необходимо контролировать доступ.

## <a name="try-it"></a>Проверьте, как это работает!

1. Из Windows выберите кнопку **Начните.**
2. Поиск OneDrive и выберите его.
3. Введите рабочий адрес электронной почты.
4. Выберите  **Вход,** а затем выберите  **Далее**.
5. Перейдите через краткий учебник, а затем выберите **Откройте OneDrive папку**. Если вы также используете личную версию OneDrive,&#39;увидите ее здесь.
6. В Обозревателе файлов перейдите к файлам, которые необходимо скопировать.
7. Выберите файлы, а затем перетащите их в OneDrive папку. Синие круглые стрелки рядом с файлами означают, что&#39;синхронизируются с облаком. При повторной&#39;синхронизации отображаются зеленые знаки проверки.

Помимо работы с файлами OneDrive локально, вы можете получить доступ к ним из веб-браузера:

1. В веб-браузере вопишитесь в office.com с помощью электронной почты.
2. Выберите **OneDrive**. Все файлы перечислены, и вы можете работать над ними в браузере с любого компьютера.
