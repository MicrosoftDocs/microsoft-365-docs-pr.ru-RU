# <a name="use-microsoft-planner-to-plan-an-event"></a>Планирование событий с помощью Microsoft Planner

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c8a41e1a-c30d-4adb-ab32-6eac922d0ec0?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Используйте Microsoft Planner для планирования события, например отслеживания запуска продукта.

### <a name="work-with-buckets"></a>Работа с ведрами

Используйте ведра для сортировки задач по типу, таких как планирование событий, маркетинг и обеспечение рекламы и PR.

1. Выберите  **Добавить новое ведро**  и введите новую категорию задач.
2. Выберите  **+**  для добавления задач в ведро и введите имя.
3. Выберите  **задачу добавить**.

### <a name="work-with-labels"></a>Работа с метами

Используйте метки, чтобы узнать, какие задачи требуют вашего внимания.

1. Введите имена для меток.
2. Выберите цветное поле для удаления или добавления метки в задачу.

## <a name="add-a-comment"></a>Добавить комментарий

Опубликовать комментарий, чтобы обсудить задачу с людьми, которые ему назначены.

1. Введите сообщение в поле **Комментарии.**
2. Выберите  **Отправить**.

### <a name="keep-track-of-tasks"></a>Отслеживание задач

Используйте  **диаграммы**  и  **представление доски**  для отслеживания прогресса для каждого члена группы и общего проекта, а также для сортировки и организации задач.

- Выберите  **Диаграммы,** **затем сгруппить** по и, наконец, **дату,**  чтобы узнать, какая задача должна быть следующей.
- Выберите  **доску,** **затем Группу по** и, наконец, Прогресс **для**  сортировки задач по ходу выполнения.

**Примечание:**  Задачи организованы  **not started,**  **In Progress** и  **Completed,**  чтобы вы знали, когда вы будете готовы к запуску продукта.
