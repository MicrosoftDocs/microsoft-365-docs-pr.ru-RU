# <a name="use-microsoft-planner-to-plan-an-event"></a>Планирование события с помощью Планировщика (Майкрософт)

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c8a41e1a-c30d-4adb-ab32-6eac922d0ec0?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Используйте Планировщик (Майкрософт) для планирования события, например отслеживания запуска продукта.

### <a name="work-with-buckets"></a>Работа с сегментами

Используйте сегменты для сортировки задач по типам, например планированию событий, маркетингу и рекламе, а также pr.

1. Выберите  **"Добавить новое сегмент"**  и введите новую категорию задач.
2. Выберите  **+**  для добавления задач в сегмент и введите имя.
3. Выберите  **"Добавить задачу"**.

### <a name="work-with-labels"></a>Работа с метами

Используйте метки, чтобы узнать, какие задачи требуют вашего внимания.

1. Введите имена меток.
2. Выберите цветное поле, чтобы удалить или добавить метку в задачу.

## <a name="add-a-comment"></a>Добавить комментарий

Опубликовать комментарий, чтобы обсудить задачу с людьми, для которых она назначена.

1. Введите сообщение в поле **"Комментарии".**
2. Выберите **"Отправить"**

### <a name="keep-track-of-tasks"></a>Отслеживание задач

Используйте  **диаграммы**  **и доски**  для отслеживания хода выполнения для каждого участника группы и всего проекта, а также для сортировки и организации задач.

- Выберите **"Диаграммы",** **"Группу по"** и "Срок", чтобы узнать, какая задача должна быть следующей. 
- Выберите  **"Доска",** **затем сгруппировать по** и, наконец, **"Ход**  выполнения", чтобы отсортировать задачи по ходу выполнения.

**Примечание.**  Задачи организованы по не запущенным,  запущенным и завершенным **задачам,** чтобы вы знали, когда будете готовы к запуску продукта.
