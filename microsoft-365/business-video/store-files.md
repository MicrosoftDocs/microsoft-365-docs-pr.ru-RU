# <a name="where-to-store-files"></a>Места для хранения файлов

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

В Microsoft 365 вы можете хранить свои работы в OneDrive для бизнеса или на сайтах SharePoint. Каждый человек в вашей организации имеет собственную библиотеку OneDrive для бизнеса для хранения личных файлов работы. При создании и экономии файлов в папке OneDrive они синхронизируются с облаком, чтобы вы могли получить к ним доступ из любого места — веб-браузера, компьютера или мобильного устройства. Для общего хранения файлов и совместной работы с командой используйте сайт SharePoint. Сайты SharePoint также синхронизируются с облаком для легкого доступа из любой точки.

## <a name="try-it"></a>Проверьте, как это работает!

### <a name="store-or-share-files-with-onedrive"></a>Хранение или совместное хранение файлов в OneDrive

1. На компьютере запустите проводник и откройте OneDrive. Или из [Office 365](https://www.office.com/)откройте  **OneDrive**  из запуска приложения.
2. Просмотр и отправка личных файлов или отправка документов или папок, щелкнув  **Share,**  а затем предложив другим просмотреть документы или отправить им ссылку. [Подробнее](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business).

### <a name="collaborate-with-a-sharepoint-site"></a>Сотрудничество с сайтом SharePoint

1. Из [Office 365](https://www.office.com/)откройте  **SharePoint**  из запуска приложения, а затем выберите сайт Team.
2. Добавьте файлы в расположение документов по умолчанию или создайте дополнительные расположения документов, называемые Библиотеками.

Вы можете настроить свой сайт SharePoint, создав под-сайты для определенных проектов, групп или клиентов. Вы также можете добавить имя компании, логотип и внешний вид, превратив сайт в бизнес-интрасети. [Подробнее](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
