# <a name="where-to-store-files"></a>Место хранения файлов

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

В Microsoft 365 вы можете хранить свою работу в OneDrive для бизнеса или на сайтах SharePoint. У каждого человека в вашей организации есть собственная библиотека OneDrive для бизнеса для хранения личных файлов работы. Когда вы создаете и сохраняете файлы в папке OneDrive, они синхронизируются с облаком, чтобы вы могли получать к ним доступ из любого веб-браузера, компьютера или мобильного устройства. Для общего хранения файлов и совместной работы группы используйте сайт SharePoint. Сайты SharePoint также синхронизируются с облаком для простого доступа из любого места.

## <a name="try-it"></a>Проверьте, как это работает!

### <a name="store-or-share-files-with-onedrive"></a>Хранение или совместное хранение файлов в OneDrive

1. На компьютере запустите проводник и откройте OneDrive. Или откройте **OneDrive** из службы запуска приложений из [Office 365.](https://www.office.com/)
2. Просматривайте и загружайте личные файлы, делитесь  документами или папками, щелкнув "Поделиться", а затем либо пригласив других, чтобы просмотреть документы, либо отправив им ссылку. [Подробнее](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business).

### <a name="collaborate-with-a-sharepoint-site"></a>Сотрудничество с сайтом SharePoint

1. В [Office 365 откройте](https://www.office.com/)  **SharePoint**  из запуска приложений и выберите сайт группы.
2. Добавление файлов в расположение документов по умолчанию или создание дополнительных расположений документов, называемых библиотеками.

Вы можете настроить сайт SharePoint, создав подузлы для определенных проектов, групп или клиентов. Вы также можете добавить название компании, логотип и внешний вид, превращая сайт в бизнес-интрасеть. [Подробнее](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
