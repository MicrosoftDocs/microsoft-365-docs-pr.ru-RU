# <a name="reset-passwords-for-your-users"></a>Сброс паролей для пользователей

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FVVP?autoplay=false]

В Microsoft 365 для бизнеса можно сбросить пароли для пользователей. Вы также можете позволить сотрудникам сбрасывать собственные пароли. Дополнительные [сведения см.](set-up-self-serve-password-reset.md) в записи "Настройка самостоятельного сброса пароля".

## <a name="try-it"></a>Проверьте, как это работает!

1. Когда пользователь запрашивает новый пароль, вы получите запрос на сброс пароля по электронной почте. Чтобы сбросить пароль, откройте приложение запуска и выберите **"Администратор".**
1. В Центре администрирования Microsoft 365 выберите "Пользователи" **и**"Активные пользователи", а затем выберите значок ключа рядом с пользователем, запросившего сброс.
1. Выберите **автоматически созданный пароль,** чтобы автоматически создать случайный пароль.
1. Выберите **"Сбросить".**
1. Выберите **"Отправить пароль" в поле электронной почты.**
1. Вы увидите свой адрес электронной почты в поле "Электронная почта с новым паролем **для следующих** получателей". Добавьте альтернативный адрес электронной почты пользователя или любой адрес, который вы выберете.
1. Choose **Send email and close**.

    Пользователь получит сообщение электронной почты с инструкциями по сбросу пароля.
