# <a name="reset-passwords-for-your-users"></a>Сброс паролей для пользователей

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FVVP?autoplay=false]

В Microsoft 365 для бизнеса можно сбросить пароли для пользователей. Вы также можете позволить сотрудникам сбросить собственные пароли. Дополнительные [сведения см.](set-up-self-serve-password-reset.md) в сброшении пароля самообслуживки.

## <a name="try-it"></a>Проверьте, как это работает!

1. Когда пользователь запрашивает новый пароль, вы получите запрос на сброс пароля в электронной почте. Чтобы сбросить пароль, откройте пусковую группу приложения и выберите **Администратор**.
1. В центре администрирования Microsoft 365 выберите Пользователи **,** Активные  **пользователи,** а затем выберите значок ключа рядом с пользователем, который запросил сброс.
1. Выберите **автоматически созданный пароль для** автоматического создания случайного пароля.
1. Выберите **Сброс**.
1. Выберите пароль **Отправка в почтовом ящике.**
1. Вы увидите свой адрес электронной почты в электронной почте **новый пароль к следующему окне получателей.** Добавьте альтернативный адрес электронной почты пользователя или любой другой адрес, который вы выберете.
1. Выберите **отправку электронной почты и закрыть**.

    Пользователь получит сообщение электронной почты с инструкциями по сбросу пароля.
