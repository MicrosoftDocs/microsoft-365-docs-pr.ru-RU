# <a name="install-the-microsoft-365-admin-mobile-app-on-your-phone"></a>Установка мобильного приложения Microsoft 365 Admin на телефон

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWe8Kw?autoplay=false]

Мобильное приложение Microsoft 365 Admin можно использовать для управления пользователями, мониторинга состояния службы и выполнения других задач администратора с телефона.

## <a name="try-it"></a>Проверьте, как это работает!

1. Чтобы получить мобильное приложение Microsoft 365 Admin, найди  **администратора Microsoft 365**  в магазине приложений для устройства и установите его.
2. Ворегистрируйся с глобальным адресом электронной почты администратора и просмотрев панель мониторинга, чтобы просмотреть состояние службы, отслеживать лицензии пользователей и просматривать сообщения и запросы на обслуживание.
3. Вот несколько распространенных задач, которые можно выполнять с приложением:

### <a name="reset-a-users-password"></a>сбросить пароль пользователя.

1. В  **статье Быстрые ссылки** выберите  **Пользователи**.
2. На странице  **Пользователи**  выберите имя пользователя и выберите пароль  **Reset**.
3. Если требуется, чтобы пользователь изменил свой новый пароль при первом входе, выберите изменение в первом  **чековом**  окне для регистрации, а затем выберите  **Сброс**.
4. Когда вас просят подтвердить, выберите  **Да,** а затем  **ОК**.
5. В  **окне пароль Share**  можно по желанию отправить новый пароль пользователю в электронной почте или текстовом сообщении.

### <a name="add-a-user-to-a-group"></a>Добавление пользователя в группу

1. В  **статье Быстрые ссылки** выберите  **Группы**.
2. На странице **Группы** выберите группу, в которая нужно добавить пользователя, а затем выберите **изменить** участников, а затем **добавить участника.**
3. Выберите пользователя, который нужно добавить.
4. Когда вас просят подтвердить, выберите  **Да,** а затем  **ОК**.

### <a name="turn-on-notifications"></a>Включаем уведомления

1. В основном меню выберите  **Параметры**.
2. В **настройках Приложения** выберите **уведомления.**
3. Выберите флажки рядом с **центром службы и** **центра сообщений.**
