# <a name="install-office-apps"></a>Установка приложений Office 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/acce002c-0756-4b64-ac5d-2198ee96a9b1?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Microsoft Office приложения можно найти в меню **"Пуск".** Если вы не&#39;их, вы можете установить их самостоятельно.

1. Перейдите в office.com. Возможно, вам потребуется войти с помощью своей учетной записи.
2. Выберите **"Установка приложений Office** Microsoft   >   **365"**   >   **"Выполнить",** а затем выберите **"Да".**
3. Будут установлены приложения Office. Процесс может занять несколько минут. По его завершению выберите **"Закрыть".**
4. Чтобы установить Microsoft Teams, перейдите на страницу office.com и выберите **Teams.**
5. Получите приложение для Windows и выберите **"Выполнить".** Teams отображает запрос после завершения установки.

Установленные приложения Office теперь отображаются в меню **"Пуск".**
