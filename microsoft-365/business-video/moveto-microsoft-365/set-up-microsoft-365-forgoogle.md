# <a name="set-up-microsoft-365-for-google-workspace-migration"></a>Настройка Microsoft 365 для миграции рабочей области Google

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4JD2Y?autoplay=false]

Чтобы перенести электронную почту, календарь и контакты из Рабочей области Google, необходимо настроить Microsoft 365 бизнес премиум с временным доменом, пока бизнес-домен не будет готов к работе с подпиской.

## <a name="try-it"></a>Проверьте, как это работает! 

1. Во входе [office.com](https://office.com)и выберите **администратора**.
1. В центре Microsoft 365 администрирования выберите **Перейти к управляемой настройке.** 
1. Необязательно установите Office приложения, а затем выберите **Продолжить**. 
1. Так как вы мигрируете из рабочей области Google, выберите домен по умолчанию и выберите **Используйте этот домен.** 
1. Введите имена пользователей и желаемые имена пользователей, убедитесь, что ваш адрес электронной почты позволяет получать пароли пользователя, а затем выберите Добавить пользователей и назначить **лицензии.** 
1. Введите адреса электронной почты, где пользователи получат новые учетные данные, выберите **Отправить электронную почту и продолжить.**
1. Затем **снова подключите** временный домен. 
1. Необязательный выбор для информирования пользователей в организации о Teams, а затем выберите **Продолжить**.
1. Расширите и просмотрите параметры Android и **iOS,** а затем выберите **Политику управления мобильными приложениями.**
1. Предоставление обратной связи при желании и выберите **Перейти в центр администрирования.**

Следующим шагом в этом процессе будет настройка защиты для вашей организации.
