# <a name="create-an-org-wide-team"></a>Создание команды для всей организации

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2GCG9?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Создайте группу в Microsoft Teams, в которую может быть вовсю участвовать каждый в вашей компании. При добавлении дополнительных пользователей в Microsoft 365 они автоматически будут добавлены в эту команду.

1. В Microsoft Teams выберите  **Teams,** а затем **создайте команду.**
2. Выберите **Создание команды с нуля.**
3. Выберите **Создать команду для всей организации.**

> [!NOTE]
> Для просмотра этого параметра необходимо быть администратором Microsoft 365.

1. Дайте команде имя, добавьте описание и выберите  **Create**.

В вашей команде находятся каналы. Общий канал создается автоматически. Можно добавить дополнительные каналы, отдельные беседы и файлы для различных отделов, событий или проектов.

Попробуйте любой из следующих ниже.

- Используйте  **вкладку "Сообщения"** для разговоров и объявлений по всей организации, таких как приветствие для всех.
- На  **вкладке Files** можно создать новые документы, в которые каждый может внести свой вклад, например документ Microsoft Word. Или перетащите и сбросите существующие документы на **вкладку Files.**
- Используйте  **вкладку Вики,** чтобы делать быстрые заметки. И выберите знак плюс **+** () для добавления приложений или файлов в качестве дополнительных вкладок.
- На вкладке слева можно просмотреть последние действия, пообщаться с другими участниками группы, просмотреть команды, в которые вы входите, и календарь. Вы можете делать голосовые и видеозвонков другим пользователям группы и просматривать файлы, над которые вы недавно работали.
