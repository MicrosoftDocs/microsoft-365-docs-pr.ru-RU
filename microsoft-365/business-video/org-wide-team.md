# <a name="create-an-org-wide-team"></a>Создание команды для всей организации

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2GCG9?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Создайте команду для всей организации в Microsoft Teams, частью которую может быть любой человек в вашей компании. По мере добавления дополнительных пользователей в Microsoft 365 они будут автоматически добавлены в эту команду.

1. В Microsoft Teams выберите  **Teams** и **создайте команду.**
2. Выберите **команду "Создать команду с нуля".**
3. Выберите **"Создать команду для всей организации".**

> [!NOTE]
> Для просмотра этого параметра необходимо быть администратором Microsoft 365.

1. Придайте команде имя, добавьте описание и выберите команду **"Создать".**

Внутри команды находятся каналы. Канал "Общие" создается автоматически. Вы можете добавлять дополнительные каналы, отдельные беседы и файлы для разных отделов, событий или проектов.

Попробуйте любое из следующих ок.

- Вкладка  **"Записи"** используется для бесед и объявлений на всей организации, например приветствия для всех пользователей.
- На  **вкладке "Файлы"** можно создать новые документы, в которые может внести свой вклад каждый, например документ Microsoft Word. Или перетащите существующие документы на вкладку **"Файлы".**
- Используйте  **вкладку "Вики-сайт",** чтобы делать быстрые заметки. И выберите знак "плюс" **+** () для добавления приложений или файлов в качестве дополнительных вкладок.
- С помощью вкладок слева вы можете просматривать недавние действия, общаться в чате с другими участниками команды, просматривать команды, в которые вы входите, и свой календарь. Вы можете делать голосовые и видеозвонков другим пользователям группы и просматривать файлы, над которые вы недавно работали.
