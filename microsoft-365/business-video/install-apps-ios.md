# <a name="install-outlook-and-office-apps-on-ios-devices"></a>Установка приложений Outlook и Office на устройствах с iOS

После установки&#39;Microsoft 365 Бизнес Премиум можно установить приложения Office на iPhone. После установки приложений вам&#39;создать ПИН-код для защиты данных Office.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWee2n?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

1. На iPhone откройте магазин Приложений.
2. Выберите  **поиск,** введите  **Outlook** в панели поиска и выберите его.
3. Выберите **установите**   >   **.**
4. Введите пароль Apple ID, выберите **вход в**  >   **Open.**
5. Введите рабочий адрес электронной почты.
6. Выберите **добавить учетную** запись, введите пароль и введите **вход.**
7. (Необязательный), Выберите **Let's Do It ** для добавления дополнительных учетных записей или выберите  **Maybe Later,**  чтобы пропустить этот шаг.
8. Выберите  **ОК** для перезапуска приложения, чтобы Microsoft 365 Business защитила данные Outlook.
9. Откройте Outlook снова, и появится сообщение подтверждения. Выберите  **ОК**.
10. Создайте PIN-код для Outlook и других приложений Office и подтвердим его. С этого времени этот ПИН-код будет использоваться для защиты всех данных приложений Office.
11. Выберите  **Включить,**  чтобы включить уведомления электронной почты.
12. Выберите  **Разрешить** включить push-уведомления.

Теперь вы можете использовать Outlook на iPhone для проверки электронной почты, календаря и контактов.

Остальные приложения Office можно установить таким же образом, и данные в этих приложениях будут защищены Microsoft 365 Business Premium.
