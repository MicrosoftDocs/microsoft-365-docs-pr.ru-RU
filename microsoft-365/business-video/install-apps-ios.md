# <a name="install-outlook-and-office-apps-on-ios-devices"></a>Установка Outlook и Office приложений на устройствах iOS

После&#39;настроив Microsoft 365 бизнес премиум, вы можете Office приложения на iPhone. После установки приложений вам&#39;создать ПИН-код для защиты Office данных.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWee2n?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

1. На iPhone откройте Магазин Приложений.
2. Выберите **поиск,** **Outlook** в панели поиска, а затем выберите его.
3. Выберите **установите**   >   **.**
4. Введите пароль Apple ID, выберите **вход в**  >   **Open.**
5. Введите рабочий адрес электронной почты.
6. Выберите **добавить учетную** запись, введите пароль и введите **вход.**
7. (Необязательный), Выберите **Let's Do It ** для добавления дополнительных учетных записей или выберите  **Maybe Later,**  чтобы пропустить этот шаг.
8. Выберите **ОК,** чтобы перезапустить приложение, Microsoft 365 бизнес защитить Outlook данных.
9. Откройте Outlook, и появится сообщение подтверждения. Выберите  **ОК**.
10. Создайте ПИН-код для Outlook и Office приложений и подтвердим его. С этого времени этот ПИН-код будет использоваться для защиты всех Приложение Office данных.
11. Выберите  **Включить,**  чтобы включить уведомления электронной почты.
12. Выберите  **Разрешить** включить push-уведомления.

Теперь вы можете использовать Outlook на iPhone для проверки электронной почты, календаря и контактов.

Вы можете установить остальные Office таким же образом, и данные в этих приложениях будут защищены Microsoft 365 бизнес премиум.
