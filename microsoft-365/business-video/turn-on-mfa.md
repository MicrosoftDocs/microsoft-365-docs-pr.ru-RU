# <a name="turn-on-multi-factor-authentication"></a>Включаем многофакторную проверку подлинности

## <a name="watch-turn-on-multi-factor-authentication"></a>Часы: Включив многофакторную проверку подлинности

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MuO3?autoplay=false]

Многофакторная проверка подлинности означает, что вы и ваши сотрудники должны предоставить несколько способов войти в Microsoft 365 это один из самых простых способов обеспечения безопасности вашего бизнеса.

## <a name="try-it"></a>Проверьте, как это работает!

1. Перейдите в Центр администрирования [https://admin.microsoft.com](https://admin.microsoft.com).
1. Выберите **Показать все,** а затем **Azure Active Directory Центр администрирования**.
1. Выберите **Azure Active Directory**, **свойства**, **управление безопасностью по умолчанию**.
1. В **статье Включить по умолчанию безопасность** выберите **Да,** а затем **сохраните**.

> [!NOTE]
> Для новых подписок автоматически включаем безопасные по умолчанию.

При следующем входе выбранных сотрудников им будет предложено настроить приложение Microsoft Authenticator на своих телефонах для второй формы проверки подлинности.
