# <a name="turn-on-multi-factor-authentication"></a>Включить многофакторную проверку подлинности

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MuO3?autoplay=false]

Многофакторная проверка подлинности означает, что вы и ваши сотрудники должны предоставить несколько способов войти в Microsoft 365. Это один из самых простых способов защиты бизнеса.

## <a name="try-it"></a>Проверьте, как это работает!

1. Перейдите в Центр администрирования [https://admin.microsoft.com](https://admin.microsoft.com).
1. Выберите **"Показать все"** и выберите **Центр администрирования Azure Active Directory.**
1. Выберите **Azure Active Directory**, **свойства**, **Управление безопасностью по умолчанию**.
1. В **области "Включить значения безопасности по умолчанию"** выберите **"Да",** а затем **"Сохранить".**

> [!NOTE]
> Для новых подписок безопасные значения по умолчанию будут автоматически включены.

При следующем входе выбранных сотрудников им будет предложено настроить приложение Microsoft Authenticator на телефонах для второй проверки подлинности.
