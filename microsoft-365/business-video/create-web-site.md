# <a name="create-a-website"></a>Создание веб-сайта

> [!VIDEO https://www.microsoft.com/videoplayer/embed/4839abc6-9323-4cbf-a79d-2907235f9ebb?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Microsoft 365 не включает общедоступный веб-сайт для вашей компании. Чтобы настроить его, рассмотрите возможность использования компании веб-хостинга, например GoDaddy, Wix или WordPress.

1. Найщите поставщика услуг размещения веб-сайтов, где можно создать свой веб-сайт.

     Вам может оказаться проще всего выбрать построитель веб-сайтов в той же компании, где зарегистрировано ваше доменное имя.
1. Разработать свой бизнес-веб-сайт, включив логотип компании, бизнес-цель, текст, изображения и ссылки.
1. Чтобы интегрировать веб-сайт с Microsoft 365, вы можете добавить на веб-сайт свою электронную почту и ссылку на Microsoft Teams, SharePoint или другое средство совместной работы.
1. Обязательно подключите свой веб-сайт к доменным именам, которые&#39;также использовать для электронной почты в Microsoft 365.
