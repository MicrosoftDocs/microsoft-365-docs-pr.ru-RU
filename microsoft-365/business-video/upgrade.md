# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Обновление с Windows 10 Домашняя до Windows 10 Pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Microsoft 365 бизнесу требуется Windows 10 Pro, чтобы обеспечить безопасность данных. Если компьютер работает на Windows 10 Домашняя, выполните следующие действия, чтобы обновить Windows 10 Pro.

1. На рабочем столе выберите меню **Пуск,** **Microsoft Store** и выберите его.
2. В Microsoft Store нажмите в панели Поиска, **введите Windows 10 Pro,** а затем выберите **обновление Windows 10 Pro**.
3. В **статье Обновление Windows 10 Pro** выберите **Дополнительные дополнительные** и выберите **Купить**.
4. В поле Безопасность Windows введите **ПИН-код,** чтобы проверить свою личность.
5. Просмотрите страницу оплаты, ознакомьтесь с условиями продажи, внести изменения в способ оплаты, если хотите, а затем выберите  **Купить**.
6. В **статье Обновление Windows 10 Pro** выберите **Установите**.
7. Сохраните все файлы, если это необходимо, а затем выберите **Да,&#39;перейти** к началу установки Windows.
