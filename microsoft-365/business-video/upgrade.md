# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Обновление с Windows 10 Home до Windows 10 Pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Для обеспечения безопасности данных для бизнеса Microsoft 365 требуется Windows 10 Pro. Если у вас есть компьютер с Windows 10 Home, выполните следующие действия, чтобы перейти на Windows 10 Pro.

1. На рабочем столе выберите меню  **"Пуск",**  выберите  **Microsoft Store** и выберите его.
2. В Microsoft Store щелкните в панели поиска введите **Windows 10 Pro,** а затем выберите обновление до **Windows 10 Pro.**
3. В  **статье Обновление до Windows 10 Pro** выберите  **Дополнительные** дополнительные и выберите  **Купить**.
4. В поле  **Безопасности Windows**  введите ПИН-код, чтобы проверить свою личность.
5. Просмотрите страницу оплаты, ознакомьтесь с условиями продажи, внести изменения в способ оплаты, если хотите, а затем выберите  **Купить**.
6. В  **статье Обновление до Windows 10 Pro** выберите  **Установите**.
7. Сохраните все файлы, если это необходимо, а затем выберите Да,&#39; **перейти**  к началу установки Windows.
