# <a name="set-up-self-service-password-reset"></a>Настройка самостоятельного сброса паролей

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3AY8S?autoplay=false]

Чтобы позволить пользователям сбрасывать свои пароли, не связываясь с вами каждый раз, можно настроить самостоятельный сброс паролей в Microsoft 365 бизнес премиум.

## <a name="try-it"></a>Проверьте, как это работает!

1. В Центре администрирования Microsoft 365 в области  навигации слева выберите параметры, а затем параметры **безопасности & конфиденциальности.**
1. Under **Let your people reset their own passwords,** select Azure **AD admin center**.
1. В левой области навигации выберите "Пользователи", а затем на странице "Пользователи — все **пользователи"** выберите **"Сброс пароля".**
1. Выберите **"Все",** чтобы включить самостоятельный сброс пароля, а затем выберите **"Сохранить".**

Когда пользователь в следующий раз войдет в свою учетную запись, у пользователя будет запросят его ИД и пароль. Они выбирают **"Далее"** и затем выбирают, следует ли проверить подлинность с помощью телефона, электронной почты или обоих вариантов. Они введите получаемый код, выберите **"Проверить"** и **"Готово".** После этого они могут сбросить свой пароль.
