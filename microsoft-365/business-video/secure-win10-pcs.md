# <a name="secure-windows-10-pcs"></a>Защита компьютеров с Windows 10

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4kcAF?autoplay=false]
 
## <a name="try-it"></a>Проверьте, как это работает!  

После того как вы настроите защиту устройств в Microsoft 365 бизнес, выполните следующие действия для защиты компьютеров с Windows 10.

1. В Центре администрирования Microsoft 365 выберите **"Программа установки".**
2. В **области "Защита компьютеров с Windows"** выберите **"Просмотреть".**
3. Просмотрите сведения на странице "Защита компьютеров  **с Windows 10",**  чтобы понять политики и их влияние на пользователей.
4. Выберите **"Начало работы".**
5. В области  **параметров Windows 10**  просмотрите параметры и при необходимости настройте их. Наведите на нее курсор, чтобы узнать больше о политике.
6. К  **кому должна применяться политика?** , выберите, будут ли ваши выборы применяться к всем в вашей организации или к определенным группам безопасности.
7. Выберите **"Сохранить изменения".**

Microsoft 365 бизнес теперь помогает защитить компьютеры с Windows 10.
