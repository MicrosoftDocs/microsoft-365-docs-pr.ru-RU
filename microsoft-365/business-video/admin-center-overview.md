# <a name="overview-of-the-microsoft-365-admin-center"></a>Обзор Центра администрирования Microsoft 365

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfvDL?autoplay=false]

Центр администрирования Microsoft 365 — это место, где вы управляете бизнесом в облаке. Вы можете выполнять такие задачи, как добавление и удаление пользователей, изменение лицензий и сброс паролей. 

Специализированные области работы, такие как управление безопасностью или устройством, позволяют более детально управлять. Дополнительные сведения о совместной работе центров администрирования см. в этой статье О конкретных типах ролей ИТ и других областях работы, таких как [Security, Device Management или Exchange?](#what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange)

Чтобы добраться до центра администрирования Microsoft [](https://admin.microsoft.com) 365, перейдите в admin.microsoft.com или, если вы уже вписались, выберите пусковую группу приложения и выберите **администратора**.

На домашней странице можно создавать карты для часто выполняемых задач. Чтобы добавить новую карту, выберите **добавить** карточку, а затем выберите знак плюс рядом с карточкой, вы хотите добавить. По завершению закрой окно. Вы можете переставить карты, выбрав и перетащив их туда, где вы хотите. Чтобы удалить карточку, выберите **Дополнительные действия,** а затем выберите **Удалить**.

Чтобы просмотреть дополнительные задачи администратора, разорите меню навигации. Дополнительные параметры конфигурации можно найти в дополнительных центрах администрирования в нижней части.

Одной из распространенных задач, которую можно выполнить в центре администрирования, является добавление пользователя. Для этого выберите **Пользователи**, **Активные пользователи,** а затем **выберите Добавить пользователя**. Введите имя пользователя и другие сведения, а затем выберите **Далее**. Следуйте подсказкам, чтобы завершить добавление пользователя. Когда вы закончите, выберите **добавление Готово,** а затем выберите **Закрыть**.

Вы можете сортировать активных пользователей по столбцам, таким как **Display name** или **Licenses.** Чтобы добавить больше столбцов, **выберите Выберите столбцы,** выберите столбцы, которые необходимо добавить, а затем выберите **Сохранить**.

Выберите пользователя, чтобы увидеть дополнительные параметры, например управление лицензиями на продукт.

Чтобы включить дополнительные функции, которые приходят с подпиской, выберите **установку**. Здесь можно включить безопасность входов, защиту мобильных приложений, DLP и другие функции, включенные в подписку.

Если вам нужна поддержка в любое время, выберите **need help**. Введите вопрос, а затем проверьте ссылки, которые отображаются. Если вы не получите здесь ответ, выберите службу **поддержки контактов,** чтобы открыть запрос на обслуживание. 

Дополнительные сведения об управлении биллингом, паролями, пользователями и администраторами см. в других уроках этого курса.

## <a name="who-is-an-admin"></a>Кто является администратором? 

По умолчанию человек, подписывавший и приобретавший Microsoft 365 для бизнес-подписки, получает разрешения администратора. Этот человек может назначить разрешения администратора другим людям, чтобы помочь им управлять Microsoft 365 для своей организации.

Если вы получаете сообщение **"У вас** нет разрешения на доступ к этой странице или выполнить это действие", вы не администратор.

### <a name="who-has-admin-permissions-in-my-business"></a>У кого есть разрешения администратора в моей организации?
<a name="bkmk_admin"> </a>

При поиске администратора для сброса пароля, удаления учетной записи или выполнения других задач вам необходимо обратиться к следующим лицам.
  
- **Университеты и школы:** свяжитесь с командой технической поддержки. Обычно ссылку можно найти на сайте университета. В небольших школах может быть всего несколько лиц, у которых есть разрешения администратора. 
    
- **Крупные организации.** Обратитесь во внутреннюю службу технической поддержки. 
    
- **Небольшие организации.** Обратитесь к владельцу или совладельцу организации. Зачастую они предоставляют разрешения администратора ИТ-консультанту, который полностью обслуживает компьютерный парк организации. 
  
Если вы не знаете, к кому обратиться за помощью в вашей работе или школе, попробуйте спросить человека, который дал вам учетную запись пользователя и пароль.

> [!NOTE]
> Администраторы целевых выпусков имеют первый доступ к новым функциям. Новые функции позже выкатываются всем администраторам. Это означает, что центр администрирования может не выглядеть иначе, чем описано в статьях справки. Чтобы быть одним из первых, кто увидит новые функции, см. ниже статью Участие в центре администрирования. 

## <a name="turn-on-targeted-release"></a>Включите целевой выпуск

1. Войдите [в admin.microsoft.com,](https://admin.microsoft.com)перейдите на области навигации и выберите **профиль Организации** \> **параметров**.

2. Перейдите к карте **"Параметры выпуска",** а затем выберите **Изменить**. 
    
3. Выберите **целевой выпуск для всех или** целевой выпуск для **выбранных пользователей.** Если вы выбираете целевой выпуск для выбранных пользователей, убедитесь, что вы добавите свою учетную запись администратора (и любых других администраторов в вашей организации, которые хотят участвовать) в список выбранных пользователей.

### <a name="what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange"></a>Как насчет определенных типов ролей ИТ и других пространств работы, таких как Security, Device Management или Exchange?

Центр администрирования Microsoft 365 является общей точкой входа для всех групп и ролей, управляющих Microsoft 365. Интерфейсы, сведения и элементы управления настраиваются и настраиваются для каждого администратора и роли. Кроме того, в специализированных рабочего пространствах обеспечивается глубокий и детализированный контроль. Эти специализированные области работы включают SharePoint, Teams &amp; Skype, Exchange, Security, Compliance, Device Management и Azure Active Directory. В центре администрирования Microsoft 365 в центре администрирования Microsoft 365 можно найти специализированные области [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) работы.
