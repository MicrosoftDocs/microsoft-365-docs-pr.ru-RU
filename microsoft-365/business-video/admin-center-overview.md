# <a name="overview-of-the-microsoft-365-admin-center"></a>Обзор центра администрирования Microsoft 365

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfvDL?autoplay=false]

Центр администрирования Microsoft 365 — это место, где вы управляете бизнесом в облаке. Вы можете выполнять такие задачи, как добавление и удаление пользователей, изменение лицензий и сброс паролей. 

Чтобы добраться до центра администрирования Microsoft [](https://admin.microsoft.com) 365, перейдите в admin.microsoft.com или, если вы уже вписались, выберите пусковую группу приложения и выберите **администратора**.

На домашней странице можно создавать карты для часто выполняемых задач. Чтобы добавить новую карту, выберите **добавить** карточку, а затем выберите знак плюс рядом с карточкой, вы хотите добавить. По завершению закрой окно. Вы можете переставить карты, выбрав и перетащив их туда, где вы хотите. Чтобы удалить карточку, выберите **Дополнительные действия,** а затем выберите **Удалить**.

Чтобы просмотреть дополнительные задачи администратора, разорите меню навигации. Дополнительные параметры конфигурации можно найти в дополнительных центрах администрирования в нижней части.

Одной из распространенных задач, которую можно выполнить в центре администрирования, является добавление пользователя. Для этого выберите **Пользователи**, **Активные пользователи,** а затем **выберите Добавить пользователя**. Введите имя пользователя и другие сведения, а затем выберите **Далее**. Следуйте подсказкам, чтобы завершить добавление пользователя. Когда вы закончите, выберите **добавление Готово,** а затем выберите **Закрыть**.

Вы можете сортировать активных пользователей по столбцам, таким как **Display name** или **Licenses.** Чтобы добавить больше столбцов, **выберите Выберите столбцы,** выберите столбцы, которые необходимо добавить, а затем выберите **Сохранить**.

Выберите пользователя, чтобы увидеть дополнительные параметры, например управление лицензиями на продукт.

Чтобы включить дополнительные функции, которые приходят с подпиской, выберите **установку**. Здесь можно включить безопасность входов, защиту мобильных приложений, DLP и другие функции, включенные в подписку.

Если вам нужна поддержка в любое время, выберите **need help**. Введите вопрос, а затем проверьте ссылки, которые отображаются. Если вы не получите здесь ответ, выберите службу **поддержки контактов,** чтобы открыть запрос на обслуживание. 

Дополнительные сведения об управлении биллингом, паролями, пользователями и администраторами см. в других уроках этого курса.
