# <a name="find-docs-and-training"></a>Поиск документы и обучение

Если у вас есть вопрос или вам нужна помощь Microsoft 365 бизнес премиум, у вас есть несколько вариантов получения помощи.

## <a name="browse-all-small-business-training"></a>Просмотр всех учебных занятий по малому бизнесу

Ссылки на все службы подготовки, которые необходимо использовать Microsoft 365 для малого бизнеса, см. в Microsoft 365 подготовки по [малому бизнесу.](index.yml) 

## <a name="search-for-microsoft-365-admin-center-docs-and-resources"></a>Поиск Microsoft 365 и ресурсов центра администрирования

Чтобы найти задачи, связанные с администрированием, и помочь, например, сбросить пароли и назначить подписки Майкрософт, см. Microsoft 365 [справку центра администрирования.](../admin/index.yml)

## <a name="join-the-community"></a>Присоединяйтесь к сообществу

Начните беседу, следуйте обсуждениям и поделитесь своими знаниями с другими пользователями Microsoft 365 бизнеса, присоединившись к [microsoft Small Business Community](https://smallbusiness.microsoft.com/).

Если у вас есть конкретные вопросы о других продуктах и службах, размести вопросы и следуйте соответствующим обсуждениям в [Ask the Microsoft Community](https://answers.microsoft.com).
