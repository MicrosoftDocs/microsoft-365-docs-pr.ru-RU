# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>Настройка Outlook для Microsoft 365 для бизнес-почты 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

После установки приложений Office вам&#39;настроить Outlook для начала использования электронной почты, календаря и контактов. Здесь&#39;как.

1. Откройте меню Пуск. Ищите Outlook и выберите его.
2. Введите свой адрес электронной почты Microsoft 365 и выберите **Подключение.**
3. Введите дополнительные адреса электронной почты, которые вы хотите использовать, например предыдущий или личный адрес электронной почты. Выберите  **Далее**.
4. Если запросили, введите пароль, а затем выберите  **вход**.
5. После того как все учетные записи были добавлены, выберите, хотите ли вы настроить Outlook mobile или подождать до более позднего времени.
6. Выберите  **Готово**. Для загрузки электронной почты и других данных Outlook может потребоваться несколько минут.

Теперь вы можете просматривать электронную почту для добавленных учетных записей. Вы также можете просматривать календарь, контакты и задачи.
