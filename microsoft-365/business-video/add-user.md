# <a name="add-a-new-user"></a>Добавление нового пользователя

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOfN?autoplay=false]

Когда новый пользователь присоединяется к вашей компании, вам необходимо добавить его в подписку Microsoft 365 бизнес премиум в качестве нового пользователя.

## <a name="try-it"></a>Проверьте, как это работает!

1. В Центре администрирования Microsoft 365 выберите "Управление пользователями" и выберите "Добавить пользователя". 
1. Введите имя и **фамилию** нового **пользователя.** **Отображаемая имя** заполняется автоматически, но вы можете изменить его.
1. Введите **имя пользователя.** Это будет адрес электронной почты нового пользователя. Если у вас несколько доменов, выберите правильный домен в выпадаемом списке.
1. Чтобы пароль создавался автоматически, выберите автоматически **созданный пароль.**
1. Чтобы потребовать от пользователя изменить пароль, выберите "Требовать от этого пользователя изменить пароль при первом входе **в** учетную запись".
1. Если вы хотите, чтобы сведения об учетной записи отправляли по электронной почте, по завершении выберите пароль для **отправки** по электронной почте. Имя электронной почты будет добавлено автоматически, но вы можете добавить альтернативный адрес электронной почты нового пользователя, чтобы он также получил сведения об учетной записи. Нажмите кнопку **Далее**.
1. Выберите свое расположение в выпадаемом списке, а затем — рядом с подпиской, которая будет нужна пользователю. Выберите **Nex** t.
1. Если вы хотите, чтобы пользователь был администратором, раз expand **Role** s, clear the **User (no administrator access)** check box и select an admin role from the list. Дополнительные сведения о ролях администраторов и их настройках см. в [подстройки "Что такое администратор"?](what-is-admin.md)
1. В **области сведений** профиля введите сведения о должности, отделе, офисе и так далее. (Вы или новый пользователь всегда можете добавить эти сведения позже.) Выберите **"Далее".**
1. Просмотрите сведения о пользователе. Если вам нужно приобрести дополнительную лицензию для этого пользователя, вы увидите уведомление на странице проверки. Select **Finish adding**, and then select **Close**.
1. Убедитесь, что пользователь успешно создан: в Центре администрирования Microsoft 365 в области навигации слева выберите "Пользователи"**и** **"Активные пользователи".** Новый пользователь должен быть указан вместе с назначенной им лицензией.
