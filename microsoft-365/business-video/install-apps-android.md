# <a name="install-outlook-and-office-apps-on-android-devices"></a>Установка приложений Outlook и Office на устройствах с Android

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfquk?autoplay=false]

Если ваше android-устройство управляется Microsoft 365 Business Premium, вам может быть предложено настроить дополнительную безопасность при первой установке приложения Office. 

## <a name="try-it"></a>Проверьте, как это работает!

1. На устройстве выберите магазин Google Play.
2. Поиск приложения Office, которое необходимо установить. В&#39;мы будем использовать Outlook.
3. На экране приветствия Outlook выберите **Начало работы.**
4. На странице  **Добавить учетную запись**  введите адрес электронной почты Microsoft 365, а затем выберите  **Продолжить**.
5. Введите **пароль,** введите пароль, а затем выберите **вход.**
6. На следующей странице выберите  **Продолжить,**  если вы хотите добавить другую учетную запись. В противном случае выберите  **Skip**.
7. Если вам будет предложено добавить приложение Портал компании Intune, выберите **Перейти в магазин.**
8. Установите и откройте приложение портала Компании Intune, а затем выберите  **ОК,**  когда увидите сообщение о том, что ваша организация защищает данные в этом приложении.
9. Если вам будет предложен ПИН-код, введите 4-значный ПИН-код,  который&#39;для разблокировки телефона, выберите Отправить, подтвердите ПИН-код, а затем выберите отправить **еще** раз.
