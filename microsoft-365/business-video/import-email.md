# <a name="import-email-to-microsoft-365-for-business"></a>Импорт электронной почты в Microsoft 365 для бизнеса 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/40f7df36-9e24-44e5-8791-e9ed0dd8fd21?autoplay=false]

После регистрации в Microsoft 365 у вас может по-прежнему быть электронная почта, адресовав старую учетную запись; например, если вы использовали учетную запись Gmail или Outlook для работы. Вы можете продолжать использовать как новые, так и старые учетные записи, или импортировать данные из старой учетной записи в новую.

## <a name="try-it"></a>Проверьте, как это работает!

### <a name="step-1-export-your-old-email"></a>Шаг 1. Экспорт старых сообщений электронной почты.

1. В Outlook выберите **"Файл",** **"Открыть и** экспортировать", а затем **"Импорт/экспорт".**
2. Выберите **"Экспорт в файл"** и **"Далее".**
3. Выберите **файл данных Outlook,** а затем выберите **"Далее".**
4. Выберите старую учетную запись электронной почты.
5. Убедитесь, что вы добавили вложенные папки. Затем выберите **"Далее".**
6. Выберите  **"Обзор"** и перейдите к легко запоминаемой области, например к рабочему столу. Выберите **"ОК",** а затем **завершите.**
7. Оставьте поля пароля пустыми и выберите **"ОК".**

    Старая учетная запись электронной почты экспортируется.

### <a name="step-2-import-your-old-email-into-your-new-email-account"></a>Шаг 2. Импорт старых сообщений электронной почты в новую учетную запись электронной почты

1. В Outlook выберите **"Файл",** **"Открыть",** **"Экспорт"** и **"Импорт и экспорт".**
2. Выберите **"Импорт" из другой программы или файла,** а затем выберите **"Далее".**
3. Выберите **файл данных Outlook,** а затем выберите **"Далее".**
4. Выберите  **"Обзор"** и перейдите к только что экспортируемму файлу. Выберите его, а затем выберите **"Открыть"** и **"Далее".**
5. Убедитесь, что вы добавили вложенные папки и импортируете элементы в ту же папку в новой учетной записи электронной почты. Нажмите кнопку **Готово**.
