# <a name="import-email-to-microsoft-365-for-business"></a>Импорт электронной почты в Microsoft 365 для бизнеса 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/40f7df36-9e24-44e5-8791-e9ed0dd8fd21?autoplay=false]

После регистрации на Microsoft 365 у вас по-прежнему может возникнуть возможность отправки электронной почты на старую учетную запись; например, если вы использовали учетную запись Gmail или Outlook для работы. Вы можете продолжать использовать новые и старые учетные записи или импортировать данные из старой учетной записи в новую.

## <a name="try-it"></a>Проверьте, как это работает!

### <a name="step-1-export-your-old-email"></a>Шаг 1. Экспорт старой электронной почты.

1. В Outlook выберите **Файл, Открыть** **и Экспорт,** а затем **импортировать и экспортировать.**
2. Выберите **Экспорт в файл,** а затем выберите **Далее**.
3. Выберите **файл данных Outlook** и выберите **Далее**.
4. Выберите старую учетную запись электронной почты.
5. Убедитесь, что вы включаем подмостки. Нажмите кнопку **Далее**.
6. Выберите **Обзор** и перейдите в удобное для запоминать расположение, например на рабочем столе. Выберите **ОК,** а затем **закончить**.
7. Оставьте поля паролей пустыми, а затем выберите **ОК.**

 Ваша старая учетная запись электронной почты экспортируется.

### <a name="step-2-import-your-old-email-into-your-new-email-account"></a>Шаг 2. Импорт старой электронной почты в новую учетную запись электронной почты

1. В Outlook выберите **Файл,** **Открыть,** **Экспорт,** а затем **импортировать и экспортировать**.
2. Выберите **Импорт из другой программы или файла,** а затем выберите **Далее**.
3. Выберите **файл данных Outlook** и выберите **Далее**.
4. Выберите **Просмотр** и перейдите к экспортируемом файлу. Выберите его, а затем **выберите Открыть,** а затем **далее**.
5. Убедитесь, что вы включите подмостки и импортируете элементы в ту же папку в новой учетной записи электронной почты. Нажмите кнопку **Готово**.
