# <a name="sign-up-for-microsoft-365-business-premium-subscription"></a>Подписка на Microsoft 365 бизнес премиум

## <a name="watch-sign-up-for-microsoft-365-business-premium"></a>Часы. Зарегистрився для Microsoft 365 бизнес премиум

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3znhX?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Выполните следующие действия, чтобы зарегистрироваться Microsoft 365 бизнес премиум (ранее называлась Microsoft 365 бизнес):

1. Чтобы зарегистрироваться для Microsoft 365 бизнес премиум, выберите его в Интернете.
2. Выберите  **для бизнеса,** а затем  **купить сейчас**.
3. Введите личный адрес электронной почты, выберите **Далее,** а затем **настройка учетной записи.**
4. Введите свое имя, номер телефона и имя компании. Если&#39;хотите получить помощь от партнера Майкрософт, проверьте поле рядом с моими **сведениями.** Выберите  **Далее**.
5. Чтобы проверить номер телефона, выберите **код проверки отправки.** Введите код, полученный на телефоне, и выберите  **Verify**.
6. Если у вас уже есть домен, выберите  **Get a Microsoft domain for now** и введите имя вашей компании. Это становится временным адресом электронной почты. Он также становится частью постоянного SharePoint OneDrive URL-адресов. Выберите  **проверить доступность,** а затем  **далее**.
7. Введите имя пользователя, пароль, подтвердите пароль, а затем выберите  **Запись**  для создания учетной записи.
8. Введите число пользователей, выберите план вы выставления счета и выберите **Далее.**
9.  Введите адрес&#39;, а затем  **далее**.
10. Введите сведения о кредитной карте, а затем **раздайте заказ.** Через несколько минут будет создана учетная запись.
11. На этом этапе можно выбрать **Продолжить** настройку учетной записи или войти в центр администрирования Microsoft 365 и завершить настройку позже.
