# <a name="sign-up-for-microsoft-365-business-premium-subscription"></a>Подписка на Microsoft 365 бизнес премиум

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3znhX?autoplay=false]

## <a name="try-it"></a>Проверьте, как это работает!

Выполните следующие действия, чтобы зарегистрироваться в Microsoft 365 бизнес премиум (ранее называлась Microsoft 365 бизнес):

1. Чтобы зарегистрироваться в Microsoft 365 бизнес премиум, найщите его в Интернете и выберите его.
2. Выберите **"Для бизнеса"** и **"Купить сейчас".**
3. Введите свой личный адрес электронной почты, выберите **"Далее"** и затем **"Настройка учетной записи".**
4. Введите имя, номер телефона и название компании. Если вы&#39;хотите получить помощь от партнера Корпорации Майкрософт, проверьте этот поле рядом с тем, где я хочу, чтобы корпорация Майкрософт поделились **своими сведениями.** Выберите **"Далее".**
5. Чтобы проверить номер телефона, выберите код **проверки отправки.** Введите код, полученный на телефоне, и выберите **"Проверить".**
6. Если у вас уже есть домен, выберите "Получить  **домен Майкрософт"** и введите название своей компании. Это временный адрес электронной почты. Он также становится частью постоянных URL-адресов SharePoint и OneDrive. Выберите **"Проверить доступность"** и "Далее". 
7. Введите имя пользователя, пароль, подтвердите пароль и выберите  **"Зарегистрироваться",**  чтобы создать учетную запись.
8. Введите количество пользователей, выберите план вы выставления счета, а затем выберите **"Далее".**
9.  Введите адрес&#39;компании, а затем введите **"Далее".**
10. Введите данные кредитной карты, а затем **задайте заказ.** Через несколько секунд учетная запись будет создана.
11. На этом этапе  можно выбрать "Продолжить настройку учетной записи" или войти в Центр администрирования Microsoft 365 и завершить настройку позже.
