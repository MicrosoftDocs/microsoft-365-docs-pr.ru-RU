# <a name="create-a-group-email-address"></a>Создание адреса электронной почты группы

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2GCG3?autoplay=false]

При создании группы создается центральный адрес электронной почты и место для совместной работы. Вы можете использовать это, чтобы предоставить адрес электронной почты, такие как продажи или сведения, которые каждый может видеть и получать сообщения электронной почты.

## <a name="try-it"></a>Проверьте, как это работает!

1. В Outlook на вкладке  **Главная** выберите  **новую группу**.
2. В поле  **Create Group**  введите имя группы, более короткий адрес электронной почты, например сведения и описание.
3. Откройте отсев конфиденциальности и выберите  **Private**.
4. Проверьте  **отправку всех групповых сообщений электронной**  почты и событий в почтовые ящики участников.
5. В **статье Добавить участников** выберите сотрудников, которые должны получать сообщения электронной почты, а затем выберите Добавить **участников.**
6. На  **вкладке Главная**  в Outlook выберите  **параметры группы,** а затем **изменить группу**.
7. В поле **Изменить группу** выберите "Пусть люди за пределами организации" по электронной почте по электронной почте перенаправить групповой контрольный ящик, а затем выберите  **ОК.**
