# <a name="create-a-group-email-address"></a>Создание адреса электронной почты группы

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2GCG3?autoplay=false]

При создании группы создается центральный адрес электронной почты и место для совместной работы. С помощью этого адреса можно предоставить адрес электронной почты, например сведения о продажах или сведениях, которые могут видеть и получать сообщения электронной почты.

## <a name="try-it"></a>Проверьте, как это работает!

1. В Outlook на вкладке **"Главная"** выберите **"Новая группа".**
2. В поле  **"Создание группы"**  введите имя группы, более короткий адрес электронной почты, например сведения, и описание.
3. Откройте dropdown конфиденциальности и выберите **"Частный".**
4. Check the  **Send all group email and events to members' inboxes**  check box.
5. В **области "Добавить участников"** выберите сотрудников, которые должны получать сообщения электронной почты, а затем выберите **"Добавить участников".**
6. На **вкладке "Главная"** в Outlook выберите **"Параметры группы"** и **"Изменить группу".**
7. В поле **"Изменить группу"** выберите "Позволить людям за пределами организации отправить сообщение электронной почты по электронной почте" и затем выберите  "ОК".
