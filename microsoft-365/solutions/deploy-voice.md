# <a name="plan-and-deploy-a-teams-voice-solution"></a>Планирование и развертывание решения голосовых служб Teams

Голосовое решение Teams позволяет сотрудникам Организации совершать звонки как внутри организации, так и за ее пределами. Полноценное решение голосовой связи состоит из Teams, телефонной системы Microsoft и выбора параметров для подключения к телефонной сети общего пользования (PSTN).

![Общие сведения о голосовых решениях Teams](..\media\solutions-architecture-center\voice-concepts.png)

Телефонная система предоставляет комплексные возможности обмена данными между филиалами Организации. Звонки между пользователями в вашей организации — независимо от их географических расположений — обрабатываются внутри телефонной системы, тем самым удаляя междугородные затраты на эти внутренние звонки.  

Подключив телефонную систему к телефонной сети общего пользования (PSTN), пользователи Teams смогут выполнять звонки за пределами Организации.

Это руководство по решению поможет вам:

- Выбор решения голосовой связи, который подходит для вашей организации
- Развертывание выбранного решения голосовой связи

Выполните следующие действия, чтобы выбрать, спланировать и настроить свое решение голосовой связи:

![Выбор решения для голосовой связи](..\media\solutions-architecture-center\voice-solutions-overview-1.png)

1. [Выбор решения для голосовой связи](https://docs.microsoft.com/MicrosoftTeams/cloud-voice-landing-page?toc=/microsoft-365/solutions/toc.json&bc=/microsoft-365/solutions/breadcrumb/toc.json)

2. [Настройка телефонной системы](https://docs.microsoft.com/microsoftteams/setting-up-your-phone-system?toc=/microsoft-365/solutions/toc.json&bc=/microsoft-365/solutions/breadcrumb/toc.json)

3. Настройте подключение по протоколу PSTN, выбрав одну или несколько из приведенных ниже параметров.
   - [Тарифный план](https://docs.microsoft.com/microsoftteams/set-up-calling-plans?toc=/microsoft-365/solutions/toc.json&bc=/microsoft-365/solutions/breadcrumb/toc.json) — решение корпорации Майкрософт "все в облаке" с корпорацией Майкрософт в качестве оператора PSTN
   - [Прямая маршрутизация](https://docs.microsoft.com/microsoftteams/direct-routing-configure?toc=/microsoft-365/solutions/toc.json&bc=/microsoft-365/solutions/breadcrumb/toc.json) — используйте прямую маршрутизацию для подключения собственной перевозчикской сети PSTN к Teams. 

Кроме того, вы можете получить сведения о том, как крупная и многонациональной корпорации была перенесена в решение голосовых вызовов в [компании Contoso](https://docs.microsoft.com/MicrosoftTeams/voice-case-study-overview?toc=/microsoft-365/solutions/toc.json&bc=/microsoft-365/solutions/breadcrumb/toc.json).
