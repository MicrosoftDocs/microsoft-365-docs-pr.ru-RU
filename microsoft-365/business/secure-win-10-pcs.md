# <a name="secure-windows-10-computers"></a>Защита компьютеров с Windows 10

Эта статья применима к Microsoft 365 бизнес премиум.

После того [как](set-up.md) вы Microsoft 365 бизнес премиум, настало время защитить компьютеры Windows 10 в вашей организации от кражи и вредоносных угроз, таких как вирусы и вредоносные программы.

## <a name="to-secure-your-windows-10-computers"></a>Обеспечение безопасности Windows 10 компьютеров

1. Войдите в [Центр администрирования Microsoft 365](https://admin.microsoft.com) с учетными данными глобального администратора. 
2. В левом nav выберите **установку,** а затем в соответствии с входом и безопасностью **выберите** Secure **Windows 10 компьютеров.** Выберите **View** для начала работы.
3. На странице **Secure your Windows 10** компьютеры ознакомьтесь со всеми сведениями, чтобы понять, что вы включите и какое влияние оказывает пользователь.

    В верхней части страницы выберите Начало **работы.**

4. На области **Secure Windows 10 выберите** параметры, которые необходимо включить. Дополнительные сведения о параметрах см. в Windows 10 [устройствах Secure.](secure-windows-10-devices.md) 
    
    Для большинства организаций эти параметры обеспечивают хороший уровень безопасности, однако, если у вашей организации более сложные потребности в безопасности, вы также можете использовать заранее определенные базовые показатели безопасности для защиты Windows 10 устройств. Дополнительные сведения см. в базовых данных по безопасности [Windows 10 устройств.](/mem/intune/protect/security-baselines)   

1. Выберите **Параметры Apply**.

    Эти параметры будут применяться для всех пользователей в вашей организации. Чтобы настроить различные политики для различных групп безопасности, см. в рубке Настройка параметров защиты устройств [для Windows 10 ПК.](protection-settings-for-windows-10-pcs.md)
