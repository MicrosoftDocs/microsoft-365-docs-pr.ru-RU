# <a name="secure-windows-10-computers"></a>Защита компьютеров с Windows 10

Эта статья относится к Microsoft 365 бизнес премиум.

После того как вы настроили [Microsoft](set-up.md) 365 бизнес премиум, настало время защитить компьютеры с Windows 10 в вашей организации от кражи и вредоносных угроз, таких как вирусы и вредоносные программы.

## <a name="to-secure-your-windows-10-computers"></a>Защита компьютеров с Windows 10

1. Войдите в [Центр администрирования Microsoft 365](https://admin.microsoft.com) с учетными данными глобального администратора. 
2. On the left nav, select **Setup** and then, under **Sign-in and security,** choose **Secure your Windows 10 computers**. Choose **View** to get started.
3. На странице **"Защита компьютеров с Windows 10"** прочитайте все сведения, чтобы понять, что вы включите и как это влияет на пользователя.

    В верхней части страницы выберите "Начало **работы".**

4. В области **"Защита компьютеров с Windows 10"** выберите параметры, которые необходимо включить. Дополнительные сведения о параметрах см. в сведениях [о безопасных устройствах с Windows 10.](secure-windows-10-devices.md) 
    
    Для большинства организаций эти варианты обеспечивают хороший уровень безопасности, но если в вашей организации более сложные требования к безопасности, вы также можете использовать предварительно определенные базовые параметры безопасности для защиты устройств с Windows 10. Дополнительные сведения [см. в базовых параметрах безопасности для устройств с Windows 10.](https://docs.microsoft.com/mem/intune/protect/security-baselines)   

1. Выберите **"Применить параметры".**

    Эти параметры будут применяться для всех пользователей в организации. Чтобы настроить разные политики для разных групп безопасности, см. параметры защиты устройств для компьютеров [с Windows 10.](protection-settings-for-windows-10-pcs.md)
