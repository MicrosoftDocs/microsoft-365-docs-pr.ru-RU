# <a name="secure-windows-10-computers"></a>Защита компьютеров с Windows 10

Эта статья применима к Microsoft 365 Бизнес Премиум.

После того, как вы настроите [Microsoft](set-up.md) 365 Бизнес Премиум, настало время защитить компьютеры Windows 10 в вашей организации от кражи и вредоносных угроз, таких как вирусы и вредоносные программы.

## <a name="to-secure-your-windows-10-computers"></a>Защита компьютеров Windows 10

1. Войдите в [Центр администрирования Microsoft 365](https://admin.microsoft.com) с учетными данными глобального администратора. 
2. В левом окантове выберите **установку,** а затем в соответствии с входом и безопасностью **выберите** **Безопасные компьютеры Windows 10**. Выберите **View** для начала работы.
3. На странице **Secure your Windows 10 computers** read all the information to understand what you are turning on, and what the user impact is.

    В верхней части страницы выберите Начало **работы.**

4. На области **Безопасность компьютеров Windows 10** выберите параметры, которые необходимо включить. Дополнительные сведения о параметрах см. в сведениях [о устройствах Secure Windows 10.](secure-windows-10-devices.md) 
    
    Для большинства организаций эти параметры обеспечивают хороший уровень безопасности, однако если у организации более сложные потребности в безопасности, вы также можете использовать заранее определенные базовые показатели безопасности для защиты устройств с Windows 10. Дополнительные сведения см. [в базовых данных по безопасности для устройств с Windows 10.](/mem/intune/protect/security-baselines)   

1. Выберите **Параметры Apply**.

    Эти параметры будут применяться для всех пользователей в вашей организации. Чтобы настроить различные политики для различных групп безопасности, см. в рубке Настройка параметров защиты устройств [для ПК Windows 10.](protection-settings-for-windows-10-pcs.md)
