# <a name="configure-windows-10-devices"></a>Настройка устройств с Windows 10

Во время настройки создается политика устройств для Windows 10, которая применяется ко всем пользователям в вашей организации. Когда пользователь вошел в Приложение Office на Windows 10 с его учетной записью или учебной учетной записью, настраиваемые параметры автоматически применяются для обеспечения безопасного подключения пользователя.
  
Советуем во время настройки принять стандартную конфигурацию. После настройки можно создать больше политик и нацелить их на определенные группы пользователей.
  
Сведения о том, как добавить политику для управления конфигурацией устройств с Windows 10, см. в статье [Настройка конфигурации устройств для компьютеров с Windows 10](protection-settings-for-windows-10-pcs.md).
