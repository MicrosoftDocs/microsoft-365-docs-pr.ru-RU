# <a name="configure-windows-10-devices"></a>Настройка устройств с Windows 10

Во время настройки создается политика устройств для Windows 10, которая применяется ко всем пользователям в организации. Когда пользователь подключается к приложению Office на своем устройстве с Windows 10 с помощью своей учебной или учебной учетной записи, настраиваемые параметры автоматически применяются, чтобы убедиться, что пользователь подключается безопасным способом.
  
Советуем во время настройки принять стандартную конфигурацию. После настройки можно создать дополнительные политики и нацелить их на определенные группы пользователей.
  
Сведения о том, как добавить политику для управления конфигурацией устройств с Windows 10, см. в статье [Настройка конфигурации устройств для компьютеров с Windows 10](protection-settings-for-windows-10-pcs.md).
