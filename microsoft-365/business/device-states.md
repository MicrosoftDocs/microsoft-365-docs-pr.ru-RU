# <a name="device-states"></a>Состояния устройств

Эта статья относится к Microsoft 365 бизнес премиум.

Устройства в списке **Действия устройств** (домашняя страница администратора \> **Действия устройств**) могут находиться в перечисленных ниже состояниях.
  
![In the Device actions list, you can see the Devices states.](../media/a621c47e-45d9-4e1a-beb9-c03254d40c1d.png)
  
|**Состояние**|**Описание**|
|:-----|:-----|
|Под управлением Intune  <br/> |Управляется Microsoft 365 бизнес премиум.  <br/> |
|Снятие с учета ожидается  <br/> |Microsoft 365 бизнес премиум готов к удалению данных компании с устройства.  <br/> |
|Снятие с учета выполняется  <br/> |Microsoft 365 бизнес премиум в настоящее время удаляет данные компании с устройства.  <br/> |
|Не удалось снять с учета  <br/> | Не удалось удалить данные компании.  <br/> |
|Отмена отмены  <br/> |Действие отмены отменено.  <br/> |
|Ожидает очистки  <br/> |Ожидается сброс до заводских настроек.  <br/> |
|Очистка выполняется  <br/> |Сброс до заводских настроек был инициирован.  <br/> |
|Не удалось выполнить очистку  <br/> |Не удалось восстановить заводские настройки.  <br/> |
|"Wipe canceled" (Стирать отменено  <br/> |Заводская стирка отменена.  <br/> |
|Неработоспособное  <br/> |Действие ожидает выполнения (или находится в процессе), но устройство не находится в состоянии ожидания в течение 30 дней.  <br/> |
|Ожидает удаления  <br/> |Ожидается удаление.  <br/> |
|Обнаружено  <br/> |Microsoft 365 бизнес премиум обнаружил устройство.  <br/> |
