# <a name="review-detected-threats-and-take-action"></a>Просмотр обнаруженных угроз и выполнение необходимых действий

Как только обнаруживается вредоносный файл или программное обеспечение, антивирусная программа Microsoft Defender блокирует его и блокирует его работу. Кроме того, если включена облачная защита, новые обнаруженные угрозы добавляются в антивирусную и антивторою, чтобы защитить и другие устройства и пользователей.

Антивирусная программа "Microsoft Defender" обнаруживает и защищает от следующих видов угроз:

- Вирусы, вредоносные программы и веб-угрозы на устройствах
- Попытки фишинга
- Попытки кражи данных

Как ИТ-специалист или администратор вы можете просматривать сведения об обнаружении угроз на устройствах [с Windows 10,](/mem/intune/enrollment/device-enrollment) зарегистрированных в Intune, в Центре администрирования Microsoft 365. Вы увидите сводные сведения, например:

- Количество устройств, которые нуждаются в антивирусной защите
- Сколько устройств не соответствует политикам безопасности
- Количество активных, устраненных или устраненных в настоящее время угроз

Существует несколько вариантов просмотра определенной информации об обнаружении угроз и устройствах:

- Страница **"Активные** устройства" в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Центре администрирования Microsoft 365.</a> См. [статью "Управление обнаружением угроз" на странице "Активные устройства".](#manage-threat-detections-on-the-active-devices-page)
- Страница **"Активные угрозы"** в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Центре администрирования Microsoft 365.</a> См. [статью "Управление обнаружением](#manage-threat-detections-on-the-active-threats-page) угроз" на странице "Активные угрозы" в этой статье.
- Страница **антивирусной программы** в <a href="https://go.microsoft.com/fwlink/p/?linkid=2150463" target="_blank">Microsoft Endpoint Manager.</a> См. [статью "Управление обнаружением угроз в Microsoft Endpoint Manager".](#manage-threat-detections-in-microsoft-endpoint-manager)

Дополнительные см. в записях об угрозах, обнаруженных антивирусной программой ["Microsoft Defender".](threats-detected-defender-av.md)

## <a name="manage-threat-detections-on-the-active-devices-page"></a>Управление обнаружением угроз на странице **"Активные устройства"**

Следующая процедура применима к клиентам с Microsoft 365 бизнес премиум.

1. Перейдите в Центр администрирования Microsoft 365 и <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> войдите в нее.

2. На странице навигации выберите устройства  >  **"Активные устройства".** Вы увидите список активных устройств и подробные сведения, такие как состояние защиты, состояние антивирусной защиты и количество обнаруженных активных угроз.

3. Выберите устройство, чтобы просмотреть дополнительные сведения об этом устройстве и доступных действиях. Откроется окно с рекомендациями и доступными действиями, такими как "Обновление **политики",**"Обновление антивирусной программы", "Выполнить **быструю** проверку", "Выполнить полную проверку" и другие. 

## <a name="manage-threat-detections-on-the-active-threats-page"></a>Управление обнаружением угроз на **странице "Активные угрозы"**

Следующая процедура применима к клиентам с Microsoft 365 бизнес премиум. [Устройства с Windows 10 должны быть защищены](/microsoft-365/business/secure-win-10-pcs) и [зарегистрированы в Intune.](/mem/intune/enrollment/windows-enrollment-methods)

> [!NOTE]
> Антивирусная **карта Microsoft Defender** и страница **"Активные** угрозы" развяываются поэтапно, поэтому у вас может не быть немедленного доступа к ним.

1. Перейдите в Центр администрирования Microsoft 365 и <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> войдите в нее.

2. На **антивирусной карте Microsoft Defender** выберите **"Просмотр активных угроз".** (Кроме того, в области навигации выберите **"Состояние"**  >  **Угрозы & антивирусной** программой.)

3. На странице **"Активные угрозы"** выберите обнаруженную угрозу, чтобы узнать больше о ней. Откроется окно с подробными сведениями об этой угрозе, в том числе о затронутых устройствах.

4. Во flyout выберите устройство для просмотра доступных действий, таких как политика **обновления,** обновление антивирусной **программы,** запуск **быстрой** проверки и другие.

## <a name="actions-you-can-take"></a>Действия, которые можно принять

При просмотре сведений об определенных угрозах или устройствах вы увидите рекомендации и одно или несколько действий, которые можно принять. В следующей таблице описываются действия, которые могут от вас потребоваться.<br><br>

| Действие | Описание |
|--|--|
| Настройка защиты | Необходимо настроить политики защиты от угроз. Выберите ссылку, чтобы перейти на страницу конфигурации политики.<br><br>Нужна помощь? См. "Управление безопасностью устройств с помощью политик безопасности [конечных точек" в Microsoft Intune.](/mem/intune/protect/endpoint-security-policy) |
| Обновление политики | Необходимо обновить или настроить политики защиты от вирусов и защиты в режиме реального времени. Выберите ссылку, чтобы перейти на страницу конфигурации политики.<br><br>Нужна помощь? См. "Управление безопасностью устройств с помощью политик безопасности [конечных точек" в Microsoft Intune.](/mem/intune/protect/endpoint-security-policy) |
| Запуск быстрой проверки | Запускает быструю антивирусную проверку на устройстве, уделяя особое внимание общим расположениям, в которых может быть зарегистрирована вредоносная программа, таким как ключи реестра и известные папки запуска Windows. |
| Запуск полной проверки | Запускает полную антивирусную проверку на устройстве, уделяя особое внимание общим расположениям, в которых может быть зарегистрирована вредоносная программа, и включая все файлы и папки на устройстве. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Обновление антивирусной программы | Требует, чтобы устройство было получать [обновления аналитики](https://go.microsoft.com/fwlink/?linkid=2149926) безопасности для защиты от вирусов и защиты от вирусов. |
| Перезапуск устройства | Заставляет устройство с Windows 10 перезапуститься в течение пяти минут.<br><br>**ВАЖНО!** Владелец или пользователь устройства не получает автоматическое уведомление о перезапуске и может потерять несбереженную работу. |

## <a name="manage-threat-detections-in-microsoft-endpoint-manager"></a>Управление обнаружением угроз в Microsoft Endpoint Manager

Для управления обнаружением угроз можно использовать Microsoft Endpoint Manager. Устройства с Windows 10 должны быть [зарегистрированы в Intune](/mem/intune/enrollment/windows-enrollment-methods) (в составе Microsoft Endpoint Manager).

1. Перейдите в Центр администрирования Microsoft Endpoint Manager и <a href="https://go.microsoft.com/fwlink/p/?linkid=2150463" target="_blank">https://endpoint.microsoft.com</a> войдите в нее.

2. В области навигации выберите **"Безопасность конечных точек".**

3. В **области "Управление"** выберите **"Антивирусная программа".** Вы увидите несколько вкладок, например "Сводка", "Неработоспособные конечные точки **Windows 10"** и "Обнаруженное вредоносное ПО **в Windows 10".**

4. Просмотрите сведения на доступных вкладок и затем примите все необходимые меры.

Например, предположим, что устройства перечислены на вкладке "Обнаруженная вредоносная **программа" в Windows 10.** При выборе устройства доступны определенные действия, такие как "Перезагрузка", "Быстрая проверка", "Полная проверка", "Синхронизация" или     **"Обновление".** Выберите действие для этого устройства.

В следующей таблице описаны действия, которые можно увидеть в Microsoft Endpoint Manager.<br><br>

| Действие | Описание |
|--|--|
| Перезапуск | Заставляет устройство с Windows 10 перезапуститься в течение пяти минут.<br><br>**ВАЖНО!** Владелец или пользователь устройства не получает автоматическое уведомление о перезапуске и может потерять несбереженную работу. |
| Быстрая проверка | Запускает быструю антивирусную проверку на устройстве, уделяя особое внимание общим расположениям, в которых может быть зарегистрирована вредоносная программа, таким как ключи реестра и известные папки запуска Windows. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Полная проверка | Запускает полную антивирусную проверку на устройстве, уделяя особое внимание общим расположениям, в которых может быть зарегистрирована вредоносная программа, и включая все файлы и папки на устройстве. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Синхронизировать | Требуется устройство для регистрации с помощью Intune (в составе Microsoft Endpoint Manager). При этом устройство получает все ожидающих действий или политик, которые назначены устройству. |
| Обновление подписей | Требует, чтобы устройство было получать [обновления аналитики](https://go.microsoft.com/fwlink/?linkid=2149926) безопасности для защиты от вирусов и защиты от вирусов. |

> [!TIP]
> Дополнительные сведения см. в [удаленных действиях для устройств.](/mem/intune/protect/endpoint-security-manage-devices#remote-actions-for-devices)

## <a name="how-to-submit-a-file-for-malware-analysis"></a>Отправка файла для анализа вредоносных программ

Если у вас есть файл, который, по вашему мнению, был пропущен или ошибочно классифицирован как вредоносный, вы можете отправить его в корпорацию Майкрософт для анализа вредоносных программ. Пользователи и ИТ-администраторы могут отправить файл для анализа. Посетите [https://www.microsoft.com/wdsi/filesubmission](https://www.microsoft.com/wdsi/filesubmission) сайт .
