# <a name="review-detected-threats-and-take-action"></a>Просмотр обнаруженных угроз и выполнение необходимых действий

Как только обнаруживается вредоносный файл или программное обеспечение, антивирус Microsoft Defender блокирует его и не позволяет ему работать. При включенной облачной защите в антивирусную и антивирусную движок добавляются вновь обнаруженные угрозы, чтобы защитить и другие устройства и пользователей.

Антивирус Microsoft Defender обнаруживает и защищает от следующих видов угроз:

- Вирусы, вредоносные программы и веб-угрозы на устройствах
- Попытки фишинга
- Попытки кражи данных

Как ИТ-специалист/администратор вы можете просматривать сведения об обнаружении угроз на устройствах [Windows 10,](/mem/intune/enrollment/device-enrollment) зарегистрированных в Intune в центре администрирования Microsoft 365. Вы увидите сводную информацию, например:

- Сколько устройств нуждается в антивирусной защите
- Сколько устройств не соответствует политикам безопасности
- Сколько угроз в настоящее время активны, устранены или устранены

У вас есть несколько вариантов просмотра определенных сведений об обнаружении угроз и устройствах:

- Страница **Активные устройства** в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">центре администрирования Microsoft 365</a>. В [этой статье см. статью](#manage-threat-detections-on-the-active-devices-page) Управление обнаружением угроз на странице Активные устройства.
- Страница **Активные угрозы** в <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">центре администрирования Microsoft 365.</a> Управление [обнаружением угроз см. на странице Active threats](#manage-threat-detections-on-the-active-threats-page) в этой статье.
- Страница **Антивирус в** Microsoft <a href="https://go.microsoft.com/fwlink/p/?linkid=2150463" target="_blank">Endpoint Manager</a>. В этой статье см. в статье Управление обнаружением угроз [в Microsoft Endpoint Manager.](#manage-threat-detections-in-microsoft-endpoint-manager)

Дополнительные новости см. в [сайте Threats detected by Microsoft Defender Antivirus](threats-detected-defender-av.md).

## <a name="manage-threat-detections-on-the-active-devices-page"></a>Управление обнаружением угроз на странице **Активные устройства**

Следующая процедура применяется к клиентам, у которых есть Microsoft 365 Бизнес Премиум.

1. Перейдите в центр администрирования Microsoft 365 и <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> войдите.

2. На странице навигации выберите **устройства Devices**  >  **Active.** Вы увидите список активных устройств и сведений, таких как состояние защиты, состояние защиты от антивирусных вирусов и количество обнаруженных активных угроз.

3. Выберите устройство, чтобы просмотреть дополнительные сведения об этом устройстве и доступных действиях. Вылет открывается с рекомендациями и доступными действиями, такими как политика **обновления,** обновление антивируса, быстрое **сканирование,** запуск полного сканирования **и** другие.

## <a name="manage-threat-detections-on-the-active-threats-page"></a>Управление обнаружением угроз на странице **Активные угрозы**

Следующая процедура применяется к клиентам, у которых есть Microsoft 365 Бизнес Премиум. [Устройства Windows 10 должны быть защищены](./secure-win-10-pcs.md) и [зарегистрированы в Intune.](/mem/intune/enrollment/windows-enrollment-methods)

> [!NOTE]
> Антивирусная **карта Microsoft Defender** и страница Active **threats** выкатываются поэтапно, поэтому у вас может не быть немедленного доступа к ним.

1. Перейдите в центр администрирования Microsoft 365 и <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> войдите.

2. На **антивирусной карте Microsoft Defender** выберите **Просмотр активных угроз.** (Кроме того, в области навигации выберите   >  **Угрозы & антивируса**.)

3. На странице **Активные угрозы** выберите обнаруженную угрозу, чтобы узнать больше об этом. Откроется вылет с подробными сведениями об этой угрозе, в том числе о том, какие устройства затронуты.

4. На вылете выберите устройство для просмотра доступных действий, таких как политика **обновления,** обновление антивируса, быстрое сканирование и другие.

## <a name="actions-you-can-take"></a>Действия, которые можно принять

При просмотре сведений о конкретных угрозах или устройствах вы увидите рекомендации и одно или несколько действий, которые можно принять. В следующей таблице описываются действия, которые можно увидеть.<br><br>

| Действие | Описание |
|--|--|
| Настройка защиты | Необходимо настроить политики защиты от угроз. Выберите ссылку, чтобы перейти на страницу конфигурации политики.<br><br>Нужна помощь? См. в рубке Управление безопасностью устройств с политиками безопасности конечной [точки в Microsoft Intune.](/mem/intune/protect/endpoint-security-policy) |
| Обновление политики | Необходимо обновить или настроить антивирусные политики и политики защиты в режиме реального времени. Выберите ссылку, чтобы перейти на страницу конфигурации политики.<br><br>Нужна помощь? См. в рубке Управление безопасностью устройств с политиками безопасности конечной [точки в Microsoft Intune.](/mem/intune/protect/endpoint-security-policy) |
| Выполнить быстрое сканирование | Запускает быстрое антивирусное сканирование на устройстве, ориентируясь на общие расположения, в которых может быть зарегистрирована вредоносная программа, например ключи реестра и известные папки запуска Windows. |
| Запуск полного сканирования | Запускает полное антивирусное сканирование на устройстве с упором на общие расположения, в которых может быть зарегистрировано вредоносное ПО, и включая каждый файл и папку на устройстве. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Обновление антивируса | Требуется, чтобы устройство было [получать](https://go.microsoft.com/fwlink/?linkid=2149926) обновления сведений о безопасности для защиты от антивирусных и антивирусных программ. |
| Устройство перезапуска | Заставляет устройство Windows 10 перезапуститься в течение пяти минут.<br><br>**ВАЖНО:** Владелец устройства или пользователь не получает автоматического уведомления о перезапуске и может потерять незаверяемую работу. |

## <a name="manage-threat-detections-in-microsoft-endpoint-manager"></a>Управление обнаружением угроз в Microsoft Endpoint Manager

Для управления обнаружением угроз можно использовать Microsoft Endpoint Manager. Устройства Windows 10 должны быть [зарегистрированы в Intune](/mem/intune/enrollment/windows-enrollment-methods) (входит в Microsoft Endpoint Manager).

1. Перейдите в центр администрирования microsoft Endpoint Manager и <a href="https://go.microsoft.com/fwlink/p/?linkid=2150463" target="_blank">https://endpoint.microsoft.com</a> войдите.

2. В области навигации выберите безопасность **конечных точек.**

3. В **статье Управление** выберите **антивирус**. Вы увидите несколько вкладок, таких как **Сводка,** **Windows 10** нездоровые конечные точки и обнаруженная вредоносная программа **Windows 10.**

4. Просмотрите сведения о доступных вкладок и примите все необходимые действия.

Например, предположим, что устройства перечислены на вкладке **Обнаруженная вредоносная программа Windows 10.** При выборе устройства будут доступны определенные действия, такие как перезапуск, быстрое **сканирование,** полное **сканирование,** синхронизация или обновление **подписей.** Выберите действие для этого устройства.

В следующей таблице описываются действия, которые можно увидеть в Microsoft Endpoint Manager.<br><br>

| Действие | Описание |
|--|--|
| Перезапуск | Заставляет устройство Windows 10 перезапуститься в течение пяти минут.<br><br>**ВАЖНО:** Владелец устройства или пользователь не получает автоматического уведомления о перезапуске и может потерять незаверяемую работу. |
| Быстрая проверка | Запускает быстрое антивирусное сканирование на устройстве, ориентируясь на общие расположения, в которых может быть зарегистрирована вредоносная программа, например ключи реестра и известные папки запуска Windows. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Полное сканирование | Запускает полное антивирусное сканирование на устройстве с упором на общие расположения, в которых может быть зарегистрировано вредоносное ПО, и включая каждый файл и папку на устройстве. Результаты отправляются [в Microsoft Endpoint Manager.](/mem/intune/fundamentals/tutorial-walkthrough-endpoint-manager) |
| Синхронизировать | Требуется, чтобы устройство проверяли в Intune (входит в Microsoft Endpoint Manager). При проверке устройства устройство получает все ожидающих действий или политик, назначенных устройству. |
| Обновление подписей | Требуется, чтобы устройство было [получать](https://go.microsoft.com/fwlink/?linkid=2149926) обновления сведений о безопасности для защиты от антивирусных и антивирусных программ. |

> [!TIP]
> Дополнительные сведения см. в [дополнительных сведениях о удаленных действиях для устройств.](/mem/intune/protect/endpoint-security-manage-devices#remote-actions-for-devices)

## <a name="how-to-submit-a-file-for-malware-analysis"></a>Отправка файла для анализа вредоносных программ

Если у вас есть файл, который, по вашему мнению, был пропущен или неправильно классифицирован как вредоносный, вы можете отправить этот файл в Корпорацию Майкрософт для анализа вредоносных программ. Пользователи и ИТ-администраторы могут отправлять файл для анализа. Посетите [https://www.microsoft.com/wdsi/filesubmission](https://www.microsoft.com/wdsi/filesubmission) .
