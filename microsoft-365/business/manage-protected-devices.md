# <a name="manage-protected-devices-with-microsoft-365-business-premium"></a>Управление защищенными устройствами с помощью Microsoft 365 Business Premium

После подключения устройства к организации с помощью учетной записи работы с устройством или приложениями Office на устройстве можно защитить устройство с помощью политик, которые вы настроили. Подключенные устройства перечислены на странице **Устройства.** Если устройство потеряно или украдено, вы можете управлять им с этой страницы, чтобы удалить любые данные компании. Вы также можете сбросить устройства Windows 10 в заводские параметры, чтобы стереть все настраиваемые параметры. 

Дополнительные сведения о задачах, которые можно использовать для управления устройствами, см. в статьях:
