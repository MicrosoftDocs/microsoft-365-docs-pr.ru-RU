# <a name="manage-protected-devices-with-microsoft-365-business-premium"></a>Управление защищенными устройствами с помощью Microsoft 365 бизнес премиум

После того как пользователь подключит свое устройство к вашей организации, войди с помощью своей учетной записи на устройство или к приложениям Office на устройстве, вы сможете защитить устройство с помощью политик, которые вы настроили. Подключенные устройства перечислены на странице **"Устройства".** Если устройство потеряно или украдено, вы можете управлять им на этой странице, чтобы удалить любые данные компании. Вы также можете восстановить заводские параметры устройств с Windows 10, чтобы сбросить все пользовательские параметры. 

Дополнительные сведения о задачах, которые можно использовать для управления устройствами, см. в указанных здесь статьях. 
  
[Удаление данных организации с устройств](remove-company-data.md)
  
[Восстановление заводских настроек для устройств с Windows 10](reset-devices-to-factory-settings.md)

[Добавление управляемого устройства](https://docs.microsoft.com/microsoft-365/business/app-protection-settings-for-android-and-ios)
