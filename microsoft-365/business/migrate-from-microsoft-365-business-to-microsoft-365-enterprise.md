# <a name="migrate-from-microsoft-365-business-premium-to-microsoft-365-e3"></a>Миграция с Microsoft 365 бизнес премиум на Microsoft 365 E3

В Microsoft 365 бизнес премиум есть все необходимое для малого бизнеса, сочетающее лучшие в своем классе облачные приложения для повышения производительности с простым управлением устройствами и безопасностью, которые позволяют сотрудникам работать наилучшим образом. Однако в некоторых случаях может потребоваться перенести подписку Microsoft 365 бизнес премиум на Microsoft 365 E3. 

Например, ваш бизнес стал больше, и ему требуется более 300 лицензий (поздравляем, к тому же).

Кроме того, вашей организации необходимы корпоративные функции, такие как приложения Microsoft 365 для предприятий, Windows 10 Корпоративная E3 или корпоративные лицензии клиентского доступа (CALs).

Обновление легко: вы можете начать обновление [из Центра администрирования.](../commerce/subscriptions/upgrade-to-different-plan.md) Все ваши данные и конфигурация в текущей подписке поддерживаются. Вам не нужно ничего делать для подготовки к миграции и ничего делать после этого, кроме использования новых функций.

>[!Note]
>Вы также можете использовать подписку Microsoft 365 бизнес премиум на 300 мест и получить подписку Microsoft 365 E3 для более чем 300 мест. Однако Microsoft Defender для Office 365 не входит в состав Microsoft 365 E3. Для дальнейшей защиты от угроз следует добавить дополнительные лицензии Защитника для Office 365, чтобы лицензии были у всех пользователей, в области применения ваших полиции Защитника для Office 365.
>

## <a name="migration"></a>Миграция

Для миграции вместе с партнером переместите подписку и лицензии Microsoft 365 бизнес премиум на подходящую подписку Microsoft 365 E3 с ее лицензиями.

В следующих разделах описаны изменения, которые необходимо внести (если они есть) и что можно сделать после миграции.

### <a name="microsoft-365-subscription-configuration-and-data"></a>Конфигурация и данные подписки На Microsoft 365

Перед миграцией не нужно вносить какие-либо изменения в текущую подписку или данные, в том числе:

- Конфигурация подписки, например доменные имена DNS.
- Учетные записи пользователей и групп и параметры проверки подлинности, такие как многофакторная проверка подлинности или политики условного доступа.
- Конфигурации служб повышения производительности и их данные, такие как Teams, почтовые ящики Exchange Online, сайты SharePoint Online, папки OneDrive для бизнеса и записные книжки OneNote.

Теперь пользователи могут пользоваться неограниченной памятью в почтовых ящиках Exchange Online и в папках OneDrive для бизнеса.

Вы можете начать использовать Cloud App Discovery, Azure AD Connect Health и службу SSO для более чем 10 приложений.

>[!Note]
>Пользователи, перенесенные в Microsoft 365 E3, больше не могут использовать MileIQ.
>

<a name="threat-protection"></a>
### <a name="threat-protection"></a>Защита от угроз

Windows 10 для бизнеса включает следующие средства защиты:

- Принудительных целостности процесса загрузки операционной системы
- Принудителен к целостности конфиденциальных операционных компонентов
- Дополнительные уязвимости и меры по смягчению эксплойтов нулевого дня
- Защита сети на основе репутации для Microsoft Edge, Internet Explorer и Chrome
- Брандмауэр на основе хостов
- Меры для программ-вымогателей
- Аппаратная изоляция Microsoft Edge
- Управление приложениями на сфере интеллектуальной безопасности Graph
- Устройство управления (USB)
- Защита сети от веб-угроз
- Правила предотвращения вторжений в хозяйства

Windows 10 Корпоративная E3 также включает управление корпоративной изоляцией на основе оборудования для Microsoft Edge.

>[!Note]
>Пользователям, перенесенным в Microsoft 365 E3, потребуется лицензия Microsoft Defender для Office 365 для дальнейшей защиты от угроз. Обязательно приобретите дополнительные лицензии Защитника для Office 365, чтобы лицензии были у всех пользователей, в области применения ваших полиции Защитника для Office 365. 
>

### <a name="device-management-with-intune"></a>Управление устройствами с помощью Intune

Перед миграцией не нужно вносить изменения в текущую конфигурацию Intune, которая включает зарегистрированные устройства, устройства и параметры приложения.

### <a name="windows-10"></a>Windows 10

Microsoft 365 бизнес премиум включает Windows 10 бизнес, которую можно установить с помощью Windows AutoPilot. При переходе на Microsoft 365 E3 каждая лицензия пользователя включает Windows 10 Корпоративная E3, которую также можно установить с помощью Windows Autopilot.

<a name="office-365-business"></a>
###  <a name="microsoft-365-apps-for-business"></a>Приложения Microsoft 365 для бизнеса

Клиент приложений Microsoft 365 для бизнеса, установленный на ваших устройствах, автоматически начнет использовать функции приложений Microsoft 365 для предприятий. После миграции вы можете использовать:

 - Поддержка групповой политики
 - Сравнение и запрос электронных таблиц
 - Бизнес-аналитика
