# <a name="use-the-step-by-step-guide-to-add-autopilot-devices-and-profile"></a>Добавление устройств и профиля AutoPilot с помощью пошагового мастера

Вы можете использовать Windows AutoPilot  для Windows 10 устройств для бизнеса, чтобы они были готовы к использованию при их отдав их сотрудникам.
  
## <a name="device-requirements"></a>Требования к устройству

Устройства должны соответствовать этим требованиям:
  
- Windows 10 версии 1703 или более поздней версии
    
- Новые устройства, которые не прошли Windows вне окна
    
## <a name="use-the-setup-guide-to-create-devices-and-profiles"></a>Создание устройств и профилей с помощью мастера настройки

[![Надпись, оповещающая об изменении Центра администрирования. Дополнительные сведения см. на сайте aka.ms/aboutM365preview.](../media/m365admincenterchanging.png)](/office365/admin/microsoft-365-admin-center-preview)

Если вы еще не создали группы устройств или профили, лучше всего приступить к работе с помощью пошагового руководства. Вы также можете [добавлять устройства и](create-and-edit-autopilot-devices.md) назначать им [профили](create-and-edit-autopilot-profiles.md) без использования руководства. 
  
1. Перейдите в Центр администрирования <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>.

2. На левой области навигации выберите **Устройства** \> **АвтоПилот**.

    ![В центре администрирования выберите устройства, а затем AutoPilot.](../media/AutoPilot.png)
  
2. На странице **АвтоПилот** щелкните или нажмите кнопку **Начните руководство**.
    
    ![Click Start guide for step-by-step instructions for Autopilot.](../media/31662655-d1e6-437d-87ea-c0dec5da56f7.png)
  
3. На Upload .csv **файл со** списком страниц устройств просмотрите место, где у вас есть готовый файл .CSV, а затем **Откройте** \> **далее**. Файл должен иметь три загона:
    
    - Столбец A: Device Serial Number (Серийный номер устройства)
    
    - Столбец B: Windows Product ID (Идентификатор продукта Windows)
    
    - Столбец C: Hardware Hash (Хэш оборудования)
    
    Эти сведения можно получить у поставщика оборудования или с помощью скрипта [Get-WindowsAutoPilotInfo PowerShell](https://www.powershellgallery.com/packages/Get-WindowsAutoPilotInfo) для создания CSV-файла. 
    
    Дополнительные сведения см. в статье [CSV-файл со списком устройств](../admin/misc/device-list.md). Вы также можете скачать образец файла на странице **Отправка CSV-файла со списком устройств**. 
    
> [!NOTE]
> Этот скрипт использует WMI для получения свойств, необходимых для регистрации устройства с Windows автопилотом. Обратите внимание, что обычно в результате CSV-файл не собирает значение Windows продукта (PKID), так как это не требуется для регистрации устройства и PKID является NULL в выходном CSV полностью нормально. Заполняется только серийный номер и аппаратный хаш.
    
4. На странице **Назначение профиля** можно выбрать существующий профиль или создать новый. Если у вас еще нет его, вам будет предложено создать его. 
    
    Профиль  это совокупность настроек, которые можно применить к одному устройству или группе устройств.
    
    Функции по умолчанию необходимы и устанавливаются автоматически. К ним относятся следующие параметры:
    
    - Пропустить регистрацию Кортаны, OneDrive и OEM.
    
    - Создание интерфейса входа с использованием фирменной символики компании.
    
    - Подключение устройствам Azure Active Directory учетные записи и автоматически регистрируются для Microsoft 365 бизнес премиум.
    
    Дополнительные сведения см. [в дополнительных сведениях о параметрах профилей автопилота.](autopilot-profile-settings.md) 
    
5. Также доступны параметры **Пропустить параметры конфиденциальности** и **Не разрешать пользователю становиться локальным администратором**. По умолчанию они **отключены**. 
    
    Нажмите кнопку **Далее**.
    
6. **Это означает,** что созданный (или выбранный) профиль будет применен к группе устройств, созданной путем отправки списка устройств. Параметры будут действовать при входе пользователей устройства в следующий. Нажмите кнопку **Закрыть**.
