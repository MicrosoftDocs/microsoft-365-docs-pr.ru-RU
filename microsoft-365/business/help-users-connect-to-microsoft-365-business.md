# <a name="help-users-connect-to-microsoft-365-business-premium"></a>Помощь пользователям в подключении к Microsoft 365 бизнес премиум

После завершения настройки политики устройств по умолчанию будут на месте и готовы к установке. На мобильных устройствах пользователи скачивает приложения Office. Политики применяются, когда пользователь вошел с помощью своей учебной или учебной учетной записи. Прежде чем политики будут применяться к ПК с Windows, например обновление на месте, необходимо сделать еще немного. См. следующие разделы и поделитесь ими, чтобы помочь пользователям настроиться и установить подключение:
  
[Настройка устройств с Windows для пользователей Microsoft 365 бизнес премиум](set-up-windows-devices.md)
  
[Добавление устройств и профиля AutoPilot с помощью пошагового мастера](add-autopilot-devices-and-profile.md)
  
[Настройка мобильных устройств для пользователей Microsoft 365 бизнес премиум](set-up-mobile-devices.md)
