# <a name="help-users-connect-to-microsoft-365-business-premium"></a>Помогите пользователям подключиться к Microsoft 365 Бизнес Премиум

После завершения настройки политики устройств по умолчанию будут на месте и готовы к применении. На мобильных устройствах пользователи загружают приложения Office. Политики применяются, когда пользователь вошел в свою работу или учетную запись школы. Перед тем, как политики применяются к ПК с Windows, например обновление на месте, необходимо еще немного поработать. См. и поделитесь следующими разделами, которые помогут пользователям настроиться и подключаться к сети:
  
[Настройка устройств Windows для пользователей Microsoft 365 Бизнес Премиум](set-up-windows-devices.md)
  
[Добавление устройств и профиля AutoPilot с помощью пошагового мастера](add-autopilot-devices-and-profile.md)
  
[Настройка мобильных устройств для пользователей Microsoft 365 Бизнес Премиум](set-up-mobile-devices.md)
