# <a name="remove-company-data-from-devices"></a>Удаление данных организации с устройств

Эта статья относится к Microsoft 365 бизнес премиум.

## <a name="remove-company-data"></a>Удаление данных компании

Microsoft 365 для бизнеса можно использовать для удаления данных [](app-protection-settings-for-android-and-ios.md) компании, которые есть у пользователей на их устройствах или пк с [Windows,](protection-settings-for-windows-10-devices.md) защищенных с помощью Microsoft 365. **После удаления данных компании с устройства их нельзя будет восстановить**. 
  
1. Перейдите в Центр администрирования <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>.
    
2. On the left nav, choose **Devices** \> **Manage**.  
  
3. На странице **"Управление"** выберите или найщите пользователя, данные которого вы хотите удалить, и выберите имя. 
    
4. На следующей области выберите устройство или устройства в списке **устройств.** На открываемой области устройств можно сбросить заводские параметры устройства или удалить данные компании в зависимости от типа устройства. 
    
    ![В области "Удаление данных компании" выберите устройство, с которого необходимо удалить данные.](../media/resetorremove.png)
  
5. В области подтверждения выберите **"Подтверждение** \> **закрытия".**
