# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Необходимые условия для защиты данных на устройствах с Microsoft 365 для бизнеса

Эта статья применима к Microsoft 365 бизнес премиум.

Первый шаг к настройке организации с помощью Microsoft 365 для бизнеса — убедиться, что вы можете выполнить необходимые условия.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Требования к настройке организации с помощью Microsoft 365 для бизнеса

- Windows устройства должны работать Windows 7 Профессиональная, Windows 8 Профессиональная или Windows 8.1 Профессиональная.
    
    [Обновление устройств с Windows до Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md)
    
    Если вы работаете Windows 10 Домашняя, необходимо  приобрести Windows 10 Pro. См. [Windows 10 Домашняя Windows 10 Pro](../business-video/upgrade.md) инструкции. 
    
- Удаление устройств из мобильных решений управления (Mobile Iron, AirWatch и так далее). Все люди в вашей организации будут зарегистрированы в Microsoft 365 для управления мобильными устройствами для бизнеса.
    
- Apple iOS 8.0 и более поздние версии.
    
    Google Android 4.0 и более поздней версии (в том числе Samsung KNOX Standard 4.0 и выше). Дополнительные сведения см. в дополнительных [сведениях о устройствах, поддерживаемых Intune.](/mem/intune/fundamentals/supported-devices-browsers)
    
- Если у вас есть Office приложений на компьютерах [](prepare-for-office-client-deployment.md) пользователей, ознакомьтесь с подготовкой к установке Office клиента, чтобы понять действия, которые необходимо предпринять, прежде чем можно настроить Microsoft 365 для бизнеса для установки Office 2016 г. на компьютерах пользователей.
