# <a name="transition-a-microsoft-365-business-csp-subscription"></a>Перевод подписки CSP Microsoft 365 бизнес

Если у вас есть подписка CSP на предварительную версию Microsoft 365 бизнес, то в этом руководстве вы узнаете, как перевести существующую подписку с предварительной версии Microsoft 365 бизнес на общедоступную.

**Процедура перевода подписки с предварительной версии на общедоступную**

1. Войдите в <a href="https://partnercenter.microsoft.com" target="_blank">Центр партнеров</a>.
2. На информационной панели выберите **Клиенты**, а затем найдите и выберите название компании.

    Отобразится перечень подписок выбранной компании.

    ![Подписки клиента в Центре партнеров](../../media/pc_customer_subscriptions_1.png)
    
3. На странице Подписки **компании** выберите **Добавить подписку**.
4. На странице **Новая подписка** выберите **малый** бизнес **и** выберите Microsoft 365 бизнес из списка.
5. Добавьте количество лицензий и выберите **Далее: просмотр**, чтобы просмотреть подписку, а затем выберите **Отправить**.

    ![Просмотр новой подписки на Microsoft 365 бизнес](../../media/pc_customer_reviewnewsubscription.png)

    В разделе **Подписки на основе лицензии** отобразятся пункты **Предварительная версия Microsoft 365 бизнес** и **Microsoft 365 бизнес**. Далее будет приостановлена подписка На просмотр.

6. Выберите **Предварительная версия Microsoft 365 бизнес**.
7. На странице **Microsoft 365 бизнес предварительного** просмотра выберите **Suspended,** чтобы приостановить подписку На просмотр.

    ![Приостановка подписки на предварительную версию Microsoft 365 бизнес](../../media/pc_customer_m365bpreview_suspend.png)

8. Выберите **Отправить** для подтверждения.

    На странице **Подписки** подтвердим, что Microsoft 365 бизнес **предварительный** просмотр показывает **приостановлено.**

    ![Проверка приостановленного состояния подписки на предварительную версию](../../media/pc_customer_m365bpreview_suspend_confirm.png)

9. Кроме того, можно проверить лицензионное соглашение. Для этого выполните следующие действия:
    1. Выберите **Пользователи и лицензии** на странице **Подписки** компании.
    2. На странице **Пользователи и лицензии** выберите пользователя.
    3. На странице пользователя проверьте раздел **Назначение** лицензий и убедитесь, что в нем **Microsoft 365 бизнес**.

        ![Проверка назначения лицензии Microsoft 365 бизнес пользователю](../../media/pc_customer_userslicenses_m365b_validate.png)

## <a name="mobile-device-impacts-upon-subscription-expiration"></a>Влияние истечения срока действия подписки на мобильные устройства

В следующей таблице обобщается влияние политик управления приложениями на мобильные устройства.

|                            | Полная лицензия                      | 60 дней после истечения срока действия подписки          |
|----------------------------|------------------------------------------------|------------------------------------|
| **Удаление рабочих файлов с неактивного устройства** | Рабочие файлы удаляются по прошествии указанного количества дней | Рабочие файлы остаются на личных устройствах пользователей |
| **Принудительное сохранение рабочих файлов пользователей в OneDrive для бизнеса** | Рабочие файлы можно сохранить только в OneDrive для бизнеса | Рабочие файлы можно сохранить в любом месте |
| **Шифрование рабочих файлов** | Рабочие файлы шифруются | Рабочие файлы больше не шифруются.</br> Удаляются политики безопасности и данные Office в приложениях. |
| **Для доступа к приложениям Office требуется PIN-код или отпечаток пальца** | Ограниченный доступ к приложениям | Нет ограничений доступа на уровне приложений |
| **Сброс PIN-кода при ошибке входа** | Ограниченный доступ к приложениям | Нет ограничений доступа на уровне приложений |
| **После простоя приложений Office пользователям требуется повторно выполнить вход** | Требуется вход | Вход не требуется |
| **Запрет доступа к рабочим файлам на устройствах со снятой защитой или с административным доступом** | Файлы работы не могут быть доступны на устройствах jailbroken/rooted | Доступ к рабочим файлам на устройствах со снятой защитой или с административным доступом разрешен |
| **Разрешение пользователям копировать содержимое из приложений Office в личные приложения** | Копирование и вклейка, ограниченная приложениями, доступными в Microsoft 365 подписки | Копирование и вставка доступны во всех приложениях |
