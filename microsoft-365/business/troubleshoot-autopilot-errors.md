# <a name="troubleshoot-autopilot-device-errors"></a>Устранение ошибок, связанных с устройствами AutoPilot

## <a name="device-file-error-messages"></a>Сообщения об ошибках файла устройства

Вот сведения о некоторых ошибках, которые могут возникнуть при работе с файлами устройств AutoPilot в Microsoft 365 бизнес премиум. 
  
|**Код ошибки**|**Исправление для попробовать**|
|:-----|:-----|
|Недопустимый тело запроса  <br/> |Эта ошибка должна происходить редко, если вы видите эту ошибку, повторите операцию еще раз.  <br/> |
|Неправильное значение аппаратного hash-значения для устройства.  <br/> |Если вы видите эту ошибку, это означает, что значение, предоставленное в CSV-файле для аппаратного hash одного устройства, не является правильным. Сначала убедитесь, что значение введите правильно. Если вы считаете, что значение правильное, но эта ошибка все еще происходит, попросите своего поставщика оборудования за помощью.  <br/> |
|Устройство, назначенное другому арендатору  <br/> |Если вы видите эту ошибку, это означает, что значение, предоставленное в CSV-файле для серийного номера или ключа продукта одного или нескольких устройств, является некорректным. Сначала убедитесь, что значение введите правильно. Если вы считаете, что значение правильное, но эта ошибка все еще происходит, попросите своего поставщика оборудования за помощью.  <br/> |
|CSV-файл содержит недопустимый серийный номер или ключ продукта  <br/> |Если вы видите эту ошибку, это означает, что устройство, которое вы пытаетесь зарегистрировать, уже зарегистрировано другой организацией. Чтобы устранить эту ошибку, попросите поставщика оборудования обратиться за помощью.  <br/> |
|Это устройство не поддерживается для установки с помощью AutoPilot  <br/> | Эта ошибка означает, что устройство не соответствует требованиям к развертыванию AutoPilot. Устройства должны отвечать следующим требованиям:  <br/>  Windows 10 версии 1703 или более поздней;  <br/>  Новые устройства, которые не прошли через функцию "Windows при вехах".  <br/> |
|Устройство не найдено  <br/> |Эта ошибка означает, что одно или несколько устройств в CSV-файле не зарегистрированы в вашей организации. Чтобы устранить эту проблему, попросите своего поставщика оборудования обратиться за помощью.  <br/> |
