# <a name="troubleshoot-autopilot-device-errors"></a>Устранение ошибок, связанных с устройствами AutoPilot

## <a name="device-file-error-messages"></a>Сообщения об ошибках файлов устройств

Вот сведения о некоторых ошибках, которые можно увидеть при работе с файлами устройств AutoPilot в Microsoft 365 Business Premium. 
  
|**Код ошибки**|**Исправление, чтобы попытаться**|
|:-----|:-----|
|Недействительное тело запроса  <br/> |Эта ошибка должна происходить редко, если вы видите эту ошибку, повторите операцию.  <br/> |
|Значение хаши оборудования для устройства неправильно.  <br/> |Если вы видите эту ошибку, это означает, что значение, которое вы предоставили в CSV-файле для аппаратного хаши одного устройства, неправильно. Сначала убедитесь, что значение было введите правильно. Если вы считаете, что значение правильное, но эта ошибка все еще происходит, попросите поставщика оборудования о помощи.  <br/> |
|Устройство, назначенное другому клиенту  <br/> |Если вы видите эту ошибку, это означает, что значение, которое вы предоставили в CSV-файле для серийного номера или ключа продукта одного или нескольких устройств, неправильно. Сначала убедитесь, что значение было введите правильно. Если вы считаете, что значение правильное, но эта ошибка все еще происходит, попросите поставщика оборудования о помощи.  <br/> |
|Файл CSV содержит недопустимый серийный номер или ключ продукта  <br/> |Если вы видите эту ошибку, это означает, что устройство, которое вы пытаетесь зарегистрировать, уже зарегистрировано другой организацией. Чтобы устранить эту ошибку, попросите поставщика оборудования о помощи.  <br/> |
|Это устройство не поддерживается для установки с помощью AutoPilot  <br/> | Эта ошибка означает, что устройство не соответствует требованиям развертывания AutoPilot. Устройства должны отвечать следующим требованиям:  <br/>  Windows 10 версии 1703 или более поздней;  <br/>  Новые устройства, которые не прошли через windows вне окна.  <br/> |
|Устройство не найдено  <br/> |Эта ошибка означает, что одно или несколько устройств в вашем CSV-файле не зарегистрированы в организации. Чтобы исправить это, попросите поставщика оборудования о помощи.  <br/> |
