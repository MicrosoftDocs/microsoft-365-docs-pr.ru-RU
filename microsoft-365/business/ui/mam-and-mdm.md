# <a name="difference-between-mdm-and-mam"></a>Разница между MDM и MAM

Microsoft 365 бизнес премиум предлагает несколько способов защиты бизнес-данных. Дополнительные сведения о различных средствах защиты, которые устанавливаются автоматически, и о том, что вы можете настроить самостоятельно для дополнительной защиты бизнеса, см. в обзоре [Microsoft 365](../microsoft-365-business-overview.md) бизнес премиум. Вы также можете настроить политики, которые защищают устройства с Windows 10 и данные на мобильных устройствах.
[Настройка параметров защиты приложений для устройств с Windows 10.](../protection-settings-for-windows-10-devices.md)

## <a name="mobile-device-management-or-mdm"></a>Управление мобильными устройствами или MDM

Microsoft 365 бизнес премиум позволяет настроить политики, которые защищают данные на устройствах с Windows 10. Если устройство находится под управлением мобильных устройств, вы управляете всем устройством и можете сбросить с него данные, а также восстановить заводские настройки. Дополнительные сведения см. в подстроке "Настройка параметров защиты устройств для компьютеров с [Windows 10".](../protection-settings-for-windows-10-pcs.md)

## <a name="mobile-application-management-or-mam"></a>Управление мобильными приложениями или MAM

Управление мобильными приложениями позволяет управлять бизнес-данными на личных устройствах пользователей, таких как iPhone и Android, а также на личных компьютерах Win 10. Вы можете использовать политики управления приложениями, чтобы запретить пользователям копировать бизнес-данные из приложений Office в личные приложения. Вы также можете удалить все данные из приложений Office на их личных устройствах. Дополнительные сведения см. в настройках защиты приложений для устройств с Android или [iOS](../app-protection-settings-for-android-and-ios.md) и настройке параметров защиты приложений [для устройств с Windows 10.](../protection-settings-for-windows-10-devices.md)
