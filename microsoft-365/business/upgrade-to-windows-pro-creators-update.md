## <a name="upgrade-to-windows-10-pro"></a>Переход на Windows 10 Pro
  
Для обновления до Windows 10 Pro у вас есть несколько вариантов. Вы можете выполнить указанные ниже действия.
    
- Установите обновление с [](https://go.microsoft.com/fwlink/?LinkID=836951 ) сайта загрузки программного обеспечения Майкрософт. Выберите этот параметр, если устройство, на которое вы вошли, находится на том же устройстве, что и нужно &ndash; обновить. 

    - На сайте загрузки программного обеспечения нажмите кнопку **"Обновить",** чтобы начать обновление устройства до Windows 10 Pro Creators Update. 
    
- Создайте установщик [](https://go.microsoft.com/fwlink/?LinkID=836960) с помощью средства создания мультимедиа. Выберите этот параметр, чтобы создать установщик &ndash; Windows 10 Pro Creators Update (USB-накопитель или ISO-файл) для установки Windows 10 на компьютере, который отличается от используемого.

    - Ознакомьтесь с инструкциями по использованию средства и созданию установок носителя. 

> [!NOTE]
> Если у вас есть устройства с Windows 7 Pro, Windows 8 Pro или Windows 8.1 Pro, ваша подписка на Microsoft 365 для бизнеса дает право на обновление До Windows Pro 10.
    
См. ["Настройка устройств с Windows для пользователей Microsoft 365 для бизнеса",](set-up-windows-devices.md) чтобы завершить настройку устройств с Windows 10. 
  
См. ["Настройка мобильных устройств для пользователей Microsoft 365](set-up-mobile-devices.md) для бизнеса", чтобы завершить настройку устройств с Android и iOS.
