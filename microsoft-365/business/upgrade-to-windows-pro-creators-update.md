## <a name="upgrade-to-windows-10-pro"></a>Переход на Windows 10 Pro
  
Для обновления до Windows 10 Pro у вас есть несколько вариантов. Вы получаете перечисленные ниже возможности.
    
- Установите обновление с сайта [Загрузка](https://go.microsoft.com/fwlink/?LinkID=836951 ) программного обеспечения Майкрософт Выберите этот параметр, если устройство, в которое вы вошли, находится на том же устройстве, что и необходимо &ndash; обновить. 

    - На сайте загрузки программного обеспечения нажмите **кнопку Обновление сейчас,** чтобы начать обновление устройства до Windows 10 Pro Creators Update. 
    
- Создайте средство установки [](https://go.microsoft.com/fwlink/?LinkID=836960) с помощью средства создания мультимедиа Выберите этот параметр для создания средства установки Windows 10 Pro Creators Update (USB-флеш-накопитель или файл ISO) для установки Windows 10 на компьютере, который отличается от &ndash; используемого.

    - Ознакомьтесь с инструкциями по использованию средства и созданию носителя установки. 

> [!NOTE]
> Если у вас есть устройства Windows под управлением Windows 7 Pro, Windows 8 Pro или Windows 8.1 Pro, ваша подписка microsoft 365 для бизнеса дает вам право на обновление Windows Pro 10.
    
Настройка [устройств Windows для Microsoft 365](set-up-windows-devices.md) для бизнес-пользователей для завершения настройки устройств Windows 10. 
  
Настройка [мобильных устройств для Microsoft 365](set-up-mobile-devices.md) для бизнес-пользователей для завершения настройки устройств Android и iOS.
