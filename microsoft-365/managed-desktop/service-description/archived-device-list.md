# <a name="microsoft-managed-desktop-archived-devices"></a>Архивные устройства Microsoft Managed Desktop

Чтобы обеспечить наилучший пользовательский опыт, Microsoft Managed Desktop в определенный момент отойдут от поддержки устройств. В этой статье перечислены устройства, которые близят к завершению работы в ближайшие несколько лет, которые мы называем "архивными". Если вы начинаете работу с microsoft Managed Desktop с помощью устройств, которые у вас уже есть, необходимо тщательно проверить этот список.

>[!IMPORTANT]
>Вы все еще можете записать эти устройства сегодня, но они будут иметь более короткий срок службы с Microsoft Managed Desktop. Если вы покупаете новые устройства, следует использовать модели из [списка активных устройств.](./device-list.md)

Чтобы быть зарегистрированным в Microsoft Managed Desktop, устройство должно быть одной из следующих моделей и соответствовать перечисленным спецификациям для оперативной памяти, семейства процессоров и дискового пространства. Никакая другая настройка не поддерживается.



>[!NOTE]
>В этой таблице "дата выхода на пенсию" — это дата, на которую эта модель больше не будет поддерживаться Microsoft Managed Desktop. В этот день все зарегистрированные устройства этой модели будут удалены из Microsoft Managed Desktop независимо от того, когда они были зарегистрированы.
