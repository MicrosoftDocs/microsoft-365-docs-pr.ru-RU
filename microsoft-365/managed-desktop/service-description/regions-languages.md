# <a name="microsoft-managed-desktop-supported-regions"></a>компьютеры, управляемые Майкрософт поддерживаемые регионы

В этой статье данная статья содержит сведения о регионах, в которых можно использовать компьютеры, управляемые Майкрософт и где она поддерживается. Дополнительные сведения о языках, поддерживаемых компьютеры, управляемые Майкрософт для пользовательского интерфейса, см. в дополнительных сведениях о том, как [локализовать устройства для пользователей.](../get-started/localization.md)

Вы по-прежнему можете использовать управляемые устройства за пределами этих регионов, не прерывая работу компьютеры, управляемые Майкрософт службы. Например, сотрудник из Великобритании может безопасно работать и получать обновления на управляемом устройстве во время путешествий по Азии, Европе или Южной Америке.

Дополнительные сведения о языках, поддерживаемых компьютеры, управляемые Майкрософт для пользователей и [](../get-started/localization.md)администраторов, см. в этой информации.

Дополнительные сведения о поддержке пользователей с помощью компьютеры, управляемые Майкрософт см. в [компьютеры, управляемые Майкрософт.](support.md)

## <a name="availability-of-the-service"></a>Доступность службы

Организации в следующих странах могут подписаться на компьютеры, управляемые Майкрософт:

- Соединенные Штаты
- Канада (за исключением Квебека)
- Великобритания
- Ирландия
- Бельгия
- Люксембург
- Нидерланды
- Швеция
- Финляндия
- Норвегия
- Дания
- Исландия
- Австралия
- Новая Зеландия
