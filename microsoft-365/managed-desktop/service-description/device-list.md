# <a name="microsoft-managed-desktop-devices"></a>Устройства Microsoft Managed Desktop 

Microsoft Managed Desktop регулярно оценивает устройства, которые будут включены в службу. В этой статье перечислены конкретные устройства, поддерживаемые в настоящее время. Более общее заявление о требованиях к устройству см. в этой [части.](device-requirements.md)

Дополнительные сведения о microsoft Managed Desktop см. в документации и ресурсах [Microsoft Managed Desktop.](https://docs.microsoft.com/microsoft-365/managed-desktop/)

Чтобы быть зарегистрированным в Microsoft Managed Desktop, устройство должно быть одной из следующих моделей и соответствовать перечисленным спецификациям для оперативной памяти, семейства процессоров и дискового пространства. 

Ссылки на устройства здесь можно найти только для ссылки. Если вы хотите заказать устройства, работайте с контактами коммерческих каналов, чтобы убедиться, что вы выбираете правильные конфигурации. В этой таблице датой *архива* является дата, на которую эти модели будут подлежать более короткому сроку службы Microsoft Managed Desktop; любые такие устройства, на которые после этой даты будет влететь, не будут получать полную трехлетнюю поддержку от Microsoft Managed Desktop. Архивные устройства также перечислены в [архивных устройствах Microsoft Managed Desktop.](archived-device-list.md) *Дата выхода* на пенсию — это дата, в которую эта модель больше не будет поддерживаться microsoft Managed Desktop. В этот день все зарегистрированные устройства этой модели будут удалены из Microsoft Managed Desktop независимо от того, когда они были зарегистрированы.

>[!NOTE]
>Устройства могут быть добавлены в эту таблицу в любое время, но мы предоставляем уведомление за 90 дней для любых изменений, влияющих на архив или дату выхода на пенсию. Недавно добавленные устройства отмечены **\*** .
