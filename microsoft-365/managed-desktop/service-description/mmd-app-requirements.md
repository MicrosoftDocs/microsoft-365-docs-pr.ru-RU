# <a name="microsoft-managed-desktop-app-requirements"></a>Требования к классическим приложениям, управляемым Майкрософт

<!--This topic is the target for aka.ms/app-req. This is aka link is used from EA agreement for MMD. do not delete.-->

<!--Application addendum -->
 
Компьютеры, управляемые Майкрософт, требуют, чтобы мы управляли устройствами с помощью определенного подхода, чтобы гарантировать производительность, надежность и работоспособность устройств.


|Область управления  |Подход к компьютеру, управляемому Майкрософт  |
|---------|---------|
|Конфигурация устройства или управление политиками     |  Microsoft Intune       |
|Управление приложениями     | Microsoft Intune и портал компании        |
|Развертывание драйверов     |  Драйверы, включенные в устройство, Windows Update или Intune       |
|Безопасность устройств     | См. ["Безопасность устройств"](security.md#device-security)      |
|Управление идентификацией и доступом     | См. ["Управление удостоверением и доступом"](security.md#identity-and-access-management)        |
|Безопасность сети     | См. ["Безопасность сети"](security.md#network-security)        |
|Информационная безопасность     |  См. ["Информационная безопасность"](security.md#information-security)       |
|Восстановление данных     | OneDrive для бизнеса        |
|Основная производительность     | Приложения Microsoft 365 для предприятий    |
|Браузер     | Microsoft Edge        |




Компьютеры, управляемые Майкрософт, могут отслеживать другое программное обеспечение, запущенное на управляемых устройствах. Если это отрицательно сказывается на управлении устройствами, безопасности, производительности или надежности устройств, может потребоваться запросить исключение из [плана обслуживания.](customizing.md)
