# <a name="overview"></a>Обзор

компьютеры, управляемые Майкрософт — это служба it-as-a-Service (ITaaS) для корпоративных облачных клиентов, предназначенная для обеспечения развертывания и обновления устройств Windows сотрудников. Он также предоставляет управление ИТ-службами и операции, отслеживает безопасность и реагирование на инциденты, а также оказывает поддержку пользователям. Эта документация содержит дополнительные сведения о платформе данных и соблюдении конфиденциальности для компьютеры, управляемые Майкрософт.

## <a name="microsoft-managed-desktop-data-sources-and-purpose"></a>компьютеры, управляемые Майкрософт источников и целей данных

компьютеры, управляемые Майкрософт предоставляет свои услуги корпоративным клиентам и правильно управляет зарегистрированными устройствами клиентов с помощью данных из различных источников. Эти источники, включая Azure Active Directory, Microsoft Intune, Microsoft Windows 10 и Microsoft Defender для конечной точки, предоставляют полное представление устройств, компьютеры, управляемые Майкрософт управляет. Служба также использует эти службы Майкрософт, чтобы компьютеры, управляемые Майкрософт возможности ITaaS:

- [Microsoft Windows 10 Корпоративная](/windows/windows-10/) — для управления опытом установки устройств, управления подключениями к другим службам и операционной поддержки ИТ-специалистов.
- [Windows для бизнеса](/windows/deployment/update/waas-manage-updates-wufb) — Windows 10 Корпоративная диагностические данные для предоставления дополнительных сведений о Windows 10 обновлении. 
- [Microsoft Endpoint Manager](/mem/endpoint-manager-overview) — для управления устройствами и обеспечения безопасности данных.
  - [Microsoft Azure Active Directory](/azure/active-directory/) — для проверки подлинности и идентификации всех учетных записей пользователей. 
  - [Microsoft Intune](/mem/intune/) — для распространения конфигураций устройств, управления устройствами и управления приложениями.
  - [Endpoint Analytics](/mem/analytics/overview) — аналитические сведения об использовании устройств и приложений.
  - [Windows автопилот —](/microsoft-365/windows/windows-autopilot) для обеспечения и развертывания устройств.
  - [Microsoft Defender для конечной точки](/microsoft-365/security/defender-endpoint/) — предоставляет службы безопасности, такие как мониторинг безопасности устройств и данные разведки безопасности.
- [компьютеры, управляемые Майкрософт](https://endpoint.microsoft.com/#home) — данные, предоставляемые клиентом или созданные службой во время работы службы.
- [Microsoft 365 для предприятия](https://www.microsoft.com/en-us/microsoft-365/enterprise/compare-office-365-plans?rtc=1) — для управления Приложения Microsoft 365.
