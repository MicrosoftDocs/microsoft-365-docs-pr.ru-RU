# <a name="work-with-reports"></a>Работа с отчетами

Рабочий стол, управляемый Майкрософт, предоставляет отчеты, которые администраторы ИТ могут использовать для представления различных аспектов заполнения устройств. Вы получите доступ к отчетам на [портале администрирования ](https://aka.ms/mmdportal)в разделе **Создание отчетов**.



## <a name="device-inventory-report"></a>Отчет об инвентаризации устройств

В представлении **устройства** отображается вкладка **Экспорт всех** , в которой [загружается подробный отчет по запасам](device-inventory-report.md).
