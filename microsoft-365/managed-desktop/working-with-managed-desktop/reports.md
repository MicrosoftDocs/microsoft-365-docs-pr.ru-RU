# <a name="work-with-reports"></a>Работать с отчетами

Рабочий стол, управляемый Майкрософт, предоставляет отчеты, которые администраторы ИТ могут использовать для представления различных аспектов заполнения устройств. Вы получите доступ к отчетам на [портале администрирования ](https://aka.ms/mmdportal)в разделе **Создание отчетов**.


## <a name="device-compliance-report"></a>Отчет о соответствии устройств требованиям

В основном представлении отображается процентное отношение управляемых устройств для настольных компьютеров Майкрософт, которые соответствуют политикам соответствия, установленным в Microsoft Intune. Кроме того, можно просмотреть конкретные сведения для всех настольных устройств с управлением Майкрософт или только тех устройств, которые не соответствуют политикам соответствия требованиям и не экспортируют файл с одинаковыми данными.

Дополнительные сведения об [отчете о соответствии устройств](device-compliance-report.md).

## <a name="device-inventory-report"></a>Отчет по инвентаризации устройств

В представлении **устройства** отображается вкладка **Экспорт всех** , в которой [загружается подробный отчет по запасам](device-inventory-report.md).
