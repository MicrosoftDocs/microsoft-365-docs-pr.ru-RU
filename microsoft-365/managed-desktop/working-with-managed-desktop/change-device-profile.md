# <a name="reassign-profiles"></a>Перенамерять профили

Профили [устройств,](../service-description/profiles.md) присвоенные устройству, можно изменить с помощью портала администрирования.

> [!IMPORTANT]
> Изменение профиля устройства стирет все данные на устройстве и приведет к его перезапуску. Убедитесь, что вы выбрали нужные устройства перед началом и сохраняем любые данные, которые необходимо сохранить.

Выбранный профиль устройства будет применен к всем устройствам, выбранным на первом шаге. Чтобы переместить отдельные устройства в различные профили, необходимо повторить этот процесс для каждого профиля устройства. 

1. В Microsoft Endpoint Manager выберите **Устройства**   в левой области. В разделе **Microsoft Managed Desktop** меню выберите  **Устройства**.  
2. Выберите флажки для устройств, которые необходимо изменить. 
3. Выберите **профиль устройства Change;** откроется флайер.
4. Чтобы выбрать новый профиль устройства, используйте выпадающее меню.
5. Выберите **профиль Изменения**.
