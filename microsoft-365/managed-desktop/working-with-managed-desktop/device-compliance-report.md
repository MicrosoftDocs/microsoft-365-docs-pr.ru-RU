# <a name="device-compliance-report"></a>Отчет о соответствии устройств требованиям

В этом представлении отображается процент управляемых устройств для настольных компьютеров Майкрософт, которые соответствуют политикам соответствия, установленным в Microsoft Intune.

![Отчет, в котором отображается процент устройств, которые соответствует политикам](../../media/mmd-device-compliance-percent.png)


Выберите **Просмотреть сведения** , чтобы просмотреть конкретные сведения для всех настольных устройств, управляемых Майкрософт, или только устройств, которые не соответствуют политикам соответствия требованиям. Чтобы переключаться между этими представлениями, используйте меню раскрывающемся меню. Для отображения в этом представлении устройство должно быть физическим компьютером и синхронизировано с Intune за последние 30 дней.

![Область сведений с вкладками для всех устройств или устройств, не соответствующих требованиям. Показывает подробную схему, включая серийный номер, имя устройства, состояние соответствия и другие сведения. Кнопка "Загрузить полные данные" в нижней части страницы.](../../media/mmd-device-compliance-detail.png)

В этом представлении выберите **загружать полные данные** , чтобы экспортировать файл с разделителями запятыми и теми же сведениями. В файле экспорта области, отображаемые как **схема** , отображаются как столбцы.
