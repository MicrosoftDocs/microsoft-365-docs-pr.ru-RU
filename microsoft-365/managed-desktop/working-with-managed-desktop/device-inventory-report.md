# <a name="device-inventory-report"></a>Отчет по инвентаризации устройств

В представлении **устройства** можно выбрать вкладку **Экспорт всех** , чтобы скачать файл с разделителями запятыми, включая следующие сведения:

- серийный номер;
- Имя устройства
- Последний возврат устройства
- Управляется
- Вычислитель
- Модель
- СОВМЕСТИМ
- Лицензия на ОС
- Версия ОС
- Собственность
- Дата регистрации
- Название звонка
- Свободно для хранения в ГБ
- Хранение — всего в ГБ
- Регистрируется с помощью имени участника-пользователя
- Назначенный пользователь
- Состояние
- Возраст (мес.)
- Профиль
- Вход пользователей
