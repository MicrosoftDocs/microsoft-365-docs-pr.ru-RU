# <a name="device-inventory-report"></a>Отчет об инвентаризации устройств

В **представлении Devices** можно выбрать вкладку **"Экспорт** всех", чтобы скачать файл с запятой, включая следующие сведения:

- серийный номер;
- Имя устройства
- Последняя проверка устройства
- Управляется
- Производитель
- Модель
- OS
- Лицензия на ОС
- ВЕРСИЯ ОС
- Собственность
- Дата регистрации
- Имя кольца
- служба хранилища - Бесплатно в ГБ
- служба хранилища - Итого в ГБ
- Регистрация пользователя UPN
- Назначен пользователь
- Статус
- Возраст (месяцы)
- Профиль
- Вход в систему Пользователей
- Версия TPM
- Включена безопасная загрузка
- Тип основного диска
- Общая физическая память
