# <a name="device-inventory-report"></a>Отчет об инвентаризации устройств

В **представлении "Устройства"**  можно выбрать вкладку "Экспортировать все", чтобы скачать файл с запятой, включив следующие сведения:

- серийный номер;
- Имя устройства
- Последний регистрации устройства
- Управляется
- Manufacturer
- Модель
- ОС
- Лицензия НА ОС
- Версия ОС
- Собственность
- Дата регистрации
- Имя звонка
- Хранилище — бесплатно в ГБ
- Хранилище — всего в ГБ
- Зарегистрировано upN пользователя
- Пользователь с назначенной подписью
- Status
- Возраст (месяцы)
- Профиль
- Во время входа в систему пользователи
