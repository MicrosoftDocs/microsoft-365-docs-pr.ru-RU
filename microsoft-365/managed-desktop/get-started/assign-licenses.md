# <a name="assign-licenses"></a>Назначение лицензий

В рамках подготовки к регистрации в компьютеры, управляемые Майкрософт необходимо убедиться, что вы получили необходимые лицензии. Если вы еще не получили лицензии, дополнительные сведения о лицензиях см. в материале о том, какие именно лицензии вам нужны. [](../get-ready/prerequisites.md#more-about-licenses)


Если все лицензии выстроили в очередь, настало время назначить их пользователям. Чтобы назначить лицензии, рекомендуется использовать [](/azure/active-directory/fundamentals/active-directory-licensing-whatis-azure-portal) функцию группового лицензирования Azure Active Directory.

Если у вас есть трудности с назначением лицензии, обратитесь в службу поддержки [администратора.](../working-with-managed-desktop/admin-support.md)
