# <a name="assign-licenses"></a>Назначение лицензий

При подготовке к регистрации на компьютере, управляемом Майкрософт, необходимо убедиться, что вы получили необходимые лицензии. Если вы еще не получили лицензии, дополнительные сведения о лицензиях см. в дополнительных сведениях о необходимых лицензиях. [](../get-ready/prerequisites.md#more-about-licenses)


Если все ваши лицензии выстроили в очередь, пришло время назначить их пользователям. Чтобы назначить лицензии, рекомендуется воспользоваться [](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-licensing-whatis-azure-portal) функцией лицензирования на основе групп в Azure Active Directory.

Если у вас есть трудности с назначением лицензии, обратитесь в службу поддержки [администраторов.](../working-with-managed-desktop/admin-support.md)
