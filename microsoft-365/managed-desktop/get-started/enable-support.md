# <a name="enable-user-support-features"></a>Включить функции поддержки пользователей

Независимо от того, предоставляете ли вы собственную поддержку пользователей или работаете с партнером для ее предоставления, выполните следующие действия, чтобы поставщик поддержки запрашивал повышенный доступ к устройству или при необходимости компьютеры, управляемые Майкрософт проблемы.

1. Если у них еще нет такой учетной записи, пользователям нужна учетная запись в том же домене Azure Active Directory (AAD), что и компьютеры, управляемые Майкрософт устройства.
2. Добавьте учетные записи пользователей из шага 1 в группу безопасности **Roles-Support партнеров** в AAD.
