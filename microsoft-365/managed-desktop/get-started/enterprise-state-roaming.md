# <a name="enable-enterprise-state-roaming"></a>Включение службы Enterprise State Roaming

Для обеспечения наилучшего интерфейса [](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview)работы с настольными компьютерами, управляемыми Майкрософт, в состоянии предприятия в роуминге, что позволяет пользователям безопасно синхронизировать данные параметров пользователей и приложений с облаком. Это означает, что они будут иметь одинаковый опыт независимо от того, на какое устройство с Windows они впишутся. Например, если заменить одно из настольных устройств, управляемых Майкрософт, новым, оно будет выглядеть и вести себя точно так же, как и последнее.

Чтобы включить роуминг состояния предприятия, выполните действия, которые необходимо включить в [Azure Active Directory,](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable)а затем вернись к этой документации.

Если у вас есть какие-либо трудности с роумингом корпоративного состояния, обратитесь в службу поддержки [администраторов.](../working-with-managed-desktop/admin-support.md)
