# <a name="enable-enterprise-state-roaming"></a>Включение службы Enterprise State Roaming

Для лучшего опыта работы с [](/azure/active-directory/devices/enterprise-state-roaming-overview)управляемым рабочим столом Microsoft впустите корпоративный государственный роуминг, который позволяет пользователям безопасно синхронизировать данные параметров пользователей и приложений в облако. Это означает, что они будут иметь одинаковый опыт независимо от того, на какое устройство Windows они впишутся. Например, если вы замените одно из своих устройств Microsoft Managed Desktop новым, оно будет выглядеть и вести себя точно так же, как и последнее.

Чтобы включить корпоративный государственный роуминг, выполните действия в включить государственный роуминг предприятия в [Azure Active Directory,](/azure/active-directory/devices/enterprise-state-roaming-enable)а затем вернись к этой документации.

Если у вас есть какие-либо трудности с корпоративным государственным роумингом, обратитесь в службу [поддержки администрирования.](../working-with-managed-desktop/admin-support.md)
