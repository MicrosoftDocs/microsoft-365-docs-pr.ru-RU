# <a name="order-microsoft-managed-desktop-devices"></a>Заказ настольных устройств, управляемых Майкрософт

Мы рекомендуем работать с одним из наших утвержденных партнеров по устройствам. Для получения дополнительной помощи по настройке связи с устройством вы можете обратиться к контактной записи Майкрософт.

В то же время вы можете [](../service-description/device-list.md) просмотреть список утвержденных в настоящее время устройств, которые поддерживаются для использования с компьютерами, управляемыми Майкрософт.
