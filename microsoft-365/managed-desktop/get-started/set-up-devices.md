# <a name="set-up-microsoft-managed-desktop-devices"></a>Настройка компьютеров, управляемых Майкрософт

На компьютерах, управляемых Майкрософт, можно использовать как новые, так и существующие устройства.

## <a name="to-obtain-new-devices"></a>Получение новых устройств

Мы рекомендуем работать с одним из наших утвержденных партнеров по устройствам. Вы можете обратиться к контакту с учетной записью Майкрософт для получения дополнительной помощи по настройке связи с устройством. Однако по сути процесс выглядит так:

1. Просмотрите список [утвержденных в настоящее время устройств.](../service-description/device-list.md)
2. Заказав устройства, работая с утвержденным партнером по устройствам.
3. После их прибытия сделайте следующее:
    - [Самостоятельная регистрация новых устройств](register-devices-self.md)
    - Работа с партнером для регистрации устройств
4. [Подготовьтесь к использованию](get-started-devices.md) настольных устройств, управляемых Майкрософт

## <a name="to-reuse-existing-devices"></a>Повторное использование существующих устройств

1. Просмотрите список утвержденных [в настоящее](../service-description/device-list.md) время устройств, чтобы определить, какие из ваших устройств утверждены для использования с компьютерами, управляемыми Майкрософт. Если вы повторно будете использовать существующее устройство, вам придется его перезапракть.
2. Сделайте следующее:
    - [Самостоятельная регистрация существующих устройств](register-reused-devices-self.md)
    - Работа с партнером для регистрации устройств
3. [Подготовьтесь к использованию](get-started-devices.md) настольных устройств, управляемых Майкрософт
