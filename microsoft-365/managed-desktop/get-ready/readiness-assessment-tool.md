# <a name="readiness-assessment-tools"></a>Средства оценки готовности

При регистрации в компьютеры, управляемые Майкрософт при регистрации необходимо заранее установить параметры и другие параметры, а также выполнить определенные требования к устройству и сети. Один инструмент, доступный через компьютеры, управляемые Майкрософт администратора, проверяет параметры, связанные с управлением. Другой инструмент, который можно загрузить, проверяет требования к отдельным устройствам и сетевые параметры. Эти средства можно использовать для проверки этих параметров и получения подробных действий по исправлению ошибок.

## <a name="downloadable-readiness-assessment-checker-for-devices-and-network"></a>Проверка готовности к загрузке для устройств и сети

Подробные сведения об использовании проверяемой проверки готовности для скачивания см. в публикации [Downloadable readiness assessment checker.](readiness-assessment-downloadable.md)

## <a name="online-readiness-assessment-tool-for-management-settings"></a>Средство оценки готовности в Интернете для параметров управления

[Он-лайн](https://aka.ms/mmdart) инструмент проверяет параметры Microsoft Endpoint Manager (в частности, Microsoft Intune), Azure Active Directory (Azure AD) и Microsoft 365, чтобы убедиться, что они будут работать с компьютеры, управляемые Майкрософт. компьютеры, управляемые Майкрософт сохраняет данные, связанные с этими проверками, в течение 12 месяцев после последнего запуска проверки в вашей организации Azure AD (клиент). Через 12 месяцев мы сохраняем его в деидентифицированной форме. Вы можете удалить собранные данные.

Любой человек с ролью глобального читателя или администратора Intune сможет запустить этот инструмент, [](readiness-assessment-fix.md#multifactor-authentication) но для двух проверок[(политики](readiness-assessment-fix.md#conditional-access-policies) условного доступа и многофакторная проверка подлинности требуются дополнительные разрешения.
 
Средство оценки проверяет эти элементы:

## <a name="microsoft-intune-settings"></a>Microsoft Intune параметров

|Чек  |Описание  |
|---------|---------|
|Профиль развертывания автопилота     | Проверяет, что назначение профиля развертывания автопилота распространяется не на  все устройства (профиль не должен быть назначен на компьютеры, управляемые Майкрософт устройствах.)       |
|Соединители сертификатов     | Проверяет состояние соединители сертификатов, чтобы убедиться, что они активны   |
|Условный доступ     | Проверяет, что политики условного доступа не назначены всем пользователям (политики  условного доступа не должны быть назначены компьютеры, управляемые Майкрософт учетным записям службы.)    |
|Политики соответствия требованиям устройств     | Проверяет, что политики соответствия intune не назначены всем  пользователям (политики не должны быть назначены любым компьютеры, управляемые Майкрософт устройствам.)    |
|Профили конфигурации устройств     | Подтверждает, что профили конфигурации не назначены всем пользователям или  всем устройствам (профили конфигурации не должны быть назначены любым компьютеры, управляемые Майкрософт устройствам.)     |
|Ограничения типа устройства     | Проверки Windows 10 устройств в организации разрешены для регистрации в Intune        |
|Страница Состояние регистрации     | Подтверждает, что страница состояние регистрации не включена      |
|Регистрация intune     | Проверяет, Windows 10 устройства в вашей организации Azure AD автоматически регистрируются в Intune         |
|Microsoft Store для бизнеса     | Подтверждает, что Microsoft Store для бизнеса включена и синхронизирована с Intune        |
|Многофакторная проверка подлинности | Проверяет, что многофакторная проверка подлинности не применяется компьютеры, управляемые Майкрософт учетных записей службы.
|Сценарии PowerShell     | Проверки Windows PowerShell сценариев не  назначены таким образом, чтобы они были нацелены на компьютеры, управляемые Майкрософт устройствах    |
|Регион     | Проверяет, поддерживается ли ваш регион компьютеры, управляемые Майкрософт        |
|Базовые показатели безопасности     | Проверяет, что базовый профиль безопасности не ориентирован на всех пользователей  или все устройства (базовые политики безопасности не должны компьютеры, управляемые Майкрософт устройств.)       |
|Windows приложения     | Просмотрите, какие приложения необходимо назначить компьютеры, управляемые Майкрософт устройствам      |
|Windows Hello для бизнеса     | Проверка включения Windows Hello для бизнеса        |
|Windows 10 кольцо обновления     | Проверяет, что политика "кольцо Windows 10 обновления" intune не ориентирована на всех пользователей  или все устройства (политика не должна быть нацелена на компьютеры, управляемые Майкрософт устройства.)     |


## <a name="azure-active-directory-settings"></a>Azure Active Directory параметров

|Чек  |Описание  |
|---------|---------|
|Подписки на "Ad hoc" для Enterprise государственного роуминга     | Советы по проверке параметра, который (если установлено на "ложный") может запретить Enterprise роуминге штата от правильной работы  |
|Enterprise State Roaming     | Советы по проверке включения Enterprise государственного роуминга       |
|Лицензии     | Проверка получения необходимых [лицензий](prerequisites.md#more-about-licenses)         |
|Многофакторная проверка подлинности     | Проверка того, что многофакторная проверка подлинности не применяется для всех пользователей (проверка подлинности multifactor не должна случайно применяться компьютеры, управляемые Майкрософт учетных записей службы.)|
|Имена учетных записей безопасности   | Проверка того, что имена пользователей не конфликтуются с компьютеры, управляемые Майкрософт для собственного использования        |
|Роли администратора безопасности     | Подтверждает, что пользователям с ролями "Считыватель безопасности", "Оператор безопасности" или "Глобальный читатель" назначены эти роли в Microsoft Defender for Endpoint.         |
|Параметры безопасности по умолчанию | Проверяет, включены ли в вашей организации Azure AD по умолчанию Azure Active Directory |
|Самостоятельный сброс пароля     | Подтверждает, что перезагрузка пароля самообслуживляется        |
|Стандартная роль пользователя     | Проверяет, что пользователи являются стандартными пользователями и не имеют прав локального администратора         |


## <a name="microsoft-365-apps-for-enterprise-settings"></a>Приложения Microsoft 365 для предприятий параметров

|Чек  |Описание  |
|---------|---------|
|OneDrive для бизнеса     | Проверяет, используется ли OneDrive для бизнеса неподтверченные параметры.        |


Для каждой проверки средство будет сообщать об одном из четырех возможных результатов:


|Результат  |Смысл  |
|---------|---------|
|Ready     | До завершения регистрации не требуется никаких действий.        |
|Предупреждение    | Следуйте шагам в инструменте для наилучшего опыта с регистрацией и для пользователей. Вы *можете* завершить регистрацию, но перед развертыванием первого устройства необходимо устранить эти проблемы.        |
|Не готов | *Если вы* не исправите эти проблемы, регистрация не состоится. Выполните действия в инструменте для их устранения.        |
|Error | Используемая вами роль Active Director (AD) Azure не имеет достаточного разрешения для запуска этой проверки. |
