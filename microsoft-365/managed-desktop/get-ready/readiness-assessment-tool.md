# <a name="readiness-assessment-tool"></a>Средство оценки готовности

Для обеспечения оптимальной работы при регистрации на настольном компьютере, управляемом Майкрософт, существует ряд параметров и других параметров, которые необходимо настроить заранее. Это средство можно использовать для проверки этих параметров и получения подробных инструкций по исправлению неправильного.

Средство проверяет параметры в диспетчере конечных точек Майкрософт (в частности, Microsoft Intune), Azure Active Directory (Azure AD) и Microsoft 365, чтобы убедиться, что они будут работать с настольным компьютером, управляемым корпорацией Майкрософт. На рабочем столе корпорации Майкрософт хранятся данные, связанные с этими проверками, в течение 12 месяцев после последнего запуска проверки в Организации Azure AD (клиенте). По истечении 12 месяцев он сохраняется в неопознанной форме.  Вы можете удалить собранные данные.

Любой пользователь, имеющий по крайней мере роль администратора Intune, сможет запускать это средство, но две проверки ([политики условного доступа](readiness-assessment-fix.md#conditional-access-policies) и [многофакторной проверки подлинности](readiness-assessment-fix.md#multifactor-authentication) требуют дополнительных разрешений).
 
Средство оценки проверяет следующие элементы:

## <a name="microsoft-intune-settings"></a>Параметры Microsoft Intune

|Проверка  |Описание  |
|---------|---------|
|Профиль развертывания для автопилота     | Проверка того, что назначение профиля развертывания автопилота не применимо ко всем устройствам (профиль *не* должен быть назначен ни одному из компьютеров, управляемых Майкрософт.)       |
|Соединители сертификатов     | Проверяет состояние соединителей сертификатов, чтобы убедиться, что они активны   |
|условный доступ;     | Проверяет, что политики условного доступа не назначены всем пользователям (политики условного доступа *не* должны назначаться учетным записям службы, управляемой корпорацией Майкрософт).    |
|Политики соответствия требованиям устройств     | Проверка того, что политики соответствия Intune не назначены всем пользователям (политики *не* должны быть назначены для настольных устройств, управляемых корпорацией Майкрософт).    |
|Профили конфигурации устройств     | Проверяет, что профили конфигурации не назначены всем пользователям или всем устройствам (профили конфигурации *не* должны быть назначены каким-либо управляемым устройствам Майкрософт).     |
|Ограничения на типы устройств     | Проверяет, разрешена ли регистрация устройств с Windows 10 в Организации в Intune        |
|Страница состояния регистрации     | Проверяет, отключена ли страница состояния регистрации      |
|Регистрация в Intune     | Проверяет, что устройства Windows 10 в вашей организации Azure AD автоматически регистрируются в Intune         |
|Microsoft Store для бизнеса     | Подтверждает, что Microsoft Store для бизнеса включен и синхронизирован с Intune        |
|Многофакторная проверка подлинности | Проверяет, не применяется ли многофакторная проверка подлинности для учетных записей служб, управляемых рабочими столами Майкрософт.
|Сценарии PowerShell     | Проверка того, что сценарии Windows PowerShell *не* назначены таким образом, что будут целевыми устройствами для настольных компьютеров Майкрософт    |
|Регион     | Проверка поддержки вашего региона на настольном компьютере, управляемом Майкрософт        |
|Базовые показатели безопасности     | Проверка того, что базовый профиль безопасности не предназначен для всех пользователей или всех устройств (базовые политики безопасности *не* должны направляться на настольных компьютерах, управляемых корпорацией Майкрософт).       |
|Приложения для Windows     | Просмотр приложений, которые необходимо назначить управляемым устройствам для настольных компьютеров Майкрософт      |
|Windows Hello для бизнеса     | Проверяет, включена ли функция Windows Hello для бизнеса        |
|Windows 10 — обновление звонков     | Проверяет, что политика "Windows 10 Update Ring Ring" в Intune не предназначена для всех пользователей или устройств (политика *не должна быть* нацелена на настольные устройства, управляемые Майкрософт).     |


## <a name="azure-active-directory-settings"></a>Параметры Azure Active Directory

|Проверка  |Описание  |
|---------|---------|
|Подписки "ad hoc" для роуминга состояния предприятия     | Показывает, как проверить, что параметр, для которого задано значение "false"), может препятствовать правильной работе перемещения корпоративных состояний.  |
|Enterprise State Roaming     | Показывает, как проверить, включен ли роуминг состояния предприятия.       |
|Лицензии     | Проверка того, что вы получили необходимые [лицензии](prerequisites.md#more-about-licenses)         |
|Многофакторная проверка подлинности     | Проверяет, что многофакторная проверка подлинности не применяется ко всем пользователям (многофакторная проверка подлинности не должна быть случайно применена к управляемым учетным записям служб Microsoft Desktop.)|
|Имена учетных записей безопасности   | Проверка того, что имена пользователей не конфликтуют с теми, которые зарезервированы на рабочем столе Майкрософт для собственного использования        |
|Роли администраторов безопасности     | Подтверждает, что роли в защитнике Майкрософт для конечной точки были назначены пользователям с помощью средства чтения безопасности, оператора безопасности или глобальных ролей читателей         |
|Параметры безопасности по умолчанию | Проверка наличия в Организации Azure AD параметров безопасности по умолчанию, включенных в Azure Active Directory |
|Самостоятельный сброс пароля     | Подтверждает, что включена функция самостоятельного сброса паролей        |
|Роль обычного пользователя     | Проверяет, являются ли пользователи стандартными пользователями и не имеют прав локального администратора.         |


## <a name="microsoft-365-apps-for-enterprise-settings"></a>Приложения Microsoft 365 для корпоративных параметров

|Проверка  |Описание  |
|---------|---------|
|OneDrive для бизнеса     | Проверяет, используются ли неподдерживаемые параметры в OneDrive для бизнеса.        |


Для каждой проверки средство сообщит один из четырех возможных результатов:


|Результат  |Смысл  |
|---------|---------|
|Ready     | Перед регистрацией не требуется никаких действий.        |
|Рекомендация    | Выполните действия, описанные в этом инструменте, чтобы получить лучшие сведения о возможностях регистрации и для пользователей. Вы *можете* выполнить регистрацию, но вам необходимо устранить эти проблемы перед развертыванием первого устройства.        |
|Не готов | Если вы не устраните эти проблемы, *Регистрация завершится с ошибками* . Выполните действия, описанные в средстве, чтобы устранить их.        |
|Error | Используемая роль Azure Active Directory (AD) не обладает разрешениями, достаточными для запуска этой проверки. |

## <a name="after-enrollment"></a>После регистрации

После того как вы закончите регистрацию на рабочем столе Майкрософт, не забудьте вернуться и скорректировать определенные параметры Intune и Azure AD. Дополнительные сведения см. в разделе [Настройка параметров после регистрации](../get-started/conditional-access.md).
