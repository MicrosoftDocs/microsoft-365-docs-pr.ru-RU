# <a name="learn-about-document-understanding-models-through-a-sample-model"></a>Сведения об использовании моделей в документе с помощью примера модели

Microsoft SharePoint Синтекс предоставляет примерную модель, которую можно использовать для проверки, чтобы лучше понять, как создавать собственные модели. Пример модели также позволяет исследовать компоненты модели, такие как классификатор, средства извлечения и объяснения. Кроме того, можно использовать примеры файлов для обучения модели.

## <a name="import-the-sample-model"></a>Импорт образца модели

Чтобы получить доступ к образцу модели, необходимо сначала импортировать модель в центр управления контентом.

1. В центре управления контентом выберите **модели** , чтобы просмотреть список моделей.</br>
2. На странице " **модели** " выберите пункт **Импорт образца модели**.</br>

    ![Импорт образца модели](../media/content-understanding/import-sample-model.png) </br>

3. Найдите образец модели под названием *бенефитсчанженотице. классификатор* , который отображается в списке моделей.</br>

    ![Пример модели](../media/content-understanding/sample-model.png) </br>

4. Выберите *бенефитсчанженотице. классификатор*, чтобы открыть домашнюю страницу модели.</br>
  
     ![Пример домашней страницы](../media/content-understanding/sample-home-page.png)

5. На домашней странице модели внимательно изучите модель, чтобы узнать, как она была создана.
 
- Просмотр образцов файлов, используемых для обучения модели.
- Более подробно изучите средства извлечения объектов модели, чтобы увидеть, как учебная модель настроила объяснения.

   ![Примеры средств извлечения моделей](../media/content-understanding/entity-extractors.png)
