# <a name="create-tag-groups"></a>Создание групп тегов

Прежде чем вы или другие люди смогут пометить документы в наборе отзывов, необходимо создать теги. Это можно сделать, создав группы тегов, содержащие детские теги. Теги отображаются на панели тегов при просмотре документов в наборе отзывов.

Чтобы создать группу тегов:

1.  В наборе обзоров нажмите **кнопку Управление набором обзоров**.

    ![Нажмите кнопку Управление набором обзоров](../media/ED-managews.png)

2.  В **плитке Теги** нажмите **кнопку Управление тегами**.

    ![Нажмите кнопку Управление тегами в плитке Теги](../media/ED-managetags.png)

После внутри управления тегами можно создать теги, чтобы соответствовать требованиям вашего дела. Вы можете начать с создания группы тегов.

1.  Нажмите **кнопку Добавить раздел**.

    ![Добавление группы тегов](../media/ED-addtagsection.png)

    Экран предварительного просмотра будет обновляться при экономии, чтобы можно было просмотреть панель тегов, не закрывая экран управления тегами и возвращаясь к рабочему набору.

2. Введите название и необязательное описание. 

3. Чтобы создать тег в этой группе, щелкните эллипсы в новой группе тегов, чтобы создать теги в этом новом разделе.
    
    ![Создание тегов в группе тегов](../media/ED-createtag.png)

   - **Теги параметра** заставляют пользователей выбирать один тег из группы тегов.
   
   - **Теги check box** позволят пользователям выбирать любое сочетание тегов. |

## <a name="nested-tags"></a>Вложенные теги

Чтобы вложить теги, щелкните эллипсы рядом с тегом, а затем выберите новый тег для добавления.

![Вложенные теги](../media/ED-tagnesting.png)
