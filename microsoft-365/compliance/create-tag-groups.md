# <a name="create-tag-groups"></a>Создание групп тегов

Прежде чем вы или другие люди сможете пометить документы в наборе для проверки, необходимо создать теги. Это можно сделать, создав группы тегов, содержащие child tags. Теги отображаются на панели тегов при просмотре документов в наборе для проверки.

Чтобы создать группу тегов:

1.  В наборе для проверки щелкните **"Управление набором для проверки".**

    ![Click Manage review set](../media/ED-managews.png)

2.  На **плитке "Теги"** щелкните **"Управление тегами".**

    ![Щелкните "Управление тегами" на плитке "Теги"](../media/ED-managetags.png)

После внутри управления тегами можно создать теги, чтобы соответствовать требованиям вашего дела. Для начала создайте группу тегов.

1.  Нажмите **кнопку "Добавить"** в разделе .

    ![Добавление группы тегов](../media/ED-addtagsection.png)

    Экран предварительного просмотра будет обновлен при сохранения, чтобы можно было просмотреть панель тегов, не закрывая экран управления тегами и возвращаясь к рабочему набору.

2. Введите название и необязательное описание. 

3. Чтобы создать тег в этой группе, щелкните многолинейные кнопки в новой группе тегов, чтобы создать теги в этом новом разделе.
    
    ![Создание тегов в группе тегов](../media/ED-createtag.png)

   - **Теги вариантов** будут принудительно выбирать один тег из группы тегов.
   
   - **Теги check box** позволяют пользователям выбирать любое сочетание тегов. |

## <a name="nested-tags"></a>Вложенные теги

Чтобы вложить теги, щелкните многолинейки рядом с тегом, а затем выберите новый тег, который нужно добавить.

![Вложенные теги](../media/ED-tagnesting.png)
