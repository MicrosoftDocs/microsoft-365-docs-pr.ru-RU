# <a name="configure-and-view-alerts-for-dlp-polices-preview"></a>Настройка и Просмотр оповещений для политик защиты от потери данных (Предварительная версия)

В этой статье показано, как определить расширенные политики оповещений, связанные с политиками защиты от потери данных (DLP). Вы узнаете, как использовать новую панель мониторинга управления оповещениями DLP в [центре соответствия требованиям Microsoft 365](https://compliance.microsoft.com/) для просмотра оповещений, событий и связанных метаданных о нарушениях политики DLP.

## <a name="features"></a>Возможности

В предварительной версии доступны следующие функции:

-   **Панель мониторинга управления оповещениями DLP**: в [центре соответствия требованиям Microsoft 365](https://compliance.microsoft.com/)эта панель мониторинга отображает оповещения для политик защиты от потери данных, которые применяются к следующим рабочим нагрузкам:

    -   Exchange
    -   SharePoint;
    -   OneDrive;
    -   Teams
    -   Устройства
-   **Расширенные параметры настройки оповещений**: эти параметры являются частью процесса создания политики защиты от потери данных. Используйте их для создания расширенных конфигураций оповещений. Вы можете создать оповещение с одним событием или сводное оповещение на основе количества событий или размера потерянных данных.

## <a name="before-you-begin"></a>Прежде чем начать

Прежде чем приступать к работе, убедитесь, что у вас есть необходимые компоненты:

-   Лицензирование панели управления оповещениями DLP
-   Лицензирование параметров конфигурации оповещений
-   Роли

### <a name="licensing-for-the-dlp-alert-management-dashboard"></a>Лицензирование для панели мониторинга управления оповещениями DLP

Все подходящие клиенты для Office 365 DLP могут получить доступ к новой панели мониторинга управления оповещениями DLP. Чтобы приступить к работе, вам нужно иметь право на Office 365 DLP для Exchange Online, SharePoint Online и OneDrive для бизнеса. Для получения дополнительных сведений о требованиях лицензирования для Office 365 DLP, Узнайте, [какие лицензии предоставляют пользователю права на преимущества от службы?](https://docs.microsoft.com/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance#which-licenses-provide-the-rights-for-a-user-to-benefit-from-the-service-16).

Пользователи, участвующие в общедоступной предварительной версии [DLP DLP](https://docs.microsoft.com/microsoft-365/compliance/endpoint-dlp-learn-about?view=o365-worldwide) или имеющие соответствующие права для [группы DLP](https://docs.microsoft.com/microsoft-365/compliance/dlp-microsoft-teams?view=o365-worldwide) , увидят оповещения политики DLP конечной точки и оповещения политики DLP в панели мониторинга управления оповещениями DLP.

### <a name="licensing-for-alert-configuration-options"></a>Лицензирование параметров конфигурации оповещений

-   **Конфигурация оповещений с одним событием**: Организации с подпиской E1, F1 или G1 или с подпиской E3 или G3 могут создавать политики оповещений, только когда оповещение инициируется каждый раз при возникновении какого-либо действия.
-   **Сводная конфигурация оповещений**: чтобы настроить статистические политики оповещений на основе порогового значения, необходимо иметь любую из следующих конфигураций:
    -   Подписка "а" или "G5"
    -   Подписка E1, F1 или G1 либо подписка E3 или G3, включающая один из следующих компонентов:
        -   Office 365 Advanced Threat Protection (план 2)
        -   Соответствие требованиям Microsoft 365 E5
        -   Лицензия на обнаружение электронных данных и аудит для надстройки Microsoft 365

### <a name="roles"></a>Роли

Если вы хотите просмотреть панель мониторинга управления оповещениями DLP или изменить параметры настройки оповещений в политике защиты от потери данных, необходимо быть участником одной из этих групп ролей:

-   Администратор соответствия
-   Администратор данных соответствия требованиям
-   Администратор безопасности
-   Оператор безопасности
-   Читатель сведений о безопасности

Для доступа к панели мониторинга управления оповещениями DLP необходима роль "Управление оповещениями" и любая из следующих ролей:

-   Управление соответствием DLP
-   View-Only управления соответствием DLP

## <a name="alert-configuration-experience"></a>Настройка оповещений

Если вы можете использовать [Объединенные параметры настройки оповещений](#licensing-for-alert-configuration-options), то вы увидите следующие параметры, встроенные в интерфейс создания политики DLP.

:::image type="content" source="../media/incident-reports-options-aggregated-alerts.png" alt-text="Снимок экрана, на котором показаны параметры отчетов об инцидентах для пользователей, которые могут быть объединены с параметрами конфигурации оповещений." border="false":::

Вы можете использовать эти параметры настройки оповещений, чтобы настроить параметр, определяющий частоту, с которым может происходить сравнение правила защиты от потери данных перед срабатыванием оповещения. Эта конфигурация позволяет настроить политику для создания оповещения каждый раз, когда действие соответствует условиям политики или при превышении определенного порогового значения, на основе количества действий или объема данных перехвачена.

Если вы можете использовать [Параметры конфигурации оповещений с одним событием](#licensing-for-alert-configuration-options), то вы увидите следующий параметр настройки оповещений в интерфейсе создания политики DLP. Используйте этот параметр, чтобы создать оповещение, которое вызывается каждый раз при соотнесении правила защиты от потери данных из-за активности пользователя.

:::image type="content" source="../media/incident-reports-options-single-event-alerts.png" alt-text="Снимок экрана, на котором показаны параметры отчетов об инцидентах для пользователей, которые могут быть объединены с параметрами конфигурации оповещений." border="false":::

1.  В [центре соответствия требованиям Microsoft 365](https://www.compliance.microsoft.com)перейдите к разделу **Защита от потери данных**.

2.  Перейдите на вкладку **оповещения** , чтобы просмотреть панель мониторинга оповещений о защите от потери данных.

    -   Выберите Filters (фильтры), чтобы уточнить список оповещений. Выберите элемент **Настройка столбцов** , чтобы отобразить список свойств, которые вы хотите просмотреть. Вы также можете сортировать оповещения по возрастанию или убыванию в любом столбце.
    -   Выберите оповещение, чтобы просмотреть сведения:
