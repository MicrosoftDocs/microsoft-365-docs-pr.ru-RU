# <a name="create-a-litigation-hold"></a>Создание удержания для судебного разбирательства

Вы можете разместить почтовый ящик в удержании судебного разбирательства, чтобы сохранить все содержимое почтовых ящиков, включая удаленные элементы и оригинальные версии измененных элементов. При найме почтового ящика пользователя на удержание судебного разбирательства содержимое архивного почтового ящика пользователя (если включено) также сохраняется. При создании удержания можно указать продолжительность удержания (также называемую удержанием на основе времени), чтобы удаленные и измененные элементы сохраняли в течение указанного периода и затем окончательно удалялись из почтового ящика.  Или вы можете просто хранить контент на неопределенный срок (называется *бесконечным* удержанием) или до тех пор, пока не будет удалено удержание судебного разбирательства. Если указать период длительности удержания, он рассчитывается с даты, когда будет получено сообщение или будет создан элемент почтового ящика. 
  
Вот что происходит при создании судебного удержания.
  
- Элементы, которые постоянно удаляются пользователем, сохраняются в папке "Извлекаемые элементы" в почтовом ящике пользователя на время удержания.
    
- Элементы, извлекаемые пользователем из папки "Извлекаемые элементы", сохраняются на время удержания.
    
- Квота хранилища для папки "Извлекаемые элементы" увеличивается с 30 ГБ до 110 ГБ.
    
- Элементы в основных и архивных почтовых ящиках пользователя сохраняются
    
## <a name="assign-an-exchange-online-plan-2-license"></a>Назначение лицензии Exchange Online Plan 2

- Чтобы разместить почтовый ящик Exchange Online на удержание судебного разбирательства, ему должна быть назначена лицензия Exchange Online Plan 2. Если почтовому ящику назначена лицензия Exchange Online Plan 1, необходимо назначить ему отдельную лицензию архивация на базе Exchange Online для ее удержания.
    

## <a name="place-a-mailbox-on-litigation-hold"></a>Перевод почтового ящика в режим хранения для судебного разбирательства

Ниже 1. Действия, необходимые для того, чтобы разместить почтовый ящик в удержании судебного разбирательства с помощью центра администрирования Exchange.

1. Перейдите [https://outlook.office.com/ecp](https://outlook.office.com/ecp) и войдите в свою учетную запись глобального администратора.

2. Щелкните **Получатели > почтовые ящики в** левой области навигации.

3. Выберите почтовый ящик, который необходимо разместить в удержании судебного разбирательства, а затем нажмите **кнопку Изменить**.

4. На странице свойств почтового ящика щелкните **Функции почтового ящика**.
    
5. В разделе **Хранение для судебного разбирательства: отключено** нажмите кнопку **Включить**, чтобы применить к почтовому ящику хранение для судебного разбирательства.
    
6. На странице **Удержание судебного разбирательства** введите следующие необязательные сведения: 
    
    - **Длительность хранения (дни)** для судебного разбирательства . Используйте это поле для создания удержания на основе времени и укажите, как долго находятся элементы почтовых ящиков при размещении почтового ящика в удержании судебного разбирательства. Этот срок рассчитывается с даты получения или создания элемента почтового ящика. По истечении срока действия удержания для определенного элемента этот элемент больше не будет сохраняться. Если оставить это поле пустым, элементы сохраняются на неопределенный срок или до удаления удержания. Длительность указывается в днях.
    
    - **Примечание.** Используйте это поле, чтобы сообщить пользователю, что его почтовый ящик находится в удержании судебного разбирательства. Примечание появится на странице Сведения об учетной записи в почтовом ящике пользователя, если они используют Outlook 2010 или более поздней. Чтобы получить доступ к этой странице, пользователи могут **щелкнуть Файл в** Outlook.
    
    - **URL-адрес** . Используйте это поле, чтобы направить пользователя на веб-сайт, чтобы получить дополнительные сведения о удержании судебного разбирательства. Этот URL-адрес отображается на странице Сведения о учетной записи в почтовом ящике пользователя, если он использует Outlook 2010 или более поздней версии. Чтобы получить доступ к этой странице, пользователи могут щелкнуть **Файл** в Outlook..

7. Нажмите **кнопку Сохранить** на странице **Удержание** судебного разбирательства, а затем нажмите **сохранить** на странице свойства почтовых ящиков.

### <a name="create-a-litigation-hold-using-powershell"></a>Создание судебного удержания с помощью PowerShell

Вы также можете создать удержание для судебного разбирательства, запуская следующую команду [в Exchange Online PowerShell:](/powershell/exchange/connect-to-exchange-online-powershell)

```powershell
Set-Mailbox <username> -LitigationHoldEnabled $true
```

Предыдущая команда сохраняет элементы бесконечно, так как продолжительность удержания не указана. Чтобы создать удержание на основе времени, используя следующую команду:

```powershell
Set-Mailbox <username> -LitigationHoldEnabled $true -LitigationHoldDuration <number of days>
```

Дополнительные сведения см. в статье [Set-Mailbox](/powershell/module/exchange/set-mailbox).

## <a name="how-does-litigation-hold-work"></a>Как работает хранение для судебного разбирательства?

Если почтовый ящик не на хранении, после удаления с помощью клавиш Shift+Delete или удаления из папки "Удаленные" элемент перемещается в подпапку удаления в папке "Элементы с возможностью восстановления". По истечении срока хранения, заданного политикой удаления (тег хранения, для которого настроено действие хранения), элементы также перемещаются в подпапку удаления. Когда пользователь удаляет элемент в папке "Элементы с возможностью восстановления" или истекает срок хранения удаленного элемента, он перемещается в подпапку очистки и помечается для окончательного удаления. Он будет удален из Exchange при следующей обработке почтового ящика помощником для управляемых папок.

Если почтовый ящик находится на хранении для судебного разбирательства, элементы в подпапке очистки хранятся в течение указанного срока. Срок хранения рассчитывается от даты получения или создания элемента. По истечении срока хранения элемент в подпапке очистки помечается для окончательного удаления и удаляется из Exchange при следующей обработке почтового ящика помощником для управляемых папок. Если почтовый ящик помещен на бессрочное хранение, элементы никогда не удаляются из подпапки очистки.
