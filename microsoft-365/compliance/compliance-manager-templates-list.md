# <a name="compliance-manager-templates-list"></a>Список шаблонов диспетчера соответствия требованиям

**В этой статье:** Просмотрите полный список **шаблонов** , доступных для создания оценок в диспетчере соответствия требованиям.

> [!IMPORTANT]
> Шаблоны оценки, доступные для Организации, зависят от лицензионного соглашения. [Просмотрите подробные сведения](https://go.microsoft.com/fwlink/?linkid=2132371).

## <a name="overview"></a>Обзор

[Диспетчер соответствия требованиям (Майкрософт](compliance-manager.md) ) предоставляет исчерпывающий набор шаблонов для создания оценок. Эти шаблоны могут помочь Организации в соблюдении требований национальной, региональной и отрасли, регулирующих сбор и использование данных.

Шаблоны добавляются в Диспетчер соответствия требованиям в соответствии с новыми законами и нормативами. Диспетчер соответствия требованиям обновляет свои шаблоны при изменении соответствующих законов или нормативных актов. Узнайте больше о том [, как проверка и прием обновлений](compliance-manager-assessments.md#accepting-updates-to-assessments).

Подробное руководство по [работе с шаблонами](compliance-manager-templates.md) для создания оценок.

## <a name="list-of-templates-and-where-to-find-them"></a>Список шаблонов и место их поиска

Ниже приведен полный список шаблонов в диспетчере соответствия требованиям. Каждая страница сведений о шаблоне содержит сведения о регламентировании или стандартном стандарте, к которому применяется. Ссылки в приведенных ниже названиях шаблонов относятся к связанной документации о стандартном, нормативном или закону.

**Где можно найти шаблоны**

В диспетчере соответствия требованиям перейдите на страницу " **шаблоны оценки** ". Вы увидите список всех шаблонов, доступных в Организации.

- **Включенные шаблоны** — это шаблоны, входящие в состав лицензионного соглашения Организации.
 - В **расширенных шаблонах** отображаются дополнительные шаблоны, которые может выбрать организация (обратитесь к [условиям обслуживания](https://go.microsoft.com/fwlink/?linkid=2132371)).

Узнайте больше о [том, как просматривать шаблоны и управлять ими](compliance-manager-templates.md#viewing-and-managing-templates-from-the-assessment-templates-page).
