# <a name="compliance-manager-templates-list"></a>Список шаблонов диспетчера соответствия требованиям

**В этой статье:** Просмотреть полный список **шаблонов, доступных** для создания оценок в диспетчере соответствия требованиям.

> [!IMPORTANT]
> Шаблоны оценки, доступные вашей организации, зависят от лицензионного соглашения. [Просмотрите сведения.](/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance)

## <a name="overview"></a>Обзор

[Microsoft Compliance Manager](compliance-manager.md) предоставляет полный набор шаблонов для создания оценок. Эти шаблоны могут помочь вашей организации соблюдать национальные, региональные и отраслевые требования, регулирующие сбор и использование данных.

Шаблоны добавляются в Диспетчер соответствия требованиям по мере принятия новых законов и правил. Диспетчер соответствия требованиям обновляет свои шаблоны при изменении законодательства или нормативных актов. Узнайте больше о [том, как просмотреть и принять обновления.](compliance-manager-assessments.md#accepting-updates-to-assessments)

Подробные рекомендации по [работе с шаблонами для](compliance-manager-templates.md) создания оценок.

## <a name="list-of-templates-and-where-to-find-them"></a>Список шаблонов и их поиск

Ниже приведен полный список шаблонов в диспетчере соответствия требованиям. На каждой странице сведений о шаблоне содержится информация о правилах или стандартах, к которым оно применяется. Ссылки на имена шаблонов, которые ниже приведены, связаны с документацией по этому стандарту, регулированию или законодательству.

**Где найти шаблоны**

В Диспетчер соответствия требованиям перейдите на **страницу шаблонов оценки.** Вы увидите список всех шаблонов, доступных для вашей организации. Важно понимать различие между включенными и премиум-шаблонами:

- **Включенные шаблоны** — это шаблоны, включенные в лицензионное соглашение вашей организации.
 - **Premium шаблоны отображают** дополнительные шаблоны, которые может получить организация (обратитесь к [условиям службы).](/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance)

Узнайте больше о [том, как просматривать и управлять шаблонами.](compliance-manager-templates.md#viewing-and-managing-templates-from-the-assessment-templates-page)

> [!NOTE]
> Дополнительные [сведения о](compliance-manager-templates.md#template-types-included-and-premium-active-and-inactive) включенных и премиум-шаблонах и о том, какие из них доступны для использования в организации, см. в шаблонах Шаблонов.
