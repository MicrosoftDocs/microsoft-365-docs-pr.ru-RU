# <a name="compliance-manager-templates-list"></a>Список шаблонов диспетчера соответствия требованиям

**В этой статье:** Просмотреть полный список **шаблонов, доступных** для создания оценок в диспетчере соответствия требованиям.

> [!IMPORTANT]
> Шаблоны оценки, доступные вашей организации, зависят от лицензионного соглашения. [Просмотрите сведения.](https://go.microsoft.com/fwlink/?linkid=2132371)

## <a name="overview"></a>Общие сведения

[Диспетчер соответствия требованиям (Майкрософт)](compliance-manager.md) предоставляет полный набор шаблонов для создания оценок. Эти шаблоны могут помочь вашей организации соблюдать национальные, региональные и отраслевые требования, управляющие сбором и использованием данных.

Шаблоны добавляются в диспетчер соответствия требованиям по мере принятия новых законов и нормативов. Диспетчер соответствия требованиям обновляет свои шаблоны при изменении законодательства или нормативных актов. Узнайте больше о [том, как просмотреть и принять обновления.](compliance-manager-assessments.md#accepting-updates-to-assessments)

Просматривайте подробные [инструкции по работе с шаблонами](compliance-manager-templates.md) для создания оценок.

## <a name="list-of-templates-and-where-to-find-them"></a>Список шаблонов и их поиск

Ниже приведен полный список шаблонов в диспетчере соответствия требованиям. На каждой странице сведений о шаблоне содержится информация о нормативе или стандарте, к которому оно применяется. Ссылки в именах шаблонов ниже приведены в связанной документации по этому стандарту, нормативу или законодательству.

**Где найти шаблоны**

В диспетчере соответствия требованиям перейдите на **страницу шаблонов оценки.** Вы увидите список всех шаблонов, доступных для вашей организации.

- **Включенные шаблоны** — это шаблоны, включенные в лицензионное соглашение вашей организации.
 - **В шаблонах premium** отображаются дополнительные шаблоны, которые может выбрать организация (см. [условия обслуживания).](https://go.microsoft.com/fwlink/?linkid=2132371)

Узнайте больше о [том, как просматривать шаблоны и управлять ими.](compliance-manager-templates.md#viewing-and-managing-templates-from-the-assessment-templates-page)
