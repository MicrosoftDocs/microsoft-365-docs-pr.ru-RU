# <a name="set-analyze-advanced-settings-in-advanced-ediscovery"></a>Настройка расширенных параметров анализа в Advanced eDiscovery

> [!NOTE]
> Чтобы можно было использовать Advanced eDiscovery, требуется подписка на Office 365 E3 с надстройкой Advanced Compliance или E5 для организации. Если у вас этого плана нет и вы хотите попробовать Advanced eDiscovery, можете [зарегистрироваться для получения пробной версии Office 365 корпоративный E5](https://go.microsoft.com/fwlink/p/?LinkID=698279). 
  
Advanced eDiscovery предоставляет расширенные параметры по умолчанию для параметров модуля анализа. В следующей процедуре описаны параметры, которые могут быть заданы.
  
1. На **вкладке "Подготовка \> анализа \> программы** установки" щелкните **"Дополнительные параметры"** (в нижней части страницы). Отобразилась следующая панель. 
    
    ![Настройка дополнительных параметров на вкладке "Анализ"](../media/c9ea3017-e19a-456b-a742-c3d07121a3f6.png)
  
2. В **параметрах практически дубликатов** и потоков электронной почты при необходимости выберите значения для следующих значений:
    
  - **Минимальное число слов:** минимальное число слов, ниже которого файл не передается для анализа с почти дубликатом. 
    
  - **Максимальное число слов:** максимальное число слов, над которыми файл не передается для анализа практически дубликатов.
    
  - **Сходство сообщений электронной** почты: минимальный уровень повторной записи для двух сообщений электронной почты, которые считаются похожими. Значение всегда равно или больше сходства документов. Значение по умолчанию — 90%.
    
3. В **параметрах Themes выберите** включаем **номера** в анализе темы, чтобы включить числа в обработку тем во время анализа. 
    
4. Щелкните **Сохранить**.
