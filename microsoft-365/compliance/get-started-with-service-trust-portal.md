# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Начало работы с порталом доверия службы (Майкрософт)

Портал доверия службы Майкрософт предоставляет разнообразные материалы, средства и другие ресурсы, посвященные безопасности, конфиденциальности и рекомендациям Майкрософт по обеспечению безопасности, конфиденциальности и соответствия требованиям.

## <a name="accessing-the-service-trust-portal"></a>Доступ к порталу доверия службы

Портал доверия службы содержит сведения о реализации элементов управления и процессов Майкрософт, которые защищают облачные службы и данные о клиентах. Чтобы получить доступ к некоторым ресурсам на портале доверия службы, необходимо войти в систему с учетной записью пользователя, прошедшего проверку подлинности, с помощью учетной записи Microsoft Cloud Services (учетная запись организации Azure Active Directory или учетная запись Майкрософт), а также проанализировать и принять соглашение о неразглашении Майкрософт для материалов соответствия требованиям.

### <a name="existing-customers"></a>Существующие клиенты

Существующие клиенты могут получить доступ к порталу доверия службы по <https://aka.ms/STP> одной из следующих интерактивных подписок (пробная или платная):

- Microsoft 365
- Dynamics 365
- Azure

 > [!NOTE]
 > Учетные записи Azure Active Directory, связанные с организациями, имеют доступ к полному диапазону документов и функций, таких как диспетчер соответствия требованиям. Учетные записи Майкрософт, созданные для личного использования, имеют ограниченный доступ к содержимому портала доверия служб.

### <a name="new-customers-and-customers-evaluating-microsoft-online-services"></a>Новые клиенты и клиенты, оценивающие Microsoft Online Services

Чтобы создать новую учетную запись или создать пробную учетную запись, используйте одну из следующих форм регистрации (также используемых для пробных учетных записей), чтобы получить доступ к STP.

- Регистрация новых [пробных учетных записей приложений Microsoft 365 для бизнеса](https://go.microsoft.com/fwlink/p/?LinkID=507653) или новой [пробной учетной записи Office 365 корпоративный](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- Регистрация новой [пробной учетной записи Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=252780)

- Зарегистрируйтесь для получения новой [пробной учетной записи Azure](https://go.microsoft.com/fwlink/?LinkId=722737).

Если вы подписываетесь на бесплатную пробную версию или подписку, необходимо включить Azure Active Directory для поддержки доступа к STP.

## <a name="using-the-service-trust-portal"></a>Использование портала доверия службы

В главном меню доступны функции и контент портала доверия службы.

![Меню портала доверия службы](../media/STPMenus1.png)

В следующих разделах приводится описание каждого элемента в главном меню.

### <a name="service-trust-portal"></a>Service Trust Portal

Ссылка на **портал службы доверия** отображает домашнюю страницу. Он позволяет быстро вернуться на домашнюю страницу.

### <a name="compliance-manager"></a>Диспетчер соответствия требованиям

> [!IMPORTANT]
> Диспетчер соответствия требованиям перешел с портала Trusting Service на новое место в [центре соответствия требованиям Microsoft 365](https://compliance.microsoft.com/). Все данные клиента перемещены на новое место, поэтому вы можете продолжать использовать диспетчер соответствия требованиям без перерывов. Сведения о настройке и сведения о новых возможностях можно найти в [документации по диспетчеру соответствия требованиям](compliance-manager.md) . Несмотря на то, что классическая версия диспетчера соответствия требованиям остается на портале доверия службы, всем пользователям рекомендуется использовать диспетчер соответствия требованиям в центре соответствия требованиям Microsoft 365.

### <a name="trust-documents"></a>Документы о защите данных

Предоставляет обширные сведения о реализации и проектировании системы безопасности с целью упрощения соответствия нормативным требованиям, а также сведения о том, как облачные службы Майкрософт обеспечивают безопасность данных. Чтобы просмотреть контент, выберите один из следующих параметров в раскрывающемся меню " **доверять документам** ".

- **Отчеты аудита:** Отображается список независимых отчетов аудита и оценки в облачных службах Майкрософт. В этих отчетах представлены сведения о соответствии требованиям облачных служб Майкрософт требованиям стандартов защиты данных и нормативных требований, таких как:

  - Международная организация для стандартизации (ISO)
  - Средства контроля обслуживающей организации (SOC)
  - требованиям Национального института стандартов (NIST);
  - требованиям Федеральной программы управления рисками и авторизацией (FedRAMP);
  - общему регламенту по защите данных (GDPR).

- **Защита данных:** Содержит множество таких ресурсов, как контролируемые элементы управления, технические документы, ответы на вопросы, проверки на уязвимости, средства оценки риска и руководства по обеспечению соответствия требованиям.

- **Планы обеспечения безопасности и соответствия требованиям Azure:** Ресурсы, которые помогают создавать защищенные и соответствующие облачные приложения. В этой области содержатся рекомендации по чертежу для государственных организаций, финансирования, здравоохранения и розничных вертикальных продаж.

### <a name="industries--regions"></a>Отрасли и регионы

Предоставляет сведения о соответствии отраслевым службам Майкрософт для конкретных отраслевых и регионов.

- **Отрасли:** На этом этапе на этой странице представлена специальная целевая страница для отрасли финансовых служб. Содержит такие сведения, как требования к обеспечению соответствия требованиям, вопросы и ответы, а также статьи об успешном выполнении. Ресурсы для других отраслей будут выпущены в будущем, но вы сможете найти ресурсы для большей отрасли, перейдя на страницу **Trust documents > Data Protections** in STP.

- **Регионы:** Предоставляет юридическую информацию о соответствии требованиям облачных служб Майкрософт различным законам разных стран. К определенным странам относятся Австралия, Канада, Чешская Республика, Дания, Германия, Польша, Румыния, Испания и Соединенное Королевство.

### <a name="trust-center"></a>Центр управления безопасностью

Ссылки на [центр управления безопасностью Майкрософт](https://www.microsoft.com/trust-center), в которых представлены дополнительные сведения о безопасности, соответствии требованиям и конфиденциальности в Microsoft Cloud. Сюда входят сведения о возможностях в облачных службах Майкрософт, которые можно использовать для удовлетворения конкретных требований GDPR, документации, полезной для вашего учета GDPR, и понимания технических и организационных показателей, которые корпорация Майкрософт предприняла для поддержки GDPR.

### <a name="my-library"></a>Моя библиотека

Эта новая функция позволяет сохранять (или *закреплять*) документы, чтобы вы могли быстро получить к ним доступ на странице "Моя библиотека". Вы также можете настроить уведомления, чтобы корпорация Майкрософт отправляла вам сообщение электронной почты при обновлении документов в вашей библиотеке. Для получения дополнительных сведений обратитесь к разделу [Моя библиотека](#my-library-1) этой статьи.

### <a name="more"></a>Дополнительно

Перейдите к разделу **дополнительные > администратору** , чтобы получить доступ к функциям администрирования, доступным только для учетной записи глобального администратора. Этот параметр доступен только в том случае, если вы вошли как глобальный администратор. В раскрывающемся меню **Администратор** есть два варианта:

- **Параметры:** На этой странице вы можете назначить роли пользователей для диспетчера соответствия требованиям (классическое).

- **Параметры конфиденциальности пользователя:** На этой странице вы можете экспортировать отчет, который содержит назначения элементов действий в диспетчере соответствия требованиям (классическое) для определенного пользователя. Вы также можете переназначить всех элементов Action другому пользователю и удалить назначенного элемента действия от указанного пользователя.

### <a name="search"></a>Поиск

Щелкните значок лупы в правом верхнем углу страницы "портал доверия службы", чтобы развернуть поле, введите поисковые условия и нажмите клавишу **Ввод**. Отображается страница **поиска** , в которой термин поиска отображается в поле поиска и в приведенных ниже результатах поиска.

![Service Trust Portal: поиск документов с примененным фильтром](../media/86b754e1-c63c-4514-89ac-d014bf334140.png)

По умолчанию поиск возвращает результаты документа. Вы можете отфильтровать результаты с помощью раскрывающихся списков, чтобы уточнить список отображаемых документов. Можно использовать несколько фильтров для сужения списка документов. Фильтры включают в себя конкретные облачные службы, категории соответствия и рекомендации по обеспечению безопасности, регионы и отрасли. Щелкните ссылку имя документа, чтобы скачать документ.

Чтобы получить список элементов управления от оценок в диспетчере соответствия требованиям (классическое), связанных с условиями поиска, щелкните **Диспетчер соответствия требованиям**. В результатах поиска отображается дата создания оценки, имя группы оценки, соответствующая облачная служба Майкрософт, а также от того, является ли этот элемент управления корпорацией Майкрософт или клиентом. Щелкните имя элемента управления, чтобы просмотреть элемент управления в оценке в диспетчере соответствия требованиям (классическое).

![Service Trust Portal: поиск по средствам контроля в диспетчере соответствия требованиям](../media/bafb811a-68ce-40b5-ad16-058498fe5439.png)

> [!NOTE]
> Отчеты о портале доверия служб и документы доступны для скачивания по крайней мере за 12 месяцев после публикации или до тех пор, пока не станет доступна новая версия документа.

## <a name="my-library"></a>Моя библиотека

С помощью функции "Моя библиотека" можно добавлять документы и ресурсы на портале доверия службы на страницу "Моя библиотека". Благодаря этому вы можете получить доступ к документам, которые важны для вас в отдельном месте.  Чтобы добавить документ в библиотеку "Моя библиотека", щелкните меню **...** справа от документа и выберите команду **сохранить в библиотеку**. Вы можете добавить в свою библиотеку несколько документов, установив флажок рядом с одним или несколькими документами, а затем нажав кнопку **сохранить в библиотеку** в верхней части страницы.

Кроме того, функция уведомлений позволяет настроить свою библиотеку, чтобы сообщение электронной почты отправлялось вам, когда Майкрософт обновляет документ, добавленный в библиотеку "Моя библиотека". Чтобы настроить уведомления, перейдите в свою библиотеку и щелкните **Параметры уведомлений**. Вы можете выбрать частоту уведомлений и указать адрес электронной почты в Организации для отправки уведомлений. Уведомления по электронной почте содержат ссылки на документы, которые были обновлены, и краткое описание обновления.

Кроме того, обратите внимание, что в библиотеке "Мои библиотеки" определены документы, которые были обновлены в течение последних 30 дней, независимо от того, включены ли уведомления. Краткое описание обновления также отображается в подсказке.

## <a name="starter-packs"></a>Начальные пакеты

Starter Packs — это набор документации Майкрософт, посвященный облачным службам Майкрософт для конкретных отраслей. В настоящее время портал доверия службы предлагает три следующих начального пакета для организаций финансовых служб. Эти стартовые пакеты помогают организациям оценивать и оценивать безопасность, соответствие требованиям и конфиденциальность в облаке Майкрософт и предоставляют рекомендации по внедрению облачных служб Майкрософт в отраслевой отрасли строго регулируемых финансовых служб.

- **Начальный пакет для ознакомления:** Используется для раннего ознакомления с Microsoft Cloud для организаций финансовых служб.

- **Начальный пакет оценки:** После ознакомления воспользуйтесь контрольными списками и другими рекомендациями в этом стартовом пакете, чтобы определить риски, связанные с безопасностью, соответствием требованиям и конфиденциальностью.

- **Аудит начального пакета:** Этот пакет Starter Pack содержит рекомендации по использованию элементов управления аудитом и другим средствам для упрощения внедрения облачных служб (Майкрософт), что помогает снизить риск выдержки Организации.

Чтобы получить доступ к этим стартовым пакетам, перейдите в раздел **Служба доверия портал > отрасли & области > отраслевые решения > финансовые услуги**. Вы можете открывать или загружать документы из начального пакета или сохранять их в библиотеке "Моя библиотека".
