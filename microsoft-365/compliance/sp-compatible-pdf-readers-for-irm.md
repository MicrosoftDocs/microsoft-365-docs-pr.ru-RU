# <a name="sharepoint-compatible-pdf-readers-that-support-microsoft-information-rights-management-services"></a>SharePoint-Compatible считывателей PDF, которые поддерживают службы управления правами на информацию (Майкрософт)

Microsoft SharePoint 2013 поддерживает защиту pdf-документов службой управления правами на данные (IRM). С помощью этой поддержки пользователи могут загружать PDF-документы в библиотеки, защищенные службой IRM, и после загрузки файлы будут защищены с помощью Microsoft Office IRM.
