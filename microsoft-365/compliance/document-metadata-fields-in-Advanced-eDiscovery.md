# <a name="document-metadata-fields-in-advanced-ediscovery"></a>Поля метаданных документа в Advanced eDiscovery

В следующей таблице перечислены поля метаданных для документов в наборе для проверки в случае Advanced eDiscovery. В таблице следующую информацию:

- **Имя поля** и отображаемого **поля:** имя поля метаданных и имя поля, отображаемого при просмотре метаданных выбранного документа в наборе для проверки. Некоторые поля метаданных не включаются при просмотре метаданных файла документа. Эти поля выделены звездочкой (*).

- **Имя поля для поиска:** Имя свойства, которое можно найти при запуске [запроса набора для проверки.](review-set-search.md) Пустая ячейка означает, что вы не можете найти поле в запросе набора для проверки.

- **Имя экспортируемой поля:** Имя поля метаданных, которое включается при экспорте документов.  Пустая ячейка означает, что поле не включено в экспортные метаданные.

- **Описание:** Описание поля метаданных.

> [!NOTE]
> В **поле "Ключевые слова"** в [поиске набора для проверки](https://docs.microsoft.com/microsoft-365/compliance/review-set-search) используется язык запросов по ключевым словам (KQL). Поля, перечисленные  в столбце "Имя поля с возможностью поиска", можно использовать в поле **"Ключевые** слова" в наборе для проверки поиска для формирования сложных запросов без использования построитель запросов. Дополнительные сведения о языке KQL см. в справочнике по [синтаксис языка](https://go.microsoft.com/fwlink/?LinkId=269603)запросов по ключевым словам.
