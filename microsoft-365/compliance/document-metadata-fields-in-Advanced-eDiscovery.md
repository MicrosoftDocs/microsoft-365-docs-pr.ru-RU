# <a name="document-metadata-fields-in-advanced-ediscovery"></a>Поля метаданных документов в Advanced eDiscovery

В следующей таблице перечислены поля метаданных для документов в обзоре, заданного в случае Advanced eDiscovery. В таблице приводится следующая информация:

- **Имя поля** и имя поля **отображения:** имя поля метаданных и имя поля, отображаемого при просмотре метаданных файла выбранного документа в наборе обзоров. Некоторые поля метаданных не включаются при просмотре метаданных файла документа. Эти поля выделены звездочкой (*).

- **Имя поля для поиска:** Имя свойства, которое можно искать при запуске запроса [набора отзывов.](review-set-search.md) Пустая ячейка означает, что вы не можете искать поле в запросе набора обзоров.

- **Имя экспортируемой области:** Имя поля метаданных, включаемого при экспорте документов.  Пустая ячейка означает, что поле не включено в экспортируемую метаданные.

- **Описание:** Описание поля метаданных.

> [!NOTE]
> Поле **"Ключевые слова"** в [поиске набора обзоров](./review-set-search.md) использует язык запросов по ключевым словам (KQL). Поля, перечисленные в столбце имя поля **Searchable,** можно использовать в поле **Keywords** в поиске набора обзоров для формирования сложных запросов без необходимости использования строителя запросов. Дополнительные сведения о KQL см. в статье Ссылка на язык [синтаксиса запроса](/sharepoint/dev/general-development/keyword-query-language-kql-syntax-reference)ключевых слов.
