# <a name="document-metadata-fields-in-advanced-ediscovery"></a>Поля метаданных документа в Advanced eDiscovery

В следующей таблице перечислены поля метаданных для документов в наборе для проверки в случае Advanced eDiscovery. В таблице представлены следующие сведения:

- **Имя поля** и **отображаемое имя поля:** имя поля метаданных и имя поля, которое отображается при просмотре метаданных выбранного документа в наборе для проверки. Некоторые поля метаданных не включаются при просмотре метаданных файла документа. Эти поля выделены с помощью звездочки (*).

- **Имя поля, доступного для поиска:** Имя свойства, которое вы ищете при выполнении запроса [набора для проверки.](review-set-search.md) Пустая ячейка означает, что неудается искать поле в запросе набора для проверки.

- **Имя экспортируемого поля:** Имя поля метаданных, которое включается при экспорте документов.  Пустая ячейка означает, что поле не включается в экспортированные метаданные.

- **Описание:** Описание поля метаданных.

> [!NOTE]
> Поле **"Ключевые слова"** при [поиске в наборе](https://docs.microsoft.com/microsoft-365/compliance/review-set-search) использует язык запросов по ключевым словам (KQL). Поля, указанные в **столбце имен поле имени поля с возможностью** поиска, можно использовать в поле **"Ключевые слова"** в поиске набора для формирования сложных запросов без необходимости использования построителя запросов. Дополнительные сведения об этом языке см. в справочнике по [синтаксису языка запросов по ключевым словам.](https://go.microsoft.com/fwlink/?LinkId=269603)
