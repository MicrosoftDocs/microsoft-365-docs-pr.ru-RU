# <a name="set-up-a-connector-to-archive-blackberry-data-preview"></a>Настройка соединителю для архива данных BlackBerry (предварительный просмотр)

Используйте [BlackBerry DataParser](https://www.17a-4.com/BlackBerry-dataparser/) от 17a-4 LLC для импорта и архива корпоративных данных BlackBerry в почтовые ящики пользователей в Microsoft 365 организации. DataParser включает соединителю BlackBerry, который настроен для захвата элементов из стороннего источника данных и импорта этих элементов в Microsoft 365. Соединиттель BlackBerry DataParser преобразует данные BlackBerry в формат сообщений электронной почты, а затем импортирует эти элементы в почтовые ящики пользователей в Microsoft 365.

После хранения данных BlackBerry в почтовых ящиках пользователей можно применить Microsoft 365, такие как хранение судебного разбирательства, электронные сведения, политики хранения и метки хранения, а также соответствие требованиям связи. Использование соединиттеля BlackBerry для импорта и архива данных в Microsoft 365 может помочь вашей организации соблюдать государственные и нормативные политики.

## <a name="overview-of-archiving-blackberry-data"></a>Обзор архива данных BlackBerry

В следующем обзоре объясняется процесс использования соединиттеля данных для архивации данных BlackBerry в Microsoft 365.

![Архивавка рабочего процесса для данных BlackBerry с 17a-4](../media/BlackBerryDataParserConnectorWorkflow.png)

1. Ваша организация работает с 17a-4 для настройки и настройки BlackBerry DataParser.

2. Регулярно элементы BlackBerry собираются в DataParser. DataParser также преобразует содержимое сообщения в формат сообщения электронной почты.

3. Соединителю BlackBerry DataParser, который вы создаете в Центр соответствия требованиям Microsoft 365, подключается к DataParser и передает сообщения в безопасное служба хранилища Azure в облаке Microsoft.

4. В почтовых ящиках пользователей создается подмостки в папке "Входящие" с именем **BlackBerry DataParser,** и элементы BlackBerry импортируется в эту папку. Соединитатель определяет, в какой почтовый ящик импортировать элементы, используя значение свойства *Email.* Каждый элемент BlackBerry содержит это свойство, которое заполняется адресом электронной почты каждого участника.

## <a name="before-you-set-up-a-connector"></a>Перед настройками соединитетеля

- Создайте учетную запись DataParser для соединители Microsoft. Для этого обратитесь в [ООО "17a-4".](https://www.17a-4.com/contact/) При создании соединитетеля в шаге 1 необходимо войти в эту учетную запись.

- Пользователь, создавший соединителю BlackBerry DataParser в шаге 1 (и завершивший его в шаге 3), должен быть назначен роли экспорта импорта почтовых ящиков в Exchange Online. Эта роль требуется для добавления соединители на странице **соединители** данных в Центр соответствия требованиям Microsoft 365. По умолчанию эта роль не назначена группе ролей в Exchange Online. Вы можете добавить роль экспорта импорта почтовых ящиков в группу ролей управления организацией в Exchange Online. Или вы можете создать группу ролей, назначить роль экспортировать импорт почтовых ящиков, а затем добавить соответствующих пользователей в качестве участников. Дополнительные сведения см. в разделах [Создание](/Exchange/permissions-exo/role-groups#create-role-groups) групп ролей или [изменение](/Exchange/permissions-exo/role-groups#modify-role-groups) групп ролей в статье "Управление группами ролей в Exchange Online".

## <a name="step-1-set-up-a-blackberry-dataparser-connector"></a>Шаг 1. Настройка соединиттеля BlackBerry DataParser

Первым шагом является доступ к странице соединители данных в Центр соответствия требованиям Microsoft 365 и создание соединителя 17a-4 для данных BlackBerry.

1. Перейдите <https://compliance.microsoft.com> к и нажмите **кнопку Соединители**  >  **данных BlackBerry DataParser**.

2. На странице **описания продукта BlackBerry DataParser** нажмите **кнопку Добавить соединителю**.

3. На странице **Условия службы нажмите** кнопку **Принять**.

4. Введите уникальное имя, которое идентифицирует соединителен, а затем нажмите **кнопку Далее**.

5. Вопишите в свою учетную запись 17a-4 и выполните действия мастера подключения BlackBerry DataParser.

## <a name="step-2-configure-the-blackberry-dataparser-connector"></a>Шаг 2. Настройка соединителю BlackBerry DataParser

Работа с поддержкой 17a-4 для настройки соединиттеля BlackBerry DataParser.

## <a name="step-3-map-users"></a>Шаг 3. Пользователи карт

Соединиттель BlackBerry DataParser автоматически соберет пользователей с Microsoft 365 адресами электронной почты перед импортом данных в Microsoft 365.

## <a name="step-4-monitor-the-blackberry-dataparser-connector"></a>Шаг 4. Мониторинг соединиттеля BlackBerry DataParser

После создания соединиттеля BlackBerry DataParser можно просмотреть состояние соединителю в Центр соответствия требованиям Microsoft 365.

1. Перейдите <https://compliance.microsoft.com> и щелкните **соединители данных** в левом nav.

2. Щелкните  вкладку Соединители, а затем выберите соединитель BlackBerry DataParser, созданный для отображения страницы вылетов, которая содержит свойства и сведения о соединителе.

3. В **состоянии Соединитель с исходным кодом** щелкните ссылку **Журнал** загрузки, чтобы открыть (или сохранить) журнал состояния соединитетеля. В этом журнале содержатся данные, импортируемые в облако Майкрософт.

## <a name="known-issues"></a>Известные проблемы

В настоящее время мы не поддерживаем импорт вложений или элементов размером более 10 МБ. Поддержка более крупных элементов будет доступна позднее.
