# <a name="add-or-remove-members-from-a-case"></a>Добавление и удаление участников дела

Вы можете добавлять или удалять участников для управления доступом к делу. Однако прежде чем участник сможет получить доступ к делу Advanced eDiscovery (и выполнять задачи в этом случае),  необходимо добавить пользователя в группу ролей "Диспетчер eDiscovery" на странице "Разрешения" в Центре безопасности и соответствия требованиям. Дополнительные сведения см. в теме "Назначение разрешений [на eDiscovery"](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions)в Центре безопасности & соответствия требованиям.

1. На странице **Advanced eDiscovery** перейдите к делу, в которое нужно добавить участника.

2. Перейдите **на вкладку** "Параметры", а затем нажмите кнопку **"Выбрать"** **& "Доступ".**

3. Нажмите **кнопку "Обновить".**

4. В **области "Управление участниками"** нажмите **кнопку** "Добавить", чтобы добавить участников дела. Вы также можете добавить группу ролей в дело, нажав кнопку **"Добавить"** в группе **"Управление группами ролей".**

5. В списке людей или групп ролей, которые можно добавить в качестве участников дела, выберите этот список рядом с именами пользователей или групп ролей, которые вы хотите добавить.

6. После выбора пользователей или групп ролей для добавления в качестве участников дела нажмите кнопку **"Добавить".**

7. На странице **"Управление этим случаем"** нажмите кнопку **"Сохранить",** чтобы сохранить новый список участников дела.
