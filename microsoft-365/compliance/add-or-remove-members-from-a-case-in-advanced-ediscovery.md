# <a name="add-or-remove-members-from-a-case"></a><span data-ttu-id="39381-103">Добавление и удаление участников дела</span><span class="sxs-lookup"><span data-stu-id="39381-103">Add or remove members from a case</span></span>

<span data-ttu-id="39381-104">Вы можете добавлять и удалять участников, имеющих доступ к делу.</span><span class="sxs-lookup"><span data-stu-id="39381-104">You can add or remove members to manage who can access the case.</span></span> <span data-ttu-id="39381-105">Однако прежде чем участник сможет получить доступ к Advanced eDiscovery (и выполнять задачи в этом случае), необходимо добавить пользователя в группу ролей диспетчера электронных данных на странице **Permissions** в центре безопасности и соответствия требованиям.</span><span class="sxs-lookup"><span data-stu-id="39381-105">However, before a member can access a Advanced eDiscovery case (and perform tasks in the case), you must add the user to the eDiscovery Manager role group on the **Permissions** page in the security and compliance center.</span></span> <span data-ttu-id="39381-106">Дополнительные сведения см. в статье [Назначение разрешений для обнаружения электронных данных в Центре безопасности и соответствия требованиям](./assign-ediscovery-permissions.md).</span><span class="sxs-lookup"><span data-stu-id="39381-106">For more information, see [Assign eDiscovery permissions in the Security & Compliance Center](./assign-ediscovery-permissions.md).</span></span>

1. <span data-ttu-id="39381-107">На странице **Advanced eDiscovery** перейдите к делу, в которое нужно добавить участника.</span><span class="sxs-lookup"><span data-stu-id="39381-107">On the **Advanced eDiscovery** page, go to the case that you want to add a member to.</span></span>

2. <span data-ttu-id="39381-108">Перейдите на вкладку **Параметры**, а затем нажмите кнопку **Выбрать** на плитке **Доступ и разрешения**.</span><span class="sxs-lookup"><span data-stu-id="39381-108">Click the **Settings** tab and then click **Select** in the **Access & permissions** tile.</span></span>

3. <span data-ttu-id="39381-109">Нажмите **кнопку Обновление**.</span><span class="sxs-lookup"><span data-stu-id="39381-109">Click **Update**.</span></span>

4. <span data-ttu-id="39381-110">В разделе **Управление участниками** щелкните **Добавить**, чтобы добавить участников в дело.</span><span class="sxs-lookup"><span data-stu-id="39381-110">Under **Manage members**, click **Add** to add members to the case.</span></span> <span data-ttu-id="39381-111">Вы также можете добавить группу ролей в дело, нажав **добавить** в **группу Управление группами ролей.**</span><span class="sxs-lookup"><span data-stu-id="39381-111">You can also choose to add a role group to the case by clicking  **Add** under **Manage role groups**.</span></span>

5. <span data-ttu-id="39381-112">В списке людей или групп ролей, которые можно добавить в качестве участников дела, установите флажки рядом с именами людей или групп ролей, которые хотите добавить.</span><span class="sxs-lookup"><span data-stu-id="39381-112">In the list of people or role groups that can be added as members of the case, select the check box next to the names of the people or role groups that you want to add.</span></span>

   > [!NOTE]
   > <span data-ttu-id="39381-113">При добавлении группы ролей в дело можно добавить только группы ролей, в которые вы входите.</span><span class="sxs-lookup"><span data-stu-id="39381-113">When adding a role group to a case, you can only add the role groups that you are a member of.</span></span>

6. <span data-ttu-id="39381-114">После выбора людей или групп ролей, которые необходимо добавить в качестве участников дела, нажмите **кнопку Добавить**.</span><span class="sxs-lookup"><span data-stu-id="39381-114">After you've selected the people or role groups to add as members of the case, click **Add**.</span></span>

7. <span data-ttu-id="39381-115">На всплывающей странице **Управление делом** нажмите кнопку **Сохранить**, чтобы сохранить новый список участников дела. </span><span class="sxs-lookup"><span data-stu-id="39381-115">In the **Manage this case** flyout page, click **Save** to save the new list of case members.</span></span>
