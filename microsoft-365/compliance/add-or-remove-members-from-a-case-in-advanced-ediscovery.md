# <a name="add-or-remove-members-from-a-case"></a><span data-ttu-id="82b60-103">Добавление и удаление участников дела</span><span class="sxs-lookup"><span data-stu-id="82b60-103">Add or remove members from a case</span></span>

<span data-ttu-id="82b60-104">Вы можете добавить или удалить членов, чтобы управлять, кто может получить доступ к делу.</span><span class="sxs-lookup"><span data-stu-id="82b60-104">You can add or remove members to manage who can access the case.</span></span> <span data-ttu-id="82b60-105">Однако прежде чем участник сможет получить доступ к делу advanced eDiscovery (и выполнять задачи в этом случае), необходимо добавить пользователя в группу ролей диспетчера электронных данных на странице **Permissions** в центре безопасности и соответствия требованиям.</span><span class="sxs-lookup"><span data-stu-id="82b60-105">However, before a member can access a Advanced eDiscovery case (and perform tasks in the case), you must add the user to the eDiscovery Manager role group on the **Permissions** page in the security and compliance center.</span></span> <span data-ttu-id="82b60-106">Дополнительные сведения см. в [дополнительных сведениях о](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions)назначении разрешений на открытие электронных данных в Центре & безопасности.</span><span class="sxs-lookup"><span data-stu-id="82b60-106">For more information, see [Assign eDiscovery permissions in the Security & Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions).</span></span>

1. <span data-ttu-id="82b60-107">На странице **Advanced eDiscovery** перейдите к делу, в которое нужно добавить участника.</span><span class="sxs-lookup"><span data-stu-id="82b60-107">On the **Advanced eDiscovery** page, go to the case that you want to add a member to.</span></span>

2. <span data-ttu-id="82b60-108">Щелкните **вкладку Параметры** и нажмите **кнопку Выберите** в плитке **разрешений & доступа.**</span><span class="sxs-lookup"><span data-stu-id="82b60-108">Click the **Settings** tab and then click **Select** in the **Access & permissions** tile.</span></span>

3. <span data-ttu-id="82b60-109">Нажмите **кнопку Обновление**.</span><span class="sxs-lookup"><span data-stu-id="82b60-109">Click **Update**.</span></span>

4. <span data-ttu-id="82b60-110">В **статье Управление участниками** **щелкните Добавить,** чтобы добавить участников в дело.</span><span class="sxs-lookup"><span data-stu-id="82b60-110">Under **Manage members**, click **Add** to add members to the case.</span></span> <span data-ttu-id="82b60-111">Вы также можете добавить группу ролей в дело, нажав **добавить** в **группу Управление группами ролей.**</span><span class="sxs-lookup"><span data-stu-id="82b60-111">You can also choose to add a role group to the case by clicking  **Add** under **Manage role groups**.</span></span>

5. <span data-ttu-id="82b60-112">В списке людей или групп ролей, которые могут быть добавлены в качестве участников дела, выберите поле рядом с именами людей или групп ролей, которые вы хотите добавить.</span><span class="sxs-lookup"><span data-stu-id="82b60-112">In the list of people or role groups that can be added as members of the case, select the check box next to the names of the people or role groups that you want to add.</span></span>

   > [!NOTE]
   > <span data-ttu-id="82b60-113">При добавлении группы ролей в дело можно добавить только группы ролей, в которые вы входите.</span><span class="sxs-lookup"><span data-stu-id="82b60-113">When adding a role group to a case, you can only add the role groups that you are a member of.</span></span>

6. <span data-ttu-id="82b60-114">После выбора людей или групп ролей, которые необходимо добавить в качестве участников дела, нажмите **кнопку Добавить**.</span><span class="sxs-lookup"><span data-stu-id="82b60-114">After you've selected the people or role groups to add as members of the case, click **Add**.</span></span>

7. <span data-ttu-id="82b60-115">На странице **Управление этим случаем** щелкните **Сохранить,** чтобы сохранить новый список участников дела.</span><span class="sxs-lookup"><span data-stu-id="82b60-115">In the **Manage this case** flyout page, click **Save** to save the new list of case members.</span></span>
