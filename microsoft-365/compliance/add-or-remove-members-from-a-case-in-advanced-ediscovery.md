<span data-ttu-id="3de4b-104">Вы можете добавлять или удалять участников для управления доступом к делу.</span><span class="sxs-lookup"><span data-stu-id="3de4b-104">You can add or remove members to manage who can access the case.</span></span> <span data-ttu-id="3de4b-105">Однако прежде чем участник сможет получить доступ к делу Advanced eDiscovery (и выполнять задачи в этом случае),  необходимо добавить пользователя в группу ролей "Диспетчер eDiscovery" на странице "Разрешения" в Центре безопасности и соответствия требованиям.</span><span class="sxs-lookup"><span data-stu-id="3de4b-105">However, before a member can access a Advanced eDiscovery case (and perform tasks in the case), you must add the user to the eDiscovery Manager role group on the **Permissions** page in the security and compliance center.</span></span> <span data-ttu-id="3de4b-106">Дополнительные сведения см. в теме "Назначение разрешений [на eDiscovery"](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions)в Центре безопасности & соответствия требованиям.</span><span class="sxs-lookup"><span data-stu-id="3de4b-106">For more information, see [Assign eDiscovery permissions in the Security & Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions).</span></span>

1. <span data-ttu-id="3de4b-107">На странице **Advanced eDiscovery** перейдите к делу, в которое нужно добавить участника.</span><span class="sxs-lookup"><span data-stu-id="3de4b-107">On the **Advanced eDiscovery** page, go to the case that you want to add a member to.</span></span>

2. <span data-ttu-id="3de4b-108">Перейдите **на вкладку** "Параметры", а затем нажмите кнопку **"Выбрать"** **& "Доступ".**</span><span class="sxs-lookup"><span data-stu-id="3de4b-108">Click the **Settings** tab and then click **Select** in the **Access & permissions** tile.</span></span>

3. <span data-ttu-id="3de4b-109">Нажмите **кнопку "Обновить".**</span><span class="sxs-lookup"><span data-stu-id="3de4b-109">Click **Update**.</span></span>

4. <span data-ttu-id="3de4b-110">В **области "Управление участниками"** нажмите **кнопку** "Добавить", чтобы добавить участников дела.</span><span class="sxs-lookup"><span data-stu-id="3de4b-110">Under **Manage members**, click **Add** to add members to the case.</span></span> <span data-ttu-id="3de4b-111">Вы также можете добавить группу ролей в дело, нажав кнопку **"Добавить"** в группе **"Управление группами ролей".**</span><span class="sxs-lookup"><span data-stu-id="3de4b-111">You can also choose to add a role group to the case by clicking  **Add** under **Manage role groups**.</span></span>

5. <span data-ttu-id="3de4b-112">В списке людей или групп ролей, которые можно добавить в качестве участников дела, выберите этот список рядом с именами пользователей или групп ролей, которые вы хотите добавить.</span><span class="sxs-lookup"><span data-stu-id="3de4b-112">In the list of people or role groups that can be added as members of the case, select the check box next to the names of the people or role groups that you want to add.</span></span>

6. <span data-ttu-id="3de4b-113">После выбора пользователей или групп ролей для добавления в качестве участников дела нажмите кнопку **"Добавить".**</span><span class="sxs-lookup"><span data-stu-id="3de4b-113">After you've selected the people or role groups to add as members of the case, click **Add**.</span></span>

7. <span data-ttu-id="3de4b-114">На странице **"Управление этим случаем"** нажмите кнопку **"Сохранить",** чтобы сохранить новый список участников дела.</span><span class="sxs-lookup"><span data-stu-id="3de4b-114">In the **Manage this case** flyout page, click **Save** to save the new list of case members.</span></span>
