# <a name="plan-for-security-amp-compliance"></a>Планирование соответствия требованиям &amp; безопасности

Управление безопасностью и соответствием требованиям — это партнерство. Вы несете ответственность за защиту данных, удостоверений и устройств, в то время как Майкрософт активно защищает службы Microsoft 365. Вы можете использовать Microsoft 365 и Enterprise Mobility + Security (EMS) вместе, чтобы обеспечить соответствующий уровень защиты для вашей организации.
  
## <a name="step-1-review-capabilities"></a>Шаг 1. Просмотр возможностей

Сориентироваться на возможности защиты информации на плакате "Защита информации для Office 365". 
  
[Развертывание защиты информации для нормативов конфиденциальности данных с помощью Microsoft 365](https://docs.microsoft.com/microsoft-365/solutions/information-protection-deploy?view=o365-worldwide)
  
## <a name="step-2-check-your-secure-score"></a>Шаг 2. Проверка оценки безопасности

После настройки подписки на Microsoft 365 заметьте свой начальный показатель. Оценка безопасности предоставляет рекомендации по настройке, которые можно использовать для увеличения оценки. Цель — знать о возможностях, которые можно использовать для защиты среды, которые не будут отрицательно влиять на производительность пользователей.
  
- [Знакомство с оценкой безопасности Office 365](../security/mtp/microsoft-secure-score.md)
