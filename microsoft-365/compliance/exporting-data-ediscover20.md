# <a name="export-case-data-in-advanced-ediscovery"></a>Экспорт данных дела в Advanced eDiscovery

Существует три способа экспорта данных из набора для проверки:

**Скачать:** Скачайте (с помощью браузера) небольшой набор файлов. Это самый быстрый способ экспорта небольшого набора данных. Этот метод сохраняет имена файлов. Дополнительные сведения см. [в документе "Загрузка документов из набора для проверки".](download-documents-from-review-set.md)

**Экспорт:** Настройка экспортируемой информации. К ним относится экспорт метаданных файлов, встроенных файлов, текстовых файлов и отредактных документов, сохраненных в PDF-файле. После подготовки экспортных данных можно загрузить данные со вкладки **"Экспорт"** на локальный компьютер. Дополнительные сведения см. [в документе "Экспорт документов из набора для проверки"](export-documents-from-review-set.md)

**Добавьте в другой набор для проверки:** Скопируйте данные из одного набора для проверки в другой. Дополнительные сведения см. в подстройки "Добавление данных из [одного набора для проверки в другой".](add-data-to-review-set-from-another-review-set.md)

> [!NOTE]
> В Microsoft 365 данные хешются, и эти хеши предоставляются в выходной файл для проверки. Это дополняется журналами аудита и функциями отчетов, такими как статистика сборов, отчеты о наборе нагрузки и отчеты об экспорте (включая файл загрузки экспорта).
