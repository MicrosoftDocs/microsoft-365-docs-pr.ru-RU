# <a name="export-case-data-in-advanced-ediscovery"></a>Экспорт данных о случаях в Advanced eDiscovery

Существует три способа экспорта данных из набора обзоров:

**Скачать:** Скачайте (с помощью браузера) небольшой набор родных файлов. Это самый быстрый способ экспорта небольшого набора данных. Этот метод сохраняет имена родных файлов. Дополнительные сведения см. в [документе Download documents from a review set.](download-documents-from-review-set.md)

**Экспорт:** Настройка экспортируемой информации. Это относится к экспорту метаданных файлов, основных файлов, текстовых файлов и отредактированных документов, сохраненных в файлы PDF. После подготовки экспортируемой информации можно скачать данные со вкладки **Экспорт** на локальный компьютер. Дополнительные сведения см. [в документе Export documents from a review set](export-documents-from-review-set.md)

**Добавьте к другому набору обзоров:** Скопируйте данные из одного набора обзоров в другой набор обзоров. Дополнительные сведения см. в примере Добавление данных из [одного набора обзоров в другой набор обзоров.](add-data-to-review-set-from-another-review-set.md)

> [!NOTE]
> В Microsoft 365 данные хешются, и эти хеши предоставляются в файле вывода для целей проверки. Это дополняется журналами аудита и функциональными возможностями отчетов, такими как статистика коллекций, отчеты о наборе нагрузки и отчеты об экспорте (в том числе файл экспортной нагрузки).
