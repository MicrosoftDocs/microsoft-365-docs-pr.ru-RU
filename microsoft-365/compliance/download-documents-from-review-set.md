# <a name="download-selected-documents-from-a-review-set"></a>Скачивание выбранных документов из набора рецензирования

Загрузка предлагает простой способ загрузки контента из набора проверки в собственном формате. Средство загрузки в Advanced eDiscovery использует функции передачи данных в браузере. Когда загрузка будет завершена, появится приглашение браузера. Файлы, загруженные с помощью этого метода, переносятся в архив в файле контейнера и будут содержать файлы на уровне элементов. Это означает, что если вы выберете вложение, вы получите сообщение электронной почты с включенным вложением. Аналогичным образом, если вы экспортируете электронную таблицу Excel, внедренную в документ Word, в загружаемую файл будут включены документы Word со встроенной таблицей Excel. При загрузке элементов сохраняется последнее измененное свойство данных, которое можно просмотреть в виде свойства файла.

Чтобы скачать содержимое из набора проверки в расширенном случае обнаружения электронных данных, сначала выберите файлы, которые вы хотите скачать, а затем выберите **действие**  >  **загружать**элементы.
