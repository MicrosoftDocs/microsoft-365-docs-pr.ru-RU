# <a name="download-selected-documents-from-a-review-set"></a>Скачивание выбранных документов из набора для проверки

Скачивание предлагает простой способ загрузки содержимого из набора для проверки в формате native. Средство скачивания в Advanced eDiscovery использует функции передачи данных браузера. Когда загрузка будет готова, появится запрос браузера. Файлы, загруженные с помощью этого метода, загуют в файл контейнера и будут содержать файлы на уровне элемента. Это означает, что если вы выберете скачивание вложения, вы получите сообщение электронной почты с вложением. Аналогичным образом, если экспортировать таблицу Excel, внедренную в документ Word, документ Word с внедренной таблицой Excel будет включен в загрузку. При скачии элементов свойство "Данные о последних изменениях" сохраняется и может рассматриваться как свойство файла.

Чтобы скачать содержимое из набора для проверки в случае Advanced eDiscovery, сначала выберите файлы, которые вы хотите скачать, а затем выберите "Скачать элементы  >  **действия".**
