# <a name="supported-file-types-in-advanced-ediscovery"></a>Поддерживаемые типы файлов в Advanced eDiscovery

Расширенное обнаружение электронных данных поддерживает множество типов файлов с множеством различных уровней, описанных в следующих таблицах. Этот список не завершен, и мы добавим новые типы файлов, когда мы продолжим проверочное тестирование. В этих таблицах указывается, поддерживается ли тип файла для извлечения текста (и оптическое распознавание текста или извлечение текста OCR для файлов изображений), которые можно просматривать в встроенном средстве просмотра и поддерживаются в средстве просмотра меток в Advanced eDiscovery.
