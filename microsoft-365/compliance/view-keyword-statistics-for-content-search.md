# <a name="view-keyword-statistics-for-content-search-results"></a>Просмотр статистики ключевых слов для результатов поиска контента

После создания и запуска поиска контента можно просмотреть статистику о предполагаемых результатах поиска. Это включает сводку результатов поиска (аналогичную сводке о предполагаемых результатах поиска, отображаемой в области сведений), статистику запросов, такую как количество местоположений контента с элементами, которые соответствуют запросу поиска, и имена расположения контента, которые имеют наиболее совпадающие элементы. Вы можете вывести статистику по одному или нескольким поискам контента. Это позволяет быстро сравнивать результаты нескольких поисков и принимать решения об эффективности поисковых запросов.
  
Кроме того, вы можете настроить новые и существующие поиски, чтобы вернуть статистику по каждому ключевому слову в запросе поиска. Это позволяет сравнить количество результатов для каждого ключевого слова в запросе и сравнить статистику ключевых слов из нескольких поисковых запросов.
  
Вы также можете скачать статистику по поискам и ключевым словам в виде CSV-файла, чтобы сравнить результаты с помощью функций фильтрации и сортировки в Excel или подготовить отчеты.
  
## <a name="get-statistics-for-content-searches"></a>Получить статистику поиска контента

Отображение статистики поиска контента:
  
1. В центре соответствия требованиям Microsoft 365 перейдите к **демонстрации всех**  >  **поисков контента.**

2. В списке поисков выберите два или несколько поисков, а затем щелкните статистику поиска **на** **странице** "Массовые действия".
    
    ![Выберите несколько поисков, а затем щелкните статистику поиска](../media/1195c6c3-2e00-469d-8c29-85c1c7ebe6c7.png)
  
3. На странице **Статистика поиска** щелкните одну из следующих ссылок, чтобы отобразить статистику выбранных поисков. 
    
    **Summary**
    
    На этой странице отображаются статистические данные, аналогичные тем, которые отображаются в области сведений на странице **поиска контента.** Отображаются статистические данные для всех выбранных поисков. Обратите внимание, что вы также можете повторно выполнить выбранные поиски на этой странице, чтобы обновить статистику. 
    
    ![Сводка статистики выбранных поисков](../media/abb663eb-b3d6-4f4c-a99f-55d20b0848af.png)
  
    а.  Имя поиска контента. Как уже говорилось ранее, вы можете отображать и сравнивать статистику для нескольких поисков.
    
    б. Тип расположения контента, который был в поиске. Каждая строка отображает статистику для почтовых ящиков, сайтов и общедоступных папок из указанного поиска.
    
    в. Количество местоположений контента, содержащих элементы, которые соответствуют запросу поиска. Для почтовых ящиков эта статистика также включает число архивных почтовых ящиков, содержащих элементы, которые соответствуют запросу поиска.
    
    г. Общее число элементов всех указанных местоположений контента, которые соответствуют запросу поиска. Примеры типов элементов включают сообщения электронной почты, элементы календаря и документы. Если элемент содержит несколько экземпляров ключевого слова, которое находится в поиске, он учитывается только один раз в общем количестве элементов. Например, если вы ищете слова "stock" или "fraud", а сообщение электронной почты содержит три экземпляра слова "stock", оно учитывается только один раз в столбце **Items.** 
    
    д. Общий размер всех элементов, найденных в указанном расположении контента, совпадающий с запросом поиска. 
    
    **Queries**
    
    На этой странице отображаются статистические данные о запросе поиска.
    
    ![Статистика запросов поиска для выбранных поисков](../media/dc817526-dfb9-43d3-a14c-4c58077eb7bb.png)
  
    а. Имя поиска контента, для которого строка содержит статистику запросов.
    
    б. Тип расположения контента, к который применима статистика запросов.
    
    в. В этом столбце указывается, к какой части поискового запроса применима статистика. **Основной** указывает на весь поисковый запрос. Если при создании или редактировании поискового запроса используется список ключевых слов, в эту таблицу включаются статистические данные по каждому компоненту запроса. Дополнительные сведения см. в разделе Статистика поиска ключевых слов для раздела [Поиск](#get-keyword-statistics-for-content-searches) контента в этой статье. 
    
    г. В этом столбце содержится фактический поисковый запрос, запускаемый средством поиска контента. Обратите внимание, что средство автоматически добавляет несколько дополнительных компонентов в создающийся запрос. 

    - При поиске всего контента в почтовых ящиках (не указав ключевых слов), фактический запрос ключевого слова является таким образом, чтобы все элементы  `size>=0` были возвращены. 
    
     - При поиске сайтов SharePoint Online и OneDrive для бизнеса добавляются два следующих компонента:
    
          **NOT IsExternalContent:1** — исключает контент из локальной организации SharePoint. 
    
          **NOT IsOneNotePage:1** — исключает все файлы OneNote, так как они будут дублировать любой документ, соответствующий запросу поиска. 

    
    д. Количество расположений контента (указанных в столбце ** Тип расположения **), содержащих элементы, которые соответствуют запросу поиска, указанному в столбце **Запрос.** 
    
    е. Количество элементов (из указанного расположения контента), которые соответствуют запросу поиска, указанному в **столбце Запрос.** Как было объяснено ранее, если элемент содержит несколько экземпляров ключевого слова, которое ведется поиск, он учитывается только один раз в этом столбце. 
    
    ж. Общий размер всех найденных элементов (в указанном расположении контента), которые соответствуют запросу поиска в **столбце Запрос.** 
    
    **Верхние расположения**
    
    На этой странице отображаются статистические данные о количестве элементов, которые соответствуют запросу поиска в каждом расположении контента, которое было в поиске. Выводится первая 1000 расположений. При просмотре статистики для нескольких поисков отображаются 1000 расположения для каждого поиска. Обратите внимание, что расположение контента не включено на этой странице, если оно не содержит элементов, которые соответствуют запросу поиска.
    
    ![Статистика количества элементов, найденных в поисковых расположениях контента](../media/35a820b0-85d9-45d1-9a0c-c74bec803e67.png)
  
    а. Имя расположения контента.
    
    б. Тип расположения контента, к который применима статистика расположения.
    
    в. Для каждого поиска есть столбцы, для которых отображается статистика. В этом столбце показано количество (и общий размер) элементов, которые соответствуют запросу поиска в каждом расположении контента. Обратите внимание, что при отображение статистики для нескольких поисков в этом столбце "NA" указывается, что расположение контента не было включено в этот поиск. 

## <a name="get-keyword-statistics-for-content-searches"></a>Получить статистику ключевых слов для поиска контента

Как было объяснено ранее, на странице **Запросы** показаны запрос поиска и число (и размер) элементов, которые соответствуют запросу. Если при создании или редактировании поискового запроса используется список ключевых слов, можно получить расширенную статистику, которая показывает, сколько элементов соответствует каждому ключевому слову или ключевому слову. Это поможет быстро определить, какие части запроса являются наиболее (и наименее) эффективными. Например, если ключевое слово возвращает большое количество элементов, можно уточнить запрос ключевого слова, чтобы сузить результаты поиска. Список ключевых слов можно настроить при создании или редактировании поиска контента. 

Чтобы создать список ключевых слов и просмотреть статистику ключевых слов для поиска контента:
  
1. В центре соответствия требованиям Microsoft 365 перейдите к **демонстрации всех**  >  **поисков контента.**
    
2. В списке поиска контента нажмите кнопку и поиск, а затем нажмите **кнопку Изменить значок Изменить** ![ ](../media/ebd260e4-3556-4fb0-b0bb-cc489773042c.gif) .
    
3. Щелкните **Запрос,** а затем делайте следующие вещи: 
    
    ![Щелкните контрольный ящик списка "Показать" и введите ключевое слово в каждой строке](../media/73ef46dd-3d5c-415d-b5e7-c3559caaafe2.png)
  
    а. Щелкните **поле список ключевых слов Show.** 
    
    б. Введите ключевое слово или этап ключевого слова в строке в таблице ключевых слов. Например, введите **бюджет** в первом ряду, а затем **введите** безопасность во втором ряду. 
    
4. После добавления ключевых слов, для поиска и получения статистики нажмите **кнопку Поиск** для запуска пересмотренного поиска. 
    
5. После завершения поиска выберите его в списке поисков и нажмите кнопку **Статистика** ![ ](../media/9bf56d43-25bf-4f53-a4be-f4d55102310c.png) поиска. Вы также можете отображать и сравнивать статистику ключевых слов для нескольких поисков.
    
6. На странице **Статистика** поиска щелкните **Запрос,** чтобы отобразить статистику ключевых слов для выбранных поисков. 
    
    ![Отображаются статистические данные по каждому ключевому слову для выбранных поисков](../media/e7910fa9-af93-4df9-92d0-e1e3e089e14f.png)
  
    Как показано на предыдущем скриншоте, отображаются статистические данные по каждому ключевому слову; это включает в себя: 
    
    - Статистика ключевых слов для каждого типа расположения контента, включенного в поиск.
    
    - Фактический поисковый запрос для каждого ключевого слова, который включает все условия из поискового запроса. 
    
    - Полный запрос поиска (определен как **Основной** в столбце **Часть)** и статистика для полного запроса. Обратите внимание, что это те же статистические данные, которые отображаются на странице **Сводка.** 

> [!NOTE]
> Чтобы уменьшить проблемы, вызванные большими списками ключевых слов, теперь в списке ключевых слов поискового запроса ограничено не более 20 строк.
