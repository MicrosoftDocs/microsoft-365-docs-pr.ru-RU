# <a name="view-keyword-statistics-for-content-search-results"></a>Просмотр статистики ключевых слов для результатов поиска контента

После создания и запуска поиска контента можно просмотреть статистику о предполагаемых результатах поиска. Это включает сводку результатов поиска (аналогичную сводке о предполагаемых результатах поиска, отображаемой в области сведений), статистику запросов, такую как количество местоположений контента с элементами, которые соответствуют поисковому запросу, и имена местоположений контента, включающих наиболее совпадающие элементы. Вы можете вывести статистику по одному или нескольким поискам контента. Это позволяет быстро сравнить результаты для нескольких поисковых запросов и принять решения об эффективности поисковых запросов.
  
Кроме того, можно настроить новый и существующий поиск, чтобы возвращать статистику по каждому ключевому слову в поисковом запросе. Это позволяет сравнить количество результатов для каждого ключевого слова в запросе и сравнить статистику ключевых слов из нескольких поисковых запросов.
  
Вы также можете скачать статистику по поискам и ключевым словам в виде CSV-файла, чтобы сравнить результаты с помощью функций фильтрации и сортировки в Excel или подготовить отчеты.
  
## <a name="get-statistics-for-content-searches"></a>Получить статистику по поиску контента

Отображение статистики для поиска контента:
  
1. В Центре соответствия требованиям Microsoft 365 перейдите в центр **"Показать все поиск**  >  **контента".**

2. В списке поисковых запросов выберите два или  более поисковых запросов, а затем щелкните статистику поиска на **странице** "Массовые действия".
    
    ![Выберите несколько поисковых запросов и щелкните статистику поиска](../media/1195c6c3-2e00-469d-8c29-85c1c7ebe6c7.png)
  
3. На странице **"Статистика** поиска" щелкните одну из следующих ссылок, чтобы отобразить статистику по выбранным поисковым запросам. 
    
    **Summary**
    
    На этой странице отображается статистика, аналогичная статистике, отображаемой в области сведений на странице **поиска контента.** Отображается статистика по всем выбранным поисковым запросам. Обратите внимание, что вы также можете повторно запустить выбранные поисковые запросы на этой странице, чтобы обновить статистику. 
    
    ![Сводка статистики по выбранным поисковым запросам](../media/abb663eb-b3d6-4f4c-a99f-55d20b0848af.png)
  
    а.  Имя поиска контента. Как было сказано ранее, можно отобразить и сравнить статистику для нескольких поисковых запросов.
    
    б. Тип расположения контента, в которое был велись поиски. В каждой строке отображается статистика для почтовых ящиков, сайтов и общедоступных папок из указанного поиска.
    
    в. Количество местоположений контента, содержащих элементы, которые соответствуют поисковому запросу. Для почтовых ящиков эта статистика также включает количество архивных почтовых ящиков, содержащих элементы, которые соответствуют поисковому запросу.
    
    г. Общее количество элементов всех указанных местоположений контента, которые соответствуют поисковому запросу. Примеры типов элементов: сообщения электронной почты, элементы календаря и документы. Если элемент содержит несколько экземпляров ключевого слова, для поиска в нем учитывается только один раз в общем количестве элементов. Например, если вы ищете слова "stock" или "мошенничество" и сообщение электронной почты содержит три экземпляра слова "stock", оно учитывается только один раз в столбце **"Элементы".** 
    
    д. Общий размер всех элементов, найденных в указанном расположении контента, которое соответствует поисковому запросу. 
    
    **Queries**
    
    На этой странице отображается статистика поискового запроса.
    
    ![Статистика поисковых запросов для выбранных поисковых запросов](../media/dc817526-dfb9-43d3-a14c-4c58077eb7bb.png)
  
    а. Имя поиска контента, для которого строка содержит статистику запроса.
    
    б. Тип расположения контента, к который применима статистика запросов.
    
    в. В этом столбце указывается, к какой части поискового запроса применима статистика. **Основной** указывает на весь поисковый запрос. Если при создании или редактировании поискового запроса используется список ключевых слов, в эту таблицу включается статистика для каждого компонента запроса. Дополнительные [сведения см. в](#get-keyword-statistics-for-content-searches) разделе "Получить статистику ключевых слов для поиска контента" в этой статье. 
    
    г. В этом столбце содержится фактический поисковый запрос, запускающийся средством "Поиск контента". Обратите внимание, что средство автоматически добавляет в создадимый запрос несколько дополнительных компонентов. 

    - При поиске всего содержимого в почтовых ящиках (без указания ключевых слов) фактический запрос ключевого слова так, чтобы возвращались  `size>=0` все элементы. 
    
     - При поиске на сайтах SharePoint Online и OneDrive для бизнеса добавляются два следующих компонента:
    
          **NOT IsExternalContent:1** — исключает любой контент из локальной организации SharePoint. 
    
          **NOT IsOneNotePage:1** — исключает все файлы OneNote, так как они будут дублировать любой документ, соответствующий поисковому запросу. 

    
    д. Количество расположений контента (заданных столбцом ** Тип расположения **), которые содержат элементы, которые соответствуют поисковому запросу, указанному в **столбце "Запрос".** 
    
    е. Количество элементов (из указанного расположения контента), которые соответствуют поисковому запросу, указанному в **столбце "Запрос".** Как объяснялось ранее, если элемент содержит несколько экземпляров ключевого слова, для поиска в этом столбце он учитывается только один раз. 
    
    ж. Общий размер всех найденных элементов (в указанном расположении контента), которые соответствуют поисковому запросу в **столбце "Запрос".** 
    
    **Лучшие расположения**
    
    На этой странице отображается статистика о количестве элементов, которые соответствуют поисковому запросу, в каждом расположении контента, в которое был велись поиск. Выводится первая 1000 расположений. При просмотре статистики для нескольких поисковых запросов отображаются 1000 наиболее часто отображаемой информации о каждом поиске. Обратите внимание, что расположение контента не включается на эту страницу, если оно не содержит элементов, которые соответствуют поисковому запросу.
    
    ![Статистика о количестве элементов, найденных в расположениях контента, в которые был велись поиск](../media/35a820b0-85d9-45d1-9a0c-c74bec803e67.png)
  
    а. Имя расположения контента.
    
    б. Тип расположения контента, к какому расположению применима статистика.
    
    в. Для каждого поиска есть столбцы, для которых отображается статистика. В этом столбце показано количество (и общий размер) элементов, которые соответствуют поисковому запросу в каждом расположении контента. Обратите внимание, что при отобрагии статистики для нескольких поисковых запросов "NA" в этом столбце указывает, что расположение контента не было включено в этот поиск. 

## <a name="get-keyword-statistics-for-content-searches"></a>Получить статистику ключевых слов для поиска контента

Как было поясняется ранее, на странице "Запросы" показан поисковый запрос и число (и размер) элементов, которые соответствуют запросу.  Если при создании или редактировании поискового запроса используется список ключевых слов, можно получить расширенную статистику, которая показывает, сколько элементов соответствует каждому ключевому слову или ключевой фразе. Это поможет быстро определить, какие части запроса наиболее (и наименее) эффективны. Например, если ключевое слово возвращает большое количество элементов, можно уточнить запрос ключевого слова, чтобы сузить результаты поиска. Список ключевых слов можно настроить при создании или редактировании поиска контента. 

Чтобы создать список ключевых слов и просмотреть статистику ключевых слов для поиска контента:
  
1. В Центре соответствия требованиям Microsoft 365 перейдите в центр **"Показать все поиск**  >  **контента".**
    
2. В списке поиска контента щелкните и  выберите поиск, а затем щелкните значок ![ ](../media/ebd260e4-3556-4fb0-b0bb-cc489773042c.gif) редактирования.
    
3. Нажмите **кнопку** "Запрос", а затем сделайте следующее: 
    
    ![Нажмите кнопку "Показать список ключевых слов" и введите ключевое слово в каждой строке](../media/73ef46dd-3d5c-415d-b5e7-c3559caaafe2.png)
  
    а. Нажмите **кнопку "Показать список ключевых слов".** 
    
    б. Введите этап ключевого слова или ключевого слова в строке таблицы ключевых слов. Например, введите **бюджет** в первой строке, а затем **введите** безопасность во второй строке. 
    
4. После добавления ключевых слов, для поиска и получения  статистики щелкните "Поиск", чтобы запустить измененный поиск. 
    
5. После завершения поиска выберите его в списке и  нажмите кнопку "Статистика ![ поиска". ](../media/9bf56d43-25bf-4f53-a4be-f4d55102310c.png) Вы также можете отобразить и сравнить статистику ключевых слов для нескольких поисковых запросов.
    
6. На странице **"Статистика** поиска" щелкните **"Запрос",** чтобы отобразить статистику ключевых слов для выбранных поисковых запросов. 
    
    ![Отображается статистика по каждому ключевому слову для выбранных поисковых запросов](../media/e7910fa9-af93-4df9-92d0-e1e3e089e14f.png)
  
    Как показано на предыдущем снимке экрана, отображается статистика для каждого ключевого слова; к ним относятся: 
    
    - Статистика ключевых слов для каждого типа расположения контента, включенного в поиск.
    
    - Фактический поисковый запрос для каждого ключевого слова, который включает все условия из поискового запроса. 
    
    - Полный поисковый запрос  (в столбце **"Часть")** и статистика для всего запроса. Обратите внимание, что это та же статистика, которая отображается на странице **сводки.** 

> [!NOTE]
> Чтобы уменьшить количество проблем, вызванных большими списками ключевых слов, теперь в списке ключевых слов поискового запроса может быть не более 20 строк.
