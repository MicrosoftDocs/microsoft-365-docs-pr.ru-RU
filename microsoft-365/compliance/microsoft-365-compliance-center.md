# <a name="microsoft-365-compliance-center"></a>Центр соответствия требованиям Microsoft 365

Если вас интересует позиция соответствия требованиям организации, вам понравится центр Microsoft 365 [соответствия](https://compliance.microsoft.com)требованиям. Центр соответствия требованиям Microsoft 365 обеспечивает простой доступ к данным и инструментам, необходимым для удовлетворения потребностей вашей организации в соответствии требованиям.

Прочитайте эту статью, чтобы ознакомиться с центром [Microsoft 365,](#how-do-i-get-the-compliance-center)как его [получить,](#frequently-asked-questions)часто задав вопросы, и ваши [дальнейшие действия](#next-steps).

[![Microsoft 365 главная страница центра соответствия требованиям](../media/m365-compliance-center-home.png)](https://compliance.microsoft.com)

## <a name="welcome-to-microsoft-365-compliance"></a>Добро пожаловать в Microsoft 365 соответствие требованиям

Когда вы впервые Microsoft 365 центр соответствия требованиям, вас встречает следующее приветствие:

![Microsoft 365 центра соответствия требованиям](../media/m365-compliance-center-welcome-steps.png)

Приветственный баннер дает вам некоторые указатели о том, как начать работу, с последующими шагами, и приглашение для вас, чтобы дать нам обратную связь.

## <a name="card-section"></a>Раздел карточки

При первом посещении центра Microsoft 365, в разделе карточки на домашней странице сразу показано, как ваша организация работает с соблюдением данных, какие решения доступны для организации, а также сводка всех активных оповещений.

Отсюда вы можете:

- Просмотрите **карточку Microsoft Compliance Manager,** которая приведет вас к решению [Диспетчер соответствия](compliance-manager.md) требованиям. Диспетчер соответствия требованиям упрощает управление соответствием требованиям. Он вычисляет оценку на основе риска, измеряя ваш прогресс в выполнении рекомендуемых действий, которые помогают снизить риски в области защиты данных и нормативных стандартов. Он также предоставляет возможности рабочего процесса и встроенное сопоставление управления, чтобы помочь вам эффективно выполнять действия по улучшению.

    ![Карта Диспетчер соответствия требованиям Microsoft 365 центра соответствия требованиям](../media/m365-compliance-center-compliance-manager-card.png)

- Просмотрите **новую** карту каталога решений, [](microsoft-365-solution-catalog.md) которая ссылок на коллекции интегрированных решений, которые можно использовать для управления сценариями соответствия требованиям. Возможности и средства решения могут включать сочетание политик, оповещений, отчетов и других функций.

    ![Карта каталога решений Microsoft 365 центра соответствия требованиям](../media/m365-compliance-center-solution-catalog-card.png)

- Просмотрите карту **Активные** оповещения, которая включает сводку наиболее активных оповещений и включает ссылку, в которой можно просмотреть более подробные сведения, такие как Серьезность, Состояние, Категория и другие. [](alert-policies.md)

    ![Карточка активных оповещений Microsoft 365 центра соответствия требованиям](../media/m365-compliance-center-active-alerts-card.png)

Вы также можете  использовать функцию Добавить карты, чтобы добавить дополнительные карты, например одну, показывающую соответствие облачным [](/cloud-app-security/) приложениям организации, а другую , показывающую данные о пользователях с общими файлами, со ссылками на Cloud App Security или другие средства, в которых можно исследовать данные.

![Дополнительные сведения о карточке центра соответствия требованиям](../media/m365-compliance-center-additional-cards.png)

## <a name="how-do-i-get-the-compliance-center"></a>Как получить центр соответствия требованиям?

- Если у вас нет нового центра Microsoft 365 соответствия требованиям, он появится в ближайшее время. Центр Microsoft 365 соответствия требованиям, как правило, теперь доступен для Microsoft 365 SKU.

- Чтобы посетить центр Microsoft 365, в качестве глобального администратора, администратора соответствия требованиям или администратора данных соответствия. [https://compliance.microsoft.com](https://compliance.microsoft.com)

## <a name="frequently-asked-questions"></a>Вопросы и ответы

**Почему меня отвезли в Центр & безопасности для выполнения некоторых задач, например определения определенных политик?**

Мы по-прежнему разрабатываем центр Microsoft 365 соответствия требованиям и добавляем дополнительные функциональные возможности и решения в ближайшие месяцы. Тем временем в Центре безопасности и соответствия требованиям необходимо завершить несколько задач ([https://protection.office.com](https://protection.office.com)). В этих случаях вы будете автоматически направлены в расположение, где вы можете выполнить поставленную задачу, например, создать или изменить политику надзора.

**Почему я еще не вижу новый центр Microsoft 365 соответствия требованиям?**

Во-первых, убедитесь, что у вас есть соответствующие лицензии и разрешения. Затем впишитесь в [https://compliance.microsoft.com](https://compliance.microsoft.com) . Если новый центр соответствия требованиям еще не появится, он появится в ближайшее время.

**Некоторые функции соответствия требованиям недоступны в центре Microsoft 365 соответствия требованиям. Чем я занимаюсь?**

Мы по-прежнему добавляем функции в центр Microsoft 365 соответствия требованиям. Если вы не можете найти что-то, например поиск журнала аудита, используйте Центр & безопасности [https://protection.office.com](https://protection.office.com) ( ). Конфигурации сохраняются как в существующем центре & безопасности, так и в новом центре Microsoft 365 соответствия требованиям.

Чтобы перейти туда, в центре Microsoft 365 соответствия требованиям, в области навигации на левой стороне экрана выберите Дополнительные **ресурсы,** а затем, в Office 365 **центра**& соответствия требованиям, выберите **Open**.

## <a name="next-steps"></a>Дальнейшие действия

- **Посетите службу Microsoft Compliance Manager,** чтобы узнать о вашей оценке соответствия требованиям и приступить к управлению соответствием требованиям для вашей организации. Дополнительные дополнительные информации см. в [см. в "Диспетчере соответствия требованиям".](compliance-manager.md)

- **Настройте внутренние политики управления** рисками, чтобы свести к минимуму внутренние риски и позволить обнаруживать, исследовать и принимать меры для рискованных действий в организации. См. [в руководстве по управлению рисками insider.](insider-risk-management.md)

- **Просмотрите политики предотвращения потери** данных в организации и при необходимости внося необходимые изменения. Дополнительные сведения см. в [дополнительных сведениях о предотвращении потери данных.](dlp-learn-about-dlp.md)

- **Знакомство с и настройка Microsoft Cloud App Security**. См. [в обзоре Quickstart. Начало работы с Microsoft Cloud App Security](/cloud-app-security/getting-started-with-cloud-app-security).

- **Узнайте о политиках соответствия** требованиям к коммуникациям и создании их для быстрого выявления и устранения нарушений политики корпоративного кодекса поведения. См. [в Microsoft 365.](communication-compliance.md)

- **Часто посещайте Microsoft 365** центр соответствия требованиям и просмотрите все возникающие оповещения или потенциальные риски. Перейдите на страницу [https://compliance.microsoft.com](https://compliance.microsoft.com) и войдите.
