# <a name="microsoft-365-compliance-center"></a>Центр соответствия требованиям Microsoft 365

Если вас интересует позиция соответствия требованиям организации, вам понравится центр соответствия [требованиям Microsoft 365.](https://compliance.microsoft.com) Центр соответствия требованиям Microsoft 365 обеспечивает легкий доступ к данным и средствам, которые необходимо использовать для выполнения требований организации.

Ознакомьтесь с этой статьей, чтобы ознакомиться с [](#how-do-i-get-the-compliance-center)центром соответствия требованиям Microsoft 365, узнать, как его [получить,](#frequently-asked-questions)часто задав вопросы и предприняв [следующие действия.](#next-steps)

[![Домашняя страница Центра соответствия требованиям Microsoft 365](../media/m365-compliance-center-home.png)](https://compliance.microsoft.com)

## <a name="welcome-to-microsoft-365-compliance"></a>Добро пожаловать на соответствие требованиям Microsoft 365

Когда вы впервые идете в центр соответствия требованиям Microsoft 365, вас встречает следующее приветствие:

![Интро Центра соответствия требованиям Microsoft 365](../media/m365-compliance-center-welcome-steps.png)

Приветственный баннер дает вам некоторые указатели о том, как начать работу, с последующими шагами, и приглашение для вас, чтобы дать нам обратную связь.

## <a name="card-section"></a>Раздел карточки

При первом посещении центра соответствия требованиям Microsoft 365 в разделе карточки на домашней странице сразу показано, как ваша организация справляется с соблюдением данных, какие решения доступны для организации, а также сводка всех активных оповещений.

Отсюда можно:

- Просмотрите **карточку Microsoft Compliance Manager,** которая приведет вас к решению [Диспетчер соответствия](compliance-manager.md) требованиям. Диспетчер соответствия требованиям упрощает управление соответствием требованиям. Он вычисляет оценку на основе риска, измеряя ваш прогресс в выполнении рекомендуемых действий, которые помогают снизить риски в области защиты данных и нормативных стандартов. Он также предоставляет возможности рабочего процесса и встроенное сопоставление управления, чтобы помочь вам эффективно выполнять действия по улучшению.

    ![Центр соответствия требованиям к карточке Диспетчер соответствия требованиям Microsoft 365](../media/m365-compliance-center-compliance-manager-card.png)

- Просмотрите **новую** карту каталога решений, [](microsoft-365-solution-catalog.md) которая ссылок на коллекции интегрированных решений, которые можно использовать для управления сценариями соответствия требованиям. Возможности и средства решения могут включать сочетание политик, оповещений, отчетов и других средств.

    ![Карта каталога решений Microsoft 365](../media/m365-compliance-center-solution-catalog-card.png)

- Просмотрите карту **Активные** оповещения, которая включает сводку наиболее активных оповещений и включает ссылку, в которой можно просмотреть более подробные сведения, такие как Серьезность, Состояние, Категория и другие. [](alert-policies.md)

    ![Центр обработки активных оповещений microsoft 365](../media/m365-compliance-center-active-alerts-card.png)

Кроме того,  можно использовать функцию Добавить карты для добавления дополнительных карт, например, с указанием соответствия облачным приложениям организации, а другой — с данными о пользователях с общими файлами со ссылками на cloud [App Security](/cloud-app-security/) или другими средствами, в которых можно исследовать данные.

![Дополнительные сведения о карточке центра соответствия требованиям](../media/m365-compliance-center-additional-cards.png)

## <a name="how-do-i-get-the-compliance-center"></a>Как получить центр соответствия требованиям?

- Если у вас еще нет нового центра соответствия требованиям Microsoft 365, он появится в ближайшее время. Центр соответствия требованиям Microsoft 365 теперь доступен для клиентов SKU Microsoft 365.

- Чтобы посетить центр соответствия требованиям Microsoft 365, в качестве глобального администратора, администратора соответствия требованиям или администратора данных соответствия, зайдите [https://compliance.microsoft.com](https://compliance.microsoft.com) и войдите.

## <a name="frequently-asked-questions"></a>Вопросы и ответы

**Почему меня отвезли в Центр & безопасности для выполнения некоторых задач, например определения определенных политик?**

Мы по-прежнему разрабатываем центр соответствия требованиям Microsoft 365 и добавляем дополнительные функциональные возможности и решения в ближайшие месяцы. В то же время в Центре обеспечения безопасности & выполнения нескольких задач [https://protection.office.com](https://protection.office.com) ( ). В этих случаях вы будете автоматически направлены в расположение, где вы можете выполнить поставленную задачу, например, создать или изменить политику надзора.

**Почему я еще не вижу новый центр соответствия требованиям Microsoft 365?**

Во-первых, убедитесь, что у вас есть соответствующие лицензии и разрешения. Затем впишитесь в [https://compliance.microsoft.com](https://compliance.microsoft.com) . Если новый центр соответствия требованиям еще не появится, он появится в ближайшее время.

**Некоторые функции соответствия требованиям недоступны в центре соответствия требованиям Microsoft 365. Чем я занимаюсь?**

Мы по-прежнему добавляем функциональные возможности в центр соответствия требованиям Microsoft 365. Если вы не можете найти что-то, например поиск журнала аудита, используйте Центр & безопасности [https://protection.office.com](https://protection.office.com) ( ). Конфигурации сохраняются как в существующем центре & безопасности, так и в новом центре соответствия требованиям Microsoft 365 автоматически.

Чтобы перейти туда, в центре соответствия требованиям Microsoft 365, в области навигации на левой стороне экрана выберите Дополнительные **ресурсы,** а затем, в центре обеспечения безопасности **Office 365 &,** выберите **Open**.

![Дополнительные ресурсы](../media/m365-compliance-center-more-resources.png)

## <a name="next-steps"></a>Дальнейшие действия

- **Посетите службу Microsoft Compliance Manager,** чтобы узнать о вашей оценке соответствия требованиям и приступить к управлению соответствием требованиям для вашей организации. Дополнительные дополнительные информации см. в [см. в "Диспетчере соответствия требованиям".](compliance-manager.md)

- **Настройте внутренние политики управления** рисками, чтобы свести к минимуму внутренние риски и позволить обнаруживать, исследовать и принимать меры для рискованных действий в организации. См. [в руководстве по управлению рисками insider.](insider-risk-management.md)

- **Просмотрите политики предотвращения потери** данных в организации и при необходимости внося необходимые изменения. Дополнительные сведения см. в обзоре политик предотвращения [потери данных.](data-loss-prevention-policies.md)

- **Ознакомиться с безопасностью облачных приложений Microsoft Cloud App и настроить ее.** См. [в обзоре Quickstart. Начало работы с безопасностью облачных приложений Microsoft.](/cloud-app-security/getting-started-with-cloud-app-security)

- **Узнайте о политиках соответствия** требованиям к коммуникациям и создании их для быстрого выявления и устранения нарушений политики корпоративного кодекса поведения. См. [в сообщении о соответствии требованиям связи в Microsoft 365.](communication-compliance.md)

- **Часто посещайте центр соответствия требованиям Microsoft 365** и просмотрите все возникающие оповещения или потенциальные риски. Перейдите на страницу [https://compliance.microsoft.com](https://compliance.microsoft.com) и войдите.
