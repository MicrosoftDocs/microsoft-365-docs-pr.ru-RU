# <a name="manage-review-sets-in-advanced-ediscovery"></a>Управление наборами проверки в Advanced eDiscovery

Наборы для проверки — это статический набор документов, в котором можно анализировать, запрашивать, просматривать, помечать и экспортировать данные в случае. Дополнительные сведения о выполнении этих задач см. в указанных здесь.

- [Анализ данных в наборе для проверки](analyzing-data-in-review-set.md)

- [Запрос данных в наборе для проверки](review-set-search.md)

- [Просмотр документов в наборе для проверки](view-documents-in-review-set.md)

- [Добавление тегов к документам в наборе для проверки](tagging-documents.md)

- [Экспорт данных дела](exporting-data-ediscover20.md)

## <a name="create-a-review-set"></a>Создание набора для проверки

Наборы для проверки можно создать на **вкладке "Наборы** проверки", нажав **кнопку +Добавить набор для проверки.**

![Добавление набора для проверки](../media/f45c51d9-585d-47d1-b7fb-0288715e0b6a.png)

На странице **"Добавление набора отзывов"** введите имя набора для проверки и нажмите кнопку **"Сохранить".** Новый набор для проверки отображается в списке на вкладке **"Наборы проверки".**

![Новый набор для проверки, указанный на вкладке "Набор для проверки"](../media/AeDnewreviewset.png)

Существует три различных способа добавления данных в набор для проверки в случае Advanced eDiscovery.

![Три способа добавления в наборы для проверки](../media/1f1f4efd-c03b-4255-bc3d-df358e56549c.png)

1. [Добавление результатов поиска в набор для проверки](add-data-to-review-set.md)

2. [Загрузка данных, не относяхся к Microsoft 365, в набор для проверки](load-non-Office-365-data-into-a-review-set.md)

3. [Добавление данных в набор для проверки из другого набора для проверки](add-data-to-review-set-from-another-review-set.md)
