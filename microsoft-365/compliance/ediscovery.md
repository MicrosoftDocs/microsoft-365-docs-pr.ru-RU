# <a name="ediscovery-in-microsoft-365"></a>eDiscovery в Microsoft 365

Обнаружение электронных данных — это процесс определения и предоставления электронной информации, которая может использоваться в качестве доказательств в судебных разбирательствах. Средства eDiscovery в Microsoft 365 можно использовать для поиска контента в почтовых ящиках Exchange Online, группах Microsoft 365, Microsoft Teams, сайтах SharePoint Online и OneDrive для бизнеса, беседах Skype для бизнеса и группах Yammer. Поиск в почтовых ящиках и на сайтах с помощью одного и того же поиска методом eDiscovery можно использовать с помощью средства "Поиск контента". Кроме того, вы можете использовать основные дела eDiscovery для идентификации, удержания и экспорта контента, найденного в почтовых ящиках и на сайтах. Если в вашей организации есть подписка на Office 365 E5 или Microsoft 365 E5 (или связанные с ней подписки на надстройки E5), вы можете дополнительно управлять хранителями и анализировать контент с помощью решения Advanced eDiscovery в Microsoft 365.
  
Microsoft 365 предоставляет следующие средства eDiscovery:
  
- [Поиск контента](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [Advanced eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Решение Advanced eDiscovery в Microsoft 365 (также называемое *Advanced eDiscovery 2.0)* построено на существующих возможностях eDiscovery и аналитики в Office 365. Это решение для eDiscovery предоставляет конечный рабочий процесс для сохранения, сбора, просмотра, анализа и экспорта контента, который отвечает на внутренние и внешние исследования в организации. Он также позволяет юридическим группам управлять хранителями и всем процессом уведомлений об удержании по юридическим основаниям для связи с хранителями, участвующими в деле.
