# <a name="ediscovery-solutions-in-microsoft-365"></a>Решения по обнаружению электронных данных в Microsoft 365

Обнаружение электронных данных — это процесс определения и предоставления электронной информации, которая может использоваться в качестве доказательств в судебных разбирательствах. Средства электронного поиска можно использовать в Microsoft 365 для поиска контента в почтовых ящиках Exchange Online, Microsoft 365 Groups, Microsoft Teams, SharePoint Online и OneDrive для бизнеса, а также в группах Skype для бизнеса и yammer. Вы можете искать почтовые ящики и сайты в том же поиске по обнаружению электронных сообщений с помощью средства поиска контента. Кроме того, можно использовать случаи выявления, удержания и экспорта контента, найденного в почтовых ящиках и на сайтах. Если в вашей организации есть подписка на Office 365 E5 или Microsoft 365 E5 (или связанные с ней надстройки E5), можно дополнительно управлять хранителями и анализировать контент с помощью решения advanced eDiscovery в Microsoft 365.
  
Microsoft 365 предоставляет следующие средства для электронного разыскного анализа:
  
- [Поиск контента](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [Advanced eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Решение Advanced eDiscovery в Microsoft 365 (также называемое *Advanced eDiscovery v2.0)* построено на существующих возможностях электронного открытия и аналитики в Microsoft 365. Это решение по обнаружению электронных данных предоставляет конечный рабочий процесс для сохранения, сбора, анализа, анализа и экспорта контента, отвечающей внутренним и внешним расследованиям организации. Он также позволяет юридическим группам управлять хранителями и весь рабочий процесс уведомления об удержании для связи с хранителями, участвующими в деле.
