# <a name="filter-data-when-importing-pst-files"></a>Фильтрация данных при импорте PST-файлов

Используйте новую функцию интеллектуального импорта в службе импорта Office 365 для фильтрации элементов в PST-файлах, которые фактически импортируется в целевые почтовые ящики. Вот как это работает:
  
- После создания и отправки задания импорта PST-файлов PST-файлы загружаются в хранилище Azure в Microsoft Cloud.
    
- Microsoft 365 безопасно анализирует данные в PST-файлах, определяя возраст элементов почтового ящика и различные типы сообщений, включенных в PST-файлы.
    
- После завершения анализа и готовности к импорту данных можно импортировать все данные в PST-файлы без каких-либо настроек или обрезать импортируемые данные путем установки фильтров, которые контролируют импортируемые данные. Например, можно выбрать:
    
  - Импортировать только элементы определенного возраста.
    
  - Импортировать выбранные типы сообщений.
    
  - Исключение сообщений, отправленных или полученных определенными людьми.
    
- После настройки параметров фильтра Microsoft 365 импортирует только данные, удовлетворяющие критериям фильтрации, в целевые почтовые ящики, указанные в заданиях импорта.
    
На следующем рисунке показан интеллектуальный процесс импорта и выделены выполняемые задачи и задачи, выполняемые Office 365.
  
![Интеллектуальный процесс импорта в Office 365](../media/f2ec309b-11f5-48f2-939c-a6ff72152d14.png)

  - [Импорт PST-файлов в Office 365 с помощью отправки по сети](use-network-upload-to-import-pst-files.md)
    
  - [Импорт PST-файлов в Office 365 с помощью отправки дисков](use-drive-shipping-to-import-pst-files-to-office-365.md)
    
- После создания задания импорта с помощью отправки по сети состояние задания импорта на странице "Импорт" в Центре безопасности и & соответствия требованиям будет установлено в состояние "Анализ", что означает, что Microsoft 365 анализирует данные в загруженных PST-файлах. Нажмите **кнопку** ![ ](../media/165fb3ad-38a8-4dd9-9e76-296aefd96334.png) "Обновить", чтобы обновить состояние задания импорта. 
    
- Для заданий на импорт дисков Microsoft 365 анализирует данные после того, как сотрудники центра обработки данных Майкрософт получат ваш жесткий диск и загрузят PST-файлы в хранилище Azure для вашей организации.
  
## <a name="filter-data-that-gets-imported-to-mailboxes"></a>Фильтрация данных, импортируемых в почтовые ящики

После создания задания импорта PST-данных выполните следующие действия, чтобы отфильтровать данные перед импортом в Office 365.
  
1. Перейдите в [https://protection.office.com/](https://protection.office.com/) и войдите в систему, используя учетные данные для учетной записи администратора в вашей организации. 
    
2. Щелкните **"Управление** \> **информацией** \> **Импортировать PST-файлы".**
    
    Задания импорта для организации перечислены на странице "Импорт **PST-файлов".** Обратите **внимание, что** значение  завершения анализа в столбце "Состояние" указывает задания импорта, которые были проанализированы в Microsoft 365 и готовы к импорту. 
    
    ![Состояние завершения анализа указывает, что Microsoft 365 проанализировал данные в PST-файлах](../media/de5294f4-f0ba-4b92-a48a-a4b32b6da490.png)
  
3. Нажмите **кнопку "Готово для импорта в Office 365"** для задания импорта, которое вы хотите завершить. 
    
    Появится страница со сведениями о PST-файлах и другой информацией о задании импорта.
    
4. Нажмите **кнопку "Импорт в Office 365".**
    
    Откроется страница **Отфильтровать данные**. Он содержит данные о данных в PST-файлах для задания импорта, включая сведения о возрасте данных. 
    
    ![На странице "Фильтрация данных" показаны данные PST-файлов для задания импорта](../media/3b537ec0-25a4-45a4-96d5-a429e2a33128.png)
  
5. В зависимости от того, хотите ли вы обрезать данные, импортируемые в Microsoft 365, в области "Хотите ли вы фильтровать **данные?",** сделайте одно из следующих:
    
    а. Нажмите **кнопку "Да". Я** хочу отфильтровать его перед импортом, чтобы обрезать импортируемые данные, а затем нажмите кнопку **"Далее".**

    ![Microsoft 365 отображает подробные сведения о данных из анализа PST-файлов](../media/4881205f-0288-4c32-a440-37e2160295f2.png)

Вы можете навести курсор на каждую из них, чтобы отобразить определенную информацию об этом типе сообщения. Кроме того, существует список с различными возрастными значениями, основанными на анализе PST-файлов. При выборе возраста в выпадаемом списке график обновляется, чтобы показать, сколько данных будет импортироваться для выбранного возраста. 
    
    б. Чтобы настроить дополнительные фильтры для уменьшения объема импортируемых данных, щелкните "Дополнительные **параметры фильтрации".**

      - **Тип.** В этом разделе показаны все типы сообщений, найденные в PST-файлах для задания импорта. Вы можете отоать поле рядом с типом сообщения, который нужно исключить. Обратите внимание, что нельзя исключить другой тип сообщения. Список [элементов почтового](#more-information) ящика, включенных в категорию Other, см. в разделе "Дополнительные сведения".

Введите адрес электронной почты (SMTP-адрес) пользователя, нажмите значок "Добавить новый", чтобы добавить их в список исключенных пользователей для этого типа получателей, а затем нажмите кнопку "Сохранить", чтобы сохранить список исключенных  ![ ](../media/457cd93f-22c2-4571-9f83-1b129bcfb58e.gif) пользователей.

    в. Нажмите **кнопку "Применить"** на странице "Дополнительные **параметры** фильтрации", чтобы сохранить параметры фильтра.

Обратите внимание, что также отображается сводка параметров фильтра. Можно нажать **кнопку "Изменить"** рядом с фильтром, чтобы при необходимости изменить параметр. 
    
    ![Анализ данных обновляется на основе параметров фильтра](../media/897e20fb-3b13-44c3-9d56-9f330750f2a3.png)

    Отображается страница состояния с настройками фильтра. Опять же, вы можете изменить любой из параметров фильтра.

    или

7. Щелкните задание импорта, чтобы отобразить более подробные сведения, такие как состояние каждого PST-файла и настроенные параметры фильтра.

  
## <a name="more-information"></a>Дополнительные сведения

- Как Microsoft 365 определяет приращения для фильтра возраста? Когда Microsoft 365 анализирует PST-файл, он анализирует отправленный или полученный отметку времени для каждого элемента (если элемент имеет отметку времени "отправлен" и "получен", выбирается самая старая дата). Затем Microsoft 365 ищет значение года для этой даты и сравнивает его с текущей датой, чтобы определить возраст элемента. Затем эти возрасты используются в качестве значений в выпадаемом списке фильтра **"Возраст".** Например, если В PST-файле есть сообщения от 2016, 2015 и 2014, то значения в фильтре "Возраст" будут иметь значения **1 год,**  **2 года** и **3 года.**
