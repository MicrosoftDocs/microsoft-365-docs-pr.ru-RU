# <a name="filter-data-when-importing-pst-files"></a>Фильтрация данных при импорте PST-файлов

Используйте новую функцию "Интеллектуальный импорт" в службе импорта Office 365 для фильтрации элементов в PST-файлах, импортируемых в целевые почтовые ящики. Вот как это работает:
  
- После создания и отправки задания импорта PST файлы PST загружаются в область хранения Azure в облаке Microsoft.
  
- Microsoft 365 анализирует данные в PST-файлах безопасно и безопасно, определяя возраст элементов почтовых ящиков и различные типы сообщений, включенных в PST-файлы.
  
- Когда анализ завершен и данные готовы к импорту, у вас есть возможность импортировать все данные в PST-файлах, как это есть, или обрезать импортируемые данные, установив фильтры, которые контролируют импортируемые данные. Например, вы можете выбрать:
  
  - Импортировать только элементы определенного возраста.
  
  - Импорт отдельных типов сообщений.
  
  - Исключить сообщения, отправленные или полученные определенными людьми.
  
- После настройки параметров фильтра Microsoft 365 импортировать только данные, отвечающие критериям фильтрации, в целевые почтовые ящики, указанные в заданиях импорта.
  
На следующем рисунке показан процесс интеллектуального импорта, а также выделены выполняемые задачи и задачи, выполняемые Office 365.

## <a name="filter-data-that-gets-imported-to-mailboxes"></a>Фильтрация данных, импортируемых в почтовые ящики

После создания задания импорта PST выполните следующие действия, чтобы отфильтровать данные, прежде чем импортировать их в Office 365.
  
1. Перейдите в <https://compliance.microsoft.com> и войдите в систему, используя учетные данные для учетной записи администратора в вашей организации.
    
2. В левой области центра соответствия Microsoft 365 нажмите кнопку **Импорт управления** \> **информацией.**
    
    Задания импорта для организации перечислены на вкладке **Импорт.** Завершенное **значение Analysis** в столбце **Состояние** указывает задания импорта, которые были проанализированы Microsoft 365 и готовы к импорту.
    
    ![Полный анализ состояния показывает, что Microsoft 365 проанализировал данные в PST-файлах](../media/de5294f4-f0ba-4b92-a48a-a4b32b6da490.png)
  
3. Выберите задание импорта, которое необходимо выполнить, и нажмите **кнопку Импорт для Office 365.**
  
    Появится страница со сведениями о PST-файлах и другой информацией о задании импорта.

4. Нажмите **кнопку Импорт Office 365**.
    
    Откроется страница **Отфильтровать данные**. Он содержит сведения о данных в файлах PST для задания импорта, в том числе сведения о возрасте данных. 
    
    ![На странице Filter your data показаны сведения о PST-файлах для задания импорта](../media/3b537ec0-25a4-45a4-96d5-a429e2a33128.png)
  
5. На основании того, хотите ли вы обрезать данные, импортируемые в Microsoft 365, в соответствии с тем, хотите ли вы фильтровать **данные?**, сделать одно из следующих:
  
    а) Нажмите **кнопку Да, я хочу отфильтровать его перед импортом,** чтобы обрезать импортируемые данные, а затем нажмите **кнопку Далее**.
  
    Импорт **данных для Office 365** страницы отображается с подробными сведениями о данных из анализа, Microsoft 365 выполнен. 
  
    ![Microsoft 365 отображает подробные сведения о данных из анализа PST-файлов](../media/4881205f-0288-4c32-a440-37e2160295f2.png)
  
    На графике на этой странице показан объем импортируемых данных. Сведения о каждом типе сообщений, найденных в PST-файлах, отображаются на графике. Курсор можно навести на каждую планку, чтобы отобразить определенную информацию об этом типе сообщения. Существует также выпаданий список с различными значениями возраста на основе анализа PST-файлов. При выборе возраста в выпадаемом списке график обновляется, чтобы показать, сколько данных будет импортироваться для выбранного возраста. 
  
    б) Чтобы настроить фильтры добавления, чтобы уменьшить объем импортируемых данных, нажмите кнопку **Дополнительные параметры фильтрации.**

    Можно настроить эти фильтры:
  
      - **Возраст** . Выберите возраст, поэтому импортировать будут только те элементы, которые являются более новыми, чем указанный возраст. Дополнительные [сведения см.](#more-information) в разделе Описание того, Microsoft 365 определяет возрастные ведра для **фильтра Age.** 
  
      - **Тип** . В этом разделе показаны все типы сообщений, найденные в PST-файлах для задания импорта. Можно отсеять поле рядом с типом сообщения, который необходимо исключить. Нельзя исключить другой тип сообщения. Дополнительные [сведения см.](#more-information) в разделе Дополнительные сведения о списке элементов почтовых ящиков, включенных в другую категорию.

Чтобы исключить людей, которые отображаются в поле From: Field, To:  field или Cc: поле сообщений, нажмите кнопку Исключить пользователей рядом с этим типом получателя. Введите адрес электронной почты (SMTP-адрес) пользователя, нажмите кнопку Добавить значок New, чтобы добавить их в список исключенных пользователей для этого типа получателя, а затем нажмите кнопку  ![ ](../media/457cd93f-22c2-4571-9f83-1b129bcfb58e.gif) **Сохранить,** чтобы сохранить список исключенных пользователей.

        > Microsoft 365 не показывает данные, полученные в результате установки **фильтра People.** Однако, если установить этот фильтр, чтобы исключить сообщения, отправленные или полученные определенными людьми, эти сообщения будут исключены во время фактического процесса импорта. 
  
    В. Щелкните **Применить** на странице **Дополнительные параметры фильтрации,** чтобы сохранить параметры фильтра. 
  
    Сведения о данных на странице **Импорт** Office 365 обновляются на основе параметров фильтра, включая общий объем данных, которые будут импортироваться в зависимости от параметров фильтра. Также показан сводка параметров фильтра. Вы можете щелкнуть **Кнопку Изменить** рядом с фильтром, чтобы при необходимости изменить параметр. 
  
    ![Сведения о данных обновляются в зависимости от параметров фильтра](../media/897e20fb-3b13-44c3-9d56-9f330750f2a3.png)
  
    г. Нажмите **Далее**.
  
    Отображается страница состояния с указанием параметров фильтра. Опять же, вы можете изменить любой из параметров фильтра.
  
    д. Нажмите **кнопку Импорт данных** для запуска импорта. Отображается общий объем импортируемых данных. 
  
    или
  
    а) Нажмите **кнопку Нет,** я хочу импортировать все данные в PST-файлах для Office 365, а затем нажмите **кнопку Далее**.

6. На **вкладке Импорт** нажмите обновить **обновление** ![ ](../media/165fb3ad-38a8-4dd9-9e76-296aefd96334.png) . Состояние задания импорта отображается в столбце **Состояние.**
  
7. Щелкните импорт задания, чтобы отобразить более подробные сведения, например состояние для каждого PST-файла и настроенные параметры фильтра.

## <a name="more-information"></a>Дополнительные сведения

- Как Microsoft 365 определить приращения для возрастного фильтра? Когда Microsoft 365 PST-файл, он смотрит на отправленный или полученный отпечаток времени каждого элемента (если элемент имеет как отправленный, так и полученный период времени, выбирается самая старая дата). Затем Microsoft 365 значение года для этого периода времени и сравнивает его с текущей датой, чтобы определить возраст элемента. Эти возрасты затем используются в качестве значений в списке выпаданий для **фильтра Age.** Например, если в PST-файле есть сообщения от 2016, 2015 и 2014 г., то значения в фильтре **Age** будут **1 год,** **2** года и **3 года.**
