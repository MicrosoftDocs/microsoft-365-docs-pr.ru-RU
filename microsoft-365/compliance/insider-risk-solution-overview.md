# <a name="insider-risk-solutions-in-microsoft-365"></a>Решения для оценки рисков в Microsoft 365

Риски, связанные с предварительной версией, — это одна из главных проблем специалистов по обеспечению безопасности и соответствия требованиям на современном рабочем месте. Отраслевые исследования показывают, что риски, связанные с предварительным использованием, часто связаны с определенными событиями пользователей или действиями. Защита Организации от этих рисков может быть проблематичной для выявления и трудностей по устранению проблем. Риски, связанные с предварительной версией, включают уязвимости в различных областях и могут привести к существенным проблемам в Организации, от потери данных о интеллектуальной собственности на место и т. д. На следующем рисунке показаны распространенные риски, связанные с предварительным обзором:

![Угрозы для оценки риска](../media/ir-solution-threats.png)

Функции предотвращения риска Microsoft 365 разработаны и встроены в продукты и решения, подверженные предварительной программе. Эти решения работают совместно и используют расширенные индикаторы служб и сторонних производителей для быстрого определения, рассмотрения и действия рисков. Большинство решений предоставляют исчерпывающие рабочие процессы обнаружения, оповещений и исправлений для аналитик данных и изучений, которые используются для быстрого выполнения и минимизации этих рисков.

## <a name="microsoft-365-insider-risk-solutions"></a>Решения Microsoft 365 для оценки рисков

Чтобы помочь защитить организацию от угроз, используйте следующие возможности и функции Microsoft 365.

| **Решение и возможности** | **Описание** | **Лицензирование** |
| :------------------------ | :-------------- | :------------ |
| Соответствие требованиям к обмену данными | Обеспечение соответствия требованиям помогает свести к минимуму риски для общения, помогая обнаруживать, записывать и обрабатывать неуместные сообщения в Организации. | Microsoft 365 E5 <br> Подписка на Microsoft 365 E3 и надстройка "Управление рисками для участников программы предварительной оценки Майкрософт 365" <br> Подписка на Microsoft 365 E3 + надстройка для обеспечения соответствия требованиям корпорации Майкрософт 365 |
| Внутреннее управление рисками | Средство предварительной оценки риска позволяет минимизировать внутренние риски, позволяя обнаруживать, изучать и исследовать от вредоносных и неслучайных действий в Организации. | Microsoft 365 E5 <br> Подписка на Microsoft 365 E3 и надстройка "Управление рисками для участников программы предварительной оценки Майкрософт 365" <br> Подписка на Microsoft 365 E3 + надстройка для обеспечения соответствия требованиям корпорации Майкрософт 365 |
| Информационные барьеры | С помощью информационных барьеров можно ограничить обмен данными и совместную работу между двумя внутренними группами, чтобы избежать конфликта интересов в Организации.  | Microsoft 365/A5/A5 <br> Office 365/A5/A5 <br> Office 365 Advanced Compliance <br> Соответствие требованиям Microsoft 365 = A5/A5 <br> Управление рисками для участников программы предварительной оценки Microsoft 365 |
| Управление привилегированным доступом | Управление привилегированным доступом обеспечивает возможность детализированного управления доступом к привилегированным задачам администратора Exchange Online в Office 365. Это поможет защитить организацию от нарушений, которые используют существующие привилегированные учетные записи администратора с ограниченным доступом к конфиденциальным данным или к критически важным параметрам конфигурации. | Microsoft 365/A5/A5 <br> Office 365/A5/A5 <br> Соответствие Microsoft 365/A5 <br> Microsoft 365 и управление информацией об информационной защите и A5 |
