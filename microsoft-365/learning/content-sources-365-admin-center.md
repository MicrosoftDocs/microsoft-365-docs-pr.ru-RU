# <a name="configure-learning-content-sources-for-microsoft-viva-learning-preview-in-the-microsoft-365-admin-center"></a>Настройка источников учебного контента для Microsoft Viva Обучение (Preview) в Центр администрирования Microsoft 365

> [!NOTE]
> Сведения в этой статье относятся к продукту предварительного просмотра, который может быть существенно изменен до его коммерческого выпуска. 

Администраторы для Центр администрирования Microsoft 365 самостоятельно или путем назначения роли администратора знаний отдельным лицам в вашей организации могут управлять настройками, связанными с Viva Обучение (Preview) и настраивать источники контента для обучения.

Администратор выбирает, какие другие источники контента для обучения (например, SharePoint или поддерживаемые сторонними источниками поставщиков контента) будут доступны пользователям Viva Обучение (Preview). Затем администратор настраивает эти источники, чтобы убедиться, что содержимое доступно для поиска и обнаружения и может просматриваться сотрудниками, которые используют Viva Обучение (Preview).

> [!NOTE]
>  Пользователи во время входов в не microsoft и LinkedIn Обучение Pro в браузере или встроенном обозревателе. Это настраиваемое обучение подлежит раздельным условиям лицензии, конфиденциальности и службы между организацией и третьей стороной, а не условиям Viva Обучение (Preview). Прежде чем выбирать этот тип обучения, убедитесь, что у вас есть соглашение для вашей организации и пользователей.

## <a name="assign-the-knowledge-admin-role-optional"></a>Назначение роли администратора знаний (необязательный)

Вы должны быть глобальным администратором Microsoft 365 для выполнения этих задач.

> [!TIP]
> Администратор знаний должен быть в меру техническим и иметь существующие SharePoint учетные данные администратора, желательно тех, кто хорошо разбирается в образовании, обучении, обучении или опыте сотрудников в организации.

### <a name="add-a-knowledge-admin"></a>Добавление администратора знаний

Чтобы добавить администратора знаний для Viva Обучение (Preview), выполните следующие действия:

1. В левой навигации Центр администрирования Microsoft 365 перейти к **ролям**.

2. На странице **Роли** на вкладке **Azure AD** выберите **Администратор знаний.**
 
3. На панели **Администратор знаний** выберите **назначенного администратора,** а затем выберите **Добавить**.

     ![На странице Роли в Центр администрирования Microsoft 365 отображается панель администратора знаний для добавления пользователя.](../media/learning/learning-add-knowledge-admin-1.png)

3. На панели **Добавить администраторов** выберите выбранного вами человека для роли, а затем выберите **Добавить**.

     ![Страница Ролей в Центр администрирования Microsoft 365, на которой отображается панель Добавить администраторов для добавления пользователя.](../media/learning/learning-add-knowledge-admin-2.png)

### <a name="remove-a-knowledge-admin"></a>Удаление администратора знаний

Чтобы удалить администратора знаний для Viva Обучение (Preview), выполните следующие действия:

1. В левой навигации Центр администрирования Microsoft 365 перейти к **ролям**.

2. На странице **"Роли"** на **вкладке Azure AD** выберите **администратора знаний.**
 
3. На панели **Администратор знаний** на вкладке **Назначены** администраторы выберите **Удалить,** а затем выберите человека, которого необходимо удалить из роли. Чтобы подтвердить, выберите **Удалить**.

     ![Страница Ролей в Центр администрирования Microsoft 365, на которой отображается назначенная панель администраторов для удаления пользователя.](../media/learning/learning-remove-knowledge-admin-1.png)

## <a name="configure-settings-for-the-learning-content-sources"></a>Настройка параметров для источников учебного контента

Вы должны быть глобальным администратором Microsoft 365 или администратором знаний для выполнения этих задач.

Чтобы настроить параметры для источников контента в Viva Обучение, выполните следующие действия:

1. В левой навигации Центр администрирования Microsoft 365 перейдите **в Параметры**  >  **параметров Org.**

2. На странице **Параметры Org** на вкладке **Services** выберите **Viva Обучение (Preview)**.

     ![Параметры страницы в Центр администрирования Microsoft 365, на которой отображается Обучение приложение.](../media/learning/learning-sharepoint-configure1.png)

3. На панели **Viva Обучение (Preview)** выберите источники учебного контента, которые необходимо настроить для организации, а затем выберите **Сохранить**.

     ![Обучение панели в Центр администрирования Microsoft 365 с вариантами источников контента.](../media/learning/learning-sharepoint-configure2.png)

Среди всех источников обучения, которые существуют, некоторые из них будут включены по умолчанию. Эти источники обучения включают в себя:

- LinkedIn Обучение (бесплатный контент)
- Microsoft Learn
- Microsoft 365 Обучение

> [!NOTE]
> Бесплатный контент LinkedIn предоставляется пользователям в соответствии с политиками конфиденциальности и пользовательским соглашением LinkedIn. LinkedIn получит IP-адрес пользователя, любые файлы cookie, ранее установленные LinkedIn, и застроит новый файл cookie для отслеживания использования бесплатного контента. Для получения бесплатного контента пользователям не требуется вход в LinkedIn.<br><br>
Для премиум-контента LinkedIn вашей организации нужна подписка для вашей команды, чтобы получить доступ к этому контенту. Пользователи должны будут войти в LinkedIn, чтобы получить доступ к этому обучению, которое предоставляется в соответствии с условиями организации и условиями пользователей с LinkedIn.<br><br> Для контента, не связанного с Microsoft (за исключением бесплатного контента LinkedIn), убедитесь, что ваша организация имеет подписку на доступ к этому контенту с помощью учетной записи перед подключением к Viva Обучение (Preview). Личные подписки пользователей на поставщиков обучения, не в microsoft, не будут интегрированы с Viva Обучение (Preview). Пользователи во время входов в не microsoft и LinkedIn Обучение Pro в браузере или встроенном обозревателе. Если пользователи переходят к контенту, где у них нет организационной подписки, они могут увидеть страницу поставщика, на которой они могут зарегистрироваться для отдельной подписки. Все обучение, не в рамках Microsoft, предоставляется в соответствии с условиями поставщика, не в рамках Viva Обучение. 

Чтобы включить или отключить источник контента для обучения, выберите контрольный ящик рядом с источником. Если включен источник, будет виден контрольный знак.

## <a name="third-party-content-providers"></a>Сторонние поставщики контента 

Набор доступных подключенных поставщиков обучения может измениться в любое время. По мере роста программы к нему присоединятся все больше поставщиков. Доступные поставщики также могут отказаться от подключения к Viva Обучение (Preview).

### <a name="skillsoft-as-a-content-source"></a>Skillsoft как источник контента  

Для Viva Обучение (Preview) пользователи, у которых включена возможность skillsoft и которые выбирают просмотр контента Skillsoft, высадят на страницу Percipio, которая попросит их ввести имя сайта Percipio организации. После ввода имени сайта организации пользователи будут направлены на страницу для входа на сайт Percipio организации. Пользователи впишутся с помощью существующих учетных данных и увидят содержимое, которое они изначально выбрали. Пользователям будет предложено ввести имя сайта Percipio только один раз, пока кэш браузера не будет очищен. Чтобы упростить этот процесс для пользователей, рекомендуется включить имя сайта Percipio во внутренние сообщения, которые вы отправляете о viva Обучение (Preview).

Это будет временный опыт предварительного просмотра, и мы работаем с Skillsoft, чтобы включить интеграцию с клиентом для общей доступности, которая позволит обойти шаг, который требует, чтобы пользователи предоставили имя сайта Percipio вашей организации. 

### <a name="details-on-microsoft-substrate"></a>Сведения о субстрате Microsoft  

Для данных, которые вы копируете в Viva Обучение (Preview) из службы, не в microsoft (поставщик обучения или система управления обучением), вы не можете напрямую извлекать, исправлять или удалять эти данные в Viva Обучение (Preview). Мы обновляем данные, которые вы импортируете из поставщиков, не в корпорации Майкрософт, для отражения изменений и удалений в исходных данных, не в microsoft.

Вам необходимо работать с поставщиком службы, не входя в Корпорацию Майкрософт, для доступа, исправления, удаления или извлечения данных в соответствии с условиями лицензии, службы или конфиденциальности службы, не входя в Корпорацию Майкрософт. Внесенные изменения будут отражены в данных, обработанных для вашего использования в Viva Обучение (Preview) после завершения циклов обновления данных службы не Microsoft и Viva Обучение (Preview). Если вы отключите подключение между службой Viva Обучение (Preview) и службой, не в microsoft, все данные, которые вы ранее импортировали из этой службы, будут удалены.
