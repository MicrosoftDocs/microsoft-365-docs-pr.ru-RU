# <a name="test-base-sdk-for-python"></a>Тестовый базовый SDK для Python

## <a name="overview"></a>Обзор
SDK тестовой базы можно использовать для взаимодействия с ресурсом тестовой базы Azure. (например, управление пакетом приложений, включите создание пакета, изменение пакета и удаление пакета)

С помощью SDK можно получить сводку тестов и результаты анализа, которые можно получить: scriptExecution, надежность, memoryUtilization, cpuUtilization, memoryRegression, cpuRegression.

С помощью SDK тестовой базы можно интегрировать тестовую базу в конвейер ci/CD.

## <a name="client-library"></a>Клиентская библиотека

Установите тестовый базовый пакет с пунктом.

~~~
pip install  Microsoft.Testbase
~~~
