# <a name="windows-server-application-testing"></a>Windows Тестирование приложений на сервере 

С помощью тестовой базы для Microsoft 365 теперь вы можете проверить свои приложения в отношении Windows Server 2016 и 2019, в том числе Server Core!

Чтобы начать проверку загруженных приложений от предварительных обновлений для операционных систем Windows Server 2016 и 2019 г. в тестовой базе для Microsoft 365, следует соблюдать следующие действия:

1.   Войдите на наш портал самообслуживки. В левом меню навигации выберите ```Upload new package``` пункт "Под" ```Package catalogue``` и заполните сведения о тесте.

2.  Выберите ```Security updates``` в качестве типа обновления ОС:

![Выбор обновлений безопасности](Media/selecting-security-updates.png)

3. В версиях ОС для тестирования выберите применимые версии ОС. Вы можете выбрать Windows версии ОС сервера или сочетание версий серверной и клиентской ОС.

![Выбор версии ОС](Media/selecting-OS-versions.png)

4. Предосмотрите другие необходимые сведения, просмотрите предоставленные сведения и загрузите пакет приложений. После отправки можно просмотреть состояние пакета на вкладке Меню Управление пакетами.


5. Чтобы просмотреть результаты тестов и сведения о проверке приложения на фоне предзапустимых обновлений безопасности за Windows Server 2016 и 2019 г., перейдите на страницу Сводка тестов или страницу результатов обновления безопасности.

![Просмотр результатов тестирования](Media/access-test-results.png)
